
# Энергетики придумали комплекс мер по отключению химзавода Азот в Северодонецке

Published at: **2019-11-07T13:25:00+00:00**

Author: ****

Original: [ОстроВ](https://www.ostro.org/lugansk/society/news/579213/)

Луганское энергетическое объединение планирует отключить энергоснабжение химзаводу "Азот" в Северодонецке. Об этом пишет пресс-служба энергокомпании.
"Азоту" прислано предупреждение об обесточивании 20.11.19. Также разработан комплекс мер по отключению должника", - сказано в сообщении.
