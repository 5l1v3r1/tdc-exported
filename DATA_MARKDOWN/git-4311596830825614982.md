
# В Москве неизвестный угрожает взорвать торговый центр с помощью дронов

Published at: **2019-11-04T13:18:00+00:00**

Author: ****

Original: [EADaily](https://eadaily.com/ru/news/2019/11/04/v-moskve-neizvestnyy-ugrozhaet-vzorvat-torgovyy-centr-s-pomoshchyu-dronov)

Неизвестный потребовал у владельцев ТЦ «Калейдоскоп» 10 биткоинов. В случае отказа он угрожал взорвать центр с помощью дронов. Как сообщает Агентство городских новостей «Москва», по данному факту полиция проводит проверку.
Промышленно-финансовая группа «Сафмар» — российский многоотраслевой конгломерат. Группа «Сафмар» основана одним из крупнейших российских предпринимателей Михаилом Гуцериевым.
