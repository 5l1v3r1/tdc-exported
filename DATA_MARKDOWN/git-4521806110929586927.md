
# Ученые создали новые катализаторы для получения биоактивных молекул

Published at: **2019-11-04T09:22:00+00:00**

Author: ****

Original: [RuNews24](https://runews24.ru/science/04/11/2019/dcd7717933a4530436db3458025a9eaf)

Квантовые точки по своим действиям больше напоминают органические молекулы, чем наночастицы металлов. Их электроны сжаты в небольшом пространстве, а активность подчиняется законам квантовой механики.
Во время синтеза к катализаторам присоединяются молекулы реагентов, энергия света активирует реакционные центры, чтобы процесс смог двигаться в нужном направлении.
Крупная молекула легко отделяется от наночастицы, которая освобождает катализатор, чтобы участвовать в другом химпроцессе. Этот метод позволит разработать новые лекарства.
