
# На железнодорожном переезде в Донском погибла пенсионерка

Published at: **2019-11-03T13:58:00+00:00**

Author: ****

Original: [Тульская Служба Новостей](https://www.tsn24.ru/na-zheleznodorozhnom-pereezde-v-donskom-pogibla-pensionerka.html)

Пенсионерка погибла на железнодорожном переезде в городе Донской Тульской области днем 2 ноября.
Инцидент произошел у железнодорожной станции 23 км. По сообщению очевидцев, пожилую женщину сбил рельсовый автобус.
