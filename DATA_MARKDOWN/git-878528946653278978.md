
# １位にならないとブログ閉鎖の小柳ルミ子に、１位のモト冬樹「ブログって何だろう」

Published at: **2019-11-05T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/05/0012851279.shtml)

　歌手の小柳ルミ子（６７）が２日放送の日本テレビ系「有吉反省会」で、１週間以内にアメブロの大御所部門ランキングでタレント・モト冬樹（６８）に勝てなければブログ閉鎖を言い渡されたことについて、モトが５日に更新した公式ブログで「ブログって何だろう」と、複雑な心境をつづっている。
　モトは「今回　有吉反省会の禊で小柳ルミ子ちゃんが大御所部門で一位にならないとブログが閉鎖されてしまうなんてことが起きちゃって　改めてブログって何だろうって考えさせられたな」と打ち明けた。
　モト自身はブログについて「俺のブログは自分のために書いている」、「ブログって公のものだけどすごく個人的なもののよう気がする」、「本当のこといって何位とかいう順位は気にもならないしどうでもいいことだと思ってる」と見解を示し、「大御所部門で一位になれないとブログが閉鎖になってしまうならルミ子ちゃんには是非一位になって欲しい」と、小柳にエールを送った。
　とはいえ、あくまでもユーザー次第とあって「どう手助けしていいかわからないけど　俺はいままでのペースで楽しくブログを書いていくしかないしね」とも付け加えている。
　小柳は「有吉反省会」で「大御所部門で最近まで不動の２位だったが、高橋親子（高橋英樹・真麻）が登場して２位を取られた」と落胆し「ずるいの。親子でやってる」と嫉妬。同部門ではモトが不動の１位で、小柳はこれにも「許せないの。だいたい冬樹がなんなの？ずるいのは、奥さんがいて写真も撮る、ペットもいる。家庭料理、くっだらない内容だよ？それで１位は許せない」とぶちまけ、「私は１人よ。ファミリーもいないし、愛犬も去年死んだし…。どうしたらいいの？」と、弱気なコメントも出た。
　そんな小柳のみそぎとして掲げられたのが「１週間以内にモト冬樹さんにブログランキングで勝てなければブログ閉鎖」。５日に更新された同部門のランキングは１位がモト、２位が小柳、３位が英樹となっている。
