
# ロッテ福浦２軍コーチ、背番号７０に　「プロ野球人生スタートさせた番号、うれしい」

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/01/0012839054.shtml)

　ロッテは１日、一部コーチ陣の背番号を発表した。
　福浦和也二軍ヘッド兼打撃コーチは「７０」、伊志嶺翔大走塁コーチ兼打撃コーチ補佐兼外野守備コーチ補佐は「８１」、根元俊一二軍内野守備走塁コーチは「８７」となる。
　福浦二軍ヘッドは「第２の人生のスタートで自分の原点ともいえるプロ野球人生をスタートさせた番号を背負うことができることをうれしく思います。根元コーチには申し訳ない気持ちですが、譲っていただけて感謝をしています。懐かしくもあり新鮮でもあります。
　これから指導者として一人でも多くの選手たちを一軍に送り出せるようにバックアップしていけたらと思います」とコメントした。
