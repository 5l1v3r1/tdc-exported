
# MIT и Ford представили новый метод навигации для роботов-курьеров

Published at: **2019-11-06T03:37:03+00:00**

Author: **Игорь Савкин**

Original: [Код Дурова](https://kod.ru/3683/)

Инженеры из Массачусетского технологического института (MIT) и Ford Motor разработали новый метод ориентирования для роботов-доставщиков. Благодаря новому методу навигации роботы во время движения ориентируются по специальным подсказкам на местности без помощи GPS.
Подсказки описаны в общих для различных ситуаций терминах — это может быть входная дверь, гараж, заметные здания или другие подобные объекты.
В роботах-доставщиках установлены камеры, которые с помощью искусственного интеллекта маркируют окружающую среду, выделяя на ней семантические подсказки, такие как «тротуар» или «дверь». Затем нейросеть на основе подсказок строит карту, по которой робот двигается к месту назначения.
Даже если робот доставляет посылку в район, в котором он никогда не был, то сможет ориентироваться по подсказкам, которые уже видел в других местах.
