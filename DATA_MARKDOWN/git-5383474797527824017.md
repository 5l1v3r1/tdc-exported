
# Apple попросила Трампа отменить пошлины на Apple Watch, AirPods и на запчасти для iPhone

Published at: **2019-11-02T08:00:31+00:00**

Author: **Илья Сидоров**

Original: [iPhones.ru](https://www.iphones.ru/iNotes/apple-poprosila-trampa-otmenit-poshliny-na-apple-watch-airpods-i-na-zapchasti-dlya-iphone-11-01-2019)

Apple направила запрос в Торговое представительство США с просьбой исключить часть продуктов из пошлин, которые ввёл Дональд Трамп на товары, импортируемые из Китая.
С 1 сентября Apple платит 15% за ввоз некоторых устройств на территорию США. Компания хочет избавиться от налога, поскольку она не может перенести всё производство за пределы Китая. Это одно из условий, которое позволяет американским корпорациям добиться отмены тарифов.
Apple хочет исключить из импортных пошлин следующие товары:
▪️ Apple Watch▪️ AirPods▪️ HomePod▪️ Beats Pill+▪️ Наушники Beats▪️ iMac▪️ Зарядные кейсы для PowerBeats Pro и AirPods▪️ Жесткие диски для Mac Pro▪️ Аккумуляторы для iPhone и MacBook
По словам Тима Кука, компания уже платит дополнительные налоги на некоторые продукты.
Ранее Apple просила отменить пошлины на компоненты для Mac Pro. Власти США частично удовлетворили запрос Apple, но ей пришлось перенести производство Mac Pro из Китая в США. [Bloomberg]
