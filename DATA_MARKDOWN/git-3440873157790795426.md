
# Воду отключат в нескольких микрорайонах Алматы

Published at: **2019-11-06T03:20:00+00:00**

Author: ****

Original: [Tengrinews.kz](https://tengrinews.kz/kazakhstan_news/vodu-otklyuchat-v-neskolkih-mikrorayonah-almatyi-383383/)

В двух районах Алматы временно отключат холодную воду, передает Тengrinews.kz со ссылкой на "Алматы Су".
По данным организации, подача холодной воды будет прекращена в жилые дома и иные объекты в  Ауэзовском и Турксибском районах.
7 ноября с 9.00 до 20.00 вода будет недоступна для жителей микрорайонов Мамыр-1, Мамыр-2, Мамыр-3, Мамыр-4, Мамыр-7, Астана и Школьник.
С 9.00 до 17.00 воду отключат для жителей микрорайона Жас Канат.
8 ноября с 9.00 до 17.00 холодная вода будет недоступна для жителей в квадрате улиц Кассина -  Сейфуллина - Ровенского - Арктическая.
"Временные ограничения по услуге холодного водоснабжения связаны с подключением новых водопроводов к магистральным сетям. Просим с пониманием отнестись к временным неудобствам", - сообщили в "Алматы Су".
