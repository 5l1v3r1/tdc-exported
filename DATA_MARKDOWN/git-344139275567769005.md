
# Deux corps récupérés après le crash d’un hélicoptère transportant 7 personnes en Corée du Sud

Published at: **2019-11-03T07:52:00+00:00**

Author: **Belga**

Original: [sudinfo.be](https://www.sudinfo.be/id149757/article/2019-11-03/deux-corps-recuperes-apres-le-crash-dun-helicoptere-transportant-7-personnes-en)


        Les équipes de sauvetage ont retrouvé deux corps dans les eaux bordant la côte est de la Corée du Sud après le crash d’un hélicoptère transportant sept personnes plus tôt cette semaine, rapportent les médias locaux.
      
L’appareil venait de récupérer un pêcheur blessé sur les îles Dokdo, appelées Takeshima par Tokyo qui dispute ces îlots à son voisin coréen, quand il s’est abîmé jeudi soir. Les raisons de ce crash demeurent pour l’instant inconnues.
Les gardes-côtes ont retrouvé deux corps, masculins, et les ont transportés à la morgue, rapporte l’agence de presse Yonhap.
Les sauveteurs ont aussi repéré un troisième corps dans le fuselage de l’appareil abîmé.
Des douzaines de soldats et gardes-côtes sont mobilisées pour mener les recherches assistées de 25 plongeurs.
