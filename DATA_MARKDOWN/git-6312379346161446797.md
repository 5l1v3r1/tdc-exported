
# Росгвардейцы задержали мужчину с ножами и гранатой в Туле

Published at: **2019-11-03T13:13:23+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-03/rosgvardeycy-zaderzhali-muzhchinu-s-nozhami-i-granatoy-v-tule)

В Туле сотрудники Росгвардии задержали 54-летнего безработного мужчину, который ходил по улице с тремя ножами, ручной гранатой и магазином с боеприпасами.
О подозрительном гражданине на пульт дежурного вневедомственной охраны сообщили очевидцы. 
Задержанного передали следственно-оперативной группе для разбирательства.
Как сообщили в ведомстве, задержанный ранее был судим. По факту произошедшего проводится проверка.
