
# OPSTELLING: Antwerp hoopt op Le Canonnier verder weg te lopen van Moeskroen

Published at: **2019-11-02T19:00:00+00:00**

Author: ****

Original: [VP VoetbalPrimeur.be](https://www.voetbalprimeur.be/nieuws/902078/opstelling-moeskroen-antwerp.html)


        Moeskroen ontvangt op Le Canonnier Royal Antwerp Football Club. Scheidsrechter van dienst is Jan Boterberg.
      
Moeskroen kan al zes wedstrijden niet winnen in de Jupiler Pro League. Afgelopen speeldag speelde het 2-2 gelijk tegen rode lantaarn Cercle Brugge.
