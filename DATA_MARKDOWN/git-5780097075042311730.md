
# ​ម៉ាលី៖ ពួក​ជីហាត​ទទួល​ខុស​ត្រូវលើការ​វាយ​ប្រហារ​សម្លាប់​ទាហាន៤៩នាក់ មាន​​បារាំង​ម្នាក់​ផង​

ប្រតិបត្តិការ​សម្លាប់​ទាហាន​នៅ​ម៉ាលីអស់​ជិត​៥០នាក់​កាលពីថ្ងៃ​សុក្រ គឺ​មា​នគោល​ដៅ​សម្លាប់​កង​កម្លាំង​​រក្សា​សន្តិភាព​ និង​ទាហាន​អន្តរជាតិដែល​ដឹកនាំ​ដោយ​កងទ័ព​បារាំង នៅក្នុង​តំបន់អាហ្វ្រិក​​ ដើម្បី​ប្រឆាំង​នឹង​ពួក​ភេរវជន។ នេះ​បើតាម​ខ្លឹមសារ​នៃ​សេចក្តីថ្លែងការណ៍​របស់​ពួក​ភេរវជន​ជីហាត ដែល​ប៉ូលីសស៊ើប​អង្កេត​​រក​ឃើញ​នៅក្នុ​ង​ប្រព័ន្ធ​តេឡេក្រាម។ ប្រភព​ព័ត៌មាន​ជាច្រើន​បញ្ជាក់ថា ប្រតិ​បត្តិ​ការ​វាយ​ប្រហារសម្លាប់​ទាហានលើ​ក​​នេះ គឺ​រៀបចំ​ដឹកនាំ​ដោយ​មេភេរវជន​ជីហាត​លំដាប់​ទី២ គឺ​ Abdel Hakim al-Sahraoui ដែល​គេ​ស្គាល់​ច្បាស់​ថាក្បាល​ម៉ាស៊ីន​ជីហាត​មាន​ឥទ្ធិពល​ខ្លាំងនៅ​ក្នុង​ទីតាំង​នោះ។
នៅ​ប្រទេស​ម៉ាលី តំបន់​ភាគ​ខាង​ជើង​ដែល​ជា​កន្លែង​រងគ្រោះ​នេះ ជា​ទីតាំង​គ្មាន​សុវត្តិភាព​ទេ​ ពីព្រោះ​តំបន់​នេះ ធ្លាប់​បានត្រូវ​​ធ្លាក់​ក្នុង​កណ្តាប់​ដៃ​កាន់​​កាប់​ដោយ​ពួក​ឧទ្ទាម​ភេរវ​ករ​​អាល់កៃដា​ នៅឆ្នាំ​២០១២។ ហើយ​តាំង​តែ​ពី​ឆ្នាំ​២០១៣រហូត​មក បារាំង​បាន​បញ្ជូន​កងទ័ព​ទៅ​ឈរ​ជើងទីនោះ ដើម្បី​ជួយកង​​ទ័ពប្រទេស​​ម៉ាលី ប្រយុទ្ធ​ប្រឆាំង​នឹង​ភេរវជន​ និង​ស្វែង​រក​សន្តិភាពឱ្យ​​ក្នុង​តំបន់។ ក្នុង​​ប្រតិបត្តិការ​នៅម៉ាលី រហូត​មក​ដល់​ពេលនេះ ទាហាន​បារាំង បាន​បាត់បង់​ពលី​ជីវិត​ អស់​២៥នាក់​ហើយ រាប់ទាំង​ទាហានបារាំង​ម្នាក់ ស្លាប់​ពីថ្ងៃ​សុក្រ​នោះផង​។ ម៉ាលី​ ក្នុង​មួយ​ខែ​ចុងក្រោយ​នេះ មានការ​វាយ​ប្រហារ​​ភេរវកម្ម​ផ្ទួនៗគ្នា។​ តំណាង​បេសកជន​ពិសេស​អង្គការ​ប្រជាជាតិ​ នៅ​ម៉ាលី បញ្ជាក់ថា ទិន្នន័យ​​ចុងក្រោយ​ នៅឆ្នាំ​២០១៨ មាន​ការ​វាយ​ប្រហារ​២៣៧ដង សម្លាប់​ទាហាន​និងមនុស្ស​ អស់​ជាច្រើន៕​​
