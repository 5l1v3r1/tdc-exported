
# แซ่บ! 'ดาว' โพสต์แรงไม่สามารถเปลี่ยนสัตว์เป็นคนได้

Published at: **2019-11-04T10:13:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/entertainment/739965)


        ดาว-พิมพ์ทอง เพิ่มความแซ่บให้เผือกอีกรอบ หลังโพสต์แรงสนั่นทวิตเตอร์ ด้านชาวเน็ตโยงว่ากล่าวถึงรักครั้งเก่า
      
เพิ่งตัดสินใจบอกลาความรักที่ไม่ได้ดังใจกับอดีตแฟนหนุ่ม พอร์ช-ศรัณย์ ศิริลักษณ์ ไปหมาดๆ สำหรับนักแสดงสาวคนสวย ดาว-พิมพ์ทอง ล่าสุดดูเหมือนสาวดาวจะไปเจอเรื่องชวนเสียอารมณ์มาอีกแล้ว
หลังล่าสุดเจ้าตัวโพสต์แรงและแซ่บมากๆในทวิตเตอร์ โดยสาวดาวเขียนข้อความว่า “น้ำลดตอผุดมากๆๆๆๆๆๆๆๆ จริงที่ว่าเราไม่สามารถเปลี่ยน XX ให้กลายเป็นคนได้ ... และสุดท้ายมันก็ย้อนมากัดเรา” พอสาวดาวโพสต์ข้อความลงไป ก็มีชาวเน็ตมารีทวิตมากมาย และมีคนเข้ามาคอมเมนต์คาดเดาว่าสาวดาวน่าจะหมายถึงรักครั้งเก่าอีกด้วย
ก็ไม่รู้เนอะว่าสาวดาวหมายถึงอะไร ยังไงก็คงต้องรอให้สาวดาวออกมาบอกเรื่องนี้เองซะแล้ว และที่รู้ๆคือข้อความนี้แซ่บพริกสิบเม็ดมากๆจ้าขอขอบคุณภาพประกอบจากอินสตาแกรม @DaoPimthong
