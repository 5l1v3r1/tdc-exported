
# Испанский стартап представил электровелосипед с функцией распознавания лица

Published at: **2019-11-03T22:00:43+00:00**

Author: **Игорь Савкин**

Original: [Код Дурова](https://kod.ru/3677/)

Испанский велосипедный стартап представил новый электровелосипед X One. Сообщается, что в раме руля присутствует сенсорный дисплей с функцией распознавания, транспорт может распознавать лицо своего владельца, а также поддерживает множество других «умных» функций.
