
# حساب كنتاكي عبر تويتر يسخر من عرض ريال مدريد لضم رحيم ستيرلنج!

Author: **محمد ابو الوفا**

التقارير أشارت إلى نية النادي الملكي مبادلة جاريث بيل بستيرلنج إضافة إلى دفع مبلغ مالي يصل إلى 70 مليون يورو.
