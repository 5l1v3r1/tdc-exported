
# Microsoft обновила логотип браузера Edge

Published at: **2019-11-04T19:01:00+00:00**

Author: ****

Original: [AIN.UA](https://ain.ua/2019/11/04/microsoft-izmenila-logotip-edge/)

Компания Microsoft впервые за четыре года изменила логотип браузера Microsoft Edge. Букве «е» придали форму волны для серфинга и добавили несколько новых цветов. Теперь в нем больше нет отсылок к Internet Explorer, пишет The Verge.
Логотип создан в соответствии со стандартами Fluent Design, на которые ранее перевели офисный пакет MS Office.
Новый логотип появился в скрытой мини-игре про серфинг, которую разработчики спрятали внутри бета-версии браузера Edge 80.0.
В сети опубликовали серию загадок в виде изображений. Отгадку в виде различных слов нужно было ввести в функцию JavaScrip на сайте бета-версии Edge. После нескольких шагов появялись инструкции к секретной игре Let`s Surf, которую разместили по адресу edge://surf/.
После прохождения игры, пользователь видел уведомление о запуске нового лого:
Microsoft представит новый браузер Edge Chromium 15 января 2020 года.
