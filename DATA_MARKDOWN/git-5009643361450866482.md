
# Airbnb запретит сдавать жилье для проведения вечеринок

Published at: **2019-11-03T09:42:45+00:00**

Author: **Юлия Онодэра**

Original: [The Village](https://www.the-village.ru/village/city/news-city/366203-airbnb-bez-vecherinok)

Сервис аренды жилья Airbnb запретит сдавать дома и квартиры для проведения вечеринок, заявил генеральный директор компании Брайан Чески. Такое решение принято после того, как на одной из вечеринок в США были застрелены пять человек.
«Начиная с сегодняшнего дня мы запрещаем так называемые дома для вечеринок и удваиваем наши усилия по борьбе с незарегистрированными вечеринками и избавлению от оскорбительного поведения хозяев и гостей, которое приводит к таким ужасным событиям, какие случились в Оринде», — написал Чески в Twitter.
По словам Чески, Airbnb ужесточит ручную проверку бронирований с высоким риском, определяемых технологией обнаружения рисков, создаст специальную группу реагирования для контроля за вечеринками и будет принимать немедленные меры против пользователей, которые нарушают правила.
