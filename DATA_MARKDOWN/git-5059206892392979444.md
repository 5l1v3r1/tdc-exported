
# Transportistas anuncian bloqueos para el jueves 7 de noviembre

Published at: **2019-11-06T02:18:21+00:00**

Author: ****

Original: [Canal Antigua](https://www.canalantigua.tv/transportistas-anuncian-bloqueos/)

Transportistas informaron sobre bloqueos en distintos puntos el próximo jueves 7 de noviembre, a partir de las 9:30 am.
Según dio a conocer, el motivo de los mismos se debe a que no se ha autorizado el ajuste del seguro para transporte de productos derivados del petróleo.
Así como también, denuncian que la SAT no permite las modificaciones de cabezal a camión para las unidades de transporte.
Por otra parte, se explica que se realizarán caravanas simultáneas en diferentes puntos.
