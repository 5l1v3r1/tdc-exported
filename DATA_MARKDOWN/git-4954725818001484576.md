
# 『ラブプラス EVERY』にて、無限にログインボーナスを受け取れたり、リッチが消費しない不具合が発生していた。緊急メンテ実施中

Published at: **2019-11-02T00:34:03+00:00**

Author: **Minoru Umise**

Original: [AUTOMATON](https://automaton-media.com/articles/newsjp/20191102-105431/)

コナミデジタルエンタテインメントが配信するモバイルゲーム『ラブプラス EVERY』にて、無限にログインボーナスが受け取れる不具合が発生していた。同作ではまず「高嶺愛花」「小早川凛子」「姉ヶ崎寧々」の3人と知り合うところから物語が始まる。ストーリーを進めていずれかのキャラをカノジョにすれば、本格的にゲームがスタートする。毎日0時になると、ログインボーナスがもらえる仕組みであるが、そのログインボーナスが無限にもらえるという現象だ。
ゲームを立ち上げるとカノジョが挨拶をし、現時点では「初心者ログインボーナス」と通常に「ログインボーナス」を受け取れる仕組み。一般的なソーシャルゲームのそれである。本来は当日分を受け取れば、次の日の0時になるのを待たなければならないはずが、11月2日0時から15分間は、ホームボタンを押せばログインボーナスの取得プロセスが再び開始される挙動になっていた。日をまたいでいないにもかかわらず、すぐにログインボーナスがまた配布されるような形だ。そしてログインボーナスのアイテムはギフトから受け取ることが可能な状態だった。
検証のために何度もログインボーナスを取得してみたが、初心者ログインボーナスは15日分完走できてしまったほか、ログインボーナスはループするので、こちらは文字通り無限に受け取ることができる状態。筆者はギフトでの受け取りはしていないが、これらを取得すると間違いなく悪用になるだろう。
コナミはすぐさまこの状態に気付いたのか、現象が発生した15分後となる11月2日0時15分よりメンテに突入。おそらくロールバックなどがおこなわれると予想される。メンテナンスは早朝4時までおこなわれるようだ。
【UPDATE 2019/11/2 12:00】『ラブプラス EVERY』公式Twitterは、11月2日より発生している問題を報告した。無限にログインボーナスを受け取れるだけでなく、リッチ（金・銀に分けられたゲーム内通貨）が消費しない現象やチャレンジデータがロードされないといった不具合が起こっていたようだ。サービス再開は3日後となる連休明けの11月5日14時が予定されている。
