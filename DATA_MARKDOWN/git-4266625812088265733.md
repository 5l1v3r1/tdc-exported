
# Сегодня состоится ВНО по иностранному языку в магистратуру

Published at: **2019-11-05T08:35:00+00:00**

Author: ****

Original: [ZIK.UA](https://zik.ua/ru/news/2019/11/05/segodnya_sostoytsya_vno_po_ynostrannomu_yaziku_v_magystraturu_1683783)


        Сегодня, 5 ноября, состоится единый вступительный экзамен по иностранному языку для поступления в магистратуру.
      
Об этом сообщает Украинский центр оценивания качества образования.
Отмечается, что для участия во внешнем независимом оценивании зарегистрировано 3 837 человек.
Тестирование начнется в 14:00. Допуск абитуриентов в пункт тестирования будет начат в 13:15 и прекращена в 13:50.
Во время тестирования участникам необходимо будет ответить на 42 задания, на выполнение которых будет отведено 60 минут.
Максимальное количество баллов, которое можно набрать, правильно выполнив все задания теста, равен 42 баллам.
Напомним, 857 учителей начальной школы приняли участие в тестировании различных компетентностей.
