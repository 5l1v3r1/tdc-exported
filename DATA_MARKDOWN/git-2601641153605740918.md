
# HBO представила картонную коробку для уединения во время просмотра фильмов и сериалов

Published at: **2019-11-05T20:32:03+00:00**

Author: **Лиана Липанова**

Original: [vc.ru](https://vc.ru/marketing/91272-hbo-predstavila-kartonnuyu-korobku-dlya-uedineniya-vo-vremya-prosmotra-filmov-i-serialov)

Чтобы получить коробку, нужно поучаствовать в конкурсе.
Американская HBO опубликовала рекламу The HBO Box — чёрной картонной коробки для уединения во время просмотра фильмов и сериалов.
