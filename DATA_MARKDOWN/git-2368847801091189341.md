
# NASA萬聖節禮物：調皮的南瓜太陽

Published at: **2019-11-03T13:30:31+00:00**

Author: ****

Original: [NTD 新唐人電視台](https://www.ntdtv.com/b5/2019/11/01/a102698118.html)

【新唐人北京時間2019年11月03日訊】美國宇航局（NASA）在推特（Twitter）一張太陽的照片，看起來像一個被點亮的萬聖節的裝飾南瓜（jack-o-lantern）。
這張照片由太陽動力學天文臺（SDO）於2014年10月8日拍攝，顯示了太陽紫外線的情況，卻看起來像一個調皮的南瓜。
NASA說，某些區域發出的光線和能量更強，展現了太陽的表面日冕層懸浮著複雜的磁場。這張照片由波長為171和193埃（angstroms）的兩組紫外線光波組成，分別以金色和黃色代表。
NASA鼓勵公眾下載這張照片的高清圖，作為萬聖節的點綴。
