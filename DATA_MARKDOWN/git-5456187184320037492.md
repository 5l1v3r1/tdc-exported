
# Forbes назвал самых богатых бизнесменов планеты

Published at: **2019-11-05T22:58:00+00:00**

Author: ****

Original: [charter97.org](https://charter97.org/ru/news/2019/11/5/354566/)

Президент группы компаний Louis Vuitton Moët Hennessy (LVMH) Бернар Арно обогнал Билла Гейтса в списке богатейших людей мира. Это следует из рейтинга Forbes, который обновляется в режиме реального времени, передает РБК.
Бернар Арно поднялся на вторую строчку с показателем $107 млрд. Основатель Microsoft — на третьей, его состояние журнал оценивает в $106,9 млрд.
На первом месте по-прежнему находится основатель Amazon Джефф Безос ($112,4 млрд).
