
# Чудо-агрегат для жарки шашлыка придумали братья из Ингушетии

Published at: **2019-11-07T17:16:00+00:00**

Author: ****

Original: [ГТРК «Ставрополье»](https://stavropolye.tv/news/127413)

Двое братьев из Ингушетии создали уникальный аппарат для жарки шашлыка, который можно установить даже на балконе многоквартирного дома. Он жарит без дыма и отвечает всем требованиям противопожарной и экологической безопасности.
Для супер-мангала хватит 50 граммов угля, чтобы приготовить30 шампуров шашлыка.
Изобретатели также создали необычный коптильный аппарат. Как и шашлычный, сейчас он проходит процедуру патентования.
Чтобы запатентовать свои устройства, братья Цечоевы обратились в правительство региона.
Премьер-министр Ингушетии Константин Суриков посмотрел изобретения и порекомендовал презентовать проект минэкономразвития региона.
Его рассмотрят на заседании экспертной комиссии, которая решит, какую поддержку можно оказать в продвижении и массовом производстве устройства.
