
# Челябинские следователи ищут поезд, который насмерть сбил молодого мужчину

Published at: **2019-11-01T05:35:00+00:00**

Author: **Михаил Белокрылов**

Original: [Накануне.ru](https://www.nakanune.ru/news/2019/11/01/22557090/)

В Челябинской области на перегоне "Гогино - Бреды" поезд насмерть сбил 33-летнего мужчину.
Трагедия произошла рано утром 31 октября, сообщили Накануне.RU в Уральском СУ на транспорте СК РФ.
Сейчас по факту гибели мужчины проводится доследственная проверка. Следователи устанавливают какой конкретно поезд сбил мужчину.
