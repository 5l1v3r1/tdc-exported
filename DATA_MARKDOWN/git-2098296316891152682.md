
# Первый трейлер «Цвета из иных миров» по рассказу Говарда Ф. Лавкрафта

Published at: **2019-11-06T00:00:00+00:00**

Author: **Евгения Блинова**

Original: [Мир фантастики](https://www.mirf.ru/news/pervyj-trejler-cveta-iz-inyh-mirov-po-rasskazu-govarda-f-lavkrafta)

В сети появился дебютный трейлер фильма «Цвет из иных миров», основанного на одноимённом рассказе Говарда Ф. Лавкрафта. Главную роль в картине играет Николас Кейдж.
