
# Dieseldiebe werden auf Baustelle überrascht

Published at: **2019-11-04T09:14:00+00:00**

Author: ****

Original: [Mitteldeutsche Zeitung](https://www.mz-web.de/aschersleben/beim-abzapfen-dieseldiebe-werden-auf-baustelle-ueberrascht-33408994)

Mehringen - Da haben Dieseldiebe Pech gehabt: Am späten Sonnabendabend bemerkten Mitarbeiter einer Sicherheitsfirma bei einer Kontrollfahrt über eine Baustelle in Mehringen, dass Unbekannte Sprit aus einer selbstfahrenden Arbeitsmaschine entwenden.
Die Baustelle befindet sich auf einem Windpark an der L85. Die Sicherheitsleute informierten die Polizei. Als die Diebe bemerkten, dass sie entdeckt worden sind, flüchteten sie.  Die von den Tätern zurückgelassenen Materialien wurden sichergestellt und werden kriminaltechnisch untersucht.
Eine zurückgebliebene Diesellache ist von Mehringer Feuerwehrkameraden gebunden worden. Der Windpark an der L 85 ist erst in der vergangenen Woche Ziel von Dieseldieben geworden. (mz)
