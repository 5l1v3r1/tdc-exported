
# Даллас представил форму для Зимней классики

Published at: **2019-11-06T17:24:00+00:00**

Author: ****

Original: [Allhockey.ru](https://allhockey.ru/news/show/369207-Dallas_predstavil_formu_dlya_Zimnej_klassiki)

"Даллас" презентовал игровую форму для "Зимней классики"."Звёзды" встретятся с "Нэшвиллом" 1 января 2020 года. Игра пройдёт на стадионе "Коттон Боул" в Далласе, вмещающем более 92 тысяч зрителей.
