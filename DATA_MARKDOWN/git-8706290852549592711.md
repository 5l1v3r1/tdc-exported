
# Βρούτσης: Όχι μόνο δεν καταργείται το εφάπαξ αλλά θα δίνεται μαζί με την σύνταξη

Published at: **2019-11-02T16:50:00+00:00**

Author: ****

Original: [enikos.gr](http://www.enikos.gr/politics/681794/vroutsis-ochi-mono-den-katargeitai-to-efapax-alla-tha-dinetai-maz)

"Οι επικουρικές συντάξεις θα αυξηθουν. Δεύτερον, θα αυξηθούν και οι κύριες. Όχι όλες. Εκείνες που έχουν μικρό βαθμό ανταποδοτικότητας. Ήταν κυριαρχη πολιτική μας θέση και θα το κάνουμε πράξη στο νομοσχέδιο" δήλωσε στον ΣΚΑΙ ο υπουργός Εργασίας και Κοινωνικών Υποθέσεων, Γιάννης Βρούτσης.
"Τα εφάπαξ διατηρούνται. Δεν κινδυνεύει τίποτα. Να νιώθουν απόλυτα ασφαλείς. Κανείς να μην έχει αγωνία για το ασφαλιστικό" ανέφερε.
Πηγή: ΣΚΑΙ
