
# Netflix tiene hasta siete temporadas planeadas para The Witcher

Published at: **2019-11-04T21:51:57+00:00**

Author: **Rodolfo León**

Original: [Atomix](https://atomix.vg/netflix-tiene-hasta-siete-temporadas-planeadas-para-the-witcher/)

La próxima serie de Netflix, The Witcher, todavía ni se estrena pero su productora, Lauren S. Hissrich, ya tiene grandes planes para el proyecto. De hecho, Hissrich dice que ya tiene pensado hasta siete temporadas para el show. El querer proveer el suficiente tiempo de desarrollo para los múltiples arcos de personajes sirve como una de las principales razones para este adelanto.
Durante una entrevista reciente para SFX Magazine, Hissrich comentó lo siguiente:
Al parecer, la evolución y desarrollo de personajes será uno de los temas progresivos en esta serie. Si todo sale bien, es muy probable que los espectadores se apeguen a todos estos individuos. Sin embargo, no esperes que cada uno de ellos tenga un final feliz, pues de acuerdo con la productora, puede haber muchas muertes y no todos sobrevivirán la primer temporada.
The Witcher llega a Netflix el 20 de diciembre 2019
