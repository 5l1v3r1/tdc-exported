
# В ДТП с автобусом на кузбасской трассе погиб человек

Published at: **2019-11-02T03:54:34+00:00**

Author: **Екатерина Бухтиярова**

Original: [Сiбдепо](https://sibdepo.ru/news/v-dtp-s-avtobusom-na-kuzbasskoj-trasse-pogib-chelovek.html)

На кузбасской трассе на выезде из Ленинска-Кузбецкого утром 2 ноября произошло ДТП с автобусом, в котором погиб человек и пострадало ещё несколько.
Фотографии с места происшествия появились в соцсети. На снимках видно, что столкнувшаяся с автобусом легковая машина улетела в кювет.
По словам очевидцев, один из участников аварии погиб. Кроме того, есть ещё люди, которые в этом ДТП получили травмы.
