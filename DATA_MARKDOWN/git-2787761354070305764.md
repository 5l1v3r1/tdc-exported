
# Брянцам предложили выбрать лучшего народного участкового

Published at: **2019-11-01T07:13:04+00:00**

Author: ****

Original: [Брянск Today](https://bryansktoday.ru/article/111660)

Напомним, в Брянской области прошел конкурс «Народный участковый-2019». По итогам онлайн-голосования победителем регионального этапа Всероссийского конкурса стал старший участковый ОМВД «Карачевский» Константин Артамонов.
Финал конкурса будет проходить с 1 по 10 ноября в форме онлайн-голосования на сайте «Комсомольской правды».  Поддержать нашего земляка, проголосовав за него по ссылке.
