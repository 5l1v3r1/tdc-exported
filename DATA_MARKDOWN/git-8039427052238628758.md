
# ప్రారంభమైన విజయారెడ్డి అంతిమయాత్ర..

Published at: **2019-11-05T14:07:04+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=947094)

ADVT
తహశీల్దార్ విజయారెడ్డి అంతిమయాత్ర ప్రారంభమైంది. మరికాసేపట్లో ఆమె అంత్యక్రియలు జరగనున్నాయి. ఐఏఎస్ అధికారిణి అమ్రపాలి సహా పలు పార్టీల నేతలు, ఉద్యోగ కార్మిక సంఘాల నాయకులు నివాళులర్పించారు. జరిగిన ఘటనను మంత్రి హరీశ్‌రావు సహా పలు పార్టీల నేతలు, ఉద్యోగ సంఘాల నాయకులు ఖండించారు. ఘటనకు ప్రభుత్వమే బాధ్యత వహించాలని కాంగ్రెస్ ఎంపీ రేవంత్ రెడ్డి అన్నారు.
 
కాగా విజయారెడ్డిని కాపాడబోయిన ఆమె డ్రైవర్ గురునాథం చికిత్స పొందుతూ మంగళవారం ఉదయం మృతి చెందాడు. నిన్న తీవ్ర గాయాలపాలైన గురునాథంను డీఆర్‌డీవో ఆపోలో ఆస్పత్రికి తరలించారు. చికిత్స పొందుతూ ఆయన ఇవాళ మృతి చెందాడు.
