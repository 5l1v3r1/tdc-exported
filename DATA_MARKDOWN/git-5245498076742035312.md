
# В Сыктывкаре полицейские вычислили любителей наркотиков по поведению

Published at: **2019-11-03T10:10:00+00:00**

Author: ****

Original: [БНК](https://www.bnkomi.ru/data/news/102247/)

В Сыктывкаре внимание сотрудников Росгвардии привлекли молодые люди, которые находились в машине у дома по улице Тентюковской. При виде полицейских их поведение заметно изменилось.
При проверке документов сотрудник вневедомственной охраны заметил у одного из мужчин полимерный пакетик с неизвестным веществом, от которого он пытался избавиться, выкинув его на землю.
Во время общения с гражданами росгвардеец обратил внимание на водителя автомобиля, у которого были явные признаки опьянения. На место происшествия незамедлительно была вызвана следственно-оперативная группа для дальнейшего разбирательства.
По данному факту мужчину 1991 года рождения доставили в дежурную часть полиции для дальнейшего разбирательства. Проводится проверка, сообщает пресс-служба управления Росгвардии по Коми.
