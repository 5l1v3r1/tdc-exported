
# Τσάμπιονς Λιγκ: Η κρίσιμη τέταρτη αγωνιστική των ομίλων

Published at: **2019-11-04T21:00:26+00:00**

Author: **Ιωάννης Καράγιωργας**

Original: [euronews](https://gr.euronews.com/2019/11/04/champions-league-i-krisimi-tetarti-agonistiki-twn-omilwn)

Παραδοσιακά η τέταρτη αγωνιστική των ομίλων του Τσάμπιονς Λιγκ είναι κομβική καθώς αναδεικνύονται οι πρώτες ομάδες των «16» ενώ αλλάζουν και οι ισορροπίες, με δύο αγώνες να απομένουν.
Η Λίβερπουλ υποδέχεται τη Γκενκ και θέλει να κάνει την τρίτη σερί νίκη για να φτάσει μια ανάσα από την πρόκριση ενώ αν συμβεί αυτό και η Νάπολι κερδίσει τη Σάλτσμπουργκ, οι Ιταλοί θα περάσουν στα νοκ άουτ.
«Έχουμε μεγάλες φιλοδοξίες για τη διοργάνωση και ο όμιλος είναι ανοιχτός. Όλα είναι ακόμη πιθανά. Γι αυτό πρέπει να είμαστε 100% συγκεντρωμένοι στο ματς. Πρέπει να δημιουργήσουμε φανταστική ατμόσφαιρα και πάλι. Και έτσι να κάνουμε δύσκολη τη ζωή της Γκενκ» είπε ο Γιούργκεν Κλοπ.
Μεγάλο ματς διεξάγεται στη Γερμανία με τη Ντόρτμουντ να υποδέχεται την ισόβαθμη Ίντερ. Έχοντας χάσει 2-0 στην Ιταλία, η γερμανική ομάδα θέλει μόνο τη νίκη για να ελπίζει σε πρόκριση στους «16», με την Μπαρτσελόνα να έχει καθαρό προβάδισμα υποδεχόμενη τη Σλάβια Πράγας.
«Πρέπει να είμαστε έτοιμοι για όλα. Χρειάζεται να παίξουμε πάρα πολύ καλά εάν θέλουμε να φτάσουμε στη νίκη. Απαιτείται να βρούμε την ιδανική ισορροπία ανάμεσα σε επίθεση και άμυνα. Παίζουμε με πολύ επικίνδυνη ομάδα» είπε ο Λουσιάν Φαβρ.
Η Μπαρτσελόνα είναι αήττητη στον όμιλο και με πιθανή επικράτηση επί της Σλάβια θα χρειάζεται μόλις έναν βαθμό σε δύο ματς για να προκριθεί.
Στον έβδομο όμιλο αναμένεται να κριθούν πολλά καθώς η Ζενίτ υποδέχεται την πρωτοπόρο Λειψία και η Λυών την Μπενφίκα, με τις τέσσερις ομάδες να βρίσκονται σε απόσταση τριών βαθμών μεταξύ τους.
Τέλος στον όγδοο όμιλο, η Βαλένθια υποδέχεται τη Λιλ με στόχο τη νίκη που θα την φέρει στις πρώτες θέσεις, καθώς Τσέλσι και Άγιαξ κοντράρονται στο Λονδίνο.
