
# СМИ опубликовали видео задержания преступной группы в Крыму

Published at: **2019-11-02T08:50:59+00:00**

Author: ****

Original: [Крым.Реалии](https://ru.krymr.com/a/news-video-zaderzhaniya-prestupnikov-v-krimu/30249361.html)

Близкое к властям Крыма издание Крыминформ опубликовало оперативное видео задержания с поличным одного из участников преступной группы, сформированной выходцами с Северного Кавказа.
По данным издания, видео получено из российского управления ФСБ по Крыму и Севастополю.
