
# Εντοπίστηκαν μετανάστες μέσα σε φορτηγό-ψυγείο στην Ξάνθη - ΤΩΡΑ

Published at: **2019-11-04T12:34:00+00:00**

Author: ****

Original: [enikos.gr](http://www.enikos.gr/society/682192/entopistike-fortigo-psygeio-pou-metefere-metanastes-stin-xanthi-t)

Περισσότεροι από 80 μετανάστες εντοπίστηκαν μέσα σε φορτηγό-ψυγείο που σταμάτησαν για έλεγχο στα διόδια της Ξάνθης άνδρες του τοπικού τμήματος Συνοριακής Φύλαξης. Ο οδηγός, γεωργιανής καταγωγής, συνελήφθη.
Σύμφωνα με τα τοπικά ΜΜΕ, όταν άνοιξαν οι πόρτες του φορτηγού οι μετανάστες που ήταν στοιβαγμένοι μέσα στο κοντέινερ, επιχείρησαν να διαφύγουν με αποτέλεσμα κάποιοι να τραυματιστούν. Επίσης, αρκετοί αντιμετωπίζουν αναπνευστικά προβλήματα.
Άμεση ήταν η κινητοποίηση του ΕΚΑΒ προκειμένου να μεταφερθούν στο νοσοκομείο.
