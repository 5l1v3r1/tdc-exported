
# 首里城再建、菅長官「財政面含め全面的に支援」…沖縄知事と会談

Published at: **2019-11-01T13:05:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/politics/20191101-OYT1T50261/)

　菅官房長官は１日、沖縄県の玉城デニー知事と首相官邸で会談し、焼失した首里城の再建に全力を挙げることで一致した。菅氏は再建計画について「財政的な措置を含めて、全面的に支援していきたい」と述べた。
　菅氏は会談に先立つ記者会見で、再建に向けた関係省庁による対策チームを作る方針を示した。
　玉城氏は会談後、記者団に「首里城は県民の心のよりどころだ。できるだけ前倒しで計画を策定して、発表したい」と述べた。
