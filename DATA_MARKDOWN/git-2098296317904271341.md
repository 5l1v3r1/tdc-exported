
# Первый тизер мультфильма Pixar «Душа»

Published at: **2019-11-07T00:00:00+00:00**

Author: **Кот-император**

Original: [Мир фантастики](https://www.mirf.ru/news/pixar-vypustil-trejler-multfilma-dusha)

Pixar представил первый тизер своего следующего мультфильма — картины «Душа» (Soul). Она рассказывает о школьном учителе музыке Джо Гарднер, который случайно попадает в мир, где зарождаются человеческие увлечения, мечты и интересы. Там он знакомится с молодой душой по имени 22.
