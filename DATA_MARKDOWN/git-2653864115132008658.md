
# Дорога у главной больницы региона стала для вологжан полосой препятствий

Published at: **2019-11-03T09:39:00+00:00**

Author: ****

Original: [35медиа](https://35media.ru/news/2019/11/03/doroga-u-glavnoj-bolnicy-regiona-stala-dlya-vologzhan-polosoj-prepyatstvij)

Дорога у главной больницы региона стала для вологжан полосой препятствий. Машины с трудом объезжают ямы, а пешеходам из-за отсутствия тротуаров приходится добираться в медицинское учреждение по проезжей части.
