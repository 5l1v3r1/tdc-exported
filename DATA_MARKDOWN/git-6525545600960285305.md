
# В основу Роттердам+ заложен принцип импортного паритета, как и для ценообразования на других энергетических рынках, - эксперт

Published at: **2019-11-01T10:37:00+00:00**

Author: ****

Original: [Українські Новини](https://ukranews.com/news/663254-v-osnovu-rotterdam-zalozhen-printsip-importnogo-pariteta-kak-i-dlya-tsenoobrazovaniya-na-drugih)


        
          Формула "Роттердам+" основывается на аналогичном для всех энергорынков принципе импортного паритета. Об этом говорится в колонке советника юридической компании "Primelex" Евгения Калинина для издания "Левый берег".
        
      
Эксперт напомнил, что НАБУ расследует производства относительно законности применения импортного паритета в формулах "Дюссельдорф +" и "Роттердам+". И эти производства должны поставить окончательную точку в дискуссии, вокруг обоснованности применения этого принципа в Украине. Он также обращает внимание, что по аналогичным формулам формируются цены и на другие энергоресурсы.
"Не только цены на уголь и газ но и на другие энергоресурсы формируются по алгоритму "международный индекс + транспортировка". Например, урановый концентрат для "Энергоатома". Цены на ураносодержащее сырье, услуги конверсии и обогащения определяются исходя из цен, опубликованных в информационных обзорах Ux Weekly и Nuklear Market Review, а к расчетам тарифов включается стоимость услуг по транспортировке", - отметил Калинин.
Напомним, ранее генеральный секретарь Европейской ассоциации угля и лигнита EURACOAL Брайан Риккетс заявил, что введение импортного паритета при ценообразовании на уголь было единственно верным решением для Украины, привязка к индексу API - абсолютно корректна.
24 марта 2017 НАБУ открыло уголовное производство, в рамках которого расследует действия членов НКРЭКУ, которые утвердили формулу "Роттердам+". 8 августа 2019 детективы НАБУ сообщили о подозрении шести лицам по так называемому делу "Роттердам+", ни один из них не был взят под стражу. Позже в официальном ответе НАБУ на запрос Информбюллетеня ICC Ukraine сообщалось, что в Бюро нет претензий к индексу API2 в основных портах Западной Европы, как базового ориентира для определения цены на уголь в Украине по формуле "Роттердам+".
