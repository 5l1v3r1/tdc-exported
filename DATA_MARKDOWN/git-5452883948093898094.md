
# Trasgredisce al foglio di via obbligatorio, denunciato

Published at: **2019-11-02T12:22:00+00:00**

Author: **Redazione**

Original: [AvellinoToday](http://www.avellinotoday.it/cronaca/trasgredisce-foglio-via-obbligatorio-2-novembre-2019.html)

I Carabinieri della Stazione di Roccabascerana hanno denunciato un 65enne di Montesarchio, per aver trasgredito le prescrizioni inerenti al Foglio di Via Obbligatorio.
