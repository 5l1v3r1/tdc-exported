
# 『世界野球プレミア12』TBSと野球系YouTuber・トクサンがコラボ

Published at: **2019-11-06T08:00:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148090/full/)

　WBSC（世界野球ソフトボール連盟）世界ランキング上位12の国と地域が参加、4年に1度開催される野球の国際大会『世界野球プレミア12』の第2回大会、野球日本代表・侍ジャパンの戦いが11月5日〜17日に行われ、TBSとテレビ朝日が共同中継を行う。TBSでは侍ジャパンを応援すべく、野球系YouTubeチャンネル｢トクサンTV｣とコラボすることが決定した。

        【写真】その他の写真を見る
      
　YouTuberのトクサンは、自身も帝京高校で甲子園出場、創価大学で全国ベスト4という経歴を持ち、35歳となった現在も軟式野球の日本代表に名を連ねている。彼のYouTubeチャンネル「トクサンTV」は2016年8月にスタート。ハイレベルな草野球の試合や道具紹介、技術指導などを中心に毎日午後6時30分に配信している。チャンネル開始3年目で総視聴回数は3億6000万回以上、チャンネル登録者数は49万人に到達。現在、月間1500万回再生されている人気チャンネルだ。 　そんな「トクサンTV」が、このたび野球のファン拡大の一環として、TBSとコラボ。TBSでの『世界野球プレミア12』生中継に合わせて、「トクサンTV」のメンバーが赤坂サカスの特設ブースから生配信。熱い応援はもちろん、プレーの解説やチャットなどで侍ジャパンを盛り上げる。 ＜トクサン　コメント＞日本最高レベルの選手が集う侍ジャパンが、世界の強豪と世界最高レベルの野球を繰り広げます！侍ジャパンは野球を愛する私、トクサンの代表でもあり、皆さんの代表でもあります。今回のような機会をいただき本当にありがとうございます。テレビもYouTubeも一緒になって、日本全体で侍ジャパンを盛り上げましょう！ ■「世界野球プレミア１２」放送予定【オープニングラウンド】１１月６日（水） 日本×プエルトリコ（後７：00）ＴＢＳ系１１月７日（木） 日本×台湾（後７：00）ＴＢＳ系【スーパーラウンド】１１月１１日（月）〜【決勝戦・３位決定戦】１１月１７日（日）
