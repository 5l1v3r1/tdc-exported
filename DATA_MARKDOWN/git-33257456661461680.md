
# Дума в первом чтении поддержала запрет на продажу смартфонов без российских приложений

Published at: **2019-11-05T12:45:21+00:00**

Author: **Государственная дума**

Original: [Meduza](https://meduza.io/news/2019/11/05/duma-v-pervom-chtenii-podderzhala-zapret-na-prodazhu-smartfonov-bez-rossiyskih-prilozheniy)

Госдума приняла в первом чтении законопроект о запрете продажи смартфонов, компьютеров и умных телевизоров без российского программного обеспечения. 
По мнению авторов законопроекта, такая норма будет привлекательна для потребителей, так как при покупке иностранных устройств они будут сразу получать программы, разработанные с учетом российского рынка.
Перечень программного обеспечения, которое должно быть установлено на устройства, будет определять правительство.
