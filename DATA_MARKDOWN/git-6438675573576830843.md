
# Red Dead Redemption 2 вышла на ПК

Published at: **2019-11-05T17:40:00+00:00**

Author: ****

Original: [РИА Новости](https://ria.ru/20191105/1560610547.html)

МОСКВА, 5 ноя — РИА Новости. Компания Rockstar Games выпустила экшен Red Dead Redemption 2 для персональных компьютеров.Версия игры для консолей вышла в 2018 году. Ей удалось одержать победу в номинации "Любимая игра осени" по версии Gamers’ Choice Awards, а также "Выбор критиков" по версии Golden Joystick Awards.При этом, как сообщает портал Dark Side of Gaming, ни одна из имеющихся на рынке видеокарт не способна качественно воспроизводить игру на максимальных настройках.Многие пользователи уже отметили, что с запуском игры возникает ряд проблем. В частности, у некоторых из них приложение "крашится" после появление загрузочного экрана.Приобрести игру можно в официальном лаунчере Rockstar, в Steam игра будет доступна позднее.
