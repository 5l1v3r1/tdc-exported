
# Microsoft mostra nova logo para o navegador Edge

Published at: **2019-11-04T13:41:00+00:00**

Author: **Rafael Rigues**

Original: [Olhar Digital](https://olhardigital.com.br/noticia/microsoft-mostra-nova-logo-para-o-navegador-edge/92476)

A Microsoft anunciou via Twitter a nova logo do seu navegador Microsoft Edge. A imagem mostra um “e” estilizado com um degradê do verde para o azul escuro, que lembra mais uma onda do que o antigo logo similar ao do finado Internet Explorer.
Esta é, na verdade, a segunda versão do Edge, baseada no código fonte do Chromium, versão Open Source do Google Chrome. A versão anterior, lançada em julho de 2015 junto com o Windows 10, era baseada em “engines” próprios da Microsoft, o EdgeHTML (para interpretação de HTML) e Chakra (para interpretação de JavaScript). A mudança no ícone é uma forma de sinalizar este “rompimento” com o passado.
Versões beta do Edge baseadas no Chromium estão disponíveis desde agosto. Segundo o The Verge, o anúncio da nova logo às vésperas de uma conferência da Microsoft voltada para profissionais de TI sinaliza que uma versão estável do navegador pode ser anunciada em breve.
