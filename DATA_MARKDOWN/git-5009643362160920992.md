
# В Москве пройдет «Ночь Миядзаки»

Published at: **2019-11-01T17:24:34+00:00**

Author: **Анна Логачева**

Original: [The Village](https://www.the-village.ru/village/weekend/wknd-news/366167-noch-miyadzaki)

В ближайшее воскресенье в «Каро Октябрь» пройдет «Ночь Миядзаки». Она станет частью Большого фестиваля мультфильмов.
В кинотеатре можно будет увидеть мультфильмы японского режиссера Хаяо Миядзаки. Первым покажут «Унесенные призраками», получивший «Оскар» в 2003 году. За ним — самый кассовый мультфильм в японском прокате «Принцесса Мононоке». «Ночь» закроет «Навсикая из Долины ветров» — картина, снятая по мотивам комикса, который нарисовал Миядзаки.
