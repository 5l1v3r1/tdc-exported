
# Рассуждал, как кореной алматинец — Абаев о поручениях Токаева

Published at: **2019-11-01T10:30:00+00:00**

Author: **Арыстан Алангожин**

Original: [Тотал](https://total.kz/ru/news/mneniya/rassuzhdal_kak_korenoi_almatinets__abaev_o_porucheniyah_tokaeva_date_2019_11_01_10_33_16)


        Абаев назвал самые важные для алматинцев поручения Токаева.
      
Министр информации и общественного развития Казахстана Даурен Абаев высказался по поводу поручений Касым-Жомарта Токаева на совещании по благоустройству Алматы, передает корреспондент Total.kz.
Он назвал три наиболее важных, по его мнению, поручения главы государства на совещании по благоустройству Алматы.
