
# Запорожцам на заметку: в Украине запретили популярный антибиотик

Published at: **2019-11-02T12:50:00+00:00**

Author: **Владимир Андреев**

Original: [РИА Мелитополь](https://ria-m.tv/news/170326/zaporojtsam_na_zametku_v_ukraine_zapretili_populyarnyiy_antibiotik.html)


        В Украине запретили еще одно лекарственное средство.
      
Об этом стало известно из информации Государственной службы Украины по лекарственным препаратам и контроля по наркотикам.
Так, под запрет попал “Евроцефтаз” всех серий. Запретили данный антибактериальный препарат по причине негативных выводов касательно качества завезенного лекарственного препарата. Речь идет о порошковом препарате “Евроцефтаз”, который используют для инъекций всех серий.
Ранее его использовали для лечения различных инфекций дыхательных путей, ЛОР-органов и в частности пневмонии. Также препарат использовали для лечения инфекций органов брюшной полости, желудочно-кишечного тракта и перитонита. Инфекции мочевых путей, половых органов, включая гонорею, инфекции костей, мягких тканей, суставов, инфекции у людей со слабой иммунной системой. Также его использовали в лечении сепсиса, менингита и разных стадии болезни Лайма.
