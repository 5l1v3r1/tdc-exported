
# 'เอ๋'ลั่นพร้อมนั่งกมธ.ป.ป.ช ร่วมคนเจ้าอารมณ์-บ้าอำนาจ

Published at: **2019-11-05T11:15:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/politics/740138)


        “ปารีณา”ยัน เต็มใจนั่ง กมธ.ป.ป.ช.ลั่นพร้อมทำงานร่วมกับคนเจ้าอารมณ์ -บ้าอำนาจ”
      
เมื่อวันที่ 5 พ.ย.เวลา 18.00 น.น.ส.ปารีณา ไกรคุปต์ ส.ส.ราชบุรี พรรคพลังประชารัฐกล่าวถึงกรณีที่ 2 ส.ส.พรรคพลังประชารัฐ ก็คือนายธนะสิทธิ์ โควสุรัตน์ส.ส.อุบลราชธานี และนายพยม พรหมเพชร ส.ส.สงขลา ขอลาออกจากกรรมาธิการการป้องกันและปราบปรามการทุจริตและประพฤติมิชอบ สภาผู้แทนราษฎรว่า ตนรู้สึกเห็นใจท่านพยมและท่านธนะสิทธิ์ ที่รู้สึกหมดความอดทนกับพฤติกรรมที่มีการใช้อำนาจกรรมาธิการแบบเกินขอบเขต และเข้าใจดีถึงความอึดอัดใจหากจะต้องเข้าร่วมประขุมในทุกสัปดาห์ ซึ่งหากเกินความรู้สึกเช่นนี้การลาออกก็ไม่ถือว่าเป็นเรื่องที่ผิดหรือน่าแปลกอะไรน.ส.ปารีณา กล่าวต่อว่า วันนี้ที่ประชุมก็ได้มีมติให้ตนและนายสิระ เจนจาคะส.ส.กทม.ทำหน้าที่แทนทั้ง 2 ท่าน ตนพร้อมที่จะทำหน้าที่แทนอย่างเต็มใจ และจะทำให้ประเทศชาติและประชาชนได้รับผลประโยชน์มากที่สุด ให้สมค่ากับทุกคะแนนของชาวราชบุรีที่เลือกตนมา “ดิฉันจะพยายามสอนและตักเตือนบรรดา ส.ส.สมัยแรกบางคนว่า อำนาจของกรรมาธิการมีแค่ไหน แต่ส่วนตัวตนมั่นใจว่า จะสามารถอดทน กับคนเจ้าอารมณ์ บ้าอำนาจ ควบคุมสติไม่ได้ ได้อย่างแน่นอน”น.ส.ปารีณา กล่าว.
