
# В Венгрии насчитали наибольшее количество украинских заробитчан

Published at: **2019-11-04T20:39:00+00:00**

Author: **Корреспондент.net**

Original: [Корреспондент.net](https://korrespondent.net/world/4157222-v-venhryy-naschytaly-naybolshee-kolychestvo-ukraynskykh-zarobytchan)

Венгрия приняла согласно последней статистике 43 881 заробитчан и большинство из них - украинцы, сообщает  Világgazdaság. Согласно официальной статистике, среди рабочих из стран, которые не входят в ЕС, в Венгрии сербы заняли второе место по количеству разрешений (6199), за ними идут китайцы с 2965. На рынке труда Венгрии – 2778 вьетнамцев и 1806 индийцев. Количество рабочих по национальной принадлежности заметно не изменилась за последние несколько лет, хотя общая цифра увеличилась по сравнению с 2017 годом. Так, два года назад из трех топ-стран было зарегистрировано 11 644 работников; этот показатель вырос до 31 499 в прошлом году. Граждане Украины, в частности, составили подавляющее большинство иностранцев, которые получили вид на жительство в Венгрии с целью трудоустройства. Их число перевалило за 26 тысяч по состоянию на сентябрь.  Напомним, заместитель главы Закарпатского облсовета Йосип Барто заявил о необходимости создания в регионе Венгерского района с центром в Берегово.
