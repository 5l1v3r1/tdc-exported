
# Петербургские ученые разработали не имеющий российских аналогов уникальный прибор

Published at: **2019-11-01T18:33:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/738854-peterburgskie-uchenye-razrabotali-ne-imeyushii-rossiiskikh-analogov-unikalnyi-pribor)


        Петербургские ученые создали первую отечественную установку для неразрушающего контроля дефектов изделий. Отмечается, что в мире подобные установки изготавливает всего одна фирма в Англии.
      
Петербургские ученые создали первую отечественную установку для неразрушающего контроля дефектов изделий. Об этом сообщили в пресс-службе Петербургского государственного электротехнического университета «ЛЭТИ».
Отмечается, что уникальная российская разработка позволит проверять качество оптоволоконного глубоководного кабеля в реальных условиях прокладки в море. Разработка выполнена по заказу одного из стратегических партнеров вуза.
По информации пресс-службы, качество, надежность и долговечность глубоководных кабелей имеют большое значение для заказчиков — изделие должно быть долговечным, водонепроницаемым, выдерживать огромное давление водных масс, обладать достаточной прочностью для укладки и эксплуатации. При этом материалы должны быть подобраны так, чтобы при механических изменениях не изменялись его рабочие характеристики.
Разработка ученых ЛЭТИ позволяет провести неразрушающий контроль и выявить дефекты, предоставив исчерпывающую информацию о проверяемом изделии. Руководитель команды разработчиков, заведующий кафедрой электронных приборов и устройств СПбГЭТУ «ЛЭТИ» Николай Потрахов отметил, это первая отечественная разработка подобного назначения.
Установка полностью повторяет технические характеристики английского аналога, но значительно дешевле его. Она отличается высокой прочностью конструкции и радиационной безопасностью. При этом в пресс-службе заявили, что ученым ЛЭТИ удалось усовершенствовать зарубежный аналог, наделив его еще одной важной функцией — в отечественной разработке есть функция томографии, которой пока нет в английской версии.
Первый образец прибора в начале ноября отправится на испытания на завод по производству кабельной продукции. По их завершении установка будет использоваться по прямому предназначению.
