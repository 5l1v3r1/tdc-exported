
# НЕВСКИЕ НОВОСТИ публикуют видео из квартиры, откуда вылетел мужчина

Published at: **2019-11-05T15:49:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/740059-nevskie-novosti-publikuyut-video-iz-kvartiry-otkuda-vyletel-muzhchina)


        На улице Стасовой из окна выпал неизвестный мужчина. В квартире, откуда он вылетел, побывал корреспондент НЕВСКИХ НОВОСТЕЙ.
      
Появилось видеозапись из квартиры в Красногвардейском районе, из окна которой сегодня днем, 5 ноября, выпал мужчина.
Как ранее рассказал источник НЕВСКИХ НОВОСТЕЙ, под окнами на земле обнаружили тело неизвестного человека. По предварительной информации, он выпал из окна дома № 1 по улице Стасовой.
По словам очевидца, мужчина буквально разбил окно своим телом. По одной версии, он сильно разбил голову и перестал дышать, по другой – пострадавшего увезли на карете скорой помощи. Рассказом об увиденном поделился очевидец.
Также источник сообщил, что в квартире проживает женщина с 23-летней внучкой. На место происшествия уже прибыли правоохранители.
