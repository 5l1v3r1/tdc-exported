
# 'เอิร์ธ'แฉต่อ!บอกชุดที่เข้าพิธีแต่ง'เปา' ไม่ใช่ร้านที่สั่งตัด!

Published at: **2019-11-02T11:30:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/entertainment/739681)


        “เอิร์ธ”แฉต่อแจงชุดเข้าพิธีแต่ง”เปา” พร้อมชวนแฟนคลับเดาชุดเจ้าสาวในงานโบสถ์ ว่าจะใช้ห้องเสื้อของใคร
      
ดูท่าจะเป็นดราม่าบานปลายแน่ๆ หลังเมื่อช่วงเช้าก่อนเข้าพิธีแต่งงานในวันนี้(2 พ.ย.)หนุ่ม 'เอิร์ธ' ลูกชายหัวแก้วหัวแหวนของ 'ซูโม่กิ๊ก' พิธีกรคนดังออกมาโพสต์เฟซบุ๊กส่วนตัวระบุว่าร้านที่สั่งตัดชุดแต่งงานยังไม่ได้เอาชุดมาให้เลยทั้งๆที่จะเริ่มงานในอีก 2 ชั่วโมงข้างหน้าแล้วงานนี้พอหนุ่มเอิร์ธโพสต์ข้อความลงไป ก็มีชาวเน็ตมากดไลค์พร้อมส่งอีโมติคอนตกใจกับเรื่องดังกล่าวอย่างมาก ต่อมาหลังงานแต่งงานผ่านพ้นไปได้ด้วยดี หนุ่มเอิร์ธก็ได้มาโพสต์อัพเดทเรื่องนี้อีกครั้ง โดยหนุ่มเอิร์ธเล่าว่า "อัพเดทครับ...สรุปคือวันนี้สูทเจ้าบ่าวที่ผมสั่งตัดไปก็ยังไม่โผล่มาให้เห็น ...งานแต่งงานในวันนี้ผ่านไปได้ด้วยดี เรื่องเสื้อผ้าผมและเปาตกลงไปเช่าห้องเสื้อของ Xxxxxx ชุดแต่งงาน ชุดไทย ชุดดีสวยงามออกมาสง่ามากครับ ต้องขอขอบคุณห้องเสื้อที่มีความเป็นมืออาชีพและสามารถแก้ปัญหาไม่มีชุดด้วยเวลาอันน้อยนิด
ขอขอบคุณเป็นพิเศษ พี่ๆทีมรันคิวและ MC พี่โบว์ พี่ก้อย และทีมงานที่มาช่วยให้งานราบรื่นมากครับ ถ้าไม่ได้พี่ๆคงจะติดขัดไปซะทุกสิ่ง  ไม่ใช่หน้าที่ของพี่แต่พี่ก็เมตตาเข้ามาช่วยเหลือ  ขอบคุณครับ ส่วนชุดเจ้าสาวในงานโบสถ์ที่จะถึงนี้ คอยติดตามกันต่อไปว่าเราจะใช้ห้องเสื้อของใคร เพราะจนถึงปัจจุบันที่สั่งตัดไปก็ยังไม่เห็น Xxxxxxแต่ทุกอย่างมีทางออก"  พอหนุ่มเอิร์ธลงรูปไปก็มีแฟนๆเข้ามากดไลค์ให้มากมาย รวมถึงคุณพ่อซูโม่กิ๊กก็ได้มาคอมเมนต์เป็นอีโมติคอนรูปปรบมือให้ด้วยขอขอบคุณภาพประกอบจากเฟซบุ๊ก Karn Kitcharoen
