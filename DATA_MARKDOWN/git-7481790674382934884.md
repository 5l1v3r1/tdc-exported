
# 浅間山の噴火警戒レベル、１に引き下げ…気象庁

Published at: **2019-11-06T05:23:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191106-OYT1T50256/)

　気象庁は６日午後２時、群馬・長野県境の浅間山の噴火警戒レベルを２（火口周辺規制）から１（活火山であることに留意）に引き下げた。火口から５００メートルを超える範囲に影響を及ぼす噴火の可能性が低くなったため。
　８月２６日に小規模噴火が発生したが、火山性地震は１０月上旬から少ない状態で経過し、深部からのマグマ上昇を示す地殻変動は観測されていないという。
