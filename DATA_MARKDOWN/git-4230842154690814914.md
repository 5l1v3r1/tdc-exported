
# КБ Южное представило противокорабельную сверхзвуковую ракету Блискавка

Published at: **2019-11-02T14:38:19+00:00**

Author: ****

Original: [ФАКТИ](https://fakty.com.ua/ru/lifestyle/tehnika/20191102-kb-pivdenne-predstavylo-protykorabelnu-nadzvukovu-raketu-blyskavka/)

Противокорабельную сверхзвуковую ракету Блискавка разработали эксперты из днепровского КБ Южное.
Днепровское КБ Южное представило сверхзвуковую ракету Блискавка из нового поколения противокорабельных ракет.
Блискавка может развивать скорость порядка 2000-2500 км/час. Ракету можно оснащать осколочно-фугасной и фугасно-проникающей боевыми частями.
Ракетами этого типа планируют вооружить самолеты СУ-27 и СУ-24М ВСУ, для чего истребители придется минимально переоборудовать, рассказали в КБ Южное.
