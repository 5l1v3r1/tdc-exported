
# Τηλεφώνημα για βόμβα στο Εφετείο - ΤΩΡΑ

Published at: **2019-11-05T09:52:00+00:00**

Author: ****

Original: [enikos.gr](http://www.enikos.gr/society/682372/tilefonima-gia-vomva-sto-efeteio-tora)

Προειδοποιητικό τηλεφώνημα για βόμβα στο Εφετείο Αθηνών έγινε το πρωί της Τρίτης, στα γραφεία εφημερίδας.
Συγκεκριμένα, λίγο πριν τις 9:00 άγνωστος τηλεφώνησε στην «Εφημερίδα των Συντακτών» και προειδοποίησε για την ύπαρξη βόμβας στο Εφετείο Αθηνών.
Ο άγνωστος ανέφερε στο τηλεφώνημα πως δεν πρόκειται για φάρσα και ότι η βόμβα θα εκραγεί σε σαράντα λεπτά.
