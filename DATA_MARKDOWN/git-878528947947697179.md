
# チョコプラ長田、無念つぶやく　ＴＴ兄弟流行語ノミネートならず　ソロリソロリ～も選外

Published at: **2019-11-06T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/06/0012853364.shtml)

　「現代用語の基礎知識選　２０１９ユーキャン新語・流行語大賞」の１９年ノミネート語３０が６日、主催者から発表された。今年はラグビーワールドカップが盛り上がったことから、「ＯＮＥ　ＴＥＡＭ」「ジャッカル」などラグビー関連が５つノミネートされた一方、お笑いネタは芸人が謹慎となった「闇営業」がノミネートされたのみ。毎年何かしらノミネートされていたギャグ系は１つも入らなかった。これにチョコレートプラネットの長田庄平はイッターで無念をつぶやいた。
　毎年何かしらのギャグがノミネートされていた新語・流行語大賞。密かに？ノミネートを狙っていたのか、チョコプラ長田は「ＴＴ兄弟入らず！」とつぶやき、泣き顔の絵文字を添えた。
　今年は体操着姿で両腕を伸ばし「Ｔ、ＴＴ～」と叫ぶＴＴ兄弟ネタの他、狂言師・和泉元彌氏のモノマネの「ソロリソロリ～」もノミネートが噂されていただけに選外に無念。長田のツイッターには「子供枠なら確実に上位です」のなぐさめの言葉も送られていた。
