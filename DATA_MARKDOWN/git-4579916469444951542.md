
# Культовый фильм про супермена впервые записали на тонкий кусок стекла. Microsoft тестирует новую технологию

Published at: **2019-11-05T14:21:00+00:00**

Author: ****

Original: [Фонтанка.ру](https://www.fontanka.ru/2019/11/05/118/)

Как уточняет корпорация 5 ноября, для записи были использованы последние открытия в области сверхбыстрой лазерной оптики и искусственного интеллекта для хранения данных в кварцевом стекле, под названием Project Silica.
«Лазер кодирует данные в стекле, создавая слои трехмерных наноразмерных решеток и деформаций на разных глубинах и под разными углами. Алгоритмы машинного обучения считывают записанные данные, декодируя узоры, которые создает поляризованный свет, проходящий через стекло», – объяснили в Microsoft.
