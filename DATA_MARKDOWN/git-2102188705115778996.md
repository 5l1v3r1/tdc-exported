
# Volkswagen протестирует квантовый навигатор

Published at: **2019-11-03T15:35:00+00:00**

Author: ****

Original: [Экономическая правда](https://www.epravda.com.ua/rus/news/2019/11/3/653261/)

Германский автопроизводитель Volkswagen протестирует навигатор, который использует возможности квантового компьютера для прогнозирования количества пассажиров и оптимизации маршрута.
Об этом пишет "Интерфакс-Украина".
На следующей неделе в Лиссабоне, где в это время будет проходить технологическая конференция Web Summit, между местом проведения мероприятия и другими частями города будут курсировать девять городских автобусов, оснащенных "квантовым маршрутизатором", сообщает The Wall Street Journal.
Это приложение использует доступ к квантовому компьютеру для расчета наиболее быстрого пути для каждого из автобусов в режиме, близком к реальному времени.
Целью тестовых поездок является выбор оптимального маршрута в условиях интенсивного городского движения, заявил главный информационный директор VW Мартин Хофман.
Испытания являются частью более масштабного плана компании по внедрению подобных систем в свои автомобили. VW намерена коммерциализировать эту технологию в ближайшие три года.
В разработке приложения-навигатора участвовала компания D-Wave Systems Inc.
