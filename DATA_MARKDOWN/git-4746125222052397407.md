
# 藤田ニコル、谷間&美くびれあらわなビキニ姿披露 「谷間がエッロ」「美乳だし腹筋えぐい」

Published at: **2019-11-07T11:02:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148199/full/)

　モデルの藤田ニコルが6日、自身のインスタグラムを更新。ビキニ姿で美ボディを披露した。 　「ねぇ、楽しみにしといて??」というコメントとともに、リラックスムードが漂うビキニショットをアップ。白のシンプルなビキニからは、ふっくらとした谷間に美くびれ、鍛えられた腹筋があらわとなっており、抜群のプロポーションが確認できる。 　ファンからは「美乳だし腹筋えぐいし天才　ほんと尊敬」「腹筋すんごいきれいい」「やっばいえちえち」「エロカワ最強」「外国にいる良い女感が溢れ出てる」「ドキッとしました」といった絶賛の声のほか、藤田の呼びかけに対して「楽しみにしとく」「なになに！楽しみ！」と期待を寄せる声が相次いだ。
