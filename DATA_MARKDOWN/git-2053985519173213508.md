
# تمريض القاهرة تنظم مؤتمرها الدولي.. 8 نوفمبر

Published at: **2019-11-06T04:20:00+00:00**

Author: **ياسمين سامي**

Original: [صدى البلد](https://www.elbalad.news/4047729)
