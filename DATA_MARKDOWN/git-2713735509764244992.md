
# Жительница Ингушетии по подложным документам получила 600 тысяч рублей

Published at: **2019-11-07T09:43:00+00:00**

Author: **Яна Багдадян**

Original: [Регион Online](https://news-r.ru/news/ingushetiya/381016/)


        37-летнюю женщину обвиняют по статье «Мошенничество при получении выплат»
      
Магас, 7 ноября. Жительница Республики Ингушетия стала фигуранткой уголовного дела. 37-летняя женщина обвиняется по статье «Мошенничество при получении выплат». Расследование ведут следователи следственного отдела по городу Малгобек.
- По данным следствия, с января 2017 года по декабрь 2018 года женщина работала руководителем ООО «Норман». Подозреваемая вносила в отчеты формы 4-ФСС заведомо ложные сведения о наступлении страховых случаев по беременности и родам, по временной нетрудоспособности и уходу за ребенком. По подложным документам преступница получила около 600 тысяч рублей, - сообщили «Регион online» в пресс-службе СУ СК России по Республике Ингушетия.
Все деньги поступили из Фонда социального страхования Ингушетии на банковский счет ООО «Норман». Оттуда преступница сняла все средства и потратила их по-своему усмотрению.
Расследование уголовного дела продолжается. Устанавливаются все обстоятельства совершенного преступления.
Ранее «Регион online» сообщал, что чиновница из Республики Ингушетия стала фигуранткой уголовного дела. 40-летняя женщина присвоила себе чужие деньги.
