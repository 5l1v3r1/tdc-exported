
# «Северсталь» представит ход реализации обновленной стратегии на Дне инвестора 2019

Published at: **2019-11-07T14:52:00+00:00**

Author: ****

Original: [35медиа](https://35media.ru/news/2019/11/07/severstal-predstavit-hod-realizacii-obnovlennoj-strategii-na-dne-investora-2019)

«Северсталь», одна из крупнейших вертикально-интегрированных горнодобывающих и сталелитейных компаний в мире, сегодня, 7 ноября, проводит в Лондоне ежегодный День инвестора.
В ходе мероприятия выступят: Сакари Тамминен (старший независимый директор), Александр Шевелев (генеральный директор), Алексей Куличенко (заместитель генерального директора по финансам и экономике), Мария Шалина (директор по маркетингу и улучшению клиентского опыта), Евгений Черняков (директор по работе с компаниями строительной отрасли) и Максим Семеновых (руководитель управления корпоративной стратегии). Топ-менеджмент «Северстали» представит ключевые результаты обновленной стратегии компании, объявленной в 2018 году, а также прогноз по результатам деятельности для дальнейшего роста финансовых показателей и создания стоимости для акционеров.
