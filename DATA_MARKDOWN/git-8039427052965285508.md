
# ‘హెచ్చరిక..దేశవ్యాప్తంగా ఉద్యమాన్ని చేపడతాం’

అమరావతి: హిందువులపై కక్షతోనే సీఎస్‌ ఎల్వీ సుబ్రహ్మణ్యం బదిలీ చేశారని రాష్ట్రీయ బ్రాహ్మణ సంఘం అధ్యక్షుడు రాధాకృష్ణ ఆగ్రహం వ్యక్తం చేశారు. సీఎం వెంటనే బ్రాహ్మణులకు బహిరంగ క్షమాపణ చెప్పి ఎల్వీ సుబ్రహ్మణ్యం బదిలీ ఉత్తర్వులను వెనక్కి తీసుకోవాలని డిమాండ్ చేశారు. లేకుంటే దేశవ్యాప్తంగా ఉద్యమాన్ని చేపడతామని హెచ్చరించారు.
