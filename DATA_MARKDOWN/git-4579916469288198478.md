
# Житель Сланцев погиб после неудачной попытки разобрать снаряд для пушек. Необожжёнными остались 10% тела

Published at: **2019-11-03T11:49:00+00:00**

Author: ****

Original: [Фонтанка.ру](https://www.fontanka.ru/2019/11/03/031/)

Не совместимые с жизнью термические ожоги получил пенсионер из Ленинградской области, который пытался разобрать боеприпас военных времён в собственной квартире.
По информации 47news, медики не смогли помочь 57-летнему владельцу мини-арсенала артиллериста, поступившему накануне с термическими ожогами 90% тела. Мужчина утром 3 ноября скончался в сланцевской межрайонной больнице.
Как писала «Фонтанка», пенсионер пострадал в квартире на улице Максима Горького, когда решился разобрать снаряд для малокалиберных советских пушек. Произошёл взрыв, частично обгорел коридор, сожительница экспериментатора госпитализирована с отравлением угарным газом. За её жизнь врачи не опасаются.
Кроме частей разорвавшегося снаряда, полицейские увезли из квартиры неуправляемую авиационную ракету.
