
# В Печоре бывшего директора ЗАО «ВиД» обвиняют в налоговом преступлении

Published at: **2019-11-05T12:55:00+00:00**

Author: ****

Original: [БНК](https://www.bnkomi.ru/data/news/102320/)

Завершено расследование уголовного дела в отношении бывшего генерального директора ЗАО «ВиД», обвиняемого в сокрытии денежных средств организации, за счет которых должны были погашаться недоимки по налогам, сборам и страховым взносам.
По данным регионального следственного управления СКР по Коми, с июля по октябрь 2017 года фигурант, будучи руководителем организации, достоверно зная о наличии задолженности организации по налогам и страховым взносам на 7,9 млн рублей, а также принятых налоговым органом мерах по ее взысканию, совершил действия, направленные на сокрытие денежных средств предприятия на указанную сумму.
В ходе расследования фигурант не признал вину по предъявленному обвинению. Между тем, следствием собрана доказательственная база, что позволило уголовное дело направить в суд для рассмотрения по существу.
