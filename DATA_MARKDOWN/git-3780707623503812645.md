
# Победительница конкурса «Лидеры России» разработала руководство для регионов по управлению инвестициями

Published at: **2019-11-07T14:25:00+00:00**

Author: **Юлия Жук**

Original: [ФедералПресс](https://fedpress.ru/news/77/society/2360482)


        МОСКВА, 7 ноября, ФедералПресс. Победительница конкурса «Лидеры России» (флагманский проект президентской платформы «Россия – страна возможностей») при участии наставника Валентины Матвиенко разработала руководство для регионов по управлению инвестициями.
      
Победительница второго сезона проекта «Лидеры России» Светлана Костикова вместе с командой исследователей приняла участие в создании работы по взаимодействию регионов с РФПИ – Российским фондом прямых инвестиций. Ее наставница Валентина Матвиенко рассказала об этом во время выступления на втором ежегодном форуме «Инвестиции в регионы – инвестиции в будущее».
Как подчеркнула Матвиенко, вопрос инвестирования в регионы становится наиболее значимым в свете реализации нацпроектов, и именно РФПИ служит одним из важнейших инвестиционных инструментов.
По словам Матвиенко, государственно-частное партнерство выходит на первый план в связи с ограниченными возможностями бюджетного финансирования. В 2018 году его участие позволило реализоваться 3,5 тысячи проектов.
