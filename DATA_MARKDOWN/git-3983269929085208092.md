
# Bretxa salarial: les dones treballen gratis des d'avui en comparació amb els homes

Published at: **2019-11-02T19:58:00+00:00**

Author: **Redacció**

Original: [CCMA](https://www.ccma.cat/324/bretxa-salarial-les-dones-europees-treballaran-gratis-a-partir-de-dema/noticia/2960324/)

A partir d'aquest diumenge 3 de novembre, les dones europees ja no cobraran per les seves tasques professionals si es compara amb els sous que reben els homes.
