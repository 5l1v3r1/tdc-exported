
# ロッテ、豪雨被害のキャンプ地・鴨川でチャリティーオークション開催へ

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/01/0012839063.shtml)

　ロッテは１日、台風・豪雨などの被害にあった秋季キャンプ地・千葉県鴨川市への義援活動の一環として１０日、１５時からキャンプ参加全選手と井口監督のグッズによるチャリティーオークションを開催すると発表した。義援金は鴨川市に寄付する。
　場所は晴天時は鴨川市総合運動施設サッカー場、雨天時は鴨川市総合運動施設文化体育館正面玄関前広場。
　チャリティーオークションは１０００円から始まり、１０００円単位で行う。
