
# Беглов проверил ремонт кровли по технологии «холодный чердак» на улице Зайцева

Published at: **2019-11-01T13:06:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/738683-beglov-proveril-remont-krovli-po-tekhnologii-kholodnyi-cherdak-na-ulice-zaiceva)


        Губернатор Петербурга Александр Беглов совершил рабочий визит в Кировский район. Он проверил ремонт кровли дома № 17/23 по улице Зайцева, где применили технологию «холодного чердака».
      
Губернатор Петербурга Александр Беглов во время объезда Кировского района проверил ремонт кровли по технологии «холодный чердак» в доме № 17/23 по улице Зайцева.
В пятиэтажном доме 1959 года постройки отремонтировали балконы, заменили водосточные трубы, восстановили дверные заполнения, заменили оконные блоки в подъездах. На данный момент проходит комплекс работ по нормализации температурно-влажностного режима в чердачном помещении.
