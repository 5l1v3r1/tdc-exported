
# В управлении Минюста по Коми произошли кадровые изменения

Published at: **2019-11-04T06:10:00+00:00**

Author: ****

Original: [БНК](https://www.bnkomi.ru/data/news/102266/)

Татьяна Рыбко покинула должность главы ведомства.
Как сообщает пресс-служба Минюста ЯНАО, на должность начальника управления Министерства юстиции по Ямало-Ненецкому автономному округу Татьяна Рыбко назначена с 28 октября приказом министерста юстиции России в порядке ротации.
Как сообщает Минюст России, с 28 октября исполнение обязанностей начальника управления министерства юстиции по Коми возложено на Ирину Блохину, которая является заместителем руководителя.
