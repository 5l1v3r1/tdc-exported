
# Лёша Свик впервые в Гродно дал большой сольный концерт 

Published at: **2019-11-03T07:14:00+00:00**

Author: ****

Original: [Гродненская правда](https://grodnonews.by/news/fotonovosti/kontsert.html)

Лёша Свик является единственным артистом, четыре сольных сингла которого подряд входили в топ-10 всех стриминговых порталов России. Хитмейкер продолжает стремительно набирать обороты, и на этот раз он презентовал свой новый альбом «Алиби».
