
# 林采緹爆半年沒啪啪啪遭酸　胡睿兒護妻嗆爆網友

Published at: **2019-11-01T02:45:00+00:00**

Author: **張詠淇**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191101ent002/)

「蛇姬」林采緹昨（31日）錄影前受訪時透露自己和剛分居的老公胡睿兒至少半年以上沒有「啪啪啪」的親密行為，新聞曝光後引起不少酸民留言攻擊，胡睿兒一早在臉書po文護妻，認為林采緹唯一做錯的事就是嫁給自己。
胡睿兒坦言的確很久沒有和林采緹有床事，但這都是自己造就，「我老婆有多愛我，我自己非常的清楚，我傷她有多深我自己也非常的清楚 ，你們跟本不了解她的痛苦及無奈，卻在地下自顧自做揣測留言，她是一位好媽媽，善解人意處處替人著想 永遠把自己擺在第二位的人，這樣好的人，你們是在靠爸什麼。」
胡睿兒砲轟網友無謂攻擊， 「如果還有有什麼問題可以直接來問我，不管未來會如何，我永遠都站在我老婆這邊。」並表示為了老婆小孩不會再默不吭聲。
