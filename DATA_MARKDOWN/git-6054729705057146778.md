
# Президентский кортеж направляется в центр Омска

Published at: **2019-11-07T17:16:00+00:00**

Author: **Светлана Порицкая**

Original: [БК55](https://bk55.ru/news/article/159701/)


        Машины проехали по ул. Транссибирской.
      
Из Омского аэропорта по ул. Транссибирской в сторону центра отправился президентский кортеж.
Несколько автомобилей представительского класса, влючающих несколько черных бронированных машин, сопровождают экипажи полиции.
Самого президента России пока разглядеть не удается. Вероятнее всего, это проехал Касым-Жомарт Токаев.
