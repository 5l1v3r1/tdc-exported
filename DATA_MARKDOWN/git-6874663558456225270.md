
# סיסקו חשפה קו מוצרי סייבר חדש, פונה לעסקים קטנים

Published at: **2019-11-07T00:00:00+00:00**

Author: **שירות כלכליסט**

Original: [כלכליסט - www.calcalist.co.il](https://www.calcalist.co.il/internet/articles/0,7340,L-3773309,00.html)

סיסקו חשפה קו מוצרי סייבר חדשים, במסגרת כנס השותפים השנתי שלה בלאס וגאס שנערך השבוע. 
לפי מחקר שערכה החברה, 79% ממנהלי אבטחת המידע בארגונים סבורים שקשה לתזמר תגובה לאיומים, ואילו 48% סבורים שמוקדש יותר מדי זמן לחיסול איומי סייבר שזוהו בארגון, במקום לאתר איומים חדשים או לתקן תשתיות ורכיבים רגישים כדי למנוע חדירות עתידיות. 
כדי לתת מענה לצרכים שזיהתה, הציגה החברה פלטפורמת אבטחה שאמורה לשפר את ביצועי תחקור אירועי הסייבר והמענה אליהם, לתמוך בחיבור לתשתיות ענן והגנה עליהן ולהוריד באופן כללי את העומס שמוטל על מנהלי האבטחה בארגונים.  
כמו כן, הציגה קו מוצרים חדש לעסקים קטנים, סקטור שוק עמו היא פחות מזוהה, ויאפשר לה להגדיל את פעילותה. לצידם, הציגה החברה מתגי אבטחה חדשים בסדרת Meraki שלה.
