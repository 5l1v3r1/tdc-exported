
# DISH// 北村匠海作詞の新曲リリックビデオ公開 ツアー初日に配信リリース

Published at: **2019-11-07T17:30:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148234/full/)

　4人組ダンスロックバンドDISH//が、9日からスタートする全国ツアー『LIVE TOUR ? DISH// ? 2019』初日に新曲「PM 5:30」をリリースすることが決定し、ジャケット写真とリリックビデオがタイトルに絡めて午後5時30分に公開された。 　同曲の作詞には、俳優としても活躍するボーカルの北村匠海が参加。黄昏時に感じる心の動きを自身の言葉で紡ぎだした瑞々しいラブソングとなっている。 　北村は「夕暮れ頃、もしこれを聴いてくれる人がひとりぼっちだったら、その人を救ってくれるような曲になればいいなあと思って歌詞を書きました。是非聴いてみてください！」とのコメントを寄せた。
