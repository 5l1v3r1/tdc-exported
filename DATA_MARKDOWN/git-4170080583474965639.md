
# Тульская ветстанция закупит анализатор для химико-токсикологических исследований

Published at: **2019-11-03T14:54:00+00:00**

Author: ****

Original: [Тульская Служба Новостей](https://www.tsn24.ru/tulskaya-vetstancziya-zakupit-analizator-dlya-himiko-toksikologicheskih-issledovanij.html)

Тульская областная ветеринарная станция по борьбе с болезнями животных закупит анализатор для химико-токсикологических исследований. Извещение о проведении аукциона размещено на сайте госзакупок.
Согласно представленным документам, будет закуплен один прибор. Стоимость контракта – 11 миллионов 350 тысяч рублей.
В комплект оборудования должен входить анализатор для определения остаточных пестицидов в образцах пчеловодства на базе хроматографа с масс-селективным детектором, включая аппаратно-программный комплекс.
Напомним, летом 2019 года в нескольких районах Тульской области была зафиксирована массовая гибель пчел. После проведения исследований выяснилось, что причиной послужили пестициды.
Губернатор Тульской области Алексей Дюмин на приеме граждан сообщил, что 20 миллионов рублей направят на техническое оснащение и перевооружение отрасли пчеловодства.
