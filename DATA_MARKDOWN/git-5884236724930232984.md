
# Боровичских школьников познакомили с работой сотрудников ГИБДД

Published at: **2019-11-04T17:36:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Velikij-Novgorod/739751-borovichskikh-shkolnikov-poznakomili-s-rabotoi-sotrudnikov-gibdd)


        Ознакомительное мероприятие в Боровичах провели для будущих абитуриентов ВУЗов МВД в рамках проекта «Молодежь и полиция: шаг в будущее».
      
Будущих правоохранителей ждала экскурсия по зданию Госавтоинспекции. Учащимся рассказали о процедуре регистрации транспортных средств и выдачи номеров и показали кабинет, где сдают экзамен на знание Правил дорожного движения для получения водительского удостоверения.
Далее ребят познакомили с программой автоматизированной информационно-управляющей системы ДПС, которая позволяет систематизировать и анализировать причины аварий для принятия мер по снижению их количества. Побывали в группе технического и дорожного надзоров, а также в отделе по исполнению административного законодательства.
В завершение школьникам напомнили о правилах безопасности на дорогах и подарили светоотражающие элементы. Об этом сообщает пресс-центр УМВД по Новгородской области.
