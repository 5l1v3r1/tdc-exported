
# 本田圭佑、フィテッセ入団決定「重要な存在になりたい」

Published at: **2019-11-07T01:23:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/soccer/20191107-OYT1T50010/)

　【マドリード＝岡田浩幸】サッカーの元日本代表ＭＦ本田圭佑（３３）が、オランダ１部リーグのフィテッセに今季終了までの契約で加入することが決まった。６日に同クラブが発表し、本田はクラブの公式サイトで「チームにとって重要な存在になりたい」とコメントした。本田は５月にメルボルン・ビクトリー（豪）を退団後は無所属となり、２０２０年東京五輪出場を目指して欧州で所属先を探していた。フィテッセは現在国内リーグ５位で、本田が２０１０～１３年に所属したＣＳＫＡモスクワ（ロシア）を当時指揮していたスルツキ氏が監督を務めている。
