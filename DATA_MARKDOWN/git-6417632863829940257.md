
# Диего Симеоне мағлубиятсиз серия якунланганидан сўнг нималар деди?

Published at: **2019-11-07T07:48:00+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/diego-simeone-maglubiyatsiz-seriya-yakunlanganidan-song-nimalar-dedi)

"Жамоа биринчи бўлимда яхши ўйнай олмаганлиги бу менинг айбим. Чунки уларга биринчи бўлим учун мотивация топиб бера олмадим.
Энди буёғига ишлашдан тўхтамаслик керак. Агар нимадир ўхшамаса, биз икки карра кўпроқ ҳаракат қилишимиз керак бўлади", - дейди Симеоне.
Эслатиб ўтамиз, "Атлетико" 4та ўйинда 7 очко жамғарган ҳолда гуруҳда "Ювентус"дан сўнг иккинчи ўринда бормоқда.
