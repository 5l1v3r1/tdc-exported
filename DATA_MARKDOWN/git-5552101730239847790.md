
# 【搶鏡頭】自備辣妹長襪　李淳遭蔡瑞雪奶頂

Published at: **2019-11-01T23:58:00+00:00**

Author: **余樂珠, 攝影組**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191029ent020/)

老爸李安來台宣傳自己的電影，李淳的《陪你很久很久》頗沾光。除了爸媽現身力挺，他自己也短褲加上長襪，遠看頗有辣妹駕到的姿態～重點是同擁兩女，只見蔡瑞雪硬是比邵雨薇高招，奶奶就這樣輕輕且默默地給頂了上去耶～
　
最近蔡依林除了時不時有公開活動，或許也因為演唱會在即什麼的，她經常上臉書與粉絲互動；之前蔡依林竟轉貼一則「14年過後，蔡依林、侯佩岑，還是選侯佩岑嗎？」的新聞，裡頭網友一面倒，大推特推的還是侯佩岑～蔡依林這廂也無畏拿這報導來自婊，果然大器。躺著也中槍的倒是周董，最後選昆凌的他，過去劈腿一事又再被細細回味了呢！
