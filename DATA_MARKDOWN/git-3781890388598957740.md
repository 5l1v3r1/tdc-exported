
# สบายๆ'ปราปต์ปฎล' ซิงเกิลที่ทำด้วยใจ'ให้รักเดินทาง'

Published at: **2019-11-03T08:56:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/entertainment/739810)


        เดลินิวส์ออนไลน์ ชวนแฟนๆคุยสบายๆกับนักแสดงคนดัง ปราปต์ปฎล สุวรรณบาง ในมุมของเพลงที่หลายคนไม่ค่อยรู้
      
จัดเป็นอีกหนึ่งนักแสดงคุณภาพที่แฟนๆชื่นชอบในผลงานและกิจกรรมเพื่อสังคมมาโดยตลอด สำหรับนักแสดงรุ่นใหญ่ ปราปต์-ปราปต์ปฎล สุวรรณบาง ที่ไม่ได้มีดีแค่งานแสดงเท่านั้น แต่ด้านการร้องเพลงหนุ่มปราปต์ก็ถือว่าทำได้ดีมากๆ ล่าสุดหนุ่มปราปต์เดินทางมาเยี่ยมเยือน “เดลินิวส์ออนไลน์” พร้อมหอบซิงเกิลที่ 2 ของตัวเองที่ตั้งใจทำมากๆมาให้แฟนๆให้ได้รู้จักกันด้วย
