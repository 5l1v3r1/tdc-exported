
# ปชป.คลอดชื่อกมธ.แก้รัฐธรรมนูญสัปดาห์หน้า

Published at: **2019-11-03T09:06:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/politics/739812)


        “เทพไท” ขอบคุณ “วันนอร์” เชียร์ “มาร์ค” นั่งปธ.กมธ.แก้รธน. ระบุต้องรอปชป.เคาะมติ ส่ง “องอาจ” ทาบทาม
      
เมื่อวันที่ 3 พ.ย. นายเทพไท เสนพงศ์ ส.ส.นครศรีธรรมราช พรรคประชาธิปัตย์ กล่าวถึงกระแสการผลักดันให้นายอภิสิทธิ์ เวชชาชีวะ อดีตนายกรัฐมนตรี และอดีตหัวหน้าพรรคประชาธิปัตย์ เป็นประธานคณะกรรมาธิการวิสามัญศึกษาปัญหา และแนวทางแก้ไขรัฐธรรมนูญว่า ต้องขอขอบคุณนายวันมูหะหมัดนอร์ มะทา หัวหน้าพรรคประชาชาติ ที่นำเสนอเรื่องนี้ขึ้นมา นายวันมูหะหมัดนอร์เป็นผู้อาวุโสทางการเมือง ผ่านการเป็นรัฐมนตรีมาหลายกระทรวง เคยเป็นประธานรัฐสภามาแล้ว คงเล็งเห็นว่านายอภิสิทธิ์มีความเหมาะสมกับการเป็นประธานคณะกรรมาธิการวิสามัญชุดนี้ เพราะเคยเป็นนายกรัฐมนตรีมาก่อน มีบารมีและประสบการณ์ในทางการเมืองสูง มีความรู้ความสามารถในทุกด้าน  และที่สำคัญนายอภิสิทธิ์มีจุดยืนด้านประชาธิปไตยที่ชัดเจนคนหนึ่ง และสามารถใช้เวลาช่วงที่ไม่ได้เป็นส.ส.ทุ่มเทให้กับการทำงานเรื่องรัฐธรรมนูญได้อย่างเต็มที่ แต่เรื่องดังกล่าวจะมีความเป็นไปได้หรือไม่ ก็ต้องขึ้นอยู่กับการตัดสินใจของพรรคประชาธิปัตย์ และตัวของนายอภิสิทธิ์ด้วย เพราะพรรคประชาธิปัตย์มีขั้นตอนในการทำงานที่ชัดเจน  “ขณะนี้พรรคกำลังรอผลการประชุมของวิปรัฐบาลก่อนว่าจะกำหนดจำนวนกรรมาธิการในคณะกรรมาธิการวิสามัญชุดนี้กี่คน และพรรคประชาธิปัตย์ได้สัดส่วนกรรมาธิการกี่คน ก่อนจะนำเข้าสู่ที่ประชุมส.ส. เพื่อคัดเลือกผู้เหมาะสมกับการเป็นกรรมาธิการวิสามัญชุดนี้ หากที่ประชุมส.ส.ของพรรคมีการเสนอชื่อนายอภิสิทธิ์ และถ้าที่ประชุมมีมติเห็นชอบ ก็จะเป็นหน้าที่ของประธานส.ส.คือนายองอาจ คล้ามไพบูลย์ เป็นผู้ไปทาบทามท่านอภิสิทธิ์ด้วยตนเอง ผมเชื่อว่าในพรรคประชาธิปัตย์ มีผู้เหมาะสมมีความรู้สามารถในเรื่องรัฐธรรมนูญหลายคน ซึ่งพรรคต้องคัดเลือกผู้ที่เหมาะสม และมีประสบการณ์ทางการเมืองมากที่สุด เพื่อเข้าทำหน้าที่ในคณะกรรมาธิการวิสามัญชุดนี้ ไม่อยากให้การแก้ไขรัฐธรรมนูญถูกกล่าวหาว่า เป็นรัฐธรรมนูญที่คนเขียนไม่ได้ใช้ คนใช้ไม่ได้เขียน จึงอยากให้นักการเมืองในฐานะผู้ปฏิบัติโดยตรงมีส่วนร่วมในการแก้ไขรัฐธรรมนูญฉบับนี้ด้วย ซึ่งรายชื่อคณะกรรมาธิการวิสามัญชุดนี้ในส่วนของพรรคประชาธิปัตย์จะมีความชัดเจนในสัปดาห์หน้าอย่างแน่นอน” นายเทพไท กล่าว.
