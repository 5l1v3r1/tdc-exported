
# オリックス・宜保　５０万増で更改「まだ力負け」来春キャンプまでに３キロ増誓う

Published at: **2019-11-06T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/06/0012852855.shtml)

　オリックス・宜保翔内野手が６日、高知市内で契約更改交渉に臨み５０万円アップの５５０万円で合意した。（金額は推定）
　昨秋のドラフト５位で未来沖縄から入団した今季は８試合に出場しプロ初安打を含む６安打を放つなど打率・２３１、０本塁打、０打点だった。
　「入って来たときのイメージに比べたら成長できたかなと思います。まだすべてにおいて力負けしている。足の速さだったり体の強さだったり。差を感じました。来年のキャンプまでに３キロ増やして臨みたい」
　宜保は現在、高知市で行われている秋季キャンプに参加しているが、２３日からは台湾で開催される「２０１９アジアウインターベースボールリーグ」に参加する。
　「ほかの選手と比べて実戦の機会があるのは大きい。１軍ではストレートのはじきができなかった。ストレートに強い打者になってきたい」と目標をあげていた。
