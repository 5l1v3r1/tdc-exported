
# キンコン梶原が生告白　ユーチューバー・カジサックは「最初たたかれ、つらかった」

Published at: **2019-11-02T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/02/0012842624.shtml)

　キングコングの梶原雄太（３９）が２日、都内で行われた「トイザらス　クリスマスイベント２０１９」にユーチューバー「カジサック」として登場した。
　今や子供の人気職業となっているユーチューバーの活動を昨年１０月からスタートさせたが「最初は色んな人に『何やっとんねん』とたたかれ、つらかった」と、客席の大半を子供が埋めた会場で真面目トーク。「今は芸能界の色んな方が（動画を）上げられていてうれしい。ユーチューブの視聴者にも、もう１回、テレビを見ようと思ってもらえるかも」と、ひそかな願いを打ち明けた。
　子供たちとの質疑応答では「会って１番うれしかった有名人」を聞かれて「イチローさん。ゴルフ場でたまたま会って、遠巻きからでも分かって、あいさつに行った」と興奮気味に回想。来年２月に誕生予定の第５子の名前を質問されると「男の子だったらイチロー」と、三男になるだけに本気ともジョークともとれる回答をしていた。
