
# 【プレゼント】2019年11月7日(木)発売の「フォートナイト ダークファイアバンドル」、PS4版とSwitch版を各1名にプレゼント！

アキバ総研にて、「フォートナイト ダークファイアバンドル」が抽選で当たるプレゼントキャンペーンが実施中だ。
「フォートナイト」は、無料で遊べる、TPSタイプのバトルロイヤルゲーム。集めた素材で壁や坂を作って相手の攻撃から身を守ったり、あるいは奇襲や移動に使ったりするといったクラフト要素が軸になっており、通常のシューティングとはひと味違ったバトルを楽しめる。「フォートナイト ダークファイアバンドル」には、レジェンドのコスチュームとバックアクセサリーが3種ずつ、さらにレアのデュアルツルハシ、エピックのラップが各3つ、それらに新しいエモートを加えた全13個のコンテンツが収録されている。発売日以降、全国のゲームソフト取扱店に並ぶ予定だ。
バトル！クラフト！クリエイト！フォートナイト バトルロイヤルは、基本プレイ無料の常に進化を続けるマルチプレイゲームです。フレンドと100人のPvPモードで最後まで生き残れるかを競い合ったり、協力して夢に描いたフォートナイトワールドを作ることもできる。
今回、アキバ総研ではこの「フォートナイト ダークファイアバンドル」のPlayStation 4版、Nintendo Switch版をそれぞれ1名、計2名にプレゼント！
