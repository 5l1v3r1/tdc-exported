
# В коррупции подозревают ряд сотрудников акимата Шымкента

Published at: **2019-11-04T12:57:00+00:00**

Author: **ИА Тотал Казахстан**

Original: [Тотал](https://total.kz/ru/news/proisshestviya/v_korruptsii_podozrevaut_ryad_sotrudnikov_akimata_shimkenta_date_2019_11_04_12_24_08)


        Двух руководителей из акимата Шымкента подозревают во взяточничестве.
      
Антикоррупционной службой Шымкента начаты досудебные расследования в отношении ряда должностных лиц по подозрению в совершении коррупционных преступлений, сообщает Total.kz.
