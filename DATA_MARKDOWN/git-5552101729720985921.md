
# 【騙贏貴婦奈奈4】比基尼辣照超吸睛　富千金奢華生活臉書全曝光

Published at: **2019-11-05T21:58:00+00:00**

Author: **顏凡裴, 鄒保祥, 林育緯**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191105inv007/)

潤寅實業負責人楊文虎及王音之夫婦，賄賂8大公司高層，詐騙14家銀行上百億元，2人潛逃至美國，竟還越洋遙控女兒在台灣協助脫產。2人女兒楊宇晨檢調上月首度提訊楊宇晨出庭，原本貴為千金、氣質出眾的她，身穿囚服現身，神色顯得憔悴，和她在臉書上「白富美」的模樣簡直判若2人。
