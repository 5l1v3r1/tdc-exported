
# СБУ взяла сотрудников спецслужбы на незаконной прослушке

Published at: **2019-11-06T12:21:00+00:00**

Author: ****

Original: [Главное™](https://glavnoe.ua/news/n357008225-sbu-vzjala-sotrudnikov-specsluzhby-na-nezakonnoj-proslushke)

Оперативники Управління внутрішньої безпеки СБУ викрили злочинну діяльність діючих та колишнього співробітників Служби на незаконному прослуховуванні абонентів та збуті отриманої інформації. Встановлено, що зловмисники через особисті перевірені зв’язки підшукували клієнтів для надання «послуг».
Колишній працівник підбирав замовників, діючі ‒ технічно забезпечували зняття інформації.
Вартість прослуховування одного абонентського номера на два місяці становила дев’ять тисяч доларів США.
Досудове слідство встановлює точну кількість абонентів, які потерпіли від дій фігурантів справи. У межах кримінального провадження, розпочатого за ч. 3 ст. 368 Кримінального кодексу України, триває досудове слідство.
Наразі тривають невідкладні слідчо-оперативні дії, фігуранти провадження затримані, приймається рішення щодо оголошення їм про підозри у вчиненні кримінального правопорушення. Операція з викриття зловмисників проводилась спільно зі слідчими Державного бюро розслідувань, під процесуальним керівництвом Головної військової прокуратури.
