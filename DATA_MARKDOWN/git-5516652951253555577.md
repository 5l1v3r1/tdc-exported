
# Την Κυριακή οι εκλογές για την ανάδειξη νέων οργάνων στο ΤΕΕ Δωδεκανήσου

Published at: **2019-11-01T08:45:01+00:00**

Author: ****

Original: [Η ΡΟΔΙΑΚΗ](https://www.rodiaki.gr/article/424914/thn-kyriakh-oi-ekloges-gia-thn-anadeixh-newn-organwn-sto-tee-dwdekanhsoy)


        Πέντε συνδυασμοί πρόκειται να αναμετρηθούν στις εκλογές που θα πραγματοποιηθούν την Κυριακή 3 Νοεμβρίου στο νομό μας για την ανάδειξη νέων οργάνων στο ΤΕΕ Δωδεκανήσου.
      
Στις εκλογές θα πάρουν μέρος οι εξής συνδυασμοί:

        - Δημοκρατική Κίνηση Μηχανικών (ΔΚΜ)
        
        
          - Δωδεκανησιακή Πρωτοβουλία Μηχανικών (ΔΩ.ΠΡΩ.ΜΗ.)
          
          - Συνεργασία Μηχανικών Δωδεκανήσου
          
          - Μηχανικών Ένωσις
          
          - Πανεπιστημονική-Δημοκρατική Πανεπιστημονική Κίνηση Μηχανικών
        
      
Σημειώνεται ότι οι ίδιοι συνδυασμοί συμμετείχαν και στις τελευταίες εκλογές που έγιναν στο ΤΕΕ, κατά τις οποίες κατέλαβαν:
- ΔΩ.ΠΡΩ.ΜΗ: 11 έδρες
- Συνεργασία Μηχανικών Δωδεκανήσου: 10 έδρες
- ΔΚΜ: 8 έδρες
- Μηχανικών Ένωσις: 3 έδρες
- Πανεπιστημονική-Δημοκρατική Πανεπιστημονική Κίνηση Μηχανικών : 1 έδρα
