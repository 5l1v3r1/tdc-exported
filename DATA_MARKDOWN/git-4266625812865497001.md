
# Низкий рейтинг не является проблемой для Зеленского, - Загородний

Published at: **2019-11-06T12:20:00+00:00**

Author: ****

Original: [ZIK.UA](https://zik.ua/ru/news/2019/11/06/nyzkyy_reytyng_ne_yavlyaetsya_problemoy_dlya_zelenskogo__zagorodnyy_1684837)


        Хотя рейтинг президента Украины Владимира Зеленского немного упал - это для него не критично. Если сравнивать с Зеленским, то за это время экс-президент Петр Порошенко больше потерял.
      
Такое мнение в эфире информационно-аналитического проекта «Великолепная четверка» на телеканале ZIK выразил политтехнолог Тарас Загородний.
