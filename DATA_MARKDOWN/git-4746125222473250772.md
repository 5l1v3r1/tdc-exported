
# ドローン対人間の“究極の鬼ごっこ”にHiHi Jetsら参戦 指原莉乃も興奮「手に汗握る展開で…」

Published at: **2019-11-07T17:00:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148233/full/)

　元HKT48でタレントの指原莉乃、お笑いタレントの陣内智則がMCを務める日本テレビ系バラエティー特番『DRONE×鬼〜生き残りをかけた究極の鬼ごっこ〜』が、あす8日に放送（後7：00〜8：54）。ドローンと人間による“究極の鬼ごっこ”が行われる。 　鬼は、日本を代表するドローンパイロットが操縦する手の平サイズの“最新マイクロドローン”。テレビ局や特設ステージ、東京湾豪華クルーズ船を舞台に、現役プロアスリートを含む総勢40人の人気タレントが、ドローンから逃げまくり、賞金総額100万円をかけた死闘が繰り広げられる。 　指原は「とても楽しく手に汗握る展開で、ドローンってこんなにすごいんだという驚きがありました。結局、機械は人間には勝てないだろうと思って見ていましたが、ドローンの細かい動きは生きているみたいでした。いろんな芸能人の方がいるので、誰が生き残るのか、誰が勝ち進むのか予想しながら楽しんで見てください！」とアピール。 　一方の陣内も「正直こんなに迫力があるとは思っていませんでした。パニック映画をみているようでスリリングで感動もありました。人間のダマし合いがあったり、協力しあったり、人間が勝つのかドローンが勝つのか、どっちが上なのかぜひ見ていただきたいです」と呼びかけている。 ■『DRONE×鬼〜生き残りをかけた究極の鬼ごっこ〜』挑戦者（五十音順）阿佐ヶ谷姉妹、アルコ＆ピース、安藤美姫、伊藤剛臣、伊藤大海（日テレアナウンサー）、ヴァンゆん、岡副麻希、おばたのお兄さん、神野大地、菊地亜美、熊田曜子、クロちゃん、C.C.ガールズ3（中川知香、花岡なつみ、豊田百佳）、Gパンパンダ、鈴木奈々、鈴木福・夢きょうだい、トム・ブラウン、中村静香、HiHi Jets（高橋優斗・井上瑞稀・猪狩蒼弥）、馬場ももこ、藤光謙司、ブリリアン、丸山桂里奈、森渉、山田親太朗、四千頭身、ワタリ119
