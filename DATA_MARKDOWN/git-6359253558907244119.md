
# В России создали полностью углепластиковую спутниковую антенну

Published at: **2019-11-06T15:55:44+00:00**

Author: ****

Original: [Новый День](https://newdaynews.ru/technology/675925.html)

Российские ученые создали опытный образец наземной антенны (рефлектора) для спутниковой связи, полностью изготовив его из углепластика, рассказал гендиректор предприятия «Информационные спутниковые системы им. академика М.Ф.Решетнева» Николай Тестоедов.
