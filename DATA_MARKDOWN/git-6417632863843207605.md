
# Реал собиқ футболчиси испан жамоасига бош мураббий этиб тайинланди

Published at: **2019-11-05T07:40:00+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/real-sobiq-futbolchisi-ispan-jamoasiga-bosh-murabbiy-etib-tayinlandi)

Бу ҳақда клуб расмий сайти хабар бермоқда. "Альмерия" Испанияда мавқей жиҳатдан иккинчи ўринда турувчи дивизионда ўйнайди.
Битимнинг қанча муддатга имзолангани ҳақида маълумот йўқ.
43 ёшли Гути футболчилик фаолияти давомида "Реал" ва "Бешиктош" клубларида ўйнаган.
У 1999-2005 йиллар давомида Испания терма жамоаси шарафини ҳимоя қилган.
