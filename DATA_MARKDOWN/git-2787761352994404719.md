
# В Белых Берегах заметили ночную ловушку для прохожих

Published at: **2019-11-03T14:58:36+00:00**

Author: ****

Original: [Брянск Today](https://bryansktoday.ru/article/111839)

В поселке Белые Берега Фокинского района Брянска сфотографировали огромную яму на переулке Вокзальном. Снимки опубликовали в социальной сети «ВКонтакте» сегодня, 3 ноября.
— Уже месяц такая картина. Ночная ловушка для прохожих, — возмутился автор поста.
