
# ప్రైవేట్ ట్రావెల్స్ బస్సు ఢీకొని మహిళ మృతి

హైదరాబాద్: ప్రైవేట్ ట్రావెల్స్ బస్సు ఢీకొని మహిళ అక్కడికక్కడే మృతి చెందిన ఘటన స్థానికంగా కలకలం రేపింది. చాదర్‌ఘాట్ పరిధిలోని నల్గొండ క్రాస్ రోడ్డులో ఈ ప్రమాదం చోటు చేసుకుంది. ద్విచక్ర వాహనంపై వెళుతున్న మహిళను ప్రైవేట్ ట్రావెల్స్ బస్సు ఢీకొట్టడంతో ఆమె మృతి చెందింది.  పోలీసులు కేసు నమోదు చేసుకుని దర్యాప్తు చేస్తున్నారు. మహిళకి సంబంధించిన వివరాలు తెలియాల్సి ఉంది. మహిళ మృతదేహాన్ని పోలీసులు ఉస్మానియాకు తరలించారు.
