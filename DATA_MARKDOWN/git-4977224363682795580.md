
# V Veliki Pristavi zagorelo gospodarsko poslopje

Published at: **2019-11-05T11:19:25+00:00**

Author: ****

Original: [STA.si](https://www.sta.si/2694165/v-veliki-pristavi-zagorelo-gospodarsko-poslopje)


        Pivka, 5. novembra - Zjutraj je v vasi Velika Pristava zagorelo gospodarsko poslopje. Zagorelo naj bi seno. Požar so gasilci pogasili, prizorišče požara pa si bodo ogledali tudi kriminalisti. Po prvih ocenah gmotna škoda znaša približno 70.000 evrov, so sporočili s Policijske uprave Koper.
