
# 3 ноября дежурный по Туле – Дмитрий Городничев

Published at: **2019-11-03T07:12:00+00:00**

Author: ****

Original: [Тульская Служба Новостей](https://www.tsn24.ru/3-noyabrya-dezhurnyj-po-tule-dmitrij-gorodnichev.html)

Сотрудники администрации Тулы продолжают дежурства в выходные дни.
В воскресенье, 3 ноября, за порядком в городе проследит заместитель начальника главного управления — начальник отдела по благоустройству главного управления администрации города Тулы по Центральному территориальному округу Дмитрий Городничев.
В пресс-службе администрации Тулы напоминают, что по всем возникающим вопросам следует обращаться в Единую дежурно-диспетчерскую службу по номеру: 1391, (4872)47-20-34; (4872)47-20-37.
