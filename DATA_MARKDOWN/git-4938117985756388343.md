
# Как угоняют Kia Rio с помощью ножниц по металлу?

Published at: **2019-11-03T00:00:00+00:00**

Author: **Дмитрий Плотников**

Original: [CAR.RU](https://car.ru/news/automobili/56729-kak-ugonyayut-kia-rio-s-pomoschyu-nozhnits-po-metallu/)

Южнокорейский седан Kia Rio является одним из бестселлеров отечественного автомобильного рынка. Поэтому нередко он становится добычей угонщиков. В ход идут даже экзотические предметы.
Блогеры из YouTube канала «Угона.нет» показали, как с помощью ножниц можно завладеть автомобилем. Конечно, речь идет не о тех изделиях, которые есть в каждом доме. Злоумышленники используют специальные ножницы по металлу, которые есть в хозяйственных или строительных магазинах.
В переднем подкрылке многих азиатских автомобилей есть небольшой лючок. Через него открывается доступ к проводу питания, соединяющего аккумулятор с предохранителями. Именно его перерезают ножницами.
 
Далее угонщик легко справляется с личинкой водительской двери и проникает в салон. Вся бортовая электроника обесточена. Молчит и сигнализация, поскольку не снабжена автономным питанием. Что происходит дальше, нетрудно догадаться.
Все манипуляции можно проследить на видео. Естественно, категорически не стоит их повторять в реальной жизни.
