
# Жена G.O обвинила СМИ в использовании их изображений без согласия

Published at: **2019-11-07T03:23:44+00:00**

Author: **Хотабовна**

Original: [YesAsia](https://www.yesasia.ru/article/754701)

Чхве Е Сыль обратилась к корейским СМИ с просьбой не использовать изображения супружеской пары для новостей без их согласия.

        Смотрите также: Ли Хай отпраздновала седьмую годовщину с момента дебюта
      
На свою страницу в Intagram девушка загрузила скриншот новостной статьи, к которой была использована фотография Е Сыль и G.O. По контексту новость никак не связана с парой, поэтому под постом она написала: «Это уже слишком. Пожалуйста, не используйте наши изображения без разрешения».
В своих комментариях нетизены по этому поводу пишут: «Должно быть, ты ужасно себя из-за этого чувствуешь. Как они могли использовать фото молодоженов для такой статьи?», «На них стоит доложить», «В Корее не осталось здравомыслящих», и многое другое.
