
# Запорожцы задолжали за коммуналку 3 миллиарда

Published at: **2019-11-05T15:49:00+00:00**

Author: **Владимир Андреев**

Original: [РИА Мелитополь](https://ria-m.tv/news/170668/zaporojtsyi_zadoljali_za_kommunalku_3_milliarda.html)


        Больше всего денег жители региона должны за газ.
      
Запорожское областное управление статистики опубликовало данные по задолженности жителей Запорожской области за потребленные комуслуги, пишет Заноза.
По данным на сентябрь запорожцы заплатили 289 млн. гривен за коммунальные услуги. При этом общая задолженность за коммуналку составила 3 млрд. гривен.
Больше всего потребители должны за газ: более 983 млн. гривен, еще 915,6 млн  – за горячую воду, за содержание домов "набежал" долг свыше 307 млн.
При этом, например, в запорожских "Теплосетях" можно платить за отопление по частям, разбивая платежи на год и погашать долг до начала следующего отопительного сезона.
