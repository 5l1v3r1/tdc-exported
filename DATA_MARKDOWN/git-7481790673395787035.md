
# 容器洗浄後に微生物残った可能性…サントリー、ミルクティー自主回収

Published at: **2019-11-05T09:10:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/economy/20191105-OYT1T50186/)

　サントリー食品インターナショナルは５日、ペットボトル飲料「サントリー　クラフトボス　ミルクティー」（５００ミリ・リットル）で味が変化した商品が見つかったため、約１７０万本を自主回収すると発表した。製造委託先の工場で容器を洗浄する際、一部商品で微生物が残った可能性があるという。これまでに健康被害は確認されていない。
　問い合わせは、特設お客様対応窓口（０１２０・７７４・３１０）。
