
# 福州羽賽周天成直落二擊退王子維

Published at: **2019-11-07T09:11:04+00:00**

Author: ****

Original: [NTD 新唐人電視台](https://www.ntdtv.com/b5/2019/11/07/a102702018.html)

【新唐人北京時間2019年11月07日訊】中國福州羽球公開賽6日進行「台灣內戰」，台灣好手周天成與王子維在男子單打賽事短兵相接，最後由周天成以21：19、21：17直落二擊退王子維。
周天成一開賽就打出一波5：3，並持續拉開比分，不過王子維也鍥而不捨地追成8：8平手，雙方來回拉鋸，9度打成平手，19：19後，周天成連下2分，以21：19先下一城。
第二局開始，王子維以一波4：2攻勢先聲奪人，不過周天成隨即追上，反以6：4超前，打成9：7後，周天成連下5分，王子維持續追趕，好不容易追成17：19僅有2分落後，此時周天成一口氣連拿2分，以21：17收下晉級資格。
而在男單其他選手賽事中，中國諶龍以19：21、12：21、12：21逆轉打敗同胞林丹；女子賽事中，中國好手陳雨菲、泰國依瑟儂（Ratchanok Intanon）都順利晉級。
