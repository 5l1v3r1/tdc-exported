
# Microsoft представила гнучку мишку

Published at: **2019-11-06T17:47:00+00:00**

Author: ****

Original: [DT.ua](https://dt.ua/TECHNOLOGIES/microsoft-predstavila-gnuchku-mishku-328737_.html)

Компанія Microsoft представила новий гнучкий пристрій. Але їм став не смартфон або планшет, а мишка Microsoft Mavic Arc Mouse, яку можна зігнути по формі руки.
Головною особливістю пристрою є те, що його форму можна змінювати. Мишку можна зробити або абсолютно плоскою або ж зігнути таким чином, щоб вона була зручна для руки. За запевненням представників Microsoft, таке рішення робить пристрій ідеальним для транспортування, оскільки він практично не займає місця в багажі.

Samsung офіційно представила Galaxy Fold у лютому. Смартфон обладнаний двома дисплеями, один з яких - Infinity Flex Display діагоналлю 7,3 дюйма з роздільною здатністю 1536 на 2152 пікселів, що можна скласти навпіл. У складеному стані можна використовувати інший, менший, дисплей діагоналлю 4,6 дюйма з роздільною здатністю 840 на 1960 точок.
