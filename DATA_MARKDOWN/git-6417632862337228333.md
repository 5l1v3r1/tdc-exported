
# Гризманн: Левантега қарши баҳсда нима бўлганини тушунтириб бера олмайман

Published at: **2019-11-03T09:22:12+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/grizmann-levantega-qarshi-bahsda-nima-bolganini-tushuntirib-bera-olmayman)

Эслатиб ўтамиз, ушбу баҳсда Каталония клуби 7 дақиқа ичида 3та гол ўтказиб юборди.
"Танаффусдан сўнг бироз бўшашдик. Нега бундай бўлганини айта олмайман, аммо "Леванте"нинг 3та голига қандай жавоб беришни билмадик. Содир бўлган воқеани тушунтириб бера олмайман. Афсуски, бу баҳсда омадимиз келмади" - дейди Гризманн.
