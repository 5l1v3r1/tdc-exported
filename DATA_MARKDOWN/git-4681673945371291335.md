
# Обоснование создания легкорельсового транспорта в Воронеже появится в 2020 году

Published at: **2019-11-02T08:22:00+00:00**

Author: **Марина Соловьёва**

Original: [36on.ru — Воронежский городской портал](http://36on.ru/news/people/87485-obosnovanie-sozdaniya-legkorelsovogo-transporta-v-voronezhe-poyavitsya-v-2020-godu)


        До конца 2020 года в Воронеже должны разработать технологическое и финансово-экономическое обоснование создания системы мультимодальных пассажирских перевозок. В данный момент власти ищут подрядчика, который смог бы подготовить соответствующую документацию.
      
Согласно распоряжению, опубликованному на сайте городской администрации, подрядчик обязан будет проанализировать существующую маршрутную сеть, а также предложить наилучший вид транспорта для решения дорожных проблем. Выбор необходимо будет обосновать.
Помимо этого, подрядчик должен будет предложить трассировку сети скоростного рельсового транспорта и с учетом этого оптимизировать маршрутную сеть. Разработка финансовой модели рельсового транспорта также войдет в обязанности подрядчика.
