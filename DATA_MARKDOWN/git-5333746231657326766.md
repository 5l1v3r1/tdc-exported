
# Пожар в пермском химико-технологическом техникуме попал на видео

Published at: **2019-11-06T08:14:00+00:00**

Author: **Ксения Теплякова**

Original: [Накануне.ru](https://www.nakanune.ru/news/2019/11/06/22557409/)

Пожарные устанавливают причину возгорания в Пермском химико-технологическом техникуме по ул. Ласьвинская, передает корреспондент Накануне.RU.
Сообщение о пожаре в учебном заведении в оперативную дежурную смену Центра управления в кризисных ситуациях Пермского края поступило поступило в 17.35. По информации ГУ МЧС по Пермскому краю, сгорел потолочный электросветильник на первом этаже. Задымления в помещениях учебного заведения не было.
