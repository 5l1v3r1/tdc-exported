
# Спецслужбы предотвратили теракт в Москве

Published at: **2019-11-05T08:55:46+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-05/specsluzhby-predotvratili-terakt-v-moskve)

В Москве задержан мужчина, готовивший террористический акт.
По данным РЕН ТВ, 30-летний Талант О. был схвачен еще несколько недель назад. Известно, что мужчина исповедует радикальные формы ислама. Он изучал материалы для изготовления взрывного устройства и активно искал сообщников для совершения теракта. 
В конце минувшей недели в отношении 30-летнего подозреваемого возбуждено уголовное дело по статье 205 УК РФ ("Террористический акт").
