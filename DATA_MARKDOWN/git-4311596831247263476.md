
# Должник из Нижнего Тагила пытался сжечь пристава, облив его бензином

Published at: **2019-11-06T15:16:00+00:00**

Author: ****

Original: [EADaily](https://eadaily.com/ru/news/2019/11/06/dolzhnik-iz-nizhnego-tagila-pytalsya-szhech-pristava-obliv-ego-benzinom)

Житель Нижнего Тагила, накопивший многомиллионный долг, облил одного из приставов бензином и угрожал поджечь жидкость. Попасть в квартиру сотрудникам ФССП удалось при поддержке группы быстрого реагирования. ЧП случилось в доме на улице Дружинина.
Как сообщает пресс-служба Следственного комитета Свердловской области, начата проверка по факту инцидента.
По информации «Российской газеты», ссылающегося на комментарии регионального управления ФССП, в отношении «поджигателя» ещё в 2017 году было возбуждено три исполнительных производства, согласно которым трое проживающих в доме граждан должны освободить жилплощадь. При этом общая сумма долгов женщины разным взыскателям составляет 18 миллионов рублей, мужчины — 7 миллионов.
Ранее выселение было отложено из-за болезни должницы. Должники использовали все возможности для обжалования судебного решения — теперь оно подлежит безусловному исполнению.
Следует отметить, что сейчас решается вопрос о привлечении должника к уголовной ответственности по ст. 318 УК РФ (применение насилия в отношении представителя власти).
