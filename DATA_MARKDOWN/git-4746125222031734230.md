
# 2020年『ななにー元日スペシャル』放送決定 あっという間の1年「充実してる証拠」

Published at: **2019-11-03T23:39:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147978/full/)

　俳優の稲垣吾郎、草なぎ剛、香取慎吾3人によるAbemaTVのレギュラー番組『7.2 新しい別の窓』のお正月特別版『ななにー元日スペシャル』が、来年の1月1日に放送されることが3日、同番組内で発表された。午後3時から放送予定。 　今年に引き続いての元日スペシャルが決定。稲垣が「また元日に皆さんとお会いできますね」と振ると、「年始めにこうやって皆さんにお会いできるのは幸せなことです」と草なぎ。香取は「またいろんな方が来てくれるのかな」と待ち望んだ。さらに稲垣は「なんか（1年経つのが）速いけど…充実してる証拠なのかな」としみじみ語っていた。 　『元日スペシャル』の詳細は追って発表される。　この日の放送では、「ななにースペシャルライブ」で視聴者が「もう一度観たい」と思うライブを集めた特別企画「歌で日本に元気を！」で過去の名場面を振り返った。また、香取が10月1日にサプライズで配信リリースした新曲「10％」を生放送でメディア初披露し盛り上げた。
