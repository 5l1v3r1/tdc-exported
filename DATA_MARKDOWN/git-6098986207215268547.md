
# WhatsApp для Android получил поддержку сканера отпечатков пальцев

Published at: **2019-11-01T07:10:09+00:00**

Author: **Виктор Подволоцкий**

Original: [Лайфхакер](https://lifehacker.ru/whatsapp-android-fingerprint/)

Функция уже доступна всем пользователям. Просто обновите приложение.
В мессенджере WhatsApp для Android наконец появилась поддержка сканера отпечатков пальцев, которая ранее была доступна только в бета-версии приложения. Для включения функция перейдите в Настройки → Аккаунт → Приватность → Блокировка отпечатком пальца.
Для активации функции в системе должен быть зарегистрирован отпечаток хотя бы одного пальца. Если WhatsApp его обнаружит, то позволит вам настроить время блокировки приложения и отображение содержимого.
Функция уже доступна всем пользователям. Если вы не нашли её в настройках, обновите приложение до последней версии через Google Play.
Ранее в этом году в iOS-версии приложения появилась поддержка Face ID или Touch ID.
