
# Грабитель, которого искали по всей России, устроился в Петербурге менеджером по работе с клиентами

Published at: **2019-11-01T11:27:44+00:00**

Author: ****

Original: [ИА «Диалог»](https://topdialog.ru/2019/11/01/grabitel-kotorogo-iskali-po-vsej-rossii-ustroilsja-v-peterburge-menedzherom-po-rabote-s-klientami/)

Полицейские Приморского района задержали 21-летнего молодого человека, находившегося в федеральном розыске. Об этом сообщили в ГУ МВД РФ по Петербургу и Ленобласти.
В Северной столице разыскиваемый занимал должность менеджера по работе с клиентами в одной из фирм. Его удалось задержали 31 октября около дома №2 на улице Аккуратова.
Молодого человека обвиняют в грабеже (часть 2 статья 161 УК РФ). Подозреваемому избрана мера пресечения – арест.
