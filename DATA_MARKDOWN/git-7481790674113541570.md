
# 巨人・原監督、ＦＡ宣言選手「誰かには参戦する」

Published at: **2019-11-02T10:33:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/npb/20191102-OYT1T50245/)

　巨人の原監督は２日、ジャイアンツ球場で報道陣の取材に応じ、「価値ある人に、こちらの話を聞いてもらうのは大事なこと。誰かには参戦すると思う」として、ＦＡ宣言した選手の獲得に乗り出す姿勢を示した。
　巨人では、大塚淳弘・副代表編成担当が先月下旬、美馬と鈴木の獲得検討に向け、調査を進めていることを明らかにしている。原監督は「ＦＡというものにジャイアンツとしても興味はある」などと語った。
