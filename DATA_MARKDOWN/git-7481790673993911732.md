
# パラ世界陸上開幕、佐々木真菜が決勝進出…女子４００

Published at: **2019-11-07T06:57:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/para-sports/news/20191107-OYT1T50204/)

　【ドバイ（アラブ首長国連邦）＝畔川吉永】東京パラリンピックの代表選考を兼ねたパラ陸上の世界選手権が７日、ドバイで開幕した。
　女子４００メートル（視覚障害Ｔ１３）予選で、佐々木真菜（東邦銀行）が全体２位となる５８秒６５で同日夜の決勝に進んだ。佐々木は「走りやすいトラックで、自分より速い選手についていこうと思った。かなり良い方向に来ていると思う」と、東京パラの代表入りに向け、手応えを強調した。
　大会は１５日まで行われ、各種目４位以内の選手が東京パラ代表に内定する。
