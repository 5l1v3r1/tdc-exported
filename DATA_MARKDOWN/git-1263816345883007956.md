
# Теперь мелкие ДТП можно оформлять через мобильное приложение

Published at: **2019-11-01T17:06:00+00:00**

Author: **Инна Лутовинова**

Original: [МОЛНЕТ.RU](https://www.molnet.ru/mos/ru/important/o_717016)

Исполнительный директор РСА Евгений Уфимцев рассказал, что с 1 ноября водители, которые попали в ДТП без пострадавших, могут оформить европротокол через специальное приложение в смартфоне, сообщает ТАСС.
