
# Двое погибли, один пропал без вести в результате взрыва на танкере в Находке

Published at: **2019-11-02T01:57:53+00:00**

Author: **ТАСС**

Original: [Новости Mail.ru](https://news.mail.ru/incident/39333962/)

«На танкере “Залив Америка” в Находке произошел хлопок, два человека погибли, еще одного члена экипажа ищут», — сообщил собеседник агентства.
По его словам, на судне были девять членов экипажа. В район происшествия отправлены два спасательных буксира, готовятся к отправке водолазы для поиска пропавшего моряка.
По предварительной информации, пожара на судне нет.
