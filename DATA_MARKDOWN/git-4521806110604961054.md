
# Ученые нашли способ научить дрозофилу есть ядовитый ваточник

Published at: **2019-11-07T09:40:00+00:00**

Author: ****

Original: [RuNews24](https://runews24.ru/science/07/11/2019/24321894a8c968e655719376883552a8)

Ваточник является ядовитым для большинства насекомых, а также для человека. Его клетки содержат смесь сердечных гликозидов – токсинов, которые угнетают деятельность натрий-калиевой аденозинтрифосфатазы на поверхности клеток.
Некоторые насекомые научились жить на ваточнике, среди них бабочки-монархи. Эксперты из США, Франции и Германии с помощью генетического редактирования смогли подтвердить роль мутаций в развитии устойчивости к гликозидам.
