
# Начальник пожарной части в Хакасии поджег пять домов ради проверки подчиненных

Published at: **2019-11-07T13:46:00+00:00**

Author: **Интерфакс**

Original: [Meduza](https://meduza.io/news/2019/11/07/nachalnik-pozharnoy-chasti-v-hakasii-podzheg-pyat-domov-radi-proverki-podchinennyh)

Начальника пожарной части в Аскизском районе Хакасии арестовали по подозрению в умышленных поджогах, передает «Интерфакс».
По версии следствия, 26-летний пожарный с 30 августа по 28 октября пять раз поджигал жилые дома. Дело расследуют по статье об умышленном повреждении чужого имущества путем поджога. До решения суда увольнять подозреваемого не будут.
Как сообщает РЕН ТВ, речь идет о начальнике пожарной части села Балыкса Иване Шнайдере. Поджигая дома, он, по данным телеканала, хотел проверить «готовность своих подчиненных к нештатным ситуациям».
Ранее Шнайдера привлекали к ответственности за кражу.
