
# RADIO FISHのShow-hey、12月に第1子誕生予定「めちゃくちゃ楽しみ」

Published at: **2019-11-02T10:56:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147918/full/)

　お笑いコンビ・オリエンタルラジオ率いるダンス＆ボーカルユニット・RADIO FISHのダンサー、Show-hey（36）が1日、自身のインスタグラムを更新。12月に第1子が誕生予定であることを報告した。 　インスタでは、赤ちゃんのエコー写真と安産御守を写した1枚を添えて「私事ではございますが、皆様にご報告です。ご存知の方々もいらっしゃるとは思いますが。この度私Show-heyは父になります。出産予定は、来月12月になります」と報告。 　「体調や仕事の事もあり、なかなかお伝えするタイミングがなく、直接お伝え出来なかった方々、申し訳ありません！ これからも変わらずShow-heyとしても祥平としても、精進していきますので、皆さまどうぞ温かい目で近くから、遠くから、中距離から見守っていただければ幸いです。めちゃくちゃ楽しみで仕方ありません」とわが子との対面を待ち望んだ。 　Show-heyは19歳からダンスを習い始め、さまざまなジャンルを習得、HIP-HOPから感情の入るR＆Bまで変幻自在に表現する振り付けや演出に定評がある。14年、15年にアメリカで行われたchoreograph competition大会で2連覇を果たした。RADIO FISHでは振り付け、構成、演出の面で力を発揮している。 　昨年4月、かねてより交際していた一般女性との結婚を発表していた。
