
# В России отзывают 4 144 автомобиля BMW

Published at: **2019-11-01T10:42:00+00:00**

Author: ****

Original: [RuNews24](https://runews24.ru/auto/01/11/2019/ff092e707efdba0a1ed08131db04d91e)

По этой же причине до этого отзывали миллионные партии машин по всему миру. По информации Росстандарта, отзыву подлежат BMW третьей (E46) и пятой (E39) серий, проданные с февраля 2000 по май 2002 года. Дело в возможной неисправности газогенератора подушки безопасности Takata.
При срабатывании подушки водителя в газогенераторе может возникнуть слишком высокое давление. Он может взорваться с выбросом осколков из подушки безопасности.
Всех людей, купивших авто, известит о проводимой кампании «БМВ Русланд Трейдинг». Ремонт будет бесплатным.
