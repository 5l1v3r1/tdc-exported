
# Беглов оценил работу квадрокоптеров по поиску наледи на крышах

Published at: **2019-11-06T16:06:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/740447-beglov-ocenil-rabotu-kvadrokopterov-po-poisku-naledi-na-kryshakh)


        Губернатор Петербурга Александр Беглов приехал в Пушкин для контроля работ по технологии «Холодный чердак». Ему показали работу квадрокоптеров, при помощи которых ведется поиск наледи на крышах.
      
Губернатор Петербурга Александр Беглов сегодня, 6 ноября, приехал в Пушкин для проверки хода работ по технологии «Холодный чердак» в жилых домах.
