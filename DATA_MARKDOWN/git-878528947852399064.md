
# 巨人原監督「近未来のスター候補生」と秋季キャンプ地の宮崎入り

Published at: **2019-11-05T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/05/0012850917.shtml)

　巨人は５日、秋季キャンプ地の宮崎入りし、宮崎ブーケンビリア空港で歓迎セレモニーが行われた。
　原監督を先頭に主砲の岡本、高橋らに加え、直江、横川、與那覇、湯浅、山下航といった若手中心の選手２６人が出席。一般ファンらが多数、詰め掛けた中、原監督は「同じキャンプ地の宮崎、生目の森でキャンプをしているソフトバンクとの日本シリーズで戦い非常に悔しい思いをして２０２０年度に向かってスタートしました。これからジャイアンツで活躍する近未来のスター候補生が、来年度の東京ドーム、１軍で活躍できるよう実りある秋季キャンプにしたい」とあいさつ。飛躍のキャンプにすると誓った。
