
# ม็อบฮ่องกงปะทะตร. เปลี่ยนย่านเซ็นทรัลเป็นสมรภูมิรบ

Published at: **2019-11-02T12:23:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/foreign/739695)


        ตำรวจปราบจลาจลฮ่องกงยิงกระสุนแก๊สน้ำตาเข้าไปในสวนสาธารณะชาเตอร์ การ์เดน ทำให้ผู้ประท้วงหลายพันคน หลบหนีไปยังย่านธุรกิจเซ็นทรัล ตั้งวางสิ่งกีดขวางบนถนน และจุดไฟเผา รวมทั้งเผาสภานีรถไฟใต้ดิน ระหว่างการฝ่าฝืนคำสั่งห้าม ชุมนุมประท้วงสุดสัปดาห์ที่ 22 เมื่อวันเสาร์
      
สำนักข่าวรอยเตอร์และเอเอฟพีรายงานจากเขตบริหารพิเศษฮ่องกงของจีน เมื่อวันที่ 2 พ.ย. ว่า ย่านเซ็นทรัล ซึ่ง 2 ฟากถนนเต็มไปด้วยธนาคาร ร้านจิวเวลรี่หรู และห้างแฟชั่น กลายเป็นสมรภูมิรบขนาดย่อม เมื่อย่างเข้าสู่กลางคืนวันเสาร์  การปะทะกันระหว่างตำรวจที่ใช้กระสุนแก๊สน้ำตา และกระสุนยาง กับกลุ่มผู้ประท้วง ที่สวมใส่เสื้อผ้าและหน้ากากสีดำเป็นสัญลักษณ์ ซึ่งตอบโต้ด้วยระเบิดเพลิง และก้อนหินกลุ่มผู้ประท้วงจุดไฟเผาสิ่งของ บริเวณทางเข้าสถานีรถไฟใต้ดินเซ็นทรัล ฉีดพ่นสีสเปรย์บนผนังกระจก และทุบทำลายสิ่งของบริเวณชานชาลาสถานี ภาพจากจากทีวีท้องถิ่นยังแสดงให้เห็นกระจกหน้าสำนักงานสาขาสำนักข่าวซินหัวของรัฐบาลจีน ถูกผู้ประท้วงทุบตีจนแตก 
 การปะทะกันครั้งล่าสุดเกิดขึ้น หลังจากรัฐบาลจีนประกาศเตือนเมื่อวันศุกร์ ว่า จะไม่อดทนต่อความท้าทายต่อระบบการปกครองฮ่องกง และแผนบรรจุวิชารักชาติในหลักสูตรการศึกษาของฮ่องกง รัฐบาลจีนในกรุงปักกิ่งยังไม่แสดงความพร้อมที่จะตอบสนอง ข้อเรียกร้องหลายข้อของผู้ประท้วงในฮ่องกง ซึ่งรวมถึงการปฏิรูปประชาธิปไตย ปล่อยตัวผู้ประท้วงที่ถูกจับกุมทั้งหมด โดยไม่มีเงื่อนไข และความรับผิดชอบของตำรวจ ในการใช้ความรุนแรงกับกลุ่มผู้ประท้วง และฝ่ายประท้วงก็ไม่มีวี่แววว่าจะละเลิกการเดินขบวนบนท้องถนน.
