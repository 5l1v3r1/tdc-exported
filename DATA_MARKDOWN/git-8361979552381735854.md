
# Один титул до рекорду Шумахера. Визначився чемпіон Формули-1 2019 року

Original: [НВ](https://nv.ua/ukr/sport/other-sport/formula-1-chempion-2019-roku-lyujis-hemilton-gran-pri-ssha-2019-peremozhec-valtteri-bottas-50051381.html)

Британський гонщик посів друге місце за підсумками Гран-прі США, чого виявилося достатньо, щоб його не дістали конкуренти.
Сам етап виграв його партнер по команді Валттері Боттас. Третім до фінішу прийшов пілот Ред Булл Макс Ферстаппен.
