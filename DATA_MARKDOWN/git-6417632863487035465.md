
# Ройс Интерга қарши ўйнамаслиги мумкин

Published at: **2019-11-03T23:23:00+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/roys-interga-qarshi-oynamasligi-mumkin)

Германиялик футболчи бугун жароҳатнинг даражасини аниқлаш бўйича тиббий кўрикдан ўтди. Ройснинг ЕЧЛ доирасидаги "Интер" билан учрашувда майдонга тушиш масаласига тиббий кўрик хулосалари эълон қилингач аниқ бўлади. Бу ҳақида клуб расмий сайти хабар тарқатди.
