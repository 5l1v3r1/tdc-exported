
# Epic Gamesストア無料配布、今週はSFホラー『SOMA』とハロウィンRPG『Costume Quest』。次週は『RUINER』『Nuclear Throne』

Published at: **2019-11-01T09:58:37+00:00**

Author: **Ryuki Ishii**

Original: [AUTOMATON](https://automaton-media.com/articles/newsjp/20191101-105363/)

Epic GamesストアにてPC版『SOMA』『Costume Quest』の期間限定無料配布が開始された。配布期間は11月7日まで。前者は通常価格3080円、後者は通常価格980円のところ、いまなら無料で入手できる（いずれも公式の日本語対応は無し）。
