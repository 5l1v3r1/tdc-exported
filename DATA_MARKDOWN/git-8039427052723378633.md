
# తహశీల్దార్ సజీవదహనంపై టీటీడీపీ సంతాపం

Published at: **2019-11-04T19:42:04+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=946293)

ADVT
రంగారెడ్డి: తహశీల్దార్ విజయారెడ్డి సజీవదహనంపై టీటీడీపీ సంతాపం ప్రటించింది. అబ్దుల్లాపూర్ మెట్ తహశీల్దార్ విజయారెడ్డిపై కిరోసిన్ పోసి సజీవదహనం చేయడంపై తెలంగాణ తెలుగుదేశం పార్టతీ అధ్యక్షుడు ఎల్.రమణ, పొలిట్ బ్యూరో సభ్యుడు రావుల చంద్రశేఖర్ రెడ్డి, అరవింద్ కుమార్ గౌడ్ తదితరులు తీవ్రంగా ఖండించారు. ఇలాంటి చర్యలకు పాల్పడే వారిని ప్రభుత్వం కఠిన చర్యలు తీసుకోవాలని డిమాండ్ చేశారు. విజయారెడ్డి ఆత్మకు శాంతి చేకూరాలని కోరుకుంటూ.. ఆమె కుటుంబ సభ్యులకు తమ ప్రగాడ సానుభూతి తెలిపారు.
