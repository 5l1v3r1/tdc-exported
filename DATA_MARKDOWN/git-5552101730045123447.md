
# 【全文】張書豪認愛妮妮對話洩密　歐陽娜娜偷交陳凱歌兒子曝光

Published at: **2019-11-02T23:28:00+00:00**

Author: **娛樂組, 攝影組**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191029ent011/)

歐陽娜娜去年18歲的生日過後，正式擺脫父親歐陽龍的「禁愛令」，果然生日過後不到半年，她就交到了男朋友。日前張書豪認愛歐陽妮妮時，po出與歐陽娣娣的對話截圖，洩密了娜娜已有男友，而本刊獲得線報，直指歐陽娜娜的男友就是中國大陸導演陳凱歌的兒子陳飛宇，陳飛宇追求她超過兩年，如今總算靠著一首歌虜獲佳人芳心。
