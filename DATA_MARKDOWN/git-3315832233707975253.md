
# Adolescenta din Mileștii-Mici, dispărută acum patru zile, a fost găsită

Published at: **2019-11-01T08:43:59+00:00**

Author: **Jurnal.md**

Original: [Jurnal.md](http://jurnal.md/ro/news/2a229fbefddd8866/adolescenta-din-milestii-mici-disparuta-acum-patru-zile-a-fost-gasita.html?utm_source=RSS&utm_medium=RSS&utm_campaign=RSS)


        Adolescenta de 14 ani din Mileștii-Mici, Ialoveni, care a dispărut fără urmă pe data de 28 octombrie, după ce s-ar fi certat cu mama sa, a fost găsită vie și nevătămată.
      
Copila a fost găsită în jurul orei 9.40, de un polițist, în timp ce aceasta se întorcea acasă din Chișinău, ne-a comunicat ofițerul de presă al IP Ialoveni, Silvia Vetcu.
Adolescenta se simte bine și nu a fost victima niciunui abuz în perioada dispariției.
