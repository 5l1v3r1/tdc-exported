
# Опасная для здоровья сметана продается в Ростовской области

Published at: **2019-11-06T15:33:00+00:00**

Author: **Яна Багдадян**

Original: [Регион Online](https://news-r.ru/news/rostov_oblast/380767/)


        Специалист Россельхознадзора нашли в молочной продукции антибиотики
      
Ростов-на-Дону, 7 ноября. Специалисты Россельхознадзора Ростовской области проверили молочную продукцию на прилавках торговых точек. Были изъяты образцы сметаны. В одном из них было установлено наличие линкомицина, содержание которого не допускается техническим регламентом Таможенного союза «О безопасности пищевой продукции».
- Опасная для здоровья сметана производилась в Ростовской области. Потребление продукции, содержащей остатки антибактериальных препаратов, опасно для здоровья: вызывает аллергические реакции, дисбактериоз, нарушение функции почек, - сообщили «Регион online» в пресс-службе Россельхознадзора Ростовской области.
Сейчас решается вопрос в отношении данной продукции. Проверки будут продолжаться.
Ранее «Регион online» сообщал, что арбитражный суд Ростовской области оштрафовал ростовский колбасный завод «ТАВР». В продукции предприятия обнаружено ДНК лошади.
