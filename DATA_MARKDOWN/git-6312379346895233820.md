
# Медведев: РФ надеется на рост участников банка данных по антитеррору

Published at: **2019-11-04T08:45:40+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-04/medvedev-rf-nadeetsya-na-rost-uchastnikov-banka-dannyh-po-antiterroru)

РФ рассчитывает на рост количества участников Международного банка данных по противодействию терроризму, который был создан Россией, заявил премьер-министр страны Дмитрий Медведев.
Он выразил надежду, что такие российские инициативы последних лет, одобренные в рамках саммита АСЕАН, позволят дать коллективный отпор новым вызовам и угрозам.
По словам премьера, число участников указанного банка данных увеличивается. Сейчас к нему подключены 47 спецслужб из 36 стран.
"Мы исходим из того, что список участников будет расширяться", - сказал глава правительства во время XIV Восточноазиатского саммита.
