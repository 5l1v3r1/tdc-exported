
# మత్తులో చిత్తవుతున్న యువత

Published at: **2019-11-04T04:11:41+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=945723)

Ph: 88865 47666ప్రవేశం ఉచితం ADVT
ఎల్‌బీనగర్‌/హైదరాబాద్‌, నవంబరు 3 (ఆంధ్రజ్యోతి): ‘తెలంగాణ యువతలో మార్పు రావాల్సిన అవసరం ఉంది. మంచిమంచి కుటుంబాల వారు మత్తు ఉచ్చులో చిక్కుకుని చిత్తవుతున్నారు. సమస్యలు వస్తే కుంగిపోకుండా యువత శక్తిగా మారితే దేశానికి గొప్ప భవిష్యత్తు ఉంటుంది’ అని హిమాచల్‌ప్రదేశ్‌ గవర్నర్‌ బండారు దత్తాత్రేయ సూచించారు. హైదరాబాద్‌ కొత్తపేట హుడా కాలనీలో ఆదివారం స్వామి వివేకానంద విగ్రహావిష్కరణ కార్యక్రమానికి ఆయన ముఖ్య అతిథిగా హాజరై మాట్లాడారు. మాజీ ఎంపీ కొండా విశ్వేశ్వర్‌రెడ్డి మాట్లాడుతూ ఒక మతానికో, రాజకీయాలకో సంబంధించినవి కాని వివేకానందుడి ఆలోచనలు యువతకు స్ఫూర్తిదాయకమన్నారు. యువజన సంఘాల సమితి వ్యవస్థాపక అధ్యక్షుడు జిట్టా బాలకృష్ణారెడ్డి ఆధ్వర్యంలో ఈ కార్యక్రమం నిర్వహించారు.
