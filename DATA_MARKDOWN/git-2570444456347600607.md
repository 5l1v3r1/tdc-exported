
# Ребенка, провалившегося под лед, ищут в Татищевском районе

Published at: **2019-11-03T15:40:00+00:00**

Author: ****

Original: [Саратов 24](https://saratov24.tv/news/rebenka-provalivshegosya-pod-led-ishchut-v-tatishchevskom-rayone/)

Одного ребенка смогли вытащить из воды сами жители. Второго - не нашли, и его сейчас ищут спасатели. На месте происшествии работают специалисты областной службы спасения, водолазы. Об этом сообщает ЧП Саратов.
