
# Casetify вместе с BTS выпустят набор аксессуаров для гаджетов

Published at: **2019-11-07T11:54:51+00:00**

Author: **Die-Tote Kamille**

Original: [YesAsia](https://www.yesasia.ru/article/754826)

Совсем скоро BTS совместно с Casetify запустят по всему миру новую необычную коллекцию продукции для гаджетов.

        Смотрите также: Фанат порадовал EXO-L новой фотографией D.O.
      
Для создания нового набора аксессуаров Casetify вдохновились концепцией альбома BTS — «Map of the Soul: Persona» и заглавным треком «Boy With Luv». Продукция, как выяснилось, будет состоять из аксессуаров для iPhone, Galaxy, AirPods, Apple Watch, MacBook и многого другого.
