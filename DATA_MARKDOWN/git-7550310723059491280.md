
# Российский боксер Сергей Ковалев проиграл бой за титул чемпиона WBO

Published at: **2019-11-03T08:15:00+00:00**

Author: ****

Original: [Новая Газета](https://www.novayagazeta.ru/news/2019/11/03/156615-rossiyskiy-bokser-sergey-kovalev-proigral-boy-za-chempionskiy-titul-wbo)

Российский боксер Сергей Ковалев проиграл бой мексиканцу Саулю Альваресу за титул чемпиона мира в полутяжелом весе по версии Всемирной боксерской организации (WBO).
