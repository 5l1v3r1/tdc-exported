
# У Тарашанах виявили підпільний цех з виробництва кованих виробів

Published at: **2019-11-05T13:38:00+00:00**

Author: ****

Original: [Погляд](https://pogliad.ua/news/bukovina/u-tarashanah-viyavili-pidpilniy-ceh-z-virobnictva-kovanih-virobiv-379851)

Комісією з питань легалізації заробітної плати 4.11.2019 р. в с. Тарашани було виявлено цех по виробництву кованих виробів в якому здійснювали діяльність громадянин без державної реєстрації підприємницької діяльності та четверо осіб без укладання трудових договорів. Про це інформує hlyboka.info.
