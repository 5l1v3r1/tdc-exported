
# Нефтекумские школьники станут актерами «Мосфильма»

Published at: **2019-11-06T08:41:00+00:00**

Author: ****

Original: [ГТРК «Ставрополье»](https://stavropolye.tv/news/127351)

Нефтекумские школьники попробуют себя в качестве актеров на «Мосфильме». Воспитанники школы №2 победили в конкурсе «Финансовые истории». Одним из лучших признан их сценарий фильма по мотивам драмы «Идеальные незнакомцы».
Трое учеников и их преподаватель сыграют в кино по своему же сценарию. Это будет один из эпизодов мини-сериала о финансовой грамотности.
Съемки проводятся в рамках одноименного проекта министерства финансов.
