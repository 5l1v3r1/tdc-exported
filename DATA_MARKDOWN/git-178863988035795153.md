
# Температурные качели продолжаются - какой погода сегодня будет

Published at: **2019-11-03T08:11:00+00:00**

Author: **Владимир Андреев**

Original: [РИА Мелитополь](https://ria-m.tv/news/170382/temperaturnyie_kacheli_prodoljayutsya_kakoy_pogoda_segodnya_budet.html)


        Сегодня температура воздуха в Мелитополе снова начинает повышаться. Уже завтра днем столбики термометра достигнут отметки в +17 градусов.
      
Сегодня будет только немногим теплее, чем вчера. Очередное же бабье лето придет в регион уже завтра. Итак, нас ждет переменная облачность, без осадков, ветер юго-восточный 7-12 м / с, температура воздуха днем 6-11 ° тепла.
