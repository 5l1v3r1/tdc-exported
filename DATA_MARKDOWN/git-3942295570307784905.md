
# WhatsApp в Android скоро можно будет разблокировать с помощью отпечатка

Published at: **2019-11-01T08:54:00+00:00**

Author: **Олег Иванов**

Original: [Anti-Malware](https://www.anti-malware.ru/news/2019-11-01-1447/31189)

Разработчики WhatsApp обещают, что мессенджер скоро будет поддерживать биометрическую аутентификацию в смартфонах, работающих на базе операционной системы Android. Единственное условие — ваше устройство должно быть оснащено сканером отпечатка пальца, тогда вы сможете защитить свой аккаунт.
Найти новую опцию можно будет здесь: «Настройки -> Учётная запись -> Конфиденциальность -> Разблокировка по отпечатку пальца». Если вы активируете эту функцию, вам придётся прикладывать палец каждый раз при запуске приложения.
Помимо этого, пользователи смогут выбрать промежуток времени, в течение которого WhatsApp будет оставаться разблокированным. Это избавит от постоянных запросов от системы биометрической аутентификации.
Стоит обратить внимание на один небольшой нюанс — звонки не будут блокироваться с помощью этого метода.
Никаких точных сроков, к сожалению, разработчики WhatsApp не называют. Единственное, что известно на данный момент — разблокировка по отпечатку в Android-устройствах появится «скоро».
Напомним, что аналог этой функции для iPhone в WhatsApp реализовали ранее в этом году.
