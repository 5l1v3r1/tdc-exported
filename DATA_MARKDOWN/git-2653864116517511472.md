
# Оформить ДТП вологжане теперь могут с помощью мобильного приложения

Published at: **2019-11-07T06:49:00+00:00**

Author: ****

Original: [35медиа](https://35media.ru/news/2019/11/07/oformit-dtp-vologzhane-teper-mogut-s-pomoshhyu-mobilnogo-prilozheniya)

Оформить ДТП вологжане теперь могут с помощью мобильного приложения. В России заработал сервис «Европротокол онлайн».
Участники мелких аварий могут самостоятельно оформить ДТП без сотрудников ГИБДД с помощью телефона или планшета. На них необходимо установить приложение «Помощник ОСАГО». Условий, при которых возможно воспользоваться сервисом, несколько. В их числе — в ДТП пострадали два автомобиля; оба участника аварии застрахованы по ОСАГО; не причинен ущерб третьим лицам и нет пострадавших; собственниками машин должны быть физлица, имеющие учетные записи в сервисе госуслуг и один из них признал свою вину. Для оформления аварии участникам нужно авторизоваться в приложении и ввести серию и номер полиса, сведения о поврежденных элементах автомобиля. После этого пользователю предложат варианты оформления ДТП с фотофиксацией или без.
