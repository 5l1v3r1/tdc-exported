
# Магнитка командировала в Зауралье Вадима Антипина

Published at: **2019-11-05T17:21:00+00:00**

Author: ****

Original: [Allhockey.ru](https://allhockey.ru/news/show/369121-Magnitka_komandirovala_v_Zaural_e_Vadima_Antipina)

Защитник Вадим Антипин командирован в "Зауралье" из системы магнитогорского "Металлурга".В составе "Зауралья" Вадим будет выходить на лед под номером 79.На данный момент в заявке курганской команды находятся девять игроков из системы "Металлурга": Глеб Носов, Глеб Бабинцев, Дмитрий Зайцев, Савелий Ольшанский, Марк Янчевский, Вадим Антипин, Константин Дубин, Данил Пятин и Юрий Платонов.
