
# ‘아침마당’ 오재미 “못생긴 외모로 인기였는데 옥동자가 밥줄 끊어”

Published at: **2019-11-04T01:39:35+00:00**

Author: **스포츠조선=박아람 기자**

Original: [Chosun](https://news.chosun.com/site/data/html_dir/2019/11/04/2019110400867.html)

입력 2019.11.04 10:39
