
# ДУМК в Симферополе открывает кружок рукоделия

Published at: **2019-11-07T12:08:00+00:00**

Author: ****

Original: [Крымское информационное агентство](https://kianews24.ru/news/dumk-v-simferopole-otkrivaet-kruzhok-r/)

В Симферопольском микрорайоне Фонтаны открывается кружок рукоделия для девочек. В кружок принимаются девочки в возрасте 10-15 лет. Об этом сообщили в пресс-службе ЦРО ДУМК.
В программе кружка: — рисование акриловыми красками; — занятие квиллингом (изготовление фигурок из скрученной в спиральки полосок бумаги); — бисероплетение (броши, деревья, цветы); — изготовление цветов из лент; — изготовление декоративных конвертов; — работа с фетром; Занятия будут проходить один раз в неделю.
Оплата только за материалы, которые будут использованы на уроках.
