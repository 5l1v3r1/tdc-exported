
# Mozilla запретит всплывающие запросы об уведомлениях от сайтов в Firefox

Published at: **2019-11-06T13:08:48+00:00**

Author: **Владислав Войтенко**

Original: [Код Дурова](https://kod.ru/mozilla-firefox-restricting-notification-nov-2019/)

Firefox станет первым десктопным браузером, который по умолчанию запретит сайтам вызывать всплывающие запросы на разрешение присылать веб-пуши. Об этом рассказывается в блоге Mozilla.
С января следующего года браузер Mozilla Firefox перестанет показывать уведомления о разрешении на получение веб-пушей от сайтов. Окно подписки на уведомления с 72 версии браузера пользователям придётся активировать самостоятельно. Для этого необходимо будет нажать на соответствующую кнопку рядом с URL-строкой и согласиться на получение уведомлений от открытого сайта:
Решение отказаться от таких всплывающих запросов в Mozilla приняли после получения результатов исследования, которое показало, что 99% пользователей Firefox не принимают такие запросы, а 48% отказываются от получения веб-пушей. Повторная просьба подписаться на уведомления сайта достаточно редко вынуждает пользователей изменить своё решение.
Борьбу с назойливыми всплывающими запросами в Mozilla начали, не дожидаясь полноценного релиза январского обновления. Начиная с 70 версии в браузере при запросе о веб-пушах ответ «Не сейчас» был изменён на «Никогда не разрешать». При выборе этого варианта сайт больше не сможет вызвать всплывающий запрос о подписке на уведомления повторно.
