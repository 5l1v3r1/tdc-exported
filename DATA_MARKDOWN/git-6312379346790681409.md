
# На укравших Дзюбу девушек завели уголовное дело

Published at: **2019-11-05T17:00:41+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-05/na-ukravshih-dzyubu-devushek-zaveli-ugolovnoe-delo)

Против девушек из Подольска, которые украли металлическую фигуру футболиста Артема Дзюбы, возбуждено уголовное дело по статье "кража".
Напомним, в конце октября девушки забрали себе фигуру, которую установили жители одного из микрорайонов за 27 тысяч рублей.
Позже девушки записали видео, где объяснили, что они "просто хотели похулиганить" и попросили помощи у футболиста.
