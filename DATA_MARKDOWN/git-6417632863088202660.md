
# Григорян Арманистон термасини бошқариши мумкин

Published at: **2019-11-04T16:42:00+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/grigoryan-armaniston-termasini-boshqarishi-mumkin)

Манбанинг хабарига кўра, сўнгги ҳафтани 53 ёшли мутахассис Ереванда Арманистон футболи мутасаддилари билан ўтказмоқда. Унинг олдида икки вариант бор. Бири - "Пюник".
Иккинчиси эса Армен Гюльбудагянцнинг истеъфосидан кейин мураббийсиз қолиб кетган Арманистон термаси бўлиб турибди.
Қайд этиш керак, Григорян 2015 йилнинг кичик даври мобайнида "Пахтакор"ни бошқарганди.
