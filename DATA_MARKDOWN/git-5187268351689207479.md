
# NethServer 7.7: Esqueça o Windows Server, esta distro tem tudo

Published at: **2019-11-06T23:02:09+00:00**

Author: **Pedro Pinto**

Original: [Pplware](https://pplware.sapo.pt/linux/nethserver-7-7-esqueca-o-windows-server-esta-distro-tem-tudo/)

Ao contrário do que acontecia há uns anos, as empresas têm vindo a adotar cada vez mais sistemas baseados em Linux. Até a própria Microsoft, com o serviço de Cloud Azure é fã do Linux.
Como sabemos, o GNU/Linux é extremamente flexível e muito usado para a implementação dos mais diversos serviços de rede… não esquecendo o preço. Para quem procura uma distribuição com vários serviços de rede prontos a funcionar tem de experimentar a distro NethServer.
Chama-se NethServer e é uma distribuição baseada no fantástico CentOS. Esta distribuição Linux é extremamente modular, uma vez que disponibiliza os mais diversos serviços de rede. MailServer, WebServer, Groupware, Firewall, Web Filter, IPS/IDS, VPN, DNS, DHCP são alguns deles… que precisa mais?
sta nova versão do NethServer pode substituir por completo uma máquina com Windows Server que esteja a funcionar como controlador de domínio.
Para simplificar toda a configuração e administração de serviços, o NethServer oferece uma interface web muito bem organizada e fluída, facilitando todo o trabalho do administrador. De referir que esta distribuição não traz, por omissão, ambiente gráfico de trabalho.
Se procuram uma distro Linux com vários serviços de rede prontos a funcionar, testem a distro NethServer. Num próximo tutorial iremos mostrar como podem experimentar esta distribuição. Estejam atentos ao Pplware.
