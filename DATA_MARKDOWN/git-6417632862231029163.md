
# Аарон Рэмзи Ювентуснинг ЕЧЛдаги юбилей голига муаллифлик қилди

Published at: **2019-11-06T19:39:27+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/aaron-remzi-yuventusning-echldagi-yubiley-goliga-mualliflik-qildi)

Гап шундаки, Рэмзининг "Локомотив" дарвозасига урган голи туринликларнинг мусобақадаги 300-голи сифатида тарихга кирди. Эслатиб ўтамиз, "кекса синьора"нинг ЕЧЛдаги 100-голига Давид Трезеге, 200-сига Себастьян Жовинко муаллифлик қилганди.
