
# Появились фото с места убийства бывшего главы Киселёвска

Published at: **2019-11-01T05:53:31+00:00**

Author: **Владимир Огурцов**

Original: [Сiбдепо](https://sibdepo.ru/news/poyavilis-foto-s-mesta-ubijstva-byvshego-glavy-kiselyovska.html)

В социальных сетях появились фотографии с места убийства бывшего мэра Киселёвска Сергея Лаврентьева. На них запечатлены тела бывшего градоначальника, а также одного из нападавших, которого, по предварительным данным, застрелил чиновник.
Напомним, громкое убийство произошло поздно вечером 31 октября в загородном доме Лаврентьева в посёлке Верх-Чумыш. По версии следствия, вечером супруга 59-летнего экс-градоначальника заметила свет фонарика и силуэты двух людей на участке. Сергей Лаврентьев вышел из дома с охотничьим ружьём. Через несколько минут после этого женщина услышала несколько выстрелов, а потом обнаружила тело мужа и неизвестного мужчины.
По данному факту возбуждено уголовное дело. Следователи рассматривают все возможные версии произошедшего, одна из которых – попытка разбойного нападения.
Сейчас на месте происшествия работает следственно-оперативная группа, ведутся поиски второго нападавшего.
