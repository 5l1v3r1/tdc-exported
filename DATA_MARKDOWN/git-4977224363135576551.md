
# Policija preklicala iskanje Domna Gradišnika

Published at: **2019-11-05T06:19:45+00:00**

Author: ****

Original: [STA.si](https://www.sta.si/2694071/policija-preklicala-iskanje-domna-gradisnika)


        Ljubljana, 5. novembra - Policija preklicuje iskanje 14-letnega Domna Gradišnika z območja policijske postaje Šentjur. Pogrešanega mladoletnika so namreč s pomočjo informacij občana našli v Šentjurju, so sporočili s Policijske uprave Celje.
