
# Ротенберг продал «Стройгазмонтаж» за 75 миллиардов рублей. Эта компания строила Керченский мост

Published at: **2019-11-07T17:07:38+00:00**

Author: **Интерфакс**

Original: [Meduza](https://meduza.io/news/2019/11/07/rotenberg-prodal-stroygazmontazh-za-75-milliardov-rubley-eta-kompaniya-stroila-kerchenskiy-most)

Бизнесмен Аркадий Ротенберг закрыл сделку по продаже компании «Стройгазмонтаж».
Представитель Ротенберга официально не назвал сумму сделки и не уточнил покупателя. По данным РБК, бизнесмен мог получить 75 миллиардов рублей, а покупателем стала одна из «дочек» «Газпрома».
