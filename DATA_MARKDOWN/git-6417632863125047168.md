
# Ўзбекистон терма жамоаси йиғинни бошлади

Published at: **2019-11-02T11:07:37+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/ozbekiston-terma-jamoasi-yiginni-boshladi-2)

Вадим Абрамов мазкур йиғинга 32 нафар футболчини чақирган. Дастлабки кунги машғулотларда 27 нафар ўйинчи шуғулланди. 
Легионерлардан фақат Одил Аҳмедов ва Санжар Шоаҳмедов терма жамоа ихтиёрига етиб келган холос. Рустам Ашурматов, Отабек Шукуров, Икром Алибоев, Элдор Шомуродов ва Темурхўжа Абдухолиқов ҳали Тошкентга етиб келишмаган. 
Музаффар Музаффаров жароҳати боис терма жамоа йиғиниша ташриф буюрмади.

        Бу йиғинга чақирилган футболчилар рўйхати: 
      
Дарвозабонлар: Санжар Қувватов (“Пахтакор”), Элдорбек Суюнов (“Пахтакор”), Абдумавлон Абдужалилов (“Бунёдкор”).
Ҳимоячилар: Егор Кримец (“Пахтакор”), Рустам Ашурматов (“Гвангжу”, Жанубий Корея), Ислом Тўхтахўжаев (“Локомотив”), Олег Зотеев (“Локомотив”), Фаррух Сайфиев (“Пахтакор”), Хожиакбар Алижонов (“Пахтакор”), Санжар Қодирқулов ("Бунёдкор"), Акбар Исматуллаев ("Металлург"), Мурод Холмуҳаммедов ("Қўқон-1912").
Ярим ҳимоячилар: Одил Аҳмедов (“Шанхай СИПГ”, Хитой), Отабек Шукуров (“Шаржа”, БАА), Икром Алибоев (“Сеул ФК”, Жанубий Корея), Санжар Шоаҳмедов (“Теренггану”, Малайзия), Азиз Ғаниев (“Насаф”), Одилжон Ҳамробеков (“Пахтакор”), Жалолиддин Машарипов (“Пахтакор”), Лутфулла Тўраев ("Бунёдкор"), Достон Ҳамдамов (“Пахтакор”), Аброр Исмоилов ("Сурхон"), Хуршид Ғиёсов ("Бунёдкор"), Жамшид Искандеров ("Локомотив"), Сардор Мирзаев ("Локомотив"), Сардор Собирхўжаев ("Пахтакор"), Азиз Турғунбоев ("Навбаҳор").
Ҳужумчилар: Элдор Шомуродов (“Ростов”, Россия), Игорь Сергеев (“Пахтакор”), Темурхўжа Абдухолиқов (“Хорфаккан”, БАА), Ҳумоюн Муртозаев ("Насаф").
