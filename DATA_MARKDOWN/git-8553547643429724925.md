
# Synology выпустила компактный NAS с одним отсеком

Published at: **2019-11-04T09:25:00+00:00**

Author: ****

Original: [Компьютерное Обозрение](https://ko.com.ua/synology_vypustila_kompaktnyj_nas_s_odnim_otsekom_130723)

Компания Synology выпустила компактный NAS с одним отсеком, предназначенный для хранения и защиты личных данных.
«DS120j легко настраивается, позволяя пользователям создавать резервные копии фотографий и видео на мобильных устройствах, — заявил Майкл Ван (Michael Wang), менеджер по продукции компании Synology. — DS120j объединяет множество универсальных функций Synology — и все это по доступной цене, что делает эту систему очевидным выбором по сравнению с внешними устройствами хранения без возможности управления».
Как отмечается, с помощью Synology Drive можно легко создавать резервные копии файлов на компьютерах и синхронизировать их на разных устройствах. Служба Synology QuickConnect обеспечивает безопасный доступ к файлам в любое время и в любом месте без необходимости выполнения сложной настройки сети.
Ключевые функции:
DiskStation Manager DS120j работает на базе DiskStation Manager, платформы для управления данными.
