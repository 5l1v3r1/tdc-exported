
# С 1 ноября появятся новые изменения для автомобилистов

Published at: **2019-11-01T00:00:00+00:00**

Author: **Ирина Котова**

Original: [CAR.RU](https://car.ru/news/autogramota/56506-s-1-noyabrya-poyavyatsya-novyie-izmeneniya-dlya-avtomobilistov/)


        С 1 ноября текущего года для российских автомобилистов произойдет ряд изменений.
      
Так, для машин, ввозимых из-за заграницы, новый электронный паспорт технических средств можно будет получить только в специализированных лабораториях. При этом, владельцам бумажных документов менять их понадобиться только в случае если паспорт будет утерян или при повторной продаже/покупке автомобиля.
Кроме того, все автобусы, грузовые машины и другие маршрутные транспортные средства должны быть оборудованы тахографами. В случае если транспортное средство не оборудовано тахографом водитель и организация, в которой он работает получать штраф. Причем при повторном выявлении отсутствия тахографа, сумма штрафа увеличится в несколько раз.
 
По словам представителей Минтранса России, все вносимые изменения необходимы для того, чтобы сделать эксплуатацию машин более удобной. Кроме того, введение тахографа должно отразиться на снижение количества возникающих дорожно-транспортных происшествий с участием этого транспорта.
