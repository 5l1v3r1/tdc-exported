
# В Одессе пока будет тепло

Published at: **2019-11-03T19:11:00+00:00**

Author: ****

Original: [ТАЙМЕР](http://timer-odessa.net/news/v_odesse_poka_budet_teplo_234.html)


        В ближайшие дни в Одессе сохранится преимущественно тёплая, хотя и пасмурная погода.
      
Поддержанию благоприятных температурных условий будет способствовать южный ветер.Самым тёплым днём будет вторник, 5 ноября: в этот день температура воздуха может достигнуть 19 градусов выше нуля. Также утром во вторник возможен небольшой дождь.В остальные дни начавшейся недели ожидается переменная облачность без осадков, температура воздуха днём будет составлять 15-17, ночью — 13-14 градусов выше нуля.
