
# 西武・源田と結婚の元乃木坂・衛藤　独身最後の撮り下ろし　純白ドレス姿も

Published at: **2019-11-07T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/07/0012855066.shtml)

　西武の源田壮亮内野手（２６）と１０月に結婚した元乃木坂４６・衛藤美彩（２６）が、来年１月８日にフォトブック（タイトル未定）を発売することが６日、分かった。北欧・アイスランドで独身最後の撮り下ろしを敢行。セクシーショットから純白ドレス姿まで披露している。
　乃木坂時代や恋愛観に触れ、地元・大分では母親と対談した衛藤は「１９年は乃木坂４６を卒業し、プライベートでも激動の１年でした。そんな年だったからこそ、なかなか行けない、大地のエネルギーを感じられるアイスランドを撮影場所に選びました」とＰＲした。
