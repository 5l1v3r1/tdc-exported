
# Разбил боковые зеркала четырех авто пьяный житель Нур-Султана

Published at: **2019-11-05T12:33:00+00:00**

Author: **ИА Тотал Казахстан**

Original: [Тотал](https://total.kz/ru/news/proisshestviya/razbil_bokovie_zerkala_chetireh_avto_pyanii_zhitel_nursultana_date_2019_11_05_12_40_54)


        Разбушевавшийся 32-летний мужчина повредил автомобили во дворе одного из жилых комплексов столицы.
      
Сотрудниками управления полиции столичного района Есиль задержан мужчина, который во дворе жилого комплекса повредил несколько автомашин, сообщает Total.kz.
