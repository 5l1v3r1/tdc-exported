
# Названо головне словосполучення 2019 року

Published at: **2019-11-07T12:41:31+00:00**

Author: ****

Original: [ФАКТИ](https://fakty.com.ua/ua/lifestyle/20191107-nazvano-golovne-slovospoluchennya-2019-roku/)

У Великій Британії назвали словосполучення 2019 року.
Словосполученням 2019 року став вислів “climate strike”, що перекладається, як кліматичний страйк.
Головне словосполучення року було названо Тлумачним словником англійської мови Коллінза.

Зазначається, що вперше цей вислів використовувався для назви протестних акцій, приурочених до Конференції ООН зі зміни клімату у Парижі у 2015 році.
Цей рух став особливо популярним у 2019 через діяльність шведської школярки Грети Тунберг.
Нагадаємо, що у 2018 році головним словосполученням став вислів “fake news” – фейкові новини.
