
# Видео с места убийства экс-мэра Киселевска

Published at: **2019-11-01T03:51:56+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-01/video-s-mesta-ubiystva-eks-mera-kiselevska)

СК предоставил видео с места убийства бывшего мэра Киселевска Сергея Лаврентьева.
На кадрах видно, что на месте происшествия работает полиция.
Ранее стали известны подробности убийства Лаврентьева, которого застрелили в собственном доме.
Причины произошедшего остаются неизвестными. По данному факту возбудили уголовное дело.
