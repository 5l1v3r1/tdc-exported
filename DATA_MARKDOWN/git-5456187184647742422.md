
# Кандидат-фермер из Гомеля: Дайте частную собственность на землю

Published at: **2019-11-03T13:49:00+00:00**

Author: ****

Original: [charter97.org](https://charter97.org/ru/news/2019/11/3/354244/)

Трактор со звукоусиливающей аппаратурой, козы, бараны и коровы - так проходят агитационные пикеты кандидата в «палату представителей» по Железнодорожному избирательному округу Гомеля фермера из деревни Поколюбичи Юрия Ганцевича.
Он выступает за частную собственность на землю, передачу убыточных колхозов частникам, переориентацию сельских хозяйств на высокотоварным производство, за создание благоприятных условий для желающих жить и работать на селе, говорит о необходимости содействия именно частным хозяйствам.
На пикете у ДК «Випра» побывала корреспондент сайта «Гомельская весна» Лариса Щирякова.
