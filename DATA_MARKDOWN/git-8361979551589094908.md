
# С начала года в Украине закрылись 304 отделения банков

Published at: **2019-11-04T16:48:00+00:00**

Author: ****

Original: [НВ](https://nv.ua/biz/finance/otdeleniya-bankov-v-ukraine-zakrylis-304-otdeleniya-bankov-novosti-ukrainy-50051562.html)

За январь-сентябрь 2019 года количество отделений банков в Украине сократилось на 304 — с 8509 отделений до 8205. Об этом свидетельствует статистика НБУ.
Лидерами по сокращению количества отделений с начала года стали:
— Ощадбанк, который закрыл 159 отделений;
— Проминвестбанк — 49 отделений;
— ПриватБанк — 47;
За этот же период:
— ПУМБ открыл 20 отделений;
— Акордбанк и Кристалбанк открыли по 5 отделений;
— Укрбудинвестбанк — 4.
Лидерами по количеству отделений являются:
— Ощадбанк- 2471 отделение;
— ПриватБанк — 1974 отделения;
— Райффайзен Банк Аваль — 496.
Ранее сообщалось, что государственные ПриватБанк и Ощадбанк с начала 2019 года в разы увеличили чистую прибыль.
