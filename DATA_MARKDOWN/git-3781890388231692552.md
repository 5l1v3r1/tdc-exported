
# เรื่องน่ารู้ : สายหยุด

Published at: **2019-11-06T22:15:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/agriculture/740342)


        ชื่อเรียกอื่น ๆ กล้วยเครือ เครือเขา แกลบ สาวหยุด เสลาเพชร ไม้พุ่มรอเลื้อย กิ่งอ่อนมีขนสีน้ำตาล ปกคลุม ใบ เดี่ยว เรียงสลับ รูปรี ดอก ออกที่ซอกใบ สีเหลืองมีกลิ่นหอม ประโยชน์ หัวเป็นยาระบายปัสสาวะ ขับลมห้ามเลือด
      
   ชื่อเรียกอื่น ๆ กล้วยเครือ เครือเขา แกลบ สาวหยุด เสลาเพชร ไม้พุ่มรอเลื้อย กิ่งอ่อนมีขนสีน้ำตาล ปกคลุม ใบ เดี่ยว เรียงสลับ รูปรี กว้าง 3-6.5ซม. ยาว 6-14 ซม.ดอก ออกที่ซอกใบ สีเหลืองมีกลิ่นหอม กลีบดอก 6 กลีบ มี 2 วงกลีบชั้นในเล็กกว่าชั้นนอก ผล เป็นผลกลุ่ม ผลย่อยยาว 2-4 ซม. มี 2-5 เมล็ด มีรอยคอดระหว่างเมล็ด เมื่อสุกสีแดง   การใช้ประโยชน์ หัวเป็นยาระบายปัสสาวะ ขับลมห้ามเลือด   ที่มา ฐานข้อมูลพรรณไม้ องค์การสวนพฤกษศาสตร์ กระทรวงทรัพยากรธรรมชาติและสิ่งแวดล้อม.
