
# ఢిల్లీ: ఈ నెల 5 వరకు స్కూళ్లకు సెలవులు

Published at: **2019-11-01T15:24:42+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=943692)

Ph: 88865 47666ప్రవేశం ఉచితం ADVT
న్యూఢిల్లీ: ఈ నెల 5 వరకు దేశ రాజధానిలోని పాఠశాలన్నిటికీ ఢిల్లీ ప్రభుత్వం సెలవులు ప్రకటించింది. వాయుకాలుష్యం ప్రమాదకర స్థాయికి చేరుకున్న నేపథ్యంలో ఈ నిర్ణయం తీసుకున్నట్టు ఢిల్లీ ముఖ్యమంత్రి అరవింద్ కేజ్రవాల్ ట్విటర్లో పేర్కొన్నారు. దట్టమైన మంచు దుప్పటి ఢిల్లీని కప్పేయడంతో పాటు సరాసరి వాయు కాలుష్య సూచీ ‘‘అత్యంత ప్రమాదకర’’, ‘‘అత్యవసర’’ కేటగిరీలోకి చేరింది. వరుసగా ఐదో రోజు కాలుష్యం ప్రమాదకర స్థాయిలో కొనసాగుతుండడంతో పర్యావరణ కాలుష్య నియంత్రణ మండలి (ఈపీసీఏ) దేశ రాజధాని ప్రాంతంలో హెల్త్ ఎమర్జెన్సీ ప్రకటించింది. ఇవాళ ఉదయం ఢిల్లీలోని మొత్తం 37 వాయు నాణ్యత పర్యవేక్షక కేంద్రాల్లోనూ వాయు కాలుష్యం ప్రమాదకర కేటగిరీలో ఉన్నట్టు నమోదు కావడం గమనార్హం.
