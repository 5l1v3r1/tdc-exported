
# ５時夢タレント　実家が火事「去年買ったばかり…」　行方不明の愛猫は戻る

Published at: **2019-11-06T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/06/0012853573.shtml)

　ロシア生まれのタレント・小原ブラス（２７）が６日、水曜日の黒船特派員としてレギュラー出演するＴＯＫＹＯ　ＭＸ「５時に夢中！」に出演。実家が火事で燃えてしまったことを明かした。
　ブラスは今月３日、ツイッターに家が焼けてしまった写真を投稿。「私事ですが．．．本日私の実家が燃えました。去年買ったばかりの実家です。ショックすぎて言葉も出ない．．．」と明かし、「飼っていた猫が行方不明ですが、家族は無事です。乾燥している季節なので皆さん気をつけてください」と注意を呼びかけていた。
　３日の火事以降、初めての５時夢出演となり、番組内で、「電気系統が原因だったんですけども、実家の方、火事になっちゃいまして。３部屋ぐらい焼けちゃって。一応、住める状態ではあるんです、リフォームが必要になったけど、全焼ではないので」と説明。火事直後には驚いた愛猫が行方不明になっていたが「戻ってきて、無事なので」と報告。「お騒がせして申し訳ないです。いろんなメッセージいっぱいもらったり、本当にありがとうございました」と激励のメッセージに感謝した。
　ツイッターには、焼け焦げた室内、はがれ落ちた壁の写真などを投稿していた。
　小原はロシア生まれ、兵庫県姫路市育ち。
