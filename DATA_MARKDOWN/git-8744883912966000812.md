
# Боттас из Mercedes выиграл квалификацию Гран-при США «Формулы-1»

Published at: **2019-11-02T22:38:22+00:00**

Author: ****

Original: [Коммерсантъ](https://www.kommersant.ru/doc/4148113)

Финский пилот Валттери Боттас из команды Mercedes показал лучшее время в квалификации 19-го этапа чемпионата «Формулы-1» — Гран-при США. Он показал время 1 минуту 32,029 секунды.
Вторым стал немецкий гонщик Ferrari Себастьян Феттель (+0,012). Третье место занял голландец Макс Ферстаппен из Red Bull (+0,067). Британец Льюис Хэмилтон из Mercedes, от которого ждут шестой победы, будет стартовать с пятого места (+0,292).
Российский пилот Toro Rosso Даниил Квят стал 13-м (+1,960). Гран-при США пройдет в воскресенье, 3 ноября.
