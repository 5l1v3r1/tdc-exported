
# ルンバ4種がガシャポンになって登場！ 「ガシャポン iRobot Roomba」発売

Published at: **2019-11-06T15:00:00+00:00**

Author: ****

Original: [アキバ総研](https://akiba-souken.com/article/42258/)

アイロボットジャパン合同会社は、株式会社バンダイより発売されているカプセル玩具「ガシャポン」にて、ロボット掃除機「ルンバ」シリーズを1/7スケールのミニチュア玩具にした「ガシャポン iRobot Roomba」を2019年11月第2週より発売する。
ロボット掃除機「ルンバ」を全長約50mmのサイズで再現したミニチュア玩具。本体をうしろに引っ張って手を放すと動き出す「プルバック方式」を採用した。走行のみならず方向転換もするので、実際にルンバが掃除をしているかのように見えるという。
2002年に発売された初代「Roomba Original」をはじめ、2004年発売の自動充電機能を備えた「Roomba Discovery」や、2007年に発売され、白いデザインがブレイクしヒットした「Roomba 537J」、そして2015年に発売されたハイエンドモデルの「Roomba 980」など、歴代ルンバの中から人気の高かった4種類がラインナップされている。
【商品概要】■ガシャポン iRobot Roomba・メーカー希望小売価格：1回300円（税込）・発売日：2019年11月2週目より順次発売予定・対象年齢：15才以上・主な素材：本体・前輪：ABS/後輪：合成ゴム・POM・サイズ：全長約50mm・ラインアップ：Roomba Original/Roomba Discovery/Roomba 537J/Roomba 980・製品販売元：株式会社バンダイ・主な販売ルート：全国の玩具売場・量販店などに設置されたカプセル自販機
iRobot、iRobotロゴ、アイロボット、Roomba、ルンバは、アイロボットの登録商標または商標です。ガシャポンは、バンダイの登録商標です。
