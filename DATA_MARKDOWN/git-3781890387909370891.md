
# พยากรณ์ดวงชะตา12ราศี ประจำเดือนพ.ย.62

Published at: **2019-11-05T03:00:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/article/739971)


        สัปดาห์นี้ติดตามพยากรณ์ดวงชะตาทั้ง 12 ราศี ประจำเดือน พ.ย. 62 โดย อ.พราหมณ์เมศ ที่จะมาวิเคราะห์ปรากฏการณ์ บนฟากฟ้า ดาวพฤหัสบดีได้โคจรโยกย้ายราศีจากเดิมเคลื่อนตัวเข้าสู่ราศีธนู จะมีผลต่อดวงชะตาในทุกๆ ชาวราศีอย่างไร
      
เนื่องจากตั้งแต่เริ่มต้นของการก้าวเข้าสู่เดือนพฤศจิกายนนี้ ในปรากฎการณ์บนฟากฟ้า ดาวพฤหัสบดีได้โคจรโยกย้ายราศีจากเดิมที่เคยตั้งอยู่ในราศีพิจิก ก็ได้เคลื่อนตัวเข้าสู่ราศีธนูเป็นที่เรียบร้อยแล้ว แต่การเคลื่อนเข้ามายังราศีธนู อันนับว่าเป็นเรือนราศีของดาวพฤหัสบดีเองนั้น กลับต้องมาร่วมราศีกับดาวเสาร์ที่ได้โคจรเข้ามาครองในเรือนราศีธนูอยู่ก่อนหน้าแล้วดาวพฤหัสฯ จรย้ายเข้าราศีธนูร่วมกับดาวเสาร์จร ที่ครองราศีอยู่ก่อนหน้านี้ เป็นการที่ดาวคู่นี้จรเข้าทับดาวคู่นี้ที่ตั้งอยู่ในดวงเมืองอีกด้วย ( ๕๗ ร่วมกันในราศีธนู ) และเราควรรู้ด้วยว่าในทุก 20 ปี ดาวคู่นี้จะโคจรมาร่วมกัน ให้ตรวจสอบย้อนหลังได้ ในปี พศ. 2503 ต่อ 2504 พศ. 2523 ต่อ 2524 และ ใกล้สุดในรอบคือ พศ. 2563 ต่อ 2564แต่ครั้งนี้ ดาวพฤหัสฯจรร่วมดาวเสาร์ก่อนกาลนิดหน่อย เวลากำหนดการจริงจะอยู่ที่ ราศีมังกร ในปี 2563 ( คลอเคลียข้ามราศีกันไป ) ดาวคู่นี้ เมื่อจรร่วมกัน จะเป็นดาวเปลี่ยนยุค คือ คนในสังคม คนในโลกเราจะมีความเชื่อที่เปลี่ยนไปจากก่อนหน้าหากใครที่เรียนโหรฯมากับครูโหรฯรุ่นเก่าๆ ท่านจะเรียก ปรากฎการณ์นี้ว่า " โจรเข้าบ้านพระ "คำว่า " โจร " นั้นมิได้มีความหมายว่าเป็นโจรปล้นฆ่าแต่อย่างไร คำว่า " โจร " ในที่นี้แปลว่า คนฉลาด คนที่เก่งในทางฉกฉวยแย่งชิง หาประโยชน์ นั้นหมายถึง " ฆราวาส หรือ มัคทายก " ที่ช่วยพระหาเงินเข้าวัดในการอุปมาลักษณะนี้ มุมนี้ จะสะท้อนให้เห็นภาพของ พระอยากได้เงินไปบูรณะโบสถ์วิหารในวัด แต่ก็ขาดปัจจัย ด้วยเหตุว่า ตนเป็นพระ จึงไม่สะดวกในการจะออกไปเรี่ยร่ายเงินทอง จึงได้เรียก มัคทายกมาปรึกษาหารือว่า ในการครั้งนี้ ว่าจะดำเนินการอย่างไรฝ่ายมัคทายก ก็บอกกับพระว่า... " หลวงพ่อ วางใจได้ เดี๋ยวกระผมจัดให้ เรื่องอมตังค์ เอ้ย !!! เรื่องหาตังค์เข้าวัด ไว้ใจผมได้ขอรับ "ภาพเหล่านี้ยังรวมถึง การร่วมอำนาจกันของผู้นำสองฝ่ายที่มองดูเหมือนมีความขัดแย้งกัน แต่แท้จริงแล้ว เป็นเนื้อเดียวกันทีนี้ เราก็พอจะมองเห็นภาพที่มันจะเกิดขึ้น เกี่ยวกับเศรษฐกิจในบ้านเรา ในปีนี้ปีหน้ากันได้เลยย้อนกลับมาอ่านดวงเดิมของดวงเมืองประเทศไทยกัน การที่ดวงเมืองได้ผูกให้ ดาวพฤหัสฯกับดาวเสาร์อยู่ร่วมกันในราศีธนูนั้น จะทำให้เวลาดวงเมืองตกจะไม่ตกมาก เวลาดวงขึ้นก็จะไม่ขึ้นมาก ดังนั้นบ้านเมืองเราถึงได้รอดจากการล่าอาณานิคมฯจากชาติตะวันตก และสามารถชนะสงครามโลกถึงสองครั้งสองคราแต่ในดีย่อมมีเสีย ประเทศเราจึงเป็นประเทศที่พัฒนาไปข้างหน้าได้ช้ากว่าที่ควรจะเป็น จึงนับว่าเป็นช่วงที่สำคัญเป็นอย่างมาก ซึ่งจะมีผลต่อดวงชะตาในทุกชาวราศี จะมีอะไรเกิดขึ้นกับใครอย่างไร ติดตามอ่านได้เลยครับ......................................................................ลัคนาราศีเมษ ( หรือผู้ที่เกิดระหว่างวันที่ 13 เมย. - 13 พค. )
- ทั่วไป / จะได้รับข่าวด่วนที่ไม่ค่ดฝัน หรือมีส่วนร่วมงานพิธี- การเงิน / จะได้รับเงินส่วนแบ่งจากหุ้นส่วน หรือได้รับเงินปันผล เงินขวัญถุงจากคนใกล้ชิด- การงาน / มีความขัดแย้งเล็กๆขึ้นในส่วนของการทำงาน- สุขภาพ / ระวังการกินการอยู่ ระยะนี้ง่ายต่อการติดเชื้อในกระเพาะอาหารได้ง่าย หรือ มีปัญหาในช่องปาก- ความรัก / เกิดข้อขัดแย้งกับคนรักได้ง่าย โดยมากจะมาจากคำพูดเพียงเล็กน้อยลัคนาราศีพฤษภ ( หรือผู้ที่เกิดระหว่างวันที่ 14 พค. - 13 มิย. )- ทั่วไป / ระยะนี้ จะค่อนข้างน้อยอกน้อยใจได้ง่าย อารมณ์อ่อนไหว เจ้าน้ำตา- การเงิน / มีเรื่องจ่ายเยอะ แต่ก็ยังพอได้หนี้สินเก่าไจากลูกหนี้คืนมาบ้าง- การงาน / ทำอะไรยังติดๆขัดๆ การตัดสินใจยังไม่ค่อยแน่นอน- สุขภาพ / การขับถ่ายในระยะนี้ มักมีปัญหา ไม่เป็นไปตามปกติ- ความรัก / คนรักมักพูดจาไม่ถนอมน้ำใจลัคนาราศีเมถุน ( หรือผู้ที่เกิดระหว่างวันที่ 14 มิย. - 13 กค. )- ทั่วไป / มักมีคนชวนทำบุญเยอะ กิจกรรมหลายด้าน จนไม่รู้จะเลือกไปกับใครก่อน- การเงิน / ยังจัดระเบียบการเงินไม่ค่อยได้ ค่าใช้จ่ายในชีวิตประจำวันมากขึ้น- การงาน / มีความไม่ชัดเจนเกี่ยวกับหน้าที่การงาน ไม่รู้จะรับฟังคำสั่งจากใคร หรือ ไม่รู้จะตัดสินว่าข้อมูลทางใดเป็นข้อมูลที่เชื่อถือได้- สุขภาพ / มีเกณฑ์ได้รับความบาดเจ็บที่มือเท้า หรือปัญหาที่เล็บ รวมไปถึงการใช้ยาที่ผิดวิธีอีกด้วย- ความรัก / เกิดความสับสนในตัวคนรัก จะปล่อยมือหรือเดินต่อดีลัคนาราศีกรกฎ ( หรือผู้ที่เกิดระหว่าง 14 กค. - 16 สค. )
- ทั่วไป / ไปไหนมาไหนมักหลงทาง ทำให้ผิดนัด ผิดเวลาอยู่บ่อยครั้ง ควรเผื่อเวลาไว้มากหน่อย- การเงิน / มีรายจ่ายเกี่ยวกับที่พักอาศัย มีการชำระเงินล่วงหน้า ทำข้อตกลงในเรื่องการเงินกับคู่สัญญา- การงาน / จะได้ร่วมงานกับเพื่อนร่วมงาน หรือเจ้านายเก่าๆ ความไม่คาดคิดที่เกิดขึ้นในตัวเพื่อนร่วมงาน นายจ้าง หรือ ลูกค้า- สุขภาพ / ในหญิงจะมีปัญหาเกี่ยวข้องกับระบบภายใน ในชายมักมีปัญหากับข้อมือข้อเท้า หรือระบบทางเดินปัสสาวะ- ความรัก / จะมีคนต่างวัย ต่างชาติต่างภาษา ต่างศาสนา กรณีใดกรณีหนึ่ง เข้ามาให้ความสนใจลัคนาราศีสิงห์ ( หรือผู้ที่เกิดระหว่าง 17 สค. - 16 กย. )- ทั่วไป / จะได้พบกับสังคมใหม่ๆ ได้รับการแนะนำให้รู้จักกับคนหน้าใหม่ๆ- การเงิน / คนรอบข้างเป็นเหตุต้องให้หมุนเงินมากกว่าปกติ - การงาน / ต้องเข้าไปมีส่วนร่วมงานกับคนระดับสูง หรือมีการค้าขายสินค้าราคาแพง เข้าร่วมงานประกวดราคา ประมูลสินค้า- สุขภาพ / ระยะนี้จะมีปัญหาที่ลำคอ หลอดลม- ความรัก / คนที่เคยรักกัน ที่ห่างหายกันไปนานๆ จะหวนกลับมาให้ต้องพบเจอกันลัคนาราศีกันย์ ( หรือผู้ที่เกิดระหว่าง 17 กย. - 16 ตค. )- ทั่วไป / มีเรื่องมีเหตุที่ต้องออกหน้า ไม่ว่าจะเป็นกิจกรรมใดๆอยู่เสมอ- การเงิน / มักมีคนมาชักชวนให้ลงหุ้นลงทุน ชี้ช่องทางในการทำธุรกิจ ค้าขาย- การงาน / มีงานพิเศษนอกเหนือจากงานประจำ / หรือ ในรายที่ว่างงาน ก็จะได้งานฟรีร้านซ์ งานพาร์ททาม- สุขภาพ / มีแก๊สในกระเพาะอาหาร บางรายเกิดกรดไหลย้อน หรือหน้ามืดเป็นลมได้ง่าย- ความรัก / คนรักมักสร้างแรงกดดันให้ต้องหนักใจหลายเรื่องลัคนาราศีตุลย์ ( หรือผู้ที่เกิดระหว่าง 17 ตค. - 15 พย. )- ทั่วไป / มีงานเลี้ยง งานสัมมนาฯ  ต้องออกงานสังคมบ่อยขึ้น- การเงิน / มักมีเรื่องใช้จ่ายเป็นเงินก้อนใหญ่ ในการซื้อหาข้าวของเครื่องใช้- การงาน / ระยะนี้การงานจะเกี่ยวข้องกับของสวยงาม งานศิลป์ งานประกวดประขัน หรืองานด้านสาธารณะสุข- สุขภาพ / ระวังสัตว์เลี้ยง ข่วนกัด รวมไปถึงแมลงสัตว์กัดต่อย- ความรัก / คนรักมักมีเรื่องที่ทำให้ต้องหงุดหงิด อยู่เสมอ ลัคนาราศีพิจิก ( หรือผู้ที่เกิดระหว่าง 16 พย. - 15 มค. )
