
# PS4「新サクラ大戦 体験版」が11/21(木)配信決定！ゲームの中心となる「アドベンチャーパート」と「バトルパート」を体験可能!!

Published at: **2019-11-05T14:20:00+00:00**

Author: ****

Original: [アキバ総研](https://akiba-souken.com/article/42240/)

商品を購入する
セガゲームスは、2019年12月12日（木）に発売となるPlayStation 4用アクションアドベンチャー「新サクラ大戦」について、11月21日（木）よりPlayStation StoreにてPlayStation4用ソフトウェア「新サクラ大戦 体験版」を無料配信することを発表した。
 「新サクラ大戦」は、架空の「太正時代」の帝都・東京を舞台に、悪と戦う「帝国華撃団」の活躍を描くドラマチック3Dアクションアドベンチャー。 メインキャラクターデザインには「週刊少年ジャンプ」（集英社刊）で連載された漫画「BLEACH」の作者として知られる久保帯人さんを起用。音楽は、これまでのシリーズでも多数の楽曲を提供してきた田中公平さん。そしてストーリー構成は、ゲーム・TVアニメ・映画とメディアの枠を超えて活躍するイシイジロウさんが担当する。

        
          ⇒PS4「新サクラ大戦」、2019年冬発売決定！ 新主人公・神山誠十郎率いる、新たな「帝国華撃団」の物語
        
      
今回、11月21日（木）より、PlayStation StoreにてPlayStation4用ソフトウェア「新サクラ大戦 体験版」が無料配信されることが決定した。
「新サクラ大戦 体験版」では、ゲームの中心となる「アドベンチャーパート」と「バトルパート」をそれぞれ体験することができる。アドベンチャーパートでは大帝国劇場内を散策し、帝国華撃団・花組隊員たちとのイベントが楽しめる、体験版専用の特別構成となっている。時間制限式選択肢「LIPS」によって変化する、ヒロインたちの反応やコミュニケーションを楽しむことができる。また、バトルパートでは、主人公・神山誠十郎とヒロインのひとり・望月あざみの機体を切り替えてプレイすることができる。
また、「新サクラ大戦」に登場するキャラクター、ホワン・ユイ（CV：上坂すみれ）の歌う、「虹の彼方」ミュージックビデオ（Short ver.）が公開された。なお、「虹の彼方」のフルバージョンは「新サクラ大戦 初回限定版」に同梱されるヴォーカルトラックCD集「「サクラ大戦」歴代歌謡集」にて楽しむことができる。シリーズ6作品の歌唱曲62曲が収録されたCD6枚組に加え、シリーズをイラストで振り返るアートブック（A4判、72P）がセットになった初回限定版は数に限りがあるので、お早めの予約をオススメする。■ホワン・ユイ（上坂すみれ）「虹の彼方」ミュージックビデオ（Short ver.）公開！
