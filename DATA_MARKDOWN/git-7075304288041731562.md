
# Рианна создала коллекцию украшений для своего бренда Fenty

Published at: **2019-11-01T10:51:20+00:00**

Author: ****

Original: [Wonderzine](https://www.wonderzine.com/wonderzine/style/stylenews/246837-fenty-jewelry-collection)

На сайте бренда Fenty, который основала Рианна, появилась коллекция украшений. Главным мотивом послужила камея — в виде неё сделаны крупные серьги, кольца и подвески, дополненные жемчугом и камнями. Цены на изделия стартуют с 340 долларов.
Рекламную кампанию для бренда, в которой приняли участие только модели африканского происхождения, сняла нигерийская фотограф Рут Оссай. За сложные причёски отвечал стилист Айзек Полеон.
