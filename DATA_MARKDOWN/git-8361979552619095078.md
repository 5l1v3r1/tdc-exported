
# Антикоррупционный суд объявил перерыв в деле Дубневича

Published at: **2019-11-01T16:37:00+00:00**

Author: ****

Original: [НВ](https://nv.ua/ukraine/politics/sud-nad-dubnevichem-obyavlen-pereryv-novosti-ukrainy-50051160.html)

Об этом сообщает РБК-Украина.
Как отметила судья, судебное заседание продолжится завтра, 2 ноября в 11:30.
Напомним, прокуроры САП просят суд арестовать Дубневича с правом на внесение залога размером 93 млн гривен.
31 октября Верховная Рада сняла неприкосновенность с нардепа Дубневича. Генпрокурор Руслан Рябошапка лично сообщил ему о подозрении.
По данным следствия, будучи председателем парламентского комитета по вопросам транспорта, Дубневич мог быть причастен к махинациям при проведении госзакупок для нужд Укрзализныци.
