
# Ижемец обманул администрацию Корткеросского района

Published at: **2019-11-07T10:30:00+00:00**

Author: **Комиинформ**

Original: [Комиинформ](https://komiinform.ru/news/188514)

Заместитель прокурора Республики Коми утвердил обвинительное заключение по уголовному делу в отношении 39-летнего жителя Ижемского района. Как сообщили в надзорном ведомстве, органом предварительного расследования он обвиняется в совершении преступления, предусмотренного ч. 3 ст. 159 УК РФ (мошенничество, то есть хищение чужого имущества путем обмана, совершенное лицом с использованием своего служебного положения, в крупном размере).
По версии следствия обвиняемый с декабря 2017 года по январь 2018 года, исполняя обязанности директора одного из муниципальных предприятий Корткеросского района, заключил с Администрацией муниципального района "Корткеросский" договоры подряда на производство работ по сносу ветхого аварийного жилья в рамках республиканской адресной программы "Переселение граждан из аварийного жилищного фонда" на 2013-2017 годы. В дальнейшем мужчина передал для оплаты сотрудникам местной администрации акты о приемке выполненных работ по форме КС-2, справки о стоимости выполненных работ и затрат по форме КС-3, а также счета на оплату, содержащие заведомо ложные сведения о выполнении работ, тем самым похитив денежные средства в сумме более 406 тысяч рублей, распорядившись ими по своему усмотрению.
Вину в совершении преступления обвиняемый не признал. Уголовное дело для рассмотрения по существу направлено в Корткеросский районный суд.
Санкция ч. 3 ст. 159 УК РФ предусматривает наказание в виде лишения свободы на срок до 6 лет.
Расследование по делу осуществлено Следственной частью Следственного управления МВД по Республике Коми.
