
# Шахматисты из РФ победили на командном чемпионате Европы

Published at: **2019-11-02T12:44:51+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-02/shahmatisty-iz-rf-pobedili-na-komandnom-chempionate-evropy)

Мужская сборная РФ победила на командном чемпионате Европы по шахматам.
В заключительном туре российский шахматисты обыграли команду из Польши. Украинская сборная, у которой было равное с командой России количество очков перед решающим игровым днем, сыграла вничью с шахматистами из Хорватии.
Соревнования состоялись в Батуми.
