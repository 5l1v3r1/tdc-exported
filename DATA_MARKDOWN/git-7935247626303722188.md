
# Eminönü'de balık ekmek satan tekneler için karar

Published at: **2019-11-01T10:20:15+00:00**

Author: ****

Original: [OLAY](https://www.olay.com.tr/eminonude-balik-ekmek-satan-tekneler-icin-karar-213839h.htm)

İstanbul Büyükşehir Belediyesi (İBB), Eminönü'nde balık ekmek satan teknelerin 1 Kasım'a kadar boşaltılması için ihtarname göndermişti.
Tarihi Eminönü Turizm Balık Ekmek Kooperatifi'nin başvurusu üzerine İstanbul 4. İdare Mahkemesi yürütmeyi durdurma kararı verdi.
