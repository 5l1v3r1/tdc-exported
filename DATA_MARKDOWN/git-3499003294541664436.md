
# Полицейские УВД юга столицы задержали подозреваемого в краже

Published at: **2019-11-01T15:11:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/yuao/politseyskie-uvd-yuga-stolitsy-zaderzhali-podozrevaemogo-v-krazhe-123268/)

В ОМВД России по Даниловскому району с заявлением о краже обратился житель столицы. По словам потерпевшего неизвестный с лестничной площадки похитил его велосипеды, после чего скрылся. Ущерб составил более 153 000 рублей.В результате проверочных мероприятий полицейские задержали подозреваемого. Им оказался 22-летний житель столицы. Похищенное изъято.По данному факту следствием территориального ОМВД возбуждено уголовное дело по признакам преступления, предусмотренного статьей 158 УК РФ «Кража». В отношении подозреваемого избрана мера пресечения в виде подписки о невыезде и надлежащем поведении.Пресс-служба УВД по ЮАО(495) 734-61-22
