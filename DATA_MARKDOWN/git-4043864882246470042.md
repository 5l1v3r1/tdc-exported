
# Назвали причину масового мору риби на Херсонщині

Published at: **2019-11-06T13:21:00+00:00**

Author: ****

Original: [Українська правда](https://www.pravda.com.ua/news/2019/11/6/7231174/)

Причиною масового замору риби в Дніпро-Бузькому лимані на Херсонщині назвали аміак, який виділився через природні, а не через техногенні процеси.
Джерело: заява завідувача кафедри водних біоресурсів та акваторії Херсонського державного аграрного університету Павло Кутіщев, "UA.Новини Суспільне"
Деталі: За його словами, виділення шкідливої речовини сталося через природні, а не через техногенні процеси.
Дослівно: "Є думка, що задуха риби — явища техногенного характеру. Ми стверджуємо, що задуха риби — явище природнє, яке по Україні спостерігається по всім містах внаслідок підвищення температури клімату. Це наслідок екологічних змін, трансформації Дніпра каскадом водосховища".
