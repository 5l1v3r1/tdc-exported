
# بازی‌های دیجیتالی به مثابه دارو برای کودکان اوتیسم

محققان ایرانی با راه اندازی یک استارتاپ مستقر در مرکز رشد دانشگاه تربیت مدرس مشغول فعالیت در زمینه طراحی بازی‌‎های دیجیتال برای تقویت مهارت در بیماران اوتیسم هستند و تا کنون سه بازی طراحی کرده‌اند.
