
# โอริกีรับแย่งตัวจริงหงส์แดงลำบาก

Published at: **2019-11-06T17:35:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/sports/740359)


        "ดิวอก โอริกี" หัวหอกตัวสำรองอดทนของ ลิเวอร์พูล ยอมรับโอกาสเบียดแย่งตัวจริงจาก 3 ประสาน "เอสเอ็มเอฟ" เป็นเรื่องยากเหลือเกิน
      
ดิวอก โอริกี กองหน้าทีมชาติเบลเยียม ของ ลิเวอร์พูล จ่าฝูง พรีเมียร์ลีก อังกฤษ ยอมรับโอกาสที่ตัวเขาจะเบียดแย่งตำแหน่งตัวจริงจาก 3 แนวรุกมหากาฬอย่าง ซาดิโอ มาเน, โรแบร์โต ฟีร์มิโน และโมฮาเหม็ด ซาลาห์ มาได้นั้นเป็นเรื่องยากทีเดียวโอริกี ไม่ได้ลงเล่นมากนักในฤดูกาลนี้ เนื่องจากต้องตกเป็นตัวเลือกอันดับ 4 ถัดจากทั้ง มาเน, ฟีร์มิโน และซาลาห์ อย่างไรก็ตาม กองหน้าเลือดเบลเจียน ก็ไม่ได้ท้อ และพร้อมจะพยายามอย่างเต็มที่เพื่อทำให้ เจอร์เกน คลอปป์ ผู้จัดการทีม เรด แมชีน ต้องหนักใจมากขึ้นในการเลือกผู้เล่นลงสนาม"ผมชอบที่ทีมของเรามีนักเตะคุณภาพมากมายเพราะนั่นทำให้เราคว้าถ้วย แชมเปี้ยนส์ ลีก มาครองได้ ถ้าทุกคนอยู่ในฟอร์มที่ดีที่สุด แน่นอนว่าตัวเลือกของโค้ชก็ไม่มีอะไรใหม่ มันขึ้นอยู่กับโค้ช และหน้าที่ของนักเตะอย่างเราก็คือทำให้โค้ชเลือกจัดทีมได้ยากที่สุดเท่าที่จะสามารถเป็นไปได้" โอริกี กล่าว. ภาพ gettyimages
