
# Στον «αέρα» η Συμφωνία START για τους εξοπλισμούς: Γιατί η Ρωσία επικρίνει τις ΗΠΑ

Published at: ****

Author: ****

Original: [CNN.gr](https://www.cnn.gr/news/kosmos/story/195886/ston-aera-i-symfonia-start-gia-toys-exoplismoys-giati-i-rosia-epikrinei-tis-hpa)

Newsroom, CNN Greece 14:54 , 01 November 2019
Τη δήλωση αυτή έκανε σήμερα ο αναπληρωτής διευθυντής της αρμόδιας Διεύθυνσης του ρωσικού υπουργείου Εξωτερικών για ζητήματα μη διάδοσης και ελέγχου των εξοπλισμών Βλαντίμιρ Λεόντιεφ κατά την διάρκεια συζήτησης που έγινε στη λέσχη Valdai για τα μέλλον των στρατηγικών εξοπλισμών στο τρίγωνο ΗΠΑ - Ρωσίας - Κίνας.
«Η κατάσταση σε σχέση με τις προοπτικές της Συμφωνίας START είναι περίπλοκη και ακατανόητη. Είναι ήδη προφανές ότι στο χρονικό διάστημα που απομένει (η ισχύς της Συμφωνίας λήγει στις 5 Δεκεμβρίου 2021) δεν θα μπορέσουμε να επεξεργαστούμε ένα πλήρες κείμενο συμφωνίας που να την αντικαταστήσει», δήλωσε ο Λεόντιεφ.
«Οι διαπραγματεύσεις για τη START διεξάγονταν επί έναν χρόνο. Και αυτό επειδή η Συμφωνία δεν είναι σχετικά περίπλοκη, αλλά συνιστά μια απλουστευμένη εκδοχή της προηγούμενης συμφωνίας» δήλωσε ο Λεόντιεφ, προσθέτοντας ότι «σ’ αυτήν δεν υπάρχουν κάποια ουσιαστικά νέα στοιχεία. Την ίδια στιγμή, όπως τώρα, εγείρονται ζητήματα, τα οποία απαιτούν μια πολύ σοβαρή προκαταρκτική επεξεργασία σε επίπεδο εμπειρογνωμόνων».
Σύμφωνα με το ΑΠΕ - ΜΠΕ, η Συμφωνία START τέθηκε σε ισχύ το 2011 με διάρκεια εννέα ετών, εφόσον δεν υπάρξει νέα συμφωνία έως το 2021. Επίσης μπορεί να παραταθεί όχι για διάστημα μεγαλύτερο των πέντε ετών (έως το 2026) κατόπιν αμοιβαίας συμφωνίας των μερών.
