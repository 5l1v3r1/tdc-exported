
# В Москве штатно сел неисправный самолет из Германии

Published at: **2019-11-04T14:50:00+00:00**

Author: **Информационная служба Накануне.RU**

Original: [Накануне.ru](https://www.nakanune.ru/news/2019/11/04/22557229/)

В аэропорту Шереметьево совершил посадку неисправный самолет из Франкфурта.
Посадка прошла штатно, никто не пострадал. О необходимости приземлиться в московском аэропорту экипаж самолета сообщил в связи с неполадками в гидросистеме.
