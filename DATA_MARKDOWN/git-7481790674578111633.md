
# 設楽悠、東京マラソン出場へ…五輪札幌変更に「選手は従うだけ」

Published at: **2019-11-04T02:17:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/etc/20191104-OYT1T50090/)

　男子マラソンの前日本記録保持者、設楽悠太（ホンダ）は３日、来年３月の東京マラソンに出場する意向を明かした。東日本実業団対抗駅伝に出場後、取材に応じた。残り１枠の東京五輪代表権を得るには、２時間５分５０秒の日本記録の更新が必要となる。五輪マラソンの札幌への開催地変更は「決まった以上、選手は従うだけ」と冷静に受け止めた。
