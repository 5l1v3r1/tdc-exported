
# Гололед прогнозируется в Нур-Султане

Published at: **2019-11-04T10:31:00+00:00**

Author: **ИА Тотал Казахстан**

Original: [Тотал](https://total.kz/ru/news/zhizn/gololed_prognoziruetsya_v_nursultane_date_2019_11_04_10_23_12)


        Штормовое предупреждение для столицы распространил «Казгидромет».
      
Штормовое предупреждение для Нур-Султана распространили синоптики РГП «Казгидромет», сообщает Total.kz.
В Нур-Султане 4 ноября ожидается гололед. Вероятность шторма — 80-85%, отметили синоптики.
