
# Day N Vegas: Travis Scott out; Future, Young Thug, Playboi Carti & more added

Published at: **2019-11-01T00:21:00+00:00**

Author: **Andrew Sacher**

Original: [BrooklynVegan](http://www.brooklynvegan.com/day-n-vegas-travis-scott-out-future-young-thug-playboi-carti-more-added/)

Las Vegas rap festival Day N Vegas is this weekend, Friday (11/1) through Sunday (11/30 at the Las Vegas Festival Grounds. Unfortunately, Saturday headliner Travis Scott had to drop off, but his replacements are pretty big: Future x Metro Boomin, Playboi Carti, Trippie Redd, and Shoreline Mafia. Day N Vegas also added Young Thug to Sunday.
Travis Scott tweeted, “TO ALL RAGERS AT @DayNVegas2019 IM SORRY THAT I CANT PULL UP BUT I PROMISE TO BE BACK SO SOON ON GANG.”
The fest also features J Cole, Miguel, Denzel Curry, Saba, and more on Friday; Migos, 21 Savage, DaBaby, Rico Nasty, and more on Saturday; and Kendrick Lamar, Tyler the Creator, Schoolboy Q, Brockhampton, Megan Thee Stallion, and more on Sunday.
Select ticketing options still available.
Set times for Friday and Saturday out now (see below). Sunday set times TBA.
