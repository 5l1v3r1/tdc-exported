
# Infiniti обновила информационно-развлекательную систему

Published at: **2019-11-06T00:00:00+00:00**

Author: **Ирина Котова**

Original: [CAR.RU](https://car.ru/news/tehnologii/56946-infiniti-obnovila-informatsionno-razvlekatelnuyu-sistemu/)


        Производители автомобильного концерна Infiniti разработали обновленную мультимедийную систему, получившую название InTouch.
      
Главной особенностью обновленной системы становится современный интерфейс, который способен раздавать Wi-Fi. Кроме того, мультимедиа взаимодействует с Apple CarPlay и Android Auto.
Новый развлекательный комплекс будет доступен для автомобилей бренда, начиная с 2020 года. Сейчас все выпускаемые машины оснащаются прежней информационной системой, которая является более простой и не имеет такого количества опций.
 
Впервые новая система появится на моделях QX50 и QX80. Интерфейс имеет два экрана. Кроме того, теперь информационный комплекс отличается современной графикой. Раздать Wi-Fi одновременно можно на семь различных устройств. Получать необходимые обновления можно при помощи беспроводного сетевого подключения.
Разработчики уверены, что автомобилисты по достоинству оценят современный интерфейс.
