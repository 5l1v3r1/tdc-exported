
# 矢野監督　横山復活に期待！ブルペン熱視線　キャンプ初日のＭＶＰに

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/tigers/2019/11/01/0012838176.shtml)

　阪神の矢野燿大監督（５０）が３１日、キャンプ初日の“ＭＶＰ”に、１４年度ドラフト１位で、１８年８月の左肩手術から復活を目指す育成選手・横山を挙げた。
　ブルペンに向かうと、一番乗りした背番号「１１５」を注視。「（ストライク）ゾーンの中で質を高めろ」と助言も送った。「面白い、いいボールを投げていた。今日は横山がしっかり腕を振って投げていた」と評価した。
　復活を願う気持ちは強い。昨年は２軍監督として、横山の苦しむ姿を見てきただけに、「気持ちも伝わってくる」と５年目左腕の姿勢に目を細めた。
　横山は今キャンプ前のリポートに「何も失うモノはない」と記した。「やってやろうと思っている」。昨秋キャンプで台頭し、今季ブレークした島本、守屋に続く活躍を目指す。
