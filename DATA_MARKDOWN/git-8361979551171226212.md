
# Жителя Николаева задержали при попытке вывезти детали для кораблей флота РФ

Published at: **2019-11-01T19:28:00+00:00**

Author: ****

Original: [НВ](https://nv.ua/ukraine/events/zhitel-nikolaeva-hotel-vyvezti-detali-korabley-rf-novosti-ukrainy-50051198.html)

Об этом сообщает пресс-служба СБУ.
По предварительной оценке, стоимость оборудования составляет более полумиллиона гривен.
По месту жительства мужчины были проведены обыски, во время которых изъяли флешки и компьютерную технику, на которой сохранилась переписка с заказчиками из России.
Следователи начали расследование по факту «нарушение режима осуществления международных передач товаров, подлежащих государственному экспортному контролю».
