
# Штраф за езду без ОСАГО может вырасти многократно

Published at: **2019-11-01T00:00:00+00:00**

Author: **Екатерина Криль**

Original: [Омск Здесь](https://omskzdes.ru/society/66041.html)


        В семь раз может вырасти штраф для россиян, ездящих без ОСАГО.
      
 
На рабочей группе Государственной думы по ОСАГО обсуждался вопрос возмещения ущерба потерпевшим при ДТП, виновник которого не имеет полиса автогражданской ответственности. Законодатели рассматривают несколько инструментов, один из них - значительное повышение штрафов, сообщают "Известия".
- Часть автомобилистов не имеют полиса ОСАГО, и когда происходят аварии по их вине, потерпевшие остаются наедине с собой. Поэтому во многих странах, в том же Казахстане, установлены очень большие штрафы за это нарушение. Люди без полиса ОСАГО должны нести адекватное наказание, - прокомментировал член комитета Госдумы по финансовому рынку Алексей Изотов.
Сейчас штраф за езду без полиса составляет 800 рублей. В перспективе его размер могут приравнять к средней стоимости страховки, то есть 5400 рублей. Отмечается, что законопроекта об ужесточении наказания для водителей без ОСАГО пока нет и вопрос лишь обсуждается.
