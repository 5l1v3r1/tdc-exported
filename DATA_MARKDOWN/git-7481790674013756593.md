
# 「嵐」、五つのＳＮＳ開始…来年５月に新国立で公演

Published at: **2019-11-03T04:58:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/culture/20191103-OYT1T50129/)

　人気アイドルグループ「嵐」が３日、東京都内で記者会見を開き、同日からツイッターなど五つのＳＮＳを開始することのほか、全国ツアー「ＡＲＡＳＨＩ　Ａｎｎｉｖｅｒｓａｒｙ　Ｔｏｕｒ　５×２０」の千秋楽公演（１２月２５日）でライブビューイングを実施することや、来年５月１５、１６日に新国立競技場でコンサートを行うことなどを明らかにした。
　１１月３日は、２０年前に嵐がＣＤデビューした記念日。メンバーの桜井翔さんは「日々考えていた感謝の思いを伝えることの一端」などと話した。会見の冒頭約１５分は、ＹｏｕＴｕｂｅで生配信された。
