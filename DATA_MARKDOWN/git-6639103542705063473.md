
# Двое детей утонули в овраге в Приморье

Published at: **2019-11-05T01:15:00+00:00**

Author: ****

Original: [Interfax.ru](https://www.interfax.ru/russia/682924)

Москва. 5 ноября. INTERFAX.RU - Тела семилетнего мальчика и трехлетней девочки обнаружены в Спасском районе Приморья, по предварительным данным, они утонули, говорится в сообщении следственного управления СКР по Приморскому краю, распространенном во вторник.
По данным ведомства, тела детей были найдены в заполненном водой овраге в селе Летно-Хвалынское.
По данному факту возбуждено уголовное дело по статье о причинении смерти по неосторожности (ч. 3 ст. 109 УК РФ).
