
# 陳亞蘭日操16小時賣命　經紀人送雙人機票原因超暖

Published at: **2019-11-02T08:17:00+00:00**

Author: **梁欣喬, 李鍾泉**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191102ent004/)

歌仔戲小生天王陳亞蘭今（2日）舉辦生日會，現場狂撩上百粉，她一出場，就用電眼對粉絲放電，完成粉絲想與她眼神交流心願。歌壇天后A-lin送上花籃祝福，她演唱《忠孝節義》第三單元主題曲「斷機教子」主題曲。陳亞蘭台上也與第一單元「萬古流芳」的娘子葉麗娜重現劇中轉圈圈戲碼，更宣布將舉辦「公益辦桌」回饋社會。
陳亞蘭為戲日操16小時，透露這兩年做了很多事情，除就讀台師大GF-EMBA外，還身兼電視歌仔戲《忠孝節義》執行製作人，以及推出歌仔戲舞台劇《黑家店2：萬古流芳》，自豪體力跟以往毫無差別，更認為自己靠著意志力與衝勁在拼，笑說以前的身體還比較虛，朋友看到她現況打趣說她可以參加鐵人三項。
陳亞蘭經記人陳秀卿看到她日操16小時為戲賣命，連好好與她說話談心的時間都沒有，體貼送上日本雙人來回機票與雞精補充體力，要她好好休息度假，經記人笑說，「送日本雙人機票不是小氣，而是亞蘭姐不習慣一個人，而且已她現在虛累累情況，不適合走太遠。」驚喜送禮，讓陳亞蘭直呼感動。
