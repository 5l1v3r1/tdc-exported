
# 首相、中国・李首相と会談…習主席来日に向け連携確認へ

Published at: **2019-11-04T03:38:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/politics/20191104-OYT1T50093/)

　【バンコク＝池田慶太】安倍首相は４日、バンコクで開かれている東南アジア諸国連合（ＡＳＥＡＮ）関連首脳会議の会場内で、中国の李克強（リークォーチャン）首相と個別に会談した。来年春の習近平（シージンピン）国家主席の国賓来日に向け、日中両政府の連携を確認する方向だ。
　両首相の会談は、昨年１０月に安倍首相が中国を公式訪問して以来。
　安倍首相は会談で、中国公船による尖閣諸島周辺の領海侵入事案のほか、９月に北京で拘束された北海道大学の男性教授ら日本人の拘束が相次いでいることなども取り上げ、中国に適切な対応を求めるとみられる。
