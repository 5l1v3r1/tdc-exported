
# ധര്‍മശാല അന്താരാഷ്ട്ര ഫിലിം ഫെസ്റ്റിവല്‍ ഏട്ടാം പതിപ്പ് ആരംഭിച്ചു; ലിജോ ജോസ് പെല്ലിശ്ശേരിയുടെ ജല്ലിക്കട്ടും പ്രദര്‍ശനത്തിന്

Published at: **2019-11-07T20:51:00+00:00**

Author: **എന്റര്‍ടെയിന്‍മെന്റ് ഡെസ്‌ക്**

Original: [DoolNews](https://www.doolnews.com/diff-2019-dharamshala-international-film-festival-start452.html)

ഷിംല: ധര്‍മശാല അന്താരാഷ്ട്ര ഫിലിം ഫെസ്റ്റിവലിന്റെ എട്ടാം പതിപ്പ് ആരംഭിച്ചു.. കഴിഞ്ഞ മൂന്ന് വര്‍ഷമായി ഡി.ഐ.എഫ്.എഫ് ആതിഥേയത്വം വഹിച്ചിരുന്ന ടിബറ്റന്‍ ചില്‍ഡ്രന്‍സ് വില്ലേജില്‍ നിന്ന് ഈ വര്‍ഷം ടിബറ്റന്‍ ഇന്‍സ്റ്റിറ്റ്യൂട്ട് ഓഫ് പെര്‍ഫോമിംഗ് ആര്‍ട്സിലാണ് ഫെസ്റ്റിവല്‍ നടക്കുന്നത്.
റിതു സരിന്‍, ടെന്‍സിംഗ് സോനം എന്നിവരാണ് ഫെസ്റ്റിവല്‍ ഡയറക്ടര്‍മാര്‍, പ്രതീക് വാട്സ് സംവിധാനം ചെയ്ത ഈബ് അല്ലെ ഓ (2019) ആണ് ഓപ്പണിംഗ് ചിത്രം. നവംബര്‍ പതിനൊന്നാം തിയ്യതിയാണ് ഫെസ്റ്റിവല്‍ അവസാനിക്കുക. ഗീതഞ്ജലി റാവുവിന്റെ ഫീച്ചര്‍ ആനിമേഷന്‍ ബോംബെ റോസ് ഫെസ്റ്റിവല്‍ അവസാനിക്കുന്നത്.

        
          വാര്‍ത്തകള്‍ ടെലഗ്രാമില്‍ ലഭിക്കാന്‍ ഇവിടെ ക്ലിക്ക് ചെയ്യൂ
        
      
ലിജോ ജോസ് പെല്ലിശ്ശേരിയുടെ ജല്ലിക്കട്ട് ഫെസ്റ്റിവലില്‍ പ്രദര്‍ശിപ്പിക്കുന്നുണ്ട്. എബൗട്ട് ലവ് ( അര്‍ച്ചന ഫഡ്കെ), ഐസ് ഹീ (കിസ്ലേ), ബിര്‍ഹ (ഏക്ത മിത്തല്‍), ബോംബെ റോസ് (ഗീതഞ്ജലി റാവു), ഈബ് അല്ലെ ഓ ( പ്രതീക് വാറ്റ്‌സ്), ( ഗുര്‍വീന്ദര്‍ സിംഗ്), ഫ്രീഡം ലൈഫ് (പ്രിയ സെന്‍) എന്നിവയാണ് മറ്റ് പ്രധാന ചിത്രങ്ങള്‍.

        
          ഡൂൾന്യൂസ് യൂട്യൂബ് ചാനൽ സബ്സ്ക്രൈബ് ചെയ്യാനായി ഇവിടെ ക്ലിക്ക് ചെയ്യൂ
        
      
ഷോര്‍ട് ഫിലിമുകള്‍ക്കായും കുട്ടികളുടെ ചിത്രങ്ങള്‍ക്കായും പ്രത്യേക സെക്ഷനുകളും അവതരിപ്പിക്കുന്നുണ്ട്.
