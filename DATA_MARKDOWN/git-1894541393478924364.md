
# Свыше 160 тысяч украинцев закрыли свой бизнес в текущем году

Published at: **2019-11-06T09:41:00+00:00**

Author: ****

Original: [ОстроВ](https://www.ostro.org/general/economics/news/579100/)

Свой бизнес закрыли 167 тысяч украинских предпринимателей на начало октября 2019 года.  Об этом свидетельствуют данные Государственной налоговой службы Украины, - сообщает УНН.
Как сообщается, по состоянию на 01.10.2019 года общее количество физических лиц-предпринимателей, прекративших предпринимательскую деятельность по данным Отчета "О состоянии учета плательщиков", составило 167 тысяч человек.
По данным ведомства, в 2018 году предпринимательскую деятельность прекратили 196 тысяч человек.
Как ранее сообщалось, Украина поднялась на 7 пунктов в рейтинге Doing Business-2020 и заняла 64 место среди 190 стран. Премьер-министр Алексей Гончарук посчитал это "позитивным сигналом" для украинских предпринимателей.
