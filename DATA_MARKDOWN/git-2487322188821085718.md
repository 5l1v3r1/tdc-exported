
# Сотрудник банка в Петербурге похитил 20 тысяч рублей со счёта клиента, чтобы оплатить ипотеку

Published at: **2019-11-05T13:55:42+00:00**

Author: **Arthur Khamzin**

Original: [TJ](https://tjournal.ru/news/124417-sotrudnik-banka-v-peterburge-pohitil-20-tysyach-rubley-so-scheta-klienta-chtoby-oplatit-ipoteku)

Он сделал копию карты, с помощью которой снял деньги через банкомат.
Сотрудник петербургского банка похитил 20 тысяч рублей со счёта клиента, чтобы заплатить за ипотеку. Об этом сообщается на сайте прокуратуры города.
По данным следствия, подозреваемый украл средства в декабре 2018 года. Для получения денег он изготовил копию карты одного из клиентов. Заполучив пин-код, сотрудник обналичил деньги через банкомат.
Служащего банка обвиняют в краже средств с банковского счёта (часть 3 статьи 158 УК РФ). Дело уже направлено во Фрунзенский районный суд.
