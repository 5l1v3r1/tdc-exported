
# Dica do Dia: Como activar o modo escuro na app Microsoft To Do para Windows 10

Published at: **2019-11-05T18:01:22+00:00**

Author: ****

Original: [PCGuia](https://www.pcguia.pt/2019/11/dica-modo-escuro-microsoft-windows/)

Para activar o modo escuro na aplicação Microsoft To Do para Windows 10, clicar na foto de perfil situada no menu lateral esquerdo. De seguida, clicar em “Definições”. Por fim, localizar a categoria “Tema” e clicar na opção “Tema escuro”.
