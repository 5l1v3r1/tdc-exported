
# ¿Pensando en comprar una cerradura inteligente? Todo lo que debes saber antes de comprarla

Published at: **2019-11-01T18:00:17+00:00**

Author: **Roberto Adeva**

Original: [ADSLZone](https://www.adslzone.net/reportajes/domotica/guia-compra-cerraduras-inteligentes/)

El avance de las tecnologías ha hecho posible que hoy en día el Internet de las cosas sea una completa realidad. Tanto es así que cada vez encontremos más dispositivos capaces de conectarse a Internet, comunicarse entre sí o que podamos controlar de forma remota. Uno de estos dispositivos son las cerraduras inteligentes, que nos aportan un sistema adicional para cerrar y abrir la puerta de nuestra casa y que además nos hacen más fácil el día a día. Si estás pensando poner una cerradur inteligente en la puerta de tu casa, esto es todo lo que debes saber antes de comprar una y algunas de las mejores cerraduras inteligentes que puedes comprar a día de hoy.
Las cerraduras inteligentes son aquellas que, como su propio nombre indica han sido diseñadas para ofrecer las funciones propias de una cerradura convencional pero que además, añaden otro tipo de sistemas de acceso como son las huellas digitales, contraseñas o códigos PIN, mandos a distancia, etc. Cerraduras que además tienen la capacidad de conectarse a Internet y ser controladas por otros dispositivos como el propio smartphone.
Y es que, las cerraduras inteligentes se caracterizan por funcionar mediante una aplicación que se conecta directamente a nuestro teléfono móvil y desde donde podemos abrir y cerrar la puerta de casa a distancia.
Aunque muchos piensan que han sido diseñadas para ofrecer una mayor seguridad a la hora de entrar en una casa, son otros muchos los que piensan todo lo contrario, ya que exponen la seguridad de la casa a posibles ataques a través de Internet. Lo que si está claro es que las cerraduras inteligentes nos facilitan el día a día, tal y como veremos a continuación.
Las cerraduras inteligentes pueden hacer uso de diferentes sistemas o métodos para la apertura y cierre de las puertas, por lo tanto, cada una tendrá un sistema o mecanismo distintos para funcionar. Sin embargo, el funcionamiento general es muy similar para todas ellas, ya que cuentan con un sistema eléctrico o electromagnético que se encarga de mantener la puerta cerrada hasta que el controlador de acceso indique que se debe abrir.
Lo que hacen las cerraduras inteligentes es aportar un sistema de identificación para la persona que quiera abrirla. De esta manera, a través del sistema que incorpore cada cerradura, cada usuario deberá identificarse de manera correcta para que la puerta se abra o cierre.
Podríamos decir que se trata de un sistema similar al del bloqueo y desbloqueo de nuestros smartphones. Es decir, a través de un PIN o contraseña que indicamos a través del teclado podemos desbloquear nuestro móvil o abrir nuestra puerta, mientras que también existen cerraduras que podemos abrir o cerrar con nuestra huella dactilar, por ejemplo.Estos sistemas de seguridad suelen ser bastante sofisticados para garantizar la seguridad de las mismas
Además, el hecho de que puedan estar conectadas a Internet, permiten que podamos abrir o cerrar la puerta cuando queramos de forma remota desde otro dispositivos como el propio móvil.
Podríamos clasificar las cerraduras inteligentes en diferentes tipos según el sistema que utilice para abrir o cerrar la puerta o las tecnologías que intervienen.
