
# मुलीकडे शरीर सुखाची मागणी करणाऱ्या नराधमास महिलांनी दिला चोप 

Published at: **2019-11-07T16:30:10+00:00**

Author: **ऑनलाइन लोकमत**

Original: [Lokmat](https://www.lokmat.com/crime/man-assaulted-women-who-demanded-sex-girl/)


        ठळक मुद्दे
      
कल्याण - कल्याण नजीक असलेल्या अटाळी परिसरात एका १९ वर्षीय मुलीची धीरज राजपूत हा ५५ वर्षीय व्यक्ती दररोज छेड काढत होता. मुलीकडून शारीरिक सुखाची मागणी करत होता. अखेर मुलीने हा सर्व प्रकार आपल्या आईला सांगितला.हा प्रकार समोर आल्यानंतर परिसरातील मुलीची छेड काढणारा धीरज राजपूत या व्यक्तीला एका खांबाला बांधून त्याला महिलांनी चोप दिला आणि त्याची धिंड काढत खडकपाडा पोलिसांच्या ताब्यात दिले. खडकपाडा पोलिसांनी धीरज राजपूत याच्याविरोधात गुन्हा दाखल करत त्याला अटक केली आहे. धीरजला मारहाण करतानाचा व्हिडीओ आणि त्याची धिंड काढतानाचा व्हिडीओ समोर आला आहे. या व्हिडिओत महिलांना नराधमास बेदम चोप दिला आहे.
