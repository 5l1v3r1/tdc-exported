
# Одного из сбежавших из психбольницы преступников задержали в Чите

Published at: **2019-11-06T06:36:19+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-06/odnogo-iz-sbezhavshih-iz-psihbolnicy-prestupnikov-zaderzhali-v-chite)

Полицейские задержали одного из двух пациентов психиатрической больницы в Чите, которые сбежали из медучреждения.
Об этом сообщили в пресс-службе регионального УМВД России по региону.
Поиски второго беглеца продолжаются.
Ранее сообщалось о том, что из психбольницы в Чине сбежали пациенты, которые были осуждены за убийство и кражу. Судом они были признаны недееспособными, поэтому их поместили в психбольницу.
