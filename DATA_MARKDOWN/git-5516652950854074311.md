
# Ισοψηφία «Δημοκρατικής Κίνησης Μηχανικών» και «Συνεργασίας Μηχανικών Δωδεκανήσου» στην Κω για τις εκλογές στο ΤΕΕ

Published at: **2019-11-03T20:34:02+00:00**

Author: ****

Original: [Η ΡΟΔΙΑΚΗ](https://www.rodiaki.gr/article/425095/isopshfia-dhmokratikhs-kinhshs-mhxanikwn-kai-synergasias-mhxanikwn-dwdekanhsoy-sthn-kw-gia-tis-ekloges-sto-tee)


        Ολοκληρώθηκε στην Κω η καταμέτρηση των ψηφοδελτίων για την ανάδειξη νέας διοίκησης στο ΤΕΕ τμήμα Δωδεκανήσου. Σύμφωνα με πληροφορίες της «Ρ» οι παρατάξεις «Δημοκρατική Κίνηση Μηχανικών» και «Συνεργασία Μηχανικών Δωδεκανήσου» ισοψήφησαν και ακολούθησαν οι παρατάξεις «Δωδεκανησιακή Πρωτοβουλία Μηχανικών», «Πανεπιστημονική –Δημοκρατική Πενεπιστημονική Κίνηση Μηχανικών» και «Μηχανικών Ένωσις».
      
Τα αποτελέσματα ( ανεπίσημα) είναι τα εξής:
«Δημοκρατική Κίνηση Μηχανικών» : 32 ψήφοι«Συνεργασία Μηχανικών Δωδεκανήσου» : 32 ψήφοι
«Δωδεκανησιακή Πρωτοβουλία Μηχανικών» : 27 ψήφοι
«Πανεπιστημονική –Δημοκρατική Πενεπιστημονική Κίνηση Μηχανικών» : 3 ψήφοι
«Μηχανικών Ένωσις»: 0 ψήφοι
Σημειώνεται ότι στις εκλογές που έγιναν σήμερα σε επίπεδο νομού ψήφισαν συνολικά 696 μέλη του ΤΕΕ αριθμός ελαφρώς αυξημένος σε σχέση με την προηγούμενη εκλογική αναμέτρηση.
