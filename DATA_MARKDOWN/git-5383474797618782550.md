
# ГИБДД разрешило оформлять мелкие ДТП со смартфона

Published at: **2019-11-01T08:54:52+00:00**

Author: **Артём Баусов**

Original: [iPhones.ru](https://www.iphones.ru/iNotes/rossiyanam-razreshili-oformlyat-dtp-cherez-smartfony-11-01-2019)

С сегодняшнего дня в России в тестовом режиме запускается услуга оформления ДТП онлайн. Нужно только приложение для смартфона.
Одобрение дали ГИБДД и Российский союз автостраховщиков (РСА). Оформить автодорожное происшествие можно через бесплатную программу Помощник ОСАГО, в которой заполняется электронная версия европротокола.
Информация об участниках вносится автоматически из баз данных РСА и Госуслуг. Пользователям потребуется только сфотографировать повреждения и приложить фотографию схемы аварии. После этого появится QR-код, который отсканирует второй водитель для подтверждения.
Функция сейчас доступна в Москве, Санкт-Петербурге, Подмосковье, Ленинградской области и Татарстане. С 2021 года приложение заработает по всей России. [Известия]
