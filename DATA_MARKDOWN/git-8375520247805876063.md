
# Російського боксера Ковальова госпіталізували після жорстокого нокауту

Published at: **2019-11-03T09:24:52+00:00**

Author: ****

Original: [ГЛАВКОМ](https://glavcom.ua/sport/news/rosiyskogo-boksera-kovalova-gospitalizuvali-pislya-zhorstokogo-nokautu--637328.html)

Російського боксера Сєргєя Ковальова доправили до лікарні після жорстокого нокауту в бою проти мексиканця. Сауля Канело Альвареса.
Мексиканець переміг росіянина в 11-му раунді. Альварес відібрав у Ковальова чемпіонський титул у напівважкій вазі за версією WBO.
Наразі інформація про стан Ковальова не повідомляється. Промоутер Кеті Дува повідомив, що госпіталізація викликана міркуваннями безпеки для здоров’я спортсмена.
