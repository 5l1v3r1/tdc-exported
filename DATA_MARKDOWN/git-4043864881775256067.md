
# ІДІЛ зберегла присутність по всьому світу —  Держдеп

Published at: **2019-11-02T00:13:32+00:00**

Author: ****

Original: [Українська правда](https://www.pravda.com.ua/news/2019/11/2/7230779/)

Терористична організація "Ісламська держава" зберегла свою присутність по всьому світу завдяки розгалуженим мережним структурам і осередкам.
Джерело; щорічний звіт Державного департаменту США, передає Reuters
"Попри те що ІДІЛ втратила майже всю свою фізичну територію, угруповання довело свою здатність адаптуватися, особливо щодо зусиль, спрямованих на те, щоб надихати або керувати послідовниками через інтернет", - заявив координатор Держдепартаменту США з боротьби з тероризмом Натан Сейлс.
За його словами, переможені під час військових дій в Сирії та Іраку бойовики ІДІЛ "відправилися в треті країни, де створюють нові загрози".
