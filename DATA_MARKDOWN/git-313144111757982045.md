
# 용현지-임경진, 아시아 3쿠션 여자선수권 4강 진출

Published at: **2019-11-03T02:02:24+00:00**

Author: **스포츠조선=이원만 기자**

Original: [Chosun](https://news.chosun.com/site/data/html_dir/2019/11/03/2019110300351.html)

입력 2019.11.03 11:02
