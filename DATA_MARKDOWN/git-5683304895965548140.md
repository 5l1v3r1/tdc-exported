
# വൈദ്യുതവേലി കടക്കാൻ ആനയുടെ തന്ത്രപരമായ നീക്കം; ദൃശ്യങ്ങൾ കൗതുകമാകുന്നു!

Published at: **2019-11-06T10:16:00+00:00**

Author: **മനോരമ ലേഖകൻ**

Original: [ManoramaOnline](https://www.manoramaonline.com/environment/green-heroes/2019/11/06/clever-elephant-breaks-electric-fence-carefully-walks-over-it-to-go-to-other-side.html)

ആനകൾ പൊതുവെ ബുദ്ധിയുള്ള ജീവികളാണ്.അങ്ങനെയൊരു ബുദ്ധിമാനായ കാട്ടാനയുടെ ദൃശ്യങ്ങളാണ് ഇപ്പോൾ സമൂഹമാധ്യമങ്ങളിൽ വ്യാപകമായി പ്രചരികുന്നത് .ഐഎഫ്എസ് ഉദ്യോഗസ്ഥനായ സുശാന്ത നന്ദയാണ് ഈ അപൂർവ ദൃശ്യങ്ങൾ ട്വിറ്ററിലൂടെ പങ്കുവച്ചത്.
