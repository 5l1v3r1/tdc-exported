
# Беглов «ВКонтакте» поздравил петербуржцев с Днем народного единства

Published at: **2019-11-04T11:08:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/739620-beglov-vkontakte-pozdravil-peterburzhcev-s-dnem-narodnogo-edinstva)

Губернатор Петербурга Александр Беглов поздравил жителей города с Днем народного единства. Поздравление он опубликовал на своей странице в соцсети «ВКонтакте».
