
# Студенты провели в Каменском благотворительную акцию

Published at: **2019-11-06T06:35:00+00:00**

Author: ****

Original: [dndz.tv](http://dndz.tv/article-30044.html)

Студенты ПГМК г. Каменское (г. Днепродзержинска) для помощи военным провели благотворительную ярмарку.Акция была направлена для приобретения лекарств бойцам, находящимся на реабилитации в военном госпитале г. Днепр.Колледж металлургов проводит такие акции пять лет.К проведению акции присоединяются неравнодушные граждане – школьники, жители микрорайона:- на ярмарке продавали сладости;- реализовали созданные своими руками изделия;- для гостей организовали мастер-классы различных направлений.Лекарства передали бойцам.
