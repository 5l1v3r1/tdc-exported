
# とろサ久保田、和牛水田にそっくり診断に「なんでやねん」

Published at: **2019-11-03T19:19:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147968/full/)

　お笑いコンビ・とろサーモンの久保田かずのぶが2日、自身のインスタグラムを更新。“そっくり診断”の結果が和牛の水田信二だったことがわかると「なんでやねん」と猛ツッコミを入れた。 　そっくり診断の結果を披露しながら久保田は「なんでやねん。なんで芸人になるねん。外国の俳優とかであれや。みんなが盛り上がってるやつと俺だけ違うねん。俺が素人みたいになってるやん」とボヤき連発。続いての寝転ぶショットではオダギリジョーとなり「2枚目はそれで問題やねん。記者会見で謝罪レベルや」とつぶやいた。 　久保田は「なんやこのアプリ。おれ違うやつやってんかな」と納得いない様子で「パフェ食べて機嫌なおそう。NGK単独やばい事するよ」と予告していた。 　この投稿にコメント欄では「間にシソンヌ長谷川さん入ってます」「共通しているのは、2人とも面白い！」など大盛り上がりとなっていた。
