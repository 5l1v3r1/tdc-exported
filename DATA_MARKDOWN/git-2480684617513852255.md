
# Матч закончился победой ФК «Тамбов»

Published at: **2019-11-03T00:00:00+00:00**

Author: ****

Original: [Тамбов-информ](http://www.taminfo.ru/tambov_novosti/43798-match-zakonchilsya-pobedoy-fk-tambov.html)

На выезде наша футбольная команда «Тамбов» одержала победу в матче с ФК «Сочи» со счетом 2:1. На 41-й минуте Михаил Костюков забил, через три минуты Кристиан Нобоа из ФК «Сочи» выровнял счёт. Во втором тайме горожане забили ещё один гол на 56-й минуте, исход матча решил мяч Георгия Мелкадзе.
