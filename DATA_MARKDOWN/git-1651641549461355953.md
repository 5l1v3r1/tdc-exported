
# Big news: Microsoft and Cisco announce interoperability between Teams Rooms and Webex Room devices

Published at: **2019-11-04T15:00:00+00:00**

Author: **Pradeep**

Original: [MSPoweruser](https://mspoweruser.com/big-news-microsoft-and-cisco-announce-interoperability-between-teams-rooms-and-webex-room-devices/)

Today, Microsoft and Cisco made a huge announcement. Both the companies are working on interoperability between Microsoft Teams Rooms and Phone System with Cisco Webex Room devices and IP voice gateways respectively. This interoperability between Microsoft and Cisco will simplify conference room and IP voice gateway deployment options for organizations around the world.
Highlights of this partnership:
Products based on this partnership is expected to be available in early CY20.
