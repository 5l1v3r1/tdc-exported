
# Порядка одного триллиона тенге нужно для благоустройства перспективных сел

Published at: **2019-11-06T12:16:00+00:00**

Author: **Арыстан Алангожин**

Original: [Тотал](https://total.kz/ru/news/gossektor/poryadka_1_trilliona_tenge_nuzhno_dlya_blagoustroistva_perspektivnih_sel_date_2019_11_06_11_43_35)


        Правительство отобрало более трех тысяч наиболее перспективных сел для финансовых вливаний.
      
Заместитель премьер-министра Казахстана Роман Скляр рассказал, как правительство намерено решать насущные проблемы в казахстанских селах, передает корреспондент Total.kz.
Для исправления ситуации, по его словам, нужно применить комплексный подход в развитии села.
«В настоящее время отобрано 3477 сел с потенциалом развития, из которых 1150 опорные и 2327 спутниковые, включая 200 приграничных сел. В этих селах проживает 6,6 млн человек, или 85% сельского населения, развитие которых будет приоритетным», — добавил Скляр.
Как объяснил зампремьера, это села с ежегодной динамикой роста численности населения, валовой добавленной стоимости животноводства и растениеводства, ростом занятости населения, расположены вблизи крупных населенных пунктов. Кроме того, Скляр назвал сумму, которая будет нужна для реализации планов по развитию этих перспективных сел.
