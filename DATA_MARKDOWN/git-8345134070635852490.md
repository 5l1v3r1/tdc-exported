
# Dica do Dia: Como activar a opção “Não Monitorizar” no Microsoft Edge para Windows 10

Published at: **2019-11-06T18:01:12+00:00**

Author: ****

Original: [PCGuia](https://www.pcguia.pt/2019/11/dica-opcao-monitorizar-microsoft-edge/)

Para activar a opção “Não monitorizar” no Microsoft Edge para Windows 10, clicar no ícone do menu, situado no canto superior direito da interface do browser. De seguida, localizar a categoria “Privacidade e segurança”. Por fim, clicar no slider da opção “Enviar pedidos de Não Monitorizar”.
