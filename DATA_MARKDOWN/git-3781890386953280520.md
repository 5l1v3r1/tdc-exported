
# 'เปา-เปาวลี'ควง'เอิร์ธ'ลั่นระฆังวิวาห์ จัดพิธีอบอุ่นเรียบง่าย

Published at: **2019-11-02T03:40:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/entertainment/739593)


        นักร้องสาว "เปา เปาวลี" ได้ฤกษ์เข้าพิธีมงคลสมรสกับหวานใจหนุ่ม "เอิร์ธ" เป็นที่เรียบร้อยแล้ว โดยจัดพิธีแบบเรียบง่ายที่จังหวัดสุพรรณบุรี เพื่อนนักร้องร่วมยินดีแน่น
      
ถือฤกษ์ดีเช้าวันนี้ (2 พฤศจิกายน 2562) เข้าประตูวิวาห์เป็นที่เรียบร้อยแล้ว สำหรับนักร้องสาว "เปา-เปาวลี" กับแฟนหนุ่ม "เอิร์ธ-กานต์ กิจเจริญ" ลูกชาย "ซูโม่กิ๊ก เกียรติ กิจเจริญ" หลังคบหาดูใจมา 8 ปี ซึ่งงานแต่งจัดขึ้นแบบเรียบง่ายที่ อ.ดอนเจดีย์ จ.สุพรรณบุรี โดยมีพิธีแห่ขันหมาก พิธีสงฆ์  สวมแหวน รดน้ำสังข์ตามประเพณีไทย บรรยากาศเป็นไปอย่างอบอุ่นโดยมีผู้ใหญ่ เพื่อนพ้องน้องพี่ทั้งในและนอกวงการมาร่วมแสดงความยินดี  อาทิ ลำยอง หนองหินห่าว ,ก้านตอง พุ่มเงิน ,เอิ้นขวัญ วรัญญา และอีกมากมาย
ขอบคุณภาพจาก เพจ NOK LAMYONG ,เพจ ป้าโลม แม่บ้านแกรมมี่โกลด์,
