
# ไก่เดือยทองเฮ! 'เอฟเอ'ยกเลิกใบแดง'ซน'พร้อมบู๊ต่อ

Published at: **2019-11-05T18:19:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/sports/740174)


        "ไก่เดือยทอง" เฮ!! "เอฟเอ" ยกเลิกใบแดง "ซน เฮือง-มิน" พร้อมบู๊ต่อได้ ไม่ต้องติดโทษแบน 3 นัด
      
"ไก่เดือยทอง" ทอตแนม ฮอทสเปอร์ ได้รับข่าวดี เมื่อ ซน เฮือง-มิน แนวรุกตัวเก่งแดนกิมจิ จะพร้อมลงดวลแข้งกับ เชฟฟิลด์ ยูไนเต็ด, เวสต์แฮม ยูไนเต็ด และ บอร์นมัธ หลังสมาคมฟุตบอลอังกฤษ หรือ "เอฟเอ" ตัดสินใจยกเลิกโทษใบแดงของเขาจากเกมที่ สเปอร์ เสมอกับ เอฟเวอร์ตัน 1-1 เมื่อสุดสัปดาห์ที่ผ่านมาในแมตช์ดังกล่าว ซน ถูกใบแดงไล่ออกจากสนาม ในจังหวะที่เข้าปะทะกับ อังเดร โกเมส จนดาวเตะชาวโปรตุกีสเสียหลักไปปะทะเข้ากับ แซร์จ ออริเยร์ อีกทอดหนึ่ง ทำให้บาดเจ็บข้อเท้าขวาหักสยอง แต่ล่าสุด "เอฟเอ" ยกเลิกใบแดงดังกล่าว ทำให้ ซน ไม่ต้องติดโทษแบน 3 นัด พร้อมลงสนามให้กับ "ไก่เดือยทอง" ได้ตามปกติ.
