
# ఆర్టీసీ సమ్మె పిటిషన్‌పై హైకోర్టులో విచారణ ప్రారంభం

Published at: **2019-11-07T10:54:13+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=948786)

హైదరాబాద్: ఆర్టీసీ సమ్మె పిటిషన్‌పై హైకోర్టులో విచారణ ప్రారంభమైంది. చీఫ్ సెక్రటరీ ఎస్కే జోషి చీఫ్ జస్టిస్ ముందు నేరుగా హాజరై వివరణ ఇస్తున్నారు. ప్రభుత్వం, ఆర్టీసీ ఎండీ, జీహెచ్ఎంసీ దాఖలు చేసిన అఫిడవిట్లపై ప్రస్తుతం హైకోర్టులో వాదనలు కొనసాగుతున్నాయి.
