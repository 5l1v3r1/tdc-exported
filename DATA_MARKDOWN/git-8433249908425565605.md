
# Wasserstoff-Offensive im Norden

Published at: **2019-11-07T17:55:45+00:00**

Author: ****

Original: [Bild](https://www.bild.de/regional/hamburg/hamburg-aktuell/bis-2035-wasserstoff-offensive-im-norden-65893400.bild.html)

Lübeck – Die fünf norddeutschen Bundesländer wollen bis 2035 eine grüne Wasserstoffwirtschaft aufbauen.
Das haben die Wirtschafts- und Verkehrsminister von Schleswig-Holstein, Meck-Pomm, Hamburg, Niedersachsen und Bremen beschlossen. Bis zum Jahre 2025 sollen im Norden mindestens 500 Megawatt Leistung zur Erzeugung von Wasserstoff mit Hilfe von erneuerbaren Energien realisiert sein.

        Damit könnten rechnerisch 151 000 Autos betankt werden, so Schleswig-Holsteins Wirtschaftsminister Bernd Buchholz (58, FDP).
