
# Милнер: Агар юрагингизда муаммо бўлса, Ливерпульга мухлислик қилишни маслаҳат бермайман

Published at: **2019-11-03T11:56:02+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/milner-agar-yuragingizda-muammo-bolsa-liverpulga-muxlislik-qilishni-maslahat-bermayman)

Мерсисайдликлар 87-дақиқада ҳам 0:1 ҳисобида мағлубятга учрашаётган эди, аммо ғалабани илиб кетишга муваффақ бўлишди.
"Соғлигингиз борасида огоҳлантириш. Агар юрагингизда муаммо бўлса, "Ливерпуль"га мухлислик қилишни маслаҳат бермайман" - деб ёзди Милнер "Твиттер"даги саҳифасига.

        
          АПЛ турнир жадвали ва тўпурарлари билан танишинг.
