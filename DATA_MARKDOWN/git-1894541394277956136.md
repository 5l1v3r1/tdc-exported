
# Руководство Николаевской таможни создало схему для миллионных злоупотреблений - СБУ

Published at: **2019-11-02T20:07:00+00:00**

Author: ****

Original: [ОстроВ](https://www.ostro.org/general/criminal/news/578926/)

Служба безопасности Украины разоблачила должностных лиц Николаевской таможни Государственной фискальной службы на масштабных злоупотреблениях. Об этом сообщает пресс-центр СБУ.
Установлено, что в течение 2018-2019 годов руководство областной таможни совместно с начальниками нескольких таможенных постов систематически нарушали отечественное антидемпинговое законодательство в интересах коммерческих структур. При ввозе из Европы минеральных удобрений должностные лица безосновательно занижали налоговые ставки, в результате чего государственный бюджет недополучил почти 10 млн. гривен.
По результатам санкционированных обысков правоохранители изъяли документацию, подтверждающую масштабные злоупотребления.
В рамках уголовного производства, открытого по ч. 2 ст. 364 (злоупотребление властью или служебным положением) Уголовного кодекса Украины, проводятся следственные действия для установления всех обстоятельств дела.
Мероприятия по разоблачению проходили совместно со следователями территориального управления ГБР, расположенного в Николаеве, - говорится в сообщении.
