
# La versión de YouTube para ordenadores y tablets estrena una novedad muy esperada

Published at: **2019-11-07T00:00:00+00:00**

Author: **Diario Tecnología**

Original: [Diario Tecnología](https://diariotecnologia.es/la-versin-de-youtube-para-ordenadores-y-tablets-estrena-una-novedad-muy-esperada)

YouTube estrena nuevas e interesantes funcionalidades para la versión de ordenadores y tablets iOS y Android. Dentro de poco llegarán también a los móviles.
Google ha remodelado la página de Inicio de YouTube y ha añadido nuevas funciones muy demandadas por los fans, como añadir vídeos a la cola de reproducción. También a partir de ahora resulta más sencillo encontrar nuevos vídeos y descubrir de qué van, antes de abrirlos.
Como podemos ver en el siguiente GIF, la pantalla de inicio ha mejorado la visualización, aumentado el tamaño de las diapositivas para ver con más claridad el contenido del vídeo. Esto reduce las columnas de vídeos de cinco a cuatro y elimina algunas de las listas de recomendaciones, pero se mantienen las más importantes, como las noticias de última hora, o las mezclas de música.
