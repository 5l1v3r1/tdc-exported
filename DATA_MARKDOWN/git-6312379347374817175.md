
# Угнанный BMW столкнулся с иномаркой на западе Москвы

Published at: **2019-11-03T17:08:20+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-03/ugnannyy-bmw-stolknulsya-s-inomarkoy-na-zapade-moskvy)

Угнанный BMW столкнулся с автомобилем Skoda на западе Москвы. Авария произошла на улице Академика Анохина. 
По предварительным данным, в результате ДТП пострадал водитель Skoda. 
Обстоятельства выясняются.
