
# Ноябрьский заплыв кончился для петербуржца моргом. Его тело достали у берегов Крестовского острова

Published at: **2019-11-04T08:46:00+00:00**

Author: ****

Original: [Фонтанка.ру](https://www.fontanka.ru/2019/11/04/011/)

Погибшего в трусах вытащили спасатели ранним утром 3 ноября на территории ЦПКиО в Петроградском районе Петербурга.
По данным ИА «Журналистские расследования» (АЖУР), утопленника заметил охранник парка в семь утра. Тело лежало в Средней Невке, между Крестовским и Елагиным островами. Каких-либо признаков насилия на его теле обнаружено не было, из всех примет – рост под 180, шрам от аппендицита и синие трусы Adidas. Его одежду по берегам найти не удалось.
