
# Казахстан предложил создать в ШОС совет по цифровой экономике

Published at: **2019-11-02T06:17:00+00:00**

Author: ****

Original: [Tengrinews.kz](https://tengrinews.kz/kazakhstan_news/kazahstan-predlojil-sozdat-shos-sovet-tsifrovoy-ekonomike-383115/)

Казахстан предложил создать в ШОС совет по цифровой экономике, передает Tengrinews.kz со ссылкой на РИА Новости. 
Казахстан предлагает создать в Шанхайской организации сотрудничества (ШОС) совет по развитию цифровой экономики, в рамках которого страны объединения могли бы сотрудничать при внедрении новых технологий, таких как 5G и блокчейн, сообщил премьер-министр Казахстана Аскар Мамин.
"Предлагается создать совет по развитию цифровой экономики, который мог бы стать площадкой для развития взаимовыгодного партнерства по внедрению в наших странах таких новых технологических элементов, как 5G, "умные города", большие данные, блокчейн, цифровые активы, а также строительство и размещение data-центров по развитию транзита данных, участию в глобальном рынке цифровых услуг", - сказал Мамин на заседании совета глав правительств ШОС в Ташкенте.
