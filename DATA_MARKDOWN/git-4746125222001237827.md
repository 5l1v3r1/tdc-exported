
# 嵐・相葉雅紀×松本潤、渋谷駅をジャック ほっと甘い表情で魅せる

Published at: **2019-11-04T10:00:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147713/full/)

　人気グループ・嵐の相葉雅紀と松本潤がそれぞれイメージキャラクターを務める『明治おいしい牛乳』と『明治ミルクチョコレート』の特大サイズ広告が東京・渋谷駅にて掲出される。また、11月9日発行の読売新聞では2人を起用した『明治おいしい牛乳』『明治ミルクチョコレート』の広告を見開きで掲載する。

        【写真】その他の写真を見る
      
　広告では肌寒くなってきたこの季節にぴったりな、ホットミルクとチョコレートでほっと温まる2人の甘い表情が印象的。別々の空間にいながら机で向かいあっているような構図となっている。なお渋谷ジャックは10日まで東急田園都市線渋谷駅のコンコースB2階道玄坂ハッピーボードに登場する。 　また5日からは期間限定で、それぞれのオリジナル壁紙が各ブランドサイトにてダウンロード可能となる。
