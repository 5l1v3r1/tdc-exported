
# 傑尼斯天團嵐出道20週年 將旋風訪台謝粉絲

Published at: **2019-11-04T22:24:00+00:00**

Author: **https://www.facebook.com/cnanewstaiwan, 中央通訊社**

Original: [中央社 CNA](https://www.cna.com.tw/news/firstnews/201911040302.aspx)

（中央社記者楊明珠東京4日專電）日本「偶像製造機」傑尼斯事務所旗下的5人組天團「嵐」為紀念出道20週年，昨天在東京舉行記者會，宣布將於9日到11日間造訪雅加達、台北等城市，感謝歌迷20年來的支持。
「嵐」（ARASHI）由大野智、櫻井翔、相葉雅紀、二宮和也、松本潤5人組成，被認為比他們的5名師兄組成的天團SMAP還要吸金。SMAP於2016年底解散，今年初「嵐」宣布將於明年12月31日休團，2021年起各自單飛，成員的動向備受矚目。
「嵐」昨天上午在全球直播記者會表示，「嵐」將邁入第21年，今後的主題訂為「重生」（Reborn）。松本潤說：「心中想了很多想挑戰的事，今後也希望與大家一起向前衝。」
松本潤也說：「我們想比以前更接近粉絲，將我們的活動同步傳達給大家。」
「嵐」宣布開設臉書（Facebook）、IG（Instagram）、推特 （Twitter）、TikTok、微博（weibo）5大社群帳號，昨天上午10時同步開通，從「嵐」的出道曲A・RA・SHI到單曲BRAVE等64曲解禁外，首度進行數位播放的單曲Turning Up於日本時間昨晚7時開播。Turning Up的MV於昨晚9時首播，「嵐」的成員還進聊天室與粉絲互動。
