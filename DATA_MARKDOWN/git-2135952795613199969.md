
# Bon plan : Le OnePlus 7T a seulement 479,99 € chez Rakuten !

Published at: **2019-11-06T10:18:25+00:00**

Author: **David Peruch**

Original: [Planet Sansfil](https://www.planet-sansfil.com/telephonie/bon-plan-le-oneplus-7t-a-seulement-47999-e-chez-rakuten/)

Bonne nouvelle pour tous ceux qui veulent s’offrir le dernier smartphone sorti chez OnePlus, le OnePlus 7T. en effet, la bonne nouvelle c’est qu’il est proposé à 479,99 € au lieu de 549€ sur le site de Rakuten.
Alors oui, il n’aura pas fallu longtemps pour voir arriver les premières promos intéressantes concernant le OnePlus 7T. Aujourd’hui, l’offre arrive directement depuis le site Rakuten qui propose le OnePlus 7T en édition Double SIM Bleu Glacier au prix de 479,99 €.
Par ailleurs si vous êtes membre du Club R vous aurez droit non seulement à une remise de 20€ supplémentaire directement mais pas seulement.
En effet, le site vous offre également avec le code « ONEPLUS + » 114.9975€ € remboursés en superpoints. Des superpoints que vous pourrez utiliser lors de vos prochains achats.
