
# Убийство экс-мэра Киселевска: два человека задержаны

Published at: **2019-11-05T05:21:00+00:00**

Author: **Информационная служба Накануне.RU**

Original: [Накануне.ru](https://www.nakanune.ru/news/2019/11/05/22557256/)

Два человека задержаны после убийства экс-мэра Киселевска Сергея Лаврентьева.
Как сообщили Накануне.RU в Следственном комитете РФ, это 49- и 35-летний жители города Белово. По версии следствия, один из них исполнял роль водителя. Он привез подельников к дому Лаврентьева, ожидал их в километре от места преступления.
Один из подозреваемых арестован. По второму задержанному вопрос об избрании меры пресечения будет решаться сегодня.
Приоритетной версией преступления остается разбойное нападение, отметили в СКР.
