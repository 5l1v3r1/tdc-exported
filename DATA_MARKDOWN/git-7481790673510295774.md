
# 「たまたま見つけた」女性の首を絞め、太ももかむ…男逮捕

Published at: **2019-11-03T05:11:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191103-OYT1T50116/)

　警視庁四谷署は２日、新潟県阿賀野市、会社員の男（３０）を強制わいせつ致傷容疑で逮捕した。
　発表によると、男は８月１７日午後６時頃、東京都新宿区の路上で、１０歳代女性の首を背後から絞め付けて失神させたうえ、倒れた女性の太ももをかみ、全治１週間のけがを負わせた疑い。「たまたま見つけた女性に性欲を抑えられなかった」と容疑を認めている。
　事件当時、男は都内に住んでおり、防犯カメラの映像から特定された。
