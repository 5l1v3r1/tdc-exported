
# 歌手・谷村奈南、会社設立を報告「健康や福祉活動など、さまざまな想いを形に」

Published at: **2019-11-05T11:23:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148030/full/)

　歌手の谷村奈南（32）が5日、自身のブログを更新。健康や国内外における福祉活動など、さまざまな想いを形にしていき、発信していく会社『ONE08株式会社』（ワンオーエイト）を設立したことを報告した。 　谷村は、歌手として、2008年には日本レコード大賞優秀作品賞を受賞し、近年は女優としても活躍。しかし、約3年前からヨガを本格的に学び始め、その後、ヨガインストラクターの国際ライセンスである「全米ヨガアライアンス認定RYT200」を取得するなどしている。 　ブログでは「まずはじめに、芸能活動を長い間お休みした後もいつも応援してくれている一人一人に、感謝の気持ちでいっぱいです。ありがとうございます」と伝え「これからは、芸能だけではなく、Wellnessや国内外における福祉活動など、さまざまな想いを形にしていきたい、発信していきたい想いを込め、ONE08株式会社（ワンオーエイト）を設立しました」と発表。 　また、公式サイトでは、会社のロゴとともに「心、身体、魂のWELLNESSへ」とし「忙しいこの現代社会だからこそ、今一度自分の内側に意識を持ち、今の自身の状態を知り、さらには本来の自分という深い場所へ。無意識に溜まった毒素や、ブロックしていたエネルギーを解放していくと、見える世界が変わってくる。自分が本当の意味で満たされると、外の世界は輝き出す。皆様の身体だけでなく、心も健やかになれることを願っています」とメッセージしている。
