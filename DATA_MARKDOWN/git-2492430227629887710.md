
# File voor Van Brienenoordbrug richting Kuip door storing

Published at: **2019-11-07T19:58:11+00:00**

Author: ****

Original: [RTV Rijnmond](https://www.rijnmond.nl/nieuws/188261/File-voor-Van-Brienenoordbrug-richting-Kuip-door-storing)

De brug heeft vanaf 20:20 uur een klein kwartier in storing gestaan, zegt een woordvoerder van Rijkswaterstaat. Een monteur heeft de storing kunnen verhelpen.
Dat was slecht nieuws voor supporters van Feyenoord. De club speelt donderdagavond in De Kuip tegen Young Boys. Rijkswaterstaat adviseerde om te rijden via de Beneluxtunnel.
