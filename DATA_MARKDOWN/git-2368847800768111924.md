
# NBA里納德38分12籃板 破馬刺不敗金身

Published at: **2019-11-03T08:05:47+00:00**

Author: ****

Original: [NTD 新唐人電視台](https://www.ntdtv.com/b5/2019/11/03/a102699263.html)

【新唐人北京時間2019年11月03日訊】NBA快船1日對戰馬刺，披上快艇戰袍的里納德（Kawhi Leonard）對戰老東家，上場34分鐘就轟了38分、12籃板和4抄截，終場快船以103：97擊退馬刺，終止黑衫軍開季3連勝。
里納德一開賽就連下8分，首節打完，快船以23：24落後馬刺1分；第二節開始，里納德持續攻擊，加上哈瑞爾（Montrezl Harrell）、威廉斯（Lou Williams）合力掃下20分，船艇以50：48在上半場結束時取得領先。
兩隊第三節持續拉鋸，末節德羅贊（DeMar DeRozan）鍥而不捨猛追分，單場進帳29分，仍無法抵擋快船攻擊，終場馬刺便以6分之差吞敗。
