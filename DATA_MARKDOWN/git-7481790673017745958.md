
# 浦和・大槻監督に制裁金１００万円…相手選手突き飛ばして退場

Published at: **2019-11-02T12:31:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/soccer/20191102-OYT1T50255/)

　１日のＪ１・鹿島戦で相手選手を突き飛ばして退場処分を受けた浦和の大槻監督について、戒告処分と１００万円の制裁金を科すとクラブが発表した。大槻監督はクラブを通じ、「あの行為は間違いでした。鹿島の選手並びにチーム関係者、すべての浦和レッズに関わる人たちに謝罪します」とコメントを出した。
