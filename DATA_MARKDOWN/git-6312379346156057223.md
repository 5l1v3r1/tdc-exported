
# Внесение ФБК в реестр НКО-иноагентво признано законным

Published at: **2019-11-01T08:01:50+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-01/vnesenie-fbk-v-reestr-nko-inoagentvo-sud-priznal-zakonnym)

Суд в Москве признал законными действия Минюста РФ по внесению основанного оппозиционером Алексеем Навальным Фонда борьбы с коррупцией (ФБК) в реестр некоммерческих организаций - иностранных агентов.
"В удовлетворении административного иска ФБК, который настаивал на признании незаконными действий Минюста РФ, отказать", - цитирует "Интерфакс" судью.
Суд таким образом не нашел оснований для исключения ФБК из этого реестра.
