
# В Краматорске появилась арт-инсталляция из морских контейнеров (ФОТО)

Published at: **2019-11-04T13:04:00+00:00**

Author: ****

Original: [ОстроВ](https://www.ostro.org/donetsk/society/news/578974/)

Арт-инсталляция, состоящая из пяти сваренных между собой морских контейнеров, появилась на входе в парк «Юбилейный» в Краматорске.
Автором инсталляции под названием «Культурная независимость» стал украинский художник «уличной волны» из Киева Дима Фатум. 
Арт-объект стилизован под граффити. В рисунках на морских контейнерах автор «зашифровал» картины всемирно известных художников – Леонардо да Винчи, Анри Матисса, Микеланджело.
