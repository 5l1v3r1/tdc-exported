
# Дорожники Сыктывкара насчитали 50 самосвалов вывезенного снега

Published at: **2019-11-07T17:18:00+00:00**

Author: ****

Original: [БНК](https://www.bnkomi.ru/data/news/102454/)

За минувшие сутки с улиц Сыктывкара вывезено 800 «кубов» снега. Этот объем может вместиться в 50 самосвалов, сообщает пресс-служба мэрии Сыктывкара.
По информации муниципального предприятия «Дорожное хозяйство», суммарно техника очистила от свежевыпавшего снега 1200 километров дорог (при общей протяженности улично-дорожной сети города в 306 км.), так как по ряду направлений спецтехника проходилась по три раза и более.
Сейчас снегоуборщики продолжают ликвидировать последствия снегопада: с улиц вывозятся уплотненные снежные массы, расширяют проезжую часть дорог и очищают тротуары.
