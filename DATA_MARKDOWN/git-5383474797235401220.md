
# Вышла iOS 13.3 beta 1 для разработчиков. Что нового

Published at: **2019-11-05T21:08:44+00:00**

Author: **Артём Баусов**

Original: [iPhones.ru](https://www.iphones.ru/iNotes/vyshla-ios-133-beta-1-chto-novogo-11-05-2019)

Apple выпустила iOS 13.3 beta 1 для авторизованных разработчиков.
Номер сборки — 17C5032d.
Судя по описанию, в этом обновлении исправили ошибки и повысили общую стабильность операционной системы. На момент написания статьи больше не было информации.
Что нового:
• исправили работу оперативной памяти• появился тумблер для отключения клавиатуры анимодзи
Размер апдейта — около 4 ГБ. Скачать iOS 13.3 beta 1 можно уже сейчас «по воздуху». Нужен только профиль для разработчиков.

        Легкого апдейта!
