
# コレステロール値の高さが決定打に…45歳の英語講師が“重力に負けない”美ボディ披露

Published at: **2019-11-03T13:00:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147854/full/)

　日本最大規模の健康美コンテスト「ベストボディ・ジャパン（以下BBJ）2019」の「ジャンル別大会」が先ごろ、東京・ニューピアホールにて開催され、美ボディ自慢の男女が多数集結。ドクター、ナース、自衛隊、パーソナルトレーナーといった職業別部門に加え、グラマーやインフルエンサーなど全12部門行われ、健康的な肉体のかっこ良さ、美しさを競い合った。

        【写真】その他の写真を見る
      
　バストやヒップの形・大きさ・きれいさが整った、女性らしいメリハリのあるスタイルが評価されるグラマー部門で準グランプリに選ばれたのは、英語講師として働く榎本理栄子さん。40歳で筋トレを始めて、今年で5年になるそうだ。 「筋トレのきっかけは、ダイエットです。学生時代から運動とは無縁で、たくさん食べていたらどんどん太ってしまって…。さらに病院で『コレステロール値が高いので、このままだと薬を飲まなきゃいけなくなるよ』と言われたことが決定打になり、まずは走ることから始めました」 　「当時は今より、12〜13キロは太っていた」と振り返る榎本さん。走り始めてからは、月2〜3キロペースで体重は減少し、健康上の不安は解消したが、“40代”という年齢を考えたという。「ただ痩せるだけでなく、より重力に負けない体を作りたいなと思い、パーソナルトレーナーのもとで筋トレを開始しました」 　また「BBJに出る」という目標を作ったことも、やる気を大きく後押ししたという。「ジムでは、ずんどうにならないようパーツごとに目標とするサイズを決めて、負荷をかけながらトレーニングしてきました。（筋トレを）始める前と現在では、見た目が明らかに変わりましたね。一番の変化は、お尻が上がったことかな。一番のチャームポイントになっています」 　「30代より今の方が、健康にもボディラインにも自信が持てるようになった」と笑顔で話す榎本さん。次の目標は、17日に開催されるBBJの日本大会で上位に入賞することだ。 「せっかくいただいたチャンスなので、ベストな状態で臨めるよう、精一杯頑張りたい。そして自分がどれだけキレイになれるのか、挑戦できればと思います」
