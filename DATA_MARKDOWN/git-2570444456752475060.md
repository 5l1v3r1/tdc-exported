
# На будущий саратовский пляж начали завозить песок

Published at: **2019-11-05T11:13:00+00:00**

Author: ****

Original: [Саратов 24](https://saratov24.tv/news/na-budushchiy-saratovskiy-plyazhe-nachali-zavozit-pesok/)

Работы по строительству новой набережной активизированы по двум направлениям: завозят песок по всей территории строительства будущего пляжа; параллельно с этим идет устройство шпунтов нижнего мола.По словам подрядчика, сегодня основная задача – завершить гидротехническое сооружение и идти строительством в сторону верхней части пляжа к ул. Шелковичной. "На сегодняшний день работы по пляжной части новой набережной возобновились. Погодные условия будут только ухудшаться, поэтому стоит задача ускориться, и успеть сделать максимально возможный объем до конца этого года", - поставил задачу Валерий Радаев.
