
# 大眼正妹私售自拍無碼片被捕　網友肉搜私下竟是甜美鄰家女孩

Published at: **2019-11-02T09:26:00+00:00**

Author: **網路創意組**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191102web007/)

雖然日本情色產業十分發達，許多女性會投入拍攝成人影片的行業，但日本有法令規定，若作品未經審查私自販售，就會處法而面臨刑責。日本四國愛媛縣警方日前便破獲一起無碼A片販賣案件，逮捕一名24歲大眼正妹，她被控自拍無碼色情片並在網路販售，賺取8,800日圓（約2,400新台幣），不過才賣出2片就被逮捕。
綜合日本媒體報導，遭到逮捕的正妹名為森河佳帆，容貌姣好且身材火辣，她在5月份時上網販售自製無碼A片，但沒多久就被警方在網路上巡邏時發現，愛媛縣警方迅速循線找到森河佳帆，並在她家中搜出10多片尚未出售的A片。
警方表示，森河佳帆有一份正職工作，平常在從事汽車租賃、經銷、維修業務的公司上班，公司負責人是一名年輕人，而且在同一棟大樓經營另一家公司。附近居民透露，曾看過疑似援交妹、百萬名車在大樓周遭出沒，讓警方懷疑這起私售A片的案子背後恐怕由更大的集團操控。
森河佳帆販售自製A片被逮消息曝光後，由於她有一雙明亮大眼、外型亮麗，立刻被網友肉搜，從她的臉書中可看到她穿著細肩帶衣服時上圍十分雄偉，而且形象清新樸素，如同一名甜美的鄰家女孩。
