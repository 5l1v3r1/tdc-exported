
# ロッテ、レアードと契約合意「日本一になりたいです」

Published at: **2019-11-02T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/02/0012842300.shtml)

　ロッテは２日、ブランドン・レアード内野手（３２）と来季の契約が合意に達したと発表した。２年の複数年契約とみられ、年俸は推定で２３０万ドル（約２億５０００万円）。
　レアードは「私の家族と私は２０年もロッテでプレーできることに、とても興奮しています。ペナントレースを勝ち抜いてリーグ優勝をしてＣＳを勝ち抜いて日本一になりたいです」とコメントした。
　日本ハムから移籍した今季は１３９試合に出場して３２本塁打、打点８９、打率・２４８の成績を残した。
　「個人としてはあと３７本と迫っている２００本塁打を目指していきたい。そのために私は日々、ベストを尽くして懸命にプレーをすることを皆さまにお約束します」と改めて来季に意欲を燃やした。
　「マリーンズファンは素晴らしく、来年もきっと私のスシパフォーマンスを期待してくれていると思っています。私も千葉ロッテマリーンズと幕張スシが大好きです。幕張スシ、さいＧＯ（さあ、行こうと最高の意味）」。ファンを湧かせるパフォーマンスを約束していた。
