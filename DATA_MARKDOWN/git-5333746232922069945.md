
# Пограничники: Северокорейские браконьеры не угрожают белухам из китовой тюрьмы

Published at: **2019-11-06T09:40:00+00:00**

Author: **Информационная служба Накануне.RU**

Original: [Накануне.ru](https://www.nakanune.ru/news/2019/11/06/22557426/)

Рыбаков КНДР в районе выпуска белух из китовой тюрьмы нет, млекопитающим они угрожают. Об этом сообщил РИА Новости представитель управления погранслужбы по региону.
Ранее стало известно, что оставшихся 50 белух решили выпустить не в Охотском море, а на юге Приморья в Японском море двумя судами.
После этого эксперты Greenpeace попросили российских пограничников обеспечить безопасность морских животных.
Напомним, осенью в исключительной экономической зоне РФ задержали несколько сотен рыбаков из КНДР. Несколько браконьеров арестованы, они оказали активное сопротивление российским пограничникам при задержании 17 сентября.
