
# 정경배 코치, 한화 2군 타격코치로 부임

Published at: **2019-11-02T07:04:20+00:00**

Author: **스포츠조선=나유리 기자**

Original: [Chosun](https://news.chosun.com/site/data/html_dir/2019/11/02/2019110200831.html)

입력 2019.11.02 16:04
