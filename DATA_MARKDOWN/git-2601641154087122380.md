
# Mozilla запретит всплывающие запросы о веб-пушах от сайтов в Firefox в 2020 году

Published at: **2019-11-06T12:45:19+00:00**

Author: **Лиана Липанова**

Original: [vc.ru](https://vc.ru/services/91371-mozilla-zapretit-vsplyvayushchie-zaprosy-o-veb-pushah-ot-saytov-v-firefox-v-2020-godu)

Окно с предложением подписаться на пуши придётся открывать вручную.
Mozilla в своём браузере Firefox запретит всплывающие уведомления от сайтов с просьбой подписаться на веб-пуши. Изменения появятся в 72 версии Firefox, которая выйдет в январе 2020 года. Об этом Mozilla сообщила в своём блоге.
Нововведение коснётся настольной версии браузера. Firefox перестанет показывать всплывающие окна с предложением подписаться на веб-пуши, которые пользователь обычно видит при первом посещении сайта. Вместо этого нужно будет вручную открывать такой запрос, нажав на кнопку рядом с URL-адресом.
Mozilla объясняет такое решение результатами исследования компании. За один месяц работы пользователям Firefox 63 версии было показано 1,45 млрд запросов, из которых было принято только 23,66 млн. Примерно в 500 млн случаев пользователи нажимали на кнопку «Не сейчас».
