
# Медведев поздравил россиян с Днем народного единства фотографиями самолета, улетающего из России

Published at: **2019-11-04T08:32:00+00:00**

Author: **Информационная служба Накануне.RU**

Original: [Накануне.ru](https://www.nakanune.ru/news/2019/11/04/22557220/)

Глава Правительства Дмитрия Медведев поздравил россиян с Днем народного единства, опубликовав фотографии самолет из авиаотряда "Россия", который обеспечивает перевозки первых лиц.
