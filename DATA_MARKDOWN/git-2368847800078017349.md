
# 【胡乃文開講 】中醫師70多歲不老花 分享3招搶救惡視力

Published at: **2019-11-05T07:22:49+00:00**

Author: ****

Original: [NTD 新唐人電視台](https://www.ntdtv.com/b5/2019/11/04/a102699654.html)

老花眼已經出現年輕化的趨勢，有的人才30多歲，外表看起來很年輕，眼睛卻出現看不清楚甚至有老花的現象，為什麼會這樣呢？多數人長時間使用電腦或3C產品，下了班一直滑手機或玩線上遊戲，眼睛無法得到充分的休息，導致水晶體開始變硬，彈性變差；近的東西看不清楚，這是眼睛出現老化的現象。
70多歲也不老花的胡乃文中醫師，分享穴位按摩、眼球運動和護眼飲食3招，教你在短時間內搶救視力，預防和改善老花眼。
