
# В магазинах «Вкусвилл» в Петербурге установили боксы для сбора пластиковых карт. Их будут отправлять на переработку

Published at: **2019-11-05T00:00:00+00:00**

Author: ****

Original: [«Бумага»](https://paperpaper.ru/papernews/2019/11/05/v-magazinah-vkusvill-v-peterburge-u/)

В 50 точках сети «Вкусвилл» в Петербурге появились боксы для сбора пластиковых крат.
На переработку можно сдать как банковские, так и дисконтные карты разных магазинов. Список всех петербургских точек «Вкусвилл» с боксами доступен на сайте.
В Петербурге пластиковые карты для переработки также принимают в пекарнях «Цех 85».
