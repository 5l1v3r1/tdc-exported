
# Низький рейтинг не є проблемою для Зеленського, - Загородній

Published at: **2019-11-06T11:55:00+00:00**

Author: ****

Original: [ZIK.UA](https://zik.ua/news/2019/11/06/nyzkyy_reytyng_ne_ie_problemoyu_dlya_zelenskogo__zagorodniy_1684771)


        Хоча рейтинг президента України Володимира Зеленського трохи впав - це для нього не критично. Якщо порівнювати зі Зеленським, то за цей час експрезидент Петро Порошенко більше втратив.
      
Таку думку в ефірі інформаційно-аналітичного проєкту «Чудова четвірка» на телеканалі ZIK висловив політтехнолог Тарас Загородній.
