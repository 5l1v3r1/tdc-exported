
# Μια μαύρη γάτα... καθυστέρησε τον αγώνα - Οι ηττημένοι μιλούν για γρουσουζιά - ΒΙΝΤΕΟ

Published at: **2019-11-05T09:42:00+00:00**

Author: ****

Original: [enikos.gr](http://www.enikos.gr/sports/682347/mia-mavri-gata-kathysterise-ton-agona-oi-ittimenoi-miloun-gia-gro)

Μια μαύρη γάτα εμφανίστηκε στο γήπεδο λίγο πριν από τον αγώνα των Giants με τους Cowboys. Ο αγώνας καθυστέρησε να ξεκινήσει μέχρι να απομακρυνθεί η γάτα, αλλά τώρα οι ηττημένοι μιλούν για... γρουσουζιά. Ή μήπως φταίει ότι ήταν χειρότεροι από τους νικητές;
Η γάτα έκανε την βόλτα της κατά το δεύτερο τέταρτο και ανάγκασε τον διαιτητή να διακόψει και να καθυστερήσει τον αγώνα για αρκετά λεπτά, μέχρι οι εργαζόμενοι του MetLife Stadium να καταφέρουν να πείσουν την τετράποδη εισβολέα να απομακρυνθεί.
