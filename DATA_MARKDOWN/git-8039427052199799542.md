
# రైతు భరోసా సొమ్మును స్వాహా చేసిన వైసీపీ నేతలు

Published at: **2019-11-04T21:49:24+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=946309)

ADVT
అనంతపురం: జిల్లాలోని కూడేరు మండలం కదురకుంట గ్రామంలో రైతు భరోసా సొమ్మును వైసీపీ నేతలు స్వాహా చేశారు. కౌలు రైతుల పేర్లతో రైతు భరోసా సొమ్మును స్వాహా చేశారు. రైతు భరోసాలో అక్రమాలను ఏబీఎన్ వెలుగులోకి తీసుకువచ్చింది. దీంతో వీఆర్వో ప్రభాకర్‌ను జిల్లా కలెక్టర్ సత్యనారాయణ సస్పెండ్ చేశారు.
