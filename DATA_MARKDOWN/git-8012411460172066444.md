
# حراك العسكريين: نستنكر التصفيق لبعض الكلام العدائي للجيش

Published at: **2019-11-05T20:48:47+00:00**

Author: ****

Original: [IMLebanon](https://www.imlebanon.org/2019/11/05/military-statment-zouk/)

أدان حراك العسكريين المتقاعدين “ما صدر خلال فتح الأوتوستراد عند نقطة ذوق مصبح لجهة استحضار العدو الإسرائيلي والتصفيق لبعض الكلام العدائي للوطن والجيش”.
وناشد، في بيان، “القضاء المختص لإجراء المقتضى وفقا للقوانين المرعيَّة الإجراء”.
