
# Ученые нашли метод борьбы с возрастными заболеваниями

Published at: **2019-11-05T13:57:21+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-05/uchenye-nashli-metod-borby-s-vozrastnymi-zabolevaniyami)

Ученые нашли метод лечения всех возрастных заболеваний сразу. Об этом пишет Proceedings of the National Academy of Sciences.
Исследователи разработали метод генной терапии, позволяющий увеличить продолжительность жизни. Он основан на доставке генов, связанных с долголетием, с помощью аденоассоциированного вируса AAV8.
Используя серотип AAV8 в качестве средства доставки, ученые создали отдельные конструкции генной терапии для каждого из генов (их называют трансгенами). Вирусный вектор доставляет трансген в клетку организма и встраивает его в ДНК.
Разработанная терапия помогает комплексно бороться с диабетом, ожирением, сердечной и почечной недостаточностью.
