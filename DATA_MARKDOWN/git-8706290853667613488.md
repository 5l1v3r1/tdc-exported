
# Ολυμπιακός - Μακάμπι Τελ Αβίβ 21-24 (10')

Published at: **2019-11-01T21:53:00+00:00**

Author: ****

Original: [enikos.gr](http://www.enikos.gr/sports/681740/olympiakos-makampi-tel-aviv-21-24-10)
