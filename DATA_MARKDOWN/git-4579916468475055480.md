
# Петербуржец Хаджимурат стал 14-кратным чемпионом мира по армрестлингу

Published at: **2019-11-06T10:47:00+00:00**

Author: ****

Original: [Фонтанка.ру](https://www.fontanka.ru/2019/11/06/068/)

Триумфом завершился для петербургского спортсмена Хаджимурата Золоева 41-й чемпионат мира по армрестлингу в городе Констанце (Румыния). Рукоборец из Северной столицы завоевал золотую медаль турнира в категории до 90 кг и стал 14-кратным чемпионом мира.
Всего в соревнованиях приняли участие атлеты из 54 стран планеты. Тренеры, подготовившие спортсмена, – Андрей Политов и Маирбек Золоев.
По результатам общекомандного зачета сборная команда России по армрестлингу стала победителем чемпионата мира.
