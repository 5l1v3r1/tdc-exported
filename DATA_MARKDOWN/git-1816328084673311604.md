
# டெல்லி விமான நிலைய வளாகத்தில் சந்தேகத்திற்கிடமான பை கிடந்ததால் பரபரப்பு

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [Dailythnathi.com](https://www.dailythanthi.com/News/India/2019/11/01064208/Delhi-Security-tightened-at-Terminal-3-of-Indira-Gandhi.vpf)

புதுடெல்லி,தலைநகர் டெல்லியில் உள்ள இந்திரா காந்தி சர்வதேச விமான நிலையம், உலக அளவில் மிகவும் அதிக பயணிகளை கையாளும் விமான நிலையங்களில் ஒன்றாகும். உள்நாட்டு, வெளிநாட்டு பயணிகள் வருகை காரணமாக எப்போதும் பரபரப்பாக இயங்கும் விமான நிலைய வளாகத்தில், சந்தேகத்திற்கிடமான பை ஒன்று கிடந்தது கண்டறியப்பட்டது. இதையடுத்து, இந்திரா காந்தி விமான நிலையத்தின் முனையம் 3-ல் பாதுகாப்பு பலப்படுத்தப்பட்டுள்ளது.
