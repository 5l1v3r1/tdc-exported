
# В ТиНАО по проезжей части бегал олень

Published at: **2019-11-02T08:39:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/proisshestviya/v_tinao_po_proezzhey_chasti_begal_olen/)

Короткое видео, снятое одним из водителей на проезжей части в ТиНАО, опубликовал Mash. Где именно было снято видео, не уточняется.На кадрах испуганное животное бежит по дороге, а водители замедляют движение, чтобы его не сбить.
