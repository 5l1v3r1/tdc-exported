
# தெற்கு சாண்ட்விச் தீவில் கடுமையான நிலநடுக்கம்

Published at: **2019-11-06T00:00:00+00:00**

Author: ****

Original: [Dailythnathi.com](https://www.dailythanthi.com/News/World/2019/11/06064535/IMD-An-earthquake-hit-East-of-South-Sandwitch-Islands.vpf)

புதுடெல்லி,அட்லாண்டிக் கடலின் தென்பகுதியில் அமைந்த தெற்கு சாண்ட்விச் தீவின் கிழக்கில் இன்று அதிகாலை 2.22 மணியளவில் கடுமையான நிலநடுக்கம் ஏற்பட்டு உள்ளது.இது ரிக்டரில் 6.3 ஆக பதிவாகி உள்ளது.  இதுபற்றிய தகவலை இந்திய வானிலை ஆய்வு மையம் வெளியிட்டு உள்ளது.  எனினும், நிலநடுக்கத்தினால் ஏற்பட்ட பொருட்சேதம் உள்ளிட்ட விவரங்கள் எதுவும் உடனடியாக வெளியாகவில்லை.
