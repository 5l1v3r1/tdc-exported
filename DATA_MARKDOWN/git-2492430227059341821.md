Author: ****

De file liep op tot ruim acht kilometer met een vertraging van een half uur. Rond 07:30 uur is de file nog maar drie kilometer en doen weggebruikers maar zo'n vijf minuten langer over hun reis.
Rijkswaterstaat adviseerde weggebruikers richting Bergen op Zoom om te rijden via de A13 en de A20.
