
# В Москве прошёл «Урок цифры для учителей России»

Published at: **2019-11-01T10:54:05+00:00**

Author: **Новости на «Меле»**

В Москве прошел «Урок цифры для учителей России», а конференция стала стартом образовательного проекта для школьников в 2019-2020 учебном году. Это серия необычных уроков информатики, которые проходят в течение учебного года. Новый сезон проекта стартовал с конференции для учителей информатики.
Для учителей «Урок цифры» прошел впервые, в нём поучаствовали более 150 педагогов из 14 регионов. Все они прошли обучение использованию современных цифровых технологий для популяризации уроков информатики.
Школьники, для которых в 2019 году проводится «Урок цифры», будут работать в компаниях, так или иначе связанных с технологиями. Поэтому прекрасно, что есть проект, который дает детям возможность уже сейчас узнать больше о цифровом мире и познакомиться с профессиями будущего.
Не исключено, что проект станет для кого-то отправной точкой в выборе профессии и откроет новое понимание того, как и в какую сторону развиваться. Андрей Лобанов, CEO международной школы программирования для детей «Алгоритмика».
Проект «Урок цифры» стартовал в декабре 2018 года по инициативе Министерства просвещения, Министерства цифрового развития, связи и массовых коммуникаций и организации «Цифровая экономика».
