
# หงส์ส่ง 2 ทีมแยกลงเล่นพร้อมกัน

Published at: **2019-11-01T05:28:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/sports/739430)


        "หงส์แดง" เริ่มเจรจาฟุตบอลลีกหารือความเป็นไปได้แบ่ง 2 ทีมลงเตะถ้วยคาราบาว คัพ และ ถ้วยสโมสรโลก
      
"หงส์แดง" ลิเวอร์พูล ได้เริ่มมีการพูดคุยกับฟุตบอลลีกอังกฤษ (อีเอฟแอล) แล้วถึงความเป็นไปได้ที่จะแยกส่ง 2 ทีมลงเล่นในรอบก่อนรองชนะเลิศของศึกคาราบาว คัพ วันที่ 17-18 ธ.ค. และ รอบตัดเชือกของศึกฟีฟ่า คลับ เวิลด์ คัพ ที่ประเทศกาตาร์ ในวันที่ 18 ธ.ค. ตามรายงานจาก "เดลี มิร์เรอร์"ทั้ง 2 รายการเตะในช่วงเวลาคาบเกี่่ยวกันพอดี แถมอยู่คนละทวีป อย่างไรก็ตามโปรแกรมแข่งของลิเวอร์พูลนั้นแน่นเอี๊ยดอยู่แล้ว จนไม่อาจแทรกช่วงอื่น ที่เป็นไปได้มากสุดคือ 8 ม.ค. ซึ่งเป็นสัปดาห์ของรอบรองฯ นัดแรก คาราบาว คัพ และหาก ลิเวอร์พูล ไม่ต้องเตะรีเพลย์รอบ 3 เอฟเอ คัพ ก็ยังมีช่องว่างให้เตะคาราบาว คัพ รอบรองฯ นัดสอง ก่อนชิงชนะเลิศ 1 มี.ค.ลิเวอร์พูล ต้องส่งชื่อนักเตะ 23 คนในถ้วยสโมสรโลก รวมถึงผู้รักษาประตู 3 คน หากแยกเป็นสองชุดพวกเขาเขาอาจต้องทิ้งนักเตะจำนวนหนึ่งในเกมที่วิลลา เมื่อเล่นเสร็จจึงตามไปสบทบที่กาตาร์เพื่อลงเตะรอบชิงที่ 3 หรือรอบชิงชนะเลิศ สโมสรโลก ขณะที่สตาฟฟ์โค้ชก็ต้องแยกสองชุด
