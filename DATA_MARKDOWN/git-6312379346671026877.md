
# Китай вывел три спутника на орбиту Земли

Published at: **2019-11-03T05:51:23+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-03/kitay-vyvel-tri-sputnika-na-orbitu-zemli)

Китай в воскресенье запустил ракету-носитель "Чанчжэн-4-Би" с тремя спутниками на борту.
Спутники уже успешно вышли на заданные орбиты. Запуск был осуществлен с космодрома Тайюань, сообщает ТАСС.
В частности, на орбиту выведены аппарат дистанционного зондирования, научно-исследовательский спутник Судана и космический аппарат "Тяньи-15".
Аппарат Gaofen-7 будет использоваться для картографических съемок Земли.
Кроме того, его данные пригодятся службе статистики.
