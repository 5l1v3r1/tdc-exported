
# Росстандарт запустил сервис для проверки отзываемых автомобилей по VIN-номеру

Published at: **2019-11-02T06:56:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/transport/sstandart_zapustil_servis_dlya_proverki_otzyvaemykh_avtomobiley_po_vin_nomeru/)

Росстандарт запустил интерактивный поиск для проверки автомобилей по VIN-номеру. Сервис помогает определить, подпадает ли конкретное авто под отзыв, сообщает пресс-служба ведомства.Это открытый сервис, им может воспользоваться любой посетитель сайта.В 2020 году Росстандарт запустит функцию личного кабинета для производителей транспортных средств, а автовладельцы будут получать не только информацию о наличии транспортного средства в отзывной программе, также сервис начнет отражать информацию о ремонте конкретного автомобиля.
