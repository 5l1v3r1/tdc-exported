
# В Госдуме в первом чтении приняли законопроект об учете долговой нагрузки россиян

Published at: **2019-11-07T16:35:00+00:00**

Author: **Александр Воинков**

Original: [ФедералПресс](https://fedpress.ru/news/77/economy/2360526)


        МОСКВА, 7 ноября, ФедералПресс. Законопроект позволит облегчить доступ пользователей к получению информации из БКИ. также будет разработана база для обмена сведениями между бюро.
      
Госдума в первом чтении приняла законопроект, вводящий институт квалифицированных бюро кредитных историй (БКИ). Они будут реализовывать новый механизм расчета совокупной долговой нагрузки.
Это позволит облегчить доступ пользователей к получению информации из БКИ. Помимо этого будет создана нормативная база для обмена сведениями между БКИ.
В документе говорится, что ЦБ разработает порядок расчета долговой нагрузки пользователями кредитных историй. Помимо этого у регулятора будет право определять порядок формирования сведений в кредитной истории, устанавливая требования к форматам их обмена, передает ТАСС.
Ранее эксперты назвали самых закредитованных заемщиков. На первом месте оказались охранники.
