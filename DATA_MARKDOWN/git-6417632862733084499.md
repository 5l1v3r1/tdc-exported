
# Бавария бошқарув постида ниҳоят ўзгариш рўй берди

Published at: **2019-11-03T21:10:00+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/bavariya-boshqaruv-postida-nihoyat-ozgarish-roy-berdi)

"Бавария" клуби бош мураббийи Нико Ковач ишдан бўшатилди. Бош мураббий вазифасини вақтинча Ханс Флик бажариб туради. У жамоани ЕЧЛ доирасидаги "Олимпиакос", Бундеслига доирасидаги "Боруссия"га қарши учрашувларга тайёрлайди.
