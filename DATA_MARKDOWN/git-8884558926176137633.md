
# Prvi poraz Ankarana

Published at: **2019-11-05T14:23:13+00:00**

Author: **Aleš Sorta**

Original: [Primorske novice](https://www.primorske.si/sport/prvi-poraz-ankarana)


        Tudi pretekli vikend v enotni primorski nogometni ligi je postregel z razburljivimi razpleti. Prvi poraz so doživeli Ankarančani, Postojnčani pa so se s tesno zmago še utrdili na vrhu lestvice.
      
Največ zadetkov so gledalci videli prav v Ankaranu, kjer je presenečenje pripravil Portorož Piran. Obramba gostiteljev, ki so na prvih osmih tekmah prejeli le dva gola, je bila v soboto precej bolj prepustna. Gostje so jim zabili štiri zadetke. Gregor Pribac je sicer Ankaran popeljal v vodstvo, potem pa so sledili kar štirje goli tekmecev in poraz favoriziranih gostiteljev je bil neizbežen.
Zmagoviti niz nadaljujejo vodilni Postojnčani, ki so imeli kar precej preglavic s Cerknico. Odločilni zadetek je v 81. minuti dosegel Ephraim Onosetale. Še več dela so imeli pri najbližjih zasledovalcih, Jadranu Hrpelje-Kozina. Korte so v 53. minuti ostale brez izključenega Gregorja Rutnika, zmagoviti zadetek pa je za gostitelje v sodnikovem dodatku dosegel Dominik Furlan. Na tretje mesto so se po pričakovani zmagi v Ilirski Bistrici prebili Komenci. Na dnu še brez zmage ostajajo Renčani, ki so po prvem polčasu sicer vodili v Ajdovščini. Zmagoviti zasuk je uspel tudi Košani v Podgradu.
