
# 枚方市にある「大阪市立高校」、府移管で改名の岐路

Published at: **2019-11-07T10:06:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191107-OYT1T50213/)

　大阪府枚方市にある「大阪市立高校（おおさかしりつこうこう）」の名称変更の可能性が高まっている。大阪府教委と大阪市教委が、２０２２年度に市立の高校２１校をすべて府に移管する方針を固めたためだ。現校名のまま府立になることは想定しづらいが、卒業生からは「できることなら校名を残して」との声が上がっている。
　愛称は「いちりつ」。京阪光善寺駅近くの小高い場所にある。普通科、英語科、理数科で約１０００人が学ぶ。大阪市内から通うのは１５％ほどだという。
　１９４１年、大阪市此花区の酉島（とりしま）工業学校に間借りして、同市立では初の旧制中学校「大阪市立中学校」として開校した。市外も含めて校地を探し、枚方町（現枚方市）の誘致を受けた。
　現在の場所に移ったのは４３年。竹やぶだった土地の開墾には生徒も加わった。グラウンドの整備は長く続き、５３年入学の岡林秀幸さん（８２）は「体育の時間はいつも、スコップで山の土を運んでいた」と振り返る。
　創立５０周年記念誌（９１年刊）によると、４８年の学制改革の際に、複数の市立高校が誕生することになり、校名が問題に。職員会議を経て「大阪市立枚方高校」とすることが一度は決まり、学校印やゴム印も作った。
　しかし、大阪市から「枚方という地名をつけるのは好ましくない」と指摘され、断念。大阪市立高校となった。２年後には、生徒に希望の校名をアンケートした。校内に多数の桜が植えられていることにちなむ「桜ヶ丘」が最多だったが、同名の学校が全国に多いとして、変更は見送られた。
　７８年には「大阪市民以外も通う学校を、大阪市の費用で運営するのはおかしい」と、府への移管が持ち上がった。市は有償での譲渡を求め、折り合わなかった。
　現在、府教委と市教委が打ち出している高校の移管は、両教委の役割分担を明確にし、広域的な方針で高校を運営するのが狙い。府、市は来夏までに計画をまとめる。移管後の校名は「府、市で協議し、府が決める」とされている。
　市立高同窓会の池田隆司会長は「卒業生は皆、『いちりつ』の呼び名に愛着がある。これから先も、あの場所に今の名前で続いてほしいというのが、同窓会の思いだ」と語る。
　一方、枚方市の伏見隆市長は「寂しさを感じる人は多いだろうとは思うが、もし、校名が変更されるのなら、ぜひ『枚方』を入れてほしい」としている。
