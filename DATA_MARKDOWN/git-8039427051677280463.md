
# శ్రీశైల దేవస్థానానికి రూ.2 లక్షల విరాళం

శ్రీశైలం: శ్రీశైల దేవస్థానం నిర్వహిస్తున్న అన్నదాన పథకానికి, కార్తీక లక్ష దీపోత్సవ కార్యక్రమానికి మొత్తం రూ.2 లక్షల విరాళాన్ని భక్తులు సమర్పించా రు. ఆదివారం దేవస్థానం నిర్వహిస్తున్న అన్నదాన పథకానికి గుంటూరుకు చెందిన ఎన్‌.బాసిరెడ్డి రూ.1,00,232 విరాళాన్ని దేవస్థాన కార్యనిర్వహణాధికారికి అందజేశారు. ఆయన వారికి స్వామివారి శేషవస్త్రాన్ని, ప్రసాదాలను, అన్నదానం బాండ్‌ను అందించి సత్కరించారు.
