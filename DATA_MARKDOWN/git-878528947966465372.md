
# 沖縄出身の侍・嘉弥真　首里城火災…悲痛も１回０封！

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/01/0012838450.shtml)

　「侍ジャパン強化試合、日本５－６カナダ」（３１日、沖縄セルラースタジアム那覇）
　野球日本代表「侍ジャパン」が惜敗した。カナダ代表には３５年ぶりの黒星となった。
　沖縄県石垣市出身の嘉弥真が八回のマウンドに上がり、１イニングを無安打無失点と完璧に抑えた。この日の朝、炎上する首里城をニュースで目にし「非常に残念です。確か１９歳の時も行って…。ショックです」と沈痛な面持ち。「スライダーをしっかり投げられました」と快投で勇気を届けた。
