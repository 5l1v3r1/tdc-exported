
# В НСО заморозили проект создания птицекомплекса для поиска инвестора

Published at: **2019-11-03T01:25:00+00:00**

Author: ****

Original: [RuNews24](https://runews24.ru/novosibirsk/03/11/2019/0bee71fc33fc7fda552d446dae3b351e)

Жители Колыванского района Новосибирской области в конце прошлого года протестовали против проекта создания птицекомплекса из-за угрозы, по их мнению, экологии. В 2017 году «Инд-Сибирь» получила первый участок площадью 18,2 гектара для реализации проекта, в апреле этого года компания без торгов получила второй земельный участок площадью 10,1 гектара.
По словам совладельца торгового холдинга «Сибирский гигант», которому принадлежит «Инд-Сибирь», Александра Манцурова, проект создания крупного птицекомплекса остановлен из-за того, что компания не договорились с финансирующим банком – Россельхозбанком.
Как отметил Манцуров, проект готов к реализации, и в данный момент его инициаторы ищут нового партнера.
В декабре 2017 года власти Новосибирской области одобрили проект «Инд-Сибирь» по созданию вертикально-интегрированного птицекомплекса по промышленному производству и переработке почти восьми тысяч тонн мяса индейки живым весом в год с возможностью расширения производительности до 15 тысяч тонн в Колыванском районе региона.
Сумма инвестиций должна составить три миллиарда рублей.
