
# TVアニメ「あひるの空」新キャスト発表！！ 五月正義役は竹本英史、車谷智久役は子安武人に決定！！

Published at: **2019-11-07T11:00:00+00:00**

Author: ****

Original: [アキバ総研](https://akiba-souken.com/article/42272/)

現在放送中のテレビアニメ「あひるの空」第7話に登場する新キャラクターのキャストが発表された。九頭龍高校教員の五月正義は竹本正義さんが、空の父親、車谷智久は子安武人さんが務めることとなった。
「あひるの空」は、日向武史さんが「週刊少年マガジン」（講談社）にて連載中のバスケットボール漫画。連載開始から約15年ファンに愛され続け、累計2400万部を超える人気作。九頭龍高校（通称：クズ高）を舞台に、不良の巣窟となったバスケットボール部に入部した主人公「車谷 空」の奮闘や成長が鮮明に描かれる。TVアニメ版の総監督は、「風夏」や「艦隊これくしょん -艦これ-」などの草川啓造さん。アニメーション制作は「BEATLESS」や「アホガール」などのディオメディアが担当する。
2019年10月より放送が開始されたテレビアニメ「あひるの空」は現在第6話まで放送されており、第7話では、主人公の車谷 空が所属する九頭龍高校（クズ高）が初めての練習試合を終え、新たな展開を迎える。今後のストーリーに期待が高まる中、その中で登場するキャラクターのキャストが発表された。
 
九頭龍高校教員の五月正義は竹本正義さんが、空の父親、車谷智久は子安武人さんが務める。さらに空の祖母、車谷ヨネは片貝 薫さんが演じるとのこと。今後登場する新キャラクターたちからも目が離せない。
