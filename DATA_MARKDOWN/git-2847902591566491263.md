
# Почему банки не принимают маткапитал в качестве первого взноса

Published at: **2019-11-05T17:46:00+00:00**

Author: ****

Original: [ГТРК «Ставрополье»](https://stavropolye.tv/news/127337)


        Крупнейшие российские банки отказываются принимать материнский капитал в качестве первоначального взноса по льготной ипотеке для семей с детьми.
      
Такие жалобы появились на специализированных форумах. Банки аргументируют это решение тем, что застройщики не хотят ждать несколько месяцев, пока Пенсионный фонд переведет им государственную субсидию за второго ребенка.
Программа ипотеки с господдержкой — инициатива президента Владимира Путина. Услугой могут воспользоваться семьи, в которых второй и последующий ребёнок родился в 2018—2022 годах.
