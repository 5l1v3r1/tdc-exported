
# Резкое похолодание: кузбасские синоптики дали прогноз на воскресенье

Published at: **2019-11-02T12:45:25+00:00**

Author: **Екатерина Бухтиярова**

Original: [Сiбдепо](https://sibdepo.ru/news/rezkoe-poholodanie-kuzbasskie-sinoptiki-dali-prognoz-na-voskresene.html)

Синоптики областного Гидрометеоцентра спрогнозировали понижение температуры воздуха на воскресенье, 3 ноября.
Так, по данным синоптиков, ночью температура воздуха опустится до -3… -8 °С, при прояснении — до -13 °С. Днём тоже будет прохладно – в пределах -1… -6 °С, местами по югу Кузбасса — до +4 °С.
Ожидаются переменная облачность, ночью в отдельных районах небольшой снег, днём преимущественно без осадков, местами гололёдно-изморозевые явления. На дорогах гололедица.
