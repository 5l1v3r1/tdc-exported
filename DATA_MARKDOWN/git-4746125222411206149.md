
# 関ジャニ∞、『クロニクル』初生放送 『27時間テレビ』で“イケメンカメラ目線”

Published at: **2019-11-01T16:00:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147823/full/)

　人気グループ・関ジャニ∞が11月2日、3日に放送されるフジテレビ系『FNS27時間テレビ にほんのスポーツは強いっ！』内番組『関ジャニ∞クロニクル』（3日 前6：40頃）に出演。毎週土曜午前10時53分からレギュラー放送されている同番組が初めて生放送されることが決定した。 　今回はメンバー5人が、世の中にある知られざるマイナースポーツに青春を捧げる高校生を全力バックアップしていく企画に挑戦。番組で取り上げるのは、横山裕が応援する宇都宮短期大学付属高校のスポーツチャンバラ部、安田章大が応援する千葉聖心高校のカーレット部、大倉忠義が応援する駒沢学園女子高校の杖道部（じょうどうぶ）、丸山隆平が応援する上宮高校のクリケット部。果たしてどんなスポーツなのか。 　そしてスタジオに、これら部活の高校生たちが生出演し、その魅力を伝えていく。そして競技の生実演に関ジャニ∞メンバーが参加する場面も…。 　番組の人気企画『イケメンカメラ目線』が登場。このコーナーは、アイドルたるもの常にイケメンでなければならないということで、今回は5人がマイナースポーツ競技中の瞬間を写真におさめ、誰が一番イケメンの写真が撮影できたか、を競う。1位になると応援している部活のメンバーに豪華賞品がプレゼント。前日から『27時間テレビ』キャプテンとして生放送に出演し続ける村上は、メンバーが集合するこのコーナーで、どんな表情をみせるのかにも注目だ。
