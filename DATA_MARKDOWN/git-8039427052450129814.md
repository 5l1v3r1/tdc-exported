
# అనంతపురం జిల్లాలో దారుణం..ఇద్దరు వ్యక్తులు హత్య

Published at: **2019-11-05T09:14:41+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=946995)

ADVT
అనంతపురం: జిల్లాలోని పరిగి మండలం గొర్రిపల్లిలో దారుణ ఘటన చోటుచేసుకుంది. పొలంగట్టు దగ్గర ఇద్దరు వ్యక్తులు దారుణ హత్యకు గురైయ్యారు. దుండగులు హత్యచేసి మృతదేహాలపై కారంపొడి చల్లారు. మృతదేహాలను గుర్తించిన స్థానికులు పోలీసులకు సమాచారం ఇచ్చారు. సమాచారం అందుకున్న పోలీసులు సంఘటనాస్థలానికి చేరుకుని మృతదేహాలను స్వాధీనం చేసుకున్నారు. కేసు నమోదు చేసి విచారణ జరుపుతున్నారు.
