
# 井上拓真が判定負け、バンタム級王座統一に失敗

Published at: **2019-11-07T12:21:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/etc/20191107-OYT1T50240/)

　世界ボクシング評議会（ＷＢＣ）バンタム級王座統一戦が７日、さいたまスーパーアリーナで行われ、暫定王者の井上拓真（大橋）が、正規王者のノルディーヌ・ウバーリ（仏）に０―３の判定で敗れ、王座統一に失敗した。
　ワールド・ボクシング・スーパーシリーズ（ＷＢＳＳ）バンタム級決勝は、世界ボクシング協会（ＷＢＡ）、国際ボクシング連盟（ＩＢＦ）王者の井上尚弥（大橋）が、ＷＢＡスーパー王者のノニト・ドネア（フィリピン）と対戦する。
