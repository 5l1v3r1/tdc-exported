
# 西野七瀬、『あな番』金澤美穂&奈緒と打ち上げ以来の再会「きょう話すこと無いかも」

Published at: **2019-11-06T00:55:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148068/full/)

　女優の西野七瀬が12日放送されるカンテレのバラエティー『グータンヌーボ2』（毎週火曜　深夜0：25※関西ローカル）で、7月期ドラマ『あなたの番です』（日本テレビ系）で共演した、金澤美穂、奈緒のとドラマの打ち上げ以来の再会を果たす。 　同番組は西野のほか、女優の長谷川京子、フリーアナウンサーの田中みな実、モデルの滝沢カレン、女優の西野七瀬の4人がMCを務め、週替りでそのなかの1人がゲスト2人とガールズトークを繰り広げる。 　ロケが始まる前、奈緒は「同年代と仕事をすることや現場で一緒になることが無くて、お友達は少なかったんですけど、美穂ちゃんとは元から友達だった」、金澤は「（奈緒は）しゃべらなくても隣にいるだけですごく安心感がある女の子。（西野は）実際会ったらよくしゃべる子。今日もいっぱいおしゃべりできたらいいな」と期待。その一方で西野は「3人で会ってるときに喋り尽くしちゃってて、きょう話すこと無いかもしれない」と仲良しだからこその心配も…。 　3人は、女優を始めたキッカケや将来の夢、さらに最近トキめいたことや理想のデートなど、仕事から恋愛までを語り合う。なお、放送翌日水曜日午前10時から、Amazon Prime Video、カンテレドーガ、TVerなどで配信される予定。
