
# «Команда лидеров Новгородчины» посетила психологический тренинг

Published at: **2019-11-01T20:16:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Velikij-Novgorod/738892-komanda-liderov-novgorodchiny-posetila-psikhologicheskii-trening)


        Для потенциальных управленцев провели очередной тренинг.
      
В пятницу, 1 ноября, участники «Команды лидеров Новгородчины» побывали на очередном тренинге. Об этом НЕВСКИМ НОВОСТЯМ рассказало областное министерство государственного управления.
Спикером тренинга для победителей проекта стала бизнес-тренер кандидат психологических наук Лариса Фоменко. Основой программы мероприятия стала работа с архетипами. В рамках тренинга участники должны были преодолевать свои внутренние страхи, сомнения и конфликты.
Напомним, помимо мероприятий в рамках «Школы лидеров Новгородчины» управленцы проходят дистанционные курсы по программе «Вежливый чиновник».
Напомним, в «Команду лидеров» в 2019 году по итогам отборов попали восемь человек. Их включили в управленческий резерв области.
