
# Кассы соберут данные со смартфонов

Published at: **2019-11-06T14:25:00+00:00**

Author: ****

Original: [Computerworld Россия](https://www.computerworld.ru/news/Kassy-soberut-dannye-so-smartfonov?utm_source=cw_subscribe&utm_medium=CPC&utm_campaign=computerworld_news&utm_term=06-11-2019)


        Это должно помочь магазинам, кафе и пр. таргетировать онлайн-рекламу на клиентов и проходящих мимо людей.
      
Умные кассовые терминалы «Эвотор» начнут считывать MAC-адреса мобильных устройств с включенным Wi-Fi в радиусе 100 м, сообщает «Коммерсантъ». Сбор активирует приложение, которое владельцы касс могут скачать в магазине «Эвотор.Маркет». Собранные данные уйдут на рекламные площадки «Яндекса» и myTarget. Приложение рассчитает количество показов, будет мониторить ход рекламной кампании и отследит посещение торговой точки пользователями после просмотра рекламы.
Сервис «Квант» (СП «Газпром-медиа» и «МаксимаТелеком») дополнит МАС-адреса идентификаторами из публичных сетей Wi-Fi для их переноса на рекламные платформы. MAC-адреса уже используются для таргетирования наружной рекламы: собрать их можно при подключении пользователей к публичным сетям W-Fi, где MAC-адреса нужны для идентификации. По данным DataDarvin, 83% мобильных пользователей не отключают Wi-Fi на своих гаджетах.
Для таргетирования рекламы выгоднее использовать оборудование сотовых операторов, не согласны в МТС. Оператор связи назвал номер телефона почти единственным стабильным идентификатором абонента.
