
# В Мелитополе разыскивают пассажира, потерявшего в маршрутке банковскую карточку

Published at: **2019-11-01T09:57:00+00:00**

Author: **Владимир Андреев**

Original: [РИА Мелитополь](https://ria-m.tv/news/170192/v_melitopole_razyiskivayut_passajira_poteryavshego_v_marshrutke_bankovskuyu_kartochku.html)


        Ехал пассажир по маршруту №24. Один из пассажиров нашел карточку и отдал ее водителю.
      
 - Мелитополь, утро.1.11.2019. Маршрутка № 24. Мерседес белого цвета, АР0929СО, перевозчик "АТП АССА". Отдали водителю, - оставили неравнодушные мелитопольцы в соцсети ориентир для растеряши.
