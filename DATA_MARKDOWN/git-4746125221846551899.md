
# デビュー20周年の嵐、1日のみの集計で初の週間デジタルシングルが1位 【オリコンランキング】

Published at: **2019-11-05T04:00:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148007/full/)

　嵐の初となるデジタルシングル「Turning Up」が、週間売上2.8万DLを記録し、11/11付オリコン週間デジタルシングル（単曲）ランキングで初登場1位を獲得。配信が開始された11月3日、1日のみの集計で1位となった。 　デビュー20周年記念日である11月3日に行われた嵐の記者会見で配信リリースが発表され、同日19時より配信がスタートした本作は、明るく前向きな歌詞が印象的なポップソング。ミュージックビデオでは、世界中の人々を巻き込んでパフォーマンスする嵐の姿が描かれている。 　嵐は、グループの公式SNS開設やこれまでの全シングル曲のサブスクリプション解禁なども発表。またこれらの新たな挑戦の報告と20周年のお礼を兼ねて、アジア4都市（ジャカルタ、シンガポール、バンコク、台北）へ5人が専用ジェット機で訪れるキャンペーン『JET STORM』を実施する。さらに、2020年5月15日と16日には、東京・国立競技場で単独アーティスト初となるコンサートを行い、2020年春には中国・北京でコンサートを開催する。 ■禁無断複写転載※オリコンランキング（売上枚数のデータ等）の著作権その他の権利はオリコンに帰属していますので、無断で番組でのご使用、Webサイト（PC、スマホ、SNS）や雑誌等で掲載するといった行為は固く禁じております。
