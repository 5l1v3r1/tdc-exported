
# สิงคโปร์เรียกร้องอาเซียนเพิ่มเอกภาพทางเศรษฐกิจ

Published at: **2019-11-03T03:34:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/foreign/739752)


        นายกรัฐมนตรีสิงคโปร์เรียกร้องเอเชียตะวันออกเฉียงใต้เพิ่มการบูรณาการทางการเศรษฐกิจร่วมกัน ซึ่งจะยกระดับความเป็นเอกภาพของภูมิภาคในการเผชิญหน้าและฝ่าฟันกับความท้าทายจากภายนอก
      
สำนักข่าวต่างประเทศรายงานจากกรุงเทพมหานคร เมื่อวันที่ 3 พ.ย. ว่านายกรัฐมนตรีลี เซียน ลุง ผู้นำสิงคโปร์ กล่าวต่อที่ประชุมสุดยอดครั้งที่ 35 ของผู้นำ 10 ประเทศสมาชิกสมาคมประชาชาติแห่งเอเชียตะวันออกเฉียงใต้ ( อาเซียน ) แบบเต็มคณะ เมื่อวันเสาร์ที่ผ่านมา มีสาระสำคัญในตอนหนึ่งว่า อาเซียนรวมตัวกันเป็นประชาคมเศรษฐกิจอาเซียน ( เออีซี ) เมื่อปี 2558 แต่นับจากนั้นแทบไม่มีความคืบหน้า ตอนนี้สมาชิกต้องเพิ่มความจริงจังเพื่อการร่วมกันบรรลุเป้าหมาย "พิมพ์เขียวเศรษฐกิจอาเซียน" ที่มีการกำหนดไว้ในปี 2568
