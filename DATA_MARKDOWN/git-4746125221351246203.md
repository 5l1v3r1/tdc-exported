
# 泉里香、ショーパンで美脚大胆披露「南の島のマーメイド」「ため息出るほどの美人」

Published at: **2019-11-05T14:52:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148056/full/)

　モデルで女優の泉里香が5日、自身のインスタグラムを更新。沖縄での写真を公開した。 　泉は「晴れてきた〜　あったかくて11月と思えない過ごしやすさ　気持ちいいなぁ」と、海辺での写真をアップ。白のTシャツにデニムのショートパンツ、足元はサンダルというラフなスタイルで、美脚を大胆披露している。ハッシュタグには「＃okinawa ＃uruma ＃うるま」と記載し、沖縄県うるま市にいることを明かしている。 　この投稿にファンから「美脚〜」「脚、長っ！ 細っ！ 綺麗っ！」「南の島のマーメイド」「さすが沖縄！ 真夏のスタイルですね」「ため息出るほどの美人〜」「すごい綺麗で脚もこれぞ美脚って言葉がぴったりですごい素敵です」など、絶賛の声が相次いでいる。
