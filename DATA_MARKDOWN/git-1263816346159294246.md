
# Нехватка доноров крови в Российской детской клинической больнице

Published at: **2019-11-02T12:30:00+00:00**

Author: **Инна Лутовинова**

Original: [МОЛНЕТ.RU](https://www.molnet.ru/mos/ru/health_and_social/o_717087)

В Российской детской клинической больнице нет доноров всех групп крови, сообщается на сайте Подари жизнь.
Сдать кровь можно понедельника по пятницу, с 8:30 до 11:00 по адресу: Ленинский проспект, дом №117, корпус 3, второй этаж (на входе на этаж висит табличка "1 этаж"), отделение переливания крови.
Обратите внимание, что в РДКБ ждут только безвозмездных доноров. Для сдачи крови не требуется регистрация в Москве или Московской области.
