
# В Туле опрокинулась маршрутка: пострадали три человека

Published at: **2019-11-01T18:09:00+00:00**

Author: ****

Original: [Тульская Служба Новостей](https://www.tsn24.ru/v-tule-oprokinulas-marshrutka-postradali-tri-cheloveka.html)

Микроавтобус, следовавший по маршруту №42, попал в ДТП на Алексинском шоссе Зареченского округа Тулы вечером 1 ноября.
По предварительной информации УМВД России по региону, водитель транспортного средства не справился с управлением, в результате чего маршрутка опрокинулась.
По информации на 21.00 в результате аварии пострадали три человека – водитель «автолайна» и двое пассажиров.
