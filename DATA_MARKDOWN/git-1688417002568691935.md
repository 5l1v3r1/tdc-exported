
# Ворскла минимально уступила Десне

Published at: **2019-11-02T16:37:00+00:00**

Author: **iSport.ua**

Original: [Корреспондент.net](https://korrespondent.net/sport/football/4156195-vorskla-mynymalno-ustupyla-desne)

Ворскла одержала минимальную победу над Десной в матче 13-го тура УПЛ.
Ворскла — Десна 0:1
Гол: Имереков, 76
Ворскла: Резник — Пердута, Чеснаков, Мартыненко, Мельничук — Петрович (Скляр, 66), Габелок (Кравченко, 46) — Якубу (Ребенок, 84), Луизао, Козыренко — Коломоец
Десна: Паст — Д. Фаворов, Гитченко, Имереков, Западня (Мостовой, 46) — Богданов, Огиря — Калитвинцев, А. Фаворов, Хлебас (Картушов, 83) — Дегтярев (Филиппов, 73)
Предупреждения: Петрович, Чеснаков — Дегтярев
