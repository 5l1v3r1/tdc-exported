
# HBO выпустила черную картонную коробку для просмотра сериалов

Published at: **2019-11-06T05:20:31+00:00**

Author: ****

Original: [ФАКТИ](https://fakty.com.ua/ru/showbiz/tv/20191106-hbo-vypustyla-chornu-kartonnu-korobku-dlya-pereglyadu-serialiv/)

HBO Box – специальная коробка для просмотра сериалов – не продается, но ее можно выиграть.
Телеканал HBO представил новый продукт — HBO Box. Коробка из черного картона создана для тех, кто хочет смотреть сериалы наедине с самим собой.
Может показаться, что это первоапрельская шутка, но это не так.
HBO Box представляет собой гигантскую черную картонную коробку с отверстиями для воздушного потока, полкой внутри и вырезами под ноги.
