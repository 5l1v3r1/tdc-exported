
# 'วีระกร'เชื่อ'สุชาติ'ไม่รับนั่งประธานกมธ.แก้รธน.

Published at: **2019-11-07T04:16:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/politics/740411)


        “วีระกร”เชื่อ “สุชาติ” ไม่รับนั่งประธาน กมธ.แก้รธน.  ระบุตำแหน่ง “รองปธ.สภาฯ”เลยระดับมานั่งตำแหน่งประธาน กมธ.แล้ว
      
เมื่อวันที่   7 พ.ย.  ที่รัฐสภา นายวีระกร คำประกอบ ส.ส. นครสวรรค์ พรรคพลังประชารัฐ(พปชร.) กล่าวถึงกระแสข่าวพรรคพลั​งประชารัฐดันชื่อ นายสุชาติ ตันเจริญ  รองประธานสภาผู้แทนราษฎร คนที่หนึ่ง แข่งกับนายอภิสิทธิ์ เวชชาชีวะ. อดีตหัวหน้าพรรคประชาธิปัตย์ นั่งประธานคณะกรรมาธิการ (กมธ.)พิจารณาศึกษาหลักเกณฑ์​และวิธิการแก้ไขเพิ่มเติมรั​ฐธรรมนูญแห่งราชอาณาจักรไทย  พ.ศ.2560  ว่า ทางพรรคยังไม่ได้มีการประชุมเรื่​องเกี่ยวกับรัฐธรรมนูญ แต่ความเห็นส่วนตัว เชื่อว่านายสุชาติจะไม่รั​บตำแหน่งนี้  เพราะท่านเป็นถึ​งรองประธานสภาผู้แทนราษฎร“ผมเชื่อว่าระดับรองประธานสภาผู้​แทนราษฎร คงไม่มานั่งตำแหน่ง ประธาน กมธ. อีก เพราะท่านเลยระดับนี้ไปแล้ว” นายวีระกร กล่าว.
