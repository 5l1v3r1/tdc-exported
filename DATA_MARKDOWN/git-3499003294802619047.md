
# На Панфиловском проспекте произошло дорожно-транспортное происшествие

Published at: **2019-11-05T10:11:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/zelao/na-panfilovskom-prospekte-proizoshlo-dorozhno-transportnoe-proisshestvie--123365/)

Прибыв на место дорожно-транспортного происшествия,  инспекторы ГИБДД выяснили, что 57-летний житель Северо-Кавказского федерального округа, управляя автобусом «Volgabus», следуя по Панфиловскому проспекту в направлении улицы Болдов Ручей, на регулируемом перекрестке допустил столкновение с впереди идущей автомашиной «Форд Фокус».В дорожно-транспортном происшествии пострадала 41-летняя зеленоградка, водитель транспортного средства «Форд Фокус», которая нарядом скорой медицинской помощи доставлена городскую больницу. По факту ДТП ведется административное расследование.ОГИБДД УВД по Зеленоградскому округу
