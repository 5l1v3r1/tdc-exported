
# Podrás jugar Battlefield V gratis este fin de semana

Published at: **2019-11-01T23:44:43+00:00**

Author: **Rodolfo León**

Original: [Atomix](https://atomix.vg/podras-jugar-battlefield-v-gratis-este-fin-de-semana/)

El día de ayer, Battlefield V recibió el Capítulo 5 con una nueva actualización, trayendo escenarios del Pacífico a este FPS. Nuevas facciones, vehículos, mapas y más han sido agregados, dándole un poco de carne fresca a este título. Para celebrarlo, EA le está dando oportunidad a todo el mundo de jugarlo gratuitamente este fin de semana en PlayStation 4.
