
# На железнодорожном вокзале Одессы появился зал ожидания для военнослужащих

Published at: **2019-11-07T17:21:00+00:00**

Author: ****

Original: [ТАЙМЕР](http://timer-odessa.net/news/na_jeleznodorojnom_vokzale_odessi_poyavilsya_zal_ojidaniya_dlya_voennoslujaschih_429.html)


        На железнодорожном вокзале Одессы из депутатской комнаты оборудовали отдельный и круглосуточный зал ожидания именно для военнослужащих.
      
Об этом руководитель «Укрзализныци» Евгений Кравцов сообщил в своём Facebook.
Он написал, что, помимо Одессы, экс-депутатские были переоборудованы подобным образом на вокзалах Днепропетровска, Львова, Харькова, а с 7 ноября ещё и Винницы с Мариуполем.
Первый такой зал ожидания для военнослужащих открылся 1 октября 2019 года на Центральном железнодорожном вокзале Киева.
