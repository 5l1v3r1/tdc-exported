
# ドラッグストア屋外トイレに乳児の遺体

Published at: **2019-11-04T00:36:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191103-OYT1T50170/)

　２日午後８時４５分頃、埼玉県ふじみ野市ふじみ野のドラッグストア「マツモトキヨシ大井町店」の屋外トイレで、洋式便器の中から下半身のない乳児の遺体をトイレ修理業者の男性が発見、同店の男性従業員を通じて東入間署に通報した。同署で死体遺棄事件として、店の防犯カメラを解析するなどして、死因や身元などを調べている。
　発表によると、乳児は性別不明、全裸でへその緒が付いていたという。同日午後１時頃、女性客がトイレを利用した際に、トイレが詰まっていたため、従業員が修理業者を呼んでいた。
