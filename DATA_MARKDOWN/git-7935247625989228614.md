
# 'Çukur'daki şiddet sokağa göre masum'

Published at: **2019-11-02T14:05:32+00:00**

Author: ****

Original: [OLAY](https://www.olay.com.tr/cukurdaki-siddet-sokaga-gore-masum-213933h.htm)

Antalya'da düzenlenen söyleşide konuşan Kesal, “Çukur'daki şiddet meselesi, gündelik hayattaki şiddetle karşılaştırdığınızda, emin olun çok daha masum” dedi.
