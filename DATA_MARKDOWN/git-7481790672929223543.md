
# 「唐津くんち」で曳山１４台勢ぞろい

Published at: **2019-11-04T00:48:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191104-OYT1T50079/)

　国連教育・科学・文化機関（ユネスコ）の無形文化遺産、佐賀県唐津市の「唐津くんち」は３日、最大の見せ場となる「お旅所神幸（たびしょしんこう）」が行われ、１４台の曳山（やま）が勢ぞろいした。
　祭りは２日夜の「宵曳山」で始まり、３日は１番曳山・赤獅子を先頭に、鯛（たい）や鯱（しゃち）などの曳山が市中心部を巡行。クライマックスのお旅所では、車輪が砂地に埋まった重さ２～３トンの曳山を、綱を持った曳（ひ）き子たちが「エンヤー」「ヨイサー」の声を上げながら進めた。
　最終日の４日は、１４台が市中心部を巡る。
