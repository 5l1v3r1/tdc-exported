
# Tülin Şahin’in kızıyla ilk fotoğrafı…

Published at: **2019-11-07T09:22:00+00:00**

Author: ****

Original: [medya faresi](http://www.medyafaresi.com/haber/tulin-sahinin-kiziyla-ilk-fotografi/926410)

Doğum için İstanbul’u tercih eden Tülin Şahin, önceki gün saat 12.08’de, Siena adını verdikleri kızını dünyaya getirdi.
Şahin “4 Kasım 2019 12.08 kızımız Siena Leyla Allah'ım kelimeler yetersiz.” mesajıyla bebeğinin ilk fotoğrafını Instagram adresinden paylaştı.
Kızının dünyaya geldiğini sosyal medyadan müjdeleyen baba Pedro de Noronha, doğuma girip Siena’nın dünyaya gelişine de şahitlik etmiş.
