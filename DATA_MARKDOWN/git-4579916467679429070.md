
# На Туристской электричка сбила человека. Пешеходы карабкаются по обрыву

Published at: **2019-11-06T16:30:00+00:00**

Author: ****

Original: [Фонтанка.ру](https://www.fontanka.ru/2019/11/06/142/)

По информации очевидца, поезд травмировал пешехода на злополучном переходе в створе Туристской улицы. Состав остановлен, люди обходят его по обрыву, вдоль железной дороги.
«Фонтанка» обратилась за подробностями в пресс-службу управления на транспорте МВД России по СЗФО.
