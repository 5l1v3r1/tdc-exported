
# Μαρτίνς: «Για ποιον λόγο υπάρχει το VAR…»

Published at: **2019-11-02T21:40:37+00:00**

Author: **ΤοΒΗΜΑ Team**

Original: [Το Βήμα Online](https://www.tovima.gr/2019/11/02/sports/martins-gia-poion-logo-yparxei-to-var/)

Με μία φράση στη συνέντευξη Τύπου, ο Πέδρο Μαρτίνς εξέφρασε την απορία του για τα θαυμαστά που συμβαίνουν στον κόσμο του ελληνικού ποδοσφαίρου.
«Δεν μπορώ να καταλάβω για ποιον λόγο υπάρχει το VAR…» αναρωτήθηκε ο Πορτογάλος προπονητής και όχι άδικα, αφού εάν το χρησιμοποιούσαν οι διαιτητές θα υποδείκνυαν τις παραβάσεις υπέρ του Ολυμπιακού.

        Διαβάστε ακόμη : Ποιες περιοχές της Αττικής κινδυνεύουν περισσότερο από καταστροφικές πλημμύρες
      
Για ακόμα μία φορά, ο Μαρτίνς είδε την ομάδα του να μην παίρνει αυτά που δικαιούται από τον ρέφερι και εκφράστηκε μάλλον συγκρατημένα. Διότι δεν θέλησε να πει κάτι περισσότερο, παρότι θα μπορούσε να θίξει πολλά από τα κακώς κείμενα που παρακολουθεί κάθε Σαββατοκύριακο…
Αναλυτικά οι υπόλοιπες δηλώσεις του Πέδρο Μαρτίνς στη συνέντευξη Τύπου:
Για το παιχνίδι: «Ήμασταν απόλυτα κυρίαρχοι στο ματς. Είχαμε πολλές ευκαιρίες. Είχαμε πολλές τελικές. Εμείς υπήρχαμε στον αγωνιστικό χώρο. Δυστυχώς δεν καταφέραμε να πάρουμε τη νίκη».
Για την εικόνα της Ξάνθης: «Ο Ολυμπιακός ήταν κυρίαρχος στο παιχνίδι. Για αυτό πρέπει να ρωτήσετε τον προπονητή της Ξάνθης».
