
# Неприятный запах в Тамбове может быть связан с деятельностью предприятий

Published at: **2019-11-04T00:00:00+00:00**

Author: ****

Original: [Тамбов-информ](http://www.taminfo.ru/important/43809-nepriyatnyy-zapah-v-tambove-mozhet-byt-svyazan-s-deyatelnostyu-predpriyatiy.html)

Продолжает свою работу рабочая группа при администрации города Тамбова по вопросу установления неприятного запаха сероводорода (канализации), который последнее время одолевал жителей областного центра.
По жалобам горожан была составлена специальная карта. Вот этот район: улицы Бориса Васильева, Московская, Тулиновская, Подвойского, часть Моршанского шоссе, Куйбышева, где больше всего было жалоб. Недалеко от этого района располагается тамбовский водоканал.
Депутат Тамбовской городской Думы Алексей Власкин отметил, что потенциальные организации, от которых могло исходить подобное зловоние, предупреждены.
«И в случае, если повториться такая ситуация, будет организован контроль уже по конкретным объектам конкретных организаций», - Алексей Власкин.
