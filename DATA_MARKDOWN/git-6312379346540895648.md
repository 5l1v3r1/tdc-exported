
# Матвиенко поручила обратиться в комиссию Думы из-за заявлений Рашкина

Published at: **2019-11-06T09:16:48+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-06/matvienko-poruchila-obratitsya-v-komissiyu-dumy-iz-za-zayavleniy-rashkina)

Валентина Матвиенко поручила сенаторам направить обращение в комиссию Госдумы по этике в связи с тем, что депутат Валерий Рашкин обвинил вице-спикера Совфеда Юрия Воробьева в создании препятствий для проверки в отношении обвиняемого в убийствах экс-сенатора Рауфа Арашукова. Об этом сообщает ТАСС.
По ее словам, комиссия по этике Госдумы должна дать объективную оценку высказываниям Рашкина.
"На самом деле, мы все возмущены этой ситуацией. Так не должны поступать политики. Я прошу вас направить обращение в адрес Госдумы, в комиссию по этике. Для этого она и существует, чтобы были даны адекватные оценки таким недостойным поступкам", — сказала Матвиенко.
