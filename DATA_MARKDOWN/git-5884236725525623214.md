
# Аэропорт Мурманска начали чаще использовать для перелетов

Published at: **2019-11-06T13:55:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Murmansk/740396-aeroport-murmanska-nachali-chashe-ispolzovat-dlya-pereletov)


        Аэропорт Мурманска обслужил на 8 % больше пассажиров, чем за аналогичный период 2018 года.
      
В аэропорте Мурманска ожидают рост числа пассажиров уже в декабре этого года. Как сообщает пресс-служба воздушной гавани, за октябрь пассажиропоток вырос, по сравнению с октябрем 2018 года вырос на 14,9 %, что составляет более 83 тыс. человек.
С начала года по 31 октября услугами перелетов воспользовались более 850 тыс. пассажиров. Ведется активная работу по поиску новых перевозчиков и открытия направлений для прямых перелетов. С 27 октября аэропорт перешел на зимнее расписание, сохранив почти все востребованные направления.
