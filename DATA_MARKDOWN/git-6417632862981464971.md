
# Неймар бир неча ҳафтадан сўнг яна ПСЖ машғулотларида иштирок эта бошлади

Published at: **2019-11-06T05:21:13+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/neymar-bir-necha-haftadan-yana-psj-mashgulotlarida-ishtirok-eta-boshladi)

Бу ҳақда клуб матбуот хизмати хабар бермоқда.
Бразилиялик футболчини соннинг орқа мушаклари оғриши безовта қилмоқда.
Неймар Чемпионлар Лигаси доирасидаги "Брюгге"га қарши ўйинда иштирок этмайди.
У тез орада умумий гуруҳдаги машғулотларга қўшилиши кутилмоқда.
