
# Украинский производитель кормов для животных может получить кредит ЕБРР на строительство завода в Литве

Published at: **2019-11-07T17:23:00+00:00**

Author: ****

Original: [НВ](https://nv.ua/biz/markets/kredit-ot-ebrr-ukrainskaya-kompaniya-mozhet-poluchit-10-mln-na-stroitelstvo-zavoda-v-litve-novosti-ukrainy-50052255.html)

Европейский банк реконструкции и развития планирует предоставить украинскому производителю кормов для домашних животных ООО Кормотех кредит на сумму 10 млн евро, сообщает Интерфакс-Украина.
Как отметили в ЕБРР, кредит может быть выдан Кормотеху на строительство завода по производству кормов для домашних животных в Литве и расширение мощностей по производству кормов в Украине.
«Банк поддержит первую трансграничную инвестицию украинской компании, состоящую в строительстве нового производственного объекта в Литве», — сказано в сообщении банка.
По информации ЕБРР, Кормотех является одним из трех крупнейших производителей кормов для домашних животных в Украине.
Решение по предоставлению кредита может быть одобрено ЕБРР 11 декабря 2019 года.
Кормотех в 2018 году начал строительство завода по производству кормов в Литве с плановой мощностью 5 тыс. т мокрых кормов в год.
Кормотех экспортирует продукцию в свыше 20 стран, в том числе во Францию, Эстонию, Нидерланды, Словению, Польшу. Мощности предприятия включают два завода по изготовлению сухих и влажных кормов во Львовской области. Компания производит продукты для кошек и собак под собственными брендами Optimeal, Клуб 4 лапы (ТМ Мяу! и Гав!) и по направлению private label. Конечными бенефициарами Кормотеха являются Елена и Ростислав Вовк.
