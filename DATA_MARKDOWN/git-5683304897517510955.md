
# സ്കൂളുകളിൽ വിദ്യാർഥികൾ മൊബൈൽ ഉപയോഗിക്കരുത്; ക്ലാസ് സമയത്ത് അധ്യാപകരും

Published at: **2019-11-05T13:12:00+00:00**

Author: **മനോരമ ലേഖകൻ**

Original: [ManoramaOnline](https://www.manoramaonline.com/news/latest-news/2019/11/05/mobile-phones-banned-in-schools-kerala-circular-out.html?utm_source=dlvr.it&utm_medium=twitter)

തിരുവനന്തപുരം∙ സ്കൂളുകളിൽ വിദ്യാർഥികൾ മൊബൈൽ ഫോൺ ഉപയോഗിക്കാൻ പാടില്ലെന്ന് പൊതുവിദ്യാഭ്യാസ ഡയറക്ടറുടെ ഉത്തരവ്. ക്ലാസ് സമയത്ത് അധ്യാപകർ മൊബൈൽ ഫോണും സമൂഹമാധ്യമങ്ങളും ഉപയോഗിക്കരുതെന്നും ഉത്തരവിൽ നിർദേശിക്കുന്നു.
സ്കൂളുകളിൽ കുട്ടികൾ മൊബൈൽ ഫോൺ ഉപയോഗിക്കരുതെന്നു നിർദേശിച്ച് മുൻപും സർക്കുലർ ഇറക്കിയിരുന്നു. എന്നാൽ അതു പാലിക്കപ്പെടുന്നില്ലെന്നു പരാതികൾ ലഭിച്ച സാഹചര്യത്തിലാണ് വീണ്ടും നിർദേശങ്ങൾ നൽകിയിരിക്കുന്നത്. നിർദേശങ്ങൾ കർശനമായി നടപ്പിലാക്കണമെന്നു പ്രഥമാധ്യാപകർക്കും വിദ്യാഭ്യാസ ഓഫിസർമാർക്കും നിർദേശം നൽകിയിട്ടുണ്ട്.
