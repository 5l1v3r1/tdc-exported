
# ハイヒール・リンゴ、膝に水がたまり戦々恐々「抜くのはものスゴく痛い」

Published at: **2019-11-02T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/02/0012841993.shtml)

　ハイヒール・リンゴが２日、ＭＣをつとめる読売テレビ「あさパラ！」に出演。ロケで歩きすぎたため、膝に水がたまっていることを報告した。
　リンゴは先日、海外ロケで中欧を訪問。「オーストリアとかチェコとか、街が狭いから観光客の車が入れず、歩かなあかん。ヒールがある靴で歩いていたら、帰ってきたら膝が痛くて」とリンゴ。整形外科で診察してもらったところ「水がたまっているって」と苦笑い。
　すぐに水を抜くほどではないので、１カ月の様子見となった。「水が吸収されるかもしれないし、さらに増えるかも。坂道と階段がダメ。３～５倍、膝に負担がかかるから」と状況を説明した。
　相方のモモコの「ついにおばあちゃんになりました」との突っ込みにも苦笑いのリンゴ。「水を抜くのはものスゴく痛いらしい」と戦々恐々。「これから１カ月、私が階段や坂道を降りようとしたら止めて下さい！」と周囲に頼み込んでいた。
