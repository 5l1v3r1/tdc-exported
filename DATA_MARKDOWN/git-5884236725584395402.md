
# В Петербурге ликвидировали незаконную столовую и офис по языковым курсам

Published at: **2019-11-07T13:03:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/740753-v-peterburge-likvidirovali-nezakonnuyu-stolovuyu-i-ofis-po-yazykovym-kursam)


        Сотрудники Комитета по контролю за имуществом Санкт-Петербурга в июне 2019 года проверили объекты нежилого фонда по нескольким адресам и выяснили, что их незаконно заняли под пункт общественного питания и офис для языковых курсов.
      
Сотрудники Комитета по контролю за имуществом Санкт-Петербурга в июне 2019 года проверили объекты нежилого фонда и выявили ряд нарушений.
