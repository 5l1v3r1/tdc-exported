
# Πιθανόν να μην επιβληθούν επιπρόσθετοι δασμοί στα εισαγόμενα αυτοκίνητα στις ΗΠΑ

Published at: **2019-11-03T23:48:00+00:00**

Author: ****

Original: [enikos.gr](http://www.enikos.gr/international/682113/pithanon-na-min-epivlithoun-epiprosthetoi-dasmoi-sta-eisagomena-a)

Μετά τις «καλές συνομιλίες» με τις αυτοκινητοβιομηχανίες, οι ΗΠΑ μπορεί να μην χρειαστεί να επιβάλουν επιπρόσθετους δασμούς στα εισαγόμενα οχήματα, όπως ανέφερε σήμερα ο υπουργός Εμπορίου, μιλώντας στην τηλεόραση του Bloomberg.
«Ελπίζουμε ότι οι διαπραγματεύσεις που είχαμε με τις εταιρείες για τα επενδυτικά σχέδιά τους θα καρποφορήσουν ώστε να μην θεωρηθεί αναγκαίο να εφαρμοστεί» ο κανόνας που επιτρέπει στις ΗΠΑ να περιορίσουν τις εισαγωγές ορισμένων προϊόντων ώστε να προστατευθεί η εθνική ασφάλειά τους, είπε ο Γουίλμπουρ Ρος.
«Είχαμε πολύ καλές συνομιλίες με τους Ευρωπαίους, τους Ιάπωνες και τους Κορεάτες φίλους μας», πρόσθεσε, από τη Μπανγκόκ όπου μετέχει στην ετήσια σύνοδο του ASEAN.
Από το 2018 η κυβέρνηση του Αμερικανού προέδρου Ντόναλντ Τραμπ απειλεί ότι θα επιβάλει δασμούς ύψους 25% στα εισαγόμενα αυτοκίνητα για να στηρίξει την Αμερικανική αυτοκινητοβιομηχανία. Αφού ανέβαλε την εφαρμογή αυτού του μέτρου τον περασμένο Μάιο, ο Τραμπ θα πρέπει τώρα να αποφασίσει μέχρι τα μέσα Νοεμβρίου αν θα επιβληθούν επιπρόσθετοι δασμοί στα εισαγόμενα ευρωπαϊκά αυτοκίνητα, μια απειλή που ανησυχεί ιδίως τη Γερμανία.
Μια τέτοια ενέργεια θα σηματοδοτούσε μια νέα κλιμάκωση στον εμπορικό πόλεμο μεταξύ Βρυξελλών και Ουάσινγκτον, λίγες εβδομάδες αφότου εφαρμόστηκαν τιμωρητικοί δασμοί σε σειρά ευρωπαϊκών προϊόντων, ύψους 7,5 δισεκατομμυρίων δολαρίων.
Το 2018 οι ΗΠΑ επέβαλαν υψηλότερους δασμούς στον χάλυβα και το αλουμίνιο που παράγεται στην ΕΕ.
Τον Σεπτέμβριο η αμερικανική κυβέρνηση συμφώνησε με την Ιαπωνία να αναβληθεί η επιβολή δασμών στα ιαπωνικά αυτοκίνητα μέχρι να ολοκληρωθούν οι διαπραγματεύσεις μεταξύ των δύο πλευρών. Το 2018 είχε πετύχει μια συμφωνία με τη Νότια Κορέα, με βάση την οποία η Σεούλ δεσμεύτηκε να ανοίξει περισσότερο την αγορά της στις αμερικανικές εταιρείες.
