
# 世耕氏「大人の新婚生活は良いものだ」…菊池桃子さんと結婚の新原氏は「大変な仕事師」

Published at: **2019-11-05T09:46:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/culture/20191105-OYT1T50191/)

　自民党の世耕弘成参院幹事長は５日の記者会見で、経済産業省の新原浩朗（ひろあき）経済産業政策局長（６０）とタレントの菊池桃子さん（５１）の結婚について、「大人の新婚生活は良いものだ」と述べ、祝意を示した。
　世耕氏は５０歳だった２０１３年９月、林久美子・民主党参院議員（当時）と再婚した。９月まで経済産業相を務めており、新原氏は元部下にあたる。世耕氏は新原氏について「大変な仕事師だ」と評価。「彼が結婚すること自体、大変な驚きだったが、お相手を聞いて千倍くらい驚いた」とも語った。
