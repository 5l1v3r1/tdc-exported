
# Incidente in galleria Scoppo, lunghe code da Villafranca a Boccetta

Published at: **2019-11-04T08:43:21+00:00**

Author: **Redazione**

Original: [Tempostretto](https://www.tempostretto.it/news/incidente-in-galleria-scoppo-lunghe-code-da-villafranca-a-boccetta.html)


        Si sono create lunghe code a partire da poco oltre la barriera di Villafranca fino allo svincolo di Boccetta, così come all'ingresso dello svincolo di Giostra
      
Tamponamento tra due auto in galleria Scoppo, stamani, l’ultima prima dello svincolo di Boccetta in direzione centro città. Sul posto la Polizia Stradale e i sanitari del 118, che hanno soccorso tre persone ferite in modo lieve.
Si sono create lunghe code a partire da poco oltre la barriera di Villafranca fino allo svincolo di Boccetta, così come all’ingresso dello svincolo di Giostra. Si tratta, tra l’altro, del tratto che da due settimane è tornato a corsia unica per i lavori sul viadotto Ritiro.
