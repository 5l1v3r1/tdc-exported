
# В центре Запорожья подросток сломала таз, выпав из окна

Published at: **2019-11-05T07:23:00+00:00**

Author: **Владимир Андреев**

Original: [РИА Мелитополь](https://ria-m.tv/news/170594/v_tsentre_zaporojya_podrostok_slomala_taz_vyipav_iz_okna.html)


        16-летняя девочка получила многочисленные травмы.
      
В одном из домов по ул. Лобановского подросток выпала из окна 3-ого этажа. пишет IPnews.
Как сообщили в областном центре экстренной медпомощи, пострадавшая получила многочисленные травмы: живота, грудной клетки, черепно-мозговую, перелом костей таза.
За жизнь пострадавшей борются врачи областной детской больницы. Обстоятельства, при которых ребенок выпала из окна не сообщаются.
