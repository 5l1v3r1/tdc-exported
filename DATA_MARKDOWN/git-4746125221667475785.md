
# 『リュウソウジャー』主要キャストがぶっちゃけトーク

Published at: **2019-11-01T12:30:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147797/full/)

　テレビ朝日系で放送中のスーパー戦隊シリーズ『騎士竜戦隊リュウソウジャー』の主要キャストによる特別映像「リュウソウトーク【TTFC配信Ver.】第1弾」が、東映特撮のアプリサービス「東映特撮ファンクラブ（TTFC）」で3日午前10時より配信される。

        【写真】その他の写真を見る
      
　出演は、一ノ瀬颯（リュウソウレッド／コウ役）、綱啓永（リュウソウブルー／メルト役）、尾碕真花（リュウソウピンク／アスナ役）、小原唯和（リュウソウグリーン／トワ役）、岸田タツヤ（リュウソウブラック／バンバ役）。司会を兵頭功海（リュウソウゴールド／カナロ役）と、金城茉奈（龍井うい役）が務める。 　トーク内容は、苦労の連続だった5人そろっての初リュウソウチェンジ、悪戦苦闘のアクション、観客の声援に押された舞台あいさつなど、苦難を乗り越えて成長してきたメンバーの素顔が収録されている。 　なお、「ソウルをひとつに！リュウソウチャレンジ！【フットサル編】TTFC先行配信Ver.（前編・後編）」は、今月12日午後11時59分で配信終了となる。「リュウソウチャレンジ！【フットサル編】完全版」や別バージョンの「リュウソウトーク」、「新番組記者発表会」等を収録した『騎士竜戦隊リュウソウジャー Blu‐ray COLLECTION1』は、今月13日発売。 ■公式サイトhttp://tokusatsu-fc.jp/
