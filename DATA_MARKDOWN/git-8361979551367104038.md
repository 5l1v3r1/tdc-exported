Боксеры провели пресс-конференцию, после чего традиционно посмотрели друг другу в глаза.
На кону поединка будет стоять титул чемпиона мира по версии Всемирной боксерской организации (WBO), которым владеет россиянин.

Канело Альварес провел 55 поединков среди профессионалов, в 52-х из которых побеждал. Два боя 29-летний боксер свел вничью, один проиграл.
Ковалев провел 38 боев среди профессионалов, в 34-х из которых побеждал. Один бой 36-летний боксер свел вничью, в трех проиграл.
