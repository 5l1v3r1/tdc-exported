
# НБУ: Официальный курс гривны установлен на уровне 24,65 грн/доллар

Published at: **2019-11-07T08:03:00+00:00**

Author: ****

Original: [ZIK.UA](https://zik.ua/ru/news/2019/11/07/nbu_ofytsyalniy_kurs_gryvni_ustanovlen_na_urovne_2465_grndollar_1685577)


        По состоянию на сегодня, 7 ноября 2019 года, официальный курс гривны установлен на уровне 24,65 грн/доллар.
      
Об этом сообщили на сайте НБУ.
Таким образом, согласно информации, которую предоставил Национальный банк Украины, основные курсы иностранных валют по состоянию на 7 ноября были установлены на таком уровне:
По сравнению с предварительніми сутками курс всех этих валют понизился.
Напомним, вчера, 6 ноября, официальный курс гривны был установлен на уровне 24,72 грн/доллар.
