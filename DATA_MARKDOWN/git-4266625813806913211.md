
# В октябре количество лиц, которые пересекли админграницу с Крымом, уменьшилось на 40%

Published at: **2019-11-06T23:07:00+00:00**

Author: ****

Original: [ZIK.UA](https://zik.ua/ru/news/2019/11/06/v_oktyabre_kolychestvo_lyts_kotorie_peresekly_adminkordon_s_krimom_1685477)


        По данным Министерства по вопросам временно оккупированных территорий и временно перемещенных лиц, в октябре количество лиц, которые пересекли административную границу с аннексированным Крымским полуостровом, уменьшилось на 40% по сравнению с сентябрем.
      
Об этом сообщает пресс-служба ведомства.
Согласно сообщению, в течение октября 2019 году административную границу в Херсонской области в обоих направлениях пересекли 192 204 человека и 28 680 транспортных средств.
По сравнению с сентябрем количество пересечений гражданского населения уменьшилось на 40,56%, а также уменьшилось количество пересечений транспортных средств на 32,15%.
Как отмечается, граждане Украины, которые живут на временно оккупированной территории, ездят на контролируемую территорию для посещения семьи, для закупки необходимых товаров, получения услуг и прочее.
Напомним, в Херсонской области по обращению Одесского филиала "Укрзализныци" временно закроют контрольно-пропускной пункт въезда-выезда "Каланчак" из-за проведения строительных работ.
