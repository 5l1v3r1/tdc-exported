
# Baracoa se suma a la exportación de angula de Cuba: una tonelada cuesta al menos $ 3 millones

Published at: **2019-11-05T17:48:35+00:00**

Author: **Redacción de CiberCuba**

Original: [CiberCuba](https://www.cibercuba.com/noticias/2019-11-05-u1-e199291-s27061-baracoa-suma-exportacion-angula-cuba-tonelada-cuesta-al-menos)

Cuba continúa apostando por manjares como la angula, o cría de anguila, para remontar su maltrecha economía.
Ahora el municipio Baracoa de Guantánamo se suma a la cría de este cotizado plato que puede llegar a costar una tonelada entre tres y cinco millones de dólares, según apunta el medio oficialista Solvisión. 
La angula o rula puede valer en países como España "según fuentes oficiales, se comercializa entre 3000 y 5000 dólares. Solo un kilogramo puede alcanzar los seis mil euros", agrega el citado medio.
Guantánamo se une con el Centro de beneficio y empaque perteneciente a las Unidad Empresarial de Base Baramar a otras provincias que exportan esta exquisitez marina que no está disponible para el cubano de a pie.
