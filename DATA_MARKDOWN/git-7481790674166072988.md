
# ＲＣＥＰ、年内の妥結見送り…インドは参加せず

Published at: **2019-11-04T15:19:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/economy/20191104-OYT1T50164/)

　【バンコク＝杉目真吾、山村英隆】日中韓やインド、東南アジア諸国連合（ＡＳＥＡＮ）など、東アジア地域包括的経済連携（ＲＣＥＰ）の交渉に参加する１６か国は４日、バンコクで首脳会議を開き、目標としていた年内妥結を見送った上で、２０２０年中の署名を目指し、インドを除く１５か国が手続きを進めるとする共同声明を発表した。
　共同声明では、１６か国のうち１５か国が、全２０分野の交渉を終了したと明記した。
　一方、インド外務省幹部は、首脳会議後に開いた記者会見で、ＲＣＥＰに参加しないと他国に伝えたことを明らかにした上で、「インドの国益のためには正しい決断だ」と強調した。
　インドはこれまで、関税引き下げや撤廃によって中国などから安い物品が大量に流れ込むことを懸念し、ＲＣＥＰに慎重な姿勢だった。インドは多くの人口を抱える巨大市場で、インドが不参加となれば、ＲＣＥＰの枠組みが根底から覆る可能性がある。
　ただ、共同声明では「全ての参加国が、インドの課題解決のため共に作業していく」とし、１６か国での署名を目指す姿勢も示した。
