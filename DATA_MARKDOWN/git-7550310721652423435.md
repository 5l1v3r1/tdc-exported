
# Минстрой: только 23% российских городов имеют благоприятную среду для жизни

Published at: **2019-11-03T14:03:00+00:00**

Author: ****

Original: [Новая Газета](https://www.novayagazeta.ru/news/2019/11/03/156619-minstroy-tolko-23-rossiyskih-gorodov-imeyut-blagopriyatnuyu-sredu-dlya-zhizni)

Только 23% городов России имеют благоприятную среду для жизни. Об этом свидетельствуют данные первого индекса качества городской среды в России, разработанного Минстроем, компанией «Дом.РФ» и КБ «Стрелка», сообщает РИА Новости.
Качество городской среды определялось по 36 индикаторам, среди которых, например, благоустройство, состояние общественного транспорта, безопасность дорог и качество общественных пространств. Каждый индикатор оценивался по десятибальной шкале, соответственно максимальное количество баллов, которые мог набрать город — 360.
«Если город набирает больше 180 баллов, то он имеет благоприятную среду», — сказал на презентации глава Минстроя Владимир Якушев. По его словам, среднее значение этого показателя по стране составляет 163 балла.
«Ведомости» отмечают, что ни один город из 1114 участвовавших в исследовании не набрал наивысшего балла. Лидером по количеству баллов стала Москва (276), далее идет подмосковный Реутов (252) и Санкт-Петербург (238). В десятку также вошли еще шесть городов Подмосковья: Видное, Красногорск, Одинцово, Истра, Мытищи и Долгопрудный, пишет газета.
Городом с самой неблагоприятной средой стал самый северный город Якутии Верхоянск (77 баллов). В десятку городов с низким рейтингом также вошел город-миллионник Омск (104 балла).
Результаты исследования будут использоваться для расчета субсидий регионам из федерального бюджета на поддержку программ формирования современной городской среды, а также станут критерием оценки эффективности губернаторов. Вице-премьер Виталий Мутко заявил, что к 2024 году процент городов с благоприятной средой должен увеличиться до 50%.
