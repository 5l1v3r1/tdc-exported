
# Названа причина массового мора рыбы в Херсонском карьере

Published at: **2019-11-06T13:21:00+00:00**

Author: ****

Original: [Украинская правда](https://www.pravda.com.ua/rus/news/2019/11/6/7231174/)

Причиной массового замора рыбы в Днепро-Бугском лимане Херсонской области назвали аммиак, который выделился из-за природных, а не через техногенных процессов.
Источник: заявление заведующего кафедрой водных биоресурсов и акватории Херсонского государственного аграрного университета Павел Кутищев, "UA.Новости Суспильне"
Детали: По его словам, выделение вредного вещества произошло через естественные, а не через техногенные процессы.
Дословно: "Есть мнение, что удушье рыбы - явления техногенного характера. Мы утверждаем, что удушье рыбы - явление естественное, которое по Украине наблюдается по всем городам вследствие повышения температуры климата. Это следствие экологических изменений, трансформации Днепра каскадом водохранилища".
