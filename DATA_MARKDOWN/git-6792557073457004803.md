
# «Страшно», «Стыд» и «Нелюбовь»: группа Shortparis выпустила новый альбом

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [«Бумага»](https://paperpaper.ru/papernews/2019/11/01/strashno-styd-i-nelyubov-gruppa-s/)

Петербургская группа Shortparis выпустила новый альбом под названием «Так закалялась сталь».
Альбом состоит из девяти треков. Два из них, «Стыд» и «Страшно», вышли еще до альбома — на эти треки уже есть клипы. Сама группа описывает релиз как «движение от глянцевых форм к протесту».
Презентация альбом состоится в ходе концерта 16 ноября в СК «Юбилейный».
