
# В воскресенье в «Орленке» открывается крытый каток

Published at: **2019-11-01T05:33:00+00:00**

Author: **Александр Волков**

Original: [www.chitaitext.ru](https://www.chitaitext.ru/novosti/v-voskresene-v-orlenke-otkryvaetsya-krytyy-katok/)

В воскресенье, 3 ноября, во дворце спорта «Орленок» открывается крытый каток. В режиме массового катания он будет работать только по воскресеньям по два часа – с 17 часов до 19:45. Это два сеанса по 45 минут.
Администрация дворца спорта сообщает, что при покупке билетов сразу на два сеанса будет скидка.
