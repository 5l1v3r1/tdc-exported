
# У Протопопівці ремонтують пожежну частину

Published at: **2019-11-06T00:00:00+00:00**

Author: **Наталка Маринець**

Original: [Кировоградская правда](https://pravda-kr.com.ua/najsvizhishe/16618-u-protopopivtsi-remontuyut-pozhezhnu-chastinu.html)

У селі Протопопівка Приютівської громади за кошти селищного бюджету ремонтують приміщення пожежної команди, повідомляють у громаді.Наразі поклали плитку на підлогу, поштукатури і пофарбовали стіни та стелю. Встановлено нові меблі.
Зробили опалення у приміщенні для зберігання і технічного обслуговування пожежних автомобілів.Нагадаємо, у Приютівській громаді придбали по одному комплекту мультимедійного обладнання для Ізмайлівської та Войнівської шкіл, а також два комплекти для Користівського ліцею
