
# Мальта конфисковала 2 контейнера с деньгами для ливийского генерала, напечатанными россиянами

Published at: **2019-11-03T10:09:00+00:00**

Author: **InfoResist**

Original: [InfoResist](https://inforesist.org/malta-konfiskovala-2-kontejnera-s-dengami-dlya-livijskogo-generala-napechatannye-rossiyanami/)

В Мальте конфисковали крупную партию неофициальных ливийских банкнот, которая, возможно, предназначались генералу Халифу Хафтару, армия которого ведет наступление для смещения поддерживаемого ООН правительства в Триполи. Об этом сообщает The Guardian.
Два контейнера по 56 кубических метров, заполненные недавно введенной валютой, были обнаружены в конце сентября, когда корабль зашел в один из портов острова.
Предположительно, деньги поступили от монетного двора, который принадлежит россиянам. Он печатал деньги для непризнанного ливийского правительства, базирующегося в восточном городе Байда. Утверждается, что деньги могли быть использованы для разжигания войны в Триполи.
Ливия была охвачена гражданской войной между востоком и западом страны, которая привела к созданию конкурирующих институтов, включая правительства.
Хафтар бомбит столицу Ливии Триполи с весны, пытаясь сместить правительство национального согласия. Мальта признает правительство в Триполи во главе с Файезом аль-Сарраджем.
Хотя Россия заявляет о своей нейтральности по Ливии, Москва разрешила восточному правительства напечатать миллиарды ливийских динаров в России, некоторые из которых могли быть использованы для финансирования армии Хафтара и ее военных действий.
