
# Metro de Santiago comenzará sus operaciones desde las 06:00 este miércoles

Published at: **2019-11-05T00:00:00+00:00**

Author: **David Muñoz**

Original: [www.adnradio.cl](https://www.adnradio.cl/noticias/nacional/metro-de-santiago-comenzara-sus-operaciones-desde-las-0600-este-miercoles/20191105/nota/3975275.aspx)

Durante este martes, se dio a conocer que la apertura del Metro de Santiago para este miércoles 6 de noviembre será a las 06:00 horas, retomando su inicio de horario habitual.Además, el servicio informó que se sumará la apertura de las estaciones Parque Bustamante en la Línea 5 y Ñuble, aunque en este caso solo estará disponible para combinación entre las líneas 5 y 6.Cabe recordar que hay algunas estaciones que durante este martes se encuentran cerradas y sin detención de trenes, como es el caso de La Cisterna, por lo que en la Línea 2 solo hay servicio entre El Parrón a Zapadores.
