
# 連毛巾都沒遮　王柏傑對著范曉萱正面全裸

Published at: **2019-11-04T09:47:00+00:00**

Author: **娛樂組**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191104ent026/)

Netflix首部華語原創影集《罪夢者》正式上線後，引發觀眾迴響熱烈，原本期待的張孝全與賈靜雯的俊男美女組合，居然一下從天堂掉進地獄，出乎讓觀眾意料，另外一組王柏傑與范曉萱的關係也相當戲劇化。因為王柏傑對著范曉萱正面全裸，連毛巾都沒遮，真的太敬業了。
其實在正式上線兩個月前，在陶晶瑩主持的訪談中，就隱約透露了王柏傑正面全裸對著范曉萱的劇情。當時王柏傑在不透露劇情的前提下，表示有全裸的演出，結果被追問「有毛巾遮嗎？」「沒有！」等到《罪夢者》正式上線後，觀眾都拚命檢查，到底這場全裸沒毛巾遮的戲在哪裡。
其實這場戲是呼應王柏傑與范曉萱在戲裡首度相遇時，人在白雪大舞廳，從後方環抱范曉萱緊貼跳舞。等到逃出監獄後，王柏傑、章立衡、周洺甫三人趁著大熱天，脫光洗澡，結果范曉萱在此時出現了，於是眾人紛紛閃避，留下王柏傑正面全裸對著范曉萱。由於王柏傑在全劇身材大放送，連關在監獄裡都脫上衣尬張孝全，瞬間成為觀眾心目中的精壯鮮肉。
另外一條戲劇支線，則是張孝全與賈靜雯這對青梅竹馬，卻因為一樁意外，讓兩人有緣無分、相隔一道監獄鐵窗。加上賈靜雯劇中無人陪產，只能獨自生下小孩，連她都覺得劇中人靜芳應該從這瞬間，從愛意轉變成絕望、甚至帶有恨意。
然而更叫觀眾拍案叫絕的是，在分隔十年後，當初逮捕張孝全的警察郭子乾，居然成了賈靜雯的「好友」，還頻繁出入她家。甚至當張孝全歷經重重障礙，與賈靜雯重逢時，郭子乾也出現了，這讓觀眾忍不住猜測，難道賈靜雯後來跟郭哥在一起嗎？！各種揣測理論滿天飛，呼籲大家最好一口氣追完《罪夢者》，才不會被爆雷！
