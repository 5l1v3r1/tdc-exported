
# JFuture 2019 в ноябре: можно решить две задачи на JVM-языке и получить билет/скидку

Published at: **2019-11-06T09:26:00+00:00**

Author: **Отдел информации dev.by**

Original: [dev.by](https://dev.by/news/jfuture-2019)

Сообщество JProf запустило кодинг-конкурс для тех, кто хочет получить бесплатный билет или 50% скидки на JFuture 2019. Ежегодная международная конференция, объединяющая Java с современными технологиями, наукой и искусством, пройдёт в Минске 15–16 ноября.
JFuture — преемница JavaDay, серии событий, которые были инициированы и поддержаны компанией Oracle по всему миру в 2012 году. В 2018 году организаторы, команда SPACE совместно с сообществами Java Professionals BY и Belarus Kotlin User Group, приняли решение расширить её контекст и сосредоточились на презентации новостей и технологий из всей экосистемы JVM языков и не только. 
Сообщество JProf запустило кодинг-конкурс для тех, кто хочет получить скидку на билет. За успешное решение двух задач участник получит 50% скидку на билет, студент — 100% (при предъявлении студенческого билета). Автор лучшего решения также сможет бесплатно посетить конференцию.

В программу JFuture 2019 вошли два блока докладов:
Список тем JFuture был сформировал по итогам независимого исследования профессиональных интересов среди белорусских девелоперов, работающих с Java и JVM-языками. Например, в 2018 году процентное соотношение участников по профессиональному уровню составило: 30% — Senior developers, 30% — Middle developers, 10% — Junior developers, 10% — Team leads. Эта статистика отразилась на формировании лайнапа, в который вошли доклады разного уровня сложности — для опытных разработчиков и новичков.
Несмотря на всеобщий интерес и хайп вокруг AI, лишь около 10% респондентов сталкивались с ним на практике. Поэтому вместо абстрактного доклада про использование условного TensorFlow на Java один из спикеров JFuture 2019 расскажет о создании приложения, которое можно интегрировать в работу голосовых помощников Google и Alexa. Для этого участникам не нужно обладать особыми знаниями алгоритмов работы нейросетей и систем распознавания голоса.
Опрос показал, что лишь 3% опрошенных регулярно участвуют в OSS-движении (Open Source Software). Но при этом практически все девелоперы пользуются открытым софтом и библиотеками. Поэтому в лайн-ап конференции вошёл доклад Марит ван Дейк — активного контрибьютора в Сucumber — инструмента поддерживающего BDD (Behavior Driven Development) проекты.
Также в программе доклад Аманды Хинчман, senior инженера 47Degrees: «Exploring Neural Networking with Kotlin Metaprogramming: a Cheaper Alternative for Deep Learning?». Он потребует широкого технического кругозора от слушателей, базового понимания мета-программирования и некоторого опыта в области машинного обучения.
