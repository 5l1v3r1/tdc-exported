
# పవన్‌ను దూషించడం తగదు: రామకృష్ణ

Published at: **2019-11-04T17:06:53+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=946250)

ADVT
అమరావతి: ఇసుక సమస్యపై ప్రశ్నించిన పవన్‌ కళ్యాణ్ ను మంత్రులు దూషించడం తగదని సీపీఐ నేత రామకృష్ణ చెప్పారు. 5 నెలలుగా ఇసుక సమస్య ఉన్నా ప్రభుత్వం పట్టించుకోలేదన్నారు. ప్రభుత్వానికి చిత్తశుద్ధి ఉంటే ఉచితంగా ఇసుక సరఫరా చేయాలని ఆయన పేర్కొన్నారు. చనిపోయిన కార్మికుల కుటుంబాలకు రూ.10 లక్షల చొప్పున ఇవ్వాలని డిమాండ్ చేశారు. అలాగే కార్మికులకు రూ.20వేలు ఇవ్వాలన్నారు.
