
# Тренер ФК Бавария Ковач отправлен в отставку

Published at: **2019-11-03T20:07:53+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-03/trener-fk-bavariya-kovach-otpravlen-v-otstavku)

Нико Ковача отправили в отставку с поста главного тренера футбольного клуба "Бавария. Об этом информирует официальный сайт немецкого клуба.
Известно, что соглашение с ним было расторгнуто по взаимной договоренности.
Нико Ковач находился на посту главного тренера "Баварии" с лета 2018 года.
