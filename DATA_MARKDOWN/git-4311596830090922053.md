
# Пашинян заявил о возможности референдума в Армении по вопросу Карабаха

Published at: **2019-11-03T21:59:00+00:00**

Author: ****

Original: [EADaily](https://eadaily.com/ru/news/2019/11/03/pashinyan-zayavil-o-vozmozhnosti-referenduma-v-armenii-po-voprosu-karabaha)

Премьер-министр Армении Никол Пашинян, выступая в прямом эфире в социальной сети Facebook, заявил о том, что одним из методов принятия решения по проблеме Нагорного Карабаха может быть проведение референдума.
Глава армянского правительства заявил о манипуляциях по карабхской теме, в частности относительно перспективы «сдачи Карабаха», на которую ссылаются представители оппозиции. Согласно Пашиняну, в настоящее время, в результате усилий его правительства, напряженность на линии соприкосновения снизилась, однако никаких конкретных договоренностей с Баку не достигнуто.
Премьер-министр Армении заверил, что правительство представит на обсуждение конкретный план с вариантами решения карабахского конфликта, в том случае если таковой будет выработан.
