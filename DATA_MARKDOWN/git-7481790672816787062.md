
# マリ襲撃事件でイスラム国が犯行声明

Published at: **2019-11-03T09:01:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/world/20191103-OYT1T50135/)

　【カイロ＝酒井圭吾】アフリカ西部マリで兵士ら５０人以上が死亡した襲撃事件で、イスラム過激派組織「イスラム国」は２日、インターネット上で犯行声明を出した。事実であれば、アブイブラヒム・ハシミ・クラシが新指導者を名乗ってから初めてのテロとなる。
　声明では、「（イスラム教の預言者の後継者を意味する）『カリフ』の兵士が７０人を殺害した。現地での戦いは続いている」と主張した。「イスラム国」は１０月３１日に前指導者のアブバクル・バグダーディの死亡を認め、各国の共鳴者に報復テロを呼びかけていた。
