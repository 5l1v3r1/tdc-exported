
# 散歩中、波にさらわれたか…不明の３歳児遺体を沖合で発見

Published at: **2019-11-01T04:51:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191101-OYT1T50177/)

　１日午前７時１０分頃、千葉県南房総市和田町の和田漁港から約２・２キロの沖合で、男児が浮いているのを海上保安庁のヘリコプターが発見した。男児は、その場で死亡が確認された。
　館山署などによると、１０月３１日午後７時２５分頃、近くの海水浴場で、「散歩中に３歳男児が行方不明になった」と母親から１１０番があり、県警や海保は男児が波にさらわれたとみて、海上を捜索していた。同署が発見された男児の身元確認を進めている。
