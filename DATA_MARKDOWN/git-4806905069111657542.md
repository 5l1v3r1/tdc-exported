
# Журнал, 1927, Франция

Published at: **2019-11-01T12:20:29+00:00**

Author: **hovan**

Original: [historyscan.d3.ru](https://historyscan.d3.ru/zhurnal-1927-frantsiia-1870268/)

Этот комикс был опубликован в еженедельном литературном журнале «Иллюстрированная Россiя», выходившем в Париже с 1924 по 1939 год. Точную дату публикации найти не удалось, но по косвенным признакам 1927–й наиболее вероятен.Примечательный факт: с 1929 года журнал был организатором конкурсов красоты «Мисс Россiя».
