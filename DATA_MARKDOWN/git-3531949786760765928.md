
# СМИ: НАТО отказалось от встречи Столтенберга с освобожденными украинскими моряками

Published at: **2019-11-02T12:02:00+00:00**

Author: **InfoResist**

Original: [InfoResist](https://inforesist.org/smi-nato-otkazalos-ot-vstrechi-stoltenberga-s-osvobozhdennymi-ukrainskimi-moryakami/)

Украинская сторона предлагала организовать в Одессе встречу генерального секретаря НАТО Йенса Столтенберга с освобожденными из России украинскими моряками, но в Альянсе отказались. Об этом пишет «Зеркало недели» со ссылкой на свои источники.
По информации издания, против была Франция, которая не хочет «раздражать» Россию встречей, которая, по мнению Парижа, может «сорвать мирный процесс и негативно повлиять на организацию переговоров в «нормандском формате».
Напомним, ранее генеральный секретарь Альянса Йенс Столтенберг заявил, что на сегодняшний день для Украины нет смысла подавать новую заявку на получение Плана действий по членству в НАТО.
