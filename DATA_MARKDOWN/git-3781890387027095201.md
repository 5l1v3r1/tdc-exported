
# 7 ผู้นำอาเซียนบอยคอตประชุมร่วมกับผู้แทนสหรัฐ

Published at: **2019-11-04T05:00:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/foreign/739898)


        นายโรเบิร์ต โอไบรอัน ผู้แทนพิเศษของประธานาธิบดีโดนัลด์ ทรัมป์ กล่าวว่าผู้นำสหรัฐเชิญผู้นำอาเซียนร่วมประชุม "เป็นกรณีพิเศษ" ที่อเมริกาในปีหน้า พร้อมทั้งยืนยันรัฐบาลวอชิงตัน "ไม่มีทางทอดทิ้ง" ภูมิภาคแห่งนี้ แต่ผู้นำอาเซียน 7 ประเทศไม่เข้าร่วมประชุมกับโอไบรอัน
      
สำนักข่าวต่างประเทศรายงานจากกรุงเทพมหานคร เมื่อวันที่ 4 พ.ย. ว่าบรรยากาศของการประชุมอาเซียน-สหรัฐ ครั้งที่ 7 ซึ่งมีขึ้นเมื่อวันจันทร์ และเป็นหนึ่งในการหารือกลุ่มย่อยรายการสำคัญของการประชุมสุดยอดสมาคมประชาชาชาติแห่งเอเชียตะวันออกเฉียงใต้ ( อาเซียน ) ที่ไทยเป็นเจ้าภาพในปีนี้ "แตกต่าง" จากหลายปีที่ผ่านมา เนื่องจากผู้แทนของรัฐบาลวอชิงตันไม่ใช่ประธานาธิบดีหรือรองประธานาธิบดี แต่เป็น "ผู้แทนพิเศษในนามประธานาธิบดีสหรัฐ" คือนายโรเบิร์ต โอไบรอัน ที่ปรึกษาด้านความมั่นคงแห่งชาติประจำทำเนียบขาว
