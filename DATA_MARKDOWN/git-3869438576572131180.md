
# ОРВИ в Мурманской области не лютует

Published at: **2019-11-06T00:00:00+00:00**

Author: ****

Original: [Мурманский Вестник](https://www.mvestnik.ru/newslent/orvi-v-murmanskoj-oblasti-ne-lyutuet/)

Эпидемиологическая ситуация по гриппу и острым респираторным вирусным инфекциям (ОРВИ) в Мурманской области спокойная. Показатели заболеваемости находятся ниже значений расчётных эпидпороговых величин во всех возрастных группах. По сравнению с предыдущей неделей отмечается снижение темпа регистрации ОРВИ. Групповых заболеваний ОРВИ не зарегистрировано.Заболеваемость ОРВИ на прошедшей неделе определялась циркуляцией респираторных вирусов не гриппозной этиологии. В области продолжается прививочная кампания против сезонного гриппа. Она завершится в конце ноября.Эпидемиологическая ситуация по гриппу и ОРВИ, ход кампании по иммунизации населения области против сезонного гриппа находятся на постоянном контроле Управления Роспотребнадзора по Мурманской области.
