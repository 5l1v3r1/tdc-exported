
# Фигуристки из России завоевали золото и серебро на этапе Гран-при во Франции

Published at: **2019-11-02T20:42:00+00:00**

Author: ****

Original: [Interfax.ru](https://www.sport-interfax.ru/682799)

Москва. 2 ноября. INTERFAX.RU - Российская фигуристка Алена Косторная стала первой по сумме двух программ на этапе Гран-при во Франции Международного союза конькобежцев (ISU).
За произвольную программу Косторная получила 159,45 балла, по сумме двух программ - 236,00 балла.
Второй стала россиянка Алина Загитова с результатом 216,06 балла.
Тройку лидеров замкнула американка Мэрайа Белл, набравшая 212,89 балла.
Ранее российский фигурист Александр Самарин занял второе место на этом Гран-при.
