
# Казань Аренанинг номи ўзгаради

Published at: **2019-11-07T10:45:00+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/kazan-arenaning-nomi-ozgaradi)

Брифингда хабар берилишича, стадион "Ак Барс Банк Арена" номини олади.
Стадион номининг ўзгариши бир ой ичида амалга оширилади.
Бу стадион Россия Премьер-лигасида иштирок этувчи Қозоннинг "Рубин" клубининг уй стадионидир.
Ушбу стадион ЖЧ-2018 доирасидаги 6та ўйинни ўзида қабул қилган. "Казань Арена"нинг қурилиши учун 14,4 млрд рубль маблағ кетган.
