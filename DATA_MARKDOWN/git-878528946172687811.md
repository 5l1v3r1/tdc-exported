
# 大関貴景勝は九州場所出場

Published at: **2019-11-07T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/general/2019/11/07/0012855528.shtml)

　大相撲の大関貴景勝（２３）＝本名佐藤貴信、兵庫県出身、千賀ノ浦部屋＝が九州場所（１０日初日・福岡国際センター）に出場することが７日、決まった。師匠の千賀ノ浦親方（元小結隆三杉）が明らかにした。
　貴景勝は先場所千秋楽の優勝決定戦で左大胸筋肉離れのけがを負い加療６週間と診断され、大関に復帰した今場所への出場が懸念された。しかし、１０月３１日に相撲を取る稽古を再開すると、関取衆との稽古も行うなど慎重に調整してきた。
　貴景勝は昨年の九州場所では初優勝を果たしている。
