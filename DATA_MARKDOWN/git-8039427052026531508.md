
# గోవా సముద్రతీరంలో ఇండియన్ కోస్ట్‌గార్డ్స్ విన్యాసాలు

Published at: **2019-11-07T06:05:36+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=948655)

ADVT
పనాజీ (గోవా) : గోవా సముద్రతీరంలో ఇండియన్ కోస్ట్‌గార్డ్స్ విన్యాసాలు చేసింది. భారత కోస్ట్‌గార్డ్ గోవాలోని పనాజీలో రెండురోజుల ‘ప్రాంతీయస్థాయి శోధన రెస్క్యూ వర్క్‌షాప్-2019’ను నిర్వహించింది. గోవా సముద్ర తీరంలో కోస్ట్‌గార్డ్స్ నౌకలు, హెలికాప్టర్లతో పలు విన్యాసాలు చేశారు. సముద్రంలో దాడి జరిగినపుడు ఎలా కాపాడాలి అనే అంశంపై ప్రదర్శన ఇచ్చారు. కోస్ట్‌గార్డ్స్ అత్యంత ధైర్య సాహసాలతో చేసిన విన్యాసాలు అందరినీ ఆకట్టుకున్నాయి. సముద్ర తీరంలో ఎలాంటి అత్యవసర పరిస్థితి అయినా ఎదుర్కొనేలా కోస్ట్‌గార్డ్స్ సిద్ధంగా ఉన్నారని ఆ విభాగం అధికారి ప్రకటించారు.
