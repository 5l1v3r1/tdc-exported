
# 'ใหม่'เดือดดราม่าคนยืมเงินลาม ฉะคนเมนต์ด้วยได้ไหม!

Published at: **2019-11-05T06:20:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/entertainment/740051)


        ใหม่-สุคนธวา เดือดอีกรอบ หลังเกิดดราม่าชาวเน็ตมองโพสต์แฉคนยืมเงินเพราะต้องการสร้างกระแส
      
ก่อนหน้านี้เคยออกมาโพสต์รูปภาพข้อความที่มีชาวเน็ตมาขอยืมเงินในมุมขำๆไปแล้ว สำหรับนักแสดงสาวสุดเซ็กซี่ ใหม่-สุคนธวา เกิดนิมิตร ซึ่งก็มีแฟนๆเข้ามากดไลค์ชื่นชอบความตรงๆของสาวใหม่จำนวนมาก แต่ล่าสุดดูเหมือนดราม่าดังกล่าวจะกลับมาอีกรอบ หลังสาวใหม่โพสต์คอมเมนต์ของชาวเน็ตที่เข้ามาวิจารณ์ถึงเรื่องดังกล่าว โดยบอกในทำนองว่าสาวใหม่จะออกมาโพสต์แบบนั้นทำไม ถ้าไม่ชอบก็แค่ไม่ให้ยืมก็จบ บ้างก็ว่าสาวใหม่ต้องการสร้างกระแสให้งานแต่งตัวเองงานนี้เลยทำให้สาวใหม่เดือด พร้อมเขียนแคปชั่นว่า “#เจ้าสาวเดือดอีกแล้ว ต้องการอะไรไม่ทราบคะ จากที่จะด่าXXคนมายืมเงิน ขอเปลี่ยนใจมาด่าคนเมนต์แทนได้ป่ะคะ #ดาราก็คน #ดาราก็มีความรู้สึก #คันปากอยากด่า #แต่ขอสวดมนต์ใส่ดีกว่า #สติสมองจะช่วยหยุดมารได้ #ใกล้แต่งงาน #มารก็จะเยอะนิดหน่อย #เมนต์อะไรเห็นนะ #จะด่ากลับวันไหนก็อีกเรื่องนึง #พูดน้อย #แต่พูดเน้น #ไม่ปิดชื่อนักเลงพอ”
พอสาวใหม่ลงรูปไป ก็มีชาวเน็ตเข้ามาให้กำลังใจจำนวนมาก พร้อมบอกว่าต่างคนต่างคิด อย่าถือสาเลยและให้สาวใหม่สู้ๆอีกด้วยขอขอบคุณภาพประกอบจากอินสตาแกรม @maisukhonข่าวที่เกี่ยวข้อง'ใหม่'งงอ้างนักร้องดังยืมเงิน ตอกแซ่บ!ไม่ขำด้วยนิสัยเสีย
