
# СК РФ разбирается в деле о порезанном восьмикласснике в Кудрово

Published at: **2019-11-02T12:17:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/739043-sk-rf-razbiraetsya-v-dele-o-porezannom-vosmiklassnike-v-kudrovo)


        Следователи СК РФ по Ленобласти проводят доследственную проверку по факту ссоры с применением ножа между школьниками в Кудрово. Одного из них доставили в больницу с травмой живота.
      
Следователи СК РФ по Ленобласти проводят доследственную проверку после драки с ножом между школьниками в Кудрово. Об этом сообщает пресс-служба регионального ведомства.
