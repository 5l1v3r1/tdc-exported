
# 「モネの庭」、夜に色づくモミジやメタセコイア

Published at: **2019-11-05T00:23:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191104-OYT1T50086/)

　高知県北川村の「モネの庭」マルモッタンで、水の庭を中心としたライトアップの催し「光のフェスタ」が開かれている。１２月１日まで土、日曜日と祝日に開かれる。
　約１２０基の照明を使い、色づき始めたモミジやメタセコイアを夜空に浮かび上がらせ、青い明かりで水面や草木を照らすと、幻想的な風景が演出される。
　ライトアップは午後５～９時（入園は午後８時半まで）、入場料は一般７３０円、小中学生３１０円。問い合わせはモネの庭（０８８７・３２・１２３３）へ。
