
# Начальник омской полиции может перевестись в Тюмень

Published at: **2019-11-01T15:20:00+00:00**

Author: **Мария Ильченко**

Original: [РИА ОмскПресс](http://omskpress.ru/news/96430/nachalnik_omskoy_politsii_mojet_perevestis_v_tyume/)

Ротация руководителей силовых структур может продолжиться в регионе.
По информации «СуперОмска», начальник УМВД России по Омской области Леонид Коломиец может перебраться на аналогичную должность в Тюменскую область. Пока в соседнем регионе эту должность занимает Юрий Алтынов.
Отмечается, что Коломиец возглавил УМВД по Омской области в декабре 2016 года. До этого три года служил на посту замначальника УМВД РФ по Астраханской области. В феврале 2018 года присвоено звание генерал-майора полиции.
