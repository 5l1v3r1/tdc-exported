
# 俄羅斯主權網路法生效 數位鐵幕恐加速成形

Published at: **2019-11-03T16:42:00+00:00**

Author: **https://www.facebook.com/cnanewstaiwan, 中央通訊社**

Original: [中央社 CNA](https://www.cna.com.tw/news/firstnews/201911030122.aspx)

（中央社莫斯科2日綜合外電報導）俄羅斯備受爭議的新法「主權網路法」（sovereign internet）昨天生效，網路自由會不會就此走味，讓數位鐵幕更加速成形，科技公司和網路用戶都在密切關注。
美國有線電視新聞網（CNN）報導，俄國早對網路設下重重限制，總統蒲亭（Vladimir Putin）今年稍早把相關規定簽署成法律，允許成立自外於全球的國家網路；這條法律也允許「俄羅斯國家資訊科技、通訊及大眾傳媒監察機構」（Roskomnadzor）關閉外部網路交流，創造「純俄」網路。
俄國政府先前表示，這種種規定都是為了保護國家，培養即便外國勢力干預，也能維續國家網路的能力。
官方「俄羅斯日報」（Rossiiskaya Gazeta）表示，「主權網路法」不致影響網路用戶，但會「確保俄國網路在遭受威脅時的溝通能力」。
官方邏輯淺顯易懂，但批評者警告，由於「主權網路法」要求所有網路提供者安裝「俄羅斯國家資訊科技、通訊及大眾傳媒監察機構」的特殊軟體，恐讓俄國政府更易監測、重新導向與關閉網路，也更容易阻擋政治敏感內容。（譯者：鄭詩韻/核稿：陳政一）1081103
