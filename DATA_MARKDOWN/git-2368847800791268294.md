
# MLB肯德瑞克逆轉轟 國民入手首座世界大賽冠軍

Published at: **2019-11-01T12:41:45+00:00**

Author: ****

Original: [NTD 新唐人電視台](https://www.ntdtv.com/b5/2019/11/01/a102698070.html)

【新唐人北京時間2019年11月01日訊】華盛頓國民10月31日作客休士頓太空人主場，一路落後的國民，靠肯德瑞克（Howie Kendrick）逆轉2分全壘打，加上休士頓太空人後援投手不給力，終場以6:2勇奪成軍以來的首座世界大賽冠軍，這7戰也創下都是客場贏球的特殊紀錄。
二局下，古力歐（Yuli Gurriel）相中薛澤（Max Scherzer）拿手的滑球擊出陽春彈，兩出局還進占2、3壘，史普林格（George Springer）揮出左外野難接的飛球，索托（Juan Soto）演出美技，沒讓國民再丟分，太空人以1:0先下一城。
太空人派出的先發投手葛蘭基（Zack Greinke），前三局只用了28球就讓國民9上9下，葛蘭基教科書般的守備能力，也為自己解決危機，頻頻解決打者擊出的滾地球，並策動雙殺。
五局下，太空人再掀波瀾，兩出局後，阿瓦瑞茲（Yordan Alvarez） 被保送上壘，一、二壘有人情況下，柯瑞亞（Carlos Correa）狙擊薛澤滑球，敲出帶有一分打點安打，太空人前5局取得2:0領先。
葛蘭基將拿手曲球，與直球、滑球及變速球交互搭配，迷惑國民強打群，前6局僅被索托敲出一支安打，完全封鎖國民打線。沒想戰況在第七局風雲變色，倫登（Anthony Rendon）擎天一擊，掃出陽春砲，國民終於打破鴨蛋，葛蘭基這時又投出保送，哈里斯（Will Harris）頂替葛蘭基緊急救火，不料他卻提油桶上來救援，被肯德瑞克（Howie Kendrick）大棒擊出兩分彈，國民3:2反超前。
八局上，伊頓（Adam Eaton）被四壞保送後，立馬盜向二壘，索托隨後補上安打，國民再添1分保險分，以4:2繼續保持領先。太空人後援投手群後面幾局守不住陣腳，九局上，國民一出局將壘包填滿，伊頓不客氣擊出帶有2分打點安打，將差距擴大到4分，信心被擊潰的太空人最後半局3上3下，最後無奈看國民在自家主場歡慶勝利。
