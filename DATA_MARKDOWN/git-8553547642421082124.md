
# Azure Synapse Analytics соединит хранилище корпоративных данных и аналитику big data

Published at: **2019-11-05T14:45:00+00:00**

Author: ****

Original: [Компьютерное Обозрение](https://ko.com.ua/azure_synapse_analytics_soedinit_hranilishhe_korporativnyh_dannyh_i_analitiku_big_data_130743)

Сегодня бизнесу необходимо два вида аналитических систем — хранилища данных и озёра данных. Хранилища данных обеспечивают необходимую аналитическую информацию о состоянии бизнеса. Озёра данных помогают выявить важные сигналы, связанные с заказчиками, продуктами, сотрудниками и процессами. Обе системы критически важны, при этом они работают независимо друг от друга, что приводит к принятию необоснованных решений. В то же время бизнесу необходимо получать аналитические выводы на базе всех данных, чтобы сохранять конкурентоспособность и поддерживать развитие целенаправленных инноваций. Может ли единый аналитический облачный сервис устранить этот разрыв и обеспечить необходимую бизнесу гибкость?На конференции Ignite, проходящей на этой неделе в Орландо (шт. Флорида), компания Microsoft представила Azure Synapse Analytics, аналитический сервис, не имеющий границ и соединяющий в себе хранилище корпоративных данных и аналитику больших данных. Он дает предприятиям свободу работать с данными на их условиях, используя бессерверные вычисления по требованию или выделенные ресурсы — в любом масштабе. Azure Synapse объединяет оба этих мира в едином интерфейсе для поглощения данных, управления ими и предоставления их для решения текущих задач бизнес-аналитики и машинного обучения.Отмечается, что Azure Synapse — это следующий этап эволюции Azure SQL Data Warehouse. Компании могут эксплуатировать существующие хранилища на текущем уровне загрузки с помощью Azure Synapse, автоматически применяя новые возможности, доступные сегодня в ознакомительном режиме. Заказчики могут сделать так, чтобы данные работали намного быстрее, продуктивнее и безопаснее, объединяя аналитику из всех источников и хранилищ, а также из аналитических систем больших данных. Партнеры Microsoft могут продолжить создавать решения вместе с вендором, при этом Azure Synapse предложит богатую и динамичную экосистему из таких компаний, как Databricks, Informatica, Accenture, Talend, Attunity, Pragmatic Works и Adatis.С Azure Synapse специалисты в области данных могут с легкостью сотрудничать, создавать, управлять самыми важными данными и анализировать их — и всё это с помощью одного сервиса. Профессионалы в области данных, используя Azure Synapse, смогут получать как реляционные, так и нереляционные данные, измеряемые петабайтами, используя знакомый язык SQL. Для критически важных нагрузок они могут легко оптимизировать производительность всех запросов с помощью интеллектуального управления рабочей нагрузкой, изоляции задач и неограниченного параллелизма.С Azure Synapse активировать бизнес-аналитику и машинное обучение станет намного проще. Сервис глубоко интегрирован с Power BI и Azure Machine Learning, чтобы существенно расширить возможности получения аналитических выводов из всех данных и применить модели машинного обучения во всех  интеллектуальных приложениях предприятия. По увекрениям Microsoft, можно будет существенно снизить время разработки для проектов в области бизнес-аналитики и машинного обучения с помощью безграничного аналитического сервиса, который позволяет бесшовно применить интеллектуальную систему ко всем самым вашим данным — от Dynamics 365 и Office 365 до облачных сервисов SaaS, поддерживающих Open Data Initiative, — и без труда делиться данными в несколько кликов.Можно создавать целостные аналитические решения с унифицированными пользовательскими возможностями. Azure Synapse studio предоставляет объединенное рабочее место для подготовки данных, управления данными, хранения данных, больших данных и задач ИИ. Инженеры, работающие с данными, могут использовать графическую рабочую среду для управления конвейерами данных без программирования. Администраторы баз данных могут автоматизировать оптимизацию запросов. Специалисты по обработке данных могут проводить апробацию концепции в считанные минуты. Бизнес-аналитики получат безопасный доступ к наборам данных и использовать Power BI для создания панелей управления в считанные минуты - и это обеспечит всего один аналитический сервис.Microsoft заявила, что Azure располагает самыми совершенными характеристиками безопасности и конфиденциальности на рынке. И они встроены в ткань Azure Synapse - в частности, автоматическое распознавание угроз и всегда включенное шифрование данных.
