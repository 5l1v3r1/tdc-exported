
# В Брянске на улице Войстроченко строители выкопали авиабомбу

Published at: **2019-11-07T12:52:30+00:00**

Author: ****

Original: [Брянск Today](https://bryansktoday.ru/article/112155)

Сегодня днем, 7 ноября, в МЧС позвонили работники одной из организаций. Они сообщили, что при проведении земляных работ по улице Войстроченко в Советском районе Брянска были обнаружены взрывоопасные предметы времен Великой Отечественной войны.
На место обнаружения выехала группа спецработ Брянского пожарно-спасательного центра. Они изъяли авиабомбу весом в 50 кило. В данный момент боеприпас обезврежен.
