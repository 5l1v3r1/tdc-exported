
# 西非貝南傳海盜攻擊 挪威貨船9名菲藉船員被綁架

Published at: **2019-11-04T12:32:51+00:00**

Author: ****

Original: [NTD 新唐人電視台](https://www.ntdtv.com/b5/2019/11/04/a102699788.html)

【新唐人北京時間2019年11月04日訊】西非國家貝南（Benin）港市科托奴（Cotonou）2日早晨，港口外發生一起海盜綁架貨船上等候入港的9名船員。
港口當局說，貨船「伯尼泰號」（Bonita）在距離港口入口約14公里處遭到攻擊，船上8名船員外加船長被綁架。
伯尼泰號船東挪威的烏格蘭公司（Ugland）證實船員遭綁架事件，稱伯尼泰2日早晨在貝南科托奴港外等候停泊將石膏卸貨時，9名船員被海盜帶走，其他的船員通知地方當局，船在當天稍晚抵達科托奴港。
公司發言人告訴挪威當地媒體，被綁架船員都是菲律賓人。
