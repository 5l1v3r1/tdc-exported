
# Σφοδρή νεροποντή έπληξε την Ηλεία: Έκλεισε η εθνική οδός από πτώσεις δέντρων - ΦΩΤΟ

Published at: **2019-11-04T14:01:00+00:00**

Author: ****

Original: [enikos.gr](http://www.enikos.gr/society/682216/sfodri-neroponti-eplixe-tin-ileia-ekleise-i-ethniki-odos-apo-ptos)

Σφοδρή νεροποντή έπληξε την Ηλεία, με πολλά σημεία των δρόμων του Πύργου και της Αμαλιάδας, να είναι για αρκετή ώρα απροσπέλαστα.
Επιπλέον, πολλά δέντρα έπεσαν στην Εθνική Οδό Πατρών – Πύργου και συγκεκριμένα στο ύψος του Παλαιοχωρίου και των Σαβαλίων, με αποτέλεσμα να διακοπεί η κυκλοφορία για αρκετή ώρα.
