
# Telegram-бот ReporTax научился следить за долгами ФОП

Published at: **2019-11-07T16:30:00+00:00**

Author: ****

Original: [AIN.UA](https://ain.ua/2019/11/07/telegram-bot-reportax-nauchilsya-sledit-za-dolgami-fop/)

Бот-бухгалтер ReporTax, который создают ЛІГА:ЗАКОН и «Юскутум», теперь умеет предупреждать предпринимателей об их долгах по ЕСВ и единому налогу. Об этом редакции AIN.UA рассказали представители компании.
Теперь пользователь будет видеть данные по долгам, переплатам и штрафам каждый раз после уплаты налогов через бот. Информация будет появляться автоматически, без дополнительного запроса.
Еще одно обновление ReporTax — книга доходов предпринимателя. Функция позволяет вносить дату, сумму и валюту дохода в формате диалога. На основе этих данных бота-бухгалтер будет автоматически подсчитывать месячный доход и рассчитывать сумму налогов.
