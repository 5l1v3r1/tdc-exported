
# Син легендарного Холіфілда феєрично дебютував у професіональному боксі (відео)

Published at: **2019-11-04T06:15:32+00:00**

Author: ****

Original: [ГЛАВКОМ](https://glavcom.ua/sport/news/sin-legendarnogo-holifilda-fejerichno-debyutuvav-u-profesionalnomu-boksi-video-637412.html)

Син легендарного американського надважковаговика Евандера Холіфілда Еван дебютував у професіональному боксі. Холіфілд-молодший за 16 секунд нокаутував Ніка Вінстеда.
