
# Ahora puedes usar Visual Studio desde tu navegador

Published at: **2019-11-04T00:00:00+00:00**

Author: **Diario Tecnología**

Original: [Diario Tecnología](https://diariotecnologia.es/ahora-puedes-usar-visual-studio-desde-tu-navegador)

Visual Studio Online ya se encuentra disponible para todo el público, por ahora como una fase de pruebas gratuita.
El pasado mayo, Microsoft anunció que se encontraba trabajando en una versión web de Visual Studio, su popular editor de código. Tras varios meses de pruebas limitadas a ciertos desarrolladores, finalmente los de Redmond han puesto disponible esta herramienta para todo el público. Eso sí, continúa en fase beta, así que probablemente te encuentres con algunos fallos y problemas de rendimiento.
Es importante señalar que su propuesta para el navegador, la cual pretende impulsar su estrategia de trabajo remoto, no solo incluye el entorno de desarrollo (IDE) de Visual Studio, pues de igual manera se ofrece Visual Studio Code con soporte para algunas extensiones populares. Además, es posible enlazar un repositorio de GitHub como tu fuente principal de archivos.
Visual Studio Online es gratuito, por el momento, aunque su sitio web ya muestra algunos precios dependiendo el hardware encargado de realizar el procesamiento en la nube. Otro punto a considerar es que debes tener una cuenta de Microsoft y de Azure; esta última servirá para crear un entorno personalizado de acuerdo a tus necesidades. Puedes comenzar tu prueba desde el siguiente enlace.
Según la compañía, su objetivo principal con Visual Studio Online es que los desarrolladores "sean más productivos que nunca", pero sin preocuparse por el hardware local. Por otra parte, piensan que las herramientas colaborativas cada vez impactan más en el flujo de trabajo de las empresas, por ello deben proporcionar soluciones que permitan mantener la eficiencia en el entorno laboral. Pese a lo anterior, en este momento no tienen planes de sustituir sus aplicaciones nativas de escritorio.
