
# Dropbox запустила файлообменный сервис. Бесплатный лимит — 100 МБ

Published at: **2019-11-06T18:45:00+00:00**

Author: ****

Original: [AIN.UA](https://ain.ua/2019/11/06/dropbox-zapustila-transfer/)

Dropbox официально запустила сервис для быстрого обмена файлами — Dropbox Transfer. Он был доступен ограниченному количеству пользователей с июля 2019 года.
Для пользователей бесплатного тарифа Dropbox совокупный размер трансфера ограничили 100 МБ. Для платных юзеров лимит — 100 ГБ. Также они могут устанавливать собственный логотип и менять фон на лендинге для передачи файлов.
