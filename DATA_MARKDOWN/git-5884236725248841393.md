
# СКА объявил состав игроков в матче с рижским «Динамо»

Published at: **2019-11-03T16:10:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/739424-ska-obyavil-sostav-igrokov-v-matche-s-rizhskim-dinamo)


        3 ноября петербургский хоккейный клуб СКА встретится с «Динамо» из Латвии в Ледовом дворце. Игра стартует в 17:00. Уже известен состав участников.
      
СКА назвал состав игроков в матче с хоккейным клубом из Латвии «Динамо». Список участников опубликовали на официальной странице СКА в социальной сети Twitter.
Новость дополняется.
