
# 「タイタンの学校」太田光代氏の講義がSHOWROOM生配信 生徒のプレゼン大会を実施

Published at: **2019-11-07T15:33:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148227/full/)

　お笑いコンビ・爆笑問題や作家の辻仁成氏が所属する芸能事務所・タイタンのコミュニケーションカレッジ「タイタンの学校」で、理事長を務める太田光代氏の講義「事業の授業」がSHOWROOMで生配信されることが決定した。

　エンタメ業界に限らず、さまざまな個性を持った人材が求められる現代社会において、同校では成功する要素を『面白い＋α』とし、この「＋α」に注目。“チャレンジ”を学ぶことで人の感性を刺激していく同校の特別講師は、光代氏のほか、辻氏やテレビプロデューサーの菅賢治氏ら。レギュラー講師には元フジテレビアナウンサーの山中秀樹をはじめ、イラストレーターの326や大渕愛子弁護士、各種番組の放送作家などエンタメシーンで活躍する人材がそろっている。 　光代氏の講義では、グループワークで商品開発を学習。「ヒット商品には必ず理由がある」「市場は無限大」「おもしろさと実用性の割合」など、半年間で多くのものを学び、各班ワークで商品企画を作成した。そのプレゼン大会を9日の午後2時から、SHOWROOM「タイタンの学校2期」にて生配信する。 ■SHOWROOM「タイタンの学校2期」【総合MC】シティホテル3号室【サポーター】くらげピザ／弟／タカマッチ／くまの子としちゃん【プレゼンター】タイタンの学校2期生　A班〜E班 【内容】各班が新しい商品企画を3分間でプレゼンした上で投票を行い、最優秀賞を決定する。最優秀賞を獲得したプレゼン商品は、商品化の可能性も。
