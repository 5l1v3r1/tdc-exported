
# นาทีหามฝรั่งหิ้วปีกแฟนสาว ยิงตัวเองสาหัส-ก่อนถึงหมอ

Published at: **2019-11-06T07:47:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/crime/740274)


        เปิดคลิปตำรวจหาม "นายบาร์ท" หนึ่งในแก๊งแหกคุก ยิงตัวเองบาดเจ็บสาหัส หลังจับแฟนสาวเป็นตัวประกัน ลั่นไกยิงศีรษะแฟนบาดเจ็บด้วย ก่อนเจ้าหน้าที่ตามพบตัวรีบนำส่งโรงพยาบาล
      
กรณีเจ้าหน้าที่ตำรวจตามจับกุมตัว 3 ผู้ต้องหาที่หลบหนีจากศาลพัทยาได้เรียบร้อยแล้ว โดยนายหน่อย นิลเทศ ได้ขอมอบตัวผ่านการติดต่อจากญาติ และถูกจับกุมตัวในป่าอ้อยตรงข้ามสวนทานตะวัน แหล่งท่องเที่ยวบ้านเขาจาน ต.ท่าเกวียน อ.วัฒนานคร จึงได้ควบคุมตัวมาสอบสวนพร้อมกับญาติผู้ประสานติดต่อให้นายหน่อยมอบตัว ทั้งนี้ผู้ต้องหาที่เหลืออีก 2 ราย คือนักโทษชาวฝรั่งกับแฟนสาว นายบาร์ท อัลเลน เฮลมัส (MR.BART ALLEN HELMUS) และ น.ส.สิรินภา วิเศษฤทธิ์ ในความผิดฐาน “ร่วมกันพยายามฆ่าเจ้าพนักงาน หลบหนีอยู่ไม่ไกลจากกัน และอยู่ในป่าใกล้จุดที่นายหน่อยมอบตัว เจ้าหน้าที่จึงระดมกำลังค้นหาทั้งทางบกและทางอากาศ
