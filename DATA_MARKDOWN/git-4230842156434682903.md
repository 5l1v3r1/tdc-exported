
# Парикмахера Роналду жестоко убили в Цюрихе

Published at: **2019-11-04T14:17:59+00:00**

Author: ****

Original: [ФАКТИ](https://fakty.com.ua/ru/sport/20191104-perukarya-ronaldu-zhorstoko-vbyly-u-tsyuryhu/)

Парикмахера Криштиану Роналду, Рикардо Маркеса Феррейра, убили в одном из швейцарских отелей.
Рикардо Маркеса Феррейра нашли мертвым 2 ноября в одном из отелей Цюриха.
Его тело обнаружила горничная отеля, когда пришла убирать. Труп парикмахера лежал на кровати, залитой кровью, с признаками ножевого ранения.
