
# Врача киевского онкодиспансера поймали на взятке

Published at: **2019-11-05T15:10:00+00:00**

Author: ****

Original: [Украинская правда](https://www.pravda.com.ua/rus/news/2019/11/5/7231078/)

Прокуратура совместно с полицейскими Киева разоблачила врача "Киевского областного онкологического диспансера" на получении взятки в 2 тысячи долларов.
Источник: пресс-служба прокуратуры города Киева
Дословно: "Врач требовал и получил 2000 долларов за предоставление медицинской документации, необходимой для прохождения медико-социальной экспертизы без реального прохождения лицом медицинского обследования".
Детали: По данному факту правоохранители начали уголовное производство по ч. 3 ст. 368 (получение взятки) УК Украины.
