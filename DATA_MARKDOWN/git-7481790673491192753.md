
# 近年は暑い札幌…今年８月、３０度以上が８日間

Published at: **2019-11-01T13:30:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/olympic/2020/20191101-OYT1T50265/)

　東京の暑さを理由に会場が変更されたマラソンと競歩だが、近年は札幌も暑い。
　現行の日程では女子マラソンが８月２日、男子マラソンは同９日に開催される。過去３０年の最高気温の平均は両日とも、東京（３１・１度）より札幌（２６・９度）が、４度以上も涼しい。
　近年、東京都の気温は上昇傾向にあり、五輪開催地に立候補した２０１１年、８月の最高気温の平均は３１・２度だったが、今年８月は３２・８度に。女子マラソン開催１年前の８月２日は、最高気温３５・１度の猛暑日を記録している。
　ただ、札幌も日によって気温が上昇し、今年８月には３０度以上の真夏日が８日間あり、２日は３４・２度になった。この日は、気温や湿度などを基に算出した熱中症の目安「暑さ指数（ＷＢＧＴ）」も東京、札幌とも、日本スポーツ協会が激しい運動の中止などを求める危険域に達した。
