
# Қойил! Уокер ЕЧЛда сўнгги уч йил ичида сейв ижро этган илк инглиз дарвозабони бўлди

Published at: **2019-11-07T06:31:00+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/qoyil-uoker-echlda-songgi-uch-yil-ichida-seyv-ijro-etgan-ilk-ingliz-darvozaboni-boldi)

Иккинчи тайм бошида Клаудио Браво Эдерсонни алмаштирди. Аммо, у ўйин сўнггиларига яқин жарима майдончаси ташқарисида қоидани бузди ва қизил карточка олган ҳолда майдондан четлатилди.
Бунга қадар захирада дарвозабон қолмаган эди. "Сити" дарвозабонлари мураббийининг тавсияси билан Кайл Уокер дарвозани қўриқлаш учун захирадан туширилди.
Шундан сўнг жарима зарбасини Руслан Малиновский ижро этди. Аммо, Кайл сейв қилган ҳолда ЕЧЛдаги сўнгги 3 йил ичида сейв ижро этган илк инглиз дарвозабонига айланди.
