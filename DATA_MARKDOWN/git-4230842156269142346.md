
# Списки опрацьовують, але вони закриті – Пристайко про обмін утримуваних осіб

Published at: **2019-11-01T13:14:49+00:00**

Author: ****

Original: [ФАКТИ](https://fakty.com.ua/ua/ukraine/20191101-spysky-opratsovuyut-ale-vony-zakryti-prystajko-pro-obmin-utrymuvanyh-osib/)

Списки утримуваних осіб для нового обміну з Росією закриті через загрозу зриву процесу.
Міністр закордонних справ України Вадим Пристайко заявив, що списки утримуваних осіб для обміну з РФ закриті через загрозу зриву процесу.
Він також заявив, що новий етап обміну складніший за попередній, оскільки включає як українських політв’язнів у РФ, так і заручників, які незаконно утримуються на окупованих територіях.
