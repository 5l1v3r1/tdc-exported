
# పోర్టు నిర్మాణంపై రాష్ట్ర ప్రభుత్వమే నిర్ణయం తీసుకోవాలి: కేంద్ర మంత్రి

Published at: **2019-11-07T21:45:19+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=948936)

ADVT
విశాఖపట్నం: దుగరాజుపట్నం, మచిలీపట్నం, రామాయపట్నంలో పోర్టు ఎక్కడ నిర్మించాలన్నది రాష్ట్ర ప్రభుత్వమే నిర్ణయం తీసుకోవాలని కేంద్ర మంత్రి మాండవీయ స్పష్టం చేశారు. పోర్టు ప్రాంతాల అభివృద్ధిపై ప్రత్యేక దృష్టి పెడతామన్నారు. కాకినాడ-పుదుచ్చేరి మధ్య జలరవాణా మార్గం సాకారానికి దృష్టిపెట్టామని కేంద్ర మంత్రి చెప్పారు. బ్రహ్మపుత్ర నదిపై డ్రెడ్జింగ్‌కు ఖర్చు కేంద్రమే భరిస్తోందని మాండవీయ స్పష్టం చేశారు.
