
# Тела детей нашли в овраге с водой в Приморском крае

Published at: **2019-11-05T01:10:35+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-05/tela-detey-nashli-v-ovrage-s-vodoy-v-primorskom-krae)

В Приморском крае были обнаружены два тела детей, которых нашли в овраге с водой. Об этом информирует региональное управление СК России.
Инцидент произошел в селе Летно-Хвалынское. В овраге, наполненном водой, были найдены тела 7-летнего мальчика и 3-летней девочки.
По предварительной информации, дети утонули.
По данному факту возбуждено уголовное дело о причинении смерти по неосторожности. Обстоятельства произошедшего выясняются.
