
# ఉద్యమకారుల రక్తం తాగి పాలన చేస్తున్నారు: మోత్కుపల్లి

హైదరాబాద్ : తెలంగాణ ప్రజల పాలిట కేసీఆర్ చీడ పురుగని మోత్కుపల్లి నరసింహులు మండిపడ్డారు. నిజాం పాలనను సర్దార్ పటేల్ అంతమొందిస్తే నయా నిజాం కేసీఆర్‌ పాలనను అమిత్‌షా నాయకత్వంలో అంతం చేస్తామన్నారు. తెలంగాణ ఉద్యమకారుల రక్తం తాగి కేసీఆర్ పాలన చేస్తున్నారని ఆగ్రహం వ్యక్తం చేశారు.
