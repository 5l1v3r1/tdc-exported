
# సీపీఎం రాష్ట్ర కార్యదర్శి మధును పరామర్శించిన సీఎం జగన్

గుంటూరు: సీపీఎం రాష్ట్ర కార్యదర్శి మధును ఆంధ్రప్రదేశ్ ముఖ్యమంత్రి వైఎస్ జగన్‌మోహన్ రెడ్డి పరామర్శించారు. తాడేపల్లిలోని మధు నివాసానికి వెళ్లి జగన్ పరామర్శించారు. కొద్దిరోజుల క్రితం మధు మోకాళ్లకు శస్త్రచికిత్స చేయించుకున్నారు.
