
# สี-มาครง ยึดมั่นข้อตกลงปารีส หลังทรัมป์ถอนตัว

Published at: **2019-11-07T02:00:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/foreign/740383)


        ประธานาธิบดีสี จิ้นผิง และประธานาธิบดีเอมมานูเอล มาครง ประกาศเจตนารมณ์ร่วมกันในการให้ความสำคัญต่อข้อตกลงปารีส หลังรัฐบาลสหรัฐของประธานาธิบดีโดนัลด์ ทรัมป์ ขอลาออกจากการร่วมข้อตกลง นอกจากนี้ จีนและฝรั่งเศสยังตกลงยกระดับความร่วมมือหลายด้านเป็นมูลค่ามากกว่า 450,000 ล้านบาท
      
สำนักข่าวต่างประเทศรายงานจากเมืองเซี่ยงไฮ้ ประเทศจีน เมื่อวันที่ 7 พ.ย. ว่าประธานาธิบดีสี จิ้นผิง ผู้นำจีน และประธานาธิบดีเอมมานูเอล มาครง ผู้นำฝรั่งเศส ออกคำแถลงการณ์ร่วมกัน เมื่อวันพุธ ว่าข้อตกลงปารีส ฉบับปี 2558  ซึ่งเป็นความตกลงตามกรอบอนุสัญญาสหประชาชาติว่าด้วยการเปลี่ยนแปลงสภาพภูมิอากาศ ( ยูเอ็นเอฟซีซีซี ) ว่าด้วยการควบคุมการปล่อยก๊าซคาร์บอน เพื่อรักษาอุณหภูมิเฉลี่ยของโลกให้อยู่ในระดับต่ำกว่า 2 องศาเซลเซียสเมื่อเทียบกับยุคก่อนปฏิวัติอุตสาหกรรมนั้น เป็นสนธิสัญญาระหว่างประเทศที่ "ไม่อาจเปลี่ยนแปลงได้" หลังสหรัฐประกาศเมื่อวันจันทร์ ว่าได้แจ้งต่อสหประชาชาติ ( ยูเอ็น ) เพื่อขอลาออกจากการเป็นสมาชิกอย่างเป็นทางการ
