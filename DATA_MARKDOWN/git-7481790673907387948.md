
# 中国の対米輸出額２５％減…国連報告書「米中摩擦は世界経済の安定性損なう」

Published at: **2019-11-06T01:31:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/economy/20191106-OYT1T50181/)

　国連貿易開発会議（ＵＮＣＴＡＤ）は５日、関税対象の物品について、中国の米国向け輸出額が今年上半期、前年同期比２５％減の９５０億ドル（約１０兆３０００億円）だったとの報告書を発表した。減少額は３５０億ドル（約３兆７９００億円）に上った。米中貿易摩擦で米国による制裁関税の影響が出たという。
　報告書によると、事務・通信用機器で落ち込みが大きく、前年同期比で１５０億ドル減った。一方で、台湾やメキシコ、欧州諸国が米国向け輸出を増やし、これらの国・地域の増加分は２１０億ドルに上った。
　報告書は、制裁関税による価格上昇で米国の消費者が大きな負担を強いられているとしたほか、米中の貿易摩擦が「世界経済の安定性と将来の成長を損なっている」と指摘した。
　報告書は、米国政府の統計をもとに、物品の分野ごとに輸出額を比較するなどして試算した。
　（ジュネーブ支局　杉野謙太郎）
