
# На Украине заявили о готовности страны к прекращению транзита

Published at: **2019-11-04T22:01:46+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-05/na-ukraine-zayavili-o-gotovnosti-strany-k-prekrashcheniyu-tranzita)

Украина готова к прекращению транзита газа с Россией. С таким заявлением в понедельник, 4 ноября, выступил министр энергетики и окружающей среды страны Алексей Оржель.
По его словам, в стране имеется рекордное количество газа для отопительного сезона, с которым Украина может справиться вообще без поставок газа.
"У нас в украинских ПХГ рекордное количество газа, которое закачано, чтобы пройти отопительный сезон", — заявил Оржель в эфире телеканала ICTV.
