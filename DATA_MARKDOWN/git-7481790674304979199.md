
# 「プレミア１２」参加チームが調整…メキシコで１次ラウンドＡ組

Published at: **2019-11-02T15:00:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/npb/20191102-OYT1T50282/)

　【サポパン（メキシコ）＝佐野司】野球の東京五輪予選を兼ねる国際大会「プレミア１２」（読売新聞社協賛）は２日（日本時間３日）、メキシコのサポパンで１次ラウンドＡ組が開幕する。参加４チームは１日、試合会場で最終調整を行って本番に備えた。
　世界ランキングの上位１２チームが参加する同大会は、２０１５年に続く２回目。Ａ～Ｃ組に分かれ、Ａ組は米国、メキシコ、オランダ、ドミニカ共和国が争う。各組を勝ち抜いた計６チームが日本で行われる２次ラウンドに進出する。
　前回大会準優勝の米国は、キャッチボールや打撃練習で体を動かした。ブローシャス監督は初戦のオランダ戦を前に、「競争相手のレベルはよく理解している。ここでプレーするチームは非常に手ごわいが、私たちは東京（２次ラウンド）へ行くつもりでいる」と話した。
