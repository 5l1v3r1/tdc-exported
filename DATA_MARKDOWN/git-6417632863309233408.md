
# Венгер Барселонани инқироз ҳолатида деб ҳисобламоқда

Published at: **2019-11-06T07:30:00+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/venger-barselonani-inqiroz-holatida-deb-hisoblamoqda)

- "Барселона" инқироз ҳолатидаги жамоа каби ўйнамоқда. Уларнинг ўйини жуда ҳам секин. Улар тўпни йўқотишганда, рақиб учун ҳудудларни камайтира олишмаяпти.
Қизиқ, жамоа тарихида Лионель Мессидек футболчидан ташқари жамоавий ўйин бўлган. Ҳозир эса улар Леодан ҳаракат кутишаётганга ўхшайди, - дейди Венгер.
Эслатиб ўтамиз, "Барселона" кеча ЕЧЛ 4-тур доирасида "Славия" (0:0) билан дуранг ўйнади.
