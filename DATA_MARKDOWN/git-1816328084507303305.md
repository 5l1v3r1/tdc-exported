
# டெல்லியில் தொழிற்சாலையில் தீ விபத்து; 3 தீயணைப்பு வீரர்கள் தீக்காயம்

Published at: **2019-11-04T00:00:00+00:00**

Author: ****

Original: [Dailythnathi.com](https://www.dailythanthi.com/News/India/2019/11/04064224/Delhi-Fire-broke-out-in-a-factory-3-fire-fighting.vpf)

புதுடெல்லி,டெல்லியில் பீராகார்ஹி பகுதியில் தொழிற்சாலை ஒன்று இயங்கி வருகிறது.  4 அடுக்குகள் கொண்ட இந்த தொழிற்சாலையில் திடீரென நேற்றிரவு தீ விபத்து ஏற்பட்டது.தொடர்ந்து இந்த தீ அடுத்த கட்டிடத்திற்கும் பரவியது.  இதுபற்றி தகவல் அறிந்து சம்பவ இடத்திற்கு 28 தீயணைப்பு வாகனங்கள் அங்கு சென்றுள்ளன.
தீயை அணைக்கும் பணியில் ஈடுபட்ட 3 வீரர்கள் காயமடைந்தனர்.  அவர்கள் மீட்கப்பட்டு உள்ளனர்.  தொடர்ந்து தீயை அணைக்கும் பணியானது நடந்து வருகிறது.
