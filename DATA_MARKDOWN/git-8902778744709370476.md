
# Паром с российскими туристами сел на мель у берегов Стокгольма

Published at: **2019-11-07T00:00:00+00:00**

Author: **Ирина Архарова**

Original: [Турпром](https://www.tourprom.ru/news/43857/)

В 20:00 в среду знаменитый туристический паром «Принцесса Анастасия», который обслуживается российским оператором MOBY SPL, сел на мель возле берегов Стокгольма. На его борту находились более тысячи туристов, впрочем никто из них не пострадал.
По данным спасательных служб, судно село на мель около острова Лидингё вблизи Стокгольма, когда паром следовал в Хельсинки, откуда должен был направиться в Санкт-Петербург. На судне отключилось электричество. К нему направили буксир.
В итоге паром сняли с мели около полуночи по московскому времени. Никто не пострадал.
