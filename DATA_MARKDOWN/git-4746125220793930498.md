
# 『ジャンプ+』無料デジタル雑誌創刊 『左ききのエレン』『花のち晴れ』ら12作品掲載

Published at: **2019-11-05T00:00:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147977/full/)

　漫画アプリ『少年ジャンプ＋』の人気作品を集めた無料デジタル雑誌『ジャンプ＋デジタル雑誌版』（週刊）が5日に創刊された。創刊号は赤と青の2バージョンあり、それぞれ掲載作品が異なる（2号からは1冊に集約）。各電子書店にて毎週火曜日発売、価格は0円。

        【写真】その他の写真を見る
      
　これは、同アプリが今年9月にリリースから5周年を迎えたことを記念して企画されたもので、さらに多くの読者に同アプリ作品を楽しんでもらうため創刊された。 　ラインナップは、アプリで人気No.1の『SPY×FAMILY』、テレビドラマも放送中の『左ききのエレン』、看板作品である『地獄楽』のほか、大人気少女漫画『花より男子』の続編『花のち晴れ〜花男 Next Season〜』など選りすぐりの12作品でスタート。創刊号の「赤」が379ページ、「青」が347ページで、それぞれ6作品掲載される。 ■創刊号「赤」掲載作品『SPY×FAMILY』『サマータイムレンダ』『左ききのエレン』『奴隷遊戯』『姫様“拷問”の時間です』『悪魔のメムメムちゃん』 ■創刊号「青」掲載作品『地獄楽』『青のフラッグ』『花のち晴れ〜花男 Next Season〜』『忘却バッテリー』『阿波連さんははかれない』『スライムライフ』
