
# 台風被害で浸水した交番、ワンボックス車が代役

Published at: **2019-11-01T23:13:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191101-OYT1T50267/)

　長野県警長野中央署は１日、千曲川の堤防決壊で浸水した長野市豊野町豊野の豊野町交番に代わって、ワンボックス車の車内で住民に対応する移動交番を設置した。
　今回の浸水被害で、豊野町交番は天井に近い約２・４メートルの高さまで水が流れ込み使用できなくなった。県警は観光地などに臨時で交番を開設する際に使用するワンボックス車を災害対応としては初めて使用した。
　車内には机と椅子が置かれ、署員２、３人が平日午前９時～午後５時に常駐。この日は午前９時頃、近くの男性（８１）が先月１４日に拾った１万円札１枚を届け出た。男性は「やっと届けられてよかった。何かあったときに駆け込める交番は安心感がある」と話した。
　豊野町交番の加部登志彦所長は「気軽に相談に立ち寄ってほしい。安心の拠点として、地域住民が元の生活を取り戻す一助になれば」と語った。
　同署は５日以降、別のワンボックス車を使って市内の避難所を１日３か所ずつ巡り避難者の相談に応じる。
