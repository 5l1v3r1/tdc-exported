
# 40,185 వజ్రాలతో టాయిలెట్‌

Published at: **2019-11-07T03:29:17+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=948387)

ADVT
షాంఘై, నవంబరు 6: చిత్రంలో కనిపిస్తున్న టాయిలెట్‌ కమోడ్‌ను 40,185 వజ్రాలు, బంగారంతో తయారు చేశారు. దీని విలువ రూ. 8.5 కోట్లు. హాంకాంగ్‌కు చెందిన ఆభరణాల సంస్థ ఆరోన్‌ షమ్‌ దీనిని రూపొందించింది. అత్యధిక వజ్రాలతో టాయిలెట్‌ కమోడ్‌ను రూపొందించినట్లు గిన్నిస్‌ రికార్డుల్లో ఎక్కాలన్న లక్ష్యంలో సంస్థ యజమాని దీనిని తయారు చేశారు. ఇటీవల షాంఘైలో జరిగిన చైనా అంతర్జాతీయ దిగుమతుల ఎగ్జిబిషన్‌లో ప్రదర్శనకు పెట్టారు.
