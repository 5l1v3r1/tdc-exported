
# «ВКонтакте» на один день запустила функцию для борьбы с оскорблениями — предупреждения при попытке оставить негативный комментарий

Published at: **2019-11-04T00:00:00+00:00**

Author: ****

Original: [«Бумага»](https://paperpaper.ru/papernews/2019/11/04/vkontakte-na-odin-den-zapustila-fu/)

В День народного единства соцсеть «ВКонтакте» запустила функцию для борьбы с оскорблениями в комментариях. Эксперимент, призванный предотвратить травлю по национальному признаку и издевательства над подростками, продлится один день.
При попытке написать агрессивный комментарий, пользователь получит совет отказаться от обидных слов. Есть ли в тексте признаки оскорблений, определят алгоритмы соцсети.
По итогам эксперимента соцсеть определит, насколько эффективно механика снижает количество негатива.
