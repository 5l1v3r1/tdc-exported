
# Adobe выпустила Photoshop для айпадов

Published at: **2019-11-04T17:15:36+00:00**

Author: **The Verge**

Original: [Meduza](https://meduza.io/news/2019/11/04/adobe-vypustila-photoshop-dlya-aypadov)

Компания Adobe выпустила версию Photoshop для айпадов. 
Adobe подчеркивает, что приложение является «настоящим Photoshop», и в нем поддерживается большинство функций, которые есть у десктопной версии — в том числе PSD, слои, инструменты для ретуши и композитинг. Приложение также будет поддерживать работу со всеми версиями Apple Pencil.
Приложение было официально представлено на ежегодной конференции Adobe. В компании отметили, что первая версия — «это только начало» и в будущем количество функций будет увеличено.
