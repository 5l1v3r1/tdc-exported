
# Археологи нашли древнее поселение индейцев благодаря дронам

Published at: **2019-11-07T20:18:00+00:00**

Author: ****

Original: [РИА Новости](https://ria.ru/20191107/1560683923.html)

МОСКВА, 7 ноя — РИА Новости. Ученые из университета во Флориде при помощи дронов обнаружили древнее поселение индейцев, которое датировали от 900 до 1200 годами нашей эры, сообщает Daily Mail. Дроны сканировали побережье острова Роли в Мексиканском заливе, благодаря чему удалось составить 3D-карты его поверхности. В результате археологи раскопали руины 37 жилищ, окруженных кольцами из устричных раковин. Также были обнаружены бусы и инструменты для их изготовления. Из раковин морских моллюсков индейцы делали украшения для шеи, кружки и декоративные элементы для одежды, которые находили в разных частях Америки.
