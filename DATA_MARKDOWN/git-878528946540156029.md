
# ロッテ井上　鴨川キャンプメンバー外れる　肋間筋損傷で全治３週間

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/01/0012838909.shtml)

　ロッテ・井上晴哉内野手（３０）が１日からの鴨川秋季キャンプメンバーから外れた。
　１０月３０日、ＺＯＺＯマリンでの秋季練習最終日、左脇腹に違和感を訴えた。翌３１日に東京都内の病院で診察を受けた結果、「左の肋間（ろっかん）筋損傷」で全治３週間と診断された。１日から浦和球場でリハビリを行うことになった。
