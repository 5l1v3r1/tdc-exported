
# 서장은 한국전시산업진흥회장

Published at: **2019-11-07T18:45:04+00:00**

Author: **조선일보**

Original: [Chosun](https://news.chosun.com/site/data/html_dir/2019/11/08/2019110800356.html)

입력 2019.11.08 03:45
서장은 ㈜엑스코 대표이사 사장이 최근 열린 한국전시산업진흥회 임시총회에서 제13대 회장으로 선임됐다. 임기는 내년 8월까지다. 서울시 정무조정실장, 주히로시마 총영사를 지냈다.
