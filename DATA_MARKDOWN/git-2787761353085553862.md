
# Брянскому приюту «Второй шанс» не хватает денег на покупку корма для питомцев

Published at: **2019-11-07T17:13:05+00:00**

Author: ****

Original: [Брянск Today](https://bryansktoday.ru/article/112171)

Брянский приют для животных «Второй шанс» нуждается в финансовой помощи. Сообщение об этом появилось в социальной сети «ВКонтакте» сегодня, 7 октября.
В публикации говорится, что в скором времени сотрудникам приюта нужно будет купить корм для питомцев. При этом над организацией навис огромный долг.
Реквизиты для помощи: номер карты Сбербанка 4276080017306110 на имя Елены Анатольевны Х. Номер Яндекс-кошелька: 410011988156035. Номер QIWI-кошелька: +79065016622.
