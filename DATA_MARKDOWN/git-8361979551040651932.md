
# Девять нардепов хотят взять Ярослава Дубневича на поруки

Published at: **2019-11-01T14:29:00+00:00**

Author: ****

Original: [НВ](https://nv.ua/ukraine/politics/yaroslava-dubnevicha-hotyat-vzyat-na-poruki-novosti-ukrainy-50051124.html)

Об этом сообщает Радио Свобода.
Поручителями согласились стать представители депутатской группы За будущее Виктор Бондарь, Игорь Палица, Тарас Батенко, Анатолий Урбанский, Виктор Балога, Сергей Минько, Антон Киссе, Игорь Колихаев.
Прокуроры САП просят суд арестовать Дубневича с правом внесения залога в размере 93 млн гривен.

31 октября Верховная Рада сняла неприкосновенность с нардепа Дубневича. Генпрокурор Руслан Рябошапка лично сообщил ему о подозрении.
По данным следствия, будучи главой парламентского комитета по вопросам транспорта, Дубневич мог быть причастен к махинациям при проведении госзакупок для нужд Укрзализныци.
