
# Скандал із «Burisma» - поганий знак для української видобувної галузі, - Сіренко

Published at: **2019-11-05T14:53:00+00:00**

Author: ****

Original: [ZIK.UA](https://zik.ua/news/2019/11/05/skandal_iz_burisma__poganyy_znak_dlya_ukrainskoi_vydobuvnoi_galuzi__sirenko_1684117)


        Політичний скандал між Україною та США, в який була залучена компанія «Burisma» - поганий сигнал для української видобувної галузі.
      
Таку думку в ефірі інформаційно-аналітичного проєкту «Чудова четвірка» на телеканалі ZIK висловив експерт ринку нафтопродуктів Олександр Сіренко.
