
# 黄重坤が逆転優勝…ゴルフ・マイナビＡＢＣ

Published at: **2019-11-03T07:36:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/golf/20191103-OYT1T50131/)

　男子ゴルフのマイナビＡＢＣ選手権（７２００ヤード、パー７２）最終日は３日、兵庫ＡＢＣＧＣで行われ、２位で出た黄重坤（ハンジュンゴン）（２７）（韓国）が通算１９アンダーで、２０１５年カシオワールドオープン以来となるツアー通算４勝目を挙げた。１打差の２位に今平周吾、１６アンダーの３位に小平智が入った。
