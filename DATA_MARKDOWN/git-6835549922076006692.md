
# Авангард сообщил, что Морозов не состоит в комитете клуба по селекции

Published at: **2019-11-01T15:44:00+00:00**

Author: ****

Original: [Allhockey.ru](https://allhockey.ru/news/show/368688-Avangard_soobschil_chto_Morozov_ne_sostoit_v_komitete_kluba_po_selekcii)

"Авангард" сообщил, что Алексей Морозов был ошибочно упомянут в качестве представителя комитета клуба по селекции."В новости о создании комитета ХК "Авангард" по селекции был ошибочно упомянут Алексей Морозов, по факту являющийся советником председателя СД клуба Александра Крылова по спортивным вопросам и не имеющий непосредственного отношения к деятельности комитета по селекции", - говорится в сообщении пресс-службы "Авангарда".В состав комитета вошли:Максим Сушинский - президент клуба, председатель комитета;Сергей Белых - генеральный директор;Евгений Хацей - директор по хоккейным операциям;Андреас Йоханссон - скаут по Европе и Северной Америке;Рашид Хабибулин - скаут по России;Вячеслав Ренев - селекционер хоккейной Академии "Авангард".
