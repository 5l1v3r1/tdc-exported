
# На Ямале открыли переправу через реку Пур

Published at: **2019-11-05T12:25:00+00:00**

Author: **Редакция ФедералПресс**

Original: [ФедералПресс](https://fedpress.ru/news/89/society/2358230)


        УРЕНГОЙ, 5 ноября, ФедералПресс. Дорожники открыли зимнюю переправу через реку Пур. Она соединит поселок Уренгой с Коротчаево.
      
Зимняя переправа через реку Пур открыта для проезда транспорта. По ней можно добраться из поселка Уренгой в отдаленный район газовой столицы – Коротчаево. Сборкой понтонного моста и эксплуатацией переправы занимается компания «Ямбургтранссервис».
Переправа собрана из понтонов шириной 12 метров. Восемь метров занимает проезжая часть, по бокам по два метра отгорожено под тротуары. На понтонном мосту установлены освещение и заграждение, сообщает пресс-служба Пуровского района.
Жители Пуровского, Тазовского и Красноселькупского районов могут бесплатно проезжать по понтонному мосту. Для всех остальных водителей проезд по наплавной переправе обойдется в 120 рублей за легковой автомобиль. Стоимость будет расти в зависимости от веса автомобиля. Грузовику придется раскошелиться на большую сумму, она будет зависеть от тоннажа транспортного средства.
В настоящее время через реку Пур ведется строительство моста. Одновременно строители отсыпают подъезды к нему с обеих сторон. Мост соединит жителей поселка Уренгой с газовой столицей.
