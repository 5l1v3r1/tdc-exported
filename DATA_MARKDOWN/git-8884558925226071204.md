
# Pogrešanega 14-letnika so našli

Published at: **2019-11-05T07:28:09+00:00**

Author: **Spletno uredništvo**

Original: [Primorske novice](https://www.primorske.si/kronika/pogresanega-14-letnika-so-nasli)


        Policija je preklicala iskanje 14-letnega Domna Gradišnika. S pomočjo informacij občana so ga našli v zapuščenem objektu v Šentjurju. Z njim je vse v redu, so sporočili s Policijske uprave Celje.
