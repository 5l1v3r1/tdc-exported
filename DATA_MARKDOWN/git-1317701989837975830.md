
# KB4528630 (Manueller Download) + KBKB4528634 als SSU Windows 10 19013.1102 steht für die Insider im Fast Ring und Skip Ahead bereit

Published at: **2019-11-05T06:42:59+00:00**

Author: **moinmoin**

Original: [Deskmodder.de](https://www.deskmodder.de/blog/2019/11/05/kb4528630-manueller-download-kbkb4528634-als-ssu-windows-10-19013-1102-steht-fuer-die-insider-im-fast-ring-und-skip-ahead-bereit/)

Microsoft hat wie angekündigt ein zweites kumulatives Update KB 4528630 für die Windows 10 (20H1) um Fast Ring und Skip Ahead bereitgestellt. Die neue Versionsnummer ist die 19013.1102, genauer die 10.0.19013.1102 vb_release_svc_prod2.
Diesmal enthält das Update eine Korrektur für das Problem mit Remote Desktop. Hier konnte es vorkommen, dass die DWM abstürzte und das Fenster völlig schwarz wurde oder flackerte. Auch ein neues SSU (Servicing Stack Update) wurde bereitgestellt.
