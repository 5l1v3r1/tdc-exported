
# กัลลาร์โดคุมทัพบาร์ซาแทนบัลเบร์เดเดือนหน้า

Published at: **2019-11-07T17:43:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/sports/740554)


        "อาส" สื่อดังสเปนตีข่าว "กัลลาร์โด" กุนซือริเวอร์ เพลท คุมทัพ "บาร์ซา" แทน "บัลเบร์เด" เดือนหน้า
      
เคลาดิโอ บอร์คี อดีตผู้เล่นทีมชาติอาร์เจนตินาชุดแชมป์ฟุตบอลโลก เปิดเผยกับ "อาส" สื่อดังแดนกระทิงดุว่า มาร์เซโล กัลลาร์โด กุนซือทีมริเวอร์ เพลท ในอาร์เจนตินา จะรับงานคุมทัพ "บาร์ซา" บาร์เซโลนา แทนที่ เอร์เนสโต บัลเบร์เด ในเดือนธันวาคมนี้กัลลาร์โด วัย 43 ปี ได้รับการจับตามองจากทีมดังสเปน หลังคุมทัพ ริเวอร์ เพลท คว้าแชมป์ฟุตบอลโคปา ลิเบอร์ตาโดเรส 2018 และพาทีมทะลุถึงรอบชิงดำในปี 2019 เป็นที่เรียบร้อย โดยจะพบกับ ฟลาเมงโก จากบราซิล เตะรอบชิงดำกันวันที่ 23 พ.ย. นี้ด้าน บัลเบร์เด ตกอยู่ภายใต้ความกดดัน หลังคุมทีมลงเตะ 11 นัด แพ้ไปแล้วถึง 3 นัด ในลา ลีกา สเปน ฤดูกาลนี้ โดยแมตช์ล่าสุดบุกพ่าย เลบันเต หมดสภาพ 1-3 ขณะที่ผลงานล่าสุดในยูฟ่า แชมเปี้ยนส์ ลีก ทำได้แค่เสมอกับ สลาเวีย ปราก 0-0 ในถิ่นคัมป์ นู ทำให้โดนแฟนบอลของตัวเองโห่ใส่ ส่งผลให้อนาคตของ บัลเบร์เด กับ "บาร์ซา" ใกล้จะถึงทางตันเต็มแก่.
