
# Airbnb запретила сдавать жилые дома для вечеринок после гибели пяти людей на Хэллоуине в Калифорнии

Published at: **2019-11-03T08:09:49+00:00**

Author: **Anna Begiashvili**

Original: [TJ](https://tjournal.ru/news/124122-airbnb-zapretila-sdavat-zhilye-doma-dlya-vecherinok-posle-gibeli-pyati-lyudey-na-hellouine-v-kalifornii)

В арендованном для вечеринки доме произошла стрельба.
Airbnb запретила сдавать жилые дома для проведения частных вечеринок, сообщил гендиректор компании Брайан Чески. Также сервис «удвоит усилия по борьбе с неразрешёнными сборищами и оскорбительным поведением хозяев и гостей».
31 октября на вечеринке в калифорнийском городе Оринда произошла стрельба, в результате которой погибли пять человек. Дом для празднования Хэллоуина был арендован на Airbnb, на вечеринку собрались более 100 человек.
По словам Чески, компания планирует расширить программу ручной проверки бронирований, которые система отмечает как бронирования с высоким уровнем риска. Для работы с объявлениями о сдаче домов для вечеринок создадут группу быстрого реагирования.
