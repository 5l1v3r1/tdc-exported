
# Турчанка установила рекорд Гиннесса по фридайвингу (видео)

Published at: **2019-11-06T07:58:00+00:00**

Author: ****

Original: [Главное™](https://glavnoe.ua/news/n357008091-turchanka-ustanovila-rekord-ginnessa-po-fridajvingu-(video))

Турецкий фридайвер Шахика Эрджюмен установила новый мировой рекорд, проплыв под водой без ластов 100 метров.
Предыдущая рекордсменка преодолела на десять метров меньше.
