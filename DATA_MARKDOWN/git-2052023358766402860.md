
# Бывшему начальнику тыла камчатского управления МВД заменили реальный срок на условный по делу о пропаже 88 млн рублей

Published at: **2019-11-06T08:17:00+00:00**

Author: ****

Original: [Медиазона](https://zona.media/news/2019/11/06/rzhkv)

​Камчатский краевой суд смягчил приговор бывшему начальнику тыла регионального управления МВД Максиму Рожкову по делу о пропаже 88 млн рублей при строительстве вертодрома. Об этом сообщает «Кам24». 
По информации издания, на строительство базы авиационного отряда специального назначения выделили 200 млн рублей из бюджета. Рожков принял работы и подписал документы об их оплате. 
«Однако в реальности объект не был достроен», — пишет «Кам24». Следствие насчитало невыполненных работ на 88 млн рублей. Куда ушли эти деньги, не уточняется. 
На полковника завели уголовное дело по части 3 статьи 285 УК (злоупотребление должностными полномочиями, повлекшее тяжкие последствия). 
Суд первой инстанции приговорил его к двум с половиной годам колонии. Рожков подал апелляцию, и реальный срок ему заменили условным. Бывшего силовика освободили из-под стражи.
