
# Ολυμπιακός: Τώρα ξεκινά το… αγροτικό του τίτλου

Published at: **2019-11-02T16:45:41+00:00**

Author: **ΤοΒΗΜΑ Team**

Original: [Το Βήμα Online](https://www.tovima.gr/2019/11/02/sports/olympiakos-tora-ksekina-to-agrotiko-tou-titlou/)

Ο Ολυμπιακός έχει ξεκινήσει εξαιρετικά τη φετινή σεζόν, με την ομάδα του Πέδρο Μαρτίνς μετά τις πρώτες 8 αγωνιστικές να φιγουράρει στην κορυφή της βαθμολογίας με 22 πόντους.
Σε 8 παιχνίδια οι Ερυθρόλευκοι έχουν 7 νίκες και 1 ισοπαλία, γεγονός που αποτυπώνει την εξαιρετική δουλειά που έχει κάνει ο Πέδρο Μαρτίνς.

Για να καταφέρουν οι Πειραιώτες, ωστόσο, να πανηγυρίσουν στο τέλος της σεζόν την κατάκτηση του πρωταθλήματος θα πρέπει να πετύχουν τρεις επιμέρους στόχους.
Να είναι αποτελεσματικοί στα εντός έδρας παιχνίδια, που μέχρι στιγμής τα πηγαίνουν εξαιρετικά, να πάρουν τα «εξάποντα» στα ντέρμπι και να είναι προσεκτικοί στις εκτός έδρας δοκιμασίες.
