
# 三菱自、「ｉ―ＭｉＥＶ」など２万台リコール

Published at: **2019-11-07T08:51:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191107-OYT1T50209/)

　三菱自動車は７日、「ｉ―ＭｉＥＶ」など２車種計１万９７９９台（２００９年７月～１９年７月製造）のリコールを国土交通省へ届け出た。国交省によると、ブレーキ部品の構造が不適切なため、水分が入って腐食し、ブレーキの制動距離が長くなる恐れがある。問い合わせは、０１２０・３２４・８６０。
