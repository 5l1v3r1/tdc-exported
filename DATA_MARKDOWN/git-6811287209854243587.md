
# Venezuela fortalece relaciones con Antigua y Barbuda

Published at: **2019-11-04T19:51:23+00:00**

Author: **Narkis Blanco**

Original: [Últimas Noticias](http://www.ultimasnoticias.com.ve/noticias/destacado/venezuela-fortalece-relaciones-con-antigua-y-barbuda/)

La vicepresidente Ejecutiva de la República Bolivariana de Venezuela, Delcy Rodríguez, sostuvo este lunes una reunión con el primer ministro para Antigua y Barbudas, Gaston Browne, con el objetivo de fortalecer las relaciones y acuerdos de cooperación y hermandad entre ambas naciones.
