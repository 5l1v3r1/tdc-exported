
# В WhatsApp изменили настройки приватности

Published at: **2019-11-07T06:36:00+00:00**

Author: ****

Original: [Tengrinews.kz](https://tengrinews.kz/internet/v-whatsapp-izmenili-nastroyki-privatnosti-383501/)

В мессенджере WhatsApp обновились настройки конфиденциальности для групп, передает Tengrinews.kz со ссылкой на РИА Новости.
Теперь у пользователей появится больше возможностей контролировать приглашения в группы. Так, добавить нового участника в чат нельзя будет без его согласия.
Чтобы активировать эту функцию, пользователю в настройках мессенджера нужно перейти в раздел "Приватность/Конфиденциальность", а затем "Группы". Там можно выбирать один из нескольких вариантов порядка добавления в группы - могут ли это делать все пользователи, либо только ваши контакты.
Также существует опция "контакты, кроме", которая позволяет "фильтровать", кто именно из собеседников пользователя может приглашать его в чаты.
