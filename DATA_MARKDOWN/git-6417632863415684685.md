
# Ливерпуль футболчиси Хитой ёки АҚШ клубига ўтиши мумкин

Published at: **2019-11-04T09:51:27+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/liverpul-futbolchisi-xitoy-yoki-aqshga-yol-olishi-mumkin)

Унинг мерсисайдликлар билан тузган шартномаси мавсум якунига қадар амал қилади ва шартнома ҳозирча узайтирилгани йўқ.
Футболчига Хитой ва АҚШ клублари қизиқмоқда. Аммо, "қизиллар" ҳам қишда Лалланага янги шартнома таклиф қилишлари мумкин.
Адам "Ливерпуль"га 2014 йилда "Саутгемптон" клубидан ўтган.
Футболчи 2013 йилдан буён Англия терма жамоаси шарафини ҳимоя қилмоқда.
