
# 7 Cara Memperbaiki Flashdisk yang Tidak Terbaca di Windows 10

Published at: **2019-11-07T22:05:25+00:00**

Author: **Virgina Maulita Putri - detikInet**

Original: [detikinet](https://inet.detik.com/tips-dan-trik/d-4776336/7-cara-memperbaiki-flashdisk-yang-tidak-terbaca-di-windows-10)

Jakarta - Saat ini flashdisk menjadi menjadi solusi penyimpanan data yang banyak digunakan. Harganya yang murah serta bentuknya yang kecil dan portabel memudahkan kita untuk mengakses data di PC atau laptop di mana saja. Sayangnya, metode penyimpanan yang satu ini rentan mengalami kerusakan hingga akhirnya tidak bisa terbaca di komputer. Ini tentu mengesalkan karena kalian pasti menggunakan flashdisk untuk menyimpan data-data penting. Jangan khawatir, berikut adalah tujuh cara untuk memperbaiki flashdisk yang tidak terbaca di Windows 10 yang dirangkum detikINET:1. Gunakan Port USB LainSalah satu solusi yang paling sederhana adalah dengan mencolokkan flashdisk ke port USB lain di PC atau laptop kalian. Karena kualitas port USB juga bisa menurun seiring usia perangkat kalian, dan mungkin masalahnya terletak di port USB, bukan di flashdisk.2. Bersihkan Port USBPort USB juga bisa menjadi kotor dan dipenuhi debu sehingga tidak optimal dalam menghubungkan flashdisk. Coba bersihkan port dengan kuas pembersih untuk menghilangkan debu yang menutupi. Jangan lupa untuk membersihkan konektor USB di flashdisk kalian juga ya.3. Update DriverUntuk memperbaiki flashdisk yang tidak terbaca tanpa perlu diformat adalah dengan memperbarui driver-nya. Untuk update driver flashdisk caranya tekan tombol Windows + X dan pilih opsi Device Manager dari daftar. Ketika Device Manager terbuka, cari opsi Disk drives di daftar yang ada dan klik. Setelah itu cari perangkat USB kalian, klik kanan dan cari opsi Properties paling bawah.Masuk ke tab Driver, cari tombol Update Driver dan klik. Jika PC atau laptop kalian menemukan driver baru untuk flashdisk, driver tersebut akan otomatis diunduh.4. Matikan Fast StartupFitur Fast Startup bisa saja membuat Windows 10 kesulitan membaca flashdisk kalian. Cara menonaktifkannya mudah saja, buka Control Panel dari menu Windows, kemudian pilih opsi Hardware and Sound klik opsi Power Options.Di daftar opsi paling kiri, cari "Choose what the power button does". Klik opsi tersebut kemudian klik opsi "Change setting that are currently available".Cari opsi Fast Startup dan klik hingga tanda centang menghilang. Klik Save changes untuk menyimpan perubahan dan restart PC kalian.5. Uninstall FlashdiskJika sudah mencoba untuk update driver tapi tidak berhasil, kalian bisa memperbaiki flahsdik yang tidak terbaca dengan cara uninstall perangkat. Caranya cabut flashdisk dari PC atau laptop kemudian tekan tombol Windows + X dan pilih opsi Device Manager dari daftar. Ketika Device Manager terbuka, klik opsi Disk drives yang ada di daftar dan cari perangkat flashdisk kalian. Klik kanan dan pilih opsi Uninstall Device. Jika driver berhasil di-uninstall, restart PC atau laptop kalian dan ketika Windows 10 menyala sambungkan kembali flashdisk.6. Uninstall Perangkat TersembunyiKadang perangkat yang tidak terbaca bisa saja tersembunyi di Device Manager. Untuk meng-uninstall perangkat ini, tekan tombol Windows + R, ketik CMD dan tekan Enter.Ketika Command Prompt terbuka, ketik line di bawah ini dan tekan Enter setelah masing-masing line:- set devmgr_show_nonpresent_devices=1- cd- cd windowssystem32- start devmgmt.mscDevice Manager akan terbuka secara otomatis. Klik menu View yang ada di toolbar, kemudian klik Show hidden devices. Buka semua seksi dan hapus entri yang berwarna abu-abu. Setelah selesai, restart PC atau laptop kalian dan hubungan kembali flashdisk ke perangkat.7. Buat Partisi Cara-cara di atas bisa kalian gunakan untuk memperbaiki flashdisk tanpa perlu khawatir menghilangkan data yang ada di dalamnya. Kalau cara ini bisa saja menghilangkan isi flashdisk kalian. Untuk membuat partisi tekan tombol Windows dan ketik Computer Management. Setelah Computer Management terbuka, klik menu Disk Management dan cari flashdisk kalian di daftar. Jika flashdisk belum dipartisi sebelumnya, akan ada indikator Unallocated space. Klik kanan di disk tersebut dan pilih opsi New Simple Volume. Ikuti instruksi untuk membuat partisi. Setelah proses partisi selesai, kalian harus menetapkan huruf khusus untuk perangkat tersebut. Masih di Disk Management, cari flashdisk kalian, klik kanan dan pilih opsi "Change Drive Letter and Paths". Klik Add untuk memberikan huruf khusus, misalnya E, G atau H.
