
# 高橋メアリージュン、妹・ユウと“金さん銀さん”写真「姉妹、長生きしたいね」

Published at: **2019-11-01T15:59:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147836/full/)

　モデルで女優の高橋メアリージュン（31）が10月31日、オフィシャルブログを更新。現在第1子を妊娠中の妹でモデルの高橋ユウ（28）と“金さん銀さん”になった写真を公開した。

        【写真】その他の写真を見る
      
　「今年のダウンジャケット。金さん銀さん」と題したブログにて、宇宙服さながらのインパクトを放つゴールドのダウンジャケットをメアリージュン、シルバーのダウンジャケットをユウが着た2ショットを公開。「姉妹で色違いで今年このダウン着ます^ ^」と宣言した。 　同アイテムについて「昔から好きなアウトドアブランド・mammutのダウン。-20℃にも対応」「すごく動きやすいんです」と紹介し、「寒いのが苦手な私は冬はダウンないとやってけないのでこのダウンに出会えて良かったです！」と心境を明かした。「二人でいたら目立つね」と再度2ショットをアップし、「本当の金さん銀さんのように、姉妹、長生きしたいね^ ^」と願いを込めた。
