
# ลูกสาวซัดทอดถึงแม่แล้ว สารภาพร่วมวงเล่นแชร์ด้วย

Published at: **2019-11-03T06:59:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/crime/739793)


        “แม่มณี” รับสารภาพซัดทอด “บุพการี” ร่วมวงแชร์ด้วย แต่บอกไม่หมดปัดตอบยอดเงินร่วมแชร์เท่าไหร่ ตำรวจจ่อเรียกสอบสวนอีกครั้ง ลั่นหากพบใครเอี่ยวดำเนินการตามกฎหมายทันที
      
กรณีตำรวจออกหมายจับ “แม่มณี” หรือ น.ส.วันทนีย์ ทิพย์ประเวช อายุ 28 ปี พร้อมนายเมธี ชิณภา แฟนหนุ่ม คดีโกงแชร์โดยถูกตั้งข้อหาหนัก 4 ข้อหา ขณะที่สังคมต่างจับจ้องผู้ที่อาจจะเป็นเครือข่ายอีกหลายคนเกี่ยวกับเรื่องนี้วันที่ 3 พ.ย. พล.ต.ท.เจริญวิทย์ ศรีวณิชย์ ผบช.ตร.ภาค 4 เปิดเผยว่า แม่มณียังให้การภาคเสธ ไม่ได้มีเจตนาจะโกงผู้เสียหาย ส่วนมารดาของแม่มณี น.ส.ธวัลรัตน์ ทิพย์ประเวช  จะมีความผิดด้วยหรือไม่นั้น ต้องรอการสอบปากคำผู้เกี่ยวข้องแม่มณีก่อน แต่เบื้องต้นผู้ต้องหายอมรับว่ามารดาได้ร่วมเล่นแชร์ด้วย แต่ไม่ทราบเรื่องตัวเลข ส่วนจะว่ามีคนมีสีอยู่เบื้องหลัง และช่วยในการหลบหนีหรือไม่ เรื่องนี้จะต้องเรียกผู้ที่เกี่ยวข้องมาสืบสวนสอบสวนอีกครั้ง หากพบว่าใครมีส่วนเกี่ยวข้องก็จะดำเนินการตามกฎหมาย รวมถึงข้อมูลเชิงลึกต่าง ๆ ยังไม่สามารถบอกได้ เพราะอยู่ในสำนวนอ่านข่าวที่เกี่ยวข้อง...'แม่มณี'จากดินสู่ดาวร่วงตกสวรรค์ เท้าแชร์ลวงโลกพันล้าน
