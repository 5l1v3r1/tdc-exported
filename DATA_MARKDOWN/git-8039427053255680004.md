
# భారీగా ఎర్రచందనం దుంగలు పట్టివేత

Published at: **2019-11-02T18:15:58+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=944606)

Ph: 88865 47666ప్రవేశం ఉచితం ADVT
అనంతపురం: నగరంలోని పోలీసులు తనిఖీలు నిర్వహించారు. తనిఖీల్లో భాగంగా భారీగా ఎర్రచందనం దుంగలను స్వాధీనం చేసుకున్నారు. 1.64 టన్నుల ఎర్రచందనం, 610 కిలోల సండ్ర దుంగలు స్వాధీనం చేసుకున్న పోలీసులు 27 మంది నిందితులను అదుపులోకి తీసుకున్నారు. కేసు నమోదు చేసి విచారణ జరుపుతున్నారు.
