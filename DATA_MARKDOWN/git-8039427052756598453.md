
# కర్ణాటక ఆర్టీసీ ప్రైవేటు పరం?

Published at: **2019-11-05T02:12:26+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=946567)

ADVT
బెంగళూరు, నవంబరు 4(ఆంధ్రజ్యోతి): కర్ణాటక రోడ్డు రవాణా సంస్థ తీవ్ర నష్టాల బాటలో ఉంది. దీంతో దీనిని ప్రైవేటు పరం చేయాలనే ప్రతిపాదన తెరమీదికి వచ్చింది. దీంతోపాటు మరో 23 ప్రభుత్వ రంగ సంస్థలు కూడా నష్టాల బాటలో నడుస్తుండడంతో వాటిని కూడా వదిలించుకునేందుకు ప్రభుత్వం సిద్ధమవుతున్నట్టు తెలుస్తోంది.
