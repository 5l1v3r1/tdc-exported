
# В Омске первая рабочая неделя в ноябре станет зимней

Published at: **2019-11-05T09:21:00+00:00**

Author: ****

Original: [БК55](https://bk55.ru/news/article/159539/)


        Завтра выпадут осадки в виде мокрого снега.
      
Сегодня, 5 ноября, наступят первые серьезные заморозки — ночью температура воздуха опустится до -12 0 С. По прогнозу в некоторых районах Омской области температура достигнет даже отметки -17 0 С.
В дневные часы будет все же теплее, температура воздуха будет колебаться от 0 до — 5 0 С. Сегодня осадков не ожидается.
Однако уже завтра, 6 ноября, в дневные часы температура составит +3 0 С и местами пройдет мокрый снег.
