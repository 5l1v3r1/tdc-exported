
# По запорожским улицам колесит такси из одноимённого французского фильма (ФОТО)

Published at: **2019-11-02T08:10:00+00:00**

Author: **Владимир Андреев**

Original: [РИА Мелитополь](https://ria-m.tv/news/170298/po_zaporojskim_ulitsam_kolesit_taksi_iz_odnoimennogo_frantsuzskogo_filma_(foto).html)


        Все, кто знаком с франшизой Люка Бессона “Такси” непременно оценят такую уловку одного из запорожских таксистов.
      
Так, уже известная по фильму “Peugeot 406” (модели немного различаются), ездит по улицам Запорожья с марсельской шашкой. Об этом “Голосу Запорожья” стало известно из собственных источников.
А вот как выглядит такси из фильма.
Стоит отметить, франшиза “Такси” стала одной из самых прибыльных для киноистории Франции, принеся в общей сумме более 300 миллионов долларов в прокате.
Остаётся надеяться, что запорожский “двойник” марсельского такси не будет брать разгон, равняясь на Даниэля из фильма, а это – от 200 км/час.
