
# వజ్రాల వ్యాపారి నీరవ్ మోదీకి యూకే కోర్టులో చుక్కెదురు

Published at: **2019-11-06T20:39:13+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=948065)

లండన్: వజ్రాల వ్యాపారి నీరవ్ మోడీకి యూకే కోర్టులో చుక్కెదురైంది. బెయిల్ కోసం దరఖాస్తు చేసుకున్న మరో పిటిషన్‌ను కోర్టు తిరస్కరించింది. పంజాబ్ నేషనల్ బ్యాంకును మోసం చేయడంతోపాటు.. మనీ ల్యాండరింగ్ కేసులో మార్చి నెలలో 48 సంవత్సరాల నీరవ్ మోదీని లండన్ పోలీసులు అరెస్టు చేశారు. బెయిల్ కోసం నీరవ్ మోదీ దాఖలు చేసిన నాలుగు పిటిషన్లను కోర్టు కొట్టివేసిన విషయం తెలిసిందే.
