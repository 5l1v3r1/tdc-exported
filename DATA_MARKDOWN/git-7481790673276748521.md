
# ラグビー「プロリーグ構想」清宮副会長が自民小委員会で説明

Published at: **2019-11-05T09:50:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/etc/20191105-OYT1T50192/)

　日本ラグビー協会の清宮克幸副会長は５日、自民党のスポーツ立国調査会・スポーツビジネス小委員会で、２０２１年秋開幕を目指すプロリーグ構想について説明し、「（発足時は）８チームぐらいでスタートしたい」と実現に意欲を示した。１３日の協会理事会で議論するという。
