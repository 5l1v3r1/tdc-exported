
# Gameforge revela requisitos para versão PC Windows de Kingdom Under Fire 2 (Vídeo)

Published at: **2019-11-07T14:00:14+00:00**

Author: ****

Original: [PCGuia](https://www.pcguia.pt/2019/11/gameforge-pc-windows-kingdom-under/)

A editora de jogos Gameforge publicou, através da loja Steam da Valve, os requisitos para a versão PC Windows de Kingdom Under Fire 2.
O novo título do universo Kingdom Under Fire, que foi desenvolvido pela equipa do estúdio sul-coreano Blueside, mistura elementos dos MMORPGs com estratégia em tempo real. Adicionalmente, este jogo conta com os modos single player e multiplayer online.
Kingdom Under Fire 2 estará disponível nas lojas a partir do dia 14 de Novembro.
Requisitos mínimos:
Sistema operativo: Windows 7 (64-bit) ou uma versão mais recenteProcessador: Intel Core i3-2120Memória: 4GBPlaca gráfica: Nvidia GeForce GTX 750 Ti (2GB)Espaço em disco: 25GB
Requisitos recomendados:
Sistema operativo: Windows 7 (64-bit) ou uma versão mais recenteProcessador: Intel Core i5-4690KMemória: 8GBPlaca gráfica: Nvidia GeForce GTX 960 (4GB)Espaço em disco: 25GB
