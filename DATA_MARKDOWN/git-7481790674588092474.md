
# 英で発見の遺体３９人「全員ベトナム人とみられる」

Published at: **2019-11-02T12:56:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/world/20191102-OYT1T50259/)

　【ロンドン＝緒方賢一】英南東部グレイズで１０月、トラックのコンテナから３９人の遺体が見つかった事件で、地元警察は１日、遺体はベトナム人とみられるとの声明を出した。ベトナム政府と連絡を取り、身元の確認などを進めているという。警察は当初、遺体について「中国人と考えている」と発表していた。
