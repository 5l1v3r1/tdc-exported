
# Путин совершил массовые кадровые перестановки в силовых структурах

Published at: **2019-11-05T08:44:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/vlast/putin_sovershil_massovye_kadrovye_perestanovki_v_silovykh_strukturakh/)

ОтставкиВ частности, с поста руководителя четвертого следственного управления Главного следственного управления СКР уволен Руслан Ибиев, главный инспектором МВД перестал быть Ефрем Романов, Сергей Журавлев больше не первый заместитель начальника ГУ МЧС по Москве, Александр Еремеев покинул пост начальника Главного управления МЧС по Республике Крым.Помимо этого, уволены министр внутренних дел по Карачаево-Черкесской Республике Игорь Трифонов, начальник управления МВД РФ по Тюменской области Юрий Алтынов. Больше не занимают свои посты начальник ГУ МЧС по Кировской области Михаил Лихачев, начальник Управления информационных технологий и связи МЧС РФ Сергей Власов, первый замначальника ГУ по обеспечению охраны общественного порядка и координации взаимодействия с органами исполнительной власти субъектов РФ МВД РФ Александр Мельников, директор и заместитель директора Департамента готовности сил и специальной пожарной охраны МЧС Максим Максименко и Владимир Дежкин.НазначенияПолковник полиции Дмитрий Баранов назначен заместителем начальника ГУ МВД России по Санкт-Петербургу и Ленинградской области. Заместителем главы МЧС России — главным государственным инспектором РФ по пожарному надзору стал генерал-лейтенант внутренней службы Игорь Кобзев. Полковник внутренней службы Александр Кузьмин назначен начальником управления ФСИН по Тверской области. Во главе ФСИН по Республике Башкортостан встал полковник внутренней службы Алексей Чириков.Сергей Карапетян назначен прокурором Смоленской области, Кирилл Левит стал начальником следственного управления СКР по Воронежской области, Вячеслав Сухоруков стал руководителем следственного управления СКР по Бурятии.
