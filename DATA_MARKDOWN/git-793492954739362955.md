
# Депутат Госдумы: Пример Донбасса открыл возможности для революции в сфере предоставления гражданства РФ

Published at: **2019-11-06T21:30:08+00:00**

Author: **therion**

Original: [Новороссия](https://novorosinform.org/794906)

Российские власти планируют внести в Госдуму проект закона об упрощённом получении гражданства России, который предполагает отмену требования отказаться от гражданства другой страны для граждан Германии, Аргентины и других стран. Об этом журналистам сообщил глава комитета Госдумы по делам СНГ, евразийской интеграции и делам соотечественников Леонид Калашников, передает «Царьград».
По его словам, законопроект будет внесён до декабря и станет «революционером в сфере гражданства».
«Законопроект, отменяющий запрет двойного гражданства, будет актуален теперь не только для Донецкой и Луганской народных республик и не только для Украины, такого отказа теперь не будет требоваться», — подчеркнул он.
В начале октября стало известно, что ЕС обязал жителей Донбасса с паспортами РФ получать шенгенскую визу на территории Украины.
Российский сенатор Франц Клинцевич заявил, что требования Евросоюза касательно выдачи виз жителями Донбасса в консульствах на Украине по существу являются издевательством.
9 августа сообщалось, что ЕС готовит руководство по непризнанию российских паспортов для жителей Донбасса.
23 июля Россия упростила процедуру получения вида на жительство для граждан Украины.
Днем ранее сообщалось, что Киев введет санкции против жителей Донбасса, получивших российские паспорта.
