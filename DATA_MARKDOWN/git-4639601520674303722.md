
# Οι κόρες της Ντέμι Μουρ αποκαλύπτουν : «Ήταν σαν να εμφανιζόταν ένα τέρας»

Published at: **2019-11-03T16:56:36+00:00**

Author: **ΤοΒΗΜΑ Team**

Original: [Το Βήμα Online](https://www.tovima.gr/2019/11/03/culture/oi-kores-tis-ntemi-mour-apokalyptoun-itan-san-na-emfanizotan-ena-teras/)

Μια εκ βαθέως συνέντευξη με τις κόρες της έδωσε πριν από λίγες ημέρες η 56χρονη Ντέμι Μουρ με τις Ρούμερ, Σκουτ και Ταλούλα να μιλάνε ανοιχτά για τους εθισμούς της ηθοποιού.
«Ήταν σαν να εμφανιζόταν ένα τέρας. Θυμάμαι πως είχα το άγχος ότι θα μου ορμούσε όταν έβλεπα πως τα μάτια της να κλείνουν λίγο περισσότερο. Ή πως ήταν πολύ στοργική μαζί μου όταν ήταν νηφάλια…»
