
# Саратовцам рекомендовано есть меньше соли

Published at: **2019-11-03T06:00:00+00:00**

Author: ****

Original: [Саратов 24](https://saratov24.tv/news/saratovtsam-rekomendovano-est-menshe-soli/)

При этом стоит учитывать и соль, которая поступает в пищу вместе с хлебобулочными, кондитерскими изделиями и другими покупными продуктами питания. По словам экспертов, россияне превышают норму потребления соли в два раза. Из-за этого возникает риск гипертонии, инфаркта и инсульта. Также Минздрав рекомендует пользоваться не обычной солью, а йодированной.
