
# വാളയാർ കേസിൽ കത്തി നിയമസഭ, തർക്കം; പ്രതിപക്ഷം ഇറങ്ങിപ്പോയി

Published at: **2019-11-05T09:38:00+00:00**

Author: **മനോരമ ലേഖകൻ**

Original: [ManoramaOnline](https://www.manoramaonline.com/news/latest-news/2019/11/05/walayar-case-opposition-parties-protest-at-legislative-assembly.html?utm_source=dlvr.it&utm_medium=twitter)

തിരുവനന്തപുരം∙ വാളയാർ കേസിനെച്ചൊല്ലി നിയമസഭയിൽ തർക്കം. പ്രതിപക്ഷം നിയമസഭയിൽനിന്ന് ഇറങ്ങിപ്പോയി. അടിയന്തര പ്രമേയ നോട്ടിസ് അനുവദിക്കാനാകില്ലെന്ന് സ്പീക്കർ പ്രതിപക്ഷത്തെ അറിയിച്ചു. വിഷയം നിയമസഭയിൽ പലവട്ടം ഉന്നയിക്കപ്പെട്ടതാണ്. അതേസമയം അടിയന്തര പ്രമേയ നോട്ടിസ് പ്രതിപക്ഷത്തിന്റെ അവകാശമാണെന്നു പ്രതിപക്ഷ നേതാവ് രമേശ് ചെന്നിത്തല പ്രതികരിച്ചു. വിയോജിപ്പ് രേഖപ്പെടുത്തേണ്ടതു ഭരണഘടനാപരമായ ബാധ്യതയാണെന്നും പ്രതിപക്ഷ നേതാവു പറഞ്ഞു.

        പ്രതികൾ പാർട്ടിക്കാരെന്നു പെൺകുട്ടികളുടെ അമ്മ
      
വാളയാർ കേസിലെ പ്രതികൾ സിപിഎമ്മുകാരാണെന്ന് ആവർത്തിച്ച് മരിച്ച പെൺകുട്ടികളുടെ അമ്മ. പാർട്ടിക്കാരോടൊപ്പം പ്രതികളെ നിരവധി തവണ കണ്ടിട്ടുണ്ട്. സിബിഐ അന്വേഷണം തന്നെ കേസിൽ വേണമെന്ന് പെൺകുട്ടികളുടെ അമ്മ മനോരമ ന്യൂസിനോടു പറഞ്ഞു. വാളയാർ പീഡനക്കേസില്‍ പാലക്കാട് പോക്സോ കോടതി വിധി റദ്ദാക്കണമെന്നാവശ്യപ്പെട്ട് പെണ്‍കുട്ടികളുടെ അമ്മ നാളെ ഹൈക്കോടതിയില്‍ അപ്പീല്‍ നല്‍കും. സിബിഐ അന്വേഷണം ആവശ്യപ്പെട്ട് ഹൈക്കോടതിയിൽ സമർപ്പിച്ച പൊതുതാൽപര്യ ഹർജി കഴിഞ്ഞ ദിവസം കോടതി തള്ളിയിരുന്നു.
വാളയാർ കേസിൽ സിബിഐ അന്വേഷണ ആവശ്യം ഇപ്പോൾ പരിഗണിക്കാനാവില്ലെന്നാണു ഹൈക്കോടതി കഴിഞ്ഞ ദിവസം വ്യക്തമാക്കിയത്. വിധി പറഞ്ഞ കേസിൽ എങ്ങനെ അന്വേഷണം നടത്തുമെന്ന് ചോദിച്ച ഹൈക്കോടതി അപ്പീൽ നൽകുന്നതിന് തടസമില്ലെന്നും അറിയിച്ചു. സിബിഐ അന്വേഷണ ഹർജി മാധ്യമ വാർത്തകളുടെ അടിസ്ഥാനത്തിലാണോ എന്നു ചോദിച്ച കോടതി വാർത്തകളിൽ പറയുന്നതുപോലെയാണ് കാര്യങ്ങളെന്ന് എങ്ങനെ ഉറപ്പിക്കുമെന്നും ഹർജിക്കാരനോടു ചോദിച്ചു.
