
# Скандал с «Burisma» - плохой знак для украинской добывающей отрасли, - Сиренко

Published at: **2019-11-05T16:00:00+00:00**

Author: ****

Original: [ZIK.UA](https://zik.ua/ru/news/2019/11/05/skandal_s_burisma__plohoy_znak_dlya_ukraynskoy_dobivayushchey_otrasly__1684131)


        Политический скандал между Украиной и США, в который была вовлечена компания «Burisma» - плохой сигнал для украинской добывающей отрасли.
      
Такое мнение в эфире информационно-аналитического проекта «Великолепная четверка» на телеканале ZIK высказал эксперт рынка нефтепродуктов Александр Сиренко.
