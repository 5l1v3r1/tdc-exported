
# В декабре на орбиту выйдет стеклянный спутник

Published at: **2019-11-05T12:50:00+00:00**

Author: **Эрика Ефремова**

Original: [Ридус](https://www.ridus.ru/news/312026)

Российские ученые запустят на орбиту стеклянный спутник «Блиц-М», который с помощью сферического прозрачного отражателя, предназначенного для получения и отражения обратно лазерных лучей, будет исследовать нашу планету.
Он займется поиском информации о строении недр Земли, а также о «дырах» на нашей орбите.
Несмотря на простое описание стеклянного спутника, его внутреннее устройство довольно-таки сложное. У «Блиц-М» нет ни двигателей, ни электроники.
Сам спутник представляет собой пассивный отражатель лазерных лучей, с помощью которого можно измерять расстояния с точностью до 0,1 мм.
Если все пойдет по плану, «Блиц-М» вместе со спутниками связи «Гонец-М» на ракете-носителе «Рокот» отправятся на орбиту 25 декабря.
