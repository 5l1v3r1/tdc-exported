
# 疑「桐油」加速首里城大火 15秒攝像者成謎

Published at: **2019-11-02T07:51:10+00:00**

Author: ****

Original: [NTD 新唐人電視台](https://www.ntdtv.com/b5/2019/11/02/a102698635.html)

【新唐人北京時間2019年11月02日訊】日本沖繩縣擁有世界文化遺產的首里城，31日凌晨發生大火，火光沖天，火勢一發不可收拾，燒毀包括正殿在內的7棟建築。一段看似近距離拍攝的15秒大火延燒影像在網上熱傳，引發日本網友議論紛紛。另外，專家表示，這場大火之所以延燒如此迅速，可能跟首里城正殿紅漆使用「桐油」及消防設備不足有關。
首里城大火時，這段看似近距離拍攝的影像，是站在城內的「御庭」（奉神門到正殿間的廣場）所拍攝，長度約15秒，除拍攝到正殿、南殿及北殿遭祝融肆虐的畫面，也看到一個拿著像是擴音器物件的人影入鏡。
