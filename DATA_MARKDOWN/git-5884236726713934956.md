
# В Верхнетоемском районе открыли после ремонта два моста

Published at: **2019-11-04T20:29:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Arkhangelsk/739794-v-verkhnetoemskom-raione-otkryli-posle-remonta-dva-mosta)


        Переправы через реки Охтома и Пинега восстановлены благодаря совместной работе областной и районной власти, ГКУ АО «Дорожного агентства «Архангельскавтодор», ООО ПКП «Титан» и предприятию подрядчику АО «АвтодорМост».
      
«Титан» по соглашению о сотрудничестве с правительством региона инвестировал около 3 млн рублей на разработку проектно-сметной документации. Сейчас оба моста введены в эксплуатацию после капитального ремонта.
Данные мосты очень важны для Верхнетоемского района, так как они обеспечивают транспортное сообщение между районным центром и двумя муниципальными образованиями: «Горковским» и «Выйским», отметили в администрации.Финансирование было получено из дорожного фонда по региональной программе «Развитие транспортной системы Архангельской области», утвержденной на период с 2014 по 2020 год.
