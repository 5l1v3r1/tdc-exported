
# 油そば「春日亭 秋葉原昭和通り口店」が本日11月1日OPEN！ 味噌ラーメン専門店「味噌太善」跡地

Published at: **2019-11-02T01:40:00+00:00**

Author: ****

Original: [アキバ総研](https://akiba-souken.com/article/42223/)

油そば「春日亭 秋葉原昭和通り口店」が本日11月1日(金)よりOPEN。場所は昭和通り沿い。JR総武線高架下近くの「名代 富士そば 秋葉原店」となり。味噌ラーメン専門店「味噌太善」跡地。
油そば「春日亭」は、北海道産小麦を100%使用し特注で製粉した油そば用のオリジナル麺と、2日間かけじっくり煮込み旨味を凝縮させた鳥豚スープや、九州地方の醤油を数種類ブレンドした秘伝の醤油を使用した油そばがウリの油そば専門店。秋葉原エリアでは「春日亭 秋葉原店」に続き2店舗となる。また、11月1日(金)～11月3日(日)の間オープン記念として、食事をされた方を対象に次回使用できる「油そば1杯無料券(中盛まで)」が進呈される。なお、オープン3日間の営業時間に関しては閉店時間は21時頃を予定しているとのこと。提供されるメニューは下記のとおり。（※いずれも税込）
・鳥豚油そば 並盛 690円・炙り鳥豚油そば 並盛 890円・しょうゆ油そば 並盛 640円・炙りしょうゆ油そば 並盛 840円・とん黒油そば 並盛 780円・炙り角切り肉しょうゆ油そば 並盛 790円・生卵かけご飯セット 180円<店舗情報>
■油そば 春日亭 秋葉原昭和通り口店■住所：東京都千代田区神田平河町4-8平河町山崎ビル1F■営業時間：11:00～■定休日：なし
