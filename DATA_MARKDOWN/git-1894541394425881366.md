
# Двух участников торгов по закупке шахтного оборудования для Мирноградугля наказали гривной

Published at: **2019-11-06T18:00:00+00:00**

Author: ****

Original: [ОстроВ](https://www.ostro.org/donetsk/society/news/579154/)

Двое участников торгов по закупке шахтного оборудования для государственного предприятия "Мирноградуголь" оштрафованы за согласованные антиконкурентные действия. Об этом сообщает пресс-служба прокуратуры Донецкой области. 
Так, по результатам рассмотрения Донецким областным территориальным отделением Антимонопольного комитета Украины письма прокуратуры Донецкой области о принятии мер по привлечению к ответственности лиц, нарушивших требования Закона Украины "О публичных закупках", отменены торги на закупку подъемно-транспортировочного оборудования для шахт, заказчиком которого является государственное предприятие "Мирноградуголь". 
При изучении материалов торгов, состоявшихся 16 мая 2017 года через систему электронных торгов "ProZorro", прокуратура установила, что победителем признано предприятие, основатель которого входит в орган управления другого участника торгов. Кроме того, указанные субъекты хозяйствования во время участия в торгах осуществляли антиконкурентные согласованные действия.
Сейчас тендерным комитетом ГП "Мирноградуголь" указанные незаконные торги отменены и за искажение их результатов решением административной коллегии отделения АМКУ на обоих участников наложен штраф на общую сумму 136 тыс грн.
