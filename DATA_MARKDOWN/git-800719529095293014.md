
# ‘Casados à Primeira Vista’. Anabela regressa e Lucas arrasa-a em frente a todos

Published at: **2019-11-03T22:14:25+00:00**

Author: **Manuela Lopes**

Original: [A Televisão](https://www.atelevisao.com/sic/casados-a-primeira-vista-anabela-regressa-e-lucas-arrasa-a-em-frente-a-todos/)


        No jantar de ‘Casados à Primeira Vista’, que antecede a cerimónia de compromisso transmitido este domingo, 3 de novembro, houve um regresso que deixou Lucas surpreendido.
      
Anabela havia ‘fugido’ do programa, deixando Lucas sozinho, mas este domingo, surpreendeu todos ao aparecer para o jantar. Assim que Lucas deu de caras com ela disse: “Quem é vivo sempre aparece”. Porém a concorrente aproveitou ainda para explicar: “Tive coisas para tratar…. coisas pessoais. Mas agora estou aqui, pronta para a experiência.”
