
# Juventus Lempar Tawaran untuk Isco di Januari 2020

Published at: **2019-11-01T18:40:00+00:00**

Author: **Bola.net**

Original: [Bola.net](https://www.bola.net/italia/juventus-lempar-tawaran-untuk-isco-di-januari-2020-076f4d.html)

Bola.net - Klub Serie A, Juventus diberitakan masih penasaran terhadap sosok Isco. Si Nyonya Tua berencana melepaskan tawaran untuk playmaker Real Madrid itu di bulan Januari nanti.
Sejak musim lalu, nama Isco kerap digosipkan diminati Juventus. Sang playmaker dinilai cocok untuk menambah kekuatan lini serang Juventus.
Pada musim panas kemarin, Si Nyonya Tua sempat mencoba untuk menawar Isco. Namun Real Madrid enggan melepas sang pemain.
Dilansir Calciomercato, Juventus masih menaruh hati terhadap Isco. Mereka tertarik untuk merekrut sang playmaker di bulan Januari nanti.
