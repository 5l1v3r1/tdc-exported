
# Торпедо объявило состав на выездные матчи

Published at: **2019-11-01T09:57:00+00:00**

Author: ****

Original: [Allhockey.ru](https://allhockey.ru/news/show/368659-Torpedo_obyavilo_sostav_na_vyezdnye_matchi)

Нижегородские торпедовцы отправились на выездную серию по маршруту "Хельсинки - Подольск".В предстоящих матчах с "Йокеритом" (2 ноября) и "Витязем" (4 ноября) цвета нашей команды готовятся защищать следующие хоккеисты:Вратари: Андрей Тихомиров, Николай Мольков;Защитники: Артём Аляев, Чарлз Геноуэй, Денис Баранцев, Сергей Зборовский, Георгий Мишарин, Михаил Орлов, Максим Минеев;Нападающие: Дамир Жафяров, Михаил Смолин, Кирилл Ураков, Станислав Бочаров, Михаил Варнаков, Данил Веряев, Владислав Михальчук, Антон Шенфельд, Куинтон Хауден, Джордан Шрёдер, Даниил Ильин, Сергей Гончарук, Андрей Белевич, Тай Рэтти.
