
# Sejam Sebelum Kongres, Benhard Limbong Mundur dari Caketum PSSI

Published at: **2019-11-02T10:38:00+00:00**

Author: **Bola.net**

Original: [Bola.net](https://www.bola.net/indonesia/sejam-sebelum-kongres-benhard-limbong-mundur-dari-caketum-pssi-7da56c.html)

Bola.net - Bernhard Limbong menyatakan mundur dari bursa pencalonan ketua umum (ketum) PSSI periode 2019-2023. Dia menarik diri sejam sebelum Kongres Pemilihan dimulai di Hotel Shangri-La, Jakarta Pusat, Sabtu (2/11/2019) pagi WIB.
Kepada awak media, mantan ketua komisi disiplin (komdis) PSSI ini mengungkapkan alasannya mundur. Dia mengaku sibuk.
"Per hari ini mundur dari bursa pencalonan Ketum PSSI. Saya kira, saya tidak punya waktu karena saya sibuk," ujar Limbong.
"Dengan waktu saya yang sekarang, saya tidak bisa untuk terlibat aktif di PSSI," katanya menambahkan.
