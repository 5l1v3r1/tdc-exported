
# Видеофакт: в центре Киева прорвало трубу с горячей водой. Теперь транспорт ходит по-другому

Published at: **2019-11-05T17:30:00+00:00**

Author: **Александра Шмарова**

Original: [kiev.vgorode.ua](https://kiev.vgorode.ua/news/sobytyia/412578-vydeofakt-v-tsentre-kyeva-prorvalo-trubu)

Сегодня в центре Киева, на пересечении улиц Шота Руставели и Саксаганского, снова прорвало трубу с горячей водой. Скорее-всего, это связано с изношенностью теплокоммуникаций. 
Нескольким автомобилям не повезло. Они оказались в центре событий и провалились в яму. Чтобы перейти дорогу люди вынуждены были прыгать прямо по кипятку. Вся улица в испарениях.
