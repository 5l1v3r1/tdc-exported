
# В Сыктывкаре 7 ноября закроют движение транспорта по Стефановской площади

Published at: **2019-11-06T13:20:00+00:00**

Author: **Комиинформ**

Original: [Комиинформ](https://komiinform.ru/news/188463)

В Сыктывкаре с 7 ноября закрывается движение транспорта по Стефановской площади, сообщает пресс-служба администрации муниципалитета.
Оно было организовано временно на период работ по благоустройству улицы Советская в районе пересечения с улицей Коммунистическая.
Поскольку укладка гранита и иные работы по благоустройству по ул.Советская закончены, а движение транспорта по всей ширине её проезжей части восстановлено, Стефановская площадь закрывается для проезда.
