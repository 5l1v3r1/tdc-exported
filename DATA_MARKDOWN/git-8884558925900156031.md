
# Pokalna tolažba za Oplast

Published at: **2019-11-04T11:22:47+00:00**

Author: **Igor Mušič**

Original: [Primorske novice](https://www.primorske.si/sport/pokalna-tolazba-za-oplast)


        Prvoligaški premor so ekipe dvoranskega nogometa izkoristile za igranje tekem osmine finala slovenskega pokala.
      
Slovenski pokal v dvoranskem nogometu - sicer po pokrovitelju imenovan kot pokal Terme Olimia - je lani osvojil Dobovec, ki je - tako kot polfinalista Litija in škofijski Bronx - neposredni udeleženec četrtfinala. To mesto je bilo zagotovljeno tudi za finaliste iz Maribora, ki pa so že poleti propadli.
Tako je osmina finala dobila še dodaten dvoboj za mesto, ki so ga izpraznili Mariborčani. Uspeh v osmini finala pokala je vsaj začasna tolažba za kobariški Oplast, ki je v prvi ligi še vedno pri dnu. Tokrat so Kobaridci na gostovanju v Zagorju ob Savi zanesljivo opravili z najboljšim drugoligašem, z ekipo Mlinše. Domačini so bolje začeli, a je pred polčasom Kobaridce z izenačenjem v igro vrnil Nejc Kovačič. Ključni trenutki za Primorce so se zgodili v začetku drugega polčasa, ko je za Oplast mrežo hitro dvakrat zatresel Jurij Kikelj.
Četrtfinala se veseli tudi Kix iz Ajdovščine, ki je v domači dvorani v drugoligaškem obračunu izločil Meteorplast iz Ljutomera. Ajdovci so imeli ves čas rezultatsko pobudo, odločilni zadetek pa je po drugem izenačenju gostov dosegel Anej Batistič.
Pokal Terme Olimia, osmina finala (igrali so eno tekmo) - Mlinše : Oplast Kobarid 3:5 (1:1); strelci - 1:0 Džombić (6), 1:1 Kovačič (17), 1:2 Kikelj (25), 1:3 Kikelj (27), 2:3 Močnik Guna (27), 2:4 Kovačič (30), 2:5 Konavec (34., 3:5 Močnik Guna (35); Extrem : Tomaž 1:7, Miklavž : Sevnica 4:6, Kix : Meteoplast 3:2 (2:2); strelci - 1:0 Sukanović (8), 1:1 Cener (10), 2:1 Sluga (14), 2:2 Horvat (20),3:2 Batistič (25), Siliko : Stripy 6:1.
Četrtfinalisti so Dobovec (lanski prvak), Litija, Bronx (polfinalista, vsi neposredno uvrščeni), Oplast, Tomaž, Sevnica, Kix in Siliko.
