
# В Украине заработал онлайн-сервис платных номерных знаков

Published at: **2019-11-05T11:34:15+00:00**

Author: ****

Original: [ФАКТИ](https://fakty.com.ua/ru/lifestyle/auto/20191105-v-ukrayini-zapratsyuvav-onlajn-servis-platnyh-nomernyh-znakiv/)

В центрах МВД стал доступен новый онлайн-сервис для проверки наличия платных номерных знаков, которые выдаются по желанию владельцев легковушек.
В сервисных центрах МВД заработал новый онлайн-сервис. Теперь украинцы могут проверять наличие платных номерных знаков, которые выдают по желанию владельцев легковых авто.
Они сгруппированы по стоимости и адресам территориальных сервисных центров МВД.
Данные на сайте обновляются ежедневно.
Читайте: Езда без страховки в Украине – размеры штрафовОднако забронировать номерной знак пока невозможно, поэтому владельцу легкового автомобиля рекомендуется выбирать несколько приемлемых вариантов комбинаций цифр.
После этого с пакетом документов (те же документы, которые необходимы для регистрации (перерегистрации) транспортных средств, – Ред.) нужно прийти в сервисный центр МВД, в котором есть в наличии желаемый номерной знак.
С помощью калькулятора цен можно проверить стоимость платных номерных знаков онлайн.
Нагажаемо, что Государственная таможенная служба Украины запустила “калькулятор”, с помощью которого можно рассчитать стоимость растаможивания автомобилей на европейской регистрации.
