
# Произошло массовое отравление сотрудников «Ленты»

Published at: **2019-11-05T13:48:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/zdorove/proizoshlo_massovoe_otravlenie_sotrudnikov_lenty/)

Около 60 сотрудников центрального офиса сети супермаркетов «Лента» в Санкт-Петербурге обратились к врачам с жалобами на отравление, сообщает пресс-служба компании.У всех обратившихся присутствуют симптомы кишечной инфекции.Причина отравления устанавливают. По предварительной версии, источником могла стать еда из столовой офиса.
