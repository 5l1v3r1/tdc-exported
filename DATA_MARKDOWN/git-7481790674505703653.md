
# 日経平均終値２万３３３０円、３営業日連続で最高値更新

Published at: **2019-11-07T06:10:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/economy/20191107-OYT1T50201/)

　７日の東京株式市場で日経平均株価（２２５種）は続伸し、終値は前日比２６円５０銭高の２万３３３０円３２銭となり、３営業日連続で今年の最高値を更新した。
　２０１９年９月中間連結決算で増収増益を発表したトヨタ自動車など、業績が上向いている銘柄に買い注文が集まった。ただ、米中貿易協議の進展への期待がやや後退し、当面の利益を確定するための売り注文もみられ、小幅な値動きにとどまった。
　市場では「各社の業績を見極めようと積極的な取引を手控える投資家も多く、しばらくは小幅な値動きが続く」（大手証券）との見方も出ている。
