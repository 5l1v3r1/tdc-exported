
# 天理が５年ぶり優勝！３発１２得点で大阪桐蔭を撃破　近畿王者として明治神宮大会へ

Published at: **2019-11-04T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/04/0012848152.shtml)

　「秋季高校野球近畿大会・決勝、天理１２－４大阪桐蔭」（４日、佐藤薬品スタジアム）
　天理（奈良３位）が大阪桐蔭（大阪１位）に打ち勝ち、秋の近畿大会で５年ぶり９回目の優勝を飾った。
　天理は一回、３番の河村が低めのボールを振り抜き、左翼席へ弾丸ライナーで運ぶ先制２ラン。六回には２本の適時打でリードを広げた後、山元がバスター打法で左翼席へ３ランを運び、一挙５点を追加した。七回には瀬が左翼席へソロ。九回にも４点を加え、先発全員の１６安打で１２得点を奪った。
　投げては先発マウンドに上がった背番号１９の１年生右腕、八回途中４失点の力投。長身から威力ある直球、落差の大きいチェンジアップを駆使して強力打線に対抗した。
　大阪桐蔭は２年ぶりの優勝はならず。エース左腕の藤江が天理打線につかまり、６回を投げ７失点。その後も４投手をつぎ込んだが、失点を重ねた。打線は一回に３番・西野がソロ。７点を追う八回には西野の適時二塁打などで３点を返したが、なお２死満塁の好機を生かせなかった。
　天理は準決勝でも今夏の甲子園を制した履正社（大阪２位）を撃破。秋の近畿王者として１５日に開幕する明治神宮大会に出場し、初戦は東北王者の仙台育英（宮城）と対戦する。
