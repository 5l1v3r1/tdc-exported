
# Dica do Dia: Como activar o modo de bloqueio de rastreios no browser Opera

Published at: **2019-11-04T18:01:36+00:00**

Author: ****

Original: [PCGuia](https://www.pcguia.pt/2019/11/dica-modo-bloqueio-browser-opera/)

Para activar o modo de bloqueio de rastreios no browser Opera, utilizar o atalho de teclado Alt+P. De seguida, clicar na categoria “Básico” situada no menu lateral esquerdo. Localizar a categoria “Protecção de privacidade”. Por fim, clicar no slider de “Bloqueio de rastreios”.
