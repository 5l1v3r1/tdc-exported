
# Valais: un corps sans vie retrouvé dans une rivière

Published at: **2019-11-04T17:10:00+00:00**

Author: **jyg**

Original: [www.lenouvelliste.ch](https://www.lenouvelliste.ch/articles/valais/canton/valais-un-corps-sans-vie-retrouve-dans-une-riviere-879225)


        Une femme a fait une chute mortelle dans un ravin haut-valaisan.
      
Le corps d’une femme de 38 ans a été retrouvé sans vie dans le lit de la rivière Mattervispe, dans le Haut-Valais. L’accident s’est produit le mercredi 30 octobre dernier dans la région de Kalpetran, annonce la police cantonale dans un communiqué.
La malheureuse, qui est de nationalité française, est tombée dans un ravin, le soir vers 21 heures. C’est son compagnon qui a averti la police.
Arrivés sur place, les secouristes n’ont pu que constater le décès de la Française.
Une enquête a été ouverte par le Ministère public.
