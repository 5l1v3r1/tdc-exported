
# На известного кузбасского адвоката совершено покушение

Published at: **2019-11-06T14:55:48+00:00**

Author: **Константин Воронов, Новосибирск**

Original: [Коммерсантъ](https://www.kommersant.ru/doc/4149755)

В центре Кемерово совершено покушение на известного местного адвоката Сергея Учителя. Об этом “Ъ” сообщили источники из его ближайшего окружения. С огнестрельным ранением юрист доставлен в больницу, сейчас он находится в операционной. Сотрудники полиции, патрулировавшие данный район, задержали стрелка на месте преступления. Его личность и мотивы пока устанавливаются.
По данным СКР, покушение было совершено, когда юрист выходил из офиса коллегии адвокатов «Регионсервис». Пистолетная пуля попала адвокату в грудную клетку. Охранник Сергея Учителя вызвал «скорую».
Возбуждено уголовное дело о покушении на убийство (ст. 30, ст. 105 УК РФ).
