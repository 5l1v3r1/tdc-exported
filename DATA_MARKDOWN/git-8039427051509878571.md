
# ‘ఐదేళ్లలో చేయాల్సిన అభివృద్ధిని సీఎం ఐదు నెలల్లో చేశారు’

Published at: **2019-11-04T19:17:14+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=946286)

ADVT
అమరావతి: ఐదేళ్లలో చేయాల్సిన అభివృద్ధిని ఐదు నెలల్లో సీఎం జగన్మోహన్ రెడ్డి చేశారని నర్సీపట్నం ఎమ్మెల్యే ఉమా శంకర్ గణేష్ అన్నారు. సోమవారం ఆయన మీడియాతో మాట్లాడుతూ జగన్ చేసిన అభివృద్ధిని చూసి ఇతర పార్టీల నాయకులు వైసీపీలో చేసురుతున్నారని అన్నారు. అమ్మ ఒడి, రైతు భరోసా వంటి అద్భుతమైన పథకాలను సీఎం ప్రవేశపెట్టారన్నారు. చింతకాయల సన్యాసి పాత్రుడు వైసీపీలో చేరడం ఆనందంగా ఉందన్నారు.
