
# Viber представил новые возможности мессенджера для бизнеса

Published at: **2019-11-06T11:25:00+00:00**

Author: **Мария Рогачева**

Original: [Лайкни](https://www.likeni.ru/events/viber-predstavil-novye-vozmozhnosti-messendzhera-dlya-biznesa/)

Viber добавил в свое приложение по обмену быстрыми сообщениями новые функции для бизнеса. Теперь бизнес-сегменту доступны два новых формата: сессии и профили брендов.
Во время сессий компании общаются со своими клиентами. Визуально это похоже на обычный чат. Одна сессия может длиться 12 часов с момента отправления пользователем первого сообщения. Начать сессию может только пользователь.

Это инструмент компании могут интегрировать в свою систему обратной связи с клиентами или в CRM.
С помощью этого инструмента пользователь может увидеть подробную информацию о компании, которая отправляет ему сообщение в мессенджере (номер телефона, часы работы), а также получить краткую справку о роде деятельности компании.
Для бизнеса данный функционал даст дополнительную возможность расширить присутствие бренда в мессенджере.
Напомним, ранее сообщалось, что в мессенджере Viber появится возможность покупки товаров.
