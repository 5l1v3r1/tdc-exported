
# Труба: Расследование «дел Майдана» на время будет остановлено

Published at: **2019-11-04T17:06:00+00:00**

Author: **InfoResist**

Original: [InfoResist](https://inforesist.org/truba-rassledovanie-del-majdana-na-vremya-budet-ostanovleno/)

Расследование «дел Майдана«, которые передадут в Государственное бюро расследований из прокуратуры, на какое-то время будет остановлено. Об этом заявил директор ГБР Роман Труба в колонке на сайте Украинской правды.
По его словам, на какое-то время это остановит расследование.
Для того, чтобы расследование «дел Майдана» не было остановлено, отметил Труба, нужно в ближайшее время принять изменения к закону о ГБР, которые отменят ограничение на количество работников ведомства и введут упрощенную конкурсную процедуру для следователей ГПУ, расследующих «дела Майдана».
