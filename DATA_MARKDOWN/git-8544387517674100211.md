
# Σταματίνα Τσιμτσιλή: Το σχόλιο που προκάλεσε αντιδράσεις (vid)

Published at: **2019-11-07T20:01:47+00:00**

Author: **Marianthi Dovri**

Original: [neolaia.gr](https://neolaia.gr/2019/11/07/stamatina-tsimitsili-ratsistiko-sxolio/)

Η Σταματίνα Τσιμτσιλή προκάλεσε θύελλα αντιδράσεων με σχόλιο που έκανε κατά τη διάρκεια της εκπομπής της.
Πιο συγκεκριμένα, η γνωστή παρουσιάστρια φάνηκε να κατηγορεί τους πρόσφυγες για «έλλειψη παιδείας» δηλώνοντας τα εξής: «Μπορεί να μην έχουν να φάνε, αλλά κάνουν 5-6 παιδιά. Ο Έλληνας την περίοδο της κρίσης έκανε ένα παιδί και αυτό για να πει ας πούμε  έγινα γονιός.»
