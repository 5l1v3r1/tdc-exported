
# Perlindungan Data Siber di Indonesia, Acronis True Image 2020 Jadi Solusi

Published at: **2019-11-05T23:19:00+00:00**

Author: **Immanuel Christian**

Original: [RRI News Portal](http://rri.co.id/post/berita/743333/teknologi/perlindungan_data_siber_di_indonesia_acronis_true_image_2020_jadi_solusi.html)

KBRN, Jakarta : Acronis pemimpin pasar global untuk perlindungan siber bersama PT Optima Solusindo Informatika (Optima) dan Bhinneka.com meluncurkan Acronis True Image 2020 di Indonesia pada hari ini, Selasa (5/11/2019).
Acronis True Image 2020 adalah versi terbaru dari solusi perlindungan siner personal yang memudahkan para pengguna melakukan replikasi data berbasis Cloud secara otomatis menjadikan acronis true image 2020 sebagai pertama dengan fitur 3-2-1 Backup rule dan direkomendasikan oleh para ahli perlindungan siber dunia.Kesadaran akan pentingnya menyimpan dan menjaga data digital saat ini pun terus meningkat lebih yang menyangkut kejahatan siber. Sebagai gambaran, berdasarkan laporan tahunan Badan Siber dan Sandi Negara (BSSN), HP dalam proyek tahun 2018 tercatat ada 12.895.554 serangan siber pada 21 sensor yang tersebar, termasuk serangan Malware sebanyak 513.863 kali."Beberapa tahun belakangan kesadaran akan perlindungan siber di Indonesia semakin meningkat. Bahkan, dalam pidato kenegaraannya pada 16 Agustus lalu Presiden Jokowi dengan tegas menyampaikan bahwa Indonesia harus siap siaga menghadapi kejahatan siber sehingga kedaulatan data dapat diwujudkan. Hadirnya Acronis True Image 2020 merupakan jawaban atas hal ini," ucap Managing Director PT Optima Solusindo Informatika, Refany Iskandar kepada RRI, Selasa (5/11/2019).Ia melanjutkan bahwa Acronis True Image 2020 adalah terobosan mutakhir yang merealisasikan visi Acronis untuk perlindungan siber dengan memadukan perlindungan data konvensional dan pengamanan siber. Ini menjadikan Acronis True Image 2020 sebagai satu-satunya solusi yang mencakup Factor of Cyber protection atau lima aspek keamanan siber yang memastikan Safety, Accessibility Privacy, Authenticity, dan Security of data atau bisa disebut dengan SAPAS.Fitur replikasi dari penyimpanan lokal ke Cloud Dual Protection juga masih ditambah dengan lebih dari 100 Kelebihan lainnya yang melekat pada kinerja Acronis True Image 2020 yang memang dirancang sedemikian rupa untuk meningkatkan performa penggunaan dan keamanan.
