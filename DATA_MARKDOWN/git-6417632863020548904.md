
# Ўзбекистон - Саудия Арабистони учрашувидан олдин сайтимиз вакиллари қандай фикрда? (таҳлилий дастур)

Published at: **2019-11-05T14:24:47+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/ozbekiston-saudiya-arabistoni-uchrashuvidan-oldin-saytimiz-vakillari-qanday-fikrda-tahliliy-dastur)

Вадим Абрамов бошчилигидаги жамоамиз Қирғизистон, Саудия Арабистони ва Фаластинга қарши ўйинларга тайёргарлик олиб бормоқда. Сайтимиз вакиллари Камолиддин Алимов, Толибжон Қосимов ҳамда Шаҳбоз Беғамов таҳлилий дастурда ўз фикрларини билдиришди.
TAS-IX эмас
