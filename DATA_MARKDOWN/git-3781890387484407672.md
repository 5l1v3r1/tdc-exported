
# แยกผังเมืองยกระดับพระราม9 ติดถึงข้ามแยกรามคำแหง

Published at: **2019-11-05T10:04:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/bangkok/740128)
