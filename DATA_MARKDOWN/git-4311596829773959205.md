
# В Варшаве раздали тысячу листовок: «Голос ненависти к России достиг апогея»

Published at: **2019-11-01T16:33:00+00:00**

Author: ****

Original: [EADaily](https://eadaily.com/ru/news/2019/11/01/v-varshave-razdali-tysyachu-listovok-golos-nenavisti-k-rossii-dostig-apogeya)

В польской столице прошла акция противодействия русофобии, которую организаторы из «Братства польско-российского» хотят сделать всенародной.
Общественники раздали на улицах Варшавы более тысячи листовок, где есть такие строки на польском и русском языках:
В частности, в раздаче листовок принял участие польский общественно-политический деятель, уроженец Сирии Набиль аль Малази (на снимке), который по этому случаю прикрепил на лацкан пиджака георгиевскую ленточку. По словам председателя «Братства польско-российского» Петра Боло Радтке, работа «в полях» прошла под пристальным взором со стороны полицейского патруля и городской охраны.
По мнению лидера «Братства польско-российского», только просвещение и информирование общественности является «шансом для нас, славянских народов, не потерять наши общие ценности».
