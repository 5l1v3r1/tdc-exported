
# В WhatsApp изменились настройки приватности

Published at: **2019-11-07T06:10:00+00:00**

Author: ****

Original: [РИА Новости](https://ria.ru/20191107/1560659651.html)

МОСКВА, 7 ноя — РИА Новости. В мессенджере WhatsApp обновились настройки конфиденциальности для групп. Об этом сообщается в блоге компании.Теперь у пользователей появится больше возможностей контролировать приглашения в группы. Так, добавить нового участника в чат нельзя будет без его согласия.Чтобы активировать эту функцию, пользователю в настройках мессенджера нужно перейти в раздел "Приватность/Конфиденциальность", а затем "Группы". Там можно выбирать один из нескольких вариантов порядка добавления в группы — могут ли это делать все пользователи либо только ваши контакты.Также существует опция "контакты, кроме", которая позволяет фильтровать тех, кто именно из собеседников пользователя может приглашать его в чаты.
