
# В Ленобласти заработала горячая линия по «мусорной реформе»

Published at: **2019-11-02T02:20:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/738943-v-lenoblasti-zarabotala-goryachaya-liniya-po-musornoi-reforme)


        В регионе подключили горячую линию по вопросам, связанным с переходом на новую систему обращения с отходами.
      
В пятницу, 1 ноября, в Ленинградской области начала работу горячая линия по «мусорной реформе». Она также стартовала в первый ноябрьский день по всему региону. Об этом сообщила пресс-служба губернатора и правительства Ленинградской области.
По горячей линии можно позвонить по вопросам, которые связаны с переходом на новую систему обращения с отходами. Благодаря такому нововведению жители Ленобласти могут позвонить в случае возникновения вопросов, касающихся графика вывоз мусора, а также содержания площадок для мусорных контейнеров.
Позвонить на горячую линию можно по телефону: +7(812)454-18-18.
