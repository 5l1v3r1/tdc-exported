
# 福山雅治　クラシックギターに悪戦苦闘　台本の「吹き替えの可能性」に一念発起

Published at: **2019-11-02T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/02/0012841231.shtml)

　俳優の福山雅治（５０）が１日、都内で行われた主演映画「マチネの終わりに」の初日舞台あいさつに共演の石田ゆり子（５０）らと出席。待ちに待った公開初日に「やっと皆さんに届きましたね」と大喜びした。
　世界的クラシックギタリスト役で出演したが、音楽経験がある福山でさえもクラシックギターは「０からのスタートだった」とケタ違いの難易度。万が一のため、手の形が似たギタリストに吹き替えを頼んでいたという。
　不安尽きぬまま現場入りしたが、「台本に『吹き替えの可能性』と書かれていて『何書いてんの』と思った」と一念発起。代役を立てずにクラシックギターを弾きこなし「奥深い世界でした」と魅了されていた。
