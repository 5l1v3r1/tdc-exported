
# А на вашей улице есть наклейки с самыми популярными лозунгами?

Published at: **2019-11-05T16:01:00+00:00**

Author: ****

Original: [charter97.org](https://charter97.org/ru/news/2019/11/5/354524/)

В Минске и других городах Беларуси появляются наклейки с лозунгом «Баста».
Фотоотчет на эту тему опубликовал телеграмм-канал «Баста».
«А на вашей улице уже есть наклейки с самыми популярными лозунгами этой осени? Нет? Пишите нам. Поделимся», — пишет «Баста».
