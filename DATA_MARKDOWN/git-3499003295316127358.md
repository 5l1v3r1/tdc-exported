
# С автостанции «Теплый Стан» можно уехать в Тулу

Published at: **2019-11-06T22:16:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/yuzao/s_avtostantsii_teplyy_stan_mozhno_uekhat_v_tulu/)

По информации пресс-службы Мосгортранса, автобусы по этому маршруту отправляются каждый день десять раз.Стоимость полного билета составляет 500 рублей, детского - 250 рублей. За провоз одного места багажа понадобится отдать 50 рублей.Напомним, что с автостанции «Теплый Стан» теперь можно уехать и в Вязьму.
