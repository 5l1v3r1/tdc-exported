
# “Славия” “Камп Ноу”да 7 йилдан бери ЕЧЛда ҳеч қайси жамоа кўрсата олмаган натижани қайд этди

Published at: **2019-11-06T03:10:25+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/slaviya-kamp-nouda-7-yildan-beri-echlda-hech-qaysi-jamoa-korsata-olmagan-natijani-qayd-etdi)

“Славия” ЕЧЛ гуруҳ босқичида “Камп Ноу”да гол ўтказиб юбормаган сўнгги 7 йил ичидаги илк клуб бўлди. Сўнгги бор бундай натижага 2012 йил декабрида “Бенфика” эришганди. Ўшанда ҳам 0:0 ҳисоби қайд этилганди.
