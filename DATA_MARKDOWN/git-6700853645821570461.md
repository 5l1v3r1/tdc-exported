
# Tommy Terhenti di Bulutangkis Fuzhou China Open 2019

Published at: **2019-11-06T15:28:00+00:00**

Author: **Danang Sundoro**

Original: [RRI News Portal](http://rri.co.id/post/berita/743612/cabor/tommy_terhenti_di_bulutangkis_fuzhou_china_open_2019.html)

KBRN, Fuzhou : Pebulutangkis tunggal putra Indonesia Tommy Sugiarto terhenti langkahnya saat menghadapi Sai Praneeth asal India di babak pertama turnamen Fuzhou China Open 2019 dengan tiga game 21-15, 12-21, 10-21 dalam tempo 52 menit di Haixia, Olympic Sports Center, Rabu (6/11/2019) sore.“Di game pertama saya main lebih safe dan nggak banyak mati sendiri. Beda dengan di game kedua dan ketiga. Saya merasa kurang fokus dan pertahanan saya kurang rapat. Saya malah banyak membuka peluang bagi Sai,” kata Tommy usai laga dalam keterangan kepada Radio Republik Indonesia (RRI), Rabu (6/11/2019).“Saya tidak bisa menerapkan permainan seperti di game pertama. Saat tertinggal di game kedua pun saya terbawa irama permainan lawan,” ujar Tommy. “Performa saya masih naik-turun karena latihan saya memang nggak seintensif dulu di pelatnas. Memang ini tantangannya menjadi pemain profesional,” lanjutnya. BACA JUGA : Fuzhou China Open 2019, Jordan/Melati Melaju ke Babak KeduaTurnamen berkategori 750, Fuzhou China Open 2019 akan menjadi turnamen terakhir Tommy di tahun ini. Hal itu, karena putra dari Icuk Sugiarto itu tidak akan mengikuti turnamen Hong Kong Open 2019, pekan depan.  “Dari awal memang plan-nya nggak ikut ke Hong Kong Open, setelah ini saya akan langsung pulang ke Jakarta, rehat sebentar dan menyusun rencana turnamen untuk tahun depan,” tegas Tommy.BACA JUGA :Bulutangkis Fuzhou China Open 2019 : Hafiz/Gloria Terhenti Di Hari Kedua, Dihadang TiongkokDua tunggal putra Indonesia sudah terhenti di turnamen ini. Sebelumnya, Anthony Sinisuka Ginting juga harus angkat koper lebih awal setelah dikalahkan Ng Ka Long Angus (Hong Kong), dengan skor 18-21, 9-21. Sedangkan dua wakil lagi akan saling berhadapan satu sama lain, Jonatan Christie akan bertemu dengan Shesar Hiren Rhustavito di babak pertama, Rabu (6/11/2019) Sore.
