
# Колония из миллиона муравьев-каннибалов сбежала из ядерного бункера

Published at: **2019-11-05T15:33:09+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-05/koloniya-iz-milliona-muravev-kannibalov-sbezhala-iz-yadernogo-bunkera)

Порядка миллиона муравьев-каннибалов сбежали из советского ядерного бункера в Польше, сообщает Newsweek.
Запертые в бункере муравьи были обнаружены еще в 2013 году. В 2016-м ученые стали изучать их поведение в бункере. Численность муравьев продолжала увеличиваться, несмотря на отсутствие пищи и источников света. Как выяснилось, муравьи питались телами погибших особей.
Чтобы узнать, покинут ли муравьи свой бункер, ученые построили для них выход. В результате уже спустя год почти все муравьи сбежали из бункера.
