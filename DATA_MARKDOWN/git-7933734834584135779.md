
# Российские кузнецы установили в Симферополе 18-тонные солнечные часы

Published at: **2019-11-04T09:45:00+00:00**

Author: ****

Original: [Крымское информационное агентство](https://kianews24.ru/news/rossiyskie-kuzneci-ustanovili-v-simf/)

Сегодня в симферопольском парке Тренёва в рамках празднования Дня народного единства и к пятилетию Крымской весны кузнецы со всей России реализовали творческий проект «Время Крыма», сделав художественную композицию из металла — солнечные часы, состоящие из сегментов-регионов, передает корреспондент Крымского информационного агентства.
