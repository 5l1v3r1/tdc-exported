
# Росгвардейцы задержали в Сыктывкаре подозрительных молодых людей

Published at: **2019-11-02T20:23:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Syktyvkar/739194-rosgvardeicy-zaderzhali-v-syktyvkare-podozritelnykh-molodykh-lyudei)


        Двое жителей Сыктывкара вызвали подозрения у бойцов Росгвардии. В ходе проверки документов у одного из них был замечен пакетик с неизвестным веществом, а у водителя автомобиля имелись признаки опьянения.
      
В столице Коми бойцы Росгвардии заметили молодых людей, которые вызвали подозрения. Двое местных жителей находились в автомобиле около одного из домов по улице Тентюковской. При виде сотрудников вневедомственной охраны их поведение изменилось.
В итоге бойцы решили проверить документы молодых людей. В ходе проверки у одного из них был замечен пакетик с неизвестным порошкообразным веществом, от которого сыктывкарец пытался избавиться. Также росгвардеец обратил внимание на то, что у водителя автомобиля имеются признаки опьянения. На место была вызвана следственно-оперативная группа. По информации Управления Росгвардии по Республике Коми, в дежурную часть был доставлен мужчина 1991 года рождения для дальнейшего разбирательства.
