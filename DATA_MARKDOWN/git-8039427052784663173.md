
# ఆసుపత్రిలో చేరిన డీకే శివకుమార్

Published at: **2019-11-01T23:01:13+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=943778)

Ph: 88865 47666ప్రవేశం ఉచితం ADVT
బెంగళూరు: కర్ణాటక కాంగ్రెస్ పార్టీ సీనియర్ నేత డీకే శివకుమార్ ఆసుపత్రిలో చేరారు. హై బ్లడ్ ప్రెజర్‌తో పాటు షుగర్ లెవెల్స్‌లో మార్పుల నేపథ్యంలో డాక్టర్ల సూచన మేరకు ఆయన్ను బెంగళూరులోని అపోలో ఆసుపత్రికి తరలించారు. మనీలాండరింగ్ కేసులో అరెస్టై ఇటీవలే ఆయన తీహార్ జైలు నుంచి బెయిల్‌పై బయటకు వచ్చారు. ఇంతలోనే అనారోగ్యంతో ఆసుపత్రిలో చేరాల్సి వచ్చింది.
