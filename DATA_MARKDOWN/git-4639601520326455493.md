
# Αλμπάνης : «Τέτοιες νίκες τις εκτιμάς περισσότερο»

Published at: **2019-11-04T06:45:03+00:00**

Author: **ΤοΒΗΜΑ Team**

Original: [Το Βήμα Online](https://www.tovima.gr/2019/11/04/sports/almpanis-tetoies-nikes-tis-ektimas-perissotero/)

H AEK πήρε μια πολύ σημαντική νίκη σε ένα «τρελό» παιχνίδι κόντρα στον Ατρόμητο και ο Χρήστος Αλμπάνης στάθηκε στον τρόπο που αυτή ήρθε αλλά και στη σημασία που έχει για τη συνέχεια.

«Το σημαντικό είναι η νίκη προκειμένου να μείνουμε κοντά στη βαθμολογία. Στο δεύτερο ημίχρονο βρήκαμε τους χώρους που αναζητούσαμε, πετύχαμε τρία γκολ και είμαστε χαρούμενοι γι αυτό.
Οι μεγάλες ομάδες οφείλουν να ξεπερνούν κάθε εμπόδιο. Αυτό κάνουμε και εμείς και θέλουμε να δίνουμε τον καλύτερο μας εαυτό. Όποιος ποδοσφαιριστής καλείται να βοηθήσει δίνει τα μέγιστα.
Ίσως αυτές οι νίκες έχουν μεγαλύτερη χαρά και τις εκτιμάς περισσότερο. Οι μεγάλες ομάδες όπως η ΑΕΚ πρέπει να βρίσκουν διεξόδους σε αυτές τις καταστάσεις. Κρατάμε τη νίκη.
Ήταν ευχάριστη εβδομάδα για εμάς. Συνεχίζουμε με πείσμα τη δουλειά για καλύτερα αποτελέσματα».
