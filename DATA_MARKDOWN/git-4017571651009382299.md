
# Indígenas de Pastaza piden salida de petrolera china

Published at: **2019-11-06T20:11:26+00:00**

Author: **Redacción**

Original: [Ecuavisa](https://www.ecuavisa.com/articulo/noticias/politica/542836-emiten-boletas-libertad-exfuncionarias-correistas)

Las boletas de excarcelación de Pamela Martínez y Laura Terán, quienes permanecían detenidas desde mayo de este año por las investigaciones dentro del caso Sobornos 2012-2016, fueron emitidas la mañana de este miércoles 6 de noviembrer. Tras ello, las dos procesadas fueron trasladadas a la Unidad de Flagrancias de Quito para colocarles grilletes electrónicos.
