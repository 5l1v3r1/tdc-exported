
# Iz Kolpe potegnili dve moški trupli

Published at: **2019-11-01T07:00:00+00:00**

Author: **Ma. M., S. U.**

Original: [Slovenske novice](https://www.slovenskenovice.si/kronika/doma/clanek/resevanje-na-kolpi-oseba-se-je-utopila-245145)

Gasilci iz Črnomlja so iz Kolpe, med vasjo Vukovci in Kovačjim gradom, potegnili dve moški trupli. Kriminalisti, kriminalistični tehnik in policisti so opravili ogled kraja, o dogodku so obvestili dežurnega preiskovalnega sodnika in okrožno državno tožilko. Točen vzrok smrti bo ugotovljen s sanitarno obdukcijo.Za pomoč pri ugotavljanju identitet so kriminalisti že zaprosili tuje varnostne organe. Po prvih zbranih obvestilih naj bi moška (enega so iz Kolpe potegnili dopoldne, drugega popoldne) utonila pri poskusu nezakonitega prestopa državne meje.
