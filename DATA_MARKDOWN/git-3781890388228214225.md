
# อังกฤษ รัสเซีย นิวซีแลนด์ มุ่งยกระดับการค้ากับอาเซียน

Published at: **2019-11-04T00:44:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/foreign/739864)


        สหราชอาณาจักร รัสเซียในฐานะผู้นำสหภาพเศรษฐกิจยูเรเชีย และนิวซีแลนด์ แสดงความมุ่งมั่นในการแสวงหาโอกาสเพิ่มความร่วมมือด้านการค้าและการลงทุนกับอาเซียน
      
สำนักข่าวต่างประเทศรายงานจากกรุงเทพมหานคร เมื่อวันที่ 4 พ.ย. ว่าเจ้าชายแอนดรูว์ ดยุกแห่งยอร์ก พระราชโอรสในสมเด็จพระราชินีนาถเอลิซาเบธที่ 2 แห่งสหราชอาณาจักร กับเจ้าชายฟิลิป ดยุกแห่งเอดินบะระ เสด็จฯร่วมงานประชุมสุดยอดธุรกิจและการลงทุนอาเซียน ( เอบิส ) เมื่อวันอาทิตย์ ซึ่งเป็นการประชุมคู่ขนานรายการสำคัญของการประชุมสุดยอดสมาคมประชาชาติแห่งเอเชียตะวันออกเฉียงใต้ ( อาเซียน ) ครั้งที่ 35 ซึ่งไทยเป็นประธานในปีนี้ โดยทรงมีพระดำรัสในตอนหนึ่งว่าสหราชอาณาจักรไม่สามารถ "ใช้กลยุทธ์ดั้งเดิมตามแบบของตัวเอง" ในการดำเนินธุรกิจได้อีกต่อไป เนื่องจากสหราชอาณาจักรต้องออกจากการเป็นสมาชิกสหภาพยุโรป ( อียู ) ตามผลการลงประชามติเมื่อเดือนมิ.ย. 2559
