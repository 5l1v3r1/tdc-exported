
# Ini Penyebab WhatsApp Web Tidak Tersambung ke Ponsel

Published at: **2019-11-06T19:41:28+00:00**

Author: **Puti Yasmin - detikInet**

Original: [detikinet](https://inet.detik.com/tips-dan-trik/d-4774740/ini-penyebab-whatsapp-web-tidak-tersambung-ke-ponsel)

Jakarta - WhatsApp memudahkan penggunanya untuk menerima dan mengirim pesan melalui pc. Hanya saja, ada penyebab WhatsApp web tidak tersambung ke ponsel. Apa itu?Selama ini WhatsApp hanya bisa diakses menggunakan HP. Cara menyambungkan koneksi WhatsApp pada HP dan pc pun hanya perlu scan barcode.Berikut penyebab WhatsApp web tidak tersambung ke ponsel:1. Masalah Konektivitas HPWhatsApp web tidak bisa dibuka karena beberapa alasan. Pertama, karena masalah konektivitas pada HP. Untuk mengatasinya, pengguna bisa membuka WhatsApp pada telepon terlebih dahulu dan coba untuk mengirim pesan.Penyebab WhatsApp web tidak tersambung ke ponsel karena koneksi pada HP tak stabil.Jika langkah tersebut tetap membuat WhatsApp web loading terus, maka pengguna bisa mencoba menyetel ulang jaringan pada HP. Setelah itu, kembali untuk mencoba mengirim pesan.2. Masalah Konektivitas KomputerSelain masalah pada konektivitas HP, penyebab WhatsApp web tidak tersambung ke ponsel juga dikarenakan masalah konektivitas pada komputer.Kenapa WhatsApp web tidak bisa digunakan? Pengguna bisa mencoba untuk mengecek konektivitas komputer. Hal itu biasa ditandai dengan bar kuning pada bagian daftar chat dengan pesan 'Komputer Tidak Terhubung'.Periksa koneksi internet kembali. Jika WhatsApp web tidak bisa dibuka di PC maka coba lah untuk log out dan log in kembali untuk mengaktifkannya.Nah, semoga penyebab WhatsApp web tidak tersambung ke ponsel bisa teratasi ya!
