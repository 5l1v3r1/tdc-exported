
# В Череповце автобус № 4 изменит маршрут в День города

Published at: **2019-11-03T16:05:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Vologda/739422-v-cherepovce-avtobus-4-izmenit-marshrut-v-den-goroda)


        Жителей города металлургов предупреждают об изменении маршрута автобуса № 4. Транспорт будет объезжать часть Советского проспекта, на которой откроется ярмарка.
      
В понедельник, 4 ноября, в Череповце изменится маршрут автобуса № 4. Это связано с празднованием для города, сообщает официальный портал города металлургов.
Автобус будет съезжать с Советского проспекта на улице Ленина, затем проедет по улице Максима Горького, после чего повернет на Московский проспект и продолжит движение по привычному маршруту. По такому же пути «четверка» будет двигаться в обратном направлении.
Напомним, Милютинская ярмарка пройдет в Череповце с 12:00 до 15:00. В ходе праздника состоится конкурс частушечников. Лучший участник получит в подарок телевизор.
