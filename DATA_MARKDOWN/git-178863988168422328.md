
# Назван процент поражений для эффективного обучения

Published at: **2019-11-07T11:02:00+00:00**

Author: ****

Original: [РИА Мелитополь](https://ria-m.tv/news/170867/nazvan_protsent_porajeniy_dlya_effektivnogo_obucheniya.html)


        Ученые назвали "процент необходимых поражений", которые нужны для наиболее эффективного обучения новому.Об этом сообщает Newsweek.
      
Специалисты назвали свое соотношение правилом 85 процентов. Их исследования показали, что человек во время обучения должен терпеть неудачи только в 15% случаев.
Если данный процентный показатель будет меньше (задача будет слишком простой) - человек плохо усвоит изученное. Если задача окажется слишком тяжела, учащийся может полностью бросить обучение.
Специалисты создали алгоритм, обучивший компьютеры выполнять простые задания. Оказалось, что машины учится быстрее, если ошибаются в примерно 15% случаев.
Дальнейшие исследования данных об обучении людей и животных, подтвердили эту закономерность и для живых существ.
