
# Житель Коми предстанет перед судом за незаконное возмещение НДС

Published at: **2019-11-01T15:06:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Syktyvkar/738765-zhitel-komi-predstanet-pered-sudom-za-nezakonnoe-vozmeshenie-nds)


        Житель Коми подал в налоговую инспекцию декларацию по НДС, содержащую ложные сведения. В итоге он обманным путем заполучил 250 тысяч рублей. Материалы уголовного дела направлены в суд для рассмотрения.
      
В Коми возбуждено уголовное дело в отношении руководителя организации по факту мошенничества. Мужчина путем незаконного возмещения НДС присвоил 250 тысяч рублей. Удалось установить, что злоумышленник направил в налоговую инспекцию декларацию по НДС, но указал в ней недостоверные сведения.
По информации МВД по Республике Коми, обвиняемый подготовил ложные документы на приобретение автомобиля Volkswagen Touareg, включая договор купли-продажи, товарную накладную, счет-фактуру и другие. Расследование уголовного дела завершено, материалы направлены в суд для дальнейшего рассмотрения.
