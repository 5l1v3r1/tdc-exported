
# Равшан Эрматов Осиё чемпионатининг финал босқичига чиққан футзалчиларимизни қабул қилди

Published at: **2019-11-01T11:22:05+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/ravshan-ermatov-osiyo-chempionatining-final-bosqichiga-chiqqan-futzalchilarimizni-qabul-qildi)

Баҳодир Аҳмедов шогирдлари Эроннинг Урмия шаҳрида ўтган Осиё чемпионати-2020 саралаш гуруҳ босқичи ғолибига айланишганди. Футзалчиларимиз Афғонистонни 6:2, Непални 4:1, Тожикистон терма жамоасини эса 9:3 ҳисобида мағлуб этишган эди.
Кеча Ўзбекистон футбол ассоциациясида учрашув бўлиб ўтди. ЎФА вице-президенти Равшан Эрматов гуруҳ ғолиби сифатида Туркманистонда ўтадиган Осиё чемпионати финал босқичи йўлланмасини қўлга киритган терма жамоамиз аъзоларини қабул қилди.
Учрашувда ЎФА раҳбариятидан ташқари футбол бўйича миллий терма жамоамиз бош мураббийи Вадим Абрамов, ўзбек футзали ривожига ҳисса қўшган яна бир жонкуяр инсон, Мираброр Усмоновнинг ўғли Жаҳонгир Усмонов ҳам иштирок этди.
Эътиборлиси, учрашув якунида футзалчиларимиз ташаббус кўрсатиб, Ўзбекистон футбол ассоциациясининг собиқ марҳум президенти, Сенатор Мираброр Усмоновнинг расми ёнида суратга тушишди. Бу билан улар юртимизда футзал ривожига катта хисса қўшган Мираброр Зуфаровичга ҳурмат бажо келтиришди.
