
# కర్నూలులో క్షుద్ర పూజల కలకలం

Published at: **2019-11-05T13:02:52+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=947077)

ADVT
కర్నూలు: ఉరుకుంద ఈరన్నస్వామి క్షేత్రంలో క్షుద్ర పూజల కలకలం రేగింది. భక్తుల విడిదిగది 167 వెనక మనిషి పుర్రె, ఎముకలు, పసుపు, కుంకుమలతో పూజలు చేసిన ఆనవాళ్లు ఉన్నాయి. దీంతో ఆలయానికి వచ్చే భక్తులు, స్థానికులు భయాందోళన చెందుతున్నారు. ఈ ఘటనపై ఆలయ అధికారులు విచారణ జరుపుతున్నారు.
