
# РБК: куратором мусорной реформы может стать помощник вице-премьера Гордеева

Published at: **2019-11-06T17:23:11+00:00**

Author: ****

Original: [Коммерсантъ](https://www.kommersant.ru/doc/4149814)

Замминистра природы и экологии, который будет курировать работу по экологии и по обращению с отходами, в том числе мусорную реформу, могут назначить помощника вице-премьера Алексея Гордеева Константина Румянцева, сообщили источники РБК. В Минприроды отказались комментировать информацию о потенциальном кандидате.
Премьер-министр Дмитрий Медведев уволил заместителя министра природных ресурсов и экологии Владимира Логинова по его просьбе. Минприроды сообщило, что ведомство уже представило кандидата на эту должность. Его имя не называется.
Напомним, Владимир Логинов в начале 2019 года попросил генпрокурора Юрия Чайку проверить законность действий региональных чиновников при установлении тарифа на услугу региональных мусорных операторов.
