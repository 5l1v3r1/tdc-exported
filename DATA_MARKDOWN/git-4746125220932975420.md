
# 菅田将暉、俳優10周年記念日ライブ5曲ダイジェスト公開

Published at: **2019-11-07T13:02:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148212/full/)

　俳優で歌手の菅田将暉が、9月6日に東京・Zepp DiverCityで開催した全国ツアー『菅田将暉 LIVE TOUR 2019 “LOVE”』ファイナルの模様が、WOWOWで10日午後8時半からオンエアされる。 　2ndアルバム『LOVE』（7月発売）を引っさげてのツアーは、約20万件の応募の中から抽選で勝ち抜いた約1万人を動員。そのファイナルは俳優デビュー10周年の記念日に行われた。 　WOWOWの番組サイトでは放送に先駆け、ライブ本編の中から「クローバー」「まちがいさがし」「見たこともない景色」「7.1oz」「TONE BENDER LOVE」の5曲のダイジェスト映像を公開中。
