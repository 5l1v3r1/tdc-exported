
# ചികിത്സയ്ക്കെത്തിയ യുവതിയെ പീഡിപ്പിക്കാൻ ശ്രമം; ഗൈനക്കോളജിസ്റ്റ് പിടിയിൽ

Published at: **2019-11-07T09:07:00+00:00**

Author: **മനോരമ ലേഖകൻ**

Original: [ManoramaOnline](https://www.manoramaonline.com/news/latest-news/2019/11/07/gynaecologist-arrested-in-trivandrum.html)

തിരുവനന്തപുരം∙ തിരുവനന്തപുരത്ത് പീഡനക്കേസിൽ ഡോക്ടർ അറസ്റ്റിൽ. ഫോർട്ട് സർക്കാർ ആശുപത്രിയിലെ ഗൈനക്കോളജിസ്റ്റ് സനൽ കുമാറാണു പിടിയിലായത്. ചികിത്സയ്ക്കെത്തിയ യുവതിയെ പീഡിപ്പിക്കാൻ ശ്രമിച്ചെന്നാണു പരാതി.
