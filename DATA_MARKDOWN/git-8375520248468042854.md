
# Необычный плавучий дом под названием ufo: впечатляющие фото жилья будущего

Published at: **2019-11-04T13:57:17+00:00**

Author: ****

Original: [ГЛАВКОМ](https://glavcom.ua/scotch/foto/neobychnyy-plavuchiy-dom-pod-nazvaniem-ufo-vpechatlyayushchie-foto-zhilya-budushchego-635453.html)

Итальянская компания Jet Capsule, которая занимается частными яхтами, разработала проект необычного плавучего дома под названием UFO (Unidentified Floating Object). 
Неопознанный плавающий объект состоит из двух полусферических оболочек из стекловолокна, которые образую два этажа дома.
Гидрореактивный электромотор Torqeedo Deep Blue 1800 приводится в движение электроэнергией от солнечных батарей, площадью 40 кв.м. Они находятся на верхней полусфере.
Внутри — кухня и столовая, гостиная 20-кв.м, это верхний «этаж» дома. Лестница ведет на нижний уровень к ванной и спальне с большим окном для просмотра морской жизни.
Будет здесь и генератор воды, преобразующий дождевую и морскую воду в питьевую. Чтобы удерживать Неопознанный плавающий объект на стоянках, на судне используется специальная «эластичная якорная система».
Будет ли в реальности реализован проект Неопознанного плавающего объекта — это покажет время.
