
# В Киеве мужчина открыл огонь из автомата по соседям, недовольным его долгами по коммуналке

Published at: **2019-11-07T00:00:00+00:00**

Author: ****

Original: [ОстроВ](https://www.ostro.org/video/579192/)

В Киеве на лестничной площадке одного из многоквартирных домов произошла стрельба. Мужчина открыл огонь из автомата в соседа, ранив его в ногу. Причиной конфликта, по утверждению местной жительницы, стали воровство стрелком электроэнергии у соседей и долги за отопление, в результате чего дому не включали отопление.
