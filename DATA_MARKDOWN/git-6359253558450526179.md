
# Главное управление МЧС России по Республике Крым осталось без руководителя

Published at: **2019-11-05T00:02:56+00:00**

Author: ****

Original: [Новый День](https://newdaynews.ru/crimea/675703.html)

Главное управление МЧС России по Республике Крым осталось без руководителя: Владимир Путин освободил от этой должности Александра Еремеева.
Как передает корреспондент РИА «Новый День», Еремеев возглавил крымский главк в июне 2018 года, сменив на данном посту Сергея Шахова. Последний после отставки из ГУ МЧС РФ по РК сохранил должность министра чрезвычайных ситуаций в составе правительства Крыма.
Кто станет новым руководителем главного управления по РК в составе МЧС РФ, пока не сообщается.
