
# Пользователи WhatsApp смогут выбрать, кто может приглашать их в группы

Published at: **2019-11-07T17:39:00+00:00**

Author: **Марина Ибушева**

Original: [Лайкни](https://www.likeni.ru/events/polzovateli-whatsapp-smogut-vybrat-kto-mozhet-priglashat-ikh-v-gruppy/)

Пользователи WhatsApp получили возможность контролировать, кто может приглашать их в группы. Мессенджер обновил настройки конфиденциальности и добавил функцию, позволяющую ограничить круг людей, которые могут добавлять пользователя в общие чаты. 
Теперь пользователям доступны три настройки: 
Установить соответствующие настройки можно в разделе «Конфиденциальность» учетной записи WhatsApp. Обновлённые настройки конфиденциальности уже выкатываются по всему миру и в ближайшие дни станут доступны для всех пользователей последней версии приложения WhatsApp. Напомним, на прошлой неделе в приложении WhatsApp появилась возможность разблокировки с помощью отпечатка пальца. Источник: блог WhatsApp
