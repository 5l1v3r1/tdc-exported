
# 菲律賓約載60人渡輪翻覆 官方展開救援

Published at: **2019-11-07T09:44:17+00:00**

Author: ****

Original: [NTD 新唐人電視台](https://www.ntdtv.com/b5/2019/11/07/a102701874.html)

【新唐人北京時間2019年11月07日訊】菲律賓中部宿霧島（Cebu）附近，7日(週四)一艘載有至少60人的渡輪在洶湧海上翻覆，當局已展開救援行動。
法新社報導，這艘渡輪船員事發時透過無線電求救表示，船隻正在進水。
海岸防衛隊發言人恩希納（Michael Encina）表示，「我們現在沒有統計數字，但我們找到的人都活著，到目前為止我們沒有發現那艘船。」
菲律賓是一個群島國，數以百萬計民眾仰賴不牢固渡輪在群島間進行廉價運輸，船難事故頻傳。
