
# Мурманская область будет закупать дорогостоящее оборудование и лекарства совместно с Москвой

Published at: **2019-11-05T14:21:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Murmansk/740019-murmanskaya-oblast-budet-zakupat-dorogostoyashee-oborudovanie-i-lekarstva-sovmestno-s-moskvoi)


        Андрей Чибис озвучил итоги Госсовета, по которым у Мурманской области есть все шансы получить хорошие федеральные ресурсы на модернизацию первичного звена здравоохранения.
      
Губернатор Мурманской области заявил о высоких шансах получить федеральное финансирование для модернизации первичного звена здравоохранения.
Для получения ресурсов Андрей Чибис поручил в сжатые сроки завершить инвентаризацию медучреждений и подготовить региональную программу по модернизации первичного звена здравоохранения.
Об этом он заявил на прошлой неделе в Светлогорске на расширенном заседания президиума Государственного совета «О задачах субъектов Российской Федерации в сфере здравоохранения», которое проходило под председательством президента Владимира Путина.
При выполнении условий, Мурманская область станет одним из первых регионов, в котором начнется федеральное финансирование модернизации.
Андрей Чибис также поручил проработать с правительством Москвы совместную закупку дорогостоящего медоборудования и лекарств. Договоренность по сотрудничеству уже достигнута в ходе Госсовета.
