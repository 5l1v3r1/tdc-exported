
# Новые цифровые метеостанции установлены в Мичуринском аграрном университете

Published at: **2019-11-07T06:26:00+00:00**

Author: ****

Original: [Interfax-Russia.ru](http://www.interfax-russia.ru/Moscow/main.asp?id=1079396)

7 ноября. Interfax-Russia.ru - Два комплекса современного метеорологического оборудования установлены в Мичуринском аграрном университете (Тамбовская область), сообщает пресс-служба администрации региона.
Современные метеостанции собирают сведения о температуре и влажности воздуха и почвы, атмосферном давлении, солнечной радиации и другие полезные для сельскохозяйственного производства данные. Сведения будут использованы в сельском хозяйстве для прогнозирования погодных условий и предупреждения связанных с ними рисков.
Тестирование оборудования позволит провести сравнение метеорологической информации с многолетними данными физиологического состояния растений. По мнению экспертов, это поможет прогнозировать возможности повреждения сельскохозяйственных растений от неблагоприятных факторов при изменении климатических условий.
