
# 『SONGS』11・9はアーカイブ映像で「こころ励ます歌」特集

Published at: **2019-11-02T13:13:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147928/full/)

　NHK総合の音楽番組『SONGS』（毎週土曜　後11：00〜11：30）。9日の放送は、目覚ましい活躍で世界を驚かせたラグビー日本代表を支えた歌「ビクトリー・ロード」のような、人々の心を励ます歌を特集する。 　今回、街ゆく人に「あなたの心を励ましてくれる歌はなんですか？」とインタビューを敢行。「つらい時、くじけそうになった時に、励ましてくれた歌」「いつまでも心を支えてくれる大切な歌」を聞いた。 　「部活でレギュラーに入れず落ち込んだ時に鼓舞してくれた歌」「恋人と別れた時に聞いて癒やしてくれた歌」「台風の被害にあった実家を片付けながら思い出し、力が湧いた歌」「夢に破れ、次なる一歩を踏み出す時に背中を押してくれた歌」など、さまざまなエピソードと心にしみる数々の歌が浮かびあがってきた。 　「こころ励ます歌 2019」と題し、総数200以上のインタビューの中から、聴く人の心を励ます歌と“音楽が寄り添った”エピソードを厳選。サザンオールスターズ「希望の轍」、中島みゆき「ファイト！」、竹内まりや「元気を出して」 、Mr.Children「終わりなき旅 」…など、NHKアーカイブスをはじめとする貴重な映像で、豪華なアーティストの名曲、ヒット曲を放送する。
