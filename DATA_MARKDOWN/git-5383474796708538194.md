
# Яндекс запустил робота-курьера. Он развозит почту, но это только начало

Published at: **2019-11-07T15:42:42+00:00**

Author: **Артём Баусов**

Original: [iPhones.ru](https://www.iphones.ru/iNotes/yandeks-zapustil-testirovanie-bespilotnogo-robota-kurera-11-07-2019)

Яндекс показал своего первого самоуправляемого робота-курьера.
Сейчас устройство тестируют в штаб-квартире компании. Он используется для перевозки небольших грузов, ездит полностью самостоятельно: определяет объекты, уступает дорогу пешеходам и даже животным. А благодаря лидару гаджет умеет ориентироваться в темноте.
Гаджет передвигается на шести колёсах, а его высота достигает 50 см. Во время движения активируется белая светодиодная лента. По прибытии она загорается зелёным.
Компания планирует в будущем использовать робота для доставки Яндекс.Еды и заказов с площадки Беру. Товары помещаются в верхнюю крышку, которую можно открыть только через приложение.
