
# Ученые рассказали о вреде сигаретных фильтров для здоровья

Published at: **2019-11-04T15:19:36+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-04/uchenye-rasskazali-o-vrede-sigaretnyh-filtrov-dlya-zdorovya)

Ученые из Великобритании и США призвали запретить использовать сигаретные фильтры из-за их опасности для человеческого организма, пишет British Medical Journal.
Эксперты выяснили, что фильтры из ацетатных волокон целлюлозы небезопасны, поскольку вынуждают людей затягиваться сильнее и выкуривать большее количество сигарет. Также, по словам специалистов, при производстве табачной продукции их начали использовать из соображений экономии, поскольку за счет этого удалось увеличить размер сигарет и их стали производить в еще большем количестве.
Также ученые обращают внимание на неэкологичность фильтров - оставшиеся фильтры в виде окурков серьезно вредят экосистеме, поскольку не разлагаются. Специалисты в этой связи призвали включить их в черный список Евросоюза наравне с одноразовыми пластиковыми изделиями.
