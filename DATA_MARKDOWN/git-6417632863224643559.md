
# Ўзбекистон МТЖ машғулотларни бошлади ГАЛЕРЕЯ

Published at: **2019-11-02T13:25:52+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/ozbekiston-mtj-mashgulotlarni-boshladi-galereya)

Терма жамоамиз Вадим Абрамов бошчилигида бугунги машғулотни "Жар" стадионининг захира майдонида ўтказди. Машғулотда маҳаллий ўйинчилардан Музаффар Музаффаров иштирок этмади, бугун кундуз куни хабар берганимиз Сардор Мирзаев айнан Музаффаровнинг ўрнига чақирилган. Легионерлардан эса ҳозирча Одил Аҳмедов ҳамда Санжар Шоаҳмедовлар биринчи кунданоқ машғулотларга қўшилишди.
