
# Microsoft объединила все приложения Office для iPhone в одно

Published at: **2019-11-04T17:26:10+00:00**

Author: **Артём Баусов**

Original: [iPhones.ru](https://www.iphones.ru/iNotes/microsoft-obedinila-vse-prilozheniya-office-dlya-iphone-v-odno-11-04-2019)

Microsoft выпустила на iOS и Android новое приложение Office, в котором объединила функции Word, Excel и Power Point.
Они полностью копируют возможности отдельных приложений, включая просмотр и редактирование. Доступ к файлам предоставляется через облако Microsoft OneDrive.
Скачать Office можно уже сейчас бесплатно. Для владельцев Android-устройств нужно пройти регистрацию на сайте Microsoft, для iOS — скачать приложение через TestFlight. Оно ещё находится на стадии доработки. Количество мест ограничено.
