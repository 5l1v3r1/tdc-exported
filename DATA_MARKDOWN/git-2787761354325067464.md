
# В Брянске на «фанту» из крана жалуются жители Новостройки

Published at: **2019-11-02T09:08:47+00:00**

Author: ****

Original: [Брянск Today](https://bryansktoday.ru/article/111764)

Жители Володарского района Брянска рассказали о проблеме, с которой они столкнулись на этой неделе. В одном из домов на улице Фосфоритной в микрорайоне Новостройка из крана идёт ржавая горячая вода.
Со слов горожан, ситуация не меняется уже несколько дней. «А вокруг такая тишина», - возмущается автор фотографии, опубликованной в социальной сети «ВКонтакте» в субботу, 2 ноября.
