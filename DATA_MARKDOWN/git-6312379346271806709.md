
# Пять сел остались без света из-за аварии на Сахалине

Published at: **2019-11-04T06:59:02+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-04/pyat-sel-ostalis-bez-sveta-iz-za-avarii-na-sahaline)

Пять населенных пунктов остались без электричества на Сахалине из-за аварии на энергосетях.
Инцидент произошел в воскресенье в Тымовском районе острова. В результате сильного крена опоры вышла из-за строя одна из линий электропередачи, сообщает пресс-служба регионального главка МЧС.
Пять сел остались без электричества.
Впрочем, по резервной схеме удалось дать электричество в поселки Кировское, Ясное, Белоречье и Палево.
Сейчас без света остается только село Зональное.
