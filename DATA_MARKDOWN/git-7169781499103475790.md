
# Facebook тестирует сквозное шифрование для аудио и видеозвонков

Published at: **2019-11-01T14:39:00+00:00**

Author: **Марина Ибушева**

Original: [Лайкни](https://www.likeni.ru/events/facebook-testiruet-skvoznoe-shifrovanie-dlya-audio-i-videozvonkov/)

Facebook тестирует в Messenger сквозное шифрование для секретных чатов, а также для видео и аудиозвонков мессенджера. Система шифрования позволит защитить сообщения пользователей от перехвата на этапе отправления. Кроме того, пользователи секретных чатов смогут задать время хранения сообщения, после которого оно автоматически удалится.Исследователь приложений Джейн Манчун Вонг также заметила, что в Facebook Messenger ведется тестирование шифрования аудио и видеозвонков в секретном режиме. 
Когда эта функция станет доступна всем пользователям, пока неизвестно.На данный момент сквозное шифрование используется по умолчанию в мессенджере WhatsApp. Напомним, ранее в Facebook Messenger появились 3 новых функции для брендов. Источник: The Next Web
