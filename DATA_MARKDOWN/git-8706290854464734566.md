
# Μηχανική βλάβη σε τρένο που παρέσυρε σκύλο στη Σκύδρα

Published at: **2019-11-04T16:54:00+00:00**

Author: ****

Original: [enikos.gr](http://www.enikos.gr/society/682263/michaniki-vlavi-se-treno-pou-paresyre-skylo-sti-skydra)

Η αμαξοστοιχία 83, που εκτελούσε το δρομολόγιο Θεσσαλονίκη-Φλώρινα, υπέστη μηχανική βλάβη, όταν παρέσυρε σκύλο, στο ύψος της Σκύδρας.
Ως αποτέλεσμα, η αμαξοστοιχία ακινητοποιήθηκε, ενώ οι 50 επιβαίνοντες πρόκειται να μεταφερθούν στον τελικό τους προορισμό με λεωφορείο μισθωμένο από την ΤΡΑΙΝΟΣΕ, σύμφωνα με σχετική ανακοίνωση.
