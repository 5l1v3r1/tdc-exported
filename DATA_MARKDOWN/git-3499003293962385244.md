
# В Зеленограде жителей познакомят с благотворительными проектами

Published at: **2019-11-07T11:50:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/zelao/v_zelenograde_zhiteley_poznakomyat_s_blagotvoritelnymi_proektami/)

В ближайшее воскресенье, 10 ноября, в Коворкинг-центре НКО в Зеленограде (корп. 145) пройдёт День открытых дверей. Посетители узнают о проектах некоммерческих организаций своего округа и познакомятся с услугами Коворкинг-центра. Начало в 12:00.
В этот день Ассоциация профессионалов социального проектирования проведёт занятие по определению уровня стресса у взрослых и детей, даст рекомендации по уменьшению уровня стресса. Также все желающие смогут посетить мастер-класс по изготовлению поделок из бросовых материалов.
Благотворительный фонд помощи бездомным животным «Ника» расскажет «Как стать волонтёром приюта для бездомных животных» и организует сбор кормов для животных приюта.
