
# 朝日奈央『1万人の第九』合唱団に参加「NGなしで頑張りたい」

Published at: **2019-11-06T18:08:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148158/full/)

　タレントの朝日奈央が、12月1日に大阪城ホールで行われる『第37回　サントリー1万人の第九』に、合唱団のひとりとして参加することが決定した。 　ソプラノパートで参加する朝日は「楽しみっていうのと同時に、不安はあります。クラシックは初めてだし、曲の歌詞も全部ドイツ語。レベルは高いですが、とにかく楽しんで、NGなしで頑張りたいです」と意気込み。 　9月末頃から練習を始めたといい「時間があれば、移動中もお風呂に入っている時も楽譜を見ながら（第九の）音楽を聞いています。全部覚えないといけないので、大変ですね」と奮闘ぶりを告白。歌唱後のご褒美について「とりあえず誰でもいいので、ハグしてもらいたい」と笑わせていた。 　同イベントの総監督と指揮は佐渡裕氏が務め、ゲストに東儀秀樹、山崎まさよし、松室政哉がゲスト出演する。
