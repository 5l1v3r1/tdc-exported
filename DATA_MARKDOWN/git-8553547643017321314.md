
# Goodram SSD IRDM PRO gen. 2 на 1 ТБ обойдется в $170

Published at: **2019-11-01T09:25:00+00:00**

Author: ****

Original: [Компьютерное Обозрение](https://ko.com.ua/goodram_ssd_irdm_pro_gen_2_na_1_tb_obojdetsya_v_170_130706)

Компания Wilk Elektronik SA сообщила о доступности в Украине нового твердотельного накопителя IRDM PRO gen. 2.
Новинка является наследником накопителя IRDM PRO. Модель второго поколения базируется на чипах флэш-памяти 3D TLC вместо предыдущих MLC. Это позволило увеличить плотность хранения потребительских данных и снизить стоимость накопителя.
Новая модель твердотельного накопителя IRDM PRO gen. 2 (модельный номер IRP-SSDPR-S25C-01T) имеет форм-фактор 2,5 дюйма и подключается через порт SATA3, который обеспечивает скорость передачи данных до 6 Гбит/с. Для нового SSD заявлена скорость в последовательных операциях чтения до 555 МБ/с, а при записи — на уровне 535 МБ/с. Производительность операций случайного доступа с блоками данных 4K для чтения и записи заявлена 92000 IOPS и 86000 IOPS, соответственно.
Твердотельный накопитель IRDM PRO gen. 2 IRP-SSDPR-S25C-01T ёмкостью 1 ТБ будет доступен в розничных сетях Украины с 1 ноября по цене около $170.
