
# Бавария бугун бош мураббийсиз қолиши мумкин

Published at: **2019-11-03T10:51:51+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/bavariya-bugun-bosh-murabbiysiz-qolishi-mumkin)

Клуб бошқарув қўмитаси раиси Карл-Хайнц Румменигге ва президент Ули Хёнесс йиғилишда мураббийнинг клубдаги келажагини муҳокама қилишади.
"Бавария" шанба куни "Айнтрахт"га 1:5 ҳисобида мағлуб бўлди. Айни дамда мюнхенликлар турнир жадвалида 4-ўринни эгаллаб турибди.

        
          Бундеслига турнир жадвали ва тўпурарлари билан танишинг.
