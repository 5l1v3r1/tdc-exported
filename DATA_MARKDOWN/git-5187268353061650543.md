
# Atualização do Windows 10 está a chegar! O seu PC tem as especificações necessárias?

Published at: **2019-11-06T08:00:37+00:00**

Author: **Pedro Simões**

Original: [Pplware](https://pplware.sapo.pt/microsoft/windows/atualizacao-windows-10-especificacoes/)

As atualizações do Windows 10 chegam de forma frequente e ritmada, com duas por ano a serem lançadas. Este é um compromisso da Microsoft e que a empresa tem cumprido. As novidades não têm faltado e as melhorias também não.
Agora que estamos a uma semana da (provável) chegada da próxima versão, é o momento de saber o que esta precisa. É hora de dar resposta à questão se o seu PC tem as especificações necessárias, definidas pela Microsoft.
A cada nova versão do Windows 10 a Microsoft revê as especificações para o seu sistema. Estas definem o hardware mínimo que deve estar presente para que este possa correr sem problemas e com uma experiência de utilização aceitável para o utilizador.
Na atualização de novembro de 2019 não houve exceção e estas especificações surgiram. A empresa revelou agora o que requer, não fugindo muito do que era pedido nas versões anteriores. Além das exigências ao nível do processador, agora alargou aos restantes componentes.
Assim, para a atualização 1909, a Microsoft requer pelo menos 1GB de RAM para a versão de 32 bits e 2GB para a versão de 64 bits. No campo do armazenamento, é exigida a presença de 32GB, sendo, contudo, requerido 7GB para a atualização.
Entretanto, em termos do processador exigido, existem agora novidades nos que são suportados. Isso tem por base um processador de 1GHz ou superior, com suporte para PAE, NX e SSE2. Na área gráfica, é exigido uma resolução de ecrã de pelo menos 800×600. A ligação à Internet é um requisito para a instalação e para atualizações.
Espera-se que esta atualização não tenha o impacto das anteriores, muito por culpa do que vai trazer. A Microsoft vai lançá-la nas normais atualizações de segurança mensais, como um normal Patch Tuesday.
Depois de algumas atualizações com problemas graves, espera-se que a próxima seja imune ao passado. Questões como a perda de dados não são aceitáveis e a Microsoft tudo terá feito para sobretudo impedir que situações destas se repitam.
