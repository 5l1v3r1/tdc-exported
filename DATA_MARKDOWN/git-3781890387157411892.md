
# พยม ลั่นไม่ขอร่วมงาน เสรีพิศุทธ์ อีก

Published at: **2019-11-03T11:01:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/politics/739829)


        “พยม” เข้าวัดทำบุญอโหสิกรรม “เสรีพิศุทธ์” หวังหยุดจองเวร ย้ำไม่ขอร่วมงานกันอีก
      
เมื่อวันที่ 3 พ.ย. นายพยม พรหมเพชร  ส.ส.สงขลา พรรคพลังประชารัฐ(พปชร.) กล่าวถึงกรณีที่พล.ต.อ.เสรีพิศุทธ์ เตมียเวส หัวหน้าพรรคเสรีรวมไทย ในฐานะประธานกรรมาธิการ(กมธ.) ป้องกันปราบปรามการทุจริตและประพฤติมิชอบ สภาผู้แทนราษฎรว่า วันนี้ช่วงเช้าตนได้เดินทางไปทำบุญที่วัดมา ก็ถือเป็นวันดีที่จะขออโหสิกรรมกับพล.ต.อ.เสรีพิศุทธ์ ต่างคนต่างอยู่ หยุดจองเวรต่อกัน จากนี้ไม่ขอร่วมงานกับพล.ต.อ.เสรีพิศุทธ์ อีกไม่ว่าจะในฐานะอะไร เพราะช่วง 1 เดือนที่ผ่านมา ตั้งแต่ได้เริ่มทำงานกรรมาธิการชุดนี้ รู้สึกตนจะนอนไม่ค่อยหลับ เสียสุขภาพจิตด้วย ทั้งนี้ตนขอแนะนำ พล.ต.อ.เสรีพิศุทธ์ ให้เข้าวัดทำบุญทำทานเสียบ้าง จิตใจจะได้สงบสุข ไม่รุ่มร้อน อาฆาตจองเวรใคร.
