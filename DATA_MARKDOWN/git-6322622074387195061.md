
# 国行截至10月31日国际储备金滑至1032亿美元

Published at: **2019-11-07T07:20:45+00:00**

Author: **theedgemarkets.com**

Original: [The Edge Markets](https://www.theedgemarkets.com/article/%E5%9B%BD%E8%A1%8C%E6%88%AA%E8%87%B310%E6%9C%8831%E6%97%A5%E5%9B%BD%E9%99%85%E5%82%A8%E5%A4%87%E9%87%91%E6%BB%91%E8%87%B31032%E4%BA%BF%E7%BE%8E%E5%85%83)

（吉隆坡7日讯）国家银行宣布，截至10月31日的国际储备金下滑1亿美元或0.1%至1032亿美元，相比10月15日的1033亿美元。
国行今日发布文告指出，储备金水平足以应付7.6个月的进口和1.1倍的短期外债。
其中，占最大部分的外汇储备从967亿令吉，增至968亿美元，而国际货币基金组织（IMF）储备及特别提款权（SDR）各为11亿美元、黄金达19亿美元，与前两周保持不变，而其他储备资产由25亿美元，跌至23亿美元。
截至10月31日，国行总资产由4601亿9000万令吉，降至4586亿3000万令吉，流通货币从1089亿1000万令吉，增至1099亿7000万令吉。
