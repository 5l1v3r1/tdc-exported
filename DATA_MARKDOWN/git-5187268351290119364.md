
# Yep.PT – o encurtador de URL português

Published at: **2019-11-02T18:00:51+00:00**

Author: **Mário Sousa**

Original: [Pplware](https://pplware.sapo.pt/truques-dicas/yep-pt-encurtador-bitly-tinyurl/)

Existem vários serviços de encurtadores de links em todo o mundo, mas nem todos são iguais, nem todos fazem a mesma coisa, nem todos se destacam.
O Yep.PT é um serviço encurtador de URLs gratuito, em português, com um domínio fácil de memorizar também ele curto por si só.
Assim, podemos começar por descrever este serviço como sendo muito semelhante aos já conhecidos serviços mundiais oferecidos pela bitly ou tinyurl.
A principal particularidade em destaque prende-se com o facto de ser algo criado em Portugal e com um domínio pt.
Então, com este serviço encurtador de URLs, vamos poder transformar o link, por exemplo, do nosso produto ou página https://www.oseudominio.pt/produtos/categoria-e-nome-do-seu-produto/ num link tão curto como https://yep.pt/produto
A saber que encurtar um URL ou Link vai, para o utilizador mais exigente, muito além do aspeto visual. Essa é uma questão de organização e controlo que traz vantagens, para quem assim procurar, numa estratégia de marketing digital ou com aplicação noutras necessidades.
Ao mesmo tempo, face às vantagens, os posts no Twitter vão poder ser otimizados através uso de URLs curtos. Esta ação garante mais espaço para os textos.
Com efeito, já no Facebook, é uma boa ideia usar os links curtos para facilitar a leitura dentro de posts grandes, para nem mencionar a própria partilha dos endereços.
Deste modo, outra característica que podemos considerar interessante deste serviço é a segmentação inteligente que permite, com apenas um link curto, que o link de destino possa ser diferente de acordo com o dispositivo ou sistema operativo do utilizador. É um valor acrescentado.
