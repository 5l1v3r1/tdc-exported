
# 渋野、通算３アンダー３９位…スウィンギングスカート台湾選手権

Published at: **2019-11-03T10:12:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/golf/20191103-OYT1T50145/)

　【新北（台湾）＝野崎尉】米女子ゴルフツアーのスウィンギングスカート台湾選手権は３日、台湾・新北の美麗華クラブ（６５０４ヤード、パー７２）で最終ラウンドが行われ、渋野日向子は６バーディー、１ボギー、１ダブルボギーでスコアを伸ばし、通算３アンダーの３９位だった。横峯さくらは７オーバーの６５位。１８アンダーで並んだ３人によるプレーオフを制し、ネリー・コルダ（米）が連覇を飾った。
