
# В поселке Ёнский сгорела женщина

Published at: **2019-11-05T00:00:00+00:00**

Author: ****

Original: [Мурманский Вестник](https://www.mvestnik.ru/newslent/v-poselke-nskij-sgorela-zhenwina/)

После ликвидации пожара в квартире одного из домов на ул. Школьная в поселке Ёнский 1 ноября было обнаружено тело 85-летней женщины.
Следственно-оперативная группа установила личность погибшей местной жительницы.
Следователи проводят комплекс мероприятий для установления всех обстоятельств произошедшего. Назначена судебно-медицинская экспертиза и пожарно-техническая судебная экспертиза.
“По предварительным данным пожар мог возникнуть в результате неосторожного обращения с огнем самой погибшей. По результатам доследственной проверки будет принято процессуальное решение”, - сообщает пресс-служба областного Следкома.
