
# 羽田空港・国内線ビルで断水、トイレの手洗い用にペットボトル配布

Published at: **2019-11-06T01:57:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191106-OYT1T50193/)

　６日午前、羽田空港の国内線第１、第２ターミナルビルで断水が発生した。ビルを管理・運営する「日本空港ビルデング」によると、午前１１時現在、ビル内の飲食店が営業できず、トイレの手洗い場で水が使えない状態となっている。
　同社によると、午前９時頃から蛇口から水が出ない状態となった。トイレの便器の水は流れるといい、手洗い用に、水が入ったペットボトルを配る対応を取っている。
