
# Названы самые жуткие герои фильмов ужасов всех времен

Published at: **2019-11-02T10:02:37+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-02/nazvany-samye-zhutkie-geroi-filmov-uzhasov-vseh-vremen)

Британская газета The Independent составила рейтинг самых пугающих существ из фильмов ужасов всех времен. Всего в списке оказались 20 персонажей.
Монстр из детской книги, а также герой одноименного хоррора 2014 года Бабадук оказался во главе рейтинга.
Второе место занял лысый вурдалак из немого фильма 1922 года "Носферату. Симфония ужаса", а на третьей строчке обосновался вампир Номак из фильма "Блэйд 2" 2002-го года.
В список также вошли монстры из "Гремлины" (1984), "Кинг Конга" (1933), "Лабиринта фавна" (2006), "Франкинштейна" (1931), "Дрожи земли" (1990) и, как ни странно, Рэндалл из "Корпорации монстров" (2001).
Последнее место досталось инопланетянам из "Чужого" (1979).
