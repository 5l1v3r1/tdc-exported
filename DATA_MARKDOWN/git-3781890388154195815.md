
# ทนายเดชาโดนแล้ว เพจ....ใส่ไข่ปั้นข่าวเท็จทำเสียหาย

Published at: **2019-11-04T23:54:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/regional/740011)


        ทนายเดชาลั่น เตรียมแจ้งความจับเพจแสบ ปั้นข่าวเท็จใส่ความทำเสียชื่อเสียง วอน FCช่วยกัน Report เพจด้วย
      
เมื่อวันที่ 5พ.ย.ผู้สื่อข่าวรายงานว่า เพจเฟซบุ๊ก”ทนายคลายทุกข์”ของทนายเดชา กิตติวิทยานันท์  ทนายความชื่อดัง ได้โพสต์ข้อความว่า “เรียน FC และพี่น้องประชาชนเพื่อทราบ "เพจ....ใส่ไข่"เพจมิจฉาชีพได้นำ LINE ของอาจารย์เดชาปลอมไปโพสต์ และทำให้เกิดความเข้าใจผิดว่าอาจารย์เดชากับนายอัจฉริยะ ต่อหน้ารักกัน ลับหลังนินทากันเพื่อให้เกิดความแตกแยก ฝาก FC ทุกท่านที่รักอาจารย์เดชาช่วยกัน Report เพจ...ใส่ไข่ด้วย ส่วนอาจารย์เดชาจะดำเนินคดีตามกฎหมายต่อไป. ส่วนคนที่เข้าไปกดถูกใจและแสดงความคิดเห็นส่วนใหญ่ ##เป็นสมาชิกอยู่ในกลุ่มเพจทนายคนดัง  โดยทางเพจ”ทนายคลายทุกข์” ได้แคปภาพที่ถูกเพจดังกล่าว ปลอมข้อความแล้วนำไปโพสต์ ทำให้ทนายเดชาเสียหายมาลงยืนยันด้วย.
