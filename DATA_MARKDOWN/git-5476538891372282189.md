
# В очереди из фур в сторону пункта пропуска Привалки сбили человека

Published at: **2019-11-04T03:30:00+00:00**

Author: **autogrodno.by**

Original: [Гродненская правда](https://grodnonews.by/news/proisshestviya/v_ocheredi_iz_fur_v_storonu_pp_privalki_sbili_cheloveka.html)

В воскресенье вечером, 3 ноября, в очереди из фур вблизи пункта пропуска "Привалки" сбили человека, об этом сообщил АвтоГродно  Егор. Мужчина поделился видеозаписью последствий.
- Наезд произошел вблизи того места, где установлены офисы выдачи страховок по направлению к границе с Литвой. Человек был накрыт одеялом, рядом стояла скорая, - говорит очевидец.
