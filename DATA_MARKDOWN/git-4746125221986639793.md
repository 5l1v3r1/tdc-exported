
# インスタ映えする即席麺のアレンジレシピ、『お願いランキング』で放送

Published at: **2019-11-05T23:01:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148104/full/)

　テレビ朝日の深夜バラエティー『お願いランキング』（月〜木　深0：50〜※毎月最終週の木曜は休止　※関東ローカル）で、人気料理研究家・リュウジの魅惑レシピ企画第3弾が11日に放送される。

        【写真】その他の写真を見る
      
　リュウジは、ほぼ毎日ツイッターでレシピを配信。フォロワーが急上昇し続け、ついに先月100万人を突破した。スナック菓子とチーズを合体して作る斬新さが受け、約50万の「いいね」を獲得した「じゃがアリゴ」というメニューに代表されるように、身近な食材を使った「お手軽レシピ」が得意。そんなリュウジが、「絶対おいしいものを作れる！」と豪語するのが、インスタントラーメンの定番「サッポロ一番」（サンヨー食品）。 　今回、「サッポロ一番」のアレンジレシピを考案。「サッポロ一番は昔から食べてきたから、本当にレシピには自信がある！」と意気込むリュウジが作ったのは、「電子レンジで作る！爆盛り油そば」や、「チーズダッカルビ風・味噌ラーメン」「塩らーめんソバメシ」など、「どれも“インスタ映え”するインパクトあるメニュー」ばかり。これからパーティーシーズンということもあり、「ホットプレートを使って、仲間とワイワイしながら」作れる楽しいレシピもある。 　番組では過去に「ケンタッキーフライドチキン」や「アマニ油」を使った「バズ飯」を紹介し、わずか1日で10万回再生を記録するなどそのメニューは大反響。第3弾に向けて、すでに今月2日から　「お願い！ランキング」公式ツイッターでリュウジが太鼓判を押す4品のレシピを配信中。11日の放送では、実際にどのメニューが多く見られたのか、「バズった順」でランキングを作成し、気になる作り方を紹介する。
