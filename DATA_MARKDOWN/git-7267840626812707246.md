
# Нацсовет хочет регулировать YouTube

Published at: **2019-11-07T12:58:00+00:00**

Author: ****

Original: [Главное™](https://glavnoe.ua/news/n357008324-nacsovet-hochet-regulirovat-youtube)

Комитет по вопросам гуманитарной и информационной политики задумывается о регулировании YouTube. Сейчас его члены обсуждают расширение полномочий Нацсовета по вопросам телевидения и радиовещания.
Основание для контроля - тариф "YouTube Premium", который по функциональности приравнивают к интернет-телевидению.
Об этом рассказала заместитель главы Нацсовета Ульяна Фещук. Ее слова приводит "Детектор.медиа":
По ее словам, Нацсовет не планирует напрямую влиять на YouTube. В возможных планах регулятора - добиться регистрации YouTube как сервиса и получить возможность отправлять туда официальные жалобы на контент.
