
# Раскрыты обстоятельства стрельбы на западе Москвы

Published at: **2019-11-02T15:57:49+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-02/raskryty-obstoyatelstva-strelby-na-zapade-moskvy-gde-postradali-dvoe)

РЕН ТВ стали известны обстоятельства стрельбы на улице Академика Анохина в Москве. По предварительным данным, между двумя группами мужчин произошел конфликт возле подъезда.
Вооруженные злоумышленники расстреляли оппонентов, после чего сели в автомобиль и скрылись. В результате стрельбы ранения получили два человека. По предварительным данным, один из них находится без сознания.
Очевидцы сообщили, что слышали около восьми выстрелов.
На месте ЧП работают правоохранители.​
