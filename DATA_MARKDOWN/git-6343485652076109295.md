
# Grupo de Boston consignó documento a la AN para atender la crisis eléctrica venezolana

Published at: **2019-11-07T19:03:08+00:00**

Author: **@vasquezluisda Luis David Vásquez**

Original: [Descifrado](http://www.descifrado.com/2019/11/07/grupo-de-boston-consigno-documento-a-la-an-para-atender-la-crisis-electrica-venezolana/)

El Grupo de Boston, conformado por parlamentarios y ex diputados venezolanos, propusieron un proyecto para solicitar un préstamo especial al Banco de Desarrollo de América Latina (CAF) con administración del Programa de Naciones Unidas para el Desarrollo (PNUD).
Con el objetivo de recuperar el sistema eléctrico venezolano que atraviesa una “situación de emergencia”.

        
          De interés: El Petro no tiene ningún tipo de respaldo
        
      
“Tomando en cuenta el financiamiento gestionado y que aprobaría CAF de hasta $350.000.000 a favor de Venezuela y el análisis técnico realizado, se determinó priorizar al Zulia, Sur Occidente, Nueva Esparta y Caracas resultando una propuesta de incorporación de más de 1.000 MW”, se lee en el comunicado.
El Grupo de Boston pide a la Asamblea Nacional aprobar una “ley aprobatoria del mecanismo humanitario del sector eléctrico” para materializar el proyecto presentado.
