
# เกาหลีเหนือทดสอบ 'เครื่องยิงจรวดซูเปอร์จัมโบ้'ครั้งใหม่

Published at: **2019-11-01T01:27:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/foreign/739410)


        รัฐบาลเปียงยางประกาศความสำเร็จครั้งใหม่ในการทดสอบ "ระบบขีปนาวุธนำวิถีแบบหลายลำกล้องขนาดใหญ่มาก" ขณะที่ผู้แทนพิเศษด้านกิจการเกาหลีเหนือของสหรัฐเตรียมเข้ารับการพิจารณาจากวุฒิสภา ในตำแหน่งรมช.กระทรวงการต่างประเทศ
      
สำนักข่าวต่างประเทศรายงานจากกรุงโซล ประเทศเกาหลีใต้ เมื่อวันที่ 1 พ.ย.ว่าสำนักข่าวกลางเกาหลี ( เคซีเอ็นเอ ) กระบอกเสียงของรัฐบาลเปียงยาง รายงานเมื่อวันศุกร์ ว่านายคิม จอง-อึน ผู้นำสูงสุดของเกาหลีเหนือ ลงพื้นที่สังเกตการทดสอบประสิทธิภาพครั้งใหม่ของ "ระบบขีปนาวุธนำวิถีแบบหลายลำกล้องรุ่นใหม่และขนาดใหญ่มาก" ที่จังหวัดพย็องอันใต้ ตั้งอยู่ริมชายฝั่งทางตะวันตกของประเทศ เทื่อช่วงเย็นของวันพฤหัสบดีที่ผ่านมา โดยขีปนาวุธซึ่งถูกปล่อยออกจากเครื่องยิงรุ่นล่าสุดนี้ สามารถเดินทางไปได้เป็นระยะทางไกลประมาณ 370 กิโลเมตร
