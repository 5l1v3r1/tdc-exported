
# Экстренные службы села Корткерос отработали ликвидацию последствий ДТП

Published at: **2019-11-05T13:18:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Syktyvkar/739993-ekstrennye-sluzhby-sela-kortkeros-otrabotali-likvidaciyu-posledstvii-dtp)


        По легенде, в Корткеросском районе произошло столкновение автомобилей, один из которых в итоге загорелся. Экстренные службы выполнили все задачи в рамках учения по ликвидации последствий ДТП.
      
На автомобильной дороге республиканского значения Сыктывкар – Троицко-Печорск экстренные службы села Корткерос провели учение по ликвидации последствий ДТП. По информации ГУ МЧС России по Республике Коми, специалисты отработали взаимодействие между пожарно-спасательными подразделениями Корткеросского гарнизона и силами РСЧС. Работы выполнялись в сложных погодных условиях, включая снежные заносы.
По легенде, на трассе столкнулись два автомобиля, а пострадавшие оказались заблокированы. Кроме того, одна из машин загорелась. Экстренные службы оперативно прибыли на место аварии, пожарные приступили к ликвидации огня, после чего деблокировали пострадавших, оказали необходимую помощь и передали их бригаде скорой помощи. 
Сообщается, что все задачи были успешно выполнены, силы Корткеросского района звена РСЧС к ликвидации последствий ДТП готовы.
