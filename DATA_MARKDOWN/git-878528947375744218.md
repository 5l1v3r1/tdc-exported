
# オリックス・頓宮バックスクリーン破壊ダ～ン　特大！推定飛距離１５０メートル

Published at: **2019-11-05T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/05/0012849418.shtml)

　オリックス・頓宮裕真内野手（２２）が４日、高知市東部野球場で行われた秋季キャンプでバックスクリーンを破壊する推定飛距離１５０メートルの特大弾を放ち、来季の正捕手獲りを強烈にアピールした。
　１人で１時間打ち続ける特打。頓宮が放った打球は周囲の度肝を抜いた。バックスクリーンへ一直線、先攻の７番打者の名前が表記される部分に直撃し、粉々に砕けた。
　「フォームをチェックしていたので当たったところは見ていません。毎日いろいろ試しているのですが、きょうが一番、いい感じで打てました」
　今キャンプでは大学時代に慣れ親しんだ捕手に再転向。正捕手・若月から定位置奪取を目指している。西村監督は「なかなかあそこまでは飛ばない。いいモノ持っている」と目を細めていた。
