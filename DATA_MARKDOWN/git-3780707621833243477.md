
# Рогозин: тестирование строительства лунной базы роботами может пройти на Эльбрусе

Published at: **2019-11-06T12:54:00+00:00**

Author: **Егор Самсонов**

Original: [ФедералПресс](https://fedpress.ru/news/russia/industry/2359338)


        ОМСК, 6 ноября, ФедералПресс. Директор Роскосмоса заявил, что тестовая отработка строительства лунной базы роботами, возможно, будет проходить на Эльбрусе. Такое предложение выдвинули разработчики.
      
Директор Роскосмоса Дмитрий Рогозин выступил с заявлением о том, что одной из площадок для испытания роботов, которыми собираются строить лунную базу в будущем, рассматривается гора Эльбрус.
По словам Рогозина, с данной идеей обратились сами разработчики роботов-строителей. Также, он подчеркнул, что в перспективе робот должен будет сам преодолеть заданный маршрут, пишет ТАСС.
Ранее «ФедералПресс» писал о планах освоения Луны. Америка представила модели скафандров для выхода на Луну.
