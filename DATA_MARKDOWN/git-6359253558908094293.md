
# Рядом с ТЮЗом столкнулись 2 машины, движение троллейбусов на Шевченко парализовано (ФОТО)

Published at: **2019-11-06T16:56:28+00:00**

Author: ****

Original: [Новый День](https://newdaynews.ru/ekb/675937.html)

В Екатеринбурге на улице Шевченко около ТЮЗа несколько минут назад столкнулись две легковушки. Как передает корреспондент «Нового Дня», автомобили перегородили полосы для движения по Шевченков в сторону Луначарского.
Легковой транспорт объезжает место аварии по встречке. Троллейбусам, следующим в сторону улицы Луначарского, пришлось временно остановить движение.

        Екатеринбург, Служба информации
