
# 松本人志が『行列』初登場 めったに見られない“ゲスト出演”にスタジオ騒然

Published at: **2019-11-03T21:55:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147926/full/)

　ダウンタウンの松本人志が10日放送の日本テレビ系『行列のできる法律相談所』（毎週日曜　後9：00）に初出演することが3日、発表された。スペシャルゲストとしてサプライズ登場しスタジオを騒然とさせる。

        【写真】その他の写真を見る
      
　松本の出演は、行列レギュラーですら知らなかったそうで、大御所の突然の登場に驚きを隠せないスタジオの面々。さらにゲストには「タウンワーク」のCMで松本と共演した間宮祥太朗や、テレビ朝日を退社してフリーとなった宇賀なつみも出演する。 　弁護士紹介コーナーでは、鍛え上げられた肉体の松本を前に、「日本一の握力を持つ男」と言われる新沼大樹が登場し、130キロを超える握力でトランプ一束を真っ二つに引きちぎる芸当を披露。VTRでは、若かりしダウンタウンの姿や、初めて『24時間テレビ』でメインパーソナリティーを務めたときの秘話も公開。今の時代ではありえない、ものすごいエピソードが飛び出す。 　さらに、松本に会わせたい人として、人気YouTuberのフワちゃんが登場。破天荒すぎる言動に松本もなすすべなし!? おもしろフェイスに加工してくれるアプリで強制的に松本＆フワちゃんの2ショットを撮影したり、あげくにフワちゃんと同じ衣装に松本を無理矢理コスプレさせたりとやりたい放題。普段の番組ではめったに見られない、ゲスト・松本人志の姿は必見だ。
