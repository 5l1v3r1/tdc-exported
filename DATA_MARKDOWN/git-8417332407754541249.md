
# Fem utradisjonelle weekend-byer

Published at: **2019-11-05T17:30:00+00:00**

Author: **Helge Baardseth, Vagabond,Julie Luneborg/Vagabond,Eilert Larstorp Paulsen, Vagabond**

Original: [adressa.no](https://www.adressa.no/reise/Fem-utradisjonelle-weekendbyer-12680b.html)

Tbilisi, Tel Aviv, Tirana, Thessaloniki og Trieste. Her er fem nye storbyer som bør oppdages.
Stadig på søken etter noe nytt, uten altfor lang flyreise? Disse fem utvalgte byene kan by på noe litt annet enn de tradisjonelle byene man reiser til på helgetur.
