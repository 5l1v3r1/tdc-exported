
# Игорь Адлейба стал чемпионом мира по боксу среди молодежи

Published at: **2019-11-02T14:25:00+00:00**

Author: ****

Original: [Sputnik Абзахия](https://sputnik-abkhazia.ru/sports/20191102/1028741232/Igor-Adleyba-stal-chempionom-mira-po-boksu-sredi-molodezhi.html)


        Вечер профессионального бокса, в котором принимал участие Игорь Адлейба, состоялся в Екатеринбурге на площадке Академии единоборств 2 ноября.
      
СУХУМ, 2 ноя – Sputnik, Бадрак Авидзба. Абхазский боксер Игорь Адлейба выиграл бой за вакантный титул чемпиона мира по версии WBO среди молодежи, одолев Эдди Кольменареса из Венесуэлы на турнире RCC Boxing Promotions в Екатеринбурге.
