
# Мурманские хирурги стали использовать биодеградируемые импланты

Published at: **2019-11-06T00:00:00+00:00**

Author: **Нуреева Ольга**

Original: [Мурманский Вестник](https://www.mvestnik.ru/newslent/murmanskie-hirurgi-stali-ispolzovat-biodegradiruemye-implanty/)

Детские хирурги-травматологи Мурманского областного клинического многопрофильного центра первыми в регионе стали применять методику лечения с использованием биодеградируемых имплантов. Основной задачей детского травмотолого-ортопедического отделения является лечение повреждений и заболеваний опорно-двигательного аппарата с использованием передовых методов. Биодеградируемые импланты не только обеспечивают фиксацию переломов костей, но так же используются при более сложных ортопедических заболеваниях.После хирургического вмешательства с применением новейших материалов пациенты гораздо быстрее восстанавливаются. Важным преимуществом оперативного лечения с использованием биодеградируемого импланта является отсутствие необходимости в повторной операции для его удаления.
