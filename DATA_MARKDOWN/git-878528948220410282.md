
# きゃりー、原宿で披露“ファッションショー”

Published at: **2019-11-07T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/07/0012855059.shtml)

　歌手でモデルのきゃりーぱみゅぱみゅ（２６）が６日、都内で、スマホ決済「メルペイ」新キャンペーン発表会に出席。その後、原宿・竹下通りのクレープ店で「メルペイ」体験を行った。
　「メルペイ」の新ＣＭに出演するきゃりー。発表会ではＣＭ撮影時と同じ、爽やかな水色のワンピースで登場し、「最近はブルーにハマっていて、お財布もブルーです」とマイブームを披露した。約１時間半後にはチェック柄の衣装で竹下通りに出現。わずかな時間で髪型と衣装をガラッと変える“ファッションモンスター”ぶりを発揮していた。
