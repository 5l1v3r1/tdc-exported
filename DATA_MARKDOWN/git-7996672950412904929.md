
# Оберіг для родини виготовляли в бібліотеці Кропивницького

        Весело та цікаво пройшов наш майстер-клас з виготовлення "Підкови на щастя" в бібліотеці-філії №21
      
Діти виготовили свій оберіг-підкову, який прикрасили різними символами достатку та благополуччя. А матеріалом для виробів стало солоне тісто, ліплення з якого останніми роками стає все більш популярним. Причина дуже проста: зробити його легко і воно абсолютно безпечне для дітей.
Тож кожен з учасників майстер-класу принесе в родину оберіг, зроблений власними руками.
