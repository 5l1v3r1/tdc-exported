
# Энгельсское предприятие задолжало работникам более четырех миллионов рублей

Published at: **2019-11-05T16:21:00+00:00**

Author: ****

Original: [Саратов 24](https://saratov24.tv/news/engelsskoe-predpriyatie-zadolzhalo-rabotnikam-bolee-chetyrekh-millionov-rubley/)

Следственным отделом по городу Энгельсу по факту невыплаты заработной платы завершено расследование уголовного дела по признакам преступления, предусмотренного частью второй статьи 145.1 Уголовного кодекса (Полная невыплата руководителем свыше двух месяцев заработной платы, совершенная из корыстной или иной личной заинтересованности). Об этом сообщает пресс-служба Следственного комитета области.
По данным следствия, что руководитель одного из учреждений, специализирующихся на ремонте военной техники, осуществляющих свою деятельность на территории Энгельса, с января по декабрь 2018 года не выплачивал заработную плату 98 работникам. Общая сумма образовавшейся задолженности составила около 4 500 000 рублей. При этом фактически возможность погашения задолженности перед сотрудниками у юридического лица имелась.
"Уголовное дело возбуждено по материалам, поступившим из государственной инспекции труда по Саратовской области. Следствием собрана достаточная доказательственная база, в связи с чем, уголовное дело направлено прокурору для утверждения обвинительного заключения", - рассказали журналисту телеканала "Саратов 24" в ведомстве.
