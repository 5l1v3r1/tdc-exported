
# Найденную рядом со станцией МЖД мину направили на утилизацию

Author: ****

Original: [MOS NEWS](https://mos.news/news/vao/naydennuyu_ryadom_so_stantsiey_mzhd_minu_napravili_na_utilizatsiyu/)
