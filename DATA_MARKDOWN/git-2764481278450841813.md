
# Новый заказник для хищников создали в Приморье

Published at: **2019-11-01T00:00:00+00:00**

Author: **Телеканал Большая Азия**

Original: [Большая Азия](http://bigasia.ru/content/news/eco/novyy-zakaznik-dlya-khishchnikov-sozdali-v-primore/)

В природоохранной зоне «Комиссаровский» обитают тигры и леопарды.
Отмечается, что с запада к новому заказнику прилегают два китайских резервата «Фэнхуаншань» и «Няоциншань». Там уже ведётся работа по сохранению биоразнообразия. Площадь новой природоохранной зоны превышает 80 тысяч гектаров. 
Приморский заповедник расположен на ключевой для тигра и леопарда территории. Хорошо сохранившиеся места обитания способны обеспечить достаточно высокую численность хищников, сообщила пресс-служба WWF.Напомним, на первом международном симпозиуме по сохранению дальневосточного леопарда, который прошёл во Владивостоке более 20 лет назад, было решено создать трансграничную заповедную территорию для сохранности животных.  Организовать заказник постановила администрация Приморского края. На заповедной территории нельзя вырубать леса и охотиться.
