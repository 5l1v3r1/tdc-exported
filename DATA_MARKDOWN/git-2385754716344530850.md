
# Πετεβίνος: «Προφανώς ζουν σε άλλο πλανήτη»

Published at: **2019-11-05T14:14:00+00:00**

Author: **SigmaliveSports**

Original: [SIGMALIVE](https://www.sigmalive.com/sports/football/cyprus/a-league/595820/petevinos-profanos-zoun-se-allo-planiti)

Για τις αντιδράσεις των Τούρκων λόγω των 6.500 ελληνικών σημαιών που θα κυματίσουν στο ΓΣΠ την Πέμπτη στον αγώνα με την Καραμπάχ, τοποθετήθηκε ο εκπρόσωπος τύπου του ΑΠΟΕΛ, Νεκτάριος Πετεβίνος στο Ράδιο Πρώτο.  «Προφανώς ζουν σε άλλο πλανήτη. Είναι ανυπόστατο κομμάτι η ελληνική σημαία στον ΑΠΟΕΛ. Να υπενθυμίσω ότι τέτοιες ενέργειες είχαμε και στο πρόσφατο παρελθόν.
