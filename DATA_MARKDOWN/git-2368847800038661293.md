
# 沙特阿美石油獲准上市 可能是最大規模IPO

Published at: **2019-11-03T09:45:57+00:00**

Author: ****

Original: [NTD 新唐人電視台](https://www.ntdtv.com/b5/2019/11/03/a102699278.html)

【新唐人北京時間2019年11月03日訊】沙特阿拉伯資本市場局（CMA）今天(3日)宣布，允許國營能源巨擘沙特阿美石油公司（Aramco）在國內股市上市。消息人士透露，該公司會在10日公布計劃首次公開募股說明書，而市場預料將在12月中展開交易。這可能是世界最大規模首次公開募股（IPO）。
這筆交易可能使阿美公司估值達到1.7兆美元，也是沙國王儲穆罕默德．沙爾曼（Mohammed bin Salman）改造經濟、擺脫國家對石油長期依賴的計劃核心。
法新社引述熟知這項募股計畫的消息人士報導，阿美石油公司料將在兩次交易賣出共5%股份，分別在12月沙特阿拉伯證交所（Tadawul）首次掛牌釋出2%股票、明年在海外交易所掛牌上市將再釋出3%股票，目前尚未選出地點。
外界原本預期阿美公司會在10月展開兩階段公開募股的第一部分，但程序遭到延後，據傳是因王儲不滿意對公司的估值不到預期的2兆美元。
阿美公司表示，今年前9月淨收入（net income）達680億美元，營收以及和銷售有關的其他收入總計2440億美元。去年淨收入則達到1110億美元，不但是蘋果公司的2倍，更是埃克森美孚（Exxon Mobil）、荷蘭皇家殼牌（Royal Dutch Shell）的好幾倍。
紐約時報形容，這可能是史上規模最大的IPO，可望打破中國電商巨頭阿里巴巴（Alibaba）2014年在美國紐約上市所創下的紀錄。
