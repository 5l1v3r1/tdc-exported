
# Microsoft даст линуксоидам попробовать свой антивирус в 2020 году

Published at: **2019-11-05T18:29:00+00:00**

Author: **Екатерина Быстрова**

Original: [Anti-Malware](https://www.anti-malware.ru/news/2019-11-05-111332/31220)

В следующем году Microsoft планирует выпустить свой антивирус для систем Linux. Уже на этой неделе (на конференции Ignite Conference) корпорация обещает показать, как специалисты в области кибербезопасности могут использовать Microsoft Defender.
Эту информацию подтвердил Роб Леффертс, вице-президент Microsoft M365 Security. Так что пользователи Linux смогут опробовать Microsoft Defender в 2020 году.
Помимо этого, корпорация распространит функцию Application Guard на все документы Office 365. Ранее эта возможность была доступна только в Edge, она позволяла пользователям безопасно открывать веб-страницу в изолированной виртуальной среде.
Теперь при открытии приложений Office 365 вроде Word или Excel у пользователей будет та же защита.
Таким образом, у злоумышленника не получится загрузить дополнительные злонамеренные составляющие в систему пользователя.
