
# Как в Сыктывкаре вызволяют понтон из ледяного плена

Published at: **2019-11-05T12:56:00+00:00**

Author: ****

Original: [БНК](https://www.bnkomi.ru/data/news/102318/)

Соединяющий Заречье со столицей Коми понтон 3 ноября сорвало и унесло течением. Сегодня коммунальщики возвращали конструкцию на место. Как вызволяли понтон из ледяного плена - в репортаже руководителя фотослужбы БНК Виктора Бобыря.
Напомним, переправа до наступления морозов будет разрешена только пешеходам.
