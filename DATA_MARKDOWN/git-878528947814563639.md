
# 侍ジャパン　稲葉監督が苦渋の決断　不振坂本に代打・山田　押し出し四球から逆転

Published at: **2019-11-05T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/05/0012851956.shtml)

　「プレミア１２・１次ラウンド、日本－ベネズエラ」（５日、桃園国際野球場）
　侍ジャパン・稲葉監督が苦渋の決断を下した。
　２点を追う八回、１死満塁。不振に苦しむ１番・坂本（巨人）の打順で代打・山田（ヤクルト）を起用。その初球に特大ファウルを打ち相手にプレッシャーをかけると、フルカウントから押し出し四球を選んだ。
　１点差に迫り、なお１死満塁で菊池（広島）が同点適時打を放った。さらに近藤（日本ハム）も押し出し四球を選び、逆転した。
