
# プレミア１２、日本が白星発進…ベネズエラ破る

Published at: **2019-11-05T14:09:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/npb/20191105-OYT1T50251/)

　野球の東京五輪予選を兼ねる国際大会「プレミア１２」（読売新聞社協賛）は５日、台湾・桃園などで１次ラウンドＢ組の２試合が行われ、日本代表「侍ジャパン」は８―４でベネズエラを破り、白星発進した。台湾は６―１でプエルトリコに勝った。メキシコのサポパンでは４日、Ａ組の米国がドミニカ共和国を１０―８で下して２勝１敗とし、日本で行われる２次ラウンド進出を決めた。Ｃ組は６日にソウルで始まる。
