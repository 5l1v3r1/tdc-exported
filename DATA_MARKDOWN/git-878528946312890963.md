
# ＤＪ　ＫＯＯ「ディスコは出会いの場」も２０歳の長女には…

Published at: **2019-11-06T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/06/0012853562.shtml)

　ＴＲＦのリーダー・ＤＪ　ＫＯＯ（５８）が６日、都内で行われたミュージカル「サタデー・ナイト・フィーバー」（１２月１３～２９日、東京国際フォーラムホールＣ）の記者発表に出席した。
　ＤＪ　ＫＯＯはモデルでタレントのアンミカ（４７）とともに同ミュージカルの公式サポーターに就任しているが、ディスコにまつわる思い出を「奥さんと出会ったのはディスコでした。ＤＪをやっていて、奥さんがリクエストしに来て出会いました」と明かした。
　ＤＪ　ＫＯＯは「ディスコは出会いの場でもあるんです」と強調したが、２０歳になった長女には「彼氏？今は勉強してほしいけど、僕も１９で（奥さんと）知り合ったからね。何かあったらアンミカさんに相談します」と弱気な顔を見せた。
　会見には主演ダンサーのリチャート・ウィンザー（３７）もゲストで登場。映画で世界中の女性をとりこにしたジョン・トラボルタばりの白のスーツでビー・ジーズの名曲「Ｓｔａｙｎ，Ａｌｉｖｅ」「Ｎｉｇｈｔ　Ｆｅｖｅｒ」などに乗り、華麗なダンスを披露した。
　楽屋が一緒だったＤＪ　ＫＯＯは「カウントしながら踊りの練習をしてるんですよ」とウィンザーの熱心さに感心しきり。イングランド出身のウィンザーもラグビーの大ファンで、高校時代にラグビーをやっていたＤＪ　ＫＯＯも「ラグビーの話をしました」とニヤリ。ウィンザーも会場のファンに向け「Ｗ杯の中でも素晴らしい大会だったと思う。日本もすごく強いチームだった」と賛辞の言葉を口にした。
