
# വന മധ്യത്തിൽ തകരാറിലായി കെഎസ്ആർടിസി ബസ്, യാത്രക്കാർ കുടുങ്ങി

Published at: **2019-11-04T05:13:00+00:00**

Author: **മനോരമ ലേഖകൻ**

Original: [ManoramaOnline](https://localnews.manoramaonline.com/thrissur/local-news/2019/11/04/thrissur-ksrtc-struck-inside-forest.html)

അതിരപ്പിള്ളി∙ ചാലക്കുടി മലക്കപ്പാറ സർവിസ് നടത്തുന്ന കെഎസ്ആർടിസി. ബസ് വന മധ്യത്തിൽ തകരാറിലായി യാത്രക്കാർ വഴിയിൽ കുടുങ്ങി.അവധി ദിവസ്സമായിരുന്നതിനാൽ വിനോദ സഞ്ചാരികളാണ് ബസ്സിലുണ്ടായിരുന്നത്. ചാലക്കുടിയിൽ നിന്ന് രാവിലെ  8.10 ന് മലക്കപ്പാറയിലേക്ക് പോയ ബസ്സാണ്  12.20 ന് തിരികെ വരുന്ന വഴി ആനക്കയത്ത് ഫോറസ്റ്റ് സ്‌റ്റേഷന് സമീപം ലീഫ് സെറ്റിന് തകരാർ സംഭവിച്ച് വഴിയിൽ അകപ്പെട്ടത്.
ബസ്സിലുണ്ടായ 28 യാത്രക്കാരെ സ്വകാര്യ ബസ്സിലും വിനോദ സഞ്ചാരികളുടെ വാഹനങ്ങളിലും കയറ്റി ജനവാസ മേഖലയിൽ എത്തിച്ചു.വനത്തിൽ കുടുങ്ങിയ ബസിൽ വനിതാ കണ്ടക്ടറാണ് ഉണ്ടായിരുന്നത് .മലക്കപ്പാറ സർവീസ് നടത്തുന്ന കെ എസ് ആർ ടി സി ബസ്സുകൾ പാതി വഴിയിൽ നിലയ്ക്കുന്നത് നിരന്തര സംഭവമാണെന്ന്  നാട്ടുകാർ പറയുന്നു.
