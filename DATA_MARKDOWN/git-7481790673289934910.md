
# 日テレ、史上初の５連覇…なでしこリーグ通算１７Ｖ

Published at: **2019-11-02T06:02:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/soccer/20191102-OYT1T50226/)

　サッカーのなでしこリーグは２日、各地で最終節が行われ、日テレがマイナビ仙台に４―０で快勝し、史上初となる５連覇を達成した。通算では１７度目の優勝。
　仙台市のユアテックスタジアム仙台で行われた試合で、日テレは４０分過ぎ、ＦＷ小林のゴールで先制。６０分頃に小林がクロスを頭で合わせて２点目を決めると、終了間際には、さらに２点を追加した。
