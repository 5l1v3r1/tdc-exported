
# Яременко показал заявление об отставке без даты

Published at: **2019-11-02T16:31:00+00:00**

Author: **InfoResist**

Original: [InfoResist](https://inforesist.org/yaremenko-pokazal-zayavlenie-ob-otstavke-bez-daty/)

Глава комитета Верховной Рады по вопросам внешней политики Богдан Яременко 1 ноября опубликовал в Facebook заявление об уходе с должности, не указав дату.
Стоит отметить, что там, где должна быть указана дата написания заявления, написано только «ноября 2019» без конкретного числа.
Напомним, народный депутат Богдан Яременко (фракция «Слуга народа») написал заявление о сложении полномочий главы комитета по вопросам внешней политики.
