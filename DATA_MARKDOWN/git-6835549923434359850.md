
# Лучший бомбардир минского Динамо Шор покинул клуб

Published at: **2019-11-05T10:43:00+00:00**

Author: ****

Original: [Allhockey.ru](https://allhockey.ru/news/show/369077-Luchshij_bombardir_minskogo_Dinamo_Shor_pokinul_klub)

Контракт с нападающим Дрю Шором расторгнут по соглашению сторон без выплаты компенсации, сообщает официальный сайт минского "Динамо".Нападающий Шейн Принс выставлен на драфт отказов. Напомним, что согласно статье 39 Правового регламента КХЛ игрок, помещенный в список отказов, в течение 48 часов может быть выбран другим клубом лиги.Нападающий Андрей Степанов, защитники Роман Дюков и Даниил Бокун командированы в фарм-клуб "Юность-Минск". Нападающий Никита Феоктистов командирован в фарм-клуб "Динамо-Молодечно".
