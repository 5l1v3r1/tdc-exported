
# Роспотребнадзор: Ситуация по гриппу и ОРВИ в Мурманской области спокойная

Published at: **2019-11-06T11:49:00+00:00**

Author: ****

Original: [Информационное агентство Nord-News](https://nord-news.ru/news/2019/11/06/?newsid=116652)

Роспотребнадзор по Мурманской области заявил, что эпидемиологическая ситуация по гриппу и ОРВИ в регионе спокойная.
- Показатели заболеваемости находятся ниже значений расчётных эпидемических пороговых величин во всех возрастных группах. По сравнению с предыдущей неделей отмечается снижение темпа регистрации ОРВИ. Групповых заболеваний ОРВИ не зарегистрировано, - отметили в ведомстве.
По данным вирусологического мониторинга заболеваемость ОРВИ на прошедшей неделе определялась циркуляцией респираторных вирусов не гриппозной этиологии.
В области продолжается прививочная кампания против сезонного гриппа.
