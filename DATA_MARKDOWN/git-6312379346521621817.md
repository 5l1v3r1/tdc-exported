
# Задержанный мужчина напал на полицейского в Москве

Published at: **2019-11-04T11:57:48+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-04/zaderzhannyy-muzhchina-napal-na-policeyskogo-v-moskve)

Мужчина напал на сотрудника полиции, когда его выпускали из изолятора временного содержания в Марьино Москве.
По данным РЕН ТВ, ранее он был задержан по подозрению в разбойном нападении. 
Все произошло, когда правоохранители предложили ему подписать документ, после чего хотели отпустить его. Однако задержанный порвал бумагу, а затем - бросился на одного их силовиков и вывихнул ему руку.
