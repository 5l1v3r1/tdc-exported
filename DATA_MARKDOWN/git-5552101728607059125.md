
# 張立東撩妹慘遭連環打槍　曾智希被取笑走心爆哭

Published at: **2019-11-07T12:09:00+00:00**

Author: **蔡世偉**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191107ent033/)

17 Media馬拉松式直播綜藝節目《明星製作人》，由阿Ken和納豆各自帶隊攜手主持，兩隊每週製作節目透過直播觀眾滿意度比拚，本週再次邀請到上次代班主持的《通靈少女》學姊謝翔雅和盧彥澤擔任來賓，進行了激烈的水果運動大戰。
謝翔雅前一天在家被菜刀切傷手縫了4針，張立東在邀約她上節目時曾表示會準備好各種水果和鱸魚湯慰問她，一到現場還不忘趁機想施展招牌撩妹招式但話都還沒說到重點，就被謝翔雅打斷他吐槽說：「好囉，聽過了！」張立東不死心轉頭撩主持群中的21歲直播主思璇Soda「我吃魚只吃魚頭，你知道為什麼嗎？」沒想到思璇對他搖手說「不要給我你的餘生喔！」張立東無奈喊「她怎麼也聽過了啊？」
車程中納豆關切曾智希怎麼壓力大到嘴角冒痘痘，曾智希對上週謝翔雅代班的篡位危機仍有陰影說「我很難過，昨天就哭了一整晚，我不想面對這一切，你們變心了！」納豆安慰「變得只有張立東，我是愛你的，如果今天謝翔雅跟你掉到水裡，我一定先救你，真的啊，然後我陪謝翔雅一起淹水」笑翻眾人。
張立東念念不忘上週與謝翔雅的互動還頻頻取笑曾智希，不少網友留言挺曾智希，納豆稱讚曾智希真的很不錯，長得又漂亮，曾智希突然神色有異跟納豆說「你不要稱讚我，我會想哭」說著轉頭靠向椅背開始噴淚哭哭，在直播中哭了近20分鐘，讓直播觀眾看到都傻眼，紛紛留言安慰曾智希，張立東緊張安慰，還和納豆開始合唱《說好不哭》，張立東把智希名字融入歌詞，納豆立刻拿掉帽子眼鏡做樣子抱吉他認真模仿起阿信，兩人歌聲終於讓爆哭的曾智希破涕為笑。
