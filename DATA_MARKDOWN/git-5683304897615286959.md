
# ധവാന്റെ ഔട്ട്, പാളിയ റിവ്യൂ; പന്തിന്റെ ‘പിഴവുകളിൽ’ കുരുങ്ങി ടീം ഇന്ത്യ

Published at: **2019-11-04T09:23:00+00:00**

Author: **മനോരമ ലേഖകൻ**

Original: [ManoramaOnline](https://www.manoramaonline.com/sports/cricket/2019/11/04/pant-s-drs-move-gets-comical-reaction-from-rohit-sharma.html)

ന്യൂഡൽഹി∙ അരുൺ ജയ്റ്റ്ലി സ്റ്റേ‍ഡിയത്തിൽ നടന്ന ഇന്ത്യ– ബംഗ്ലദേശ് ഒന്നാം ട്വന്റി20 മത്സരം ടീം ഇന്ത്യയ്ക്ക് കയ്പേറിയ ഓർമയാണു സമ്മാനിച്ചത്. ഇന്ത്യയെ ബംഗ്ലദേശ് ഏഴു വിക്കറ്റിനാണു പരാജയപ്പെടുത്തിയത്. ബംഗ്ലദേശ് ക്രിക്കറ്റിന്റെ ചരിത്രത്തിൽ ആദ്യമായാണ് ഇന്ത്യയ്ക്കെതിരെ ട്വന്റി20 മത്സരത്തിൽ ഒരു ജയം സ്വന്തമാക്കുന്നത്. ഈ മത്സരം ടീം ഇന്ത്യയിൽ ആരൊക്കെ മറന്നാലും ഇന്ത്യൻ വിക്കറ്റ് കീപ്പർ ഋഷഭ് പന്ത് മറക്കാൻ ഇടയില്ല. യുവ വിക്കറ്റ് കീപ്പറുടെ ‘പിഴവുകള്‍ക്ക്’ ടീം ഇന്ത്യ നൽകേണ്ടി വന്നതു വലിയ വിലയാണ്. മികച്ച ബാറ്റിങ് കാഴ്ച വച്ച ശിഖർ ധവാന്റെ വിക്കറ്റ്, ഒരു റിവ്യു ചാൻസ് എന്നിവയെല്ലാം പന്തിന്റെ പിഴവിലാണു നഷ്ടമായതെന്നാണ് ആരാധകർ പറയുന്നത്.
ബംഗ്ലാദേശിനെതിരായ പത്താം ഓവറിലായിരുന്നു പന്തു കാരണം ഡിആർഎസ് നഷ്ടമായ സംഭവം. ബംഗ്ലദേശ് താരം സൗമ്യ സർക്കാരിന്റെ നിർണായകമായ വിക്കറ്റിനു വേണ്ടിയായിരുന്നു ഇന്ത്യയുടെ ‘പാളിയ’ പോരാട്ടം. സൗമ്യ സർക്കാരിന്റെ ബാറ്റിൽ തൊടാതെ വന്ന ബോൾ ഋഷഭ് പന്ത് പിടിച്ചെടുത്തു. ഉടൻ ഔട്ടിനായി താരം അപ്പീൽ ചെയ്തു. പക്ഷേ അംപയർ ഔട്ട് അനുവദിച്ചില്ല. തുടർന്ന് ഋഷഭ് പന്തിന്റെ ഉറപ്പിൽ ക്യാപ്റ്റൻ രോഹിത് ശർമ ഡിആർഎസിന് ആവശ്യപ്പെട്ടു. ഡിആർഎസിലും ഔട്ട് അല്ലെന്നു തെളിഞ്ഞതോടെ ഋഷഭ് പന്തിന്റെ പിഴവിനെച്ചൊല്ലി രോഹിത് കുറ്റപ്പെടുത്തുന്നതും വിഡിയോ ദൃശ്യങ്ങളില്‍ കാണാമായിരുന്നു.
