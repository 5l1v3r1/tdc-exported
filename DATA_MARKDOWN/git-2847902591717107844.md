
# Монахи из Дрепунг-Гоманга возведут две песочные мандалы

Published at: **2019-11-01T17:49:00+00:00**

Author: ****

Original: [ГТРК «Ставрополье»](https://stavropolye.tv/news/127251)


        В Калмыкии возведут две песочные мандалы. Вырисовывать священные знаки и рисунки будут служители из монастыря Дрепунг-Гоманг.
      
Начнут работы с мандалы Манджушри — Будды. Она олицетворяет жизненную энергию, силу, позволяет избежать всех преград и препятствий, способствует уменьшению негативной кармы.
Создание мандал — кропотливый процесс. Священнослужители несколько дней скрупулезно создают их из цветного песка и специальных инструментов. Все это время мастера молятся и просят благословения.
Увидеть мандалы можно будет с 3 по 10 ноября. Затем по традиции их разрушат. Этот ритуал подчеркивает один из главных постулатов буддизма — идею о непостоянстве всего сущего.
