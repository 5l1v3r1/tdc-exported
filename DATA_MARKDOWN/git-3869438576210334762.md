
# Андрей Чибис поделился планами сотрудничества с «Яндекс»

Published at: **2019-11-05T00:00:00+00:00**

Author: ****

Original: [Мурманский Вестник](https://www.mvestnik.ru/newslent/andrej-chibis-podelilsya-planami-sotrudnichestva-s-yandeks/)

На своей странице Андрей Владимирович рассказал о встрече с гендиректором «Яндекс» в России Еленой Буниной. В регионе планируют прорабатываться такие сервисы, как:
«Яндекс.Учебник» — сервис для учителей и школьников. Уже используется в регионе, но задача сделать его ещё более популярным. И детям нравится, и их оценки улучшаются.
«Яндекс.Лицей» — двухлетние курсы для старших школьников по программированию.
«Яндекс.Практикум» — он-лайн переобучение с наставником для взрослых.
Так же планируется внедрить систему «Яндекс.Транспорт», использующуюся уже по всей России. В ней можно отслеживать движение транспорта в режиме реального времени.
«В ближайшее время ждём десант от “Яндекса” в Мурманской области. Будет работать в поле, - пишет Андрей Чибис. - На этой неделе сформируем план с четкими сроками и стартуем. Отпишусь по итогам».
