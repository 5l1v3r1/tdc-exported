
# 五輪マラソン「大通公園発着・チケット販売なし」ＩＯＣコーツ氏が支持

Published at: **2019-11-02T08:32:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/olympic/2020/20191102-OYT1T50231/)

　２０２０年東京五輪のマラソンと競歩の札幌開催決定を受け、国際オリンピック委員会（ＩＯＣ）調整委員長のジョン・コーツ氏は２日、マラソンで大会組織委員会が検討している「大通公園発着のコース」「チケット販売なし」の案を支持する考えを明らかにした。読売新聞の取材に答えた。また、東京都が競技準備のために支出し、他の目的に活用できない経費の負担については「ＩＯＣは責任から逃れることはしない」と語った。
　ＩＯＣは当初、札幌ドーム発着のコースを提案していたが、コーツ氏は「大通公園発着で非公式に同意ができている」と明かした。ドーム入り口の拡張、周辺道路の改善、陸上仕様への改修に、時間とコストがかかることを勘案したという。大通公園発着の場合、大規模な観客席の設置が難しいため、チケットなしで自由に観戦できる形とすることに賛同した。コース詳細は、１２月のＩＯＣ理事会で決まる見通し。
　東京都への補償については「（正確な金額の）算出は難しい」としながらも、コース上の遮熱舗装なども含めて「検討の対象になる」とした。新たに生じるコストは、まず組織委が負担し、ＩＯＣが必要に応じて補填（ほてん）する形になるという。
　トーマス・バッハＩＯＣ会長が提案した五輪後の「祝祭マラソン」では、３月の東京マラソンに関連づけて行う可能性も探るとし、「五輪計画と同コースで、老若男女の市民が参加する形で行えれば」と語った。
