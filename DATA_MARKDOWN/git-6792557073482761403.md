
# Во Фрунзенском районе произошло ДТП с участием автобуса. Пострадали три пассажира

Published at: **2019-11-06T00:00:00+00:00**

Author: ****

Original: [«Бумага»](https://paperpaper.ru/papernews/2019/11/06/vo-frunzenskom-rajone-proizoshlo-dtp-s/)

Днем 6 октября на Бухарестской улице, 32 легковой автомобиль не уступил дорогу автобусу, после чего произошло столкновение. Об этом сообщает отдел пропаганды УГИБДД по Петербургу.
В результате аварии пострадали три пассажира автобуса, их госпитализировали с травмами средней тяжести.
По факту ДТП проводится проверка, сообщают в УГИБДД по Петербургу.
