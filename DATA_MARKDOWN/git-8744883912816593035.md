
# Украина хочет обсудить транзит газа на встрече в «нормандском формате»

Published at: **2019-11-01T12:57:41+00:00**

Author: ****

Original: [Коммерсантъ](https://www.kommersant.ru/doc/4147674)

Киев хочет обсудить на встрече лидеров стран—членов «нормандского формата» (РФ, Франция, Германия и Украина) вопрос транзита российского газа, заявил министр иностранных дел Украины Вадим Пристайко. Ранее президент РФ Владимир Путин говорил о готовности к переговорам о поставках на Украину газа по значительно сниженным ценам.
«Обойти этот вопрос невозможно, но он не является основой для восстановления мира на Украине. Это важная часть переговоров, важная для Украины, потому что десятилетний контракт заканчивается и есть возможность встретиться»,— сказал господин Пристайко (цитата по «РИА Новости»).
Ранее Еврокомиссия осталась разочарована исходом консультаций в Брюсселе 28 октября между Россией и Украиной по продолжению транзита российского газа после 1 января 2020 года, когда истекает контракт между «Газпромом» и «Нафтогазом». Как следует из высказываний вице-председателя Еврокомиссии Мароша Шефчовича, сторонам не удалось продвинуться вперед.
