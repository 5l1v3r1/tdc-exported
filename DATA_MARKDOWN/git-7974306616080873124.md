
# Основатели Kabanchik.ua запустили приложение FlashBeats для световых шоу из вспышек смартфонов

Published at: **2019-11-04T12:01:00+00:00**

Author: ****

Original: [AIN.UA](https://ain.ua/2019/11/04/osnovateli-kabanchik-zapustili-flashbeats/)

Роман Киригетов и Александр Юрьев, основатели сервиса Kabanchik.ua, запустили новый проект — приложение для синхронизации вспышек FlashBeats.
С его помощью можно создавать масштабные световые шоу с тысячами мобильных устройств. Они будут мигать вспышками в такт музыке. Сфера применения: концерты, события под открытым небом, ночные клубы.
Программа пока доступна на iOS, до конца года появится Android-версии.
Монетизация осуществляется через подписку стоимостью $2,99 месяц. Pro-версия позволяет конвертировать свою музыку (Apple Music, Spotify), организовывать «вечеринки» и приглашать в них других пользователей. Для гостей приложение бесплатное.
Если организатор знает местоположение каждого смартфона, то можно составлять из смартфонов символы, изображения. Кастомное световое шоу для крупных мероприятий организовывают по отдельной стоимости. В Украине FlashBeats уже ограниченно тестировали на Atlas Weeknd, готовятся другие интеграции.
Сейчас у FlashBeats более 50 000 загрузок, ключевые рынки: США, Япония и Китай. По словам Киригетова, в проект они вложили более $10 000. Инвесторов пока не ищут.
