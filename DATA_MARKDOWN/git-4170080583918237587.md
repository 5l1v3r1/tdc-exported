
# Новости Тулы: В Тульской области уточнят запрещенные для курения места

Published at: **2019-11-06T14:19:00+00:00**

Author: ****

Original: [Тульская Служба Новостей](https://www.tsn24.ru/novosti-tuly-v-tulskoj-oblasti-utochnyat-zapreshhennye-dlya-kureniya-mesta.html)

В Тульской области уточнят запрещенные для курения места. Соответствующий законопроект рассмотрят в региональном парламенте. Документ разработан для охраны здоровья граждан от воздействия окружающего табачного дыма.
Согласно федеральному закону, власти регионов вправе самостоятельно устанавливать ограничения в отдельных общественных местах и помещениях.
