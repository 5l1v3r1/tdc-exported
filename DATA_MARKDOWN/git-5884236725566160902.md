
# До конца года Минстрой изменит правила об ограничителях на окнах для детей

Published at: **2019-11-06T05:33:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/740210-do-konca-goda-minstroi-izmenit-pravila-ob-ogranichitelyakh-na-oknakh-dlya-detei)


        Свод правил планируется изменить в декабре 2019 года. Полное открытие створок окон коснется только строящихся зданий.
      
В декабре 2019 года Министерство строительства и жилищно-коммунального хозяйства России планирует внести в свод правил по установке ограничителей на окна в квартирах новостроек изменения. Они необходимы для обеспечения безопасности детей. Об этом сообщил ТАСС.
По словам замминистра строительства и ЖКХ Дмитрия Волкова, данный вопрос начали решать с апреля текущего года, в настоящее время правила проходят финальную стадию согласования. Свод правил станет действовать через полгода после выпуска.
Только в строящихся зданиях можно будет открывать окна квартир настежь. Дмитрий Волков отмечает, что в уже построенных зданиях по всей стране будет сложнее это сделать, а для новых решение это уже принято и будет реализовано. Кроме того, может также увеличиться высота ограждений в квартирах, где есть окна до пола.
