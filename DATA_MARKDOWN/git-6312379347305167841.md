
# Четыре человека госпитализированы с концерта в Зарядье с отравлением

Published at: **2019-11-01T07:41:56+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-01/chetyre-cheloveka-gospitalizirovany-s-koncerta-v-zaryade-s-otravleniem)

Как стало известно РЕН ТВ, четыре человека были госпитализированы из концертного зала "Зарядье" с признаками пищевого отравления.  По имеющимся данным, в больницу попал 12-летний мальчик, двое мужчин и женщина.
Устанавливаются обстоятельства инцидента.
