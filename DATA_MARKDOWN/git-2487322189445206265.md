
# Оксимирон записал трек с фигурантом «московского дела». Тот поучаствовал из заключения

Published at: **2019-11-06T19:31:33+00:00**

Author: **Сергей Звезда**

Original: [TJ](https://tjournal.ru/news/124594-oksimiron-zapisal-trek-s-figurantom-moskovskogo-dela-tot-pouchastvoval-iz-zaklyucheniya)

Рэпер поддерживает задержанных на митингах теперь и в своём творчестве.
Оксимирон записал трек для участия во втором раунде онлайн-баттла Hip-Hop.ru с фигурантом «московского дела» Самариддином Раджабовым. Того обвиняют в угрозе применения насилия к трём полицейским, поскольку он бросил пластиковую бутылку на несогласованной акции в Москве.
