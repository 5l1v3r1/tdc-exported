
# Отчитались: украинские дороги готовы к приходу зимы

Published at: **2019-11-04T12:05:00+00:00**

Author: **Александра Шмарова**

Original: [kiev.vgorode.ua](https://kiev.vgorode.ua/news/sobytyia/412399-mozhno-ne-perezhyvat-ukraynskye-dorohy-hotovy-k-prykhodu-zymy)

По его информации, правительство готовится к зиме совместно с частными компаниями, которые имеют современную снегоуборочную технику. 
Уже сегодня в регионы выехали 54 новые спецмашины. В целом для безопасности на дорогах государственного значения имеется 4641 единица техники, которая готова оперативно реагировать на изменения в погоде. В распоряжении облавтодоров - 3363 единиц техники. Другие подрядные организации имеют еще 1278 единиц. 
Кроме того, по словам премьер-министра, на дорогах государственного значения установят 7 тысяч погонных метров щитов для сдерживания снега, а также предусмотрят более 800 участков для временной остановки автомобилей в случае непогоды. Коммунальщики продолжают получать запасы технической соли.
