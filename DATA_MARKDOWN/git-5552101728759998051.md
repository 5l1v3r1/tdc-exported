
# 【新車登場】優雅與力量-- Vesta Primavera S 125 i-get ABS特仕版

Published at: **2019-11-06T11:30:00+00:00**

Author: **鞠豪傑 Jet Chu**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191104-carandwatch-vespaprimaveras125igetabs/)

繼去年成功推出大受市場歡迎的兩款Primavera特仕版— Yacht Club遊艇特仕版與50週年特仕版後，今年Vespa原廠Piaggio特別將Primavera優雅的氣質與力量結合，注入了活潑朝氣的氣息，打造出Primavera S限量特仕版，其中的S代表了Sporty，讓Vespa Primavera S 特仕版成為最具熱情的Primavera，並推出專屬全新車色：星湛灰與霓虹橘，兩色各限量60台在台上市。
在1968年誕生的Vespa Primavera （義大利語：「春天」之意），從推出之際就因為其小巧、敏捷、時尚的外型，廣受全世界速克達車迷的喜愛，並成為極具Vespa經典的象徵車款之一。
以Vespa經典的鋼製單體式車架打造的Primavera S特仕版，此次推出兩款全新的車色—質感霧面的星湛灰、以及光澤亮面的霓虹橘；在外觀上，Primavera S特仕版擁有了極具風格的專屬設計，讓人一眼就能看出屬於它的不凡魅力，包括黑灰色系的領結飾蓋（搭配搶眼的橘色V型喇叭飾蓋）、車身飾條、前土除飾蓋、輪圈、前後車燈邊框等，黑色座椅則使用橘色縫線增添活力色彩，車側並有專屬的運動車貼，讓Primavera S特仕版更為顯眼。Vespa Primavera S特仕版搭載了12吋鋁合金五幅輪圈，提供車輛較佳的穩定性與安全性，並確保即使在不同路況的道路騎乘時，依然有良好的抓地力，並配備LED頭尾燈與日行燈、晶片防盜系統、與前輪ABS碟式煞車系統，給車主安心的騎乘標準。
全新Primavera S特仕版全面到港，新車上市期間，Vespa總代理太古運通推出高額零利率分期或專屬交車好禮方案二選一，喜歡Vespa的車迷朋友們可至Vespa各授權經銷據點欣賞Primavera S特仕版的獨特魅力!
