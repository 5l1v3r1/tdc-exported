
# స్మార్ట్‌ సిటీలపై బుగ్గన సమీక్ష

అమరావతి, అక్టోబరు 31 (ఆంధ్రజ్యోతి): రాష్ట్రంలో ఆకర్షణీయ నగరాలు (స్మార్ట్‌ సిటీలు)గా ఎంపికైన తిరుపతి, కాకినాడ, విశాఖపట్నంతోపాటు హెరిటేజ్‌ సిటీగా అభివృద్ధి చేయదలచిన అమరావతికి సంబంధించిన ప్రాజెక్టులపై ఆర్థిక మంత్రి బుగ్గన రాజేంద్రనాథ్‌రెడ్డి సమీక్ష సమావేశం నిర్వహించారు. ఆశించిన లక్ష్యాలకు అనుగుణంగా వీటిని తీర్చిదిద్దేందుకు తీసుకోవాల్సిన చర్యల గురించి అధికారులు, నిపుణులతో చర్చించినట్టు సమాచారం.
