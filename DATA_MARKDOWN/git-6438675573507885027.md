
# В Бурятии военный спас семью с двумя детьми из горевшего частного дома

Published at: **2019-11-02T02:04:00+00:00**

Author: ****

Original: [РИА Новости](https://ria.ru/20191102/1560515600.html?utm_source=yxnews&utm_medium=desktop&utm_referrer=https%3A%2F%2Fyandex.ru%2Fnews)

МОСКВА, 2 ноя — РИА Новости. Военнослужащий соединения Войск радиационной, химической и биологической защиты Восточного военного округа (ВВО) спас семью, в том числе двух маленьких детей, при пожаре в частном доме в Бурятии, сообщает пресс-служба округа.По данным ведомства, рядовой контрактной службы Евгений Грешилов по пути на службу заметил густой дым, который шел из дома. Хозяева в этот момент спали. Военнослужащий разбудил семью и через окно достал двух маленьких детей, а затем помог выбраться их родителям.
