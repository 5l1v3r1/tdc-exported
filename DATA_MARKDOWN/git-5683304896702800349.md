
# മാവോയിസ്റ്റ് ഏറ്റുമുട്ടൽ നടന്ന സ്ഥലത്തേക്ക് സിപിഐ സംഘം; ആദിവാസികളെ കാണും

Published at: **2019-11-01T11:25:00+00:00**

Author: **മനോരമ ലേഖകൻ**

Original: [ManoramaOnline](https://www.manoramaonline.com/news/latest-news/2019/11/01/cpi-leaders-visit-maoist-encounter-area.html)

പാലക്കാട്∙ പെ‍ാലീസ്– മാവേ‍ായിസ്റ്റ് ഏറ്റുമുട്ടൽ നടന്ന അട്ടപ്പാടി മഞ്ചക്കണ്ടി ഊരിനുസമീപത്തുള്ള സ്ഥലത്തേക്ക് സിപിഐ അസിസ്റ്റന്റ് സെക്രട്ടറി പ്രകാശ്ബാബുവിന്റെ നേതൃത്വത്തിലുള്ള പാർട്ടി അന്വേഷണ സംഘം എത്തും.
അതീവജാഗ്രതാനിർദ്ദേശം നിലനിൽക്കുന്ന സ്ഥലമായതിനാൽ ഷെ‍ാർണൂർ ഡിവൈഎസ്പി തടഞ്ഞെങ്കിലും വിലക്ക് അവഗണിച്ചാണ് സംഘത്തിന്റെ യാത്ര. മഞ്ചക്കണ്ടി ഊരിലെത്തി ആദിവാസികളുമായി സംസാരിക്കുകയും സംഭവസ്ഥലത്തെ സ്ഥിതിഗതികൾ വിലയിരുത്തുകയുമാണ് ലക്ഷ്യം.
