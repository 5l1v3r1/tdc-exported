
# Панарин обошел Кучерова по количеству очков за первые пять сезонов в НХЛ

Published at: **2019-11-07T05:22:00+00:00**

Author: ****

Original: [Allhockey.ru](https://allhockey.ru/news/show/369237-Panarin_oboshel_Kucherova_po_kolichestvu_ochkov_za_pervye_pyat_sezonov_v_NHL)

Нападающий "Рейнджерс" Артемий Панарин набрал 2 (1+1)очка в матче регулярного чемпионата НХЛ с "Детройтом" (5:1).Это позволило россиянину обойти нападающего "Тампы" Никиту Кучерова по количеству очков за первые пять сезонов в НХЛ (2013-2018), набрав 334 очка (2015 — настоящее время).  Больше всех за указанный период (2005 — 2010) из россиян набрал очков нападающий "Вашингтона" Александр Овечкин — 529 очков.
