
# В Сыктывкаре ограничат движение транспорта по улицам для обслуживания рубежей систем фото-видео фиксации

Published at: **2019-11-01T10:30:00+00:00**

Author: **Комиинформ**

Original: [Комиинформ](https://komiinform.ru/news/188260)

В Сыктывкаре будет ограничение движение транспорта по ряду улиц, сообщает пресс-служба муниципалитета.
В связи с производством ООО "Эльф инжиниринг" плановых работ по обслуживанию рубежей систем фото-видео фиксации правил дорожного движения будет ограничено движение транспорта на территории МО ГО "Сыктывкар" согласно приложению с 5 по 8 ноября 2019 года.

        Перечень участков улично-дорожной сети г. Сыктывкара, подлежащих ограничению движения транспорта.
