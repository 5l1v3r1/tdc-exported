
# Два человека пострадали в массовом ДТП с двумя маршрутками и тремя авто на Белы Куна

Published at: **2019-11-07T12:36:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/740742-dva-cheloveka-postradali-v-massovom-dtp-s-dvumya-marshrutkami-i-tremya-avto-na-bely-kuna)

Утром 7 ноября на улице Белы Куна случилось ДТП с пятью машинами, две из которых – маршрутки. В инциденте пострадали двое. Об этом сообщили в пресс-службе ГИБДД.
