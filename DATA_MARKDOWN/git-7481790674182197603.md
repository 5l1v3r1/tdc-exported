
# 核廃絶決議案、国連総会委で採択…米は今年も棄権

Published at: **2019-11-02T03:38:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/world/20191102-OYT1T50171/)

　【ニューヨーク＝村山誠】国連総会で軍縮問題を議論する第１委員会で１日、日本が提出した核廃絶決議案が賛成多数で採択された。同種の決議案提出は２６年連続で、今回は来年の核拡散防止条約（ＮＰＴ）再検討会議を見据え核軍縮問題に焦点を絞ったが、米国が昨年に続き棄権するなど各国の意見の隔たりが目立った。
　決議案には１４８か国が賛成した。昨年と同じく中国、ロシア、シリア、北朝鮮の４か国が反対し、２６か国が棄権した。米国は、過去のＮＰＴ合意文書の履行の重要性を再確認するといった内容を支持しなかった。高見沢将林（のぶしげ）軍縮大使は「今回明らかになった課題を踏まえ、ＮＰＴ再検討会議に向けて努力を重ねたい」と述べた。
