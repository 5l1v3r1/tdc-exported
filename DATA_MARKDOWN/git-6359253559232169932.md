
# Коммунальщики перекроют перекресток Космонавтов – Челюскинцев для ремонта теплосетей

Published at: **2019-11-06T12:38:07+00:00**

Author: ****

Original: [Новый День](https://newdaynews.ru/ekb/675882.html)

Завтра Екатеринбургская теплосетевая компания частично ограничит движение на перекрестке проспекта Космонавтов и улицы Челюскинцев рядом с железнодорожным вокзалом. На этом участке будет проведен аварийный ремонт тепломагистрали. Ограничения для транспорта продлятся с полуночи 7 ноября до 4:30 8 ноября.
На этот период для автомобилистов правый поворот с Космонавтов будет организован за счет одной из полос движения в сторону улицы Мамина-Сибиряка. Поскольку перекресток является одним из самых загруженных в городе, транспортный комитет мэрии просит водителей заранее выбирать пути объезда.
Также изменятся маршруты общественного транспорта. Троллейбус №3 будет следовать по улицам от Крауля по Малышева, Карла Либнехта, Свердлова до станции «Железнодорожный вокзал». Троллейбус №5 двинется от станции «Коммунистическая» по проспекту Космонавтов до разворотного кольца у станции метро «Уралмаш». Троллейбус № 17 на время ремонта будет отменен.
