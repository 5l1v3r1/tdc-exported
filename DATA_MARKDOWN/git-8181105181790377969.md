
# Фигурист Самарин занял второе место на этапе Гран-при во Франции

Published at: **2019-11-02T14:36:00+00:00**

Author: ****

Original: [Interfax.ru](https://www.sport-interfax.ru/682784)

Москва. 2 ноября. INTERFAX.RU - Российский фигурист Александр Самарин занял второе место на этапе серии Гран-при Международного союза конькобежцев (ISU) во французском Гренобле.
Спортсмен за свое выступление по сумме двух программ набрал 265,10 балла. Первым стал американец Натан Чен с результатом 297,16 балла, тройку победителей замкнул француз Кевин Эймос, набравший 254,64 балла.
Этап Гран-при проходит с 1 по 3 ноября во французском городе Гренобль.
