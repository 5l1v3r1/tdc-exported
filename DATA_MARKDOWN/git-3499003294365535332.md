
# Назвали самое гиблое место на Земле

Published at: **2019-11-03T09:09:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/zhizn/nazvali_samoe_mertvoe_meste_na_zemle/)

Долина Даллол в Эфиопии признана одним из самых безжизненных мест на Земле, сообщила группа французских ученых из Университета Париж-Сюд. Исследование опубликовано в Nature Ecology & Evolution. Пейзаж долины больше похож на другую планету. Там находится действующий вулкан Даллол и уникальная гидротермальная система, где вода по химическому составу представляет собой горячий (до 109 градусов по Цельсию), крайне кислый и соленый раствор. В долине нельзя жить, поскольку вода состоит из рассола с преобладанием магния, которые разрушают клетки.Источник: http://4girls.news
