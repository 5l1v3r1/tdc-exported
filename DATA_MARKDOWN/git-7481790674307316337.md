
# ゆるキャラＧＰ、頂点に長野県のアルクマ…「復興に向け頑張って」

Published at: **2019-11-03T10:32:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191103-OYT1T50147/)

　全国のご当地キャラクターなどが人気を競う「ゆるキャラグランプリ２０１９」の投票結果が３日、長野市で発表され、長野県のＰＲキャラクター「アルクマ」がグランプリに輝いた。
　アルクマは、リンゴなど長野県の特産品のかぶり物をした熊がモチーフで、県内外を歩きながら同県の魅力をくまなく広める役割を果たしてきた。同県は台風１９号の被害が甚大で、阿部守一知事は「復旧・復興に向けて地域が元気を出せるよう、アルクマに頑張ってほしい」と話した。
　２位は大阪府泉佐野市の「一生犬鳴（けんめい）！　イヌナキン！」、３位は熊本県の「ころう君」だった。
