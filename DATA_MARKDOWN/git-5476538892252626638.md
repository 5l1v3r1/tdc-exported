
# В пункте пропуска «Бенякони» возможны пробки. Ремонт на литовской стороне

Published at: **2019-11-04T12:05:00+00:00**

Author: **Служба информации «ГП»**

Original: [Гродненская правда](https://grodnonews.by/news/transport/v_punkte_propuska_benyakoni_vozmozhny_probki_remont_na_litovskoy_storone.html)

Популярныеновости С 1 ноября в пункте пропуска «Шальчиникай» (с белорусской стороны – «Бенякони») осуществляются ремонтные работы дороги на въездном направлении в Литовскую Республику.
В связи с этим движение транспорта затруднено – все транспортные средства (на въезд и на выезд) следуют по выездному направлению из сопредельного государства.
По окончании работ на данном направлении аналогичные работы начнутся на направлении «выезд из Литовской Республики», соответственно движение всего транспорта будет также осуществляться по реверсивному принципу, сообщили в Госпогранкомитете.
Ориентировочное время проведения работ – 3 недели.
Стоит учитывать эту информацию при планировании следования через белорусско-литовскую границу.
