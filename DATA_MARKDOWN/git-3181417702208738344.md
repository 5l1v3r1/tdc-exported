
# Тверская область стала одним из самых популярных регионов для путешествий в новогодние каникулы

Published at: **2019-11-06T10:18:00+00:00**

Author: ****

Original: [Тверские ведомости](https://vedtver.ru/news/society/tverskaya-oblast-stala-odnim-iz-samykh-populyarnykh-regionov-dlya-puteshestviy-v-novogodnie-kanikuly/)

фото: plazaspahotels.ru

        Российский сервис бронирования жилья для отдыха Tvil.ru сообщает, что Верхневолжье вошло в топ-10 самых популярных у туристов территорий России для предстоящих новогодних туристических поездок.
      
По данным сервиса, Тверская область (наряду с Кемеровской и Ярославской областями) особенно приглянулась тем, кто желает хорошо провести будущие новогодние каникулы в компании до 6 человек.
