
# డెడ్‌లైన్లను పట్టించుకోం: అశ్వత్థామరెడ్డి

Published at: **2019-11-05T14:55:14+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=947100)

ADVT
హైదరాబాద్: ఆర్టీసీ కార్మికులను భయభ్రాంతులకు గురిచేస్తున్నారని ఆర్టీసీ జేఏసీ నేత అశ్వత్థామరెడ్డి మండిపడ్డారు. ఇప్పటికైనా ప్రభుత్వం తమతో చర్చలు జరపాలని డిమాండ్ చేశారు. ఆర్టీసీలో కేంద్రానికి 30శాతం వాటా ఉంటుందని తెలిపారు. రాష్ట్ర ప్రభుత్వానికి ఎలాంటి అధికారాలు ఉండవని ఆయన పేర్కొన్నారు. విధానపరమైన నిర్ణయాలు తీసుకోవాలంటే బోర్డు అనుమతి తప్పనిసరన్నారు. కార్పొరేషన్‌ను మార్చాలంటే కేంద్రం అనుమతి తప్పనిసరిగా ఉండాలన్నారు. ఆర్టీసీ కార్మికులు భయపడాల్సిన అవసరంలేదని ఆయన వ్యాఖ్యానించారు. ప్రభుత్వ డెడ్‌లైన్లను పట్టించుకోవాల్సిన అవసరంలేదన్నారు.
