
# 【定期存款2019】快閃3個月賺6千蚊！比較16間銀行港元定存優惠

港元 定期存款 優惠2019持續白熱化，受惠於年尾相繼開業的虛擬銀行，各傳統銀行如恒生、渣打、中銀、工銀、南洋等的定存利率愈來愈高，《經濟一週》將會每周更新，為大家比較哪一間銀行最高息，並醒你4大貼士！更為你計算好存款期內所得利息，例如敍造一百萬定存利息會是多少呢？（2019年10月28日更新）

招商永隆銀行繼續以一個月短存年利率5厘冠絕其他銀行，入場費只需10,000元，相當親民，不過亦有上限限制。假設以上限50,000元存款一個月計算，到期可獲約208元利息。
而高於4厘息的銀行有3間，包括匯豐、恒生及工銀亞洲。當中以工銀亞洲最抵玩，起存額為1,000元至10萬元，假設以上限10萬元存款3個月計算，到期可獲約1,050元利息。
若手持大筆閒錢，100萬元起存額而又高息的銀行可選擇交銀香港、富邦或創興，3間均提供2厘以上年利率，實際年利率及利息可參考下表。

要敘造定期存款未必需要一筆過，可以分批存入資金以達到最低存款額要求。恒生銀行職員指，只要在開立定存戶口時的存款期內達最低存款額要求便可，新資金可選擇分批存入或一次過存入。
而何謂新資金呢？「合資格新資金結餘」是指最新存款總結餘，與30曆日前之同一貨幣結餘對比所增加的金額，並扣減30曆日內，已享用新資金優惠的定期存款本金總額。舉例說，存戶的理財戶口內，30日前的結餘是30,000元，現時結餘是17萬元，即30日內增長14萬元（$170,000-$30,000）。假設期間享用過新資金定期存款優惠20,000元，即「合資格新資金結餘」是12萬元（$140,000-$20,000），而這筆12萬元可享優惠年利率。
不少存戶會以「螞蟻搬家」的方式，將資金搬到銀行戶口敘造定期存款。惟要留意的是，若以現鈔存入，不少銀行須收取大量現鈔存款的手續費。例如部分銀行每人每日存入200張現鈔以上，又或存入等值10萬港元，便須收取總存款金額的0.25%手續費。
以前者為例，若敘造20萬元定存，存入200張1,000元鈔票，便毋須收取手續費。以同額計算，倘若存入的全是500元鈔票，則須存入400張，銀行或會收取手續費，故存入現鈔前亦要計算好鈔票張數。又例如有意敘造25萬港元的定存，而在櫃位存入現鈔，則須收取達625元（$250,000 x 0.25%）手續費，或會較所賺取的利息高。故若要敘造大額定期存款，建議可以開支票方式，將A銀行的資金搬到B銀行開賬戶，便可豁免手續費。
為吸納客戶使用更多銀行服務，部分銀行的優惠定期存款僅限指定客戶、新客戶或新資金，又可能在開立定存賬戶時同時申請其他服務。例如花旗3個月定期存款年利率2.3%的優惠，僅適用於全新花旗私人客戶業務及Citigold戶口客戶、以全新資金開戶，並須持有包括基金、債券、證券、結構性產品或外幣存款等產品。
要留意的是，定期存款會鎖死資金一段時間。在定存期間，如存戶提早取走資金，銀行會收取最低收費200元，並且不會發放利息。故在敘造定存時要三思。
