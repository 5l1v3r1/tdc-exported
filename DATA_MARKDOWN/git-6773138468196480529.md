
# Инна Жиркова в сексуальном неоновом бикини стала королевой пляжа в Дубае

Published at: **2019-11-01T05:30:00+00:00**

Author: **Леся Ловицкая**

Original: [www.cosmo.ru](https://www.cosmo.ru/stars/news/01-11-2019/inna-zhirkova-v-seksualnom-neonovom-bikini-stala-korolevoy-plyazha-v-dubae/)

30-летняя Инна Жиркова отлично проводит время на курорте. 30-летняя обладательница титула «Миссис Россия — 2012» поделилась пикантными кадрами из Дубая.
Жена 36-летнего футболиста Юрия Жиркова отдыхает за границей. Светская львица показала интересный образ, который продумала для красочных фотосессий. Инна выбрала яркий купальник, который позволил ей стать звездой на пляже.
Жиркова облачилась неоновое бикини салатового оттенка. Модная модель бюстгальтера без лямок выгодно подчеркнула шикарную грудь своей хозяйки. Она выбрала двуслойные трусики необычного кроя — с низкой посадкой и пикантной сеточкой сверху.
