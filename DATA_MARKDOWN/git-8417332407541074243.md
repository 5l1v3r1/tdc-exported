
# - Trygdeskandalen må granskes av Stortinget

Published at: **2019-11-04T13:24:25+00:00**

Author: **NTB og Simen Granviken**

Original: [adressa.no](https://www.adressa.no/pluss/nyheter/2019/11/04/Trygdeskandalen-m%C3%A5-granskes-av-Stortinget-20317038.ece)

Arbeiderpartiets leder Jonas Gahr Støre sier trygdeskandalen må behandles i Stortingets kontrollkomité.
Gahr Støre mener Stortinget også må å igangsette egen stortinggranskning.
