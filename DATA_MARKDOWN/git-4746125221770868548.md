
# マイ・ケミカル・ロマンス、再結成を発表 12月LAでライブ

Published at: **2019-11-01T14:03:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147817/full/)

　2013年に解散した米バンドのマイ・ケミカル・ロマンスが、『RETURN』と題した再結成ライブを12月20日、米ロサンゼルスのShrine Expo Hallで行うことを1日（日本時間）、バンドの公式ツイッターなどで発表した。日本でも、所属レーベルのツイッターで正式に再結成が告知された。

        【写真】その他の写真を見る
      
　同バンドは『Three Cheers For Sweet Revenge』で2003年にメジャーデビュー。06年にリリーされたアルバム『ザ・ブラック・パレード』は世界中のティーンエージャーに支持され、大ヒットを記録。2013年3月をもって解散し、その後幾度となく再結成がうわさされてきたが、今回約6年9ヶ月ぶりにカムバックを果たすこととなる。 　併せてバンドの公式インスタグラムも開設。世界中のファンが待ちわびていた再結成の一報は、現地時間ハロウィンの日の発表ということもあり、ツイッターの世界トレンドでも1位となった。
