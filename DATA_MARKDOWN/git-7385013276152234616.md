
# クリスマスにユリとケイが帰ってくる！ 「声物語3　島津冴子 頓宮恭子の大聖夜～エンジェルの時　ラブリーな夢～」開催決定！

Published at: **2019-11-07T15:20:00+00:00**

Author: ****

Original: [アキバ総研](https://akiba-souken.com/article/42250/)

アニメ「ダーティペア」で主演を務めた声優・島津冴子さんと頓宮恭子さんによるイベント「声物語3　島津冴子 頓宮恭子の大聖夜～エンジェルの時　ラブリーな夢～」」が、2019年12月14日（土）、開催される。
1980年代に一世を風靡したSFアニメ「ダーティペア」で、ユリ役を務めた島津冴子さん、ケイ役を務めた頓宮恭子さんによるクリスマスイベント「声物語3　島津冴子 頓宮恭子の大聖夜～エンジェルの時　ラブリーな夢～」が、2019年12月14日（土）、コートヤード・マリオット銀座東部ホテルにて開催される。12月18日に発売予定の「ダーティペア」Blu-ray BOX特典のオーディオコメンタリー収録がきっかけで、実現したという今回の企画は、「ダーティペア」ファンにはたまらない出演者、おもてなしが用意されている。ゲストにグーリィ主任役の沢木郁也さん、ムギ役の巻島直樹さん、第1話にモブ役で出演した難波圭一さんを迎えるほか、PS4用ゲーム「新サクラ大戦」にて隊長・神山誠十郎役を務める若手声優の阿座上洋平さんが出演。オリジナルボイスドラマのほか、愉快なトークショーを披露する。さらに島津さんと頓宮さんが、来場者の思い出の品にサインをしてくれるということで、ぜひお宝を持参して世界に一つだけの「ダーティペア」グッズをゲットしてみてはいかがだろうか？詳細や参加申し込みにかんする情報は、島津冴子さんの公式ブログにてご確認を！
