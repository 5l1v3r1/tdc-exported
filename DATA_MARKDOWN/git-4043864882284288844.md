
# Апеляційний суд вирішив арештувати Дубневича, але знизив заставу до 90 млн

Published at: **2019-11-07T20:15:05+00:00**

Author: ****

Original: [Українська правда](https://www.pravda.com.ua/news/2019/11/7/7231319/)

Апеляційна палата Вищого антикорупційного суду визначила для нардепа Ярослава Дубневича запобіжний захід у вигляді тримання під вартою з альтернативою застави 90 мільйонів гривень.
Джерело: засідання суду 7 листопада
Деталі: Прокурор просив апеляційну палату ВАКС обрати той самий запобіжний захід для Дубневича, який він просив минулого разу: тримання під вартою з альтернативою 93 мільйони гривень.
Адвокати просили відмовити у задоволенні клопотання прокурора.
