
# Hromadná nehoda čtyř aut si vyžádala čtyři zraněné

Published at: **2019-11-06T20:04:00+00:00**

Author: **Redakce**

Original: [Deník.cz](https://www.denik.cz/nehody/dopravni-nehoda-ctyr-aut-si-vyzadala-ctyri-zranene-20191106.html)

"Čtyři zraněné osoby byly převezeny zdravotnickou záchrannou službou do nemocnice. Hasiči na místě provedli protipožární opatření a zasypali uniklé provozní kapaliny sorbentem. Po celou dobu zásahu byla silnice uzavřena," uvádí informace Hasičského sboru Středočeského kraje.
