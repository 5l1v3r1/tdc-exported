
# СКР выясняет, по чьей вине сирота не получил жилье на Ставрополье

Published at: **2019-11-01T11:46:00+00:00**

Author: ****

Original: [ГТРК «Ставрополье»](https://stavropolye.tv/news/127242)


        На Ставрополье следователи проверяют информацию, размещенную ранее в СМИ, что сирота-инвалид не может получить положенное по суду жилье. Об этом сообщили в пятницу в краевом управлении СКР.
      
В публикациях утверждалось, что в 2018 году суд обязал обеспечить мужчину квартирой, однако решение до сих пор не исполнено.
Заместитель руководителя краевого управления СКР Анатолий Демин поручил провести по данному факту процессуальную проверку. По итогам будет дана уголовно-правовая оценка действиям органов власти.
