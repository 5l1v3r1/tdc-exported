
# 高瀬アナ、朝ドラ送りで林遣都に「親近感わくんですよね…」和久田アナは困惑？

Published at: **2019-11-06T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/06/0012852744.shtml)

　ＮＨＫの高瀬耕造アナウンサーが６日、「おはよう日本」での朝ドラ送りで、林遣都が演じる信作の名前を挙げ「親近感がわくんですよね」とコメント。高瀬アナと林は顔が似ているとネットで話題となっていることから、この“朝ドラ送り”も話題となった。
　高瀬アナは「かつてドラマーを目指していた二郎さん、改め信作さん。大人になったらどういう仕事につくんだろうなって、なんか親近感がわくんですよね」と朝ドラ送り。これに和久田麻由子アナは「え？」と困惑の表情を浮かべて首を傾げ、この日の朝ドラ送りは終わった。
　「スカーレット」で信作を演じているのは林遣都。１６年の朝ドラ「べっぴんさん」にも出演し、その時はドラマー志望の青年・河合二郎を演じていた。
　高瀬アナと林はこれまでも似ていると評判で、林は１７年２月に放送された「あさイチ」で、高瀬アナに似ているとネットで話題となっていることに「昔の年上の方に似ていると言われるのは嬉しい」とコメント。当時、あさイチに出演していた井ノ原快彦からも「前髪上げると似ている」と言われていた。
　やはり高瀬アナは林が気になるのか。ネットではこの朝ドラ送りに「いつか２人が並んだところを見たい」「林さんとお顔似てるもんね」「高瀬アナも認識していたことが発覚」「林遣都にシンパシー感じまくりの高瀬アナ」「まさかべっぴんさんの二郎の名前まで出て来るとは」と高瀬アナが林の役どころに触れたことに反響が相次いでいた。
