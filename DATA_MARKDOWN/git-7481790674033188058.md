
# 「雨イジング・スパイダーマン」…雨の日、ビルからビルへ飛び移る泥棒

Published at: **2019-11-02T20:00:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191102-OYT1T50272/)

　新宿のビルに侵入して現金を盗んだとして、警視庁新宿署は２日、住所・職業不詳の男（２６）を建造物侵入と窃盗の容疑で逮捕したと発表した。雨の日に建物を飛び移って盗みを繰り返していたとみられ、同署は「アメ（雨）イジング　スパイダーマン」と呼んで捜査していた。
　新宿署幹部によると、男は４月１４日深夜～１５日未明、新宿区の５階建て雑居ビルに侵入し、屋上のプレハブから６万６０００円を盗んだ疑い。逮捕は１日。調べに対し、黙秘している。
　雑居ビルの屋上と隣接するビルの６階の窓には、男が飛び移ったとみられる形跡が見つかったという。周辺では４～１０月、ビルの高層階を狙った窃盗事件が約２０件発生しており、同署が関連を調べている。
