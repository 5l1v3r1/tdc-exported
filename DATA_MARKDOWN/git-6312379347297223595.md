
# Дело завели после нападения на водителя и кражи 6,5 млн рублей

Published at: **2019-11-01T06:33:48+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-01/delo-zaveli-posle-napadeniya-na-voditelya-i-krazhi-65-mln-rubley)

Возбуждено уголовное дело после нападения со стрельбой на водителя машины на МКАД и кражи у него из салона машины 6,5 миллиона рублей, сообщили в пресс-службе столичного ГУ МВД.
Дело возбуждено по статье о разбое.
По данным полиции, неизвестные на двух автомобилях совершили преднамеренное ДТП с машиной потерпевшего. Затем два злоумышленника разбили стекло в автомобиле, один из них сделал два выстрела из травматического пистолета в сторону водителя, после чего они схватили сумку с деньгами из салона машины, в которой было 6,5 миллиона рублей, и уехали.
По данным РЕН ТВ, раненый из оружия мужчина Александр Кирдан является владельцем мойки. Предварительно известно, что он был знаком с нападавшим. По данным телеканала, ранее стрелок был судим.
