
# Лукашенко отдал арабским шейхам тысячу гектар земли в Минске

Published at: **2019-11-06T09:19:00+00:00**

Author: ****

Original: [charter97.org](https://charter97.org/ru/news/2019/11/6/354584/)

Вчера Лукашенко встретился с премьер-министром Объединенных Арабских Эмиратов шейхом Мухаммедом бен Рашидом аль-Мактумом. В числе прочего он обсудил с ним создание с помощью ОАЭ международного финансового центра в Минске. Портал tut.by узнал, где может расположиться этот объект.
Сейчас Лукашенко находится с визитом в Объединенных Арабских Эмиратах. Как сообщила пресс-служба диктатора, там он встретился с вице-президентом, премьер-министром и министром обороны ОАЭ, правителем эмирата Дубай шейхом Мухаммедом бен Рашидом аль-Мактумом.
— На встрече, в частности, обсуждались перспективные проекты в транспортно-логистической, финансовой сферах. Один из них предусматривает создание в Минске с помощью эмиратской стороны международного финансового центра. В настоящее время завершается разработка концепции реализации данного проекта, — сообщила пресс-служба Лукашенко.
Как рассказали несколько информированных источников, пока в качестве основной площадки под финансовый центр рассматривается северный берег Свислочи в районе водохранилища Дрозды. Примерные границы участка выглядят так:
Предполагается, что в финансовом центре будут не только деловая, развлекательная и туристическая зоны, но и международный выставочный комплекс. «Это будет крупнейший инвестпроект. Площадь — до тысячи гектаров», — говорит один из собеседников портала.
Сейчас арабские инвесторы готовят концепцию комплекса, которая в итоге ляжет на стол Лукашенко. Последнее слово будет за ним, потому что для реализации такого проекта надо принимать указы правителя.
Во-первых, нужно будет вносить изменения в Генплан Минска, чтобы вывести из оборота пахотные земли, на которых хотят строить финцентр. Во-вторых, для подобных проектов обычно устанавливается другой налоговый режим.
Отметим, что шейх Мохаммед контролировал создание большого количества транснациональных бизнес-проектов, включая строительство самого высокого здания в мире — Бурдж-Халифа. Мохаммеду принадлежат два многопрофильных конгломерата — Dubai World и Dubai Holding, которые занимаются строительством, финансами, туризмом, телекомом, перевозками, логистикой.
