
# 侍ジャパン強化試合第２戦、３―０でカナダ破る

Published at: **2019-11-01T13:10:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/npb/20191101-OYT1T50263/)

　２日（日本時間３日）に開幕する野球の国際大会「プレミア１２」（読売新聞社協賛）を前に、日本代表「侍ジャパン」とカナダ代表との強化試合「ＥＮＥＯＳ侍ジャパンシリーズ２０１９」の第２戦が１日、沖縄セルラースタジアム那覇で行われ、日本が３―０で勝ち、１勝１敗で対戦を終えた。日本は２日、プレミア１２の１次ラウンドが行われる台湾に移動し、５日にベネズエラとの初戦に臨む。
