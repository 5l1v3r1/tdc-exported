
# Известный футбольный блогер Красава подписал контракт с «Арсеналом» из Беларуси

Published at: **2019-11-01T16:30:00+00:00**

Author: ****

Original: [charter97.org](https://charter97.org/ru/news/2019/11/1/354054/)

Популярный российский блогер и бывший профессиональный футболист Евгений Савин (Красава) подписал контракт с дзержинским «Арсеналом», который выступает в третьем по силе дивизионе чемпионата Беларуси, пишет tut.by.
Сейчас Евгений Савин делает фильм для своего блога про Александра Глеба. В «Арсенале» выступает его брат Вячеслав.
Савин до этого выступал за ряд российских клубов: «Ротор», «Локомотив», «Томь», «Анжи», «Химки», «Амкар», «Крылья Советов», «Урал», «Арсенал», «Луч-Энергию» и «Тюмень».
Вызывался в молодежную сборную России.
