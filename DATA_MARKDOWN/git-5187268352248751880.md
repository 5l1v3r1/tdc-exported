
# Espanha vai “rastrear” smartphones dos cidadãos durante oito dias

Published at: **2019-11-03T15:40:08+00:00**

Author: **Pedro Pinto**

Original: [Pplware](https://pplware.sapo.pt/informacao/smartphones-espanha/)

Com as tecnologias que existem hoje, facilmente se consegue perceber o paradeiro dos utilizadores. Em Espanha, o Instituto Nacional de Estatísticas (INE) vai acompanhar durante oito dias as movimentações dos cidadãos através dos seus smartphones.
O objetivo, segundo o INE, é realizar um estudo sobre mobilidade usando “informação anónima”.
Durante oito dias os cidadãos espanhóis vão ser “seguidos”. Para isso, o INE irá fazer uso de dados anónimos que resultam de um acordo pioneiro entre as três principais operadoras espanholas. A informação será muito relevante, pois irá permitir perceber para onde se desloca a população espanhola e nesse sentido que serviços públicos disponibilizar e reforçar infraestruturas.
O INE irá também analisar durante dois dias, nos dias 20 de julho e 15 de agosto, para contabilizar as migrações sazonais durante o período de férias. O mesmo será feito também nos dias 25 de dezembro (Natal) e 24 de novembro.
Relativamente a esta ação única, as operadoras já vieram referir que os dados dos smartphones serão entregues de forma agregada não havendo possibilidade de os associar a qualquer utilizador.
A ideia é bastante interessante fazendo-se assim uso de dados que muita das vezes não servem para nada.
