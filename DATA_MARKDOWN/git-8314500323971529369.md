
# Nur so viel Autoversicherung bezahlen, wie man tatsächlich fährt

Published at: **2019-11-05T22:20:00+00:00**

Author: **Daniel Hügli**

Original: [cash](https://www.cash.ch/news/top-news/friday-berlin-nur-so-viel-autoversicherung-bezahlen-wie-man-tatsaechlich-faehrt-1425477)

Eine junge Frau sitzt singend auf einem Auto, Häuserschluchten rundherum, es tauchen weitere gut gelaunte Passanten auf, und bald singen alle im Chor: «Für meine Autoversicherung - zahl ich nur was ich fahr’».
