
# 한동섭 한국방송학회장

Published at: **2019-11-07T18:45:04+00:00**

Author: **조선일보**

Original: [Chosun](https://news.chosun.com/site/data/html_dir/2019/11/08/2019110800363.html)

입력 2019.11.08 03:45
한동섭 한양대 미디어커뮤니케이션학과 교수가 9일 제32대 한국방송학회장으로 취임한다. 국가교육회의 공론화위원, 방송위원회 방송평가위원, 한양대 기획실장 등을 지냈다.
