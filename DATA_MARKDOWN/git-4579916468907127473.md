
# Полеты во сне и наяву. Прыжки по крышам в центре Петербурга закончились смертью

Published at: **2019-11-05T06:22:00+00:00**

Author: ****

Original: [Фонтанка.ру](https://www.fontanka.ru/2019/11/05/020/)

Экстремальное развлечение — прыжки по крышам — закончилось гибелью человека в Адмиралтейском районе Петербурга. 
По данным ИА «Журналистские расследования» (АЖУР), тело мужчины рядом с расселенным домом №36 по улице Шкапина нашли в четвертом часу дня 4 ноября. В полицию с места происшествия доставили 29-летнего местного жителя, который рассказал, что вместе с другом хотел перепрыгнуть с крыши одной заброшки на другую. Судя по всему, у него получилось, у друга — нет. Выживший ранее осуждался по наркотической статье. 
Напомним, четыре аварийных дома на Шкапина стоят заброшенными уже несколько лет. Сначала на их месте хотели возвести парковку, потом решили передать инвестору для проведения комплексной реконструкции. Но и этот план не реализован.
