
# Как будет выглядеть пикап GMC Sierra без кузова?

Published at: **2019-11-01T00:00:00+00:00**

Author: **Алексей Кирилов**

Original: [CAR.RU](https://car.ru/news/automobili/56570-kak-budet-vyiglyadet-pikap-gmc-sierra-bez-kuzova/)


        Один из владельцев GMC Sierra решил, что кузов для автомобиля - это лишний элемент и избавился от него.
      
Теперь GMC Sierra - это не просто грузовик, а особенный. Кроме отсутствия кузова примечателен автомобиль тем, что имеет силовой агрегат V8 с турбиной, система впуска у которого изменена программно.
Также транспортное средство получило внедорожную резину, для которой пришлось дополнительно установить проставки спереди и сзади. Кроме этого, автомобиль оснащен прожекторами из светодиодов.
 
Владелец планирует использовать получившийся «кабриолет» для поездок по лесам и полям. Аналитики считают, что транспортное средство легко справится с кукурузным полем и густой грязью.
Так как автомобиль лишился бамперов, то поездка по лесу для него не проблема, водитель легко проезжает между деревьями там, где нет даже намека на дорогу. Не трудно транспортному средству преодолевать и затопленные места.
Будет ли продавать нынешний владелец модифицированный GMC Sierra - неизвестно. Сейчас автомобиль используется для нужд собственника.
