
# В Архангельске на мосту через Соломбалку отремонтировали лопнувшее покрытие

Published at: **2019-11-02T15:26:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Arkhangelsk/739109-v-arkhangelske-na-mostu-cherez-solombalku-otremontirovali-lopnuvshee-pokrytie)


        В столице Поморья отремонтировали два моста через реку Соломбалка. Рабочие исправили дефекты деформационных швов.
      
В Архангельске отремонтировали мост через реку Соломбалку. Водители могут проезжать без ограничений, сообщает сайт мэрии Архангельска.
МУП «Архкомхоз» завершил ремонт деформационных швов на мосту через реку Соломбалка по улице Советская. Проезд для автотранспорта открыт по всем полосам. Кроме моста по улице Советской Архкомхоз отремонтирован мост по улице Мостовая. На нем также выполнен ремонт деформационного шва.
Ранее НЕВСКИЕ НОВОСТИ сообщали, что в Архангельске ремонтируют мост через протоку Кривяка. Движение закрыто для водителей и пешеходов.
