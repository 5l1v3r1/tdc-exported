
# Кантенинг қачон сафга қайтиши маълум бўлди

Published at: **2019-11-02T04:52:43+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/kantening-qachon-safga-qaytishi-malum-boldi)

“Челси” аъзоси Европа Чемпионлар Лигаси доирасида “Аякс”га қарши кечадиган 5 ноябрдаги учрашувда майдонга туша олади. Бу ҳақида лондонликлар бош мураббийи Фрэнк Лэмпард маълум қилди.
Қайд этиш жоиз, Канте салкам бир ойдан буён майдонга туша олмай келмоқда.
