
# Bejo Sugiantoro Bergabung, Aji Santoso Tetap Pertahankan Uston Nawawi

Published at: **2019-11-05T01:49:00+00:00**

Author: **Bola.net**

Original: [Bola.net](https://www.bola.net/indonesia/bejo-sugiantoro-bergabung-aji-santoso-tetap-pertahankan-uston-nawawi-ed06f8.html)

Bola.net - Komposisi tim pelatih Persebaya Surabaya kembali komplet menyusul bergabungnya, Bejo Sugiantoro. Arsitek asal Sidoarjo tersebut sebelumnya meninggalkan tim untuk menjalani kursus kepelatihan A AFC.
Namun, kehadiran Bejo tidak membuat posisi Uston Nawawi tergeser. Meskipun, Uston awalnya hanya diminta untuk menggantikan Bejo selama kursus di Yogyakarta.
Uston tetap mendampingi Aji Santoso guna menatap sisa kompetisi Shopee Liga 1 2019 yang disiarkan Indosiar. Mereka akan bekerjasama untuk mengangkat performa Persebaya.
Aji bahkan cukup senang bisa didampingi dua asisten pelatih. Menurutnya, keberadaan mereka sangat dibutuhkan untuk memberikan masukan demi kebaikan Persebaya ke depan.
"Justru saya ingin nanti asisten tiga, tidak hanya dua, karena kan banyak input kepelatihan head coach," kata Aji Santoso, pelatih Persebaya
"Kami akan berdiskusi, kalau orangnya lebih banyak lebih bagus, daripada seumpamanya satu aja," imbuh mantan arsitek Persela tersebut.
