
# Ρόδος: Λαμπάδιασαν σταθμευμένα αυτοκίνητα τα ξημερώματα

Published at: **2019-11-07T10:10:35+00:00**

Author: ****

Original: [Η ΡΟΔΙΑΚΗ](https://www.rodiaki.gr/article/425344/rodos-lampadiasan-stathmeymena-aytokinhta-ta-xhmerwmata)


        Στις φλόγες παραδόθηκαν σήμερα τα ξημερώματα δύο αυτοκίνητα που βρίσκονταν σταθμευμένα στην οδό Αισχίνης στο κάτω Ροδίνι.
      
Σύμφωνα με όσα έχουν γίνει γνωστά, περίπου στις 3:30 σήμερα τα ξημερώματα για άγνωστη μέχρι στιγμής αιτία αυτοκίνητο το οποίο ήταν σταθμευμένο σε πάρκινγκ, τυλίχθηκε στις φλόγες με αποτέλεσμα μέσα σε λίγα λεπτά να προκληθεί πυρκαγιά και στο διπλανό σταθμευμένο όχημα
Για την κατάσβεστη της φωτιάς υπήρξε άμεση κινητοποίηση απο την πυροσβεστική υπηρεσία Ρόδου.
Αξιωματικοί της πυροσβεστικής υπηρεσίας Ρόδου αφήνουν ανοιχτά όλα τα ενδεχόμενα, κλιμακώνοντας μια σειρά απο έρευνες έτσι ώστε να απαντηθεί το ερώτημα αν πρόκειται για εμπρησμό ή όχι.
ΦΩΤΟ ΑΦΟΙ ΣΙΕΧ
