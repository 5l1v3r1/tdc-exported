
# ฟิน! 'ไฮโซพก' โพสต์สตอรี่ 'อั้ม' มาป่วนถึงออฟฟิศ

Published at: **2019-11-06T06:41:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/entertainment/740251)


        ไฮโซพก และ อั้ม-พัชราภา โพสต์ไอจีสตอรี่แซวกันไปมา ด้านชาวเน็ตฟินมากๆ
      
แม้จะแย้มผ่านสื่อล่าสุดว่าสถานะกับ ไฮโซพก-ประธานวงศ์ ดีขึ้นนิดนึงแค่นั้นเอง แต่ดูท่าออร่าความสุขของสาว อั้ม-พัชราภา ไชยเชื้อ จะมีมากล้น และทำให้แฟนๆฟินกันเหลือเกินเพราะจนถึงตอนนี้แฟนคลับก็ยังคงลุ้นให้ทั้งสองคนรีเทิร์นกลับมาใช้สถานะคนรู้ใจอีกครั้ง
ล่าสุดหนุ่มพกก็เติมเชื้อฟินๆให้แฟนๆได้เห็นกันอีกครั้ง หลังโพสต์อินสตาแกรมสตอรี่เป็นรูปสาวอั้มนั่งอยู่ที่โต๊ะทำงาน พร้อมข้อความว่า “Boss lady มาป่วนถึงออฟฟิศ” จากนั้นสาวอั้มก็ได้เอารูปหนุ่มพกมาลงสตอรี่ของตัวเองเช่นกัน โดยใส่คำว่า “มาช่วยทำงานนะ” ก่อนที่หนุ่มจะเอารูปดังกล่าวไปโพสต์สตอรี่อีกรอบแล้วใส่คำว่า “มานั่งเล่นเกมส์”อีกด้วย
