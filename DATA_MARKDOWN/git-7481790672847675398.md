
# 「通話は録音されます」…アポ電撃退する電話機

Published at: **2019-11-05T04:42:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191105-OYT1T50121/)

　特殊詐欺グループなどが高齢者から資産状況などを聞き出す「アポ電（アポイントメント電話）」の急増を受け、シャープが警視庁や大阪府警と協力し、対策機能を高めた新たな固定電話機を開発した。
　「電話が鳴るとつい出てしまい、被害に遭う高齢者が多い」という警察の助言を基に、番号を登録していない相手からの電話は着信音が鳴る前に「この通話は防犯のため録音されます」と警告。不審な電話に出る機会を減らす。
　それでも相手が切らない場合は短時間の着信音が鳴った後、受話器を取らなくても相手に名乗るように促すメッセージが自動的に流れる。相手の声も本体から聞こえるため、電話に出る判断材料になる。
　会話途中でも、ボタンを押せば通話を切り、相手の電話番号を自動で着信拒否登録する。こうした機能は初期設定済みで、高齢者でも購入してすぐに使える。１１月１４日に発売予定で、想定価格は税抜き１万６０００円前後（子機１台付き）。
