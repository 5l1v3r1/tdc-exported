
# Вышел главный трейлер сериала «Ведьмак»

Published at: **2019-11-01T09:29:42+00:00**

Author: ****

Original: [Wonderzine](https://www.wonderzine.com/wonderzine/entertainment/entertainment-news/246833-the-witcher-trailer)

Netflix поделился трейлером долгожданного сериала «Ведьмак», снятого по мотивам фэнтезийной саги польского писателя Анджея Сапковского. В центре сюжета — мутант Геральт из Ривии, который скитается по миру и по заказу людей убивает различных чудовищ.
Главную роль в сериале исполнил Генри Кавилл, известный по фильмам «Агенты А.Н.К.Л.», «Человек из стали» и другим. Премьера состоится на стриминговом сервисе 20 декабря.
