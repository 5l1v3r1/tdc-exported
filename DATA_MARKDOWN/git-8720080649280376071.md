
# Дептранс объяснил, почему в Самаре не снизят оплату проезда по карте

Published at: **2019-11-06T15:10:59+00:00**

Author: **Максим Фёдоров**

Original: [Другой город](http://drugoigorod.ru/pay-stability/)

Департамент транспорта разъяснил информацию о планах снизить безналичную оплату проезда. На самом деле оплата за проезд по единой транспортной карте не станет дешевле.
Представители дептранса назвали информацию о снижении установленной стоимости проезда некорректной. По их словам, планы другие:
«Рассматривается вариант, при котором в будущем при изменении стоимости проезда безналичная оплата проезда может быть установлена на уровне ниже стоимости проезда за наличный расчет».
Напомним, жители Самары попросили областной минтранс договориться с коммерческими перевозчиками по поводу льгот. Чиновники отказались.
