
# ทองคำแท่ง-รูปพรรณเช้านี้ปรับราคาลง 50 บาท

Published at: **2019-11-04T03:29:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/economic/739882)


        ทองคำในประเทศปรับตัวลงตามต่างประเทศหลังตัวเลขจ้างงานอกภาคเกษตรสหรัฐฯเดือนต.ค.ปรับขึ้น ส่งผลทองแท่งขายออกบาทละ 21,600  บาท  ทองรูปพรรณขายบาทละ 22,100 บาท 
      
รายงานข่าวสมาคมค้าทองคำแจ้งว่า  ทองคำในประเทศปรับลง 50 บาท  หลังจากตัวเลขการจ้างงานนอกภาคเกษตรในเดือนต.ค.เพิ่มขึ้น 128,000 ตำแหน่งสูงกว่าที่นักวิเคราะห์คาดการณ์ที่ระดับ 89,000 ตำแหน่ง ส่วนอัตราการว่างงานเพิ่มขึ้นสู่ระดับ 3.6% และตัวเลขค่าจ้างรายชั่วโมงโดยเฉลี่ยของแรงงานเพิ่มขึ้น 0.2% ส่งผลให้ทองแท่งรับซื้อบาทละ 21,500 บาท ขายออกบาทละ 21,600  บาท  ทองรูปพรรณรับซื้อบาทละ 21,117.88 บาท และขายออกบาทละ 22,100 บาท ราคาต่างประเทศอยู่ที่  1,513.00  ดอลลาร์สหรัฐต่อออนซ์  จากเมื่อวันศุกร์ที่ผ่านมาอยู่ที่  1,513.50 ดอลลาร์สหรัฐต่อออนซ์ ค่าเงินบาทอยู่ที่  30.18 บาทต่อดอลลาร์สหรัฐ จากวันศุกร์อยู่ที่  30.17 บาทต่อดอลลาร์สหรัฐ
