
# СМИ назвали самого жуткого монстра из фильмов ужасов

Published at: **2019-11-02T00:13:00+00:00**

Author: ****

Original: [РИА Новости](https://ria.ru/20191102/1560503741.html)

МОСКВА, 2 ноя – РИА Новости. Журналисты из Independent составили рейтинг 20 самых жутких монстров из фильмов ужасов, в который вошли вампиры, пришельцы, мутанты и другие существа.Список возглавил Бабадук — персонаж из одноименного хоррора 2014 года, режиссера Дженнифер Кент, о сверхъестественном существе из детской книжки. На втором месте оказался герой легендарного немого фильма 1922 года "Носферату. Симфония ужаса". Лысый вурдалак с двумя торчащими зубами в исполнении Макса Шрека даже спустя 97 лет продолжает вселять ужас в зрителей. Тройку лидеров замкнул вампир Номак из фильма Гильермо дель Торо "Блэйд 2".В рейтинг также попали монстры из фильмов: "Гремлины" (1984), "Кинг Конг" (1933), "Лабиринт фавна" (2006) и даже Рэндалл из "Корпорации монстров" (2001).
На последнее место поставили инопланетян из фантастической франшизы Ридли Скотта "Чужой" (1979).
