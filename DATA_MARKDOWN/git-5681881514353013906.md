
# В Одессе повесилась на дереве 13-летняя девочка

Published at: **2019-11-01T10:01:00+00:00**

Author: ****

Original: [ТАЙМЕР](https://timer-odessa.net/news/v_odesse_povesilas_na_dereve_13_letnyaya_devochka_962.html)


        Одесские правоохранители расследуют обстоятельства смерти 13-летней девочки, которая повесилась на дереве.
      
Труп обнаружили около 23:00 в четверг, 31 октября, в Малиновском районе города.
Установлено, что девочка — 13-летняя местная жительница. По предварительным данным, признаков насильственной смерти на её теле не обнаружено.
Сведения по данному факту внесены в Единый реестр досудебных расследований по ст. 115 Уголовного кодекса Украины (умышленное убийство с пометкой «самоубийство»). Тело погибшей направлено на экспертизу, которая установит окончательную причину смерти. Полицейские выясняют обстоятельства трагедии, сообщили в отделе коммуникации.
