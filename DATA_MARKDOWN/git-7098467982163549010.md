
# Бганба о земельном кадастре Абхазии: осталось только программное обеспечение

Published at: **2019-11-04T14:44:00+00:00**

Author: ****

Original: [Sputnik Абзахия](https://sputnik-abkhazia.ru/Abkhazia/20191104/1028751704/Bganba-o-zemelnom-kadastre-Abkhazii-ostalos-tolko-programmnoe-obespechenie.html)


        Работы по созданию земельного кадастра в Республике Абхазия начались в 2012 году.
      
СУХУМ, 4 ноя – Sputnik, Бадри Есиава. Для создания земельного кадастра Абхазии осталось запустить программное обеспечение, сказал премьер-министр республики Валерий Бганба на брифинге в Сухуме.
Он добавил, что Абхазия рассматривает американскую версию программного обеспечения земельного кадастра страны в качестве одного из подходящих вариантов, но российская сторона с этим не согласна. 
Работы по созданию земельного кадастра в Абхазии начались в 2012 году. В 2012-2013 году подготавливалась государственная геодезическая сеть. В 2014 году на основании решения Межправительственной комиссии между Россией и Абхазией были выделены дополнительные средства на аэрофотосъемку.
На сегодняшний день в республике действует Земельный кодекс советского образца.
