
# Расстрелявшие мужчину из травмата преступники украли у него 5 млн

Published at: **2019-11-01T16:36:57+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-01/rasstrelyavshie-muzhchinu-iz-travmata-prestupniki-ukrali-u-nego-5-mln)

Несколько преступников, расстрелявших из травматического оружия мужчину, отобрали у него 5 миллионов рублей.
Инцидент произошел на улице Верхние Поля в Москве. 
По данным РЕН ТВ, один из нападавших задержан. В столичном главке МВД заявили, что поиск остальных нападавших продолжается. Введен план "Перехват".
