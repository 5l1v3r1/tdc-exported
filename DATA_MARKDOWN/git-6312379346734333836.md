
# В Приморье женщина потеряла сознание после подрыва петарды детьми

Published at: **2019-11-01T15:38:46+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-01/v-primore-zhenshchina-poteryala-soznanie-posle-podryva-petardy-detmi)

Во Владивостоке вечером в пятницу 1 ноября пожилая женщина потеряла сознание во дворе дома, когда рядом с ней начали взрываться петарды.
По данным РЕН ТВ, в полицию с заявлением обратилась девушка, ставшая свидетельницей произошедшего. По ее словам, с петардами баловались дети 12-13 лет. При этом они специально старались напугать прохожих.
