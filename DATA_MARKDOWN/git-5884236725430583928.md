
# Россияне озвучили, какие профессии считают самыми интересными

Published at: **2019-11-05T07:45:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/739852-rossiyane-ozvuchili-kakie-professii-schitayut-samymi-interesnymi)


        Лидером среди интересных профессий оказалось дизайнерское дело. Опрос провели аналитики сервиса «Работа.ру».
      
Россияне рассказали, какие профессии, по их мнению, являются самыми интересными. Об этом сообщает RT.По результатам опроса сервиса «Работа.ру» лидером среди интересных профессий оказались дизайнерская должность, за которую проголосовали 31,8 % опрошенных. На втором месте оказалась работа журналиста (28,9 %). Замкнула тройку лидеров профессия артиста (23,8 %).Отметим, что почти 72 % респондентов считают, что у них интересная работа.Ранее НЕВСКИЕ НОВОСТИ писали, Microsoft рассказали о результатах эксперимента по введению четырехдневки.
