
# 美眾院表決過關 正式啟動川普彈劾調查程序

Published at: **2019-11-01T08:46:00+00:00**

Author: **https://www.facebook.com/cnanewstaiwan, 中央通訊社**

Original: [中央社 CNA](https://www.cna.com.tw/news/firstnews/201911010007.aspx)

（中央社華盛頓31日綜合外電報導）嚴重分歧的美國聯邦眾議院今天針對推動總統川普的彈劾調查程序進行首次表決，最終以232對196票通過一項決議案，針對川普正式展開嶄新且公開階段的彈劾調查。
眾議院議長裴洛西（Nancy Pelosi）說，眾院往前跨一步，目的是為了建立公開聽證會的程序，確保彈劾程序的正當性與透明度，讓民眾自己看到事實。
這項表決展現民主黨的團結一致，而共和黨則無人跑票支持此一決議案。
裴洛西表示：「這是令人悲傷的一天。擔任國會議員的目的不是為了彈劾總統。」
彈劾調查的焦點放在川普要求烏克蘭總統澤倫斯基（Volodymr Zelenskiy）調查美國前副總統拜登（Joe Biden）及他的兒子韓特（Hunter）。（譯者：徐崇哲）1081101
