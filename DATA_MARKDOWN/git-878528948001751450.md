
# ヤクルト・村上が秋季キャンプから離脱「悔しい気持ち」下半身のコンディション不良で

Published at: **2019-11-02T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/02/0012842463.shtml)

　ヤクルト・村上宗隆内野手が２日、下半身のコンディション不良のため、愛媛・松山で始まった秋季キャンプから離脱した。
　１０月中に病院で診察を受けていたが、チームとともに前日に現地入り。参加の可能性を探ったものの、この日の朝の状態を踏まえて参加を見送った。
　「悔しい気持ち。キャンプでレベルアップを、と思っていた」と村上。球場内でウエートトレーニングなどを行った後、神妙な表情で帰京の途についた。
