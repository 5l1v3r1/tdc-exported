
# 外資瘋台股 持股債約當外匯存底比重創新高

Published at: **2019-11-05T21:02:00+00:00**

Author: **https://www.facebook.com/cnanewstaiwan, 中央通訊社**

Original: [中央社 CNA](https://www.cna.com.tw/news/gpho/201911050001.aspx)

央行5日發布10月底外匯存底續創歷史新高，10月因熱錢湧入，台股大漲4.9%，外資持有新台幣資產較前一月增357億美元，約當外匯存底占比衝上88%，占比來到19個月以來新高。今年10月底，台灣外匯存底增加至4724.76億美元。中央社製圖 108年11月5日
