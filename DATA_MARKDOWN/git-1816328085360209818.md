
# ஆர்.எஸ்.எஸ். தலைவர் மோகன் பகவத்துடன் பட்னாவிஸ் ஆலோசனை

Published at: **2019-11-05T00:00:00+00:00**

Author: ****

Original: [Dailythnathi.com](https://www.dailythanthi.com/News/India/2019/11/05231154/Devendra-Fadnavis-meets-RSS-chief-Mohan-Bhagwat-in.vpf)

நாக்பூர்,மராட்டிய சட்டசபை தேர்தலில் ஆட்சி அமைப்பதற்கு போதுமான இடங்களில் வெற்றி பெற்ற பா.ஜனதா-சிவசேனா கூட்டணியில் முதல்-மந்திரி பதவியை பகிர்ந்து கொள்வதில் ஏற்பட்டுள்ள மோதலால் புதிய ஆட்சி அமைவதில் இழுபறி நீடித்து வருகிறது.
சிவசேனாவை சேர்ந்தவர் முதல்-மந்திரி ஆவதற்கு 170 எம்.எல்.ஏ.க்களின் ஆதரவு உள்ளதாக அக்கட்சி பா.ஜனதாவை மிரட்டி வருகிறது. இதனால் மராட்டிய மாநிலத்தில் அரசியல் குழப்பம் நிலவி வருகிறது. இந்நிலையில், ஆர்.எஸ்.எஸ். தலைவர் மோகன் பகவத்தை மராட்டிய முதல்-மந்திரி தேவேந்திர பட்னாவிஸ் இன்று சந்தித்து பேசினார். அப்போது, மாநில அரசியல் சூழ்நிலை குறித்து இருவரும் ஆலோசனை நடத்தியதாக தகவல் வெளியாகி உள்ளது.
