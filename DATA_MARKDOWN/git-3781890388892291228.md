
# คาราบาวแตก!'เทียรี่'เดือด ซัดผมไม่ทำตาม'แอ๊ด'สั่งแล้ว

Published at: **2019-11-05T03:01:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/entertainment/740035)


        น้ำตานองอาบสองแก้ม....“เทียรี เมฆวัฒนา”โพสต์สะเทือนวงคาราบาว ตั้งแต่วันนี้ไปผมไม่ทำตามคำสั่งของพี่แอ๊ดอีกต่อไป ทำเอาแฟนคลับสงสัยเกิดอะไรขึ้น
      
กลายเป็นประเด็นขึ้นมาในโลกออนไลน์เลยทีเดียว หลัง เทียรี่ เมฆวัฒนา หนึ่งในสมาชิกของวงคาราบาว ได้ออกมาโพสต์ข้อความผ่านทางเฟซบุ๊กส่วนตัวว่า “ตั้งแต่วันนี้ไป ผมไม่ทำตามคำสั่งของพี่แอ้ดอีกต่อไปนะคับ” จนทำให้แฟนๆ หลายๆ คนเกิดความสงสัยว่าเกิดอะไรขึ้น มีปัญหาภายในวงหรือเปล่า
นอกจากนี้เทียรี่ ยังโพสต์ข้อความต่ออีกว่า “ถึงเวลา จะพูดความจริงนะคับ” ซึ่งก็มีแฟนคลับเข้ามาคอมเมนท์ให้กำลังใจและบอกว่าจะรอความจริงจากปากของเทียรี่กับแอ๊ด คาราบาว
