
# โมดียกอาเซียนเป็นหัวใจ นโยบายมองตะวันออก

Published at: **2019-11-03T08:23:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/foreign/739804)


        นายกรัฐมนตรีนเรนทรา โมดี กล่าวว่าอาเซี่ยนมีความสำคัญอย่างมากต่ออินเดียในฐานะเป็นภูมิภาคซึ่งเป็น "แกนกลาง" ของแผนยุทธศาสตร์ "มองตะวันออก" และหวังการยกระดับความร่วมมือในทุกด้าน
      
สำนักข่าวต่างประเทศรายงานจากกรุงเทพมหานคร เมื่อวันที่ 3 พ.ย.ว่านายกรัฐมนตรีนเรนทรา โมดี ผู้นำอินเดีย กล่าวเมื่อวันอาทิตย์ ต่อที่ประชุมอาเซียน-อินเดีย ครั้งที่ 16 ซึ่งเป็นหนึ่งในการหารือกลุ่มย่อยที่สำคัญของการประชุมสุดยอดสมาคมประชาชาติแห่งเอเชียตะวันออกเฉียงใต้ ( อาเซียน ) ครั้งที่ 35 ซึ่งไทยเป็นประธานในปีนี้ ว่าตอนนี้อาเซียนเป็นภูมิภาคที่กำลังมีบทบาทมากขึ้นอย่างต่อเนื่องต่อภูมิภาคอินโด-แปซิฟิก ดังนั้นการยกระดับความร่วมมือระหว่างอาเซียนกับอินเดียในด้านความมั่นคง ทั้งทางบก ทางทะเล และทางอากาศจะช่วยกระตุ้นเศรษฐกิจและการขยายตลาดของทั้งสองฝ่ายได้เช่นกัน โดยโมดีกล่าวว่าจะเป็นการเชื่อมโยงไปถึงการพัฒนาด้านเกษตรกรรม วิศวกรรม เทคโนโลยีดิจิทัล และการวิจัยทางวิทยาศาสตร์
