
# Анджеліну Джолі евакуювали зі зйомок через бомбу

Published at: **2019-11-05T13:44:34+00:00**

Author: ****

Original: [ФАКТИ](https://fakty.com.ua/ua/showbiz/20191105-andzhelinu-dzholi-evakuyuvaly-zi-zjomok-cherez-bombu/)

Голлівудську актрису Анджеліну Джолі евакуювали зі зйомок нового фільму студії Marvel через знайдену бомбу.
Анджеліну Джолі евакуювали зі зйомок нового фільму студії Marvel Вічні, які проходять в пустелі в Національному парку Каралехо на острові Фуертевентура. Причиною стала бомба, яку знайшли на території майданчика.

Зйомки проходять на місці німецької військової бази часів Другої світової війни.
Також серед евакуйованих був актор Річард Медден.
Вибухотехніки успішно знешкодили бомбу.
Раніше Факти ICTV зібрали для топ-5 фільмів, де Джолі приголомшливо зіграла головну роль.
