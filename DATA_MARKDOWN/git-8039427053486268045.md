
# పదవుల కోసమేనా ఈ మౌనం?

Published at: **2019-11-03T03:54:30+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=944995)

Ph: 88865 47666ప్రవేశం ఉచితం ADVT
విజయవాడ, గుంటూరు, నవంబరు 2(ఆంధ్రజ్యోతి): పత్రికా స్వేచ్ఛకు విఘాతం కలిగేలా జీవోలు విడుదల చేస్తున్నా పదవుల కోసం అమర్‌, రామచంద్రమూర్తి మౌనం వహించడం సరికాదని టీడీపీ నేత వర్ల రామయ్య విమర్శించారు. ‘గతంలో రాజశేఖర్‌ రెడ్డి సీఎంగా 938 జీవో ఇస్తే అది పత్రాకా స్వేచ్ఛకు ఉరితాడు అని ఉద్యమానికి నాయకత్వం వహించి జీవో ఉపసంహరింప చేసిన అమర్‌, రామచంద్రమూర్తి నేడు మౌనం వహించడానికి కారణం సీఎం విసిరిన పదవులేనా?’ అని ఆయన నిలదీశారు. బాధ్యతగల మంత్రి పదువుల్లో ఉన్న వ్యక్తులు మీడియాను సామాజిక వర్గాలుగా విభజించి, కులాలను ఆపాదించడం దారుణమని మాజీ మంత్రి ఆలపాటి రాజేంద్రప్రసాద్‌ గుంటూరులో అన్నారు.
