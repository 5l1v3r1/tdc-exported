
# На юге столицы задержан подозреваемый в хищении оружия

Published at: **2019-11-06T15:11:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/yuao/na-yuge-stolitsy-zaderzhan-podozrevaemyy-v-khishchenii-oruzhiya-123448/)

В ОМВД России по району Царицыно обратился житель столицы. По словам заявителя в одном из ресторанов быстрого питания на Кантимировской улице неизвестный тайно похитил из его сумки, оставленной без присмотра, травматический пистолет и магазин с 10 патронами, после чего скрылся.В результате проведения оперативно-розыскных мероприятий оперуполномоченные уголовного розыска ОМВД России по району Царицыно совместно с оперативниками УВД по ЮАО задержали подозреваемого. Им оказался 26-летний приезжий. Похищенное изъято.По данному факту следствием территориального ОМВД возбуждено уголовное дело по признакам преступления, предусмотренного статьей 226 УК РФ «Хищение либо вымогательство оружия, боеприпасов, взрывчатых веществ и взрывных устройств». Подозреваемый задержан в порядке статьи 91 УПК РФ.Пресс-служба УВД по ЮАО(495) 734-61-22
