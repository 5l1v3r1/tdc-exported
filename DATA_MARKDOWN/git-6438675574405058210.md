
# Украина закрыла один из трех пунктов пропуска на границе с Крымом

Published at: **2019-11-07T08:41:00+00:00**

Author: ****

Original: [РИА Новости](https://ria.ru/20191107/1560665210.html)

СИМФЕРОПОЛЬ, 7 ноя — РИА Новости. Украинская сторона закрыла автомобильный пункт пропуска "Каланчак", сообщила пресс-служба пограничного управления ФСБ по Крыму и Севастополю.Ранее СМИ писали, что там начинается ремонт.С российской стороны напротив "Каланчака" расположен пункт пропуска "Армянск", который теперь временно не работает.Крымские пограничники советуют путешественникам учитывать сложившиеся обстоятельства и пользоваться оставшимися двумя КП "Перекоп" и "Джанкой". Первый предназначен только для граждан России и Украины, остальные могут пересечь границу только в Джанкое.С начала 2019-го через три пункта пропуска прошли более 2,4 миллиона человек, что соответствует уровню прошлого года за тот же период.
