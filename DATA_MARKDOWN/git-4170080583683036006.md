
# Житель Донского украл из торгового центра в Туле пару обуви

Published at: **2019-11-04T15:06:00+00:00**

Author: ****

Original: [Тульская Служба Новостей](https://www.tsn24.ru/zhitel-donskogo-ukral-iz-torgovogo-czentra-v-tule-paru-obuvi.html)

Полицейские задержали подозреваемого в краже из торгового центра на улице Путейской Тулы.
По предварительным данным УМВД России по региону, заместитель управляющего одного из магазинов обралась в полицию с заявлением о том, что из торгового зала похищены кроссовки. На место прибыла следственно-оперативная группа. Полицейские установили личность похитителя.
По данным ведомства, им стал ранее судимый за кражи 20-летний города Донской. Он был задержан и доставлен в дежурную часть отдела полиции для разбирательства.
Решается вопрос о возбуждении уголовного дела. Похищенная обувь изъята.
