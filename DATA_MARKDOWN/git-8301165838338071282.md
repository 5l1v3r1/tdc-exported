
# No Man’s Sky: Update 2.16 veröffentlicht

Published at: **2019-11-07T16:01:06+00:00**

Author: **Thomas Ruch**

Original: [play3.de](https://www.play3.de/2019/11/07/no-mans-sky-update-2-16-veroeffentlicht/)


        "No Man’s Sky" hat von Hello Games das neue Update 2.16 spendiert bekommen, mit dem einige Fehler behoben und die allgemeine Stabilität der Weltraumsimulation verbessert wurde. Der Changelog verrät die Details.
      
Die Entwickler von Hello Games haben heute das frische Update 2.16 für die Weltraum-Simulation „No Man’s Sky“ veröffentlicht. Die neuste Aktualisierung behebt diverse Fehler und behebt verbessert somit die Stabilität auf PS4, Xbox One und PC.
Mit dem Changelog liefern die Entwickler einen Überblick über die Verbesserungen. Es wird dazu betont, dass Hello Games auf das Feedback der Spieler hört und entsprechende Verbesserungen liefert. Auf der offiziellen Seite wurde der Changelog geteilt.
