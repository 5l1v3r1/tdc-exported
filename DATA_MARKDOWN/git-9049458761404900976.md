
# ‘సచివాలయ’ అభ్యర్థుల ఆందోళన

Published at: **2019-11-05T09:27:42+00:00**

Author: ****

Original: [Andhrajyothy](http://edu.andhrajyothy.com/EduNewsArticle.aspx?SID=947007)

గ్రామసచివాలయ పరీక్షల్లో ఉత్తీర్ణత సాధించి, సర్టిఫికెట్ల పరిశీలన తర్వాత తమను అనర్హులంటున్నారని శానిటేషన్‌, ఎన్విరాన్‌మెంట్‌ సెక్రటరీ గ్రేడ్‌-బి అభ్యర్థులు సోమవారం గుంటూరులో ఆందోళన చేశారు. తొలుత ఏదైనా సైన్స్‌ డిగ్రీ ఉండాల న్నారని, ఇప్పుడు బీ ఫార్మసీ సైన్స్‌డిగ్రీఅభ్యర్థులు అనర్హులంటున్నారని వాపోయారు. తమను ఉద్యోగాల్లోకి తీసుకోవాలని డిమాండ్‌ చేశారు. -గుంటూరు తూర్పు
