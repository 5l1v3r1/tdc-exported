
# Работы по устранению дефекта трубы в Колпино завершили быстрее нормативного срока

Published at: **2019-11-05T00:02:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/739821-raboty-po-ustraneniyu-defekta-truby-v-kolpino-zavershili-bystree-normativnogo-sroka)

Восстановительные бригады ТЭКа справились с работой за четыре часа вместо положенных восьми. Ранее на улице Вавилова произошел прорыв трубы.Восстановительные бригады ТЭКа справились с коммунальной аварией на улице Вавилова в Колпина в два раза быстрее нормативного срока, устранив дефект трубы за четыре часа вместо положенных восьми. Таким образом 4 ноября  в 18:55 по московскому времени работы были полностью завершены, а дома запитаны по нормальной схеме.
Ранее НЕВСКИЕ НОВОСТИ писали, что прорыв трубы в Колпино на улице Вавилова локализовали через полчаса. Коммунальная авария произошла примерно в 14:10 по московскому времени, а в 14:45 прорыв был локализован.#колпино #прорыв #труба
