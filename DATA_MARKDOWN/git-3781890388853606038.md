
# โพสต์แรกหลังเลิกแฟนฝรั่ง 'เมญ่า' อวดหุ่นแซ่บสะท้านใจ

Published at: **2019-11-07T11:52:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/entertainment/740500)


        เมญ่า-นนธวรรณ รีเทิร์นความแซ่บ ลงรูปแรกหลังแยกทางสามีต่างชาติด้วยชุดว่ายน้ำสุดเซ็กซี่ ด้านชาวเน็ตแห่ให้กำลังใจมากมาย
      
ยังคงเป็นประเด็นที่ชาวเน็ตสนใจอย่างต่อเนื่อง หลังอดีตนักแสดงและนางงามสาว เมญ่า-นนธวรรณ บรามาส มิสไทยแลนด์เวิล์ดปี 2557  ได้ประกาศเลิกรากับสามีนักธุรกิจชาวสวิตเซอร์แลนด์ "ฟิลลิป บรามาส" เมื่อคืนที่ผ่านมา ท่ามกลางความตกใจของแฟนๆ ตามที่เสนอข่าวไปแล้วนั้น
ความคืบหน้าล่าสุดในอินสตาแกรมของสาวเมญ่าก็ได้มีการโพสต์สตอรี่เป็นรูปที่ตนเองใส่ชุดว่ายน้ำสุดเซ็กซี่นั่งเล่นกับลูกริมทะเล ขณะที่รูปปกติสาวเมญ่าก็ได้ลงรูปของตัวเองในชุดว่ายน้ำเหมือนเดิม แต่ยืนขึ้น โพสต์ท่าสวยๆ อวดความเซ็กซี่แบบเต็มสตรีม ซึ่งพอสาวเมญ่าโพสต์รูปลงไป ก็มีชาวเน็ตเข้ามาคอมเมนต์ให้กำลังใจและชมความสวยแซ่บของเธอจำนวนมาก
