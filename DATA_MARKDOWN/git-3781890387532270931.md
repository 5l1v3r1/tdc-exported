
# 'อ.เจษฏา'เชื่อ'คาราบาว'วงแตก เหตุ'อุดมการณ์เปลี่ยน'!

Published at: **2019-11-05T03:30:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/entertainment/740043)


        "อ.เจษฏา" เชื่อคาราบาว วงแตกแน่ เหตุอุดมการณ์เปลี่ยน ทำ "เทียรี่" ทนไม่ไหว ต้องโพสต์ระบายความอัดอั้นผ่านเฟซบุ๊ก
      
จากกรณี"ที่นายเทียรี่ เมฆวัฒนา"นักร้องและนักดนตรีชื่อดัง “วงคาราบาว”ได้โพสต์แตกหักกับ”แอ๊ด คาราบาว”โดยระบุว่า ตั้งแต่วันนี้ไป ผมไม่ทำตามคำสั่งของพี่แอ้ดอีกต่อไปนะคับ.ตามที่ได้เสนอข่าวไปแล้วนั้น เมื่อวันที่ 5 พ.ย. เพจเฟซบุ๊กของ ผศ.ดร.เจษฎา เด่นดวงบริพันธ์ อาจารย์ประจำภาควิชาชีววิทยา คณะวิทยาศาสตร์ จุฬาลงกรณ์มหาวิทยาลัย ได้โพสต์ข้อความโดยตั้งข้อสังเกตถึงกรณีดังกล่าวว่า เกิดอะไรขึ้นกับวงคาราบาวเนี่ย พี่เทียรี่โกรธอะไรกันกับพี่แอ๊ด ... มาต้มมาม่า ซุ่มดูดราม่า กันดีกว่าปล. เท่าที่ไปไล่ดูโพสต์ย้อนหลังในเพจพี่เทียรี่ มีแต่โพสต์เรื่องแบนสารเคมีทางการเกษตรทั้ง 3 สารเลย ไม่รู้ว่าเกี่ยวข้องกันกับประเด็นที่วงคาราบาว ก็ไปเล่นคอนเสิร์ตให้บริษัทเคมีทางการเกษตรเหมือนกัน หรือเปล่าอ่านข่าวที่เกี่ยวข้อง'แอ๊ด'ซัดขายครีมปลิ้นปล้อน 'อ.เจษ'จวกกลับตรรกะวิบัติ!...คาราบาวแตก!'เทียรี่'เดือด ซัดผมไม่ทำตาม'แอ๊ด'สั่งแล้ว..
