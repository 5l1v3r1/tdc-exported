
# 卒業電撃発表の矢作萌夏 涙の２代目女王 ＡＫＢグループ歌唱力ナンバーワン決定戦

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/01/0012838353.shtml)

　「第２回ＡＫＢ４８グループ歌唱力Ｎｏ．１決定戦」決勝大会が３１日、都内で開催され、来年２月での卒業を発表しているＡＫＢ・矢作萌夏（１７）が優勝した。２代目王者として名前を呼ばれると、歓喜の号泣。前回２位から頂点に登り詰め、「（涙が）止まらないです。最後にファンの方に、自慢の推しメンと思ってもらえる結果を残してうれしいです。（優勝報告したいのは）いつも笑顔で守ってくれるファンの皆さんです」と“有終の美”に声を震わせた。
　１３５人が参加した予選を勝ち抜いた１９人がハイレベルな歌声を響かせ、ファイナルステージは矢作、ＡＫＢ／ＳＴＵ・岡田奈々（２１）、ＳＫＥ・高柳明音（２７）ら８人で競われた。
