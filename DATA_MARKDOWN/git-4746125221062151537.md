
# HIKAKIN、YouTuber人生に感慨「12年やってきてよかった」 話題の自宅からは東京タワー一望

Published at: **2019-11-01T21:44:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147769/full/)

　人気YouTuberのHIKAKIN（30）が1日、東京タワーで行われた『東京タワー ウィンターファンタジー 〜オレンジ・イルミネーション 2019〜』の点灯式に出席した。

        【写真】その他の写真を見る
      
　東京タワーをイメージした赤い衣装で登場したHIKAKIN。「上京して、20歳ぐらいの時に母親と一緒に一番上まで昇ったのを思い出します」と笑顔。イルミネーションが点灯すると「こんな日が来るなんて思ってなかったです。ありがとうございます！」と大興奮。「18歳で東京に来たので、12年頑張ってよかったなと感じております」と感慨深げに語った。 　これまでのYouTuberとしての活動についても触れ、「（ここまで来るのは）長かったです。最初の4年間ぐらいは日の目を見なかったので…。今は自分の夢が間違いなくかなっています」と振り返り、「長く続けることが大事だと思うので、少しでも長くエンターテインメントの世界にいられたらなと思います」と今後の目標も明かした。 　また、HIKAKINの自宅から東京タワーが一望できるという話題に「東京タワーを見てると、大都会の東京で『俺、戦ってんな』という気分になる」とうれしそうに語った。 　同イルミネーションは、“都会のキャンドル”をイメージした高さ11.1メートルの東京タワー型オブジェ「キャンドルタワー」に約1000個のフルカラーLED電球が装飾されており、周辺の植栽エリア「ファンタジーガーデン」と合わせ、合計約4万6000個のLED電球とオーナメントが輝いている。きょう1日から2020年1月5日まで開催。
