
# В Москве застрелили начальника Центра «Э» по Ингушетии

Published at: **2019-11-02T20:27:00+00:00**

Author: ****

Original: [RuNews24](https://runews24.ru/incidents/02/11/2019/47df7731623e13d6953387105a8dd7cc)

На юге Москвы неизвестный расстрелял двух мужчин. Один из погибших начальник центра «Э» по Ингушетии Ибрагим Эльджаркиев. 
По данным РЕН ТВ, инцидент произошел у подъезда одного из жилых домов по улице Академика Анохика. Между двумя группами мужчин произошел конфликт. Злоумышленники несколько раз выстрелили в своих противников и скрылись с места происшествия.
Очевидцы сообщили «Пятому каналу», что на месте трагедии лежат двое мужчин без признаков жизни. Они добавили, что слышали семь выстрелов и видели мужчину, на котором была надета красная куртка.
