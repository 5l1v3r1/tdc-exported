
# Уряд придбає понад 800 поліцейських авто для сільської місцевості

Published at: **2019-11-07T09:20:00+00:00**

Author: ****

Original: [Економічна правда](https://www.epravda.com.ua/news/2019/11/7/653423/)

Кабінет міністрів виділив кошти на придбання 822 поліцейських авто для сільської місцевості.
Про це йдеться в повідомленні на урядовому порталі.
"Кабінет міністрів на своєму засіданні виділив кошти на покупку 822 поліцейських авто спеціально для маленьких міст та сільської місцевості", - зазначено в повідомленні.
Зазначається, що реалізація рішення дозволить вже в 2020 році підвищити швидкість реагування поліції на виклики в малих містах і селах.
Згідно з повідомленням, в майбутньому році проект охопить майже 30 тис маленьких міст України, де проживає близько 28 млн громадян і передбачає залучення ще 3,2 тис офіцерів громад для роботи в об'єднаних територіальних громадах.
