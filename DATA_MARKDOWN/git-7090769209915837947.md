
# Прогноз погоды на 4 ноября

Published at: **2019-11-03T06:10:32+00:00**

Author: ****

Original: [Амурская Служба Новостей: новости с характером](https://asn24.ru/news/society/69748/)

Ночью 4 ноября по северу переменная облачность, местами слабый снег, по югу ясно. Ночью и в утренние часы местами туман. Ветер слабый, переменных направлений. Температура -6... -11°, местами -15... -20°.Днем по северу облачно, по югу переменная облачность местами осадки (снег, мокрый снег). Ветер по северу северо-западный, по югу юго-восточный 3-8 м/с, с усилением до 14 м/с. Температура от -1° до +4°, местами -5... -10°.В Благовещенске ночью ясно, ветер слабый, переменных направлений. Температура -6... -8°. Днем переменная облачность, без осадков. Ветер юго-восточный 2-7 м/с. Температура +2... +4°.
