
# 絶滅危惧種などカメ６４匹、動物園から盗難か

Published at: **2019-11-07T00:14:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191107-OYT1T50118/)

　沖縄県沖縄市の動物園「沖縄こどもの国」で、国天然記念物に指定されているカメ６４匹が盗まれた可能性があることがわかった。同園から報告を受けた県が６日、明らかにした。環境省のレッドリストで絶滅危惧種に指定されているリュウキュウヤマガメ１５匹と、セマルハコガメ４９匹の行方が分からなくなっている。同県文化財課などによると、１０月２８日午後２時頃、飼育員がカメの数が少ないことに気づき、飼育場のネットの一部が外されているのを見つけた。同園はカメが盗まれた可能性があるとして、同県警沖縄署に被害届を出した。
