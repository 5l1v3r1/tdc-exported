
# Гюльчатай, покажи личико. Facebook будет проверять пользователей с помощью видео

Published at: **2019-11-06T06:14:00+00:00**

Author: ****

Original: [Фонтанка.ру](https://www.fontanka.ru/2019/11/06/022/)

Соцсеть Facebook начала тестировать новый инструмент в борьбе с фальшивыми аккаунтами. Об этом пишут «Ведомости» со ссылкой на Twitter эксперта в области мобильных приложений Джейн Маньчун Вонг.
По ее данным, теперь, чтобы верифицироваться, пользователям придется записать видеоролик с собой в главной роли. Это будет своего рода видеоселфи, где лицо нужно будет показать со всех сторон. Соцсеть в свою очередь обязуется в течение 30 дней «после подтверждения личности» удалить ролик.
