
# Минстрой назвал лучшие для жизни города России

Published at: **2019-11-02T00:44:00+00:00**

Author: **Юлия Саберова**

Original: [ФедералПресс](https://fedpress.ru/news/77/society/2356830)


        МОСКВА, 2 ноября, ФедералПресс. В Минстрое составили список российских городов с благоприятной городской средой.
      
Самым благоприятным для жизни городом России была названа Москва. Столица набрала 276 баллов из 360. На второй строчке – город Санкт-Петербург с показателем 238 баллов. Замкнул тройку Екатеринбург (191 балл). Также в пятерке лидеров Казань и Нижний Новгород. Оба города получили по 190 баллов. Пишет ТАСС.Ранее вице-премьер России Виталий Мутко на пресс-конференции рассказал, что «индекс качества городской среды» может стать ориентиром для регионов России.
