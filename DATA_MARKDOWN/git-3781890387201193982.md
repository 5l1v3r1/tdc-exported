
# เรื่องน่ารู้ : ตะชาย

Published at: **2019-11-01T22:15:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/agriculture/739393)


        หรือมะผด ใบประกอบมีปีกที่โคนก้านใบ ใบย่อยเป็นคู่ ขอบใบจัก สีน้ำตาลอ่อนถึงน้ำตาลเข้ม ดอกสีเขียวครีมออกเป็นช่อ เมื่อสุกมีสีออกแดง เมล็ดเกือบกลมขนาด 3-4 ซม.
      
   หรือมะผด เป็นไม้ต้นขนาดกลางสูง 8-10 เมตร กิ่งเปราะ ปลายกิ่งมีขนอ่อนนุ่ม ใบประกอบมีปีกที่โคนก้านใบ ใบย่อยเป็นคู่ ขอบใบจัก หลังใบมีขนนุ่มแน่น สีน้ำตาลอ่อนถึงน้ำตาลเข้ม ดอกสีเขียวครีมออกเป็นช่อ ช่อดอกขนาดใหญ่ 30-80 ซม. ดอกย่อยขนาด 0.2-0.4 ซม. ผลเป็นช่อใหญ่ เมื่อสุกมีสีออกแดง เมล็ดเกือบกลมขนาด 3-4 ซม.
