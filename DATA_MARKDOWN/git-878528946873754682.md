
# のん　期間があいた同じ役に緊張…「この世界の（さらにいくつもの）片隅に」

Published at: **2019-11-04T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/04/0012848360.shtml)

　女優・のん（２６）が４日、都内で行われたアニメ映画「この世界の（さらにいくつもの）片隅に」（１２月２０日公開）のワールドプレミアに出席した。
　興行収入２７億円を突破した前作「この世界の片隅に」の３年ぶりとなる新作。主人公のすず役を続投したのんは「期間をおいてから同じ役に挑むのは初めてで緊張した」とアフレコに向かうまでの心境を告白。
　不安がついて回ったが、何度も作品を見返すうちに「すずさんの皮膚感が戻ってきた」と３年前を思い出し、「強い気持ちで演じられた。気付かされることもあって再び挑めてよかった」と同作に感謝していた。
　本作は「この世界の片隅に」に、約３０分の新たな場面を追加した。
