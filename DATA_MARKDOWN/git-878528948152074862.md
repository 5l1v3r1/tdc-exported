
# 侍ジャパン　４番鈴木が先制の適時三塁打！吉田も適時打で２－０

Published at: **2019-11-07T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/07/0012858097.shtml)

　「プレミア１２・１次ラウンド、日本－台湾」（７日、台中インターコンチネンタル野球場）
　侍ジャパンが先制に成功した。
　一回、２死から近藤（日本ハム）が四球、暴投で二塁へ進塁。ここで４番・鈴木が左中間への適時三塁打。完全アウェーの敵地で、侍の４番が強烈な先制パンチを浴びせた。さらに２死三塁から吉田（オリックス）も右前適時打。２－０とした。
　ともに２連勝で１次ラウンド突破を決めている両チーム。この試合の勝敗はスーパーラウンドにも持ち越されるため、決勝に向けても大事な一戦となる。
