
# U-17 жаҳон чемпионати. Жанубий Корея чорак финалга йўл олди

Published at: **2019-11-06T06:57:16+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/u-17-jahon-chempionati-janubiy-koreya-chorak-finalga-yol-oldi)

ОФК номидан мусобақада иштирок этаётган Жанубий Корея U-17 терма жамоаси 1/8 финал баҳсини Анголага қарши ўтказиб 1:0 ҳисобида зафар қучди ва чорак финалга йўл олди. Ўйиндаги ягона гол 33-дақиқада Чои Мин Сео томонидан киритилди. 
1/8 финалнинг яна бир ўйинида Голландия Нигерияни 3:1 ҳисобида мағлуб этиб, бу жамоа ҳам кейинги босқичга чиқди.
