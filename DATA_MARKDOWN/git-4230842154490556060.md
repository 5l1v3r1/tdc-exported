
# Скалецька звільнила всіх своїх радників

Published at: **2019-11-01T16:49:39+00:00**

Author: ****

Original: [ФАКТИ](https://fakty.com.ua/ua/ukraine/20191101-skaletska-zvilnyla-vsih-svoyih-radnykiv/)

Міністр охорони здоров’я Зоряна Скалецька звільнила з посад своїх радників.
На думку Зоряни Скалецької, більшість радників в міністерстві мали особливий мандат, але не мали жодної відповідальності.
Читайте: Скалецька назвала пріоритети роботи МОЗ і завдання номер один
До слова, міністр охорони здоров’я додала, що МОЗ продовжить працювати з експертним середовищем, але статусів радника більше не буде.
Раніше Факти ICTV писали, що МОЗ планує скасувати медичні довідки.
