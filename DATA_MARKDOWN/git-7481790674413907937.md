
# 東証システム、処理能力１・５倍に…５日に刷新

Published at: **2019-11-04T00:03:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/economy/20191104-OYT1T50075/)

　東京証券取引所は５日、株式売買システム「アローヘッド」のシステムを刷新する。１秒間に数千回にも及ぶ高速取引の処理能力を高めたほか、急激な値動きへの対策も強化した。システムの安全性など性能向上を世界にアピールする。新システムでは、証券会社などからの注文に対し、受け取り通知を出すまでの時間が０・３ミリ秒から０・２ミリ秒に短縮される。単純計算では、通知回数は１秒間に約３３３３回から約５０００回へと１・５倍に増える。
