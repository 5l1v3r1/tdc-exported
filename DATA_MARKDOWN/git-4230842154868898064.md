
# Зеленський “в шоці” – завод Малишева за 10 років випустив один танк для ЗСУ

Published at: **2019-11-06T12:34:00+00:00**

Author: ****

Original: [ФАКТИ](https://fakty.com.ua/ua/ukraine/20191106-zelenskyj-v-shotsi-zavod-malysheva-za-10-rokiv-vypustyv-odyn-tank-dlya-zsu/)

Президент України Володимир Зеленський здивувався тому, що завод Малишева за останні 10 років випустив всього один танк для Збройних сил України.
Президент Володимир Зеленський зазначив, що його шокувала інформація про завод ім. В. О. Малишева, який з 2009 року випустив лише один танк для Збройних сил України.
