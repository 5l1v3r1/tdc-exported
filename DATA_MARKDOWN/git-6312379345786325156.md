
# Хэмилтон в шестой раз стал чемпионом Формулы-1

Published at: **2019-11-03T20:57:59+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-03/hemilton-v-shestoy-raz-stal-chempionom-formuly-1)

Британский автогонщик Льюис Хэмилтон команды "Мерседеса" в шестой раз в карьере стал чемпионом "Формулы-1".
Хэмилтон, завоевавший титул чемпиона мира в третий год подряд, обогнал по количеству титулов аргентинского автогонщика и пятикратного чемпиона "Формулы-1" Хуана Мануэля Фанхио по прозвищу Маэстро. Теперь британец уступает только семикратному чемпиону немцу Михаэлю Шумахеру.
Пять из шести титулов чемпиона мира (2014, 2015, 2017, 2018 и 2019 года) Хэмилтон получил в команде "Мерседес" в "гибридную эру". В 2008 году он выиграл чемпионат, являясь пилотом "Макларена" (2008).
