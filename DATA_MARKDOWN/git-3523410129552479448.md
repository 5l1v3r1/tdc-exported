
# 
        هوای تهران نا سالم برای گروه های حساس

به گزارش بولتن نیوز، براساس اعلام شرکت کنترل کیفیت هوای تهران شاخص کیفیت هوا روی عدد ۱۳۵ قرار دارد و امروز هوای تهران ناسالم برای گروه های حساس است.
