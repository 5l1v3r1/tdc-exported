
# Українсько-американська ракета вивела на орбіту космічний корабель

Published at: **2019-11-03T00:52:02+00:00**

Author: ****

Original: [ГЛАВКОМ](https://glavcom.ua/news/ukrajinsko-amerikanska-raketa-vivela-na-orbitu-vantazhniy-korabel-637281.html)

У суботу, 2 листопада, українсько-американська ракета-носій «Антарес» успішно вивела на орбіту вантажний корабель «Сігнус», який доставить на Міжнародну космічну станцію 3 750 кілограмів корисного вантажу, зокрема експериментальну духовку.
Про це повідомляє Конструкторське бюро «Південне», яке розробило основну конструкцію першого ступеню ракети.
Пуск ракети здійснили з Центру космічних польотів на острові Воллопс, штат Вірджинія. Стикування з МКС заплановане на 4 листопада.
«Сігнус» на замовлення NASA має доставити на станцію провізію та речі для екіпажу, матеріали для наукових досліджень, обладнання для магнітного альфа-спектрометра, устаткування для станції та виходу у відкритий космос, комп'ютери та комплектуючі.
