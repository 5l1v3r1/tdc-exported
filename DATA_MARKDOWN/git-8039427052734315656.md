
# భవన నిర్మాణ కార్మికుల ఆందోళన

విజయనగరం: జిల్లాలోని చీపురపల్లిలో భవన నిర్మాణ కార్మికులు ఆందోళన చేపట్టారు. ఎఐటియుసి సారధ్యంలో భవన నిర్మాణ కార్మికులు ఆందోళన నిర్వహించారు. ముఖ్యమంత్రి జగన్ తమ కూడులో ఇసుక పోస్తున్నారని నిరసన వ్యక్తం చేశారు. భవన నిర్మాణ కార్మికుల ఆందోళనకు టీడీపీ నేత కిమిడి నాగార్జున సంఘీభావం తెలిపారు.
