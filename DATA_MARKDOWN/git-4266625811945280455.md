
# Зеленский распорядился: Пенсии будут назначаться по-новому

Published at: **2019-11-06T08:55:00+00:00**

Author: ****

Original: [ZIK.UA](https://zik.ua/ru/news/2019/11/06/zelenskyy_rasporyadylsya_pensyy_budut_naznachatsya_ponovomu_1684685)


        Президент Украины Владимир Зеленский дал поручение Министерству социальной политики Украины разработать и внедрить систему электронного назначения пенсии.
      
Об этом сообщают «Факты».
Сейчас в Украине планируется введение подачи документов для назначения пенсий в электронном виде и получение консультаций по поводу их получения в аналогичном порядке.
После презентации приложения «Е-пенсия», планируется перенос таких же функций в телефон.
С этой целью Зеленский дал поручение Минсоцполитики, а на его выполнение дал два месяца. Соответственно, приложение должно заработать уже в конце этого года.
С его помощью можно будет проверить размер начисленной пенсии, получить консультацию специалиста и подать документы для назначения пенсии без личного участия.
Напомним, Милованов хочет отменить главный источник Пенсионного фонда.
