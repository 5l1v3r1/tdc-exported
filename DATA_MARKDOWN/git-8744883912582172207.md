
# WhatsApp обновил настройки конфиденциальности

Published at: **2019-11-07T13:51:29+00:00**

Author: **Евгений Федуненко**

Original: [Коммерсантъ](https://www.kommersant.ru/doc/4150440)

Мессенджер WhatsApp объявил, что теперь для групп действуют новые настройки конфиденциальности. Обновлена система приглашения в группы, чтобы пользователь мог самостоятельно принимать решения о том, кто может его добавлять.
Для активации функции нужно перейти в настройки и выбрать один из вариантов: «Все», «Мои контакты» (то есть те, кто сохранен в адресной книге пользователя), «Контакты, кроме…». Теперь администратору группы, если он не имеет права добавить человека в чат, необходимо будет отправить пользователю личное приглашение в группу, которое будет действовать три дня и которое пользователь может либо принять, либо отклонить.
Новые настройки уже доступны некоторым пользователям, а в ближайшие дни станут доступны для всех в последней версии приложения WhatsApp.
