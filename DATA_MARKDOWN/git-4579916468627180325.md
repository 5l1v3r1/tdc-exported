
# Во Франции опрокинулся автобус с туристами. СМИ пишут о пострадавших россиянах

Published at: **2019-11-03T15:14:00+00:00**

Author: ****

Original: [Фонтанка.ру](https://www.fontanka.ru/2019/11/03/044/)

Больше тридцати человек пострадали в аварии с туристическим автобусом, который следовал из Парижа в Лондон и не сладил со скользкой дорогой. Об этом 3 ноября сообщает Courrier Picard.
