
# Burgemeester geschokt door vuurwerkbom

Published at: **2019-11-04T09:25:00+00:00**

Author: ****

Original: [omroep flevoland](https://www.omroepflevoland.nl/nieuws/175512/burgemeester-geschokt-door-vuurwerkbom)


        Burgemeester Ineke Bakker van Urk is geschokt door de explosie van een zware vuurwerkbom in een woning aan Golfoploop. De bom richtte in de nacht van zaterdag op zondag zware schade aan.
