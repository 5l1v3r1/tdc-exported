
# Локомотив ҳимоячиси Германиянинг энг истеъдодли 3 футболчисини айтиб ўтди

Published at: **2019-11-06T08:23:24+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/lokomotiv-himoyachisi-germaniyaning-eng-istedodli-3-futbolchisini-aytib-otdi)


        - Ҳозирги кунда Германиянинг энг истеъдодли 3 футболчисини айта оласизми?
      
- Аллақачон юлдуз бўлишга уринган Лерой Сане. Яна "Бавария"дан Серж Гнабри ва Никлас Зюлелар. Никлас дунёнинг энг яхши ҳимоячиларидан бирига айланиши мумкин.
Бенедикт "Локомотив"га қадар "Шальке-04" ва "Ювентус" клубларида ўйнаган.
Футболчи 2011-2017 йилларда Германия терма жамоаси шарафини ҳимоя қилган.
