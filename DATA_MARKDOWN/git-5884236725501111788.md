
# Петербургские «Силовые машины» будут выпускать лопатки для газовых турбин

Published at: **2019-11-07T16:14:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/740840-peterburgskie-silovye-mashiny-budut-vypuskat-lopatki-dlya-gazovykh-turbin)


        Петербургская компания «Силовые машины» рассматривает возможность изготовления лопаток для газовых турбин. Для этого, возможно, создадут совместное предприятие.
      
 «Силовые машины» изучают возможность создания совместного предприятие по выпуску лопаток для турбин.
