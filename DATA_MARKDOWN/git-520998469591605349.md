
# Першокурсникам економічного факультету ЧНУ влаштували посвяту за кордоном

Published at: **2019-11-05T14:58:00+00:00**

Author: ****

Original: [Погляд](https://pogliad.ua/news/education/pershokursnikam-ekonomichnogo-fakultetu-chnu-vlashtuvali-posvyatu-za-kordonom-379858)

19 жовтня 2019 року студенти спеціальності «Міжнародні економічні відносини» та співробітники економічного факультету Чернівецького національного університету імені Юрія Федьковича відвідали місто Ясси (Румунія) з метою посвяти студентів першого курсу та ознайомлення із румунським містом та культурою сусідньої країни.
Під час подорожі група із 50 майбутніх економістів-міжнародників мали змогу познайомитися з містом та архітектурними пам’ятками, відвідати чотири музеї Палацу культури та провести інтернаціональний квест-випробування для першачків у тамтешньому парку. Про це йдеться на сайті ЧНУ.
