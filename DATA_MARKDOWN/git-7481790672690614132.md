
# 来春のＮＨＫ朝ドラ、撮影途中で脚本家交代「理由の回答控える」

Published at: **2019-11-05T10:55:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/culture/20191105-OYT1T50211/)

　ＮＨＫは５日、２０２０年春スタートの連続テレビ小説「エール」について、制作上の都合により、脚本家を林宏司さんから、清水友佳子さん、嶋田うれ葉さんと番組スタッフに交代すると発表した。
　ＮＨＫによると、同作は９月から撮影が行われていた。具体的な交代の理由について、同局は「制作過程の詳細については回答を控える」としている。
　「エール」は、福島市出身の作曲家で全国高校野球大会の大会歌「栄冠は君に輝く」などで知られる古関裕而（ゆうじ）さんをモデルにした朝の連ドラ１０２作目。
