
# ［深層ＮＥＷＳ］２閣僚辞任「野党が力を持ち、政治に緊張感を」

Published at: **2019-11-07T14:59:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/politics/20191107-OYT1T50264/)

　自民党の石原伸晃元経済再生相と国民民主党の玉木代表が７日、ＢＳ日テレの「深層ＮＥＷＳ」に出演し、菅原一秀・前経済産業相と河井克行・前法相の辞任などについて議論した。
　石原氏は「批判を重く受け止めなければいけない」と述べた。玉木氏は「（安倍内閣が）自爆している。重要課題を議論できる環境を政府・与党に整えてもらいたい」と注文を付けた。同じく番組に出演した一橋大の中北浩爾教授は「もう少し野党が力を持って、政治に緊張感を持たせることが必要ではないか」と求めた。
