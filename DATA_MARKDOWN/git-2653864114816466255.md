
# «Северсталь» проиграла лидеру Восточной конференции КХЛ

Published at: **2019-11-04T19:27:00+00:00**

Author: ****

Original: [35медиа](https://35media.ru/news/2019/11/04/severstal-proigrala-lideru-vostochnoj-konferencii-khl)

«Северсталь» нет смогла обыграть лидера Восточной конференции КХЛ — казанский «Ак Барс». Гости в середине второго периода в течение минуты забросили две шайбы.
В третьем периоде «Северсталь» отыграться не смогла. В итоге поражение — 0:2, уже восьмое подряд для череповецкой команды.
