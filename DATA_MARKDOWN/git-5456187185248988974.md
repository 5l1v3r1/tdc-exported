
# В Полоцке «Табакерку» поставили на парковочное место для инвалидов

Published at: **2019-11-01T20:14:00+00:00**

Author: ****

Original: [charter97.org](https://charter97.org/ru/news/2019/11/1/354077/)

Рядом с супермаркетом Green в Полоцке киоск «Табакерка» поставили на парковочное место для людей с ограниченными возможностями.
Сотрудники супермаркета рассказали, что киоск появился несколько дней назад, но люди уже начали писать по этому поводу жалобы.
Руководство Green говорит, что не имеет отношения к установке киоска — они просто снимают помещение в торговом доме «Стрелец». Его владелец Никита Захаров сказал «Еврорадио», что количество парковочных мест для инвалидов не изменилось, их по-прежнему четыре — одно место добавили вместо занятого «Табакеркой».
