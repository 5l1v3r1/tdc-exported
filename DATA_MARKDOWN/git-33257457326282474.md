
# «Яндекс» показал робота-курьера

Published at: **2019-11-07T13:18:42+00:00**

Author: **Meduza**

Original: [Meduza](https://meduza.io/news/2019/11/07/yandeks-pokazal-robota-kuriera)

«Яндекс» начал тестировать робота для перевозки небольших грузов, сообщается в пресс-релизе компании, поступившем в «Медузу».
Похожий на тележку «Яндекс.Ровер» может ездить по тротуарам со скоростью пешехода, пока что его используют только внутри компании. Робот доставляет документы от дверей штаб-квартиры «Яндекса» до автобуса, который развозит бумаги по другим офисам компании.
