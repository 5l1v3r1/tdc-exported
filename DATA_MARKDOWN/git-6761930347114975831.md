
# Израиль нанес удар по сектору Газа в ответ на ракетный обстрел

Published at: **2019-11-02T10:19:27+00:00**

Author: ****

Original: [Крым.Реалии](https://ru.krymr.com/a/news-israel-nanes-raketniy-udar-po-sektoru-gaza/30249433.html)

Не менее 10 ракет были выпущены вечером в пятницу из сектора Газа по южным районам Израиля, сообщает русская служба Радио Свобода.
По сообщению израильских военных, их целью были объекты, связанные с радикальной исламистской группировкой ХАМАС, которая управляет сектором.
«Израильские истребители начали наносить удары по позициям террористов в Газе. Это ответ на десять ракет, которые были выпущены из Газы по израильским гражданским лицам ранее», – говорится в сообщении пресс-службы Вооруженных сил страны.
