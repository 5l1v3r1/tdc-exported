
# Акира (EXILE) стал первой международной моделью Ralph Lauren из Азии

Published at: **2019-11-01T01:52:40+00:00**

Author: **Zarina**

Original: [YesAsia](https://www.yesasia.ru/article/752993)

Акира (EXILE, EXILE THE SECOND) стал международной моделью для элитной линии мужской одежды Ralph Lauren Purple Label из коллекции осень-зима 2019 года.

Акира являлся амбассадором Ralph Lauren с 2017 года. Теперь он стал первой международной моделью бренда из Азии. Сам Ральф Лорен дал восторженные отзывы о рекламе с участием Акиры и лично назначил его моделью для международной рекламы.
Рекламный щит с изображением Акиры уже установлен перед отелем Chateau Marmot в Лос-Анджелесе. Его будут продолжать показывать на рекламных щитах и в журналах в Соединенных Штатах, Европе и Азии.
