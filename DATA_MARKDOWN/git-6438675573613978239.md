
# ФСБ пресекла деятельность группы вымогателей в Крыму

Published at: **2019-11-02T06:58:00+00:00**

Author: ****

Original: [РИА Новости](https://ria.ru/20191102/1560518026.html)

СИМФЕРОПОЛЬ, 2 ноя - РИА Новости. Деятельность преступной группы, специализирующейся на вымогательстве, пресечена в Крыму сотрудниками управлений ФСБ и МВД по региону, сообщает пресс-служба УФСБ по Крыму.По данным ведомства, граждане РФ, выходцы из республик Северного Кавказа на протяжении нескольких недель с угрозой применения насилия вымогали у местного жителя 350 тысяч рублей."Тридцатого октября один из членов преступной группы, 1979 года рождения, задержан с поличным при получении требуемых денежных средств. В отношении него следственными органами МВД по Республике Крым возбуждено уголовное дело по признакам преступления, предусмотренного п. "в" ч.2 ст. 163 УК РФ (вымогательство, совершенное в крупном размере)", - говорится в сообщении.Санкция статьи предусматривает наказание до семи лет лишения свободы. Все обстоятельства произошедшего устанавливаются, подозреваемый задержан. Решается вопрос об избрании ему меры пресечения, сообщили в пресс-службе.
