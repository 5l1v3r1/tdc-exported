
# LIVE: Μπάγερν - Ολυμπιακός (19.55)

Published at: **2019-11-06T19:33:00+00:00**

Author: ****

Original: [enikos.gr](http://www.enikos.gr/sports/682733/live-bagern-olympiakos-1955)

Ο Ολυμπιακός έχει δύσκολο έργο στο Μόναχο (19.55) για την 4η αγωνιστική των ομίλων του Τσάμπιονς Λιγκ. 
Οι συνθέσεις των δύο ομάδων:
Μπάγερν: Νόιερ, Παβάρ, Χάβι Μαρτίνεθ, Λεβαντόφσκι, Γκορέτσκα, Ντέιβις, Γκνάμπρι, Μίλερ, Αλάμπα, Κομάν, Κίμιχ.
Ολυμπιακός: Σα, Ελαμπντελαουΐ, Τσιμίκας, Σεμέδο, Μεριά, Γκιλιέρμε, Μπουχαλάκης, Καμαρά, Ραντζέλοβιτς, Γκερέρο, Ποντένσε.
