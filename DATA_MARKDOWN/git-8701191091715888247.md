
# Специальный налоговый режим введут по всей стране

Published at: **2019-11-05T10:47:00+00:00**

Author: ****

Original: [Клерк](https://www.klerk.ru/buh/news/491971/)

На днях Минфин разработал проект по включению в эксперимент по введению НПД для самозанятых еще в 19 регионах.
Таким образом, с января 2020 НПД будут применять желающие граждане из 23 регионов РФ.
Более того, с июля 2020 этот режим планируют распространить на всю страну.
Минфин доволен итогами эксперимента по НПД. Между тем не все разделяют этот восторг. Свои сомнения на счет пользы НПД, в частности высказал депутат Госдумы Ярослав Нилов.
Он напомнил, что самозанятые не платят взносы на ОПС и в будущем смогут рассчитывать только на социальную пенсию.
Что думают наши читатели по этому поводу, выясним в ходе опроса, в котором приглашаем вас принять участие.
