
# കാന്താരിയുടെ കലിപ്പന്‍ ഇനി ബിഗ്‌സ്‌ക്രീനിലും; ഫുക്രുവിന് പിന്നാലെ ഷാരിഖും സിനിമയില്‍

Published at: **2019-11-05T13:30:00+00:00**

Author: **എന്റര്‍ടെയിന്‍മെന്റ് ഡെസ്‌ക്**

Original: [DoolNews](https://www.doolnews.com/kalippante-kanthari-fame-sharik-become-hero-on-new-movie.html)

കാന്താരിയുടെ കലിപ്പന്‍ എന്ന ചെറു ടിക് ടോക് വീഡിയോകള്‍ നിങ്ങളും കണ്ടിട്ടുണ്ടാവും. ആ വീഡിയോകളില്‍ കലിപ്പനായി വന്ന മുഹമ്മദ് ഷാരിഖിനെ ഇനി ബിഗ് സ്‌ക്രീനില്‍ കാണാം.
മുഹമ്മദ് ഷാരിഖിനെ നായകനാക്കി പുതിയ ചിത്രം വരുന്നു. ഷാരിഖ് തന്നെയാണ് ഇക്കാര്യം ഫേസ്ബുക്കിലൂടെ അറിയിച്ചത്. സിനിമയുടെ ടൈറ്റില്‍ ഇത് വരെ പ്രഖ്യാപിച്ചിട്ടില്ല. നവംബര്‍ 10ന് ചിത്രത്തിന്റെ പേര് അനൗണ്‍സ് ചെയ്യും.
നവാഗതനായ കബീര്‍ പുഴമ്പുറം ആണ് ചിത്രം സംവിധാനം ചെയ്യുന്നത്. വിഷ്ണുപ്രിയ സത്യന്‍ നായികയാവും. ഹസീബ് മേപ്പാട്ട് ആണ് നിര്‍മ്മാതാവ്. ബിനേഷ് ചിത്രത്തിന്റെ സംഗീത സംവിധാനം നിര്‍വഹിക്കും.
മറ്റൊരു ടിക് ടോക് താരമായി ഫുക്രുവും സിനിമയില്‍ അരങ്ങേറ്റം കുറിച്ചിരുന്നു. ഫുക്രുവിന്റെ ചിത്രമായ തല്ലുംപിടി റിലീസിനൊരുങ്ങുകയാണ്.
