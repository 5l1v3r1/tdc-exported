
# В столице Коми уплыл мост

Published at: **2019-11-03T07:45:00+00:00**

Author: ****

Original: [БНК](https://www.bnkomi.ru/data/news/102251/)

Сегодня около 8 утра понтонный мост с улицы Кутузова в Заречье оторвало шугой.
Как сообщили БНК в отделе ГИБДД по Сыктывкару, ремонтные работы уже начались. Движение по мосту будет восстановлено после его возвращения на прежнее место.
Госавтоинспекция обращается к водителям и пешеходам заранее планировать свой маршрут движения и быть внимательными к установленным временным дорожным знакам.
Как сообщили БНК в ГУ МЧС по Коми, на месте дежурят городские спасатели, которые осуществляют перевозку с помощью катера «Орбита». Мост будет установлен на прежнее место после прохода шуги.
