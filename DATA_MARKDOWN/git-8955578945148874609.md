
# СК проводит проверку по факту избиения врача скорой помощи в Северске

Published at: **2019-11-07T15:12:00+00:00**

Author: **Масляева Мария Сергеевна**

Original: [РИА Томск](https://www.riatomsk.ru/article/20191107/izbienie-vracha-skoroj-pomoschi-seversk-sk-proverka/)

ТОМСК, 7 ноя – РИА Томск, Мария Масляева. Следственный комитет (СК) проводит проверку по факту избиения врача скорой помощи в Северске; по предварительным данным, доктора избил родственник умершего на улице мужчины, сообщила в четверг пресс-служба ведомства.
Добавляется, что региональный СК проводит проверку по факту произошедшего.
