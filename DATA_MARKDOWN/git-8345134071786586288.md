
# Red Dead Redemption 2 para PC já tem trailer de lançamento

Published at: **2019-11-04T12:31:53+00:00**

Author: ****

Original: [PCGuia](https://www.pcguia.pt/2019/11/red-dead-redemption-pc-trailer/)

A Rockstar Games publicou o trailer de lançamento da versão PC Windows de Red Dead Redemption 2.
Os jogadores que encomendaram Red Dead Redemption 2 para PC através do Rockstar Launcher vão receber um outro jogo da Rockstar Games, bem como o War Horse, o kit Outlaw Survival e o Mapa Le Trésor des Morts para o Story Mode, 50 Barras de Ouro para o modo Red Dead Online, e descontos nos upgrades para a Edição Especial e Edição Definitiva.
A data oficial de lançamento da versão PC Windows de Red Dead Redemption 2 é dia 5 de Novembro. Contudo, este título só vai chegar à loja Steam em Dezembro. Red Dead Redemption 2 já está disponível para a PlayStation 4 e Xbox One.
