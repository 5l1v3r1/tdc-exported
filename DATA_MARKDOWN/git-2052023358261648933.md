
# «Не поддерживаем стремление сделать из него символическую фигуру» — ректор ВШЭ о Егоре Жукове

Published at: **2019-11-05T11:27:00+00:00**

Author: ****

Original: [Медиазона](https://zona.media/news/2019/11/05/kzmnv)

Ректор Высшей школы экономики Ярослав Кузьминов прокомментировал уголовное преследование студента вуза, фигуранта «московского дела» Егора Жукова. Заявление опубликовано на сайте университета. 
«Те взрослые люди, кто начинает раскручивать Егора как "политическую звезду", оказывают плохую услугу ему и его семье», — написал Кузьминов. 
По его словам, Жуков — «еще очень молодой человек, его амбиции явно превосходят его знания». Как заверил Кузьминов, студент «может еще несколько раз изменить свои политические взгляды».
