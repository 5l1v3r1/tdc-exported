
# Израиль сообщил о перехвате семи ракет, выпущенных из сектора Газа

Published at: **2019-11-01T19:46:00+00:00**

Author: ****

Original: [Interfax.ru](https://www.interfax.ru/world/682741)

Москва. 1 ноября. INTERFAX.RU - Из сектора Газа по Израилю выпущены семь ракет, их удалось перехватить, говорится в сообщении, распространенном в пятницу в твиттере армии обороны Израиля.
"Семь ракет запущены из Газы по Израилю. Противоракетная система "Железный купол" перехватила их все", - говорится в сообщении.
Также израильские военные опубликовали кадры, на которых, по их словам, запечатлен момент перехвата ракет.
