
# ＮＹダウ、２日連続で最高値…米中協議進展の見方強まる

Published at: **2019-11-05T21:40:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/economy/20191106-OYT1T50165/)

　【ニューヨーク＝小林泰明】５日のニューヨーク株式市場は、米中貿易摩擦の緩和への期待が継続し、ダウ平均株価（３０種）は２日連続で過去最高値を更新した。終値は前日比３０・５２ドル高の２万７４９２・６３ドルだった。
　英紙フィナンシャル・タイムズ（ＦＴ）電子版などが、米国が９月に発動した中国に対する制裁関税の撤廃を検討していると報じ、米中貿易協議が順調に進展しているとの見方が強まった。
　全米供給管理協会（ＩＳＭ）が５日発表した１０月の非製造業景況指数が前月から改善し、米景気に対する安心感も広がり、買い注文を支えた。
　ただ、当面の利益を確定しようとする売り注文もあり、値上がりは小幅にとどまった。
　情報技術（ＩＴ）企業の銘柄が多いナスダック店頭市場の総合指数の終値は、１・４８ポイント高の８４３４・６８だった。
