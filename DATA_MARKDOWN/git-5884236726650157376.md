
# 17-летний лешуконец угнал шесть авто, чтобы покататься по Архангельску

Published at: **2019-11-01T22:22:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Arkhangelsk/738919-17-letnii-leshukonec-ugnal-shest-avto-chtoby-pokatatsya-po-arkhangelsku)


        Парень из сельского района Поморья угонял автомобили, чтобы покататься по улицам столицы области. Ему грозит реальный срок заключения. 
      
Полицейские Архангельска раскрыли серию угонов автомобилей в столице Поморья, сообщает сайт МВД по региону. 
Заявления об угонах поступали в дежурную часть с сентября по начало октября. Все заявления объединяло то, что автомобили были дешевые, отечественного производства и не оснащенные сигнализацией. 
На днях полицейские задержали подозреваемого в угоне шести автомобилей. 17-летний житель Лешуконского района угонял отечественные авто, чтобы покататься по улицам Архангельска. Позабавившись, он бросал авто на обочинах. Сейчас все авто возвращены владельцам. 
По данным фактам возбуждено уголовное дело по признакам преступления, предусмотренного частью 1 статьи 166 Уголовного кодекса Российской Федерации «Неправомерное завладение транспортным средством без цели хищения».
