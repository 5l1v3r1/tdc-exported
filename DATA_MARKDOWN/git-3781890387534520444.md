
# นิวซีแลนด์มุ่งยกระดับสัมพันธ์กับอินโด-แปซิฟิก

Published at: **2019-11-01T05:40:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/foreign/739460)


        นายกรัฐมนตรีหญิงของนิวซีแลนด์มีกำหนดเยือนกรุงเทพมหานครในช่วงสุดสัปดาห์นี้ เพื่อร่วมการประชุมย่อยหลายรายการของการประชุมอาเซียน ซึ่งเธอมุ่งมั่นใช้เป็นโอกาสกระตุ้นความร่วมมือระดับภูมิภาคอินโด-แปซิฟิกในทุกด้าน
      
สำนักข่าวต่างประเทศรายงานจากกรุงเวลลิงตัน ประเทศนิวซีแลนด์ เมื่อวันที่ 1 พ.ย.ว่านายกรัฐมนตรีจาซินดา อาร์เดิร์น กล่าวเมื่อวันศุกร์ เกี่ยวกับเป้าหมายของเธอในการเยือนกรุงเทพมหานคร ระหว่างวันที่ 3 ถึง 4 พ.ย.นี้ เพื่อเข้าร่วมการประชุมการประชุมเอเชียตะวันออก ( อีเอเอส ) ครั้งที่ 14 และการประชุมความตกลงหุ้นส่วนทางเศรษฐกิจระดับภูมิภาค ( อาร์เซป ) ซึ่งเป็นหนึ่งในการหารือย่อยของการประชุมสุดยอดสมาคมประชาชาติแห่งเอเชียตะวันออกเฉียงใต้ ( อาเซียน ) ครั้งที่ 35 ซึ่งไทยเป็นประธานในปีนี้ ว่าเธอต้องการแลกเปลี่ยนทรรศนะกับทุกฝ่ายในประเด็นทางยุทธศาสตร์และความมั่นคงที่ภูมิภาคอินโด-แปซิฟิกกำลังเผชิญร่วมกัน ไม่ว่าจะเป็นเรื่องนิวเคลียร์ ภัยคุกคามจากการก่อการร้าย การแทรกซึมของแนวคิดหัวรุนแรง ความปลอดภัยทางทะเล และภาวการณ์เปลี่ยนแปลงของสภาพอากาศโลก
