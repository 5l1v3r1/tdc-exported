
# 【直播預告】香港11.9-10反極權反警暴活動

Published at: **2019-11-07T23:37:05+00:00**

Author: ****

Original: [NTD 新唐人電視台](https://www.ntdtv.com/b5/2019/11/07/a102702269.html)

【新唐人北京時間2019年11月08日訊】本週末，香港民眾再次發起反極權反暴政活動。週六舉辦「紀念柏林圍牆倒塌30周年集會」，週日將有「屯門牛大遊行」等。新唐人電視台和《大紀元時報》將進行全程追蹤和網絡直播，請鎖定新聞直播間。
香港民眾的抗爭活動至今已近5個月，每週多次集會遊行要求「五大訴求 缺一不可」，但屢屢遭到港警的暴力驅散，迄今超過3000人被捕，還不斷爆出被捕者被虐待、酷刑、性侵等可怕的傳聞，多人死亡被懷疑被自殺，而港府拒絕獨立調查。
本週，民意不斷創新低的香港特首林鄭月娥北上受到中國國家主席習近平和中共常委韓正的接見，其不作為還被肯定，被要求繼續「止暴制亂、恢復秩序」。港人擔憂香港步入黑暗時期。
香港民主黨黨主席胡志偉刊文說，中共以一國牢牢壓著兩制，港人的高度自治空間越來越小，而中共正「將香港從威權帶入極權」。但港人相信「正義或會遲到，總不會缺席」，學生喊出口號「希望在於人民，改變始於抗爭」，「願榮光歸香港」。
