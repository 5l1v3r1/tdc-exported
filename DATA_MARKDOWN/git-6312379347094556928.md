
# Французские истребители нанесли удары по туннелям ИГ* в Ираке

Published at: **2019-11-01T16:11:14+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-01/francuzskie-istrebiteli-nanesli-udary-po-tunnelyam-ig-v-irake)

Французские истребители Rafale нанесли удары по туннелям на северо-востоке Ирака, используемые ИГ (террористическая группировка, запрещена в РФ) как тыловые базы для своих действий. Об этом сообщает РИА Новости.
"(Операция) Chammal - французские Rafale уничтожили несколько укрытий ИГ на северо-востоке Ирака", - сообщает пресс-служба министерства обороны Франции.
По данным издания, операция, целью которой помимо прочего было ослабление логистических и военных способностей группировки, проводилась в пятницу 31 октября совместно с международной коалицией.
Министр обороны Франции Флоранс Парли подчеркнула, что Франция остается привержена борьбе с терроризмом.
* ИГ - террористическая группировка, которая запрещена на территории РФ.
