
# Ўзбекистон МТЖ. Одил Аҳмедов ва Игорь Сергеев иштирок этмаган машғулотдан ГАЛЕРЕЯ

Published at: **2019-11-06T07:10:00+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/ozbekiston-mtj-odil-ahmedov-va-igor-sergeev-ishtirok-etmagan-mashgulotdan-galereya)

Вадим Абрамов бошчилигида терма жамоамиз кеча ҳам "Жар" стадионининг захира майдонида шуғулланди. Жамоамиз сардори Одил Аҳмедов индивидуал тарзда шуғулланди. Игорь Сергеев шамоллаб қолгани боис ҳозирча терма жамоа машғулотларида иштирок этмаяпти, яқин вақтлар ичида Сергеев жамоа ихтиёрига қўшилади. Кеча Тошкентга икки нафар легионеримиз Икром Алибоев ва Рустам Ашурматов ҳам етиб келишди. Аммо мураббийлар сафардан келган футболчиларга дам беришди. Бу икки ўйинчимиз бугунги машғулотларда иштирок этишади.
