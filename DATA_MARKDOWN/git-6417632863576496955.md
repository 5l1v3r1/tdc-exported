
# Мессининг сўнгги баҳслардаги шахсий кўрсаткичлари билан танишинг

Published at: **2019-11-02T18:32:27+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/messining-songgi-bahslardagi-shaxsiy-korsatkichlari-bilan-tanishing)

Ўйиннинг 38-дақиқасида "Леванте" дарвозаси томон белгиланган пенальтини аниқ амалга оширган Месси сўнгги 5 учрашувдаги ўзининг 6-голини киритди.
Аргентиналик футболчи бунга қадар "Славия" (2:1), "Эйбар" (3:0) ва "Севилья" (4:0) жамоалари дарвозасига биттадан гол киритган бўлса, "Вальядолид"(5:1) дарвозабонини икки бор ранжитганди.
Бундан ташқари, Лео 3 бор шерикларини голли узатма билан ҳам таъминлаган.
