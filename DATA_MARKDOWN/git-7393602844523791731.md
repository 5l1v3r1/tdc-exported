
# Бассейн Орбита в Сыктывкаре информатизируют

Published at: **2019-11-05T09:20:00+00:00**

Author: **Комиинформ**

Original: [Комиинформ](https://komiinform.ru/news/188284)

Министерство физической культуры и спорта Коми совместно с региональным "Центром информационных технологий" прорабатывают вопрос информатизации отрасли физкультуры и спорта республики, сообщает пресс-служба ведомства.
В качестве пилотного проекта было выбрано подведомственное учреждение минспорта – "Спортивная школа по плаванию "Орбита".
"Основная суть состоит в том, чтобы всю имеющуюся информацию, которая есть по самому учреждению, его персоналу, спортсменам и тренерам перевести в электронный вид. В последствии это не только облегчит работу для самого учреждения, но и данные станут доступными в удобной форме для рядового пользователя", - отметили в минспорта республики.
Спортшколы смогут сформировать полный "паспорт" спортсмена с его достижениями, этапами подготовки и участием в соревнованиях, медицинскими показаниями. Такая же схема формирования данных может быть реализована и по тренерам – образование, стаж работы, достижения иные данные – станут частью открытого портфолио.
"Из общедоступной информации, например, родитель при выборе спортшколы для своего ребенка, сможет проанализировать какую секцию выбрать, подходит ли ему наставник, время проведения тренировок, стоимость услуг и просто записать ребенка на занятия. Все это поможет уже заранее определиться со спортивным будущим малыша", - рассказали в республиканском Центре информационных технологий.
В последствии информация по всем учреждения будет объединена в одну базу данных, которая будет доступна для всех желающих в онлайн режиме. Причем, по задумке, сюда войдут не только спортивные учреждения, подведомственные минспорта региона, но спортшколы муниципалитетов Коми.
"Нельзя забывать, что сегодня в стране реализуется национальный проект "Цифровая экономика Российской Федерации. В составе которого есть федеральный проект "Цифровое государственное управление", где одной из задач стоит цифровизация основных социально-значимых услуг и сервисов. По нашему мнению, сфера физической культуры и спорта региона может получить единую информационную "платформу" для доступа пользователей, желающих получить общедоступную информацию по отрасли по модели "одного окна", - подчеркнули в министерстве физической культуры и спорта Республики Коми.
