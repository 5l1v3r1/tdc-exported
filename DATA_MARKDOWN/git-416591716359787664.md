
# Facebook testa login via reconhecimento facial

Published at: **2019-11-05T13:34:00+00:00**

Author: **Matheus Luque**

Original: [Olhar Digital](https://olhardigital.com.br/noticia/facebook-testa-login-via-reconhecimento-facial/92540)

Diversas empresas de tecnologia adotam a biometria para desbloqueio de determinado serviço. A tecnologia pode até ter seus pontos negativos, mas definitivamente, é mais conveniente que digitar a senha inúmeras vezes. O Facebook, entretanto, está optando pela identificação via reconhecimento facial.
Para ativar o modo, o usuário precisará tirar algumas selfies, sempre olhando para diferentes direções. Com a nova tecnologia, os usuários da rede social de Mark Zuckerberg não precisarão mais digitar e-mail e senha toda vez que entrarem na plataforma.
