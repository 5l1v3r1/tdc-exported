
# В Украине открыли первую автозаправку без персонала

Published at: **2019-11-05T21:25:00+00:00**

Author: ****

Original: [Главное™](https://glavnoe.ua/news/n357008200-v-ukraine-otkryli-pervuju-avtozapravku-bez-personala)

В Виннице открыли первую полностью автоматизированную заправочную станцию. Заправочный комплекс работает без вспомогательного персонала - все операции водители выполняют самостоятельно, сообщает abcnews.
АЗС работает по принципу самообслуживания - водитель приезжает, выбирает горючее, вставляет пистолет в бак, а дальше на терминале вводит нужную сумму и вносит средства. Рассчитаться можно наличными, с помощью PayPass, банковской или специальной топливной карты.
По словам разработчиков, подобная автоматизированная заправка - первая в Украине. В первый день работы услугами АЗС воспользовалось более полусотни клиентов.
