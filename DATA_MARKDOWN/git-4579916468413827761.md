
# Сыграть в ящик. HBO придумала, как смотреть сериалы в переполненной комнате общежития

Published at: **2019-11-06T05:36:00+00:00**

Author: ****

Original: [Фонтанка.ру](https://www.fontanka.ru/2019/11/06/017/)

Новый гаджет, который поможет уединиться для просмотра фильмов или сериала, презентует американская компания HBO. Правда, если есть чувство юмора, что новым станет и хорошо забытое старое.
Компания запустила конкурс для студентов, в котором можно выиграть новинку – картонную коробку The HBO Box, которая предназначена для просмотра фильмов и сериалов. Коробка выполнена в чёрном цвете. Как пишет 6 ноября TJ, она создана для уединения при жизни с соседями по квартире или общежитию.
