
# తెలంగాణలో మునిసిపల్ ఎన్నికలకు రంగం సిద్ధం

హైదరాబాద్: తెలంగాణలో మునిసిపల్ ఎన్నికలకు రంగం సిద్ధమైంది. సోమవారం షెడ్యూల్ విడుదల చేసే అవకాశం ఉంది. రాష్ట్ర వ్యాప్తంగా రెండు దశల్లో మునిసిపోల్స్ నిర్వహించనున్నట్టు తెలుస్తోంది. కోర్టు సమస్యలు లేని మునిసిపాలిటీలకు మొదటిదశలో ఎన్నికలు నిర్వహించనున్నట్టు తెలుస్తోంది. కోర్టు విచారణలో ఉన్న మునిసిపాలిటీలకు రెండో దశలో ఎన్నికలు నిర్వహించనున్నట్టు సమాచారం. మొదటిదశ మునిసిపాలిటీల రిజర్వేషన్ల కసరత్తును ప్రభుత్వం ఇప్పటికే పూర్తి చేసింది. నోటిఫికేషన్ వెలువడగానే వార్డుల రిజర్వేషన్ల డ్రా జరగనుంది.
