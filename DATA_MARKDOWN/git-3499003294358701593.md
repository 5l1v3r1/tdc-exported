
# Оперативники ЗелАО задержали подозреваемых в мошенничестве

Published at: **2019-11-07T10:11:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/zelao/operativniki-zelao-zaderzhali-podozrevaemykh-v-moshennichestve-123494/)

В ОМВД России по району Крюково обратился 37-летний специалист отдела безопасности банка о хищении денежных средств, принадлежащих банку. Общая сумма материального ущерба составила более 337 000 рублей.Сотрудниками уголовного розыска ОМВД России по району Крюково совместно с коллегами из ОМВД России по районам Матушкино и Савёлки задержаны подозреваемые женщины 25-ти и 22-х лет на территории ближайшего Подмосковья.Как установлено, девушка, работавшая в одном из отделений банков Зеленограда, обманным путем завладела паспортными данными незнакомых ей граждан. Далее, используя свое служебное положение, подозреваемая оформила несколько кредитных карт. После этого ее сообщница получила их в банковской организации. Денежные средства девушки поделили между собой пополам. Следственным управлением УВД по Зеленоградскому АО ГУ МВД России по г. Москве возбуждено уголовное дело по признакам преступления, предусмотренного статьей 159 УК РФ «Мошенничество». В отношении подозреваемых избрана мера пресечения в виде подписки о невыезде и надлежащем поведении.Пресс-служба УВД по ЗелАО
