
# 「あおり殴打」無関係の女性を「同乗者」に…デマ拡散の市議辞職

Published at: **2019-11-03T05:11:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191103-OYT1T50115/)

　茨城県の常磐道で８月、あおり運転をした上、男性を殴ったとして男が逮捕された事件に絡み、無関係の女性を男の同乗者とするデマ情報を自身のフェイスブックに書き込んだ愛知県豊田市議の原田隆司氏（５７）が２日、議員の辞職願を杉浦弘高・同市議会議長に提出した。杉浦議長はこれを受理、同日付で許可した。
　読売新聞の取材に対し、原田氏は「このまま議員活動を続ければ、市民や市議会に多大な迷惑をかけると判断し、一日も早く身を引くことにした」と述べた。
