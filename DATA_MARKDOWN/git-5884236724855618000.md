
# Светловчанин без повода сломал прохожему палец на руке

Published at: **2019-11-02T13:37:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Kaliningrad/739067-svetlovchanin-bez-povoda-slomal-prokhozhemu-palec-na-ruke)


        В Светлом мужчина без каких-либо причин затеял скандал с прохожим, в ходе которого нанес удар по руке и сломал потерпевшему палец. Злоумышленника удалось задержать, по факту случившегося возбуждено уголовное дело.
      
Житель Светлого подозревается в умышленном причинении средней тяжести вреда здоровью. По информации УМВД России по Калининградской области, 33-летний мужчина без особых оснований затеял конфликт с 50-летним прохожим. В ходе конфликта злоумышленник ударил оппонента ногой по правой руке и сломал ему палец.
Сообщается, что подозреваемый не имеет постоянного места работы. В отношении мужчины по факту произошедшего было возбуждено уголовное дело. Теперь злоумышленнику может грозить наказание вплоть до лишения свободы на срок до трех лет.
