
# 三大法人買超台股99.12億元

Published at: **2019-11-01T15:41:00+00:00**

Author: **https://www.facebook.com/cnanewstaiwan, 中央通訊社**

Original: [中央社 CNA](https://www.cna.com.tw/news/gpho/201911010002.aspx)

台股1日漲40.82點，收在11399.53點，成交值新台幣1332.48億元。三大法人合計買超新台幣99.12億元，自營商買超14.23億元，投信賣超2.98億元，外資及陸資買超87.87億元。中央社製圖 108年11月1日
