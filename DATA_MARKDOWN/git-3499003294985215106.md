
# Легковушка насмерть сбила женщину на пешеходном переходе в центре Москвы

Published at: **2019-11-07T10:35:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/proisshestviya/legkovushka_nasmert_sbila_zhenshchinu_na_peshekhodnom_perekhode_v_tsentre_moskvy/)

Ранним утром 7 ноября легковой автомобиль насмерть сбил женщину на нерегулируемом пешеходном переходе на ул. Знаменка в Хамовниках, сообщает пресс-служба ГИБДД по Москве.От полученных травм женщина скончалась на месте.Водитель место ДТП не покинул.
