
# На центральном проспекте в Мелитополе уже уложили первый слой асфальта (фото, видео)

Published at: **2019-11-03T06:11:00+00:00**

Author: **Влада Хмельницкая**

Original: [РИА Мелитополь](https://ria-m.tv/news/170374/na_tsentralnom_prospekte_v_melitopole_uje_ulojili_pervyiy_sloy_asfalta_(foto_video).html)


        Ремонтные работы продолжаются в выходные. Подрядчик АБЗ-Промстрой спешит  использовать погожие дня для того, чтобы успеть сдать объект в срок. Вчера на спуске у спорткомплекса ТГАТУ на пр. Б. Хмельницкого дорожники уложили первый слой асфальта.
      
Всего их будет два. Перекрывать движение полностью на перекрестке не пришлось. Работы велись по полосам.
Завершить укладку второго слоя асфальта и установку бордюров планируется через 2 недели.
