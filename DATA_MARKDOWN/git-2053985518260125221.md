
# أهم الاخبار| السيسي يتابع تطبيق الحد الأدنى للأجور.. اكتتاب أرامكو.. وموقف محرج لـ ترامب

Published at: **2019-11-03T04:14:00+00:00**

Author: **ElBalad**

Original: [صدى البلد](https://www.elbalad.news/4043786)
