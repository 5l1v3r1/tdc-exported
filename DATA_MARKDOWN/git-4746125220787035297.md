
# 福士蒼汰&横浜流星&伊藤あさひ“特撮ヒーロー”3ショットに「キター! 夢の共演」

Published at: **2019-11-07T12:21:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148205/full/)

　俳優の福士蒼汰が主演するTBS系連続ドラマ『4分間のマリーゴールド』（毎週金曜　後10：00）の公式SNSが7日、福士、横浜流星、伊藤あさひからなる“特撮ヒーロー出身”3人組の決めポーズ写真を公開した。 　福士は『仮面ライダーフォーゼ』（仮面ライダーフォーゼ）、横浜は『烈車戦隊トッキュウジャー』（トッキュウ4号）、伊藤は『快盗戦隊ルパンレンジャーVS警察戦隊パトレンジャー』（ルパンレッド）に出演していた“特撮ヒーロー”出身組。同ドラマの劇中で横浜と伊藤の共演シーンは今までになく、“レア”な3ショットとなっており、それぞれ自身が演じたヒーローの決めポーズを披露している。 　8日の午後10時からは第5話が放送。公式ツイッターでは「今後3人の共演シーンもある！...かも！？」と、含みをもたせたコメントで「お楽しみに」と締めている。 　この投稿にファンから「すごい共演ですね〜　世界の平和も守れそう」「3人揃ったら最強ですね」「これはヤバイ！ 貴重な3ショットありがとうございます」「さすが戦隊モノ出身者はポーズが決まってる」「キター！ 夢の共演　素敵すぎる〜」など、反響が寄せられている。
