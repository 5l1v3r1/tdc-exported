
# ثانويات الضنية ومدارسها تعاود فتح أبوابها

Published at: **2019-11-02T11:51:43+00:00**

Author: **Lebanon 24**

Original: [Lebanon24](https://www.lebanon24.com/news/lebanon-news/641348/lebanon24?src=rss-feeds&utm_campaign=Softimpact-Rss-Feeds&utm_source=SI-Rss-Feeds-Documents&utm_term=Softimpact-Rss-Feeds-Documents&utm_medium=SI-Rss-Feeds-641348)

عاودت أغلب الثانويات والمدارس الرسمية والخاصة فتح أبوابها اليوم أمام التلامذة، وعاودت إعطاء الدروس بشكل طبيعي، بعدما كانت أبلغتهم أمس عبر رسائل نصية معاودة الدوام كالمعتاد.
