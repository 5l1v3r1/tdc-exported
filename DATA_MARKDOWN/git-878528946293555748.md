
# 卓球、張本が日本勢首位の５位

Published at: **2019-11-04T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/general/2019/11/04/0012848801.shtml)

　国際卓球連盟は４日、１１月の世界ランキングを発表し、男子は張本智和（木下グループ）が前回と同じ５位で日本勢トップだった。丹羽孝希（スヴェンソン）は１１位、水谷隼（木下グループ）は１３位で変わらなかった。
　女子は伊藤美誠（スターツ）が日本勢最上位の７位、石川佳純（全農）は８位を維持。平野美宇（日本生命）は一つ下げて１０位だった。
　１位は男女とも中国勢で、男子は樊振東、女子は陳夢。（共同）
