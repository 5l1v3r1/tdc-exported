
# Девять школьников погибли при взрыве в Афганистане

Published at: **2019-11-02T07:15:56+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-02/devyat-shkolnikov-pogibli-pri-vzryve-v-afganistane)

Взрыв прогремел в провинции Тахар на северо-востоке Афганистана.
По данным агенства "Пажвак", жертвами атаки стали девять школьников, которые шли в момент взрыва на учебу.
Информация о ЧП выясняется.
