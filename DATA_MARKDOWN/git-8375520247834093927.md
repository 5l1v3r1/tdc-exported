
# На Олімпіаді-2020 планують використати новий метод антидопінгового тесту

Published at: **2019-11-05T10:45:52+00:00**

Author: ****

Original: [ГЛАВКОМ](https://glavcom.ua/sport/news/na-olimpiadi-2020-planuyut-vikoristati-noviy-metod-antidopingovogo-testu-637797.html)

Міжнародний олімпійський комітет планує застосовувати нові методи антидопінгового тестування на Олімпіаді-2020 в Токіо. Про це повідомив президент МОК Томас Бах.
«Ми хотіли б, аби порушники ніколи і ніде не почували себе в безпеці», - передає слова Баха Р-Спорт.
Мова йде про раніше анонсований метод аналізу сухої крові. Очікується, що такий спосіб робить тестування дешевшим та простішим та заощаджує місце для їх зберігання.
