
# В Самаре экстренно сел самолет, летевший из Петербурга в Узбекистан

Published at: **2019-11-04T11:39:00+00:00**

Author: ****

Original: [РИА Новости](https://ria.ru/20191104/1560560228.html)

САМАРА, 4 ноября — РИА Новости. Самолет, летевший из Санкт-Петербурга в Ургенч (Узбекистан), вынужденно сел в понедельник в Самаре из-за плохого самочувствия одного из пассажиров, сообщили РИА Новости в пресс-службе аэропорта.
По ее словам, по прибытии борт встречали реанимационный автомобиль и карета скорой помощи, пассажиру оказали необходимую медицинскую помощь и оперативно госпитализировали в одну из городских больниц. По ее словам, в 14:20 мск самолет вылетел к месту назначения.
