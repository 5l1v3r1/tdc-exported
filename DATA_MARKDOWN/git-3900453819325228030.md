
# Опубликован первый постер дорамы «Люди с недостатками»

Published at: **2019-11-05T15:23:00+00:00**

Author: **gunyasha**

Original: [YesAsia](https://www.yesasia.ru/article/754225)

4 ноября создатели новой дорамы MBC «Люди с недостатками» (рабочее название) опубликовали первый постер проекта с главными героями.
Смотрите также: Ан Джэ Хён рассказал о работе над дорамой «Люди с недостатками»
С сюжетом можно ознакомиться здесь. Постер представляет собой две половинки. На первой высокомерный богач-красавчик Ли КанУ (Ан Джэ Хён) орёт в уши удивленной Джу Соён (О Ён Со): «Давай встречаться!». На второй половинке уже девушка отвечает ему тем же: «Пошёл вон!». Парень съеживается и закрывает глаза от неожиданности.
Премьера дорамы состоится в конце ноября.
