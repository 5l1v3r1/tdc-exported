
# ตกใจ! 'ต้นหอม'โพสต์ภาพ'น้องปกป้อง'ตาบวมเป่ง

Published at: **2019-11-04T05:02:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/entertainment/739891)


        ต้นหอม-ศกุนตลา โพสต์คลิปและภาพลูกชาย น้องปกป้อง ตาบวมเป่ง ด้านชาวเน็ตแห่ห่วงและส่งกำลังใจให้มากมาย
      
ทำเอาแฟนๆเป็นห่วงและต้องส่งกำลังใจให้ น้องปกป้อง ลูกชายของนักแสดงและดีเจสาว ต้นหอม-ศกุนตลา เทียนไพโรจน์ กันแบบรัวๆเลยทีเดียว หลังสาวต้นหอมโพสต์คลิปที่น้องปกป้องมีตาบวมแดงเป่งทั้งสองข้าง โดยบนตัวยังมีจุดสีแดงอีกหลายจุดเหมือนแพ้อะไรสักอย่างหนึ่ง โดยสาวต้นหอมเขียนแคปชั่นว่า “หูยยย แพ้หนักมากอ่า... อาจจะมาจากอาหาร อากาศคุณหมอเขาว่างั้น เขามีแต่ภูมิแพ้กรุงเทพ XXนี่ภูมิแพ้พิษณุโลก #นู๋หมานู๋แมว @pokpong_bigbird” พอสาวต้นหอมลงรูปไป ก็มีชาวเน็ตเข้ามาถามมากมายว่าน้องปกป้องเป็นอะไร ไปหาหรือยังจำนวนมาก
ล่าสุดสาวต้นหอมก็ได้อัพเดทรูปของน้องปกป้องอีกครั้ง พร้อมแคปชั่นว่า “โถดวงใจของแม่ ตายังบวมอยู่เลย...”จากนั้นก็มีแฟนๆเข้ามาคอมเมนต์ให้กำลังใจน้องปกป้อง แล้วขอให้หายไวๆมากมาย
