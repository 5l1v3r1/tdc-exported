
# Американка створила машину для людей, яким не вистачає обіймів

Published at: **2019-11-01T05:45:50+00:00**

Author: ****

Original: [ГЛАВКОМ](https://glavcom.ua/world/hitech/amerikanka-stvorila-mashinu-dlya-lyudey-yakim-ne-vistachaje-obiymiv-636727.html)

Американська художниця Люсі Макрей створила пристрій для людей, яким не вистачає обіймів в життя.
Про це пишуть Коментарі.
Зазначається, що машина для обіймів Compression Carpet - це відповідь на безперервну оцифровку нашого повсякденного життя.
