
# 李興文兒子假扮同志　當街騷擾熟女「我可以親你嗎？」

Published at: **2019-11-01T09:40:00+00:00**

Author: **娛樂組**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191101ent018/)

李興文兒子李堉睿為了想紅，使出各種屁孩手法拍攝YouTube影片，這回終於因潛入IKEA裸體DIY遭逮，過去拍片的各種黑歷史也一一被掀出，他曾假扮同志，在路上騙吻熟女，被批評消費同志。
李堉睿早在兩年前，就以「社會實驗」的名義，在路上到處找女生說，「我是GAY，可以親妳嗎？」想當然幾乎所有女生，都一眼看穿這是個膚淺的搭訕手法，狠狠拒絕了，更不少是直接嚇到，還有熟女說：「我一眼就知道你不是，既然是同志，應該要找男生吧。」
雖然李堉睿聲明：「本人對GAY並沒有有任何的偏見，也支持所有人都能有戀愛的自由。」但依舊遭到不少網友撻伐，有人認為這樣做不只歧視同志，也是騷擾女性，相當不得體。有網友留言批評「就只是不斷騷擾別人的影片，跟你是不是gay根本無關」
