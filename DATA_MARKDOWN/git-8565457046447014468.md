
# Viernes con cielo nublado y máxima de 21°C

Published at: **2019-11-01T12:24:00+00:00**

Author: ****

Original: [El Patagónico](https://www.elpatagonico.com/viernes-cielo-nublado-y-maxima-21c-n5063218)


        Hoy la jornada permanecerá con cielo mayormente nublado y se prevé una temperatura máxima de 21°C. Se registrarán vientos leves del oeste. ¿Cómo estará el tiempo durante el fin de semana?
      
Según lo anuncia el Servicio Meteorológico Nacional, mañana sábado el cielo estará ligeramente nublado y la temperatura oscilará entre 7°C y 22°C.
Para el domingo el cielo permanecerá mayormente nublado y la temperatura rondará entre 10°C y 23°C.
El lunes el cielo estará parcialmente nublado y la temperatura oscilará entre 12°C y 22°C.
