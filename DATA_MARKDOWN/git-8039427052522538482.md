
# జనసేన లాంగ్‌ మార్చ్‌లో ప్రకార్డుల ప్రదర్శన

అమరావతి: ఇసుక కొరత పై జనసేన పార్టీ నిర్వహిస్తున్న లాంగ్‌మార్చ్‌లో వివిధ రకాల నినాదాలతో ప్లకార్డులను ప్రదర్శించారు. ఇసుక బ్లాక్‌ మార్కెట్‌ను అరికట్టాలని, అన్నిరీచ్‌లు తెరిచి ఇసుకను అందుబాటులోకి తీసుకువాలన్న డిమాండ్‌తో ప్లకార్డులు ప్రదర్శించారు. ప్రభుత్వ నిర్లక్ష్యం, చేతకానితనం కార్మికులను బలితీసుకుంటోందని ఆరోపించారు. ప్రభుత్వం చేసేది ఇసుక వారోత్సవాలు కాదు, వైసీపీ నాయకుల ఇసుక దోపిడీ వారోత్సవాలు అంటూ పెద్దయెత్తున ప్లకార్డులు ప్రదర్శించారు.
