
# 女子７８キロ級、梅木が２年連続３度目Ｖ…柔道講道館杯

Published at: **2019-11-02T10:44:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/etc/20191102-OYT1T50246/)

　柔道・講道館杯全日本体重別選手権第１日（２日・千葉ポートアリーナ＝読売新聞社など後援）――来年の東京五輪代表１次選考会を兼ねて行われ、女子７８キロ級は２０１６年リオデジャネイロ五輪代表の梅木真美（ＡＬＳＯＫ）が２年連続３度目の優勝。７０キロ級は田中志歩（環太平洋大）が連覇を果たした。男子７３キロ級は原田健士（日体大）が決勝で海老沼匡（パーク２４）を破って初優勝した。
