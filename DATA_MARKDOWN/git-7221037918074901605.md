
# Hazal Kaya ile Ali Atay’ın Zekeriyaköy’deki evi yandı!

Original: [medya faresi](http://www.medyafaresi.com/haber/hazal-kaya-ile-ali-atayin-zekeriyakoydeki-evi-yandi/926237)

Talihsiz olayın ardından güzel oyuncu, annesinin evinde kalmaya başladı.
