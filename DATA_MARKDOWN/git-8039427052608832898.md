
# బోటు ప్రమాదాల నివారణపై జగన్‌ సమీక్ష

Published at: **2019-11-06T20:26:05+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=948062)

ADVT
అమరావతి: బోటు ప్రమాదాల నివారణకు తీసుకోవాల్సిన చర్యలపై సీఎం జగన్‌ సమీక్ష నిర్వహించారు. ఎమ్మార్వోల ఆధ్వర్యంలో 8 చోట్ల కంట్రోల్‌ రూమ్‌లు ఏర్పాట్లు చేయాలని అధికారులకు సీఎం ఆదేశించారు. ప్రతి కంట్రోల్‌ రూమ్‌లో 13 మంది సిబ్బంది ఉండాలని సూచించారు. ఈ నెల 21న కంట్రోల్‌ రూమ్‌ల నిర్మాణానికి శంకుస్థాపన చేస్తామని వెల్లడించారు. బోట్లలో ఎట్టి పరిస్థితుల్లో లిక్కర్‌ వినియోగం ఉండ కూడదని తెలిపారు. అలాగే సిబ్బందికి బ్రీత్‌ అనలైజర్‌ పరీక్షలు నిర్వహించాలని చెప్పారు. బోట్లకు జీపీఎస్‌ పెట్టాలన్నారు. అలాగే టికెట్లు ఇచ్చే అధికారం కంట్రోల్‌ రూమ్‌లదేనన్నారు.
