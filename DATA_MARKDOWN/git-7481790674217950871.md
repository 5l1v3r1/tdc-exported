
# ＮＹダウ、２日ぶり最高値更新…２万７６７４ドル

Published at: **2019-11-07T22:12:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/economy/20191108-OYT1T50147/)

　【ワシントン＝山内竜介】７日のニューヨーク株式市場で、ダウ平均株価（３０種）は２日ぶりに最高値を更新した。終値は前日比１８２・２４ドル高の２万７６７４・８０ドルだった。
　情報技術（ＩＴ）企業の銘柄が多いナスダック店頭市場の総合指数の終値は、２３・８９ポイント高の８４３４・５２だった。
