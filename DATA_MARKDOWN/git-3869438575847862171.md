
# В Мурманск с гастролями прибудет Костромской театр кукол

Published at: **2019-11-03T00:00:00+00:00**

Author: ****

Original: [Мурманский Вестник](https://www.mvestnik.ru/newslent/v-murmansk-s-gastrolyami-pribudet-kostromskoj-teatr-kukol/)

С 7 по 10 ноября в Мурманском областном театре кукол будут принимать своих коллег из Костромы. В столицу Заполярья прибудет один из старейших театров кукол России и неоднократный лауреат премии «Золотая маска». Для деток постарше костромичи привезут спектакли «Не Ёжик» и «Котёнок по имени Гав». А малыши увидят постановку «Чудо-зонтик».
