
# Rugby : les All Blacks se contentent du bronze

Published at: **2019-11-01T13:49:35+00:00**

Author: **Olivier Peguy**

Original: [euronews](https://fr.euronews.com/2019/11/01/rugby-les-all-blacks-se-contentent-du-bronze)

Les All Blacks ont décroché ce vendredi un lot de consolation au Mondial de Rugby. Ils ont remporté la "petite finale" face au Pays de Galles.
Une large victoire 40-17, avec notamment six essais contre deux pour les Gallois.
