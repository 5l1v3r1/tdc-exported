
# Telegram для Android обновился до версии 5.12.1. Что нового?

Published at: **2019-11-03T15:45:44+00:00**

Author: **Kirill Sergeev**

Original: [Код Дурова](https://kod.ru/telegram-5-12-1-android/)

Состоялся релиз новой версии Telegram для Android с индексом 5.12.1. В нем имеется несколько полезных изменений, о которых мы сейчас вам расскажем.
В новой версии появился улучшенный плеер с возможностью слушать подкасты и песни длиннее 20 минут со скоростью 2x, а также продолжать воспроизведение с того места, где вы остановились в прошлый раз.
Появился поиск элементов при отправке файлов и музыки.
При отправке контента из других приложений теперь можно выбирать сразу несколько адресатов по долгому удержанию на нужном контакте.
Все чаты их архива теперь помечаются прочитанными в контекстном меню по долгому нажатию на архив.
