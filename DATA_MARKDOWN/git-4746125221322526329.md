
# 深田恭子、濡れ肌あらわな谷間&太ももショット披露「エロい、かわいすぎ」「完璧だ」

Published at: **2019-11-01T11:59:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147801/full/)

　女優の深田恭子が10月31日、自身のインスタグラムを更新。濡れ肌あらわな色っぽい写真を投稿した。 　深田は「2020年壁掛けカレンダーが出来あがりました！」と、濡れ肌＆髪に、白のタンクトップで谷間や太ももなどを大胆に披露した表紙カットを公開。「お部屋に飾るにはとっても大きな物だし、それを飾っていただけるなんて心から感謝なので、今までとは違う感じの色々なバリエーションで撮って頂きました」と、写真2枚目にはお団子頭でピンクのオフショルダーワンピースを着た、美脚ショットを披露した。 　「どうかお手に取って頂けたら幸いです」のコメントともに、ハッシュタグで「＃皆様カレンダーは卓上派ですか　＃壁掛けしかなくてすみません　＃1年間あなたのお部屋にいさせてください」と茶目っ気たっぷりにアピールしている。 　この投稿にファンから「きゃぁーーーッ」「めっちゃ色っぽくて可愛い〜」「とってもセクシーです」「エロい、かわいすぎ」「な、な、な、な、な、な、なんて美しいんでしょうか」「ボディラインも憧れます!! コメントも可愛すぎる。完璧だ」「来年も再来年も部屋にいて下さい」など、絶賛の声が寄せられている。
