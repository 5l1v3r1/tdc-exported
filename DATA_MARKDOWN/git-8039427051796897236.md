
# తహసీల్దార్ హత్య దురదృష్టకరం: హరీశ్‌రావు

Published at: **2019-11-05T11:18:53+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=947044)

ADVT
హైదరాబాద్: తహసీల్దార్ విజయారెడ్డి హత్య దురదృష్టకరమని మంత్రి హరీష్‌రావు అన్నారు. ఘటనను తీవ్రంగా ఖండిస్తున్నామని చెప్పారు. ఇలాంటి ఘటనలు పునరావృతం కాకుండా చర్యలు తీసుకుంటామని పేర్కొన్నారు. బాధిత కుటుంబాన్ని ప్రభుత్వం తరపున ఆదుకుంటామన్నారు.
