
# 載620人法國列車與汽車相撞 釀2死3傷

Published at: **2019-11-05T05:11:35+00:00**

Author: ****

Original: [NTD 新唐人電視台](https://www.ntdtv.com/b5/2019/11/05/a102700333.html)

【新唐人北京時間2019年11月05日訊】法國東部安省托西亞地區(Tossiat)一列火車3日(週日)與一輛汽車相撞，造成2人死亡，3人受傷，警方仍在調查事發原因。
事發於3日下午5時左右，一列載着620名乘客的火車，從貝爾福(Belfort)開往里昂(Lyon)，在一處平交道與一輛汽車相撞。
汽車遭到撞擊後嚴重損毀，車上一名57歲女子和一名69歲男子喪生，火車上則有3人受輕傷。
意外導致當地鐵路交通停頓，車上乘客被困車廂超過3小時，約至晚上9時左右才離開。
