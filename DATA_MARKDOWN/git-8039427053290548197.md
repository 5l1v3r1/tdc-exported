
# జగన్ పాలన గాడి తప్పుతోంది: వేదవ్యాస్

Published at: **2019-11-06T12:23:42+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=947938)

ADVT
విజయవాడ: ఆంధ్రప్రదేశ్ రాష్ట్రంలో జగన్ పాలన గాడి తప్పుతోందని మాజీ డిప్యూటీ స్పీకర్ బూరగడ్డ వేదవ్యాస్ అన్నారు. బుధవారం ఆయన మీడియాతో మాట్లాడుతూ జగన్ ముఖ్యమంత్రిగా ప్రమాణస్వీకారం చేసిన దగ్గర నుండి తప్పుటడుగులు వేస్తున్నారని విమర్శించారు. ప్రజలకు ఉపయోగ పడే అనేక విషయాల్లో ఆయన ఎటువంటి నిర్ణయాలు తీసుకోవడం లేదన్నారు. రాష్ట్రంలో పరిస్థితి మరీ అద్వానంగా ఉందన్నారు.
 చీఫ్ సెక్రటరీ షోకాజ్ నోటీసులు ఇవ్వడం చాలా ఆశ్చర్యంగా ఉందని వేదవ్యాస్ అన్నారు. ఇది పరిపాలనలో చాలా పరిణామాలకు దారితీస్తుందని అభిప్రాయం వ్యక్తం చేశారు. అజయ్ కలం వంటి సీనియర్ వ్యక్తి ఉన్నప్పటికీ రాష్ట్రంలో ఇలాంటి పరిపాలన జరుగుతుందని..ఆయన సీఎంకు మంచి సలహాలు ఎందుకు ఇవ్వలేకపోతున్నారో అర్ధం కావడం లేదన్నారు. ఇప్పటికైనా అజయ్ కలం ముందుండి జగన్‌కు మంచి సలహాలు ఇవ్వాలని వేదవ్యాస్ సూచించారు.
