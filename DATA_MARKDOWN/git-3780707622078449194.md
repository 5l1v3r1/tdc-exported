
# Новый способ оплаты по QR-коду через Систему быстрых платежей введут в России в 2020 году

Published at: **2019-11-07T13:17:00+00:00**

Author: **Егор Самсонов**

Original: [ФедералПресс](https://fedpress.ru/news/russia/finance/2360422)


        МОСКВА, 7 ноября, ФедералПресс. С 2020 года по всей стране начнет действовать новый способ оплаты через QR-код по Системе быстрых платежей. Об этом заявили в ЦБ.
      
Глава Центрального банка Российской Федерации Эльвира Набиуллина анонсировала ввод новой системы оплаты через QR-код по Системе быстрых платежей.
Данный способ запустят по всей стране в 2020 году. На данный момент эту технологию уже тестируют несколько торговых сетей, передает ТАСС.
Дополнительно ожидается сниженная комиссия для предпринимателей, которые будут использовать новый вид оплаты, а не карточный эквайринг, сообщают «Новые известия».
Ранее «ФедералПресс» писал о способах оплаты в России. Российская молодежь готова полностью отказаться от наличных.
