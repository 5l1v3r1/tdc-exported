
# 山本彩、ファンクラブライブで来年のホールツアー全日程発表

Published at: **2019-11-05T21:00:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148081/full/)

　シンガー・ソングライター山本彩が5日、『山本彩ファンクラブLIVE「Sayaka Yamamoto Congregation 2019」』の東京初日公演で、来年開催する全国ホールツアーの詳細を発表した。 　2016年のZeppツアー、17年のホールツアー、2月から6月まで行われたライブハウスツアーに続く、自身4度目となるソロツアーとなる今回は、来年2月22日の千葉公演を皮切りに4月30日の東京公演まで全国17ヶ所20公演を開催する。 　この発表とあわせて、きょう5日午後9時よりファンクラブ先行が開始となった。■2020年山本彩全国ホールツアー（タイトル未定）▽2020年2月22日（土）：千葉・市川市文化会館2月24日（月・祝）：京都・京都ロームシアター2月26日（水）：東京・NHKホール3月1日（日）：香川・レクザムホール3月8日（日）：兵庫・神戸国際会館3月13日（金）：石川・金沢市文化ホール3月18日（水）：福岡・福岡市民会館3月20日（金・祝）：熊本・熊本県立劇場3月28日（土）：宮城・トークネットホール仙台3月29日（日）：岩手・盛岡市民文化ホール4月3日（金）：神奈川・カルッツ川崎4月5日（日）：広島・上野学園ホール4月8日（水）：大阪・フェスティバルホール4月9日（木）：大阪・フェスティバルホール4月12日（日）：長野・長野市芸術館4月17日（金）：北海道・カナモトホール4月22日（水）：愛知・日本特殊陶業市民会館フォレストホール4月23日（木）：愛知・日本特殊陶業市民会館フォレストホール4月29日（水・祝）：東京・LINE CUBE SHIBUYA4月30日（木）：東京・LINE CUBE SHIBUYA
