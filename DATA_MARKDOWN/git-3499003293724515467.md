
# Публичные слушания по реновации пройдут сегодня вечером в Люблино

Published at: **2019-11-05T17:47:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/yuvao/publichnye_slushaniya_po_renovatsii_proydut_segodnya_vecherom_v_lyublino/)

Как сообщалось ранее, мероприятие состоится в актовом зале школы №2121 по адресу: ул.Ставропольская, 20.В 19:00 начало, регистрация начнется в 18:00 мск, при себе нужно иметь паспорт.На слушаниях планируется обсудить новый облик кварталов А, Б, Е, Г, параметры будущей застройки, развитие транспортной инфраструктуры и реорганизацию улично-дорожной сети.Источник: http://uvaomos.news
