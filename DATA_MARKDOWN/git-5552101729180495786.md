
# 102秒激情不倫片流出　中村江莉香醉吻人夫事業全停擺

Published at: **2019-11-02T09:09:00+00:00**

Author: **娛樂組**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191102ent006/)

日本知名部落客中村江莉香2月參加真人秀《突然的Marriage》，與熊田勇太成為假想夫婦，兩人經過30天的婚姻生活後，假戲真作公開交往，她也順勢開設YouTube頻道，經營網路的事業。不過日前她卻在直播中失控，喝醉後和一旁的灰衣男子亂親一通，兩人還吻到出鏡，短短102秒內就親了6次。
從影片中可見到先由中村江莉香主動出擊，男方也非常享受，更數度說出充滿暗示性的話語：「妳這是什麼滑滑的，好滑呀」、「妳的腳好滑哦」，她則春心蕩漾捧著對方的臉，嘴對嘴狂親，讓男子嗨到直說「怎麼辦會愛上耶」。
事件爆發後，與中村江莉香以結婚為前提交往的熊田勇太，隔日火速開直播切割，表示兩人早已分手，而中村江莉香則是慘遭外界批評：「不會再支持了」、「看不起她」。
面對各界的謾罵聲， 中村江莉香發文公開謝罪，而且去見已婚男的老婆道歉，解釋當時開直播本來是要聊工作，不料喝醉後卻失去理智，同時她也將所有錯攬在自己身上，強調是自己主動。並希望外界別再肉搜、打擾對方。不過她的個人事業目前已全面暫停。
