
# Ώρες αγωνίας για τον Θοδωρή Μαραντίνη: Εσπευσμένα στο χειρουργείο για διπλή επέμβαση!

Published at: **2019-11-04T23:30:16+00:00**

Author: ****

Original: [europost.gr](https://europost.gr/ores-agonias-gia-ton-thodori-marantini-espeysmena-sto-cheiroyrgeio-gia-dipli-epemvasi/)

Χειρουργείο για τον τραγουδιστή των Onirama, Θοδωρή Μαραντίνη! Τι συνέβη;
Σύμφωνα με πληροφορίες του Athensmagazine.gr, ο Θοδωρής Μαραντίνης θα υποβληθεί σε δύο χειρουργικές επεμβάσεις, οι οποίες είναι ρουτίνας. Ναι, καλά διαβάσατε!
Πρόκειται για μια επέμβαση, η οποία έχει να κάνει με το διάφραγμα και μία ακόμη η οποία αφορά την γαστροοισοφαγική παλινδρόμηση οισοφάγου. Οι επεμβάσεις θα πραγματοποιηθούν σήμερα σε κλινική της Αθήνας. Εμείς να ευχηθούμε περαστικά και καλή ανάρρωση.
