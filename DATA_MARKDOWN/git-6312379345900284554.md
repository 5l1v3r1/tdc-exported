
# Ранивший парня из травмата в Москве оказался женским фитнес-тренером

Published at: **2019-11-03T09:59:18+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-03/ranivshiy-parnya-iz-travmata-v-moskve-okazalsya-zhenskim-fitnes-trenerom)

Правоохранители задержали подозревамого в нападении на парня на улице Кантемировской в Москве. Инцидент произошел вечером 2 ноября: мужчина ранил 27-летнего Мишу Мхитаряна из травматического оружия, после чего скрылся. Пострадавший в тяжелом состоянии госпитализирован. 
Подозреваемым оказался 36-летний тренер фитнес-клуба для женщин.
Против него возбужденго уголовное дело.
