
# Különös baleset történt Budapesten

Published at: **2019-11-03T17:45:00+00:00**

Author: **Infostart**

Original: [infostart](https://infostart.hu/belfold/2019/11/03/kulonos-baleset-tortent-budapesten)

A hatalmas eszköz egy gépkocsi utánfutójáról esett egy másik autóra a XXIII. kerületi Grassalkovich úton, a Hősök tere közelében. A különleges baleset helyszínén a fővárosi hivatásos tűzoltók dolgoztak - olvasható a katasztrofavedelem.hu oldalon.
