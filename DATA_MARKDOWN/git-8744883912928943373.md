
# Медведев предложил создать проекты для помощи афганской экономике

Published at: **2019-11-02T05:57:03+00:00**

Author: ****

Original: [Коммерсантъ](https://www.kommersant.ru/doc/4148032)

Премьер-министр России Дмитрий Медведев предложил подумать о создании проектов в рамках Шанхайской организации сотрудничества (ШОС), которые помогут экономике Афганистана выйти из кризиса.
«Нужно подумать о конкретных проектах, которые позволят афганской экономике выйти из кризиса, стать активным участником политических и экономических процессов в регионе»,— сказал господин Медведев на расширенном заседании Совета глав правительств государств—членов ШОС (цитата по ТАСС).
По его словам, эта идея уже обсуждалась с главой исполнительной власти страны Абдуллой Абдуллой.
Ранее господин Медведев заявил, что Москва внимательно наблюдает за тем, как развиваются события в Афганистане. Страна является важным партерном для России, сказал премьер-министр. Он добавил, что Афганистан имеет «ключевое значение для обеспечения безопасности и стабильности в регионе».
