
# Markakis, Flowers to return to Braves (source)

Published at: **2019-11-04T19:30:11+00:00**

Author: ****

Original: [MLB.com](https://www.mlb.com/news/nick-markakis-returns-to-braves)

The Braves have reached deals to bring back outfielder Nick Markakis and catcher Tyler Flowers, sources told MLB.com's Mark Bowman on Monday. The club hasn't confirmed the new contracts for the pair, which had team options for 2020 declined before new deals were struck.
