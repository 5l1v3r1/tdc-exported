
# ఆందోళన విరమించిన ఢిల్లీ పోలీసులు

Published at: **2019-11-05T20:36:39+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=947170)

ADVT
న్యూఢిల్లీ: తీస్ హజారీ కోర్టుల సముదాయంలో పోలీసులు, న్యాయవాదుల మధ్య జరిగిన ఘర్షణలపై ఢిల్లీ హైకోర్టు తీర్పునకు నిరసనగా ఆందోళన చేపట్టిన ఢిల్లీ పోలీసులు ఎట్టకేలకు శాంతించారు. పోలీసు సిబ్బంది డిమాండ్లన్నీ నెరవేర్చుతామంటూ ఉన్నతాధికారులు హామీ ఇవ్వడంతో 11 గంటల పాటు సాగిన ధర్నాను విరమించుకున్నారు. ఢిల్లీ హైకోర్టు తీర్పుపై రివ్యూ పిటిషన్ వేస్తామనీ.. పోలీసులు ఆందోళన విరమించి విధుల్లో చేరాలంటూ ఢిల్లీ ప్రత్యేక పోలీస్ కమిషనర్ (క్రైమ్) సతీష్ గొల్చా కోరారు. గాయపడిన పోలీసులకు రూ.25 వేల చొప్పున పరిహారం ఇస్తామని ఆయన హామీ ఇచ్చారు.
