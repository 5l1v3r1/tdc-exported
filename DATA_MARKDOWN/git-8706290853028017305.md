
# Δραματική διάσωση γυναίκας από φλεγόμενο κτίριο - ΒΙΝΤΕΟ

Published at: **2019-11-01T05:56:00+00:00**

Author: ****

Original: [enikos.gr](http://www.enikos.gr/international/681420/dramatiki-diasosi-gynaikas-apo-flegomeno-ktirio-video)

Πέντε αστυνομικοί δημιούργησαν ανθρώπινη σκάλα για να σώσουν μια ηλικιωμένη γυναίκα που είχε παγιδευτεί μέσα στο διαμέρισμά της που είχε πάρει φωτιά, στην πόλη Nanjing.
Έσωσαν την γυναίκα από τον πρώτο όροφο και άλλο ένα άτομο από διαμέρισμα του δευτέρου ορόφου. Ευτυχώς, η φωτιά σβήστηκε και δεν υπήρξαν τραυματίες.
