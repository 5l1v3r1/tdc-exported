
# Умер директор Оборонпромэкологии Хамзат Беков

Published at: **2019-11-02T15:30:59+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-02/umer-direktor-oboronpromekologii-hamzat-bekov)

Скончался директор Оборонпромэкологии Хамзат Беков.
По данным РЕН ТВ, он ушел из жизни в своем доме в поселке Внуково после продолжительной болезни. Ему было 64 года.
Отметим, ФГУП "Оборонпромэкология" совместно с научно-исследовательскими организациями и предприятиями выполняет работу по разработке, производству, хранению вооружения и утилизации военной и спецтехники для нужд Минобороны.
