
# U-17 жаҳон чемпионатида Австралия, Жанубий Корея ва Япониянинг хурсандчилиги

Published at: **2019-11-03T06:07:41+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/u-17-jahon-chempionatida-avstraliya-janubiy-koreya-va-yaponiyaning-xursandchiligi)

Австралия Нигерияни, Жанубий Корея Чилини, Япония эса Сенегални мағлуб этган ҳолда гуруҳдан чиқиш вазифасини уддалади. Қуйида Осиё жамоаларининг хурсандчилигидан фотолавҳаларни тақдим этамиз.
Эслатиб ўтамиз, бугун тунда Тошкент вақти билан 01:00да Тожикистон 3-турни ўтказади. Ён қўшниларимиз Аргентинага қарши баҳс олиб боришади. Ҳозирча Тожикистон ҳисобида 3 очко бор.
