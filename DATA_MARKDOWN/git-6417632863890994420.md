
# Бугун 17:00. Ўзбекистон U-19 – Ҳиндистон U-19

Published at: **2019-11-06T06:10:20+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/bugun-1700-ozbekiston-u-19-hindiston-u-19)

Ўзбекистон U-19 терма жамоаси саралашнинг “F” гуруҳидан ўрин олган ва гуруҳда Саудия Арабистони, Ҳиндистон ҳамда Афғонистон терма жамоалари билан баҳс олиб боради. U-19 Осиё Чемпионати-2020 саралаш баҳслари Саудия Арабистонидаги Хобар шаҳрида ўтказилади.
