
# Экс-посол Украины в США: НАТО — безальтернативный путь для страны, другие варианты закрыты

Published at: **2019-11-01T11:15:00+00:00**

Author: ****

Original: [НВ](https://nv.ua/ukraine/politics/nato-eto-bezalternativnyy-put-dlya-ukrainy-eks-posol-chalyy-50051022.html)

По его словам, другие варианты для государства закрыты.
«С точки зрения безопасности, у нас сейчас нет альтернативного пути в будущем для обеспечения безопасности Украины, кроме как членства в коллективной организации обороны и безопасности. Россия нам другие варианты закрыла своей агрессией, поэтому НАТО — это на сегодня безальтернативный путь для нас», — сообщил Чалый.

Напомним, генеральный секретарь НАТО Йенс Столтенберг 30−31 октября находился с официальным визитом в Украине. В четверг Столтенберг выступил в Верховной Раде и возглавил заседание Комиссии НАТО-Украина с участием президента Владимира Зеленского.
На заседании генсек Альянса заявил, что двери НАТО открыты для желающих вступить в него, в том числе и для Украины.
