
# В Иркутске начали устанавливать площадки для раздельного сбора мусора

Published at: **2019-11-01T00:00:00+00:00**

Author: **АСИ-Иркутск, Ольга Жаркова**

Original: [Агентство социальной информации](https://www.asi.org.ru/news/2019/11/01/irkutsk-kontejnery-razdelnyj-sbor-musora/)

Контейнерную площадку для раздельного сбора мусора обустроили в иркутском Академгородке.
Она появилась на площадке ТСЖ «Прогресс». На площадке установлены три цветных контейнера: зеленый – для стекла, оранжевый – для пластика, синий – для бумаги. Площадку организовала группа компаний «Прогресс», которая стала победителем конкурса «Позиция активна – разделяй селективно!». Конкурс провел региональный оператор РТ-НЭО. Подобные площадки появятся в других городах: Ангарске и Шелехове.
«Сохранение природы начинается с себя. Каждый человек может принять в этом посильное участие, например разделять мусор дома, — рассказала АСИ Елена Чубакова, президент ассоциации «Большая Байкальская тропа» (ББТ). — Сейчас в Иркутске есть несколько мест, куда можно сдать макулатуру, пластик, стекло. Но не у всех есть возможность каждую неделю сдавать раздельно мусор. Поэтому очень важно создавать условия возле дома».
В офисе ассоциации мусор собирают раздельно уже год и теперь ждут установки контейнеров, как в Академгородке. ББТ также обучает волонтеров правилам утилизации мусора.
