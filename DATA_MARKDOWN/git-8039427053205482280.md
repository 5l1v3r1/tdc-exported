
# ‘కేసీఆర్‌కు ధనదాహం, పదవి వ్యామోహం పెరిగింది’

Ph: 88865 47666ప్రవేశం ఉచితం ADVT
వరంగల్: రెండోసారి అధికారంలోకి వచ్చిన సీఎం కేసీఆర్‌కు ధనదాహం, పదవివ్యామోహం పెరిగిందని బీజేపీ అధికార ప్రతినిధి కృష్ణసాగర్ రావు విమర్శించారు. శుక్రవారం హన్మకొండ బీజేపీ కార్యాలయంలో ఆయన మాట్లాడుతూ ముఖ్యమంత్రి
ఫాసిస్టు నిర్ణయాలతో ధనిక రాష్ట్రాన్ని దివాళా రాష్ట్రంగా మార్చారన్నారు. తన జేబులు నింపుకోవడం తప్ప.. సంక్షేమ పథకాలకు కూడా నిధులు ఇవ్వలేని దుస్థితి ఏర్పడిందని విమర్శించారు. ఆర్టీసీ కార్మికుల సమ్మె 30 రోజుల నుండి నడుస్తోందని, ప్రజలు రవాణా వ్యవస్థ లేక తీవ్ర ఇబ్బందులు పడుతున్నారని కృష్ణసాగర్ అన్నారు.
 
కార్మికుల సమస్యలు న్యాయబద్ధమైనవి కాబట్టే బీజేపీ మద్దుతు ఇస్తుందని కృష్ణసాగర్ రావు అన్నారు. ఆర్టీసీ పూర్తి స్థాయి ఎండిని నియమించకుండా కేసీఆర్ నియంతలా వ్యవహరిస్తున్నారని విమర్శించారు. కార్మికులకు జీతాలు ఇవ్వకుండా 50 వేల మంది కార్మికుల కుటుంబాలను రోడ్డుపై పడేసి చోద్యం చూస్తున్నారని, ఆర్టీసీ లెక్కలపై గవర్నర్‌ను కోరుతామని, ఆర్టీసీ ఆడిట్‌పై లెక్కులు అడిగే అధికారం గవర్నర్‌కు ఉందన్నారు. రాష్ట్రంలో జరిగే పరిస్థితులుపై రాష్ట్రపతికి ఫిర్యాదు చేస్తామని కృష్ణసాగర్ రావు పేర్కొన్నారు.
