
# Видео: девушка-боец MMA нокаутировала соперницу подножкой

Published at: **2019-11-02T05:18:02+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-02/video-devushka-boec-mma-nokautirovala-sopernicu-podnozhkoy)

Девушка-боец MMA из Австралии Челси Хаккет нокаутировал свою соотечественницу Николь Шепешвари на турнире Eternal MMA 49.
В начале второго раунда Хаккет смогла поймать ногу соперницы, после чего она сделала подсечку. Ее противница жестко рухнула на канвас, рефери остановил бой, присудив Челси победу нокаутом.
Она стала второй в профессиональной карьере для Хаккет. Для Шепешвари это второе поражение при двух победах.
