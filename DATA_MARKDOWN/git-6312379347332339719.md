
# Иностранные студенты устроили поножовщину около РУДН

Published at: **2019-11-03T10:20:05+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-03/inostrannye-studenty-ustroili-ponozhovshchinu-okolo-rudn)

Студенты из Нигерии устроили поножовщину недалеко от здания РУДН в Москве. Инцидент произошел 1 ноября. 
По данным РЕН ТВ, между двумя парнями произошел конфликт. Один из них достал нож и несколько раз ударил оппонента. Пострадавший с ранениями в область груди, спины и плеча госпитализирован. Нападавшего задержали. 
Против нападавшего возбуждено уголовное дело.
