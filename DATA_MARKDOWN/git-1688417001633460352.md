
# SpaceX протестировала парашют для Crew Dragon

Published at: **2019-11-04T10:33:00+00:00**

Author: **Корреспондент.net**

Original: [Корреспондент.net](https://korrespondent.net/tech/space/4156500-SpaceX-protestyrovala-parashuit-dlia-Crew-Dragon)

Компания SpaceX Илона Маска провела подряд 13 успешных тестов парашюта для своего нового многоразового космического корабля Crew Dragon. Об этом компания накануне сообщила в Twitter.
Команда SpaceX осуществила подряд 13 успешных тестов парашюта обновленной версии Mark 3 для Crew Dragon, – говорится в сообщении.
По данным SpaceX, тесты показали, что у парашютной системы есть возможность безопасно доставить космический корабль на поверхность в случае неоткрытия одного из четырех основных парашютов.
