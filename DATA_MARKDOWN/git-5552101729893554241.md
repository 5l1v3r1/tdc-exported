
# 【豪門爆姦情3】不倫六年每週二固定約會　千金小三臉書放閃露餡

Published at: **2019-11-05T21:58:00+00:00**

Author: **娛樂組, 攝影組**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191105ent019/)

根據讀者的爆料，元祖食品少東張劭緯從2013年，就開始與大黑松小倆口的千金邱威樺搞外遇。兩人6年來暗偷款曲，不僅一起親暱打球、看診、到陽明山玩樂，還多次出國至日本遊玩、到夜店玩耍，甚至到旅館開房、在浴室自拍，兩人甚至於2014年在晶華酒店買情侶對戒。
