
# В КЧР после смерти в горах пропавшего мужчины собирают спасательный отряд

Published at: **2019-11-03T04:19:00+00:00**

Author: **Ася Семенченко**

Original: [Регион Online](https://news-r.ru/news/karachay_cherkess_republic/379436/)


        Активисты набирают по пять-шесть человек с каждого района республики, чтобы в будущем они могли организовывать поиски людей с обычными добровольцами
      
Черкесск, 3 ноября. В Карачаево-Черкесской Республике паблик "ЧП КЧР" в "Инстаграм" решил собрать свой спасательный отряд с волонтерами-поисковиками. Такое решение администраторы приняли после того, как включились в поиск пропавшего недавно Рашида Байрамкулова - мужчину с больным сердцем не успели спасти, он замерз в горах и скончался.
- Дорогие друзья, в связи с тем, что в нашей республике очень часто начали пропадать люди, возникла идея создать свой спасательный отряд добровольцев, тем более, что у нас есть друзья с "Лиза Алерт", которые окажут нам всю необходимую помощь и будут проводить обучение. На данный момент нам нужны по пять-шесть человек с каждого района КЧР, чтобы при необходимости они грамотно организовывали поиски с обычными добровольцами.
