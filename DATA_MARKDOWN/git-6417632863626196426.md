
# Дани Себальос жароҳат сабаб 3 ҳафтани ўтказиб юборади

Published at: **2019-11-06T22:30:00+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/dani-sebalos-jarohat-sabab-3-haftani-otkazib-yuboradi)

Лондоннинг "Арсенал" клубида ижара асосида ҳаракатланаётган ярим ҳимоячи Дани Себальос "Виктория Гимарайнш"(1:1) клубига қарши учрашувда жароҳат орттириб олди. 23 ёшли футболчи учрашувнинг 54-дақиқасида алмаштирилди. Себальос яқин 3 ҳафтани ўтказиб юбориши кутилмоқда. Испаниялик футболчининг айнан қачон сафга қайтиши чуқурлаштирилган тиббий кўрик хулосалари чиққач аниқ бўлади.
