
# 瑛太 宮崎あおいとようやく結婚！？ドラマで恋人同士役「ようやくたどり着けた」

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/01/0012838357.shtml)

　女優の宮崎あおい（３３）がこのほど、都内で行われたＴＢＳ系主演ドラマ「あしたの家族」（来年１月放送）の会見に俳優の瑛太（３６）らと登壇した。宮崎にとっては、昨年１０月に第１子となる男児を出産して以来、初のドラマとなる。
　宮崎と瑛太は、２００５年公開の映画「好きだ、」と０８年放送のＮＨＫ大河ドラマ「篤姫」で共演。両作では瑛太が宮崎に思いを寄せながらも成就しない役を演じたが、今作では結婚目前の恋人同士という役どころだ。
　“３度目の正直”となった瑛太は「『好きだ、』も『篤姫』も追いかけて、でも、成就せず、今回はどうなるのかと思い、台本を開いてみたら、１５、１６年かけてあおいちゃんとようやく結婚するところまでたどり着きました。ありがとうございました」と感謝。宮崎は「瑛太君とは約１０年ぶりの共演。尊い時間だったなと感じました」と笑顔だった。
　結婚式当日に新郎に逃げられた過去を持つヒロイン（宮崎）が、家に恋人（瑛太）を連れてきたら、父親の元部下で現上司だった…という家族ドラマ。「渡る世間は鬼ばかり」などで知られ、本作を手がける石井ふく子プロデューサー（９３）は「家族の話しかできません。家族に魅せられてきました」と思いを語った。
