
# Зафиксировано снижение числа случаев заболевания ОРВИ и гриппом

Published at: **2019-11-06T16:00:00+00:00**

Author: **Инна Лутовинова**

Original: [МОЛНЕТ.RU](https://www.molnet.ru/mos/ru/health_and_social/o_717643)


        Эпидемиологическая ситуация по гриппу и ОРВИ и ход кампании по вакцинации против сезонного гриппа находятся на постоянном контроле управления Роспотребнадзора по Москве.
      
С 28 октября по 3 ноября в Москве отмечается спокойная эпидемиологическая ситуация по гриппу и ОРВИ, сообщает управление Роспотребнадзора по городу. Уровень заболеваемости низкий.
Более того, уже шестую неделю подряд отмечается снижение числа зарегистрированных случаев заболевания.
