
# Вышла бета Telegram 5.12.2 для iOS: что нового?

Published at: **2019-11-06T11:54:06+00:00**

Author: **Владислав Войтенко**

Original: [Код Дурова](https://kod.ru/telegram-5-12-2-beta-ios-nov-2019/)

Команда Telegram выпустила очередную бета-версию мессенджера для iOS с индексом 5.12.2. Рассказываем о доступных нововведениях.
В новой бета-версии Telegram для iOS разработчики улучшили функцию отложенных сообщений. Отныне пользователи могут отложить отправку сообщений не только на определённое время, но и до момента, когда собеседник зайдёт в Сеть.
К сожалению, если пользователь скрыл от других время последней активности, данная функция работать не будет.
Теперь в меню очистки кэша конкретного диалога появилась возможность переходить непосредственно в сами настройки. Также был переработан дизайн раздела «Использование памяти».
Возможность открывать ссылки прямо из мессенджера теперь обрела более гибкую настройку. Пользователи могут выбрать определённый браузер для открытия ссылок.
Для того, чтобы стать участником бета-тестирования Telegram для iOS, необходимо выполнить несколько простых действий.
— Установить приложение TestFlight и войти в него.
— Перейти по ссылке со смартфона или планшета.
— Нажать на кнопку «Открыть» в правом верхнем углу.
— Принять приглашение на тестирование.
— Скачать бета-обновление для Telegram.
