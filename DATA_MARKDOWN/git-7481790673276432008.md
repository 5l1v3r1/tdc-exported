
# 広島・長野「ふがいない」、ＦＡ権行使せず残留

Published at: **2019-11-01T10:41:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/npb/20191101-OYT1T50238/)

　ＦＡ権を保有している広島・長野が１日、鈴木球団本部長との交渉に臨み、権利を行使せず残留する意向を伝えた。交渉後、長野は「（広島でプレーを続けるのは）もちろん。（今季は）チームが勝てなかったことがすごく悔しいし、（自身も）ふがいないシーズンだった」と再起を誓った。鈴木本部長は「引き続き、うちでやってくれるという意思を持ってくれて、楽しみが増えた。レギュラーとして頑張ってほしい」と期待した。
