
# บลูเวฟ ฟอร์มโหดต้อนรองบ่อนยับลิ่ว16ทีมเอฟเอคัพ

Published at: **2019-11-02T11:56:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/sports/739692)


        โชว์ฟอร์มสมราคา "ฉลามพลังเพลิง" ไล่ถล่ม "ณัฐวุฒิ ใจใส" 14-2 ด้าน "กรมทางหลวง" ต้อน "สโมสรรถไฟ" 10-1 ทะลุสู่รอบ 16 ทีมสุดท้าย ศึกฟุตซอล เอฟเอ คัพ 2019 ได้สำเร็จ
      
ศึกฟุตซอล เอฟเอ คัพ 2019 ที่ศูนย์การค้าแฟร์ชั่นไอส์แลนด์ เมื่อวันที่ 2 พ.ย. ในรอบ 32 ทีมสุดท้าย "ฉลามพลังเพลิง" พีทีที บลูเวฟ ชลบุรี รองแชมป์ลีกและรองแชมป์เก่ารายการนี้ โชว์ฟอร์มยอดเยี่ยมถล่ม ทีมณัฐวุฒิ ใจใส 14-2 ผ่านเข้ารอบ 16 ทีมสุดท้ายตามความคาดหมาย ส่วนผลคู่อื่นๆ กรมทางหลวง ชนะ สโมสรรถไฟ 10-1, บางกอก ซิตี้ ชนะ กัสดาฟี่ 5-2, นครปฐม ชนะ วายเอฟเอ วอริกซ์ ศรีราชา 8-5, ฟุตซอลไทยอาร์มี่ ชนะจุดโทษ นนทบุรี 9-8
