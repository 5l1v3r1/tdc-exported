
# Хэмилтон стал шестикратным чемпионом Формулы-1

Published at: **2019-11-03T20:50:00+00:00**

Author: ****

Original: [Interfax.ru](https://www.sport-interfax.ru/682854)

Москва. 3 ноября. INTERFAX.RU - Британец Льюис Хэмилтон из команды Mercedes в шестой раз в карьере завоевал звание чемпиона мира по автогонкам класса "Формула-1".
Досрочный триумф в чемпионате он оформил по итогам прошедшей в воскресенье гонки в рамках Гран-при США, после которой он стал недосягаемым до ближайшего преследователя в лице напарника по Mercedes финна Валттери Боттаса.
