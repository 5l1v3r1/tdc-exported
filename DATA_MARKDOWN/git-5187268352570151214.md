
# Triunfo em português no Nordic Game Discovery

Published at: **2019-11-04T20:00:31+00:00**

Author: **Paulo Silva**

Original: [Pplware](https://pplware.sapo.pt/jogos/triunfo-em-portugues-no-nordic-game-discovery/)

Esta é mais uma prova de que o “que é nacional, é bom”. No desenvolvimento de videojogos em Portugal este é apenas mais um exemplo!
Chronos, jogo que o Pplware a href=”https://pplware.sapo.pt/jogos/chronos-e-o-poder-de-abrandar-o-tempo/” rel=”noopener” target=”_blank”>tem vindo a acompanhar com atenção, venceu a edição do Nordic Game Discovery Contest 2019.
Ainda falta muito para que o desenvolvimento de videojogos em Portugal atinja outra divisão. No entanto, os primeiros (e segundos passos) estão dados e com firmeza.
Ao longo dos últimos anos temos vindo a assistir a um número crescente de projetos criados pela massa cinzenta nacional. Esta é uma atitude de louvar, não só pela arte e engenho de todos os que envolvidos como também pela coragem, pois este não é, definitivamente, um caminho fácil.
