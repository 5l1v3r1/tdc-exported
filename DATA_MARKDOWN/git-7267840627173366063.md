
# СМИ: в НАТО отказались от встречи генсека с освобожденными украинскими моряками

Published at: **2019-11-02T09:33:00+00:00**

Author: ****

Original: [Главное™](https://glavnoe.ua/news/n357007852-smi-v-nato-otkazalis-ot-vstrechi-genseka-s-osvobozhdennymi-ukrainskimi-morjakami)

По информации ZN.UA, украинская сторона предлагала организовать в Одессе встречу генерального секретаря НАТО Йенса Столтенберга с освобожденными из России украинскими моряками, но в Альянсе отказали в этом.
Основным противником, по информации источников, оказалась Франция, не желающая раздражать Россию встречей, которая, по мнению Парижа, может "сорвать мирный процесс и негативно повлиять на организацию переговоров в нормандском формате".
