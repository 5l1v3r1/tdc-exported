
# 高庭驳回政府申请 Prestariang应声走高

Published at: **2019-11-04T03:53:23+00:00**

Author: **Wong Swee May**

Original: [The Edge Markets](https://www.theedgemarkets.com/article/%E9%AB%98%E5%BA%AD%E9%A9%B3%E5%9B%9E%E6%94%BF%E5%BA%9C%E7%94%B3%E8%AF%B7-prestariang%E5%BA%94%E5%A3%B0%E8%B5%B0%E9%AB%98)

（吉隆坡4日讯）Prestariang Bhd今早最高上扬3仙或5.83%，至54.5仙，因高庭上周驳回政府将Prestariang Skin私人有限公司的原诉传票转为传讯令状的申请。
Prestariang向大马交易所报备，高庭的裁决为法庭现在裁定政府应向Prestariang Skin支付多少赔偿金，以终止先前批准的35亿令吉特许协议铺路。
截至11时35分，该股报53仙，达3777万5300股成交，是第五大热门股。
