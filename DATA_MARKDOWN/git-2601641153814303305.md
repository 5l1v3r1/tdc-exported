
# «ВКонтакте» на один день запустит функцию для борьбы с оскорблениями в комментариях

Published at: **2019-11-04T10:00:35+00:00**

Author: **Альберт Хабибрахимов**

Original: [vc.ru](https://vc.ru/social/91030-vkontakte-na-odin-den-zapustit-funkciyu-dlya-borby-s-oskorbleniyami-v-kommentariyah)

Соцсеть будет советовать пользователям «не тратить время на агрессию».
«ВКонтакте» запустила экспериментальную функцию для борьбы с оскорблениями в комментариях. Она проработает в тестовом режиме один день, 4 ноября, рассказали vc.ru в соцсети.
Функция поможет защитить подростков, которые чаще других сталкиваются с издевательствами, и предотвратить травлю по национальному признаку, рассчитывают в соцсети.
Определять содержание комментариев будут алгоритмы. Если они обнаружат признаки оскорблений, автор получит рекомендацию отказаться от обидной реплики.
По результатам эксперимента «ВКонтакте» соберёт обратную связь, оценит эффективность функции и примет решение о её запуске на постоянной основе.
