
# WhatsApp libera uso de figurinhas na versão web e desktop

Published at: **2019-11-06T15:57:00+00:00**

Author: **Luiz Nogueira**

Original: [Olhar Digital](https://olhardigital.com.br/noticia/whatsapp-libera-uso-de-figurinhas-na-versao-web-e-desktop/92623)

Quem usa o WhatsApp pelo navegador tem quase todas as funções da versão para smartphones, porém, uma das principais reclamações dos usuários é a falta da opção de enviar as famosas figurinhas que viraram febre no aplicativo.
Mas o problema foi resolvido. A versão para navegadores e desktop do aplicativo de mensagens acaba de disponibilizar a opção do envio de figurinhas. É possível enviar "adesivos" criados pelo próprio usuário ou baixados em aplicativos especializados na disponibilização desse tipo de conteúdo. A informação vem do site WaBetaInfo.
Até então, os usuários poderiam apenas mandar os adesivos que eram padrão no WhatsApp, impossibilitando que os usuários mandassem suas criações fora do celular.
O recurso está sendo disponibilizado aos poucos aos usuários do aplicativo. A partir de agora, todas as figurinhas salvas pelo usuário ficam disponíveis para serem usadas em outras plartaformas.
Entretanto, a aba de figurinhas favoritas continua como exclusividade da versão para smartphones. Além disso, não é possível salvar pelo desktop ou web figurinhas mandadas pelos seus contatos. Essa opção também continua restrita às versões de Android e iOS do WhatsApp.
