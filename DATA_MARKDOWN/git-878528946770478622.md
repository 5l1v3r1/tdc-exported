
# 日本代表・具　悔し泣きした途中交代「すごく痛くて骨が動いてた」

Published at: **2019-11-04T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/general/rugbywc2019/2019/11/04/0012849092.shtml)

　ラグビー日本代表の具智元が４日放送の日本テレビ系「くりいむしちゅーのＴＨＥレジェンド２０１９」に出演。スコットランド戦で右肋骨を痛めて途中交代を命じられたときに男泣きした心境を語った。
　中学２年で韓国から来日した具。スコットランド戦の前半１４分に激しいタックルを受けて右の肋骨を痛めた。激痛に顔をゆがめ、脂汗を流しながらも試合出場を続行しようとしていたが、途中交代を告げられた。
　無念の表情で涙を流しながらベンチに下がる具にスタンドからは拍手が送られていた。
　司会のくりいむしちゅー・上田晋也から「あの場面は続けられる状態じゃなかった？」と問われた具は「すごく痛くて、骨が動いてた。すごい痛みがあって代えられた」と激痛があったことを打ち明けた。
　涙の理由を尋ねられると「日本に来たときから一番の夢がＷ杯出場。Ｗ杯、もう出られないと思った」と悔しくてたまらなかった思いを吐露した。
　具は次戦の南アフリカ戦では患部をテーピングで固定して最後まで出場。現在も「まだ治ってない」と完治には至ってないという。
