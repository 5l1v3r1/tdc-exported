
# В сыктывкарском парке появится место для кормления грудью

Published at: **2019-11-06T04:39:00+00:00**

Author: ****

Original: [БНК](https://www.bnkomi.ru/data/news/102355/?utm_source=telegram.me&utm_medium=social&utm_campaign=roditelyam-na-zametkuv-stolitse-komi-e)

Общественная организация «Проект 1780» выиграла президентский грант на создание в Сыктывкаре сквера для молодых мам. Планируется открыть его в Кировском парке.
«Проект 1780» стал победителем второго конкурса президентских грантов 2019 года. Они получили свыше 497 тыс. рублей на инициативу «Умный город», одна из частей которых — создание сквера для молодых мам в столице республики.
На брифинге с журналистами и.о. мэра Сыктывкара Наталья Хозяинова рассказала, что активисты уже связались с администрацией по вопросу реализации проекта. В качестве места для сквера определен участок напротив дома №24 по улице Кирова в Кировском парке. Сотрудники мэрии уже осмотрели границы территории. Как отметила Наталья Хозяинова, они готовы помогать активистам по всем вопросам.
Как следует из заявки проекта, в сквере для мам будут, помимо традиционных скамеек и дорожек, пеленальный столик и место для кормления грудью, скрытые от ветра и глаз окружающих. На участке хотят разместить информацию о принципах воспитания великих педагогов, например, Марии Монтессори, Януша Корчака.
