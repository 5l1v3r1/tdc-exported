
# 愷樂爆炸上圍差點撐破緊身背心　粉絲狂喊：姐穿這樣太扯

Published at: **2019-11-07T04:32:00+00:00**

Author: **張詠淇**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191107ent009/)

愷樂自從轉型成性感辣妹後，經常公開大曬好身材，6日她在IG上PO出表演照，穿著緊身背心及超短熱褲在舞台上演出，爆炸的上圍快把背心撐破，隱約露出雪白雙峰，連黑色底褲也從邊邊露出，讓粉絲直噴鼻血。
愷樂穿著黑色緊身背心，身材曲線前凸後翹，一眼就看到她的雄偉上圍，臀部的飽滿線條也一覽無遺，照片曝光後，引起粉絲直呼說，「這樣不行啦！！！喂～簡爸爸」、「姐穿這樣太扯」。愷樂過去曾分享過養胸秘訣，從小就會吃豬腳和青木瓜的習慣，還會用「胸膜」美胸，穿對內衣對胸部形狀也很重要，而且一定要把副乳撥到正確位置才不會下垂。
為了維持好身材，愷樂還有自己的減重食譜，「早餐通常是無糖豆漿、飯糰，中午吃攝影棚便當，晚餐只吃青菜和蛋白質，宵夜則會吃蘋果或優酪乳。
