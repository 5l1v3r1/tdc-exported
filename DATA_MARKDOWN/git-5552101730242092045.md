
# 【獨家】【金主與愛的距離】女版小玉尬炫富網紅　美麗妄娜肉貼金主烏鴉

Published at: **2019-11-05T08:58:00+00:00**

Author: **娛樂組, 攝影組**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191104ent006/)

有40.6萬人訂閱的YouTuber美麗妄娜，曾因買了50包吸管拍影片，遭批不環保、浪費，並被封為「女版小玉」，雖然行事有爭議，但外型亮麗的她依舊受到許多男性粉絲追捧，感情狀態神祕，最近卻被本刊直擊，早就尬上經紀公司老闆，也是知名YouTuber的烏鴉。
