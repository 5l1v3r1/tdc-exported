
# تحطم طائرة فى ولاية كاليفورنيا.. فيديو

Published at: **2019-11-07T10:08:00+00:00**

Author: **البهي عمرو**

Original: [صدى البلد](https://www.elbalad.news/4049478)

البهي عمرو عرضت قناة إكسترا نيوز فى خبر عاجل عبر شريط العاجل يفيد بـ تحطم طائرة فى ولاية كاليفورنيا، بمدينة سان بيرناردينو، بالولايات المتحدة الأمريكية.ولم تذكر القناة أي تفاصيل حول الحادث ما إذا كان هناك وفيات او اصابات.
