
# Μας κούρασε η Παιδεία! Ετοιμάζονται να σνομπάρουν τα τετράμηνα

Published at: **2019-11-03T09:29:00+00:00**

Author: **SigmaLive**

Original: [SIGMALIVE](https://www.sigmalive.com/news/local/595496/mas-kourase-i-paideia-etoimazontai-na-snomparoun-ta-tetramina)

Καθηγητές και μαθητές ψάχνουν τώρα τρόπους να αποφύγουν τις εξετάσεις των τετραμήνων. Μετά το δημοψήφισμα της ΟΕΛΜΕΚ στις 11 Νοεμβρίου, ακολουθούν αποφάσεις της ΠΣΕΜ. «Στην περίπτωση, που δεν αποφασίσουν να κατέλθουν σε απεργίες, θα λάβουμε εμείς τα μέτρα μας», τονίζει στη «Σ» ο πρόεδρος των μαθητών. «Οφείλουμε να σεβαστούμε τη νομοθεσία», επισημαίνουν οι γονείς. Οι εξετάσεις θα γίνουν όπως προνοεί η νομοθεσία, διαμηνύει το Υπουργείο Παιδείας. Διαβάστε περισσότερα στη «Σημερινή» που κυκλοφορεί.
