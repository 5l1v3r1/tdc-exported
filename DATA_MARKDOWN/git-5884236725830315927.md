
# Появились кадры с места ДТП с попавшей под колеса пенсионеркой на набережной Грибоедова

Published at: **2019-11-06T06:12:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/740214-poyavilis-kadry-s-mesta-dtp-s-popavshei-pod-kolesa-pensionerkoi-na-naberezhnoi-griboedova)


        Вечером вторника водитель Volkswagen сбил женщину, переходившую по нерегулируемому пешеходному переходу. В результате ДТП пешехода в тяжелом состоянии госпитализировали. В Сети появились кадры с места аварии.
      
Минувшим вечером вторника, 5 ноября, на набережной Канала Грибоедова, около дома № 117, в Адмиралтейском районе произошел дорожный инцидент, который закончился больницей. В Сети появились кадры с места случившегося.
