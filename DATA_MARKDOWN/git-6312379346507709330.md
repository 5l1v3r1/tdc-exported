
# В Сети обсуждают видео с жесткой дракой в торговом центре

Published at: **2019-11-04T13:09:25+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-04/v-seti-obsuzhdayut-video-s-zhestkoy-drakoy-v-torgovom-centre)

В Интернете появился видеоролик с кадрами жестокой драки между несколькими мужчинами. Пользователи социальных сетей пишут, что инцидент якобы произошел в одном из торговых центров Казани.
"Охранники магазина одежды здесь повздорили с посетителем", - пишет один из комментаторов.
По информации казанских телеграм-каналов, охранники магазина одежды якобы заподозрили посетителя в краже товара.
Обстоятельства происшествия устанавливаются.
