
# Apple попередила про відключення від інтернету старих гаджетів

Published at: **2019-11-04T02:05:00+00:00**

Author: ****

Original: [НВ](https://techno.nv.ua/ukr/gadgets/apple-poperedila-pro-vidklyuchennya-vid-internetu-starih-gadzhetiv-novini-svitu-50051395.html)

Про це повідомляє CNN.
Відключення від інтернету відбудеться на гаджетах, які були випущені до 2012 року.
Також без оновлення перестануть працювати AppStore, iCloud, електронна пошта та пошук сайтів.
Проблеми торкнуться власників iPhone5, iPhone4, iPad mini, iPad 2.
