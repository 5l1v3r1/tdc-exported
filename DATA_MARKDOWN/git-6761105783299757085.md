
# Для продолжения улицы Сибирская может быть снесен шестиэтажный дом

Published at: **2019-11-06T05:30:00+00:00**

Author: **Александр Волков**

Original: [www.chitaitext.ru](https://www.chitaitext.ru/novosti/dlya-prodolzheniya-ulitsy-sibirskaya-mozhet-byt-snesen-shestietazhnyy-dom-/)

Власти Перми рассматривают вариант продления улицы Сибирская, при котором будет разобран полностью или частично шестиэтажный дом по адресу ул Чернышевского, 23. Сейчас кроме квартир здесь располагается супермаркет «СемьЯ» и магазины. 
Как сообщает «Коммерсантъ», проект обсуждался на публичных слушаниях, но окончательного решения по нему не принято. Новая дорога нужна для обеспечения транспортной связи с активно застраивающимся микрорайоном Красные Казармы. 

        фото Яндекс.Карты
      
По проекту, новый участок улицы Сибирская будет двухполосным и выйдет к  улице Чкалова между домами №8 и №10.
