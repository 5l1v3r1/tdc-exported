
# Камеры сняли смертельную стрельбу на западе Москвы

Published at: **2019-11-02T16:16:58+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-02/kamery-snyali-smertelnuyu-strelbu-na-zapade-moskvy)

 
Камеры наблюдения зафиксировали смертельную стрельбу на западе Москвы. Инцидент произошел в одном из дворов на улице Академика Анохина. 
По предварительным данным, несколько человек напали на двоих мужчин. По информации РЕН ТВ, нападавшие скрывали лица за масками. В результате ЧП смертельные ранения получили два человека. 
Нападавшим удалось скрыться. По данным РЕН ТВ, их автомобиль находится в розыске.
