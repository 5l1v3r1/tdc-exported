
# Home page: Valdivia pede suspensão de Campeonato Chileno

Published at: **2019-11-05T18:31:29+00:00**

Author: ****

Original: [UOL Noticias](https://noticias.uol.com.br/ultimas-noticias/erratas/2019/11/05/home-page-valdivia-pede-suspensao-de-campeonato-chileno.htm)

A home page do UOL errou ao noticiar que Valdivia pede suspensão de Campeonato Chileno com a imagem do jogador Wanderson Ferreira de Oliveira, também conhecido como Valdívia, ex-São Paulo. A reportagem se refere ao jogador chileno Jorge Valdivia. A informação foi corrigida.
