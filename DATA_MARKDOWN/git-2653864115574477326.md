
# Сервис «Европротокол онлайн» заработал в России

Published at: **2019-11-04T10:37:00+00:00**

Author: ****

Original: [35медиа](https://35media.ru/news/2019/11/04/servis-evroprotokol-onlajn-zarabotal-v-rossii)

Сервис «Европротокол онлайн» заработал в России. Теперь оформлять ДТП можно через интернет. Сервис позволяет оформлять мелкие аварии без сотрудников ГИБДД и без заполнения протокола прямо на своем смартфоне и планшете. Для этого необходимо только скачать бесплатно мобильное приложение «Помощник ОСАГО».
