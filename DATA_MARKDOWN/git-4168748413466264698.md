
# Госдума поддержала запрет на продажу смартфонов и компьютеров без российского ПО

Published at: **2019-11-05T22:12:26+00:00**

Author: **GenrichMatushka3**

Original: [it.d3.ru](https://it.d3.ru/gosduma-podderzhala-zapret-na-prodazhu-smartfonov-i-kompiuterov-bez-rossiiskogo-po-1872489/)

Согласно депутанам, он «направлен на обеспечение для российских пользователей сети Интернет и услуг связи возможность использовать приобретаемое оборудование (смартфоны, компьютеры, телевизоры с функцией „смарт–ТВ“) без необходимости установки дополнительных мобильных приложений и иных программ для ЭВМ, ориентированных на работу с российскими пользователями таких устройств»
