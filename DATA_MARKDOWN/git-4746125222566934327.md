
# ユウキロック、12年交際の女性と結婚していた「我が家は大黒柱2本体制」

Published at: **2019-11-04T13:47:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147992/full/)

　お笑い芸人のユウキロック（47）が4日、自身のツイッターを更新し、12年交際していた一般女性と今年4月に結婚していたことを発表した。 　ユウキロックは「今年4/16に12年交際していた一般女性と入籍しました」と報告。「二人共結婚指輪はしないので結婚マスクを作成。俺は『スーパーロック』妻は『スターパンク』です」とプロレスのマスクをかぶった2ショットを公開した。 　今後の結婚生活に向けて「我が家は大黒柱2本体制でハンセン・ブロディに負けないタッグとして一生戦っていこうと思います」と決意を新たにしている。 　ユウキロックは大上邦博と1995年にハリガネロックを結成。『第1回 M-1グランプリ』（2001年）で準優勝を果たしたほか、『爆笑オンエアバトル』の第4代チャンピオンにも輝いたが、14年3月のライブをもって解散。ユウキロックは自ら考案した“演芸インストラクター”の肩書で、講師を務めるほか、タレントとしても活動、大上は16年2月に芸人を引退した。
