
# Fogo em Água de Pena mobilizou bombeiros

Published at: **2019-11-01T10:19:00+00:00**

Author: **Andreína Ferreira**

Original: [dnoticias](https://www.dnoticias.pt/casos-do-dia/fogo-em-agua-de-pena-mobilizou-bombeiros-BA5398627)

Três elementos dos Bombeiros Municipais de Machico deslocaram-se esta tarde à Água de Pena para extinguir um fogo que surgiu numa zona de mato. As chamas foram extintas com o apoio de uma viatura ligeira de combate a incêndios.
