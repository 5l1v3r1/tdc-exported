
# 【豪門爆姦情】得知老公外遇自己客戶　安晨妤第一時間先嗆小三

Published at: **2019-11-06T06:08:00+00:00**

Author: **娛樂組**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191106ent011/)

歌手安晨妤與元祖小開張劭緯結婚13年，皆幸福甜蜜樣，沒想到老公卻驚爆外遇，且對象是大黑松小倆口的千金邱威樺。
據悉，安晨妤得知老公長期外遇多女，且其中一位小三就是自己所開的「123網路電商」客戶， 第一時間完全沒辦法接受，在自己和邱威樺的一個共同公務群組中公開嗆聲，群組中的其他人皆嚇到，紛紛避而遠之，就怕掃到颱風尾。
安晨妤於今（6日）發聲明，表示已對小三提出通姦罪刑法訴訟，由於目前已進入司法審判階段，因此細節無法一一陳述。
