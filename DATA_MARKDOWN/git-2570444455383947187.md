
# Железнодорожный переезд закроется в Заводском районе

Published at: **2019-11-01T10:50:00+00:00**

Author: ****

Original: [Саратов 24](https://saratov24.tv/news/zheleznodorozhnyy-pereezd-zakroetsya-v-zavodskom-rayone/)

Это связано с ремонтом. Работы продлятся с двенадцати часов ночи до пяти часов утра. Автомобильный транспорт сможет объехать этот участок через железнодорожный переезд на 6-м километре перегона Саратов-3 – Примыкание (пересечение с 5-м Динамовским проездом в Заводском районе). Об этом сообщает Приволжская железная дорога.
