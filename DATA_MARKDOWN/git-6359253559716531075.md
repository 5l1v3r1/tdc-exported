
# Челябинск не может похвастаться качеством городской среды – данные Минстроя

Published at: **2019-11-02T00:03:50+00:00**

Author: ****

Original: [Новый День](https://newdaynews.ru/chel/675651.html)

Челябинск вошел в пятерку аутсайдеров среди городов миллионников по качеству городской среды.
Как передает корреспондент РИА «Новый День», индекс городов опубликовал Минстрой РФ. Учитывались качество жилья, улично-дорожной сети, показатели озеленения, общественно-деловой инфраструктуры, социально-досуговой инфраструктуры, общегородское пространство.
Максимальное количество баллов для каждого параметра – 60.
Челябинск набрал 160 баллов из 360 и расположился на 11 месте или на 5-ой строчке с конца: относительное благополучие отмечено только по качеству жилья (32 балла), по остальным параметрам показатели не добрались даже до половины максимального значения индекса.
Хуже обстоят дела в Воронеже, Перми, Волгограде и Омске.
Тройка лидеров выглядит так: Москва – 276 баллов, Санкт-Петербург – 238, Екатеринбург – 191.
