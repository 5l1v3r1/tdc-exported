
# Esta web te permite editar y añadir efectos a tus vídeos en tiempo real

Published at: **2019-11-02T00:00:00+00:00**

Author: **Diario Tecnología**

Original: [Diario Tecnología](https://diariotecnologia.es/esta-web-te-permite-editar-y-aadir-efectos-a-tus-vdeos-en-tiempo-real)

Existen muchas aplicaciones y herramientas online que nos permiten editar y añadir filtros a nuestras fotografías, pero la oferta se reduce cuando buscamos una solución similar para vídeos.
VEED.io es una plataforma que nos permitirá, de una manera muy rápida y sencilla, subir nuestros vídeos, modificarlos y poder descargarlos de nuevo para utilizarlos como queramos.
