
# الرئيس السيسى يجتمع بـ رئيس الوزراء ووزير المالية لمتابعة تطبيق قرار رفع الحد الأدنى للأجور

Published at: **2019-11-03T02:15:00+00:00**

Author: **آيات أحمد**

Original: [صدى البلد](https://www.elbalad.news/4043614)

آيات أحمد يجتمع الرئيس عبد الفتاح السيسى مع رئيس الوزراء الدكتور مصطفى مدبولى، ووزير المالية لمتابعة تطبيق قرار رفع الحد الأدنى للأجور.جاء ذلك فى خبر عاجل، أفادت به فضائية " اكسترا نيوز" منذ قليل، وسنوافيكم بالتفاصيل..
