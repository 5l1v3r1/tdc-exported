
# HBO показала рекламу картонной коробки, которая позволит уединиться для просмотра фильмов и сериалов

Published at: **2019-11-06T04:39:23+00:00**

Author: **Arthur Khamzin**

Original: [TJ](https://tjournal.ru/news/124481-hbo-pokazala-reklamu-kartonnoy-korobki-kotoraya-pozvolit-uedinitsya-dlya-prosmotra-filmov-i-serialov)

Её могут выиграть американские студенты в специальном конкурсе.
Американская компания HBO выпустила рекламу картонной коробки The HBO Box, которая предназначена для просмотра фильмов и сериалов. Чтобы её получить, нужно поучаствовать в конкурсе.
