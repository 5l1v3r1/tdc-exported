
# కేసీఆర్ పాలన అంతానికి ఎంతోకాలం పట్టదు: లక్ష్మణ్‌

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=948830)

జగిత్యాల: కేసీఆర్ నయా నిజాంలా వ్యవహరిస్తున్నారని బీజేపీ రాష్ట్ర అధ్యక్షుడు లక్ష్మణ్ ధ్వజమెత్తారు. కేసీఆర్ పాలన అంతానికి ఎంతో కాలం పట్టదని హెచ్చరించారు. ఆర్టీసీ కార్మికులను నడిరోడ్డున పడేసిన కేసీఆర్‌పై.. ప్రజలు తిరగబడే రోజులు వస్తాయన్నారు. రెవెన్యూ ఉద్యోగులను కేసీఆర్ వేధిస్తున్నారని ఆరోపించారు. తెలంగాణపై మోదీ, అమిత్ షా ప్రత్యేక దృష్టి పెట్టారని తెలిపారు.
