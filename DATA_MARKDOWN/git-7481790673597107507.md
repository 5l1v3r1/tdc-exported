
# トラック荷台の冷蔵庫から移民４１人、全員生存

Published at: **2019-11-05T02:48:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/world/20191105-OYT1T50131/)

　【ローマ＝笹子美奈子】ロイター通信によると、ギリシャ北部クサンチ近郊で４日、トラックの荷台から４１人の移民が見つかった。多くはアフガニスタン出身とみられ、全員の生存が確認された。
　ギリシャ警察がトラックを検問したところ、荷台の冷蔵庫の中から移民が発見され、運転手を拘束した。英国では先月２３日、トラックのコンテナから密航者と見られる３９人の遺体が見つかっており、欧州への危険渡航が問題となっている。
　現場付近は、アジアや中東からバルカン半島を北上する「バルカンルート」に位置している。
