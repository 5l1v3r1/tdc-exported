
# Эрдоган предоставил ООН план по зонам поселения для сирийских беженцев

Published at: **2019-11-01T17:08:51+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-01/erdogan-predostavil-oon-plan-po-zonam-poseleniya-dlya-siriyskih-bezhencev)

ООН создаст группу для изучения предложений президента Турции Тайипа Эрдогана по новым зонам поселения для возврата сирийских беженцев. Об этом сообщает РИА "Новости".  
"Президент Эрдоган предоставил генсеку турецкий план по новым зонам поселения для возврата сирийских беженцев. Генсек указал на основные принципы, касающиеся добровольного, безопасного и достойного возврата беженцев", — заявил заместитель официального представителя генсека ООН Фархан Хак в ходе брифинга.
В свою очередь, ООН проинформировала турецкого лидера, что Управление верховного комиссара ООН по делам беженцев незамедлительно соберет группу для изучения предложения Эрдогана.
