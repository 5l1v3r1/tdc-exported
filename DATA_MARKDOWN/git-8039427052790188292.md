
# అర్వపల్లి ఎస్సైకు రోడ్డు ప్రమాదం.. తలకు తీవ్ర గాయాలు

సూర్యాపేట: రోడ్డు ప్రమాదంలో అర్వపల్లి ఎస్ఐ లోకేష్‌ తలకు తీవ్ర గాయాలయ్యాయి.
నాగారం, ఫణిగిరి మధ్య ఈ ప్రమాదం జరిగింది. గుమ్మడవెల్లి కాంగ్రెస్, టీఆరెస్ కార్యకర్తల మధ్య ఘర్షణలు చోటు చేసుకున్న నేపథ్యంలో అక్కడికి వెళ్లి వస్తుండగా ఈ రోడ్డు ప్రమాదం జరిగింది. గుమ్మడవెల్లిలో పరిస్థితిని సమీక్షించిన ఎస్సై లోకేష్ తన వ్యక్తిగత వాహనంలో తిరిగి వస్తుండగా వాహనం పల్టీ కొట్టింది. ఆయనను హుటాహుటిన సూర్యాపేట జనరల్ ఆసుపత్రికి తరలించారు. అనంతరం మెరుగైన చికిత్స నిమిత్తం హైదరాబాద్ యశోద ఆసుపత్రికి తరలించారు.
