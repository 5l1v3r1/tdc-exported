
# 林志玲台南辦婚宴　言承旭7次狂親小20歲少女

Published at: **2019-11-07T04:57:00+00:00**

Author: **娛樂組**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191107ent012/)

在前女友林志玲嫁給日星Akira之後，言承旭的感情狀態總令人好奇，不過他似乎全心投入工作，近來拍攝中國大陸偶像劇《交換吧愛情》，和飾演陸版《流星花園》的「杉菜」沈月大談相差20歲的老少戀。
言承旭在劇中飾演霸道總裁，他和沈月都曾演出過《流星花園》，但兩人實際年齡相差20歲，格外引起關注，有些本來有點疑慮的網友，看了預告之後發現，除了年齡差不違和之外，竟還有點甜。
言承旭還在劇中重現《流星花園》經典台詞「如果道歉有用，要警察幹嘛！」勾起不少人的回憶。此外他和沈月在曝光的2分半預告中，光是吻戲就多達7次，讓網友稱讚，「這次真的可以」「言承旭更配少女」。
