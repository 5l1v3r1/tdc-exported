
# Украина назвала новый способ заблокировать «Северный поток — 2»

Published at: **2019-11-07T18:49:00+00:00**

Author: ****

Original: [charter97.org](https://charter97.org/ru/news/2019/11/7/354804/)

Украина обжалует в суде ЕС возможное решение властей Германии сделать исключение из правил Еврокомиссии для трубопровода «Северный поток-2». Об этом заявил торговый представитель Украины Тарас Качка на конференции «Торговые войны: искусство защиты», сообщает «Европейская правда».
В соответствии с директивой Еврокомиссии одна компания не может быть владельцем газопровода и одновременно прокачивать по нему газ (как это намерен делать «Газпром»). Однако как ранее сообщало издание Bild, ХДС/ХСС и СДПГ подготовили законопроект, в соответствии с которым для «Северного потока-2» может быть сделано исключение, поскольку следует учитывать инвестиции, вложенные в проект за несколько предыдущих лет.
По мнению Качки, такая поправка будет напрямую противоречить решению Еврокомиссии, а потому Украина или Польша ее обжалует. «Россия же стремится сохранить устаревшую модель отношений — и это путь в никуда. Это динозавр, который сопротивляется, но его конец неизбежен», — сказал украинский торговый представитель.
В качестве довода, что подобная тактика может быть успешной, Качка привел судебный спор по поводу трубопровода Opal, который проходит через территорию Польши. «Газпрому» разрешили использовать 90% мощностей трубы, однако Европейский суд отменил это решение по требованию Польши.
В конце октября Энергетическое агентство Дании дало разрешение на строительство участка газопровода «Северный поток-2», проходящего по континентальному шельфу страны. Таким образом, Дания стала последней страной, согласовавшей маршрут трубопровода для «Газпрома».
