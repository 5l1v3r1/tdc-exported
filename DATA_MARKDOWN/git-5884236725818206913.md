
# В Кировском парке Сыктывкара откроется сквер для молодых мам

Published at: **2019-11-06T09:53:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Syktyvkar/740257-v-kirovskom-parke-syktyvkara-otkroetsya-skver-dlya-molodykh-mam)


        Напротив дома № 24 по улице Кирова в Сыктывкаре планируется создать сквер для молодых мам, в котором они смогут проводить время с детьми. На территории будет установлен пеленальный столик, появится место для кормления.
      
В столице Коми планируется создать сквер для молодых мам. Общественная организация «Проект 1780» выиграла президентский грант на реализацию данной идеи. БНК сообщает, что сквер может появиться в Кировском парке.
Организация получила более 497 тысяч рублей на инициативу «Умный город», часть из них будет направлена на создание сквера. Наталья Хозяинова, и.о. мэра Сыктывкара, отметила, что определен участок напротив дома № 24 по улице Кирова. Специалисты осмотрели территорию и выразили готовность оказать помощь активистам.
Предполагается, что в сквере будут установлены не только скамейки, но также пеленальный столик, будет организовано и место для кормления грудью.
