
# Кейн: Тоттенхэм Чемпионлар Лигасини ютишга қодир

Published at: **2019-11-06T09:35:42+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/keyn-tottenxem-chempionlar-ligasini-yutishga-qodir)

"ЕЧЛнинг ўтган мавсуми биз учун жуда ҳам муваффақиятли ўтди ва биз узоқ йўлни босиб ўтдик.
Биз яна ғолиб чиқишни истаймиз. Агар "Тоттенхэм" ҳеч бўлмаса битта мусобақана ютса ҳам, бу клуб учун муваффақият ҳисобланади.
ЕЧЛни шу мавсум ютишимиз мумкинми? Ўйлайманки, ҳа. Ўтган мавсум стартини эсланг ва биз охирида нималар қилганимизни ҳам. "Тоттенхэм" одамлар хато қилганини исботлаган эди", - дейди Кейн.
