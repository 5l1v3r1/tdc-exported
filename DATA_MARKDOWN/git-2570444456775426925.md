
# В Балашове грабители вынесли из банка 13 миллионов рублей

Published at: **2019-11-06T12:20:00+00:00**

Author: ****

Original: [Саратов 24](https://saratov24.tv/news/v-balashove-grabiteli-vynesli-iz-banka-13-millionov-rubley/)

Крупной суммы лишился один из офисов «Россельхозбанка». Происшествие случилось в минувшие выходные. Воры попали в банк днем, спрятались по одной из версий в офисе Управления сельского хозяйства. Когда рабочий день закончился и все служащие разошлись по домам, злоумышленники похитили деньги. Грабители до сих пор не пойманы. Есть информация, что они уехали на светлом автомобиле «Нива». В городе работают полицейские из областного центра. О преступлении сообщает информационное агентство «Балашовер». Правоохранительные органы пока не комментируют случившееся. Банк работает в обычном режиме.
