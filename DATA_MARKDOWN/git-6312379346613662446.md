
# Один человек погиб в ДТП с участием рейсового автобуса под Кемеровом

Published at: **2019-11-02T08:48:27+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-02/odin-chelovek-pogib-v-dtp-s-uchastiem-reysovogo-avtobusa-pod-kemerovym)

Автомобиль Nissan столкнулся с рейсовым автобусом в Кемеровской области. Один человек погиб, еще двое пострадали.
Авария произошла около 2:30 по московскому времени в ночь на субботу, сообщает пресс-служба регионального управления ГИБДД.
По предварительным данным, водитель Nissan выехал на встречную полосу, где врезался в автобус "Нефаз".
Водитель иномарки скончался на месте, травмы получили две пассажирки автобуса.
По факту аварии начато расследование.
