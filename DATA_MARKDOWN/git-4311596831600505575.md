
# Движение автомобилей с прицепом запрещено на дороге Гудаури — Коби

Published at: **2019-11-01T15:52:00+00:00**

Author: ****

Original: [EADaily](https://eadaily.com/ru/news/2019/11/01/dvizhenie-avtomobiley-s-pricepom-zapreshcheno-na-doroge-gudauri-kobi)

В Грузии ввели ограничения для движения транспорта на участке Коби — Гудаури автомагистрали Мцхета — Степанцминда — Ларс, сообщает департамент автомобильных дорог Министерства регионального развития и инфраструктуры страны.
По информации ведомства, «из-за обильных осадков снега на данном участке запрещено движение автотранспорта с прицепом и полуприцепом».
Для других видов автотранспорта движение свободное.
