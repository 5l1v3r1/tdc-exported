
# บาร์ซาจ๊ากซัวเรซข้อเท้าพังพัก3วีก

Published at: **2019-11-04T15:18:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/sports/740001)


        "บาร์เซโลนา" งานเข้าอีกรอบ เมื่อ "หลุยส์ ซัวเรซ" หัวหอกตัวเก๋า ข้อเท้าพักต้องนอนพักรักษาตัวนานถึง 3 สัปดาห์
      
บาร์เซโลนา ทีมดังแห่งศึกลา ลีกา สเปน ต้องพบข่าวร้ายอีกครั้ง เมื่อจะหมดสิทธิใช้งาน หลุยส์ ซัวเรซ ดาวยิงตัวเก่ง นานถึง 3 สัปดาห์ หลังเจ้าตัวได้รับบาดเจ็บข้อเท้าจากเกมบุกพ่าย เลบันเต ยับเยิน 1-3 เมื่อวันเสาร์ที่ผ่านมาลอนดอน อีฟนิ่ง สแตนดาร์ด สื่อดังของอังกฤษเผยว่า ซัวเรซ ที่โดนถอดออกจากสนามตั้งแต่นาทีที่ 41 ในเกมพ่าย "ค้างคาวน้อย" นั้น จะพลาดลงสนามให้ "เจ้าบุญทุ่ม" ถึง 3 นัดคือในเกม แชมเปี้ยนส์ ลีก นัดเปิดคัมป์ นู ดวลเกือก สลาเวีย ปราก วันอังคารนี้, เกม ลา ลีกา นัดเปิดบ้านรับมือ เซลตา บีโก วันเสาร์ที่ 9 พ.ย.และเกมบุกรัง เลกาเนส ในวันเสาร์ที่ 23 พ.ย.ด้วย.ภาพ gettyimages
