
# Зеленский назвал главную победу Рады IX созыва

Published at: **2019-11-07T16:13:06+00:00**

Author: ****

Original: [ФАКТИ](https://fakty.com.ua/ru/ukraine/20191107-zelenskyj-nazvav-golovnu-peremogu-rady-ix-sklykannya/)

Владимир Зеленский считает главной победой нынешней Верховной Рады принятие ряда важных законов.
Европейские партнеры и кредиторы довольны турборежимом, в котором работает Верховная Рада IX созыва.
Об этом заявил президент Украины Владимир Зеленский во втором выпуске своего блога Откровенно с президентом на YouTube.
По его словам, народные депутаты уже успели принять очень важные для Украины законы.
