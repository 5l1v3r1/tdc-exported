
# Квест для щкольников

Published at: **2019-11-01T08:30:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/tinao/kvest_dlya_shchkolnikov/)

3 ноября Дом культуры «Юбилейный» поселения Московский примет участие в седьмой акции «Ночь искусств».— Мероприятие посетят ребята с 8 по 11 классы. Акцию проведут в виде квеста по Дому культуры «Юбилейный». Учеников разделят на две команды по десять человек. Квест пройдет в несколько этапов. Например, в библиотеке организуют игру «Что? Где? Когда?». Ребятам предстоит ответить на вопросы по литературе, — поделилась информацией художественный руководитель Дома культуры  «Юбилейный» Олеся Загузина.Акция начнется в 12.00 по адресу: поселок Рогово, улица Юбилейная, дом 1А.Материал сайта http://rogovskoe.org/all_news/news/akciyu_noch_iskusstv_provedut_v_dome_kultury_yubilejnyj/Источник: http://tinaomos.news
