
# Минстрой предложил ремонтировать ветхие дома за счет граждан

Published at: **2019-11-07T07:43:00+00:00**

Author: ****

Original: [RuNews24](https://runews24.ru/society/07/11/2019/0efcf5bf958ff3c500f6c59338abcede)

С такой инициативой выступило Министерство строительства, пишут «Известия» со ссылкой на пресс-службу ведомства.
Министерство считает, что такие дома следует включить в региональные трехлетние подпрограммы капремонта. Финансировать их предлагают за счет денег граждан либо софинансировать ремонт из местных бюджетов. Речь идет только о работах, необходимых для поддержания безопасного для жильцов состояния изношенных домов, пока их не признают аварийными.
Разработка программы ремонта зданий предусматривается новым проектом закона по расселению аварийного жилья. Положения данного документа сейчас обсуждаются.
