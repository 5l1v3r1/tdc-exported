
# టిప్పర్ ఢీ.. జీహెచ్‌ఎంసీ కార్మికురాలి మృతి

హైదరాబాద్: కూతురి ద్విచక్ర వాహనంపై వెళుతున్న జీహెచ్ఎంసీ కార్మికురాలిని టిప్పర్ వాహనం ఢీకొట్టడంతో ఆమె అక్కడికక్కడే మృతి చెందింది. ముషీరాబాద్ ప్రధాన రహదారిలో ఈ ప్రమాదం చోటు చేసుకుంది. నేటి ఉదయం 6:30 గంటలకు ద్విచక్ర వాహనంపై తన కూతురుతో టూ వీలర్ వాహనంపై వెళుతున్న లక్ష్మమ్మ అనే జీహెచ్ఎంసీ కార్మికురాలు లక్ష్మమ్మను టిప్పర్ వాహనం ఢీకొనడంతో ఆమె అక్కడికక్కడే మృతి చెందింది.
