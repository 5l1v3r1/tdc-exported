
# Ледокол «Обь» готовится к уходу из Мурманска

Published at: **2019-11-06T10:56:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Murmansk/740298-ledokol-ob-gotovitsya-k-ukhodu-iz-murmanska)


        Ледокол «Обь» уйдет из порта Мурманска в порт Ямало-Ненецкого автономного округа 7 ноября для выполнения поставленных перед ним задач.
      
Ледокол «Обь», принадлежащий «Атомфлоту», готовится к уходу из порта Мурманска. Судно будет направлено в порт Ямало-Ненецкого автономного округа 7 ноября.
Первый неатомный ледокол Росатомфлота позволит завершить проект в порту Сабетта по договору с Ямал СПГ на оказание услуг до 31 декабря 2040 года, передает пресс-служба ФГУП «Атомфлот».
