
# ＷＴＯ、中国の対米報復関税を承認…年３６億ドル

Published at: **2019-11-02T04:58:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/economy/20191102-OYT1T50184/)

　【ロンドン＝池田晋一】世界貿易機関（ＷＴＯ）は１日、中国が米国に対し、最大で年約３６億ドル（約３９００億円）の報復関税を課すことを承認した。米国による中国製品への反ダンピング（不当廉売）措置が不当だとして中国が２０１３年に提訴していたもので、米中貿易摩擦が本格化する前の紛争に関する決定。
　報復関税の規模は、ＷＴＯが今年１０月、約７５億ドル分の欧州連合（ＥＵ）製品に課税することを米国に認めた裁定などに次ぎ、３番目となる。中国は近く、ＷＴＯから最終承認を受ける見通しだ。だが、実際には発動せず、今後の米国との貿易協議での駆け引きに使うとの見方もある。
　中国は１３年１２月、米国が中国製の太陽光パネルや鉄鋼製品に対して行った反ダンピング措置は不当としてＷＴＯに提訴。１審にあたる紛争処理小委員会は１６年１０月、米国の措置は貿易ルール違反と認定し、是正を求めた。
　しかし、米国が応じなかったため、中国が１８年９月にＷＴＯに裁定を求めていた。ＷＴＯは米国の措置について、中国の複数の輸出業者を中国政府の支配下にあるとみなし、一律に反ダンピング税を課したことなどをルール違反と認定した。
