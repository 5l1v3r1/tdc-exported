
# Робопчела совершила управляемый полет

Published at: **2019-11-06T09:42:00+00:00**

Author: ****

Original: [RuNews24](https://runews24.ru/science/06/11/2019/dca9285f3dd199590cd38c629f841f3f)

Чтобы разработать устройство, специалисты использовали для создания двигательной системы мягкие электрические приводы, которые изготовлены на основе диэлектрических эластомеров.
Для проверки работоспособности робопчелы создали несколько моделей с программным управлением. Так, модель с двумя крыльями оказалась непригодной к использованию – инженеры теряли над ней контроль сразу после того, как она взлетала.
Модель с четырьмя крыльями исследователи одобрили. Инженеры отметили, что разработка очень устойчива к внешним воздействиям.
