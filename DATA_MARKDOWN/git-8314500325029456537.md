
# US-Anleihen: Kurse legen noch etwas zu

Published at: **2019-11-06T20:53:00+00:00**

Author: ****

Original: [cash](https://www.cash.ch/news/boersenticker-ausland/us-anleihen-kurse-legen-noch-etwas-zu-1428278)

Zweijährige Anleihen stiegen um 1/32 Punkt auf 99 25/32 Punkte. Sie rentierten mit 1,605 Prozent. Fünfjährige Anleihen kletterten um 6/32 Punkte auf 99 13/32 Punkte. Sie rentierten mit 1,626 Prozent. Zehnjährige Anleihen legten um 13/32 Punkte auf 98 10/32 Punkte zu. Sie rentierten mit 1,812 Prozent. Longbonds mit einer Laufzeit von dreissig Jahren gewannen 28/32 Punkte auf 98 31/32 Punkte. Sie rentierten mit 2,296 Prozent./bek/he
