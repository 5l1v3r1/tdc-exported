
# Серіал Слуга народу крутитимуть на центральному ТБ в Білорусі

Published at: **2019-11-06T10:33:50+00:00**

Author: **Daria Trokhimovich**

Original: [Рубрика](https://rubryka.com/2019/11/06/serial-sluga-narodu-krutytymut-na-tsentralnomu-tb-v-bilorusi/)


        
          З 11 листопада основний державний білоруський телеканал “Білорусь 1” покаже український серіал “Слуга народу” в вечірній прайм-тайм.
        
      
Повідомляє агентство теленовин Белтелерадіокомпанія.
“Усі сезони цього багатосерійного фільму глядачі “Білорусь 1” зможуть побачити в вечірній прайм-тайм – о 20:00 з понеділка по п’ятницю. В ефір виходитиме по два епізоди картини “Слуга народу””.
Серіал “Слуга народу”, створений студією “Квартал 95”, виходив в Україні з листопада 2015 року по березень 2019 року. Головну роль в ньому зіграв нинішній президент України Володимир Зеленський.
“Слуга народу” – комедійний політичний телесеріал, який розповідає історію обрання президентом України учителя історії Василя Голобородька.
Серіал складається з 3 сезонів.
