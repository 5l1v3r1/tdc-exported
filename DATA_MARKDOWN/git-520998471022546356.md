
# На будівлі художнього музею в Чернівцях ремонтують аварійний карниз

Published at: **2019-11-06T12:54:00+00:00**

Author: ****

Original: [Погляд](https://pogliad.ua/news/chernivtsi/na-budivli-hudozhnogo-muzeyu-v-chernivcyah-remontuyut-avariyniy-karniz-379906)

У Чернівцях на будівлі художнього музею біля міської ради за допомогою спеціальної машини ремонтують карниз.
Перекрито тротуар. Пішоходам радять бути обережними на цій ділянці.
Руху транспорту по Центральній площі роботи не заважають, пише ТВА.
