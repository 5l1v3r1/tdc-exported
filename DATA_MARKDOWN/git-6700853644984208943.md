
# Gubernur Bengkulu Sambut Api Abadi Torch Relay Porwil ke X se Sumatera

Published at: **2019-11-01T18:57:00+00:00**

Author: **Roki EP**

Original: [RRI News Portal](http://rri.co.id/post/berita/741886/cabor/gubernur_bengkulu_sambut_api_abadi_torch_relay_porwil_ke_x_se_sumatera.html)

KBRN, Bengkulu : Setelah diambil dari pusat energi panas bumi di Kabupaten Lebong dan melalui perjalanan cukup jauh melewati tiga wilayah Kabupaten, yakni Kabupaten Rejang Lebong, Kabupaten Kepahiang dan Bengkulu Tengah, pada Jumat (1/11/2019) sore, api abadi Torch Relay Pekan Olahraga Wilayah (Porwil) ke X se Sumatera, telah sampai di Kota Bengkulu.
Sebelum rombongan yang membawa api abadi Torch Relay Porwil ke 10 se Sumatera, disambut oleh Gubernur Bengkulu Rohidin Mersyah bersama Forum Komunikasi Pimpinan Daerah (Forkopimda) serta Organisasi Perangkat Daerah (OPD) di lingkup Pemprov Bengkulu, dengan didampingi pengurus KONI dan sejumlah atlit bertempat di gedung Balai Raya Semarak Bengkulu, dijemput oleh Wakil Walikota Dedy Wahyudi bersama jajaran Pemerintah Kota Bengkulu di simpang 4 Nakau, perbatasan masuk Kota Bengkulu.
Seusai menerima api obor Porwil ke X se Sumatera, Gubernur Rohidin Mersyah berharap, dalam Porwil ini bisa menorehkan prestasi bagi Bengkulu. Apalagi sebelumnya cabang olahraga sepak bola yang tidak disangka-sangka, dipastikan akan berlaga di seri final pada Minggu lusa, melawan tim sepak bola dari Aceh.
“Untuk itu saya mengajak seluruh elemen masyarakat agar bersama-sama mendukung dan mensuport para atlit Bengkulu yang akan berlaga pada Porwil ini. Mengingat dengan dukungan penuh tersebut diyakini akan menambah semangat sekaligus motivasi untuk menjadi juara nantinya,” harap Gubernur.
Dengan telah berada api obor Porwil ini, akan diinapkan semalam di Gedung Balai Raya Semarak, dan besok pagi akan kembali diarah keliling kota Bengkulu sebelum menuju opening Porwil di lapangan sepak bola stadion Semarak Bengkulu, yang direncanakan di buka oleh Menteri Pemuda dan Olahraga, Zainudin Amali.
“Insya Allah Menpora akan hadir secara langsung membuka Porwil ke X se Sumatera Bengkulu, besok malam,” tukas Gubernur Rohidin.
