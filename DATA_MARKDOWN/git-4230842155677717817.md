
# HBO випустила чорну картонну коробку для перегляду серіалів

Published at: **2019-11-06T05:20:31+00:00**

Author: ****

Original: [ФАКТИ](https://fakty.com.ua/ua/showbiz/tv/20191106-hbo-vypustyla-chornu-kartonnu-korobku-dlya-pereglyadu-serialiv/?utm_source=yxnews&utm_medium=desktop&utm_referrer=https%3A%2F%2Fyandex.ua%2Fnews)

HBO Box – спеціальна коробка для перегляду серіалів – не продається, але її можна виграти.
Телеканал HBO представив новий продукт – HBO Box. Коробка з чорного картону створена для тих, хто хоче дивитися серіали наодинці з собою.
Може здатися, що це першоквітневий жарт, але це не так.
HBO Box – це гігантська чорна картонна коробка з отворами для повітряного потоку, полицею всередині та вирізами під ноги.
