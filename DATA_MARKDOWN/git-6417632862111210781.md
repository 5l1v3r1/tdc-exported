
# ОФК Кубоги финали фотосуратларда

Published at: **2019-11-05T07:51:21+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/ofk-kubogi-finali-fotosuratlarda)

Малайзияда бўлиб ўтган мусобақа финалида Ливаннинг "Ал Аҳед" клуби Шимолий Кореянинг "25 апрель" жамоасига қарши баҳс олиб бориб 1:0 ҳисобида зафар қучди. Шундай қилиб ливанликлар тарихда биринчи бор ОФК Кубогини қўлга киритишди. Қуйида финалдан фотосуратларни келтириб ўтамиз.
