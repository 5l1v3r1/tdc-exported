
# На админгранице с Крымом временно закрывают КПВВ Каланчак

Published at: **2019-11-05T17:01:21+00:00**

Author: ****

Original: [ФАКТИ](https://fakty.com.ua/ru/ukraine/20191105-na-adminkordoni-z-krymom-tymchasovo-zakryvayut-kpvv-kalanchak/)

Контрольно-пропускной пункт Каланчак, расположенный на границе с оккупированным Крымом, временно приостанавливает пропуск граждан.
С 7 по 15 ноября КПВВ Каланчак временно не будет пропускать граждан.
Контрольные пункты Чаплинка и Чонгар продолжают работать в штатном режиме.
Читайте: На Донбассе боевики обстреляли КПВВ Марьинка
Пункт пропуска временно закрывают для проведения Укрзализныцей строительных работ, направленных на реконструкцию контрольных пунктов на административной границе с Крымом.
Это делается для того, чтобы избежать травмирования людей.
Напомним, ранее КПВВ Каланчак закрывали для автомобилей.
