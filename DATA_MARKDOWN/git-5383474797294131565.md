
# Adobe выпустила 18 тысяч новых шрифтов для iPhone

Published at: **2019-11-01T21:59:00+00:00**

Author: **Илья Сидоров**

Original: [iPhones.ru](https://www.iphones.ru/iNotes/adobe-vypustila-18-tysyach-novyh-shriftov-dlya-iphone-11-01-2019)

Adobe добавила 1300 бесплатных шрифтов в Creative Cloud.
Их можно использовать во всех приложениях на iOS 13.1 и iPadOS 13.1, которые поддерживают соответствующую функцию, например, в Pages.
Ранее установка нестандартных шрифтов было очень сложным процессом. Начиная с iOS 13.1 Apple упростила эту процедуру, предоставив API разработчикам.
В Adobe Creative Cloud есть следующие типы шрифтов:
▪️ С засечками▪️ Без засечек▪️ Рукописные▪️ Брусковые▪️ Готические▪️ Моноширинные
Подписка Creative Cloud открывает доступ ещё к 17 000 шрифтов. [The Verge]
