
# США сообщили о выходе из Парижского соглашения по климату

Published at: **2019-11-05T12:47:00+00:00**

Author: ****

Original: [ZIK.UA](https://zik.ua/ru/news/2019/11/05/ssha_soobshchyly_o_vihode_yz_paryzhskogo_soglashenyya_po_klymatu_1683977)


        Соединенные Штаты Америки официально уведомили ООН о начале процедуры выхода из Парижского соглашения по климату 2015 года.
      
Об этом сообщает пресс-служба Государственного департамента США.
Как отмечается, Соединенные Штаты Америки официально выйдут из Парижской соглашения через год после получения ООН сообщение об этом.
По словам государственного секретаря США Майк Помпео, несмотря на выход из соглашения, страна продолжит проводить исследования в области зеленой энергетики и пользоваться возобновляемыми источниками энергии. Кроме этого, США будет и в дальнейшем "протягивать руку помощи своим друзьям и партнерам по всему миру".
Напомним, недавно испанское правительство предложило экологической активистке, шведской школьнице Грете Тунберг помочь попасть на саммит ООН по вопросам климата в Мадриде, который состоится в декабре.
