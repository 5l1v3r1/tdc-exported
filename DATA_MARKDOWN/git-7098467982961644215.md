
# В Sputnik расскажут об итогах первого этапа проекта Тайны Великой Абхазской стены

Published at: **2019-11-04T10:23:00+00:00**

Author: ****

Original: [Sputnik Абзахия](https://sputnik-abkhazia.ru/Abkhazia/20191104/1028747539/V-Sputnik-rasskazhut-ob-itogakh-pervogo-etapa-proekta-Tayny-Velikoy-Abkhazskoy-steny.html)


        5 ноября в 10:00 в мультимедийном пресс-центре Sputnik Абхазия состоится пресс-конференция по итогам первого этапа проекта "Тайны Великой Абхазской стены".
      
Завершилась первая из трех археологических экспедиций в рамках проекта "Тайны Великой Абхазской стены". Об исследовании и окончании археологических раскопок в селе Хиацха Галского района расскажут представители инициативной группы:
Приглашаем представителей СМИ принять участие в пресс-конференции, которая пройдет в мультимедийном пресс-центре Sputnik Абхазия по адресу: город Сухум, ул. Пушкина, 16 (бизнес-центр "Гудоу Плаза", пятый этаж).
