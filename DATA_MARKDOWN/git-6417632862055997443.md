
# “Барселона” бундай натижани ЕЧЛда 3-марта қайд этди

Published at: **2019-11-06T03:43:09+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/barselona-bunday-natijani-echlda-3-marta-qayd-etdi)

“Барселона” шу тариқа ЕЧЛдаги иштироки давомида ўз майдонида учинчи марта кучли бешлик чемпионати вакили бўлмаган клуб дарвозасини ишғол қила олмади. Жамоа бунга қадар “Бенфика”ни 2012 йилда мағлуб эта олмаган (0:0). Бундан ташқари 1997 йилда Киевнинг “Динамо”сига 0:4 ҳисобида мағлуб бўлганди.
