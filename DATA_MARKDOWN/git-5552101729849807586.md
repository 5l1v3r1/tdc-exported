
# 《黑澀會》薔薔誇張行徑惹流氓　差點被西瓜刀砍

Published at: **2019-11-06T09:50:00+00:00**

Author: **張詠淇**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191106ent024/)

緯來綜合台實境節目《歡迎光琳》邀請女藝人楊繡惠、薔薔、蘿莉塔、解婕翎齊聚一堂吃飯聊天，這4位女星都有著性格直率，幽默犀利的共通點，是演藝圈的「正義俠女」，她們分享許多精彩的事蹟，主持人沈玉琳聽了都驚嘆：「這些都可以當『楊繡惠接班人』了！」
林嘉凌（薔薔）自《我愛黑澀會》出道，作風嗆辣、敢說敢言，大膽風格深受許多粉絲喜愛。薔薔更自豪不向惡勢力低頭，她自爆：「有次跟朋友去逛夜市，我們莫名其妙被流氓盯上，而且竟還被他腳『不小心』踢到。」這一腳讓她氣炸：「換我回擊一個『不是故意的』腳踹回去！」結果流氓開始對薔薔挑釁，她竟還比出中指回應，這時對方已經亮出西瓜刀，薔薔坦言：「其實我當時超怕，雙腿根本都在顫抖，但氣勢要有，我依然怒狠狠瞪他！」沈玉琳聽聞玩笑表示：「那妳趕快拿顆西瓜給他砍半，讓他消消氣。」
同桌女星蘿莉塔個性直爽率真，骨子裡同樣存在『正義之氣』，甚至還因此研究不雅影片！她解釋：「因為常會號稱某藝人不雅影片外流，我都會反覆研究然後給專家朋友們鑑定。」然後自己會再回去留言，告訴大家是假的。沈玉琳反嗆她：「妳幹嘛雞婆讓男人幻想破滅！」
蘿莉塔還說對爸爸的外遇很不能諒解，「爸爸還在世時我還能忍氣吞聲，爸爸過世後我再也受不了！」她坦言為了幫家人出氣，在告別式致詞台上直接指著爸爸的再娶是「破壞家庭的小三、狐狸精」，還欲罷不能搶著麥克風繼續講，脫稿演出讓全場都很傻眼！
