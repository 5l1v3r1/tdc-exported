
# 新田真剣佑、誕生日ケーキのサプライズに『自撮り』で対応

Published at: **2019-11-06T16:54:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148092/full/)

　俳優の新田真剣佑が6日、都内で行われた舞台『星の大地に降る涙』製作発表記者会見に出席した。今月16日に23歳の誕生日を迎える新田だが、誕生日ケーキがサプライズで登場した際、「すげー！ 写真撮っていいですか!?」と興奮した姿を見せた。

        【写真】その他の写真を見る
      
　初主演舞台となる同作の初演を見た新田は「日本に来る前にこの作品を見て衝撃を受けました。素晴らしい作品にしたいと思います」と話した。会見場で、10年前の初演で主役を務めた三浦春馬からVTRメッセージが寄せられると「あとで動画ください。本当にうれしいです。前作を超える作品にしたい。」と意気込んでいた。 　フォトセッション時には、岸谷五朗と寺脇康文がバースデーソングを歌いながら誕生日ケーキをサプライズで運んでくると、驚いた表情を見せ「ありがとうございます！」と感謝を述べ、自らのスマホで出演者と一緒にケーキを囲み自撮りをする一幕も。 岸谷、寺脇が主宰する演劇ユニット「地球ゴージャス」の結成25周年記念公演となる本作は、歴史の大きな渦に巻き込まれた人間たちの生き様を描いたファンタジー作品となる。2020年3月10日〜4月13日に舞浜アンフィシアターにて東京公演、2020年5月3日〜5月14日にフェスティバルホールにて大阪公演が開催される。 　会見では、ほかに笹本玲奈、松本利夫、湖月わたる、愛加あゆ、島ゆいか、森公美子が出席した。
