
# Белорусское приложение помогает врачам свериться с Минздравом

Published at: **2019-11-04T15:02:00+00:00**

Author: **Андрей Кожемякин**

Original: [dev.by](https://dev.by/news/softgarantservis-lekar-pro)

Белорусская компания «Софтгарантсервис» разработала бесплатное мобильное приложение для врачей — Lekar Pro. Медики получат быстрый доступ к стандартам лечения и диагностики, которые разработал Минздрав.
Lekar Pro доступно на Android и iOS. В приложении — полная база клинических протоколов Минздрава Беларуси. Это нормативные стандарты, на которые должны опираться врачи при постановке диагноза и назначении лечения. Они рекомендованы государственным и частным медучреждениям.
В приложении можно:
Бэк-платформа позволяет любым разработчикам получать по API актуальные протоколы, инструкции препаратов, встраивать информацию в свои информационные системы.
В планах добавить в приложение сервисы для общения между врачами и обучения.
Разработчик информационных систем для управления предприятий реального сектора экономики, а также для организаций здравоохранения. Занимается локализацией в Беларуси медицинской информационной системы международной компании «Медэлемент». Эта платформа объединяет облачные сервисы и справочные системы для врачей, клиник и пациентов в разных странах. Мобильное приложение с клиническими протоколами — первый этап локализации.
Разработчик программного обеспечения для автоматизации бизнес-процессов и управленческих функций, а также мобильных и веб-приложений для медицины. На рынке информационных систем с 2001 года. Головной офис — в Алматы. Представительства и партнеры в России, Беларуси, Узбекистане.
