
# ヤクルト・清水ブルペン３連投「投げることで得られることある」

Published at: **2019-11-04T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/04/0012849179.shtml)

　ヤクルト・清水昇投手が４日、松山秋季キャンプで３日連続のブルペン投球を行った。初日から６０球、７４球ときて、この日は６７球。５日はシート打撃登板も予定している。
　「投げることで得られることがある。疲れた時はどこが悪くなるとか、何か見つかればいい」。ドラフト１位の即戦力として期待されながら、ルーキーイヤーの今季は未勝利だっただけに、精力的に投手陣で最も多くの球数を投げ込んだ。高津監督も「丈夫なヤツは投げた方がいい。体で（コツを）つかむというところを目指してほしい」と、投げ込みを奨励した。
