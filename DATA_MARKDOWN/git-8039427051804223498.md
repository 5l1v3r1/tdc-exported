
# విజయవాడ మొత్తం వారిపై ఆధారపడి ఉంది: ఎమ్మెల్యే విష్ణు

విజయవాడ: ఫోటోగ్రాఫర్స్‌కి అవార్డు ఇవ్వడం చాలా సంతోషంగా ఉందని ఎమ్మెల్యే మల్లాది విష్ణు పేర్కొన్నారు. విజయవాడ నగరం అంతా ఫోటోగ్రాఫర్స్‌పై ఆధారపడి ఉందన్నారు. ఆంధ్ర రాష్ట్ర అవతరణ రోజున ఫోటోగ్రాఫర్ ఫంక్షన్ జరగడం చాలా సంతోషంగా ఉందన్నారు. ప్రపంచం అంతమయ్యేవరకు ఫోటోగ్రఫీ ఉంటుందన్నారు. సీఎం ప్రమాణ స్వీకారం సమయంలో తల్లీకొడుకుల భావోద్వేగాన్ని ఫొటోస్‌లో బంధించడం చాలా ఆనందంగా ఉందని విష్ణు పేర్కొన్నారు.
