
# 'നായകനാക്കിയാലും അനില്‍ രാധാകൃഷ്ണന്‍ മേനോന്റെ സിനിമയില്‍ അഭിനയിക്കാനില്ല'; നിലപാട് വ്യക്തമാക്കി ബിനീഷ് ബാസ്റ്റിന്‍

Published at: **2019-11-03T19:58:00+00:00**

Author: **ന്യൂസ് ഡെസ്‌ക്**

Original: [DoolNews](https://www.doolnews.com/will-not-act-in-anil-radhakrishnan-menon-s-films927.html)

കോഴിക്കോട്: നായകനാക്കിയാലും സംവിധായകന്‍ അനില്‍ രാധാകൃഷ്ണന്‍ മേനോന്റെ സിനിമയില്‍ ഇനി അഭിനയിക്കാനില്ലെന്ന് നടന്‍ ബിനീഷ് ബാസ്റ്റിന്‍. തന്നോടല്ല, സമൂഹത്തോടാണ് മാപ്പ് പറയേണ്ടതെന്നും ബിനീഷ് മീഡിയാ വണ്‍ ചാനലിലെ വ്യൂ പോയിന്റില്‍ പറഞ്ഞു.
അനിലുമായുള്ള പ്രശ്‌നങ്ങള്‍ നാളെ നടക്കുന്ന സമവായ ചര്‍ച്ചയിലൂടെ പരിഹരിക്കുമെന്നും ബിനീഷ് വ്യക്തമാക്കി.
‘അനില്‍ രാധാകൃഷ്ണന്‍ മേനോന്റെ സിനിമയില്‍ നായകനാക്കിയാലും ഇനി അഭിനയിക്കില്ല. അഞ്ചോളം സിനിമകള്‍ സംവിധാനം ചെയ്ത അദ്ദേഹം എന്തുകൊണ്ട് എനിക്കു കഥാപാത്രം തന്നില്ല? ഇപ്പോള്‍ ഈ പ്രശ്‌നം വന്നതിനു ശേഷമാണോ എനിക്കു വേഷം തരുന്നത്?’- ബിനീഷ് ചോദിച്ചു.

        
          വാര്‍ത്തകള്‍ ടെലഗ്രാമില്‍ ലഭിക്കാന്‍ ഇവിടെ ക്ലിക്ക് ചെയ്യൂ
        
      
ഒരു രാത്രി മുഴുവനും ലോകത്താകമാനമുള്ള ജനങ്ങള്‍ ഈയൊരു കാര്യത്തില്‍ എനിക്കൊപ്പം നിന്നതാണ്. ഇക്കാര്യത്തില്‍ ജനങ്ങളോടു പ്രതിബദ്ധതയുണ്ട്. ഞാന്‍ ആ ചിത്രത്തില്‍ അഭിനയിച്ചാല്‍ അതു ജനങ്ങളോടു ചെയ്യുന്ന വലിയ തെറ്റായിരിക്കും.
എന്നോടു മാപ്പ് ചോദിക്കേണ്ട ആളല്ല അനില്‍ രാധാകൃഷ്ണന്‍ മേനോന്‍. എന്റെ മനസ്സില്‍ ഇപ്പോഴും അദ്ദേഹത്തിനു വലിയ സ്ഥാനമാണുള്ളത്. അദ്ദേഹത്തെ ഏതു വേദിയില്‍ക്കണ്ടാലും ഞാന്‍ എഴുന്നേറ്റു നില്‍ക്കും. അദ്ദേഹത്തോടു ബഹുമാനമുണ്ട്. എന്നെ ദ്രോഹിച്ചിട്ടുള്ള ആളൊന്നുമല്ല അദ്ദേഹം.’- ബിനീഷ് പറഞ്ഞു.
മെഡിക്കല്‍ കോളേജിലെ കോളേജ് ഡേയ്ക്ക് അതിഥിയായെത്തിയ നടന്‍ ബിനീഷ് ബാസ്റ്റിനൊപ്പം വേദി പങ്കിടാന്‍ കഴിയില്ലെന്ന് കോളേജ് മാസിക പ്രകാശനം ചെയ്യാനെത്തിയ സംവിധായകന്‍ അനില്‍ രാധാകൃഷ്ണന്‍ മേനോന്‍ പറഞ്ഞതായിരുന്നു വിവാദമായത്. ഇന്നലെ വൈകീട്ട് ആറ് മണിക്കായിരുന്നു പരിപാടി.

        
          ഡൂൾന്യൂസ് യൂട്യൂബ് ചാനൽ സബ്സ്ക്രൈബ് ചെയ്യാനായി ഇവിടെ ക്ലിക്ക് ചെയ്യൂ
        
      
തന്റെ സിനിമയില്‍ അവസരം ചോദിച്ച് നടക്കുന്ന ഒരു മൂന്നാംകിട നടനൊപ്പം വേദി പങ്കിടാനാകില്ലെന്ന് സംവിധായകന്‍ പറഞ്ഞെന്നും അതിനാല്‍ പരിപാടി കഴിഞ്ഞ് വന്നാല്‍ മതിയെന്ന് കോളേജ് അധികൃതര്‍ തന്നോട് ആവശ്യപ്പെട്ടതായും ബിനീഷ് വെളിപ്പെടുത്തുകയായിരുന്നു.
