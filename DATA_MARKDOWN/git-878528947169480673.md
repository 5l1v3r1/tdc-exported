
# ＮＨＫ　田代まさし容疑者逮捕で公式サイトから削除　Ｅテレで７月に講師

Published at: **2019-11-06T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/06/0012853943.shtml)

　元タレントの田代まさし容疑者（６３）が６日、覚せい剤取締法違反（所持）容疑で逮捕された。覚醒剤による逮捕はこれで４度目。田代容疑者の逮捕を受けて、ＮＨＫはＥテレ「バリバラ」の公式サイトから今年７月の同容疑者の出演回を削除した。
　田代容疑者は７月４、１１日の２回にわたって同番組に出演。「教えて★マーシー先生」と題し、講師として、薬物依存症をテーマに出演していた。
　放送発表後には、講師として起用することに批判的な声も大きかったが、ＮＨＫ大阪の制作部部長は「依存症の人がどう回復しているか、どう社会と向き合うか、いい材料になる」と説明していた。
　同番組の収録を終えた田代容疑者は６月末、ＮＨＫを通じ、「依存症のことを知らない人とか、いま悩んでいる人とかに届いたらいいな。依存症から回復している姿を見せていきたい」とコメントを寄せていた。
