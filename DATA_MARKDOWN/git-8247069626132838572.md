
# Синоптики объяснили, нормально ли ноябрьское тепло для Кузбасса

Published at: **2019-11-01T08:16:56+00:00**

Author: ****

Original: [Сiбдепо](https://sibdepo.ru/news/sinoptiki-obyasnili-normalno-li-noyabrskoe-teplo-dlya-kuzbassa.html)

Тёплая погода в ноябре – абсолютно нормальное явление для Кузбасса, сообщили Сибдепо в пресс-службе областного Гидрометеоцентра.
По данным синоптиков, максимальная температура воздуха в Кемеровской области, зафиксированная накануне, составляла +16 градусов Цельсия в горной местности региона. А вот по северу области температура воздуха днём даже опускалась до минусовых отметок.
