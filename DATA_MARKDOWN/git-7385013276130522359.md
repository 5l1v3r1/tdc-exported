
# 秋の夜長に聴くアニソンは!? 名曲多数の公式投票企画「2019秋アニメOPテーマ人気投票」スタート！

Published at: **2019-11-06T13:40:00+00:00**

Author: ****

Original: [アキバ総研](https://akiba-souken.com/article/42252/)

アキバ総研にて、本日11月6日（水）より、公式投票企画「2019秋アニメOPテーマ人気投票」がスタートした。

        
          ⇒2019秋アニメOPテーマ人気投票
        
      
毎シーズン恒例の、お気に入りのOP主題歌を決める本企画。今回は、2019秋アニメから46曲をピックアップ。今期は定番のアニソンアーティスト系の楽曲、声優ソングに加え、NoBさんのような大御所シンガーから清春さん、EGO-WRAPPIN'やthe pillows、浅井健一さんのような、まさかのJ-POP、Rock系アーティストと非常に幅広いラインアップが並び、アニソンファンとしてはかなりエキサイティングなシーズンといえるのではないだろうか？
そんな群雄割拠の中、今期はいったいどのOP主題歌がナンバーワンに輝くのか？　ひとり7票まで投票できるので、好きな楽曲にすべて投票するもよし、気になる楽曲に票を振り分けるもよし。思いのままに投票してほしい。
 
なお、投票受付は11月20日（水）まで！
