
# 渋野３４位に後退、横峯は６３位…米女子ツアー台湾選手権第２Ｒ

Published at: **2019-11-01T09:42:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/golf/20191101-OYT1T50236/)

　米女子ゴルフツアーのスウィンギングスカート台湾選手権は１日、台湾・新北の美麗華クラブ（６５０４ヤード、パー７２）で第２ラウンドが行われ、米女子ツアー２戦目に臨んでいる全英女子オープン覇者の渋野日向子は３バーディー、３ボギー、１ダブルボギーで通算１アンダーにスコアを落とし、３４位に後退した。横峯さくらは４オーバーの６３位。首位タイで出たＭ・Ｊ・ホ（韓国）がスコアを六つ伸ばし、１２アンダーで単独首位に立った。
