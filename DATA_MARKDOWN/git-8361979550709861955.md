
# В НАТО отказались от встречи Столтенберга с освобожденными украинскими моряками — СМИ

Published at: **2019-11-02T09:29:00+00:00**

Author: ****

Original: [НВ](https://nv.ua/world/geopolitics/v-nato-otkazalis-ot-vstrechi-stoltenberga-s-osvobozhdennymi-ukrainskimi-moryakami-smi-novosti-ukrainy-50051251.html)

По информации источников, основным противником встречи секретаря Альянса с освобожденными из русского плена моряками была Франция. Они «не хотят раздражать Россию встречей, которая может сорвать мирный процесс и негативно повлиять на организацию переговоров в нормандском формате».
