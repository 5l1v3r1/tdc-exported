
# Полиция открыла дело по факту драки Ляшко с Герусом

Published at: **2019-11-07T11:39:00+00:00**

Author: ****

Original: [Главное™](https://glavnoe.ua/news/n357008316-policija-otkryla-delo-po-faktu-draki-ljashko-s-gerusom)

Главное управление полиции в Киевской области открыло уголовное производство по факту драки бывшего главы фракции "Радикальной партии", экс-депутата Верховной Рады Олега Ляшко с председателем комитета Рады по вопросам энергетики и жилищно-коммунальных услуг Андреем Герусом в международном аэропорту "Борисполь".
Об этом Українським Новинам сообщил источник в правоохранительных органах.
"Полиция открывает дело и будет расследовать", - сказал он.
По его словам, в полицию никто из сторон драки не обращался - ни Ляшко, ни Герус.
В рамках дела полиция намерена допросить самого Ляшко, а также очевидцев - сотрудников аэропорта.
