
# Украина заявила о достатке газа для отопительного сезона без транзита

Published at: **2019-11-05T00:30:14+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-05/ukraina-zayavila-o-dostatke-gaza-dlya-otopitelnogo-sezona-bez-tranzita)

Украина справится с отопительным сезоном без транзита российского газа. Об этом заявил выступил министр энергетики и окружающей среды страны Алексей Оржель.
Он отметил, что страна готова к прекращению транзита газа с Россией, так как на Украине имеется рекордное количество газа для того, чтобы справиться с отопительным сезоном своими силами.
"У нас в украинских ПХГ рекордное количество газа, которое закачано, чтобы пройти отопительный сезон", — заявил Оржель в эфире телеканала ICTV.
