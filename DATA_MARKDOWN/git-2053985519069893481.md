
# كل ما تريد معرفته عن مباراة المنتخب الودية أمام ليبيريا

Published at: **2019-11-07T07:02:00+00:00**

Author: **ياسمين تيسير**

Original: [صدى البلد](https://www.elbalad.news/4048481)

ياسمين تيسير تنطلق مساء اليوم الخميس أولى مباريات المنتخب الأول الودية استعدادا لـ مباريات الجولة الأولى والثانية في التصفيات المؤهلة لبطولة كأس الأمم الأفريقية 2020 .ويرصد موقع "صدى البلد" أبرز المعلومات عن مباراة المنتخب الودية. يواجه المنتخب الاول نظيره ليبيريا في تمام الساعة السادسة مساء على استاد برج العرب وسيذاع المباراة عبر قناة تايم سبورت " بث أرضي " و BEIN SPORT . وأعلن حسام البدرى المدير الفني للمنتخب ضم 32 لاعبا لصفوف المنتخب وجاءت كالآتى : محمد الشناوي (الأهلي ) محمد عوّاد (الزمالك ) محمد بسام (طلائع الجيش) أحمد فتحي (الأهلي ) رجب بكار (بيراميدز) محمد هاني (الأهلي ) عبدالله جمعة( الزمالك ) محمود وحيد (الأهلي ) محمد حمدي (بيراميدز ) محمود علاء (الزمالك ) رامي ربيعة (الأهلي ) باهر المحمدي (الإسماعيلي ) أحمد أيمن منصور (بيراميدز) طارق حامد (الزمالك) عمرو السولية (الأهلي) حمدي فتحي (الأهلي) عمر جابر(بيراميدز) عبدالله السعيد (بيراميدز) محمد مجدي أفشة (الأهلي) حسين الشحات (الأهلي) أحمد السيد زيزو (الزمالك) محمد فاروق (بيراميدز) اسلام عيسي (بيراميدز ) حسام حسن (سموحة ) مروان حمدي (وادي دجله) احمد جمعه (المصري ) أحمد حجازي (وست بروميتش ألبيون الإنجليزي) ومحمد صلاح (ليفربول الإنجليزي ) محمد النني (بشكتاش التركي) أحمد حسن كوكا (براجا البرتغالي) محمود حسن تريزيجيه (أستون فيلا الإنجليزي ) محمود عبدالمنعم كهربا (ديسبورتيفو أفيس البرتغالي ).وسيدير المباراة الحكم الدولي أحمد الغندور وسيعاونه هاني خيري وشريف عبد الله مساعدان، محمود بسيوني حكما رابعا.ومن المقرر أن يحضر المباراة 10 آلاف مشجع لدعم المنتخب الأول .
