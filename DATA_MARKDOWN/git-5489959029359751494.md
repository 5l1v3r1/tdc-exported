
# Tak Dapat Hadiah Penalti, Real Madrid Keluhkan VAR

Published at: **2019-11-03T19:19:00+00:00**

Author: **Bola.net**

Original: [Bola.net](https://www.bola.net/spanyol/tak-dapat-hadiah-penalti-real-madrid-keluhkan-var-b686dd.html)

Bola.net - Real Madrid lewat salah satu petinggi mereka, Emiliano Butragueno mengeluhkan penggunaan VAR dalam laga melawan Real Betis di Santiago Bernabeu, Minggu (3/11/2019) dini hari tadi.
Dalam laga ini, para pemain Real Madrid sempat meminta hadiah penalti kepada wasit ketika umpan silan Karim Benzema dari sisi kiri mengenai tangan pemain Betis, Zou Feddal pada menit-menit akhir babak kedua.
Namun setelah berkonsultasi dengan tim VAR, wasit Jose Maria Sanchez Martinez memutuskan untuk tidak memberi penalti kepada Real Madrid.
