
# Барселона тез орада Куманни бош мураббийликка таклиф қилмоқчи

Published at: **2019-11-05T12:21:00+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/barselona-tez-orada-kumanni-bosh-murabbiylikka-taklif-qilmoqchi)

"Кўк-анорранглилар" келаси йил шартнома муддати тугайдиган Эрнесто Вальверде билан ҳамкорликни давом эттирмоқчи эмас.
Хабар қилинишича, Куман "Барселона" бош мураббийлигига Евро-2020дан сўнг келади.
Эслатиб ўтамиз, Куман 1998-2000 йиллар давомида "Барселона"да ишлаган.
