
# Михаил Казиник в программе «Интервью с Литвиненко», – полное видео

Published at: **2019-11-02T16:40:00+00:00**

Author: ****

Original: [ZIK.UA](https://zik.ua/ru/news/2019/11/02/myhayl_kazynyk_v_programme_yntervyu_s_lytvynenko__onlayntranslatsiya_1682267)


        Сегодня в эфире телеканала ZIK транслировали программу «Интервью с Литвиненко». Гостем стал музыкант, музыковед Михаил Казиник.
      
В разговоре с ведущей проекта Юлией Литвиненко Михаил Казиник, помимо прочего, поделился мыслями относительно исторического общего между Украиной, Россией и Беларусью. По его мнению, эти три страны больные одной исторической болезнью, которая началась с отменой крепостного права - бывшие рабы не социализировались в свободных людей.
