
# В Госдуме призвали выработать сбалансированное решение по вопросу ветхого жилья

Published at: **2019-11-07T10:30:00+00:00**

Author: **Информационная служба Накануне.RU**

Original: [Накануне.ru](https://www.nakanune.ru/news/2019/11/07/22557555/)

В Госдуме, комментируя предложение Минстроя по ремонту ветхого жилья, заявили, что необходимо выработать сбалансированное решение в этом вопросе.
Напомним, в министерстве предложили ветхое жилье ремонтировать за счет граждан до тех пор, пока оно не получит статус аварийного.
Речь идет о домах с износом более 70%. Для них планируется разработать отдельные подпрограммы капремонта. По решению местных властей, эти подпрограммы могут софинансироваться из регионального и муниципального бюджетов.
Органы власти должны быть мотивированы быстрее расселить такие дома, а собственники - не доводить до аварийного состояния, считает первый заместитель председателя Комитета по жилищной политике и ЖКХ Сергей Пахомов.
"Президент поставил задачу, чтобы аварийного жилья расселялось больше, чем признается. Правительство предлагает добиваться этого двумя способами – ускорением процесса расселения жильцов, а также сокращением объема выбывания жилищного фонда из числа пригодного для проживания. С одной стороны, это логичное управленческое решение, с другой - опирается исключительно на терпение людей, проживающих в сложных жилищных условиях", - цитирует парламентария пресс-служба Госдумы.
По его словам, в ходе рассмотрения законопроекта предстоит выработать сбалансированное решение по ветхому жилью. Эти вопросы будут обсуждаться 7 ноября на парламентских слушаниях.
