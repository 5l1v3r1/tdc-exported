
# Захарова призвала Украину сделать шаги для достижения мира

Published at: **2019-11-06T07:46:39+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-06/zaharova-prizvala-ukrainu-sdelat-shagi-dlya-dostizheniya-mira)

Официальный представитель МИД Мария Захарова призвала Украину сделать шаги для достижения мира. Об этом пишет РИА "Новости".
По ее словам, для урегулирования конфликта Киев должен выполнить минские договоренности и "формулу Штайнмайера". 
"Если на Украине есть желание начать движение в сторону мира, то надо это реализовывать", - заявила Захарова.
