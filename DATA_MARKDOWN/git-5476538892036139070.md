
# В Сморгони на пешеходном бульваре начали установку скамеек с гербами всех районов Гродненщины

Published at: **2019-11-07T09:09:00+00:00**

Author: **www.shliah.by**

Original: [Гродненская правда](https://grodnonews.by/news/stroitelstvo/v_smorgoni_na_peshekhodnom_bulvare_nachali_ustanovku_skameek_s_gerbami_vsekh_rayonov_grodnenshchiny.html)

Работы по установке скамеек на пешеходном бульваре по ул. Советской в Сморгони начались утром 6 ноября.Напомним, этот объект реконструировался в рамках подготовки Сморгони к областному фестивалю-ярмарке «Дожинки-2019», который пройдет здесь 23 ноября
