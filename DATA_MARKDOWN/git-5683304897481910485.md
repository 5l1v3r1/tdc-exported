
# സ്കൂള്‍ കന്റീനുകളില്‍ ജങ്ക് ഫുഡിന് നിരോധനം; വിലക്ക് ഈ ഭക്ഷ്യവസ്തുക്കൾക്ക്

Published at: **2019-11-05T15:52:00+00:00**

Author: **മനോരമ ലേഖകൻ**

Original: [ManoramaOnline](https://www.manoramaonline.com/news/latest-news/2019/11/05/junk-food-will-be-banned-in-school-canteen.html?utm_source=dlvr.it&utm_medium=twitter)

തിരുവനന്തപുരം∙ സ്കൂള്‍ കന്റീനുകളില്‍ ജങ്ക് ഫുഡിന് നിരോധനം വരുന്നു. 50 മീറ്റര്‍ ചുറ്റളവിലും ജങ്ക് ഫുഡുകൾ  വില്‍ക്കാനാവില്ലെന്ന് കേന്ദ്ര ഭക്ഷ്യ സുരക്ഷാ അതോറിറ്റി അറിയിച്ചു. കോള, ചിപ്സ്, ബര്‍ഗര്‍, പീസ, ഗുലാബ്ജാമൂന്‍, കാര്‍ബണേറ്റഡ് ജ്യൂസുകള്‍ തുടങ്ങിയവക്കാണ് വിലക്ക്. അടുത്ത മാസം മുതല്‍ നിരോധനം പ്രാബല്യത്തില്‍ വരും. കായികമേളകളില്‍ ഇത്തരം ഭക്ഷ്യവസ്തുക്കളുടെ പരസ്യം അനുവദിക്കില്ലെന്നും അധികൃതർ വ്യക്തമാക്കി. 'ഈറ്റ് റൈറ്റ്' എന്ന കേന്ദ്രസർക്കാരിന്റെ ക്യാംപെയിനിന്റെ ഭാഗമായാണ് ഭക്ഷ്യസുരക്ഷ അതോറിറ്റി സ്കൂൾ കുട്ടികളുടെ ഭക്ഷ്യക്രമം സംബന്ധിച്ച മാർഗനിർദേശങ്ങളിൽ സുപ്രധാന തീരുമാനം ഉൾപ്പെടുത്തിയിട്ടുള്ളത്.
