
# Na ministrstvu za delo prejeli bel prah

Published at: **2019-11-04T10:44:26+00:00**

Author: **STA**

Original: [Primorske novice](https://www.primorske.si/kronika/na-ministrstvu-za-delo-prejeli-bel-prah)


        Na ministrstvu za delo, družino, socialne zadeve in enake možnosti s sedežem za Bežigradom so danes zjutraj prejeli kuverto, iz katere se je vsul bel prah. Ta je po rezultatih prve analize nenevaren, so sporočili s Policijske uprave Ljubljana. Policisti nadaljujejo z zbiranjem obvestil o dogodku.
      
LJUBLJANA > Pošiljko, iz katere se je usul bel prah, so odprli okoli 8.30. Osebe, ki so bile v stiku s snovjo, so za nujno potreben čas izolirali od okolice. Kraj dogodka so zavarovali policisti, gasilci in delavci veterinarske fakultete ter izvedli ukrepe, ki so v teh primerih predvideni.
Rezultati prve analize, ki so jo opravili na inštitutu za mikrobiologio in parazitologijo ljubljanske veterinarske fakultete, so pokazali, da ne gre za nevarno snov, so sporočili s policije.
