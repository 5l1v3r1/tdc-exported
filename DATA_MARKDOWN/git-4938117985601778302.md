
# Сеть рассмешил «стильный» тюнинг ВАЗ-2107

Published at: **2019-11-06T00:00:00+00:00**

Author: **Дмитрий Плотников**

Original: [CAR.RU](https://car.ru/news/automobili/56985-set-rassmeshil-stilnyiy-tyuning-vaz-2107/)


        В социальной сети «ВКонтакте» в одном из сообществ была опубликована фотография тюнингованного ВАЗ-2107, вызвавшая массу едких комментариев интернет-пользователей.
      
На фото автомобиль снят исключительно сзади, дабы выделить основной элемент тюнинга. Им оказался приобретенный не по размеру спойлер, вызвавший смех пользователей популярной соцсети.
Назначение спойлера на спорткарах понять не сложно. Он служит для увеличения прижимной силы на заднюю часть кузова и увеличения, как вследствие, аэродинамических свойств автомобиля. Габаритный же спойлер на «семерке» вызвал бурный смех в рядах пользователей и массу едких комментариев, назвавших такой тюнинг «колхозным».
 
«Машина - огонь! Как раз для дворовых посиделок. Осталось только поляну накрыть», «Судя по величине спойлера, с хозяином жигуля лучше не спорить. Мужик сурьезный», «Это что, лезвия от бритвы Gillette 2», «Братан, с какого танка ты этот спойлер снял?», - пишут остряки.
Словом, не оценило интернет-сообщество пламенные порывы владельца советского автомобиля. А как вам?
