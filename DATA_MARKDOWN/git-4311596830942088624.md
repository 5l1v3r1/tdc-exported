
# Эксперт: Говорить слово «да» по телефону незнакомцам стало опасно

Published at: **2019-11-02T15:21:00+00:00**

Author: ****

Original: [EADaily](https://eadaily.com/ru/news/2019/11/02/ekspert-govorit-slovo-da-po-telefonu-neznakomcam-stalo-opasno)

Российским гражданам не следует при разговоре по телефону с незнакомцами говорить слово «да», так как в этом случае они рискуют стать жертвой аферистов.
Как пояснил изданию АиФ директор по стратегическим коммуникациям Infosecurity Softline Company Александр Дворянский, именно с помощью таких простых слов злоумышленники собирают биометрию — цифровые отпечатки голоса.
Чаще всего, аферисты под видом социологов задают стандартные вопросы, провоцируя собеседника отвечать односложно, например, словом «да. Таким образом они собирают цифровые отпечатки голоса, которые впоследствии могут использовать при работе с банком.
Этот новый тип мошенничества стал актуальным после того, как российские банки начали оказывать услуги без личного присутствия клиента. Сейчас биометрию можно сдать в 140 банках.
Поскольку система защиты банка может не поверить одному только телефонному «да», вырванному из контекста, мошенники в разговоре пытаются добиться, чтобы жертва также произнесла фразу «я подтверждаю».
По словам эксперта, чтобы обезопасить себя, лучше отвечать на звонки обычным «алло». А лучше вовсе не отвечать на незнакомые номера, а реальный номер своего банка добавить в контакты на телефоне.
