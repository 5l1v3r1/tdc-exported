
# ผู้นำจีนให้กำลังใจหัวหน้าคณะบริหารเกาะฮ่องกง

Published at: **2019-11-05T01:16:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/foreign/740031)


        ประธานาธิบดีสี จิ้นผิง ยืนยันความสนับสนุนอย่างเต็มที่ของรัฐบาลกลางในกรุงปักกิ่ง ต่อคณะผู้บริหารฮ่องกงของนางแคร์รี แลม ด้านตำรวจฮ่องกงจับกุมผู้ต้องสงสัยเกี่ยวข้องกับเหตุนองเลือดถึงขั้นใช้มีดแทงและมีการกัดหู ระหว่างการประท้วงเมื่อวันอาทิตย์ที่ผ่านมา
      
สำนักข่าวต่างประเทศรายงานจากเมืองเซี่ยงไฮ้ ประเทศจีน เมื่อวันที่ 5 พ.ย. โดยอ้างจากรายงานของสำนักข่าวซินหัว ว่าประธานาธิบดีสี จิ้นผิง พบกับนางแคร์รี แลม หัวหน้าคณะผู้บริหารฮ่องกง ที่งานมหกรรมแสดงสินค้านำเข้านานาชาติจีน ( ซีไอไออี ) เมื่อวันจันทร์ ซึ่งผู้นำจีนยืนยัน "ความเชื่อมั่นในระดับสูง" ต่อการปฏิบัติหน้าที่ของคณะผู้บริหารฮ่องกงชุดปัจจุบันภายใต้การนำของแลม ขณะเดียวกันรัฐบาลปักกิ่งรับทราบและมีความเข้าใจต่อความพยายามของแลม ในการฟื้นฟูความสงบและเสถียรภาพให้กลับคืนสู่สังคม อย่างไรก็ตาม ผู้นำจีนขอให้แลมและคณะเพิ่มความพยายาม ในการหาหนทางยกระดับคุณภาพชีวิตของชาวฮ่องกงในทุกภาคส่วน และย้ำว่าการภิจสำคัญที่สุดในตอนนี้คือการยุติการประท้วงที่ใช้ความรุนแรง
