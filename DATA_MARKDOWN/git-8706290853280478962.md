
# To μήνυμα της αδερφής του Σάλα που ραγίζει καρδιές

Published at: **2019-11-02T05:46:00+00:00**

Author: ****

Original: [enikos.gr](http://www.enikos.gr/sports/681656/to-minyma-tis-aderfis-tou-sala-pou-ragizei-kardies-photo)

Αν ζούσε, θα είχε τα γενέθλιά του στις 31 Οκτωβρίου. Ο Εμιλιάνο Σάλα, ο οποίος σκοτώθηκε σε αεροπορικό δυστύχημα τον περασμένο Ιανουάριο, θα γινόταν 29 χρόνων.
Η αδερφή του, Ρομίνα, μοιράστηκε τις σκέψεις της σε ένα σπαρακτικό μήνυμα στο instagram: «Δεν υπάρχουν λέξεις για να περιγράψουν τόσο μεγάλο πόνο. Ευτυχισμένα γενέθλια αδερφέ!! Όπου και αν είσαι, δώσε μας τη δύναμη για να συνεχίσουμε. Θα ήθελα να ήταν όλα διαφορετικά. Σου υπόσχομαι πως δεν θα σταματήσουμε ποτέ, μέχρι τέλους. Φιλιά στον παράδεισο».
