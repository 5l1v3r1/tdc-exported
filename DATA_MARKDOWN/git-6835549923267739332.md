
# Лучич нокаутировал хоккеиста Коламбуса [видео]

Published at: **2019-11-03T06:58:00+00:00**

Author: ****

Original: [Allhockey.ru](https://allhockey.ru/news/show/368849-Luchich_nokautiroval_hokkeista_Kolambusa_video)

"Коламбус" уступил "Калгари" со счетом 0:3.  Нападающий "Калгари" Милан Лучич вновь отметился грубой игрой. В одном из эпизодов матча Лучич ударил кулаком в лицо игрока "Коламбуса" Коула Шервуда. Шервуд, пытаясь сыграть на добивании, попал клюшкой по щиткам голкиперу "Калгари" Давиду Риттиху, после чего Лучич решил разобраться с форвардом "Блю Джекетс". По итогам эпизода Шервуд заработал малый штраф, в свою очередь Милан был наказан двойным малым штрафом.  Посмотреть видео эпизода можно в телеграмм аккаунте AllHockey
