
# เอ๋ ชนม์สวัสดิ์โดนแล้ว สาวทอมปลอมเฟซหลอกเรี่ยไร

Published at: **2019-11-01T10:17:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/crime/739518)


        ตำรวจปากน้ำตามรวบสาวหล่อ ถึงหาดใหญ่ สุดแสบปลอมเฟซบุ๊ก”เอ๋ ชนม์สวัสดิ์” ขอเรี่ยไรเงินทำบุญ อ้างตกงาน อายุมาก แค่หาเงินใช้
      
เมื่อวันที่ 1 พ.ย. พ.ต.อ.ธงรบ  แจ้งจิต ผกก.สส.ภ.จว.สมุทรปราการ พ.ต.ท.อรรถพล  มิขำ สว.สส. พร้อมกำลัง ร่วมกันจับกุมตัวน.ส.ชยดล  คิดโสดา อายุ 33 ปี อยู่บ้านเลขที่ 39/3 ม.3 ต.สุไหงปาดี อ.สุไหงปาดี จ.นราธิวาส ผู้ต้องหาตามหมายจับของศาลจั​งหวัดสมุทรปราการ ที่ 686 / 2562 ลงวันที่ 29 ตุลาคม 2562 ในข้อหา ฉ้อโกงโดยการแสดงตนเป็นคนอื่น โดยจับได้ที่ ห้องเช่ารายวันแห่​งหนึ่งใน อ.หาดใหญ่ จ.สงขลา ก่อนควบคุมตัวกลั​บมาทำการสอบสวนที่ กองสืบสวนตำรวจภูธรจังหวัดสมุ​ทรปราการ   สืบเนื่องจากก่อนหน้านี้​มีผู้เสียหายจำนวนหลายรายได้เข้​าแจ้งความร้องทุกข์ต่อพนั​กงานสอบสวน สภ.พระประแดง และ สภ.บางแก้ว จ.สมุทรปราการ ว่า ถูกน.ส.ชยดล  ปลอมเฟซบุ๊ก โดยใช้ชื่อเฟส Chonsavat Asawahame ซึ่งเป็นชื่อของนายเอ๋ ชนม์สวัสดิ์  อัศวเหม ประธานหอการค้าจังหวัดสมุ​ทรปราการ แล้วเรี่ยไรเงิน โดยอ้างว่าไปทำบุ​ญทอดกฐินและผ้าป่า ทำให้ผู้เสียหายเข้าใจผิดคิดว่า เป็นเฟซของนายชนม์สวัสดิ์  จริง จึงได้โอนเงินเข้​าบัญชีที่ลงไว้ในเฟซบุ๊กดั​งกล่าว ต่อมาเหยื่อได้ตรวจสอบและพูดคุยกั​บนายชนม์สวัสดิ์  แต่เจ้าตัวไม่ทราบเรื่องดังกล่าวและไม่​เคยเปิดเฟซบุ๊กเพื่อเรี่ยไรเงิ​นไปทำบุญแต่อย่างใด จึงรู้ว่าถูกหลอก จากนั้นพากันเข้​าแจ้งความ จนชุ​ดสืบสวนติดตามจับกุมตัวผู้​ก่อเหตุได้ดังกล่าว     
จากการสอบสวนน.ส.ชยดล ยอมรับสารภาพว่า ก่อนหน้านี้ทำงานคัดผลส้มอยู่​กับพี่สาวที่จ.เชียงใหม่ แต่เงินไม่พอใช้จึงลาออกและเดิ​นทางกลับมาที่บ้านเกิดในจ.นราธิวาส และหางานทำไม่ได้ จึงทำให้ไม่มีเงินใช่ ประกอบกับตนชื่นชอบความใจกว้​างของนายชนม์สวัสดิ์ และเคยพบเจอนายชนม์สวัสดิ์ มาแล้วหลายครั้งจึงรู้ว่าเป็​นคนใจดีและชอบทำบุญช่วยเหลือคน จึงได้เอารูปที่เคยถ่ายกั​บนายชนม์สวัสดิ์ มาขึ้นเป็นโปรไฟล์ และใช้ชื่อของนายชนม์สวัสดิ์ และนามสกุล เป็นภาษาอังกฤษ มาเปิดเฟสเพื่อเรี่ยไร เพื่อหาเงินใช้ ยอมรับว่าที่ทำลงไปเพราะเป็​นอารมณ์ชั่ววูบ แต่เงินก็ไม่ได้เข้ามาทุกวัน และยอดเงินที่ได้ก็ไม่ได้​มากมายอะไร ที่ทำไปเพราะตนตกงานจริง ๆ  อายุก็เยอะแล้ว จะไปสมัครงานที่ไหนก็ลำบาก ตอนนี้ตนสำนึกผิดแล้​วอยากจะขอกราบขอโทษนายชนม์สวั​สดิ์ ในเรื่องที่ได้ทำลงไป  เบื้องต้นเจ้าหน้าที่ได้แจ้งข้​อกล่าวหา ก่อนควบคุมตัวส่งพนักงานสอบสวน สภ.พระประแดง เพื่อดำเนินคดีตามกฎหมายต่อไป.
