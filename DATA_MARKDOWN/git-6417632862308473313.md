
# Реал директори белгиланмаган пенальти ҳақида: ВАР нега ҳар хил қарор қабул қилишини тушунмайман

Published at: **2019-11-03T05:59:10+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/real-direktori-belgilanmagan-penalti-haqida-var-nega-har-xil-qaror-qabul-qilishini-tushunmayman)

82-дақиқада севильяликлар ҳимоячиси Зухаир Феддаль ўз жарима майдончаси ичида Карим Бензема узатмасини қўли билан қайтарди, аммо ВАР текширувидан кейин 11 метрлик жарима зарбаси белгиланмади.
"Бу қарорга тушунмадим. Ўша вазиятда қўл тўп йўналишини тўлиқ ўзгартирди, аммо пенальти белгиланмади.
ВАР нега ҳар хил қарор қабул қилишини тушунмайман. Бундай вазиятларда тизим орқали аниқ қарор чиқарилиши керак, бироқ ушбу ўйинда бу ҳолат кузатилмади.
11 метрлик жарима зарбаси бўлади деб ўйлагандик, аммо кутганимиздек бўлмади" - дейди Бутрагеньо.
"Реал" - "Бетис" учрашувини Хосе Мария Мартинес бошқарди.
