
# Опубликованы индивидуальные постеры героев дорамы «Люди с недостатками»

Published at: **2019-11-05T22:49:48+00:00**

Author: **gunyasha**

Original: [YesAsia](https://www.yesasia.ru/article/754242)

Создатели новой дорамы MBC «Люди с недостатками» (рабочее название) опубликовали индивидуальные постеры основных героев проекта.

С сюжетом можно ознакомиться здесь.
Постер Джу Соён (О Ён Со) гласит: «Ненавижу красавчиков».
Постер Ли Кану (Ан Джэ Хён) гласит: «Я такой красавчик!».
Постер Ким Мигён (Ким Сыль Ги) гласит: «Найти_богатого_мужа».
Постер Ли Минхёка (Гу Вон) гласит: «Я не могу врать».
Постер Пак Хёнсу (Хо Чон Мин) гласит: «Я люблю деньги».
Премьера дорамы состоится 27 ноября в 20:55 по корейскому времени.
