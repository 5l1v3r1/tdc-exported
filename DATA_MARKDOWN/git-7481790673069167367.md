
# 新東名でトラック炎上、一部区間で通行止め９時間

Published at: **2019-11-03T02:53:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191103-OYT1T50123/)

　２日午後１１時３５分頃、愛知県新城市川田の新東名高速道路下り線で、大型トラックから火が出たと、運転していた会社員男性（５１）から１１０番があった。消防が消火に当たったが、約１時間後に全焼した。男性にけがはなかった。
　県警高速隊によると、この火災で、新城インターチェンジ（ＩＣ）―岡崎東ＩＣ間が２日午後１１時４０分頃から３日午前８時５０分までの計９時間、通行止めになった。現場の路面が損傷したため、通行止め解除後も、２車線のうち、１車線（走行車線）を約５キロにわたって規制し、修復作業が行われている。
