
# 「少女☆歌劇 レヴュースタァライト」再生産総集編＆完全新作劇場版の制作が決定！

Published at: **2019-11-07T11:00:00+00:00**

Author: ****

Original: [アキバ総研](https://akiba-souken.com/article/42273/)

株式会社ブシロードは、2019年11月3日（日）の3rdスタァライブ“Starry Diamond”にて、アニメ「少女☆歌劇 レヴュースタァライト」の再生産総集編と、完全新作劇場版2作品の制作決定を発表した。
「少女☆歌劇 レヴュースタァライト」は、ミュージカルとアニメーションがリンクして展開する新感覚エンターテインメント。それぞれの夢、願いを胸に“スタァ”を目指して名門音楽学校に通う9人の少女たちのもとに、突然舞い込んだ謎の招待メール。このメールをきっかけに、乙女たちの壮絶なバトルロワイヤルが始まるという設定だ。2017年9月よりミュージカル「少女☆歌劇 レヴュースタァライト -The LIVE- #1」が上演開始されたのをきっかけに、2018年夏にはTVアニメが放送され、話題を呼んだ。また、同年10月には舞台「少女☆歌劇 レヴュースタァライト -The LIVE-#2 Transition」も上演。アニメとミュージカルを軸に展開する「二層展開式少女歌劇」となっており、メインキャストの9人がアニメでの声優とミュージカルキャストのどちらも演じるのが特徴のプロジェクトだ。
 
■「劇場版 少女☆歌劇 レヴュースタァライト」制作決定2019年11月3日（日）に横浜アリーナにて行われた3rdスタァライブ“Starry Diamond”にて、テレビアニメシリーズを再編集した総集編映画「少女☆歌劇 レヴュースタァライト 再生産総集編 ロンド・ロンド・ロンド」が、2020年初夏公開と発表された。
 
 加えて、完全新作劇場版「劇場版 少女☆歌劇 レヴュースタァライト」についても制作決定が発表されている。※今後の展開については続報をお待ちください。
 
■【特報PV】劇場版「少女☆歌劇 レヴュースタァライト」また、劇場版2作品の制作決定を受けて、特報PVも解禁された。
