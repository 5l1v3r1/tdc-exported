
# Экс-директор госпредприятия организовал контрабанду оборудования для ВМФ России

Published at: **2019-11-07T16:30:56+00:00**

Author: ****

Original: [ФАКТИ](https://fakty.com.ua/ru/ukraine/20191107-eks-dyrektor-derzhpidpryyemstva-organizuvav-kontrabandu-obladnannya-dlya-vmf-rosiyi/)

В Запорожской области бывшего руководителя госпредприятия уличили в контрабанде оборудования для военных кораблей РФ.
Бывший коммерческий директор оборонного завода по производству оборудования для военных катеров погорел на контрабанде его продукции в страну-агрессор.
Он скупал оборудование у отечественных частных структур, работающих по лицензии завода, а затем поставлял в Россию через так называемые фирмы-прокладки.
Его незаконной деятельности способствовали должностные лица Запорожской таможни.
Задержали его вместе с сообщником на таможенном посту Мелитополь при оформлении оборудования, предназначенного для охлаждения жидкости на военных кораблях.

В офисе и дома у фигурантов сотрудники СБУ изъяли много запчастей к военному оборудованию и бухгалтерскую документацию, подтверждающую нелегальные сделки.
Открыто производство по ст. 333 УКУ – Нарушение режима осуществления международной передачи товаров, подлежащих государственному экспортному контролю.
Решается вопрос о привлечении к ответственности таможенников, причастных к контрабанде военного оборудования в Россию.
