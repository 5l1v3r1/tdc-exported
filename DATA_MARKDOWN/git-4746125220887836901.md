
# ノンスタ井上、そっくり診断で“中村倫也似”の結果「これは似てる(笑)」「目の錯覚です」

Published at: **2019-11-01T15:58:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147829/full/)

　お笑いコンビ・NON STYLEの井上裕介が1日、自身のツイッターを更新。“そっくり診断”の結果を公表した。 　井上は「そっくり診断やってみた!! こりゃ、世論に叩かれること間違いなしだわね^_^ でも、この診断は僕が決めたんじゃないからね」と、アプリを使った診断結果を画像投稿。 　そこには「あなたとそっくりの芸能人は…中村倫也」と書かれており、両者の顔写真が表示されている。この結果に井上は「ウフフっ!! 嬉しい^_^嬉しい^_^嬉しい^_^ イェーーイ!! 今日1日、ハッピーな気持ちで過ごせるぜ」と上機嫌な様子をみせた。 　コメント欄には「ちょっと似てるとか思ってしまった自分を殴りたい……」「これは似てる（笑）」「まぁ、似てるっちゃ似てるね」という肯定派、「井上さんは好きだし、別に特別ブサイクだとも思ってないけど、これは許せない！笑笑」「目の錯覚です」「やめろっ！」という否定派が、それぞれの意見をつぶやいている。
