
# Лайфхак: как быстро подвесить цветочный горшок при помощи верёвки

Published at: **2019-11-03T00:00:06+00:00**

Author: **Виктор Подволоцкий**

Original: [Лайфхакер](https://lifehacker.ru/kashpo-rope/)

Подойдёт даже бельевая верёвка.
Способов подвесить цветочный горшок к потолку или крючку на стене довольно много. Самый простой из них — купить специальное кашпо с уже готовыми стропами и петелькой. Если же вы хотите обойтись без лишних трат, надёжную обвязку для горшка можно сделать и самостоятельно.
Этот способ подойдёт практически для любого сосуда округлой формы. Подойдёт даже бельевая верёвка. Главное — рассчитать с длиной. Чем она больше, тем ниже будет висеть горшок.
