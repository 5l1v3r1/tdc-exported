
# มาเนโนสนโนแคร์ถ้าพุ่งแล้วได้จุดโทษก็จะพุ่งอีก

Published at: **2019-11-07T13:41:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/sports/740534)


        "มาเน" โนสนโนแคร์ ถ้าพุ่งแล้วได้จุดโทษก็จะพุ่งอีก แต่ยืนยันจังหวะในเกมกับ "วิลลา" ถูกทำฟาวล์จริง
      
ซาดิโอ มาเน แนวรุกตัวเก่งของ "หงส์แดง" ลิเวอร์พูล เปิดเผยถึงประเด็นที่ถูก เปป กวาร์ดิโอลา กุนซือชาวสแปนิชของ "เรือใบสีฟ้า" แมนฯ ซิตี โจมตีว่าเป็นพวกชอบพุ่งว่า หากมันทำให้เขาได้จุดโทษ เขาก็จะทำอีก โดย มาเน ถูกโจมตีจากจังหวะในเกมที่ ลิเวอร์พูล บุกชนะ แอสตัน วิลลา 2-1 ว่าเขาพยายามพุ่งล้มเอาจุดโทษ ทำให้ถูกผู้ตัดสินแจกใบเหลืองลงโทษมาเน กล่าวว่า "ถ้าการพุ่งจะทำให้ผมได้จุดโทษ ผมก็จะทำมันอีก ทำไมจะไม่ละ แต่ เจอร์เกน (คลอปป์) พูดถูก ผมไม่ได้พุ่ง มีการสัมผัสตัวเกิดขึ้น บางทีมันก็ไม่ใช่จุดโทษ เพราะกรรมการไม่ให้ และเขาแจกใบเหลืองผม ด้วยความสัตย์จริง ผมไม่มีปัญหากับเรื่องนี้เลย มันเป็นอะไรที่ผมจะทำต่อไป ถ้าผมได้สิ่งตอบแทนเป็นจุดโทษ ถ้าไม่ได้จุดโทษ มันก็จะไม่เปลี่ยนแนวทางการเล่นของผม หรือสิ่งที่ผมทำ สิ่งเดียวที่ผมจะพูดได้ก็คือ ผมพร้อมจะมอบทุกสิ่งทุกอย่างเพื่อช่วยเหลือทีม".
