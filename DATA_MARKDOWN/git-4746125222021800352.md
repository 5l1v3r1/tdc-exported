
# 市井紗耶香、芸能事務所との契約を報告「タレント活動の幅を広げてまいります」

Published at: **2019-11-01T10:17:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147789/full/)

　今年7月の参院選に立憲民主党から比例代表で立候補し、落選した元モーニング娘。市井紗耶香（35）が1日、自身のインスタグラムを更新。芸能事務所・アンカンシェル（代表：中野尚美）とマネジメント業務提携契約を結んだことことを報告した。 　「タレント活動の業務提携について」として市井は「株式会社エスダブル（代表市井紗耶香）と、株式会社アルカンシェル（代表中野尚美）は、タレント市井紗耶香のマネジメントについて、このたび業務提携を行うこととなりました」と報告。今後「株式会社エスダブル及び株式会社アルカンシェル両者が連携して、市井紗耶香のタレント活動の幅を広げてまいります」と伝えた。 　「尚美ママとはかれこれ数年前からお世話になっていて、先の参議院選の後の活動について色々とご相談に乗っていただいておりました」といい「この先4人の子どもを抱え守っていく中で、選挙に挑戦させていただいた経験は決して無駄ではなく、むしろ、私のこの先の人生の中で学びの場をもっと増やしていけたらと可能性を生み出すことができました」。 　「毎日頑張っている全国のお母さん方お父さん方ともっと沢山話せる場所も作っていきたい。芸能も勿論、枠にはまらず、人生の勉強となることはとことん挑戦して行ったら良い。と尚美ママは背中を押して下さり、このような私でも何か力になれればと手を差し伸べてくださいました」と明かし「3年前に小さくはじめた個人事務所ですが今こうしてまた新たな門出を迎えることができとても幸せです」と思いを記した。 　最後は「今後も個人事務所（株）エスダブルでいただけるお仕事に関しましては引き続き責任を持ち背筋を正しながら行ってまいります」と気持ちを新たにし「皆様、いつもありがとうございます。今後とも宜しくお願い申し上げます」と呼びかけている。
