
# Манчестер Юнайтеднинг октябрь ойидаги энг яхши футболчиси маълум бўлди

Published at: **2019-11-02T06:02:39+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/manchester-yunaytedning-oktyabr-oyidagi-eng-yaxshi-futbolchisi-malum-boldi)

Унга кўра ўтган ойдаги энг яхши футболчи деб Скотт Мактоминей эътироф этилди.

        Скотт Мактоминей - 48%
      
Маркус Рэшфорд - 39%
Дэниэль Жеймс - 13%
Шотландиялик ўйинчи жорий мавсум 13та ўйинда майдонга тушиб, 2та гол ва 1та ассистга эришди.
Олдинроқ, айнан Скотт "қизил иблислар"нинг АПЛдаги 2000-голига ҳам муаллифлик қилган эди.
