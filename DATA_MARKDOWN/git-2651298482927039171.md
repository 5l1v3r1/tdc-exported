
# РУСЛАН АЛХАНОВ ПРОИНСПЕКТИРОВАЛ СТРОИТЕЛЬСТВО СОЦИАЛЬНЫХ ОБЪЕКТОВ В ШЕЛКОВСКОМ РАЙОНЕ

Published at: **2019-11-01T21:36:17+00:00**

Author: ****

Original: [ЧГТРК &#34;Грозный&#34;](https://grozny.tv/news.php?id=34789)


        Министр внутренних дел ЧР курирует сектор, в который входит муниципалитет.
      
В районе строят новую школу, детский сад, фельдшерско-акушерский пункт, игровые и спортивные площадки. Также здесь возводят жилой комплекс для переселения 500 семей из аварийного жилья. Все работы финансирует РОФ имени Первого Президента ЧР, Героя России Ахмата-Хаджи Кадырова.
Напомним, по поручению Рамзана Кадырова Чеченская Республика была разделена на сектора. Каждый участок закреплен за ответственным лицом. Задача кураторов обеспечить прямой контроль над решением вопросов социально-экономического развития каждого из курируемых районов.
