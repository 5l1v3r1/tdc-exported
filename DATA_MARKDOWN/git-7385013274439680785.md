
# 「ゆるキャン△」の保冷バッグ、スマホケース、アクリルスタンドの受注が開始！

Published at: **2019-11-07T18:20:00+00:00**

Author: ****

Original: [アキバ総研](https://akiba-souken.com/article/42271/)

株式会社アルマビアンカは、「日常でも使用できる」をコンセプトにしたオリジナルグッズを展開する通販サイト「AMNIBUS」にて、「ゆるキャン△」商品の受注を2019年11月1日（金）より開始した。
「ゆるキャン△」は、あfろさんが「まんがタイムきららフォワード」（芳文社）にて連載している人気コミック。山梨県を舞台に、女子高校生たちがキャンプをしたり野外料理をしたりと、アウトドアライフを満喫する様子をゆるやかに描いた人気作。アニメ版では、スタッフ陣が数回に渡るロケハンを敢行、徹底したアウトドア・日常・風景の描写が好評を博した。また2020年1月より、シリーズ最新作として、ショートアニメ「へやキャン△」が放送されることも決定している。⇒TVアニメ「ゆるキャン△」シリーズ最新作、ショートアニメ「へやキャン△」が、2020年1月に放送決定！
今回受注を開始したのは、保冷バッグ、NordiQ 手帳型スマホケース、トレーディング NordiQ アクリルスタンドの3種だ。
 
■NordiQ 保冷バッグなでしこ、リン、千明、あおい、恵那を北欧風テイストで描き起こした保冷バッグ。テントやバイク、木のモチーフと合わせてレイアウトしたデザインに仕上がっている。内側はアルミ蒸着シートで保冷保温に対応し、500mlのペットボトルが約8本収納できる。ポリエステル生地のため、汚れもさっと拭けてお手入れも簡単なアイテムだという。
価格：3,980円（税別）サイズ：（約）H21cm×W26cm×D15cm素材：ナイロン、アルミ
 
■NordiQ 手帳型スマートフォンケースなでしこ、リン、千明、あおい、恵那を北欧風テイストで描き起こしたスマートフォンケース。テントのモチーフと組み合わせて、ナチュラルなカラーリングでデザインした。スマートフォンが上下に移動するスライド式で、iPhoneやAndroid端末などに対応するとのこと。
