
# Adobe представила бесплатное приложение, которое редактирует снимки с помощью нейросети

Published at: **2019-11-07T10:34:45+00:00**

Author: **Анна Логачева**

Original: [The Village](https://www.the-village.ru/village/city/news-city/366521-adobe)

Adobe представила бесплатное приложение Photoshop Camera, которое позволяет автоматически редактировать фотографии с помощью искусственного интеллекта.
Программа работает на платформе Sensei, которая распознает объекты как на готовых снимках, так и в режиме реального времени. После чего приложение предлагает подходящие фильтры и эффекты.
