
# Latest Microsoft To Do for iOS update brings Face ID support and more

Published at: **2019-11-06T09:34:00+00:00**

Author: **Pradeep**

Original: [MSPoweruser](https://mspoweruser.com/latest-microsoft-to-do-for-ios-update-brings-face-id-support-and-more/)

Microsoft today released a new update for Microsoft To Do for iOS app in Apple App Store. This Version 2.4 update brings support for FaceID if you use To Do app with Mobile Device Management.
Microsoft last month updated the To Do app with dark mode support for iOS 13 and iPadOS 13 users. With the new update, Microsoft has extended the dark mode support. Dark mode is now supported in the help articles too, so you can rest your eyes while you search through our tips and tricks.
Microsoft recently added URL preview in To Do. With this update, if don’t want to see a preview of a URL in your notes, you can turn it off in To Do app settings.
