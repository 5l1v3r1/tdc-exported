
# Мэрия Новосибирска снесет кафе «Карат»

Published at: **2019-11-02T07:26:00+00:00**

Author: ****

Original: [RuNews24](https://runews24.ru/novosibirsk/02/11/2019/474039e9c361aed9d8ee917991ac6c23)

По информации пресс-службы мэрии Новосибирска, накануне на улице Авиастроителей, 1в в Дзержинском районе начали демонтаж незаконного объекта – кафе «Карат». Работы проводятся по решению суда.
Владельцев точки ранее предупреждали, что они должны самостоятельно демонтировать здание, однако этого не случилось. Теперь демонтаж принудительно проведут Служба судебных приставов и мэрия города.
На данной территории планируется организовать сквер, который будет включать площадки для занятий спортом.
