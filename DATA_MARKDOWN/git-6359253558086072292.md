
# В Челябинской области девятимесячный младенец отравился крысиным ядом

Published at: **2019-11-05T12:16:27+00:00**

Author: ****

Original: [Новый День](https://newdaynews.ru/chel/675753.html)

В Челябинской области проводится проверка по факту отравления 9-месячного ребенка.
Как сообщили РИА «Новый День» в следственном управлении СКР по Челябинской области, днем 2 ноября в больницу Магнитогорска из одного из домов по улице Ярославского с признаками отравления поступила 9-месячная девочка.
По данному факту проводится процессуальная проверка. По предварительным данным, ребенок отравился крысиным ядом. Девочка помещена в реанимационное отделение, медики оценивают состояние ребенка, как стабильно тяжелое.
В настоящее время устанавливаются все обстоятельства произошедшего. Проведение проверки взято на контроль руководством следственного управления. По результатам проверки будет принято процессуальное решение.
