
# В Мелитополе перекроют центральный проспект

Published at: **2019-11-01T08:42:00+00:00**

Author: **Владимир Андреев**

Original: [РИА Мелитополь](https://ria-m.tv/news/170182/v_melitopole_perekroyut_tsentralnyiy_prospekt.html)


        Перекрывают движение в связи с ремонтными работами.
      
В связи с проведением ремонтных работ по пр. Б. Хмельницкого (от ул. Покровской до ул. Гетмана Сагайдачного) 01.11.2019 с 08.00 до 18.00 часов будет затруднено движение транспорта с возможным перекрытием.
