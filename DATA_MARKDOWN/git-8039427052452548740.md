
# డెంగీతో ఇద్దరు మృతి

Published at: **2019-11-04T03:25:26+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=945638)

Ph: 88865 47666ప్రవేశం ఉచితం ADVT
పాలకొల్లు టౌన్‌/శంఖవరం, నవంబరు 3: డెంగీ బారిన పడి తూర్పు, పశ్చిమగోదావరి జిల్లాల్లో ఇద్దరు మృతిచెందారు. పశ్చిమగోదావరి జిల్లా పాలకొల్లుకు చెందిన కోట అన్నామణి(55)కి నాలుగు రోజుల క్రితం జ్వరం రాగా ఒక ప్రైవేటు ఆసుపత్రిలో చేర్చారు. అక్కడ చికిత్స పొందుతున్న ఆమె ఆదివారం ఉదయం మృతి చెందినట్టు బంధువులు తెలిపారు. తూర్పుగోదావరి జిల్లా శంఖవరం గ్రామం అంబేడ్కర్‌ కాలనీకి చెందిన రాయి తాతాజీ(29) వారం రోజులుగా జ్వరంతో బాధపడుతున్నాడు. ఆయనను మూడు రోజుల క్రితం కుటుంబ సభ్యులు కాకినాడ ప్రభుత్వ ఆసుపత్రికి తరలించారు. చికిత్స పొందుతూ ఆదివారం మృతి చెందాడు.
