
# История запретной любви. Опубликован тизер украинско-польского исторического сериала Кофе с кардамоном

Published at: **2019-11-05T15:52:00+00:00**

Author: ****

Original: [НВ](https://style.nv.ua/kultura/ukrainskiy-serial-kofe-s-kardamonom-smotret-tizer-video-50051764.html)

В Сети появился тизер украинско-польского исторического сериала Кофе с кардамоном. Главные роли в нем исполнят украинская актриса Алена Лавренюк, которая воплотит образ юной украинской красавицы Анны и польский актер Павел Делонг, который исполнит роль женатого польского шляхтича Адама. Действия фильма разворачиваются в ХIX веке.
Алена Лавренюк знакома украинским зрителям по театральным работам, а также по фильмам 11 детей из Моршина и Dzidzio Контрабас. Павла Делонга украинский зритель знает по драме Стивена Спилберг Список Шиндлера.
Тизер будущего сериала был снят во Львове. Съемки сериала начнутся весной 2020 года. Режиссер фильма Станислав Капралов.
