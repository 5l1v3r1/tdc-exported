
# Gedung Kosong di Ancol Kebakaran, 14 Damkar Dikerahkan

Published at: **2019-11-04T23:52:28+00:00**

Author: **Faiq Hidayat**

Original: [detiknews](https://news.detik.com/berita/d-4772140/gedung-kosong-di-ancol-kebakaran-14-damkar-dikerahkan)

Jakarta - Sebuah gedung milik Hailai mengalami kebakaran. Sudin Pemadam Kebakaran Jakut mengerahkan 14 unit mobil ke lokasi tersebut.
"Betul, untuk saat ini 14 unit mobil pemadam sudah di lokasi," ucap petugas Dinas Pemadam Kebakaran DKI, Jono, saat dimintai konfirmasi, Senin (4/11/2019).
Lokasi kebakaran tepatnya berada di Jalan Pelabuhan Ratu No 8 RW 11, Ancol, Pademangan, Jakarta Utara. Jono mengatakan gedung yang terbakar sudah tidak terpakai.
"Jadi gedung sudah tidak terpakai, tapi gedung miliki Hailai," imbuh dia.
Kebakaran tersebut terjadi sekitar pukul 23.00 WIB. Saat ini petugas sedang memadamkan api di lokasi. Penyebab kebakaran belum diketahui karena masih dalam proses pemadaman.
