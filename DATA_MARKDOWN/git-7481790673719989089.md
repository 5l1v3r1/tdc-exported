
# 奈良時代を再現、貴族・女官の衣装で「天平行列」

Published at: **2019-11-04T01:26:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191104-OYT1T50078/)

　奈良時代を再現した色鮮やかな衣装を身にまとい、東大寺に参詣する「天平行列」が３日、奈良市内で行われた。聖武天皇に扮（ふん）した俳優加藤雅也さんや、貴族や女官の装束姿の市民ら１２０人が、ＪＲ奈良駅から東大寺まで約３キロを練り歩いた。
　聖武天皇や光明皇后ゆかりの宝物が展示される正倉院展の時期に合わせ、２０１３年から毎年開催。参加者は、開催中の「御即位記念　第７１回正倉院展」に出展されている仏具「紫檀金鈿柄香炉（したんきんでんのえごうろ）」など宝物のレプリカを手に、ゆっくりと歩みを進めた。
