
# Сульшернинг таъкидлашича, МЮ қишда йирик харидларни амалга оширмайди

Published at: **2019-11-06T17:57:45+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/sulsherning-takidlashicha-myu-qishda-yirik-xaridlarni-amalga-oshirmaydi)

"Режаларимизга тўхталадиган бўлсак, бизнинг устувор мақсадимиз - мавжуд футболчиларни тайёрлаш ва Европа Лигасида кейинги босқич йўлланмасини қўлга киритиш. Трансферлар? Биз доим узоқ истиқболни кўзлаб ҳаракат қиламиз. Январь ойида трансферлар бўлмаслиги мумкин, эҳтимол бир ёки иккита бўлар. Йирик битимлар асосан ёзда тузилади, январда бу сал қийинроқ масала", - дейди мутахассис Goal.com сайтига берган интервьюсида.  Манба : Goal
