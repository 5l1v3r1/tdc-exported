
# 知英　想定体重１００キロのどすこい姿を演じた映画公開に「本当にうれしい」

Published at: **2019-11-02T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/02/0012843125.shtml)

　女優の知英（２５）が２日、都内で、映画「どすこい！すけひら」の公開記念舞台あいさつを行った。
　目が覚めるとナイスボディーに大変身した主人公が繰り広げるラブコメディーで、知英は劇中で想定体重１００ｋｇの“どすこい姿”を披露するため、ファットスーツを着用して撮影に臨んだ。
　共演の草川拓弥（２４）は、現場の様子について「近すぎず遠すぎず、良い距離感でみんな芝居したりしてましたよ」と報告。知英は「ここからはもうお客さまのものですので、この日が来てくれて本当にうれしいです」と封切りを喜んだ。
