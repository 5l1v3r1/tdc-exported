
# Минстрой РФ назвал российские города с благоприятной городской средой

Published at: **2019-11-04T01:59:00+00:00**

Author: ****

Original: [RuNews24](https://runews24.ru/society/04/11/2019/0b74d63a960e1a2f6fba08ec61edf0c1)

Согласно исследованию «Индекса качества городской среды», благоприятной может быть названа среда каждого четвертого города России. Таким образом, в прошлом году в индекс попали 262 крупных муниципалитета.
Глава российского Минстроя Владимир Якушев отметил, что среднее значение индекса по стране составляет 163 балла.
Городом с самой благоприятной средой стала Москва. Столица набрала 276 баллов из 360 возможных. За ней с небольшим отрывом следует Санкт-Петербург, набравший 238 баллов. Тройку лидеров замыкает Екатеринбург с 191 баллом.
За ними расположились Казань и Нижний Новгород, у которых показатель в 190 баллов. В пятерку также попал Красноярск, набравший 189 баллов.
По словам вице-премьера РФ Виталия Мутко, индекс качества городской среды сформирован не для соревнования городов, а как ориентир для руководства, которые должны видеть, над чем работать.
Рост качества городской среды является одой из задач национального проекта «Жилье и городская среда». Для оценки данного параметра утвержден индекс из 36 параметров: благоустройство, безопасность, транспорт, качество дорог, развитие социальной сферы и другое.
