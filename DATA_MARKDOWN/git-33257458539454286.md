
# Восстание машин близко — Microsoft показала консоль Xbox One X с расплавленным черепом терминатора. При включении у него загорается глаз

Published at: **2019-11-01T15:50:27+00:00**

Author: **Meduza**

Original: [Meduza](https://meduza.io/shapito/2019/11/01/vosstanie-mashin-blizko-microsoft-pokazala-konsol-xbox-one-x-s-rasplavlennym-cherepom-terminatora-pri-vklyuchenii-u-nego-zagoraetsya-glaz)

Компания Microsoft представила 1 ноября версию игровой консоли Xbox One X — на ее корпусе изображен череп смертоносного киборга Rev-9 из фильма «Терминатор: Темные судьбы». Когда пользователь включает устройство, на приставке загорается красный глаз нового терминатора.
