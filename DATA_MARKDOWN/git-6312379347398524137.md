
# Украина заявила, что справится без транзита российского газа

Published at: **2019-11-05T03:28:27+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-05/ukraina-zayavila-chto-spravitsya-bez-tranzita-rossiyskogo-gaza)

На Украине сделали громкое заявление о том, что страна сможет справиться без транзита газа с Россией. С таким заявлением выступил министр энергетики и окружающей среды страны Алексей Оржель.
Он отметил, что Украина готова прекратить транзит российского газа без убытков для страны.
"У нас в украинских ПХГ рекордное количество газа, которое закачано, чтобы пройти отопительный сезон", — отметил Оржель в эфире телеканала ICTV.
По его словам, на Украине достаточно газа для того, чтобы справиться с отопительным сезоном без транзита.
