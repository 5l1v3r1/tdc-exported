
# Кўнгли бўшлар кирмасин! Андре Гомешнинг кечаги жароҳати

Published at: **2019-11-04T03:57:04+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/kongli-boshlar-kirmasin-andre-gomeshning-kechagi-jarohati)

Учрашувнинг 79-дақиқасида мезбонлар вакили Андре Гомеш Сон Хон Мин ва Серж Орье билан тўқнашувдан сўнг оёғини синдириб олди. Қуйида Гомешнинг жароҳати қанчалик жиддий эканини кўриш мумкин.
