
# Внезапно вышла iOS 13.2.2. Что нового

Published at: **2019-11-07T21:14:47+00:00**

Author: **Артём Баусов**

Original: [iPhones.ru](https://www.iphones.ru/iNotes/vnezapno-vyshla-ios-1322-chto-novogo-11-07-2019)

Apple выпустила iOS 13.2.2 для всех желающих.
Номер сборки — 17B102.
В этом обновлении компания исправила сразу несколько ошибок:
• приложения больше не выгружаются из оперативной памяти• интернет мог не работать после совершения вызова; теперь все в порядке• интернет мог не работать при полном сигнале, теперь все в порядке
Размер прошивки — около 100 МБ. Скачать iOS 13.2.2 можно уже сейчас в разделе Основные -> Обновление ПО.

        Легкого апдейта!
