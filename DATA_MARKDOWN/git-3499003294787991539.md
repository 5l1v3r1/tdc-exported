
# Задержанный за мелкое хулиганство избил полицейского в отделении на севере Москвы

Published at: **2019-11-02T06:36:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/proisshestviya/zaderzhannyy_napal_na_politseyskogo_v_otdelenii_na_severe_moskvy/)

Задержанный за мелкое хулиганство мужчина был доставлен в отделении полиции на севере Москвы, передает пресс-служба МВД России по столице.В отделении задержанный вел себя агрессивно, напал на участкового, ударив его руками по голове.Заведено уголовное дело о применении насилия в отношении представителя власти.
