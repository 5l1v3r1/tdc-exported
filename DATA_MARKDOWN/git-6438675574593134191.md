
# В Сети высмеяли призыв Климкина назло всем сделать из Украины державу

Published at: **2019-11-04T06:32:00+00:00**

Author: ****

Original: [РИА Новости](https://ria.ru/20191104/1560554051.html)

МОСКВА, 4 ноя — РИА Новости. Пользователи социальных сетей скептически отнеслись к заявлению экс-министра иностранных дел Украины Павла Климкина, предложившего сделать из Украины державу назло всем, кто в нее не верит."А как называется то образование, которое Климкин и соратники уже построили на Украине? Шалман "Диканька", — написал @zigzagsudbi. "Назло — нельзя" — отметила Оксана Товарянська."Хоть "на слабо" что-то сделают, причем, сами", — сыронизировала De_Gabriak. "Назло можно только уши отморозить", — добавил @arazymovsky.
