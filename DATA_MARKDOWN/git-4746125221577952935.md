
# EXILE・松本利夫、3歳長男の近影披露「そっくり」「こんなに大きくなったんですね!」

Published at: **2019-11-01T11:20:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147798/full/)

　EXILEの松本利夫が10月31日、自身のインスタグラムを更新。長男の近影を撮影した写真を披露した。 　松本は「自分が若い頃には日本でのハロウィンはそんなに大きなイベント事ではなかった気がする」と振り返りつつ、「なのでハロウィンを楽しむ習慣がなかったが今は新たにハロウィンを楽しむ事を見つけた^_^」（本文ママ）と、ハロウィン時期に東京ディズニーランドで撮影した、3歳になる長男の写真をアップした。 　この投稿に「大切な家族が増えると色んなイベントが特別なものになりますよね」「お子様こんなに大きくなったんですね！」「後ろ姿がまっちゃんにそっくり」「うわー感慨深いですね」「素敵なパパですね」などの声が寄せられている。 　松本は2014年1月1日に、一般女性と結婚。2015年12月31日をもって、EXILEのパフォーマーを卒業し、以降俳優として活動。2016年4月、第1子となる男児の誕生を発表した。
