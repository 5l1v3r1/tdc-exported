
# Канте Аяксга қарши баҳсда қатнашиши мумкин

Published at: **2019-11-04T15:44:00+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/kante-ayaksga-qarshi-bahsda-qatnashishi-mumkin)

"Аристократлар" ярим ҳимоячиси Нголо Канте октябрь ойида олган жароҳатидан кейин илк марта умумий гуруҳда машғулот ўтказди. Жамоа бош мураббийи Фрэнк Лэмпард ҳам унинг ҳолати яхши эканлигини ва "Аякс"га қарши ўйин қайдномасига киритилишини тасдиқлаган.
28 ёшли хавбек Франция термаси сафида Исландияга қарши ўйиндан олдинги машғулотда жароҳатланганди.
"Челси" - "Аякс" учрашуви эртага тунда ўтказилади.
