
# Трампу не підходять письмові відповіді інформатора по справі імпічменту

Published at: **2019-11-04T16:49:00+00:00**

Author: ****

Original: [DT.ua](https://dt.ua/WORLD/trampu-ne-pidhodyat-pismovi-vidpovidi-informatora-po-spravi-impichmentu-328445_.html)

Президент США Дональд Трамп відхилив пропозицію адвоката про письмових свідоцтвах анонімного "високопоставленого чиновника адміністрації Трампа" на питання республіканців Палати представників, написав політик у Twitter.
