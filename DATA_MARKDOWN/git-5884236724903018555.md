
# Водителям предложили оценить удобство и качество ЗСД

Published at: **2019-11-06T16:37:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/740454-voditelyam-predlozhili-ocenit-udobstvo-i-kachestvo-zsd)


        Петербургским водителям предложили пройти опрос для оценки качества и удобства Западного скоростного диаметра. Отзывы и предложения направят на развитие магистрали.
      
«Магистраль северной столицы» запустила большой опрос для жителей Петербурга, пользующихся Западным скоростным диаметром. Каждый желающий может поделиться мнением об удобстве трассы и оценить условия пользования дорогой по этой ссылке.
