
# В Петербурге задержали забравшихся на ЖК «Международный» руферов

Published at: **2019-11-04T14:19:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/739692-v-peterburge-zaderzhali-zabravshikhsya-na-zhk-mezhdunarodnyi-ruferov)


        Росгвардейцы задержали часть руферов, которые забрались на крышу одного из городских зданий. Экстремалов привлек 25-ти этажный ЖК «Международный». Видео «восхождения» выложили в интернете.
