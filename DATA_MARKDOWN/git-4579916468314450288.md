
# Петербуржцы вернулись из Тюмени. На этот раз обошлось без форс-мажора

Published at: **2019-11-01T20:59:00+00:00**

Author: ****

Original: [Фонтанка.ру](https://www.fontanka.ru/2019/11/01/161/)

Резервный борт «Ямала» прибыл из Тюмени в Петербург. Об этом свидетельствуют данные онлайн-табло аэропорта «Пулково».
Рейс YC811 приземлился в воздушной гавани Петербурга в 23.17 1 ноября. Посадка прошла штатно.
Напомним, у самолёта авиакомпании «Ямал» при вылете из Тюмени отказал левый двигатель из-за попадания в него птицы.  Воздушному судну пришлось кружить над городом, чтобы выработать топливо. За пассажирами, застрявшими в городе, отправили резервный борт, однако, семеро человек отказались лететь.
