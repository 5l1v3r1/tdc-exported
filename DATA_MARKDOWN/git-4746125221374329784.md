
# 熊切あさ美、20年前“チェキッ娘”時代の写真公開「初々しい」「懐かしい」

Published at: **2019-11-05T12:48:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148046/full/)

　タレントの熊切あさ美が3日、自身のインスタグラムを更新。20年前の写真を公開した。 　熊切は「チェキッ娘旅立ちライブから今日で20年」と、アイドルグループ・チェキッ娘の解散ライブを懐古しながら20年前の写真をアップ。「なんだかんだ早かったなー」と振り返り、「デビューして21年目　初心を忘れずにこれからも頑張ります」と決意を新たにした。 　この投稿に「懐かしい〜 初々しいあさみちゃんも素敵です」「あの頃と変わらない若々しさですね」「やっぱり昔からお綺麗でキラキラしてましたね」「この頃からずぅーーーと大好きです　21年目もご活躍楽しみにしてます」などの声が寄せられている。
