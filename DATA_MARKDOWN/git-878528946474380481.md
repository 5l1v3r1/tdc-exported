
# 侍・高橋　ピンチ招くも後続シャットアウト

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/01/0012838446.shtml)

　「侍ジャパン強化試合、日本５－６カナダ」（３１日、沖縄セルラースタジアム那覇）
　野球日本代表「侍ジャパン」が惜敗した。カナダ代表には３５年ぶりの黒星となった。
　高橋が中継ぎとしての仕事を果たした。四回からマウンドに上がり、五回に２死一、二塁のピンチを招くも後続をシャットアウト。６日・プエルトリコ戦の先発が有力な岸が、発熱のため２日連続でグラウンドに姿を現さず。代役の候補に挙がるサブマリンは「全力で岸さんの穴を埋めないといけない」と気合を込めた。
