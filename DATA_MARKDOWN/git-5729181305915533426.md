
# Siguen las consecuencias del clásico platense

Published at: **2019-11-05T00:49:00+00:00**

Author: ****

Original: [El Esquiú](https://www.elesquiu.com/deportes/2019/11/5/siguen-las-consecuencias-del-clasico-platense-340024.html)

Continúan las consecuencias después del clásico que protagonizaron Gimnasia y Estudiantes en el Bosque, que fue para el Pincha por 1-0 con el tanto de Mateo Retegui.
Este lunes se confirmó que el árbitro del encuentro informó a Horacio Tijanovich y no a Mariano Andújar. El delantero del Lobo, según el informe del Pitana, le propinó una patada al arquero del equipo ganador mientras realizaba declaraciones a los medios una vez finalizado el clásico.
Tijanovich deberá asistir el próximo jueves a AFA para declarar sobre esta situación para definir si le cabe una sanción.
