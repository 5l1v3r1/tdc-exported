
# Куман: Барселона учун Голландия билан шартномани бекор қилишим мумкин

Published at: **2019-11-03T12:12:35+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/kuman-barselona-uchun-gollandiya-bilan-shartnomani-bekor-qilishim-mumkin)

"Бу маълумот тасодифан Нико Ян Хугма (Голландия футбол федерацияси вакили) оғзидан чиқиб кетди. У ҳамма билади деб ўйлаган бўлса керак.
Шартда фақат "Барселона" номи ёзилган ва агар таклиф бўлса, фақат Евро-2020дан сўнг уни кўриб чиқишим мумкин. Ҳозир ҳеч қаерга кетмайман.
Европа чемпионатидан сўнг Федерация турнирда ёмон иштирок этганимизни айтса, йўлларимиз айрилиши мумкин.
Аммо мен фақат битта клубга кетаман. Голландия терма жамоасини бошқа жамоа учун тарк этмайман. "Барселона" - орзуимдаги жамоа" - дейди Куман.
