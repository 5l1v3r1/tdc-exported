
# వేలేరులో ఎలా పని చేస్తావో చూస్తా..

Published at: **2019-11-06T02:27:10+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=947413)

ADVT
హన్మకొండ రూరల్‌, నవంబరు 5: తహసీల్దార్‌ విజయారెడ్డి సజీవ దహనం జరిగిన రోజే వరంగల్‌ అర్బన్‌ జిల్లాలో ఓ మహిళా తహసీల్దార్‌ను సర్పంచ్‌ బెదిరించిన సంఘటన చోటు చేసుకుంది. పోలీసుల కథనం ప్రకారం.. సాదా బైనామా ద్వారా కొన్న భూమిని పట్టా చేయాలంటూ వేలేరు తహసీల్దార్‌ రజనిని మల్లికుదుర్ల గ్రామ సర్పంచ్‌ గోదుల రాజిరెడ్డి కొన్ని రోజులుగా కోరుతున్నాడు. సోమవారం హన్మకొండ కేఎల్‌ఎన్‌ రెడ్డి కాలనీలోని రజని ఇంటికి వెళ్లారు. తహసీల్దార్‌ భర్త వారిస్తున్నా.. లోపలకు దూసుకెళ్లేందుకు రాజిరెడ్డి ప్రయత్నించాడు. చివరకు ‘నువ్వు (రజని) వేలేరులో ఎలా పనిచేస్తావో చూస్తా’ అని బెదిరించి వెళ్లిపోయాడు. తహసీల్దార్‌ రజని.. సుబేదారి స్టేషన్‌లో ఫిర్యాదు చేయగా రాజిరెడ్డిపై పోలీసులు కేసు నమోదు చేసి మంగళవారం అరెస్టు చేశారు.
