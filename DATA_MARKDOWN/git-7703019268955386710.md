
# ¿Lluvia, calor o ambos?: Así estará el tiempo este martes

Published at: **2019-11-05T00:00:00+00:00**

Author: **Santiago**

Original: [El Liberal](https://www.elliberal.com.ar/noticia/509823/lluvia-calor-o-ambos-asi-estara-tiempo-este-martes)

Santiago del Estero vivirá hoy una jornada calurosa y durante la cual se mantendrán vigentes posibles chaparrones y lluvias. La máxima escalará a 30º.
El pronóstico elaborado por el Servicio Meteorológico Nacional indica para la mañana posibilidad de que se registren chaparrones con vientos del este a 12 kilómetros por hora.
Para la tarde, se anuncia un 70% de probabilidad de lluvias aisladas, con vientos del este y mejorando hacia la noche. Mientras que, para mañana, se anticipa un día nublado, con una mínima de 19º y una máxima de 32º.
