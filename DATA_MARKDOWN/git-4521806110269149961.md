
# США решили ужесточить санкции против Ирана

Published at: **2019-11-01T10:17:00+00:00**

Author: ****

Original: [RuNews24](https://runews24.ru/world/01/11/2019/62f5cf1bd83b11cac8ff5c1217c99559)

По мнению Госдепа, строительный сектор Ирана прямо или косвенно контролируется Корпусом стражей исламской революции (КСИР), признанной в США международной террористической организацией.
В связи с этим будет введен запрет на поставку Ирану необработанных и обработанных металлов, графита, угля и программного обеспечения для строительства сектора.
Также под запрет попадают поставки четырех видов материалов, которые, по мнению США, Иран может использовать в ядерной или ракетной программах.
США продолжат усиливать экономическое давление на Иран до тех пор, пока не будет прекращено расширение ядерной программы.
