
# Ухта выделила резервные средства на ремонт моста через автодорогу Керки-Том

Published at: **2019-11-05T13:15:00+00:00**

Author: **Мария Майер**

Original: [Комиинформ](https://komiinform.ru/news/188403)

На ухтинском участке автодороги Керки-Том восстановят мост. Соответствующее постановление подписал руководитель администрации Ухты.
Согласно постановлению, финансовому управлению администрации поручено выделить Управлению ЖКХ из резервного фонда средства в размере 190 тыс. 585 рублей.
Средства требуются на проведение неотложных аварийно-восстановительных работ моста через реку Вылыс-Койю на участке  км 6+182,35 автомобильной дороги Керки-Том. Этот мост был разрушен в результате паводка, вызванного прохождением обильных осадков в виде проливных дождей.
В течение месяца после выполнения работ УЖКХ обязано представить финансовому управлению отчет об использовании выделенных средств.
Напомним, Ухта, Сосногорск и Ижма объединили усилия для восстановления дороги Том-Керки. В связи с аномальным количеством осадков летом этого года был разрушен мост через реку Вылыс Койю на ухтинском участке, бетонный мост на сосногорском участке, также на данном участке паводковыми водами размыто дорожное полотно. Тогда три населенных пункта СП "Том" остались без автомобильного сообщения.
