
# Самострои Юга и Кавказа проверят эксперты

Published at: **2019-11-05T11:27:00+00:00**

Author: **Мария Гридина**

Original: [ФедералПресс](https://fedpress.ru/news/23/economy/2358110)


        КРАСНОДАР, 5 ноября, ФедералПресс. Для реализации проекта по борьбе с самостроями несколько ведомств объединятся и будут работать на базе Минстроя России.
      
Регионы Южного федерального округа и Кавказа изучат эксперты – для составления единого реестра самостроев, который будет называться «Роскапстрой». В него войдут данные всей России, но начнут с Крыма, Краснодарского края, Ростовской области, Дагестана.
По данным всероссийского центра национальной строительной политики, ситуация с самостроями в ЮФО и СКФО стоит особенно остро. Для реализации проекта по борьбе с самостроями несколько ведомств объединятся для работы на базе Минстроя России.
Планируются выезды экспертов в каждый регион отдельно. Каждый объект будет рассматриваться отдельно. Для этого будут выделяться в том числе федеральные средства. О каких суммах идет речь, пока не сообщалось.
ФАУ «Роскапстрой» станет главным экспертом при рассмотрении не только объектов, но и правовой и технической экспертизы по всей стране.
