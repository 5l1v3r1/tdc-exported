
# Российский рэпер Оксимирон записал трек о протестах в Москве

Published at: **2019-11-07T12:18:00+00:00**

Author: ****

Original: [charter97.org](https://charter97.org/ru/news/2019/11/7/354745/)

Российский рэпер Оксимирон записал трек для участия во втором раунде онлайн-баттла Hip-Hop.ru с фигурантом «московского дела» Самариддином Раджабовым. Того обвиняют в угрозе применения насилия к трем полицейским, поскольку он бросил пластиковую бутылку на несогласованной акции в Москве, пишет TJ.
