
# Ολυμπιακός : Σκάουτερ για τον Σα στο Μόναχο

Published at: **2019-11-06T08:00:11+00:00**

Author: **ΤοΒΗΜΑ Team**

Original: [Το Βήμα Online](https://www.tovima.gr/2019/11/06/sports/olympiakos-skaouter-gia-ton-sa-sto-monaxo/)

Σε ένα από τα δυσκολότερα φετινά παιχνίδια των ερυθρόλευκων λόγω και της δυναμικής του αντιπάλου, αναμένεται να υπάρχουν σκάουτερ για τον Ζοσέ Σα.
Ο πορτογάλος γκολκίπερ είναι ένας από τους κορυφαίους παίκτες του Ολυμπιακού τη φετινή σεζόν και όπως είναι λογικό έχει κλέψει τις εντυπώσεις με τις εμφανίσεις του.

        Κύπρος : Περιέλουσε με οξύ την πρώην γυναίκα του – Παλεύει για την ζωή της
      
Όπως αναφέρει η Sportday, το παιχνίδι με την Μπάγερν στο Μόναχο θα είναι μια καλή ευκαιρία για τις ομάδες που ενδιαφέρονται για τον Σα να τσεκάρουν τις ικανότητες του και να τον παρακολουθήσουν από κοντά κόντρα σε μια μεγάλη ομάδα και σε ένα ματς που θα νιώσει πολλή πίεση.
Αξίζει να σημειωθεί πως δεν γίνονται γνωστές οι ομάδες που τσεκάρουν τον πορτογάλο διεθνή κίπερ, όμως είχαν βρεθεί σκάουτερ τόσο στο παιχνίδι με την Μπάγερν στο Καραϊσκάκη αλλά και στο ματς με την ΑΕΚ.

        Κτηματολόγιο : Καλοκαίρι τα πρόστιμα για τις ληξιπρόθεσμες δηλώσεις
