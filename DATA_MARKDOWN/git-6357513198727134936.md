
# Мировой босс, подземелья со своими задачами и несбалансированный друид — что показали в демо Diablo IV

Published at: **2019-11-04T19:11:45+00:00**

Author: **Руслан Присекин**

Original: [Stopgame](https://stopgame.ru/newsdata/40659)

Несмотря на то что Diablo IV находится на раннем этапе разработки, демоверсию игры уже давали пощупать на BlizzCon 2019. Этой возможностью воспользовался автор Game Informer, успевший поделиться впечатлениями от увиденного.В демо было доступно несколько квестов из основной цепочки, два подземелья, периодическое событие и так называемый мировой босс — особо сильный противник, которого, судя по всему, можно убить только в команде с другими игроками.По словам журналиста, одно из самых интересных событий демонстрационной версии — сражение с мировым боссом по имени Ашава. Чудовище обладает множеством способностей, некоторые из них могут убить вашего персонажа с одного удара. Конечно, уничтожение такого босса щедро вознаграждается горами лута, включая легендарный вещи.
