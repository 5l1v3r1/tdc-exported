
# ロッテ、レアードと２年契約

Published at: **2019-11-02T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/02/0012842345.shtml)

　ロッテは２日、レアード内野手（３２）と来季の契約に合意したと発表した。２年契約で、年俸は２２０万ドル（約２億３７６０万円）。日本ハムから今季加入し、来季が来日６年目となる。球団を通じて「とても興奮している。リーグ優勝し、クライマックスシリーズ（ＣＳ）を勝ち抜き、日本一になりたい」とコメントした。
　今季は１３９試合に出場し打率２割４分８厘、３２本塁打、８９打点だった。通算１６３本塁打で「２００本塁打を目指していきたい」と意気込んだ。（金額は推定）
