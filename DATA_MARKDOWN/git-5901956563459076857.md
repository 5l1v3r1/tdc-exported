
# Najstnik (19) v Metliki povozil peški, odpeljali so ju v bolnišnico

Published at: **2019-11-07T11:12:00+00:00**

Author: **N. Č.**

Original: [Slovenske novice](https://www.slovenskenovice.si/kronika/doma/clanek/19-letnik-v-metliki-povozil-peski-odpeljali-so-ju-v-bolnisnico-246740)

METLIKA – Policisti so bili nekaj po 17. uri obveščeni o prometni nesreči v Metliki z udeležbo dveh pešk in voznika osebnega avtomobila.Lažje poškodovano 43- in 20-letno peško so reševalci odpeljali v bolnišnico.Ob ogledu so ugotovili, da se je nesreča zgodila zaradi nepravilnega ravnanja pešk, ki sta prečkali cesto zunaj prehoda za pešce, vanju pa je trčil 19-letni voznik osebnega avtomobila.
