
# СКА 3 ноября сыграет в Ледовом с рижским «Динамо»

Published at: **2019-11-03T12:04:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/739326-ska-3-noyabrya-sygraet-v-ledovom-s-rizhskim-dinamo)


        Петербургский хоккейный клуб СКА на втором матче выходных встретится с «Динамо» из Латвии. Игра состоится в Ледовом дворце в 17:00.
      
В воскресенье, 3 ноября, в 17:00 в Ледовом дворце состоится хоккейный матч между петербургской командой СКА и сборной из Латвии «Динамо». Соответствующая информация появилась на официальной странице хоккейного клуба СКА в соцсети Twitter.
