
# Новые «Сокольники»: Названия строящихся станций БКЛ будут дублировать уже существующие

Published at: **2019-11-05T17:14:03+00:00**

Author: **Юлия Онодэра**

Original: [The Village](https://www.the-village.ru/village/city/news-city/366357-znakomye-nazvaniya)

Мэр Сергей Собянин утвердил названия пяти станций Большой кольцевой линии метрополитена. Все они на стадии строительства носили другие имена. Названия строящихся станций БКЛ продублируют уже существующие. По мнению властей, так будет удобнее для пассажиров.
Станция с проектным названием «Можайская», которая находится на пересечении с Арбатско-Покровской и Филевской линиями метро, станет «Кунцевской» — по названию уже существующих станций, на которые будет доступна пересадка.
Станцию «Шереметьевская» назовут «Марьиной Рощей», с нее можно будет пересесть с БКЛ на Люблинско-Дмитровскую линию. Станция «Ржевская» после открытия окажется «Рижской», «Стромынка» — «Сокольниками», а «Рубцовская» — «Электрозаводской».
