
# ソフトバンク・工藤監督　若手へ成長を求む「ここが来年に向けての出発」

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/01/0012838441.shtml)

　ソフトバンク・工藤監督が、１日からの宮崎秋季キャンプを前に若手へ成長を求めた。３１日、１軍選手らとともに宮崎入り。到着後に行われたミーティングで「ここが来年に向けての出発。個々が能力を上げられるように頑張ってくれ」と妥協しない姿勢を求めた。
　来季の３年ぶりのリーグＶ奪回と４年連続日本一へ向け、若手の底上げは必要不可欠だ。
