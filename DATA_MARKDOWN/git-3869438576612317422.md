
# Грузовик повредил воздушную теплосеть в Мурманске

Published at: **2019-11-06T00:00:00+00:00**

Author: ****

Original: [Мурманский Вестник](https://www.mvestnik.ru/newslent/gruzovik-povredil-vozdushnuyu-teploset-v-murmanske/)

Единая дежурно-диспетчерская служба сообщает, что в 07:53 на пункт управления ММБУ «ЕДДС» по системе 112 поступила информация о ДТП с участием грузового автомобиля Scania по адресу: ул. Промышленная, 15.
Открытый кузов автомобиля задел и повредил воздушную теплосеть. В результате без отопления оказался абонент ООО «Ривьера», Промышленная,16. Абонент переключен на электрообогрев.
Сроки устранения аварии уточняются.
