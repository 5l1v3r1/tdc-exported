
# Вышел первый трейлер нового «Человека-невидимки» с Элизабет Мосс

Published at: **2019-11-07T18:45:47+00:00**

Author: **Дарья Громова**

Original: [Лайфхакер](https://lifehacker.ru/unvisible-man-trailer-1/)

Перезапуск знаменитого хоррора выйдет на большие экраны уже весной.
Universal Pictures опубликовали первый трейлер фильма ужасов «Человек-невидимка». Это перезапуск одноимённого фильма 1933 года по роману Герберта Уэллса.
