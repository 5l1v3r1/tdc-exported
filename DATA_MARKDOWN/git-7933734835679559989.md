
# Полиция разыскивает несовершеннолетнего крымчанина

Published at: **2019-11-07T14:26:00+00:00**

Author: ****

Original: [Крымское информационное агентство](https://kianews24.ru/news/policiya-raziskivaet-nesovershennolet/)

Крымские полицейские разыскивают пропавшего без вести несовершеннолетнего Илью Северьяна 2002 года рождения. Об этом сообщили в управлении уголовного розыска МВД по РК.
Сотрудники отдела уголовного розыска УМВД России по Ялте устанавливают местонахождение без вести пропавшего несовершеннолетнего Ильи Вадимовича Северьяна (06.10.2019), который ушел из дома и до настоящего времени его местонахождение неизвестно.
Приметы: на вид 16-17 лет, рост около 165 см, волосы светлые, глаза темного цвета, брови густые дугообразные, телосложение худощавое. При себе имел документы, удостоверяющие личность.
«Просьба ко всем, кто владеет информацией о местонахождении разыскиваемого несовершеннолетнего, незамедлительно сообщить в УМВД России по Ялте по телефонам: 8 (3654) 23-50-46, 23-09-20, или в отдел координации деятельности органов внутренних дел по розыску лиц УУР МВД по РК по телефону: (3652) 667-588, 668-818», — говорится в сообщении.
