
# 大阪梅田駅、深夜の「阪急三番街」で出火…男性搬送

Published at: **2019-11-07T00:09:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191107-OYT1T50116/)

　６日午後１１時５０分頃、大阪市北区芝田の「阪急三番街」南館１階の靴店から出火。商品などを焼き、警備員が消火器でまもなく消した。市消防局によると、この火事で一帯に煙が充満し、通行人の４０歳代の男性が煙を吸って気分不良を訴え、病院に搬送された。店は当時無人だったという。
　阪急三番街は、阪急大阪梅田駅ターミナルの階下にある商業施設。
