
# 藤田ニコル“働く女性50人”にエール 夢のかなえ方を初告白「リストをメモして…」

Published at: **2019-11-01T18:49:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147314/full/)

　タレントの藤田ニコルが、3日放送のTBSラジオ『LUX presents 〜女性の輝きに、限界なんてない〜藤田ニコルと働く女性50人』（前9：00）の公開収録を行った。

        【写真】その他の写真を見る
      
　番組では、50人の働く女性をゲストに迎えて、藤田、フリーアナウンサーの吉田明世、人材コンサルタントの森本千賀子氏が直接対話し、女性としての生きがい、仕事の悩み、職場で感じた性別に関する偏見や違和感などについてトークを展開。 　参加者に対して行われた事前アンケートでは「自分の仕事と部下の指導、偏りなくやりたいがバランスが難しい」「仕事で上を目指したいが、20代で仕事をやめて結婚するのが幸せだという空気が職場にある」といった声が寄せられた。 　参加者からのリアルな声に、3人は自身の経験談やアドバイスをトーク。「夢のかなえ方」というテーマでは、藤田が「これ初めて言ったけど…」と前置きした上で「やりたいことや夢リストを携帯にメモして、かなったら消しています」と打ち明けていた。
