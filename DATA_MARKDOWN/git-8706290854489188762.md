
# Αγγελοπούλου: Είναι μια ευκαιρία να επανασυστήσουμε την Ελλάδα στον κόσμο - ΒΙΝΤΕΟ

Published at: **2019-11-07T12:47:00+00:00**

Author: ****

Original: [enikos.gr](http://www.enikos.gr/politics/682838/angelopoulou-einai-mia-efkairia-na-epanasystisoume-tin-ellada-sto)

"Τα 200 χρόνια από την ελληνική επανάσταση του 1821 και τη σύσταση του ελληνικού κράτους αποτελούν ορόσημο για κάθε Έλληνα και Ελληνίδα", επισήμανε μιλώντας στη Βουλή, κατά την έναρξη των εργασιών της Επιτροπής «Ελλάδα 2021», η πρόεδρος της Επιτροπής Γιάννα Αγγελοπούλου.
"Είναι μια ευκαιρία αναμέτρησης με το μέλλον και μια ευκαιρία να επανασυστήσουμε την Ελλάδα στον κόσμο", τόνισε.
