
# Nezemnaya world tour 2020: Макс Барських їде у світове турне

Published at: **2019-11-04T13:37:53+00:00**

Author: ****

Original: [ФАКТИ](https://fakty.com.ua/ua/showbiz/20191104-nezemnaya-world-tour-2020-maks-barskyh-yide-u-svitove-turne/)

Популярний український співак Макс Барських їде у світове турне Nezemnaya world tour 2020.
Макс Барських відправляється в тур Nezemnaya world tour 2020, який охопить Австралію, США, Канаду, Велику Британію, Німеччину, Швецію, Фінляндію, Данію, країни Балтії та Казахстан.
