
# В Вологде адвокат обокрал бизнесмена

Published at: **2019-11-06T14:23:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Vologda/740412-v-vologde-advokat-obokral-biznesmena)


        Юриста заподозрили в хищении 200 тыс. рублей. Деньги адвокат взял якобы для взятки, по факту же просто забрал себе. Бизнесмен, который остался без денег, написал заявление в полицию.
      
В Вологде адвокат обманул клиента на 200 тысяч рублей, сообщает пресс-служба СУ СКР области. Юрист вызвался передать взятку сотруднику прокуратуры, чтобы клиент избежал ответственности за проваленную проверку соблюдения требований противопожарной безопасности в торгово-развлекательном центре. По факту же адвокат не имел возможности повлиять на ход проверки, а деньги украл. Когда предприниматель понял, что произошло, сам обратился в полицию. Адвоката задержали, против него возбудили уголовное дело. Идет расследование.
