
# Diyarbakır kırsalında 3 terörist etkisiz hale getirildi

Published at: **2019-11-05T17:47:19+00:00**

Author: ****

Original: [OLAY](https://www.olay.com.tr/diyarbakir-kirsalinda-3-terorist-etkisiz-hale-getirildi-214252h.htm)

İçişleri Bakanlığından, Diyarbakır kırsalında İl Jandarma Komutanlığınca gerçekleştirilen hava destekli operasyonda, 3 teröristin etkisiz hale getirildiği bildirildi.
Bakanlık'tan yapılan açıklamaya göre, iç güvenlik operasyonları kapsamında Diyarbakır kırsalında İl Jandarma Komutanlığınca hava destekli operasyon düzenlendi.
3 teröristin etkisiz hale getirildiği bölgede operasyonlar devam ediyor.
