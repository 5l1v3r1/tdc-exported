
# Филадельфия вызвала Воробьёва из АХЛ, Рубцов отправлен в фарм-клуб

Published at: **2019-11-06T15:32:00+00:00**

Author: ****

Original: [Allhockey.ru](https://allhockey.ru/news/show/369199-Filadel_fiya_vyzvala_Vorob_eva_iz_AHL_Rubcov_otpravlen_v_farm_klub)

"Филадельфия" вызвала из фарм-клуба нападающего Михаила Воробьёва. В обратном направлении проследовал форвард Герман Рубцов.22-летний Воробьёв провел в нынешнем сезоне в НХЛ пять матчей и отметился результативной передачей. 21-летний Рубцов в этом сезоне дебютировал в НХЛ. На его счету показатель полезности "-1" в трёх матчах.
