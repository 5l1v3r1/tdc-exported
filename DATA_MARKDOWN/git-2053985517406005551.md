
# طريقة عمل كباب الحلة الأصلي بتاع زمان مع الشيف يسري

Published at: **2019-11-02T02:37:00+00:00**

Author: **قسم التوك شو**

Original: [صدى البلد](https://www.elbalad.news/4042311)

قسم التوك شو قدّمت الشيف يسري خميس، خلال حلقته الأولى ببرنامج "تتبيلة" عبر فضائية "صدى البلد"، طريقة عمل كباب الحلة الاصلي بتاع زمان بطريقة سهلة.كيلو لحم مكعبات بصلة كبيرة مفرومة 2 ملعقة ك زيت 2 ملعقة زبدة ملح فلفل اسودبهاراتورقة لورا جوزة الطيب قطعة فحم سمن
