
# Генассамблея ООН одобрила резолюцию РФ по контролю над вооружениями

Published at: **2019-11-06T22:01:16+00:00**

Author: ****

Original: [Коммерсантъ](https://www.kommersant.ru/doc/4149954)

Первый комитет Генассамблеи ООН одобрил российский проект резолюции об укреплении системы договоров по контролю над вооружениями, передает «РИА Новости». Документ призывает участников подобных договоров выполнять их в полном объеме.
Резолюция предлагает поддерживать меры, которые способствуют заключению дополнительных соглашений такого рода. Проект называется «Укрепление и развитие системы договоров и соглашений по контролю над вооружениями, разоружению и нераспространению».
Ранее комитет принял три проекта российских резолюций по разоружению в космосе. Первая резолюция — «Неразмещение первыми оружия в космосе».
Вторая резолюция — «Дальнейшие практические меры по предотвращению гонки вооружения в космическом пространстве». Третий документ называется «Меры по обеспечению транспарентности и укреплению доверия в космической деятельности».
