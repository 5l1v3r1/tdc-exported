
# Зірка збірної Бразилії похрестився у власній ванній

Published at: **2019-11-01T16:07:48+00:00**

Author: ****

Original: [ФАКТИ](https://fakty.com.ua/ua/sport/20191101-zirka-zbirnoyi-brazyliyi-pohrestyvsya-u-vlasnij-vannij/)

Хавбек збірної Бразилії Філіппе Коутінью пройшов обряд хрещення.
Хавбек мюнхенської Баварії та збірної Бразилії Філіппе Коутінью вибрав власну ванну місцем для хрещення.

Урочисту церемонії хрещення футболіста провів бразильський пастор Тіаго Брунет, який працює у церкві Case de Destino у Німеччині.
Разом з Коутінью обряд хрещення пройшла його дружина Айне.
Нагадаємо, у серпні Баварія орендувала Коутінью у каталонської Барселони.
