
# Браво - ЕЧЛда захирадан тушиб қизил карточка олган илк дарвозабон бўлди

Published at: **2019-11-07T06:05:00+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/bravo-echlda-zaxiradan-tushib-qizil-kartochka-olgan-ilk-darvozabon-boldi)

Чилилик посбон иккинчи тайм бошида майдонга тушди. Аммо, 81-дақиқада қизил карточка олган ҳолда майдондан четлатилди.
Шундай қилиб, Браво ЕЧЛда захирадан тушиб майдондан четлатилган илк дарвозабон бўлди.
Эслатиб ўтамиз, Браво "Манчестер Сити" 2016 йилда "Барселона"дан ўтган эди.
