
# Резервный рейс с пассажирами Sukhoi Superjet 100 приземлился в Санкт-Петербурге

Published at: **2019-11-01T21:48:44+00:00**

Author: ****

Original: [Коммерсантъ](https://www.kommersant.ru/doc/4147830)

Резервный рейс с пассажирами из Тюмени, которые не смогли долететь до Санкт-Петербурга из-за отказа двигателя лайнера Sukhoi Superjet 100 авиакомпании «Ямал», приземлился в аэропорту «Пулково», сообщило «РИА Новости» со ссылкой на представителя управляющей компании аэропорта ООО «Воздушные ворота северной столицы». Ранее самолет с отказавшим двигателем успешно приземлился в Тюмени.
«Рейс приземлился в 23:17»,— сказал собеседник агентства.
Самолет совершал рейс из Тюмени в Санкт-Петербург. После вылета у него произошел помпаж левого двигателя, который вышел из строя. Экипаж принял решение выработать топливо и вернуться в Тюмень. Пострадавших в результате инцидента нет.
