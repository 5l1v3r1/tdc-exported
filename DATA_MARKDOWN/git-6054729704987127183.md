
# В Омске появятся 30 новых вольеров для бездомных животных

Published at: **2019-11-01T13:05:00+00:00**

Author: ****

Original: [БК55](https://bk55.ru/news/article/159432/)


        По срокам они должны быть к окончанию этого года.
      
Стало известно, что между благотворительным фондом «Общий мир» и БУ «Спецавтохозяйство» заключено соглашение, по которому хотят улучшить условия содержания отловленных бездомных собак в городе. Теперь проводится акция по сбору средств и материалов для строительства послеоперационного блока на территории учреждения. Теплое чистое помещение необходимо для послеоперационного содержания собак после оперативных вмешательств (операции, стерилизация).
Для того, чтобы послеоперационный блок начал функционировать, необходимо еще смонтировать вольеры для раздельного содержания собак, установить кварцевые лампы, оснастить помещение необходимой мебелью. Полный перечень работ и материалов, которые необходимы для подготовки помещения, опубликован на странице фонда помощи животным «Общий мир» в социальной сети «ВКонтакте» (https://vk.com/fond_omir?w=wall-147140033_5619).
Кроме того, на уличной территории БУ «Спецавтохозяйства» продолжается строительство вольеров для содержания собак, отловленных на улицах Омска. До конца года в учреждении появятся 30 новых вольеров.
