
# Солнце обманчиво - в Мелитополе сегодня будет также холодно

Published at: **2019-11-02T09:08:00+00:00**

Author: **Владимир Андреев**

Original: [РИА Мелитополь](https://ria-m.tv/news/170303/solntse_obmanchivo_v_melitopole_segodnya_budet_takje_holodno.html)


        Сегодня антициклон еще продолжит господствовать в регионе. 
      
Ожидается переменная облачность, без осадков, ветер южный 5-10 м / с, температура воздуха ночью 4-5 ° мороза, днем 2-7 ° тепла.
