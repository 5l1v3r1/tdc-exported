
# よゐこ濱口　ＴＫＯ木下との共演ＮＧ説を否定「びっくり」「誤解です」

Published at: **2019-11-05T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/05/0012849579.shtml)

　お笑いコンビ・よゐこの濱口優（４７）が４日、妻でタレントの南明奈（３０）と都内で「ピノガチャ」発表会に登場した。
　一部週刊誌などで報じられていたＴＫＯ・木下隆行（４７）との騒動について「僕も記事見てびっくりしている」と笑った。この件は、濱口と南の結婚祝賀パーティーの幹事を務めた木下が、後輩から集めた金を着服したのではと報道されていたもの。濱口は「誤解です。ネタとして言っていた」と説明。木下との共演ＮＧ説も「びっくりしました。仲良いです」と否定した。
　１８年５月に結婚した２人は、この日もラブラブ。濱口は今年９月３日に行った披露宴を行った際の南のウエディングドレス姿に「めちゃめちゃキレイでした」とのろけた。
