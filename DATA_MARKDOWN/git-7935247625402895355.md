
# Bakanlık açıkladı! 3 terörist etkisiz hale getirildi

Published at: **2019-11-03T13:22:00+00:00**

Author: ****

Original: [OLAY](https://www.olay.com.tr/bakanlik-acikladi-3-terorist-etkisiz-hale-getirildi-213993h.htm)

Kars kırsalında Jandarma Özel Harekat timlerince düzenlenen operasyonda 3 terörist etkisiz hale getirildi. İçişleri Bakanlığından yapılan açıklamaya göre, iç güvenlik operasyonları kapsamında, Kars kırsalında, İl Jandarma Komutanlığı  Jandarma Özel Harekat timlerince operasyon gerçekleştirildi.     
Operasyonda, 3 terörist etkisiz hale getirildi. Bölgede operasyonlar sürüyor.
