
# Волейбольная команда «ДМК» г. Каменское стала победителем Спартакиады «Здоровье»

Published at: **2019-11-01T07:18:00+00:00**

Author: ****

Original: [dndz.tv](http://dndz.tv/article-30015.html)

В Каменском (Днепродзержинске) завершен волейбольный этап двадцатой Спартакиады «Здоровье».30.10.19 убедительной победой металлургов завершились состязания волейбольных команд:- «ДМК»  переиграла волейболистов «ДХЗ» - 25:17, 25:22;- болельщики смогли создать атмосферу поддержки любимой команды;- финал для металлургов оказался победным;- 3 место заняли спортсмены «ДнепрАзота».
