
# El año que viene estas características importantes podrían aparecer en Twitter

Published at: **2019-11-06T00:00:00+00:00**

Author: **Diario Tecnología**

Original: [Diario Tecnología](https://diariotecnologia.es/el-ao-que-viene-estas-caractersticas-importantes-podran-aparecer-en-twitter)

Ayer a través de un tweet el vicepresidente de diseño e investigación de Twitter, Dantley Davis, ofreció una lista de nuevas características y funciones que podrían llegar a desarrollar para el 2020. Claro que esto no es algo definitivo, ni oficial, es un supuesto de Davis sobre características que está pensando.  Aún así el tiene un puesto importante como para pensar que varias de estas características tienen posibilidades.
