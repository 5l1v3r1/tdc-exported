
# 「織田シナモン信長」EDテーマを堀内犬友らメインキャラが歌唱！ キャスト登壇の先行上映イベントも12/28に開催！

Published at: **2019-11-01T18:00:00+00:00**

Author: ****

Original: [アキバ総研](https://akiba-souken.com/article/42219/)

2020年1月より放送予定のテレビアニメ「織田シナモン信長」のエンディングテーマを、メインキャラクターのシナモン・ブーたち（堀内犬友さん/犬川登志夫さんら）が担当することが決定した。さらに、キャストたちが登壇する先行上映イベントが2019年12月28日（土）に開催される。
「織田シナモン信長」（目黒川うなさん・著）は、「月刊コミックゼノン」（徳間書店）にて2016年より大好評連載中の人気作品。群雄割拠の戦国時代に覇王と呼ばれた織田信長が、現代の日本によみがえったその姿は、なんと柴犬。信長と同じ時代にしのぎを削った名将たちも、なぜかみんな犬に転生してしまい、現代の群雄割拠がドッグランで繰り広げられる、という究極のギャップに癒やされる異色コメディ作品。
スタッフは、本作の監督として髙橋英俊さん、シリーズ構成に経塚丸雄さん、そしてキャラクターデザインは、「美少女戦士セーラームーン」 をはじめとする数々の人気作品の作画監督・原画を担当する香川 久さんが務める。
 
そして、本作のエンディングテーマは、メインキャラクターのシナモン・ブーたち（堀内犬友さん/犬川登志夫さんら）が担当することに決定した。そのほかのワンちゃんたちが合いの手で盛り上げ、ポップで思わず口ずさみたくなるようなキャラクターソングに仕上がっているという。
 
作詞作曲は、「マタアイマショウ」「ルパン・ザ・ファイヤー」など超ロングヒット曲をリリースしているSEAMOさんが務める。今回、犬を題材にしたアニメのエンディングテーマを手がけることとなり「尾張生まれ尾張育ちの、犬の歌まで書いている愛犬家でもある僕が作るなんて…これもなにかの縁ですね。武将達、いや犬達が入り乱れるハチャメチャ感を、ラップも織り交ぜ歓喜と悲哀を存分に表現した曲になりました」と楽曲への期待を高めるコメントを寄せた（原文ママ）。
 
 さらに、本作の先行上映イベントが2019年12月28日（土）新宿ピカデリーにて開催されることも決定。本イベントでは、第1話から2話を上映するほか、堀内犬友さん（織田シナモン信長 役）、犬川登志夫さん（伊達ブー政宗 役）が登壇し、制作の裏側など本作についてのトークを繰り広げるという。
