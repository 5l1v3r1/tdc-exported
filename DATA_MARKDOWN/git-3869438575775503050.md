
# Кировск признан самым комфортным городом для жизни в Мурманской области

Published at: **2019-11-05T00:00:00+00:00**

Author: ****

Original: [Мурманский Вестник](https://www.mvestnik.ru/newslent/kirovsk-priznan-samym-komfortnym-gorodom-dlya-zhizni-v-murmanskoj-oblasti/)

Индекс качества городской среды впервые рассчитали в России. Разработали его Минстрой РФ, АО «Дом РФ» и КБ Strelka, оценивали города по 36 параметрам, таким как благоустройство, состояние соцобъектов, качество дорог, услуг общественного транспорта и другим.В итоге список составили по 1114 городам, среди которых Кировск оказался самым комфортным городом для жизни в нашем регионе: он набрал 202 из 360 баллов. На втором месте — Мончегорск, на третьем — Гаджиево. Мурманск в рейтинге городов Мурманской области оказался на четвертом месте, набрав 198 баллов.Стоит отметить, что данные мониторинга Минстрой использует для размера субсидий, необходимых муниципалитетам на дальнейшие работы по созданию комфортной городской среды.
