
# Визитная карточка Украины: В эфире ZIK презентовали уникальный альманах об истории украинской моды

Published at: **2019-11-06T21:45:00+00:00**

Author: ****

Original: [ZIK.UA](https://zik.ua/ru/news/2019/11/06/vyzytnaya_kartochka_ukrayni_v_efyre_zik_prezentovaly_unykalniy_almanah_ob_1685435)


        Президент всеукраинского объединения работодателей легкой промышленности Александр Соколовский презентовал уникальный для Украины альманах истории украинской моды 1991-2017 лет. По его словам, эта книга может стать своеобразной визитной карточкой Украины за границей.
      
Об этом он рассказал в эфире проекта «HARD с Влащенко» на телеканале ZIK.
«Это уникальная книга, которая только вышла из типографии. Даже ее официальная презентация будет в начале февраля. Это первый в Украине альманах истории отечественной моды с 1991 года до 2017-го», – рассказал Соколовский.
