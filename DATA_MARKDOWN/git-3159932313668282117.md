
# Ata: BC altera projeção de reajuste de administrados em 2019 (mercado) para 5,2%

Published at: **2019-11-05T00:00:00+00:00**

Author: **O Povo**

Original: [O Povo](https://www.opovo.com.br/noticias/economia/2019/11/05/ata--bc-altera-projecao-de-reajuste-de-administrados-em-2019--mercado--para-5-2.html)

O Banco Central alterou sua projeção para a alta dos preços administrados em 2019, conforme a ata do último encontro do Comitê de Política Monetária (Copom), divulgada nesta terça-feira, 5. O índice calculado foi de 3,5% para 5,2% no cenário de mercado. No caso de 2020, o porcentual foi de 4,5% para 4,0%, e para 2021, passou de 4,2% para 3,8%. As estimativas anteriores constavam no Relatório Trimestral de Inflação (RTI) de setembro. O cenário de mercado utiliza como parâmetros as projeções para câmbio e juros do Relatório de Mercado Focus. Cenário híbridoNo cenário que utiliza câmbio constante e Selic do relatório Focus, a projeção para a alta dos preços administrados em 2019 foi de 3,6% para 5,2%. No caso de 2020, passou de 4,6% para 4,1%, para 2021. Já a projeção de elevação dos administrados em 2021 no cenário híbrido está em 4,0%. Ontem, o Focus indicou que a estimativa para 2019 no mercado financeiro é de elevação de 4,69% dos administrados. Para 2020, a expectativa está em 4,10% e, para 2021, em 4,00%. As projeções para os preços administrados ajudaram a formar a base para que o colegiado cortasse na semana passada a Selic (a taxa básica de juros) de 5,50% para 5,00% ao ano. Foi o terceiro corte consecutivo.
