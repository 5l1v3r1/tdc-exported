
# สีกับมาครงและ นโยบายการทูตไวน์

Published at: **2019-11-05T09:34:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/foreign/740122)


        ประธานาธิบดีสี จิ้นผิง ผู้นำจีน และประธานาธิบดีเอมมานูเอล มาครง แวะชิมอาหารและไวน์ของฝรั่งเศส พร้อมทั้งหารือสถานการณ์โลก ที่บูธในงานมหกรรมแสดงสินค้าที่เมืองเซี่ยงไฮ้
      
สำนักข่าวต่างประเทศรายงานจากเมืองเซี่ยงไฮ้ ประเทศจีน เมื่อวันที่ 5 พ.ย.ว่าประธานาธิบดีสี จิ้นผิง ผู้นำจีน และประธานาธิบดีเอมมานูเอล มาครง เยี่ยมชมศาลาแสดงสินค้าของฝรั่งเศส ที่งานมหกรรมแสดงสินค้านำเข้านานาชาติจีน ( ซีไอไออี ) ซึ่งปีนี้จัดขึ้นเป็นครั้งที่ 2 ที่เมืองเซี่ยงไฮ้ ทางตะวันออกของจีน เมื่อวันอังคาร โดยผู้นำทั้งสองประเทศร่วมชิมอาหารและไวน์ชั้นดีของฝรั่งเศสซึ่งเจ้าหน้าที่ประจำศาลาจัดเตรียมไว้ พร้อมทั้งสนทนาแลกเปลี่ยนทรรศนะกันในประเด็นความสัมพันธ์ระหว่างจีนกับฝรั่งเศส และสถานการณ์โลก ซึ่งน่าจะมีการพูดคุยเกี่ยวกับการที่สหรัฐยื่นเรื่องอย่างเป็นทางการต่อสหประชาชาติ ( ยูเอ็น ) เพื่อขอลาออกจากการเป็นสมาชิกข้อตกลงปารีส ว่าด้วยการแก้ไขภาวะโลกร้อน ฉบับปี 2558 อย่างเป็นทางการด้วย ที่ผู้นำจีนกล่าวว่า "เป็นเรื่องน่าเสียใจ"
