
# Vrachtwagen verliest lading op A16; parallelbaan dicht

Published at: **2019-11-06T05:57:10+00:00**

Author: ****

Original: [RTV Rijnmond](https://www.rijnmond.nl/nieuws/188181/Vrachtwagen-verliest-lading-op-A16-parallelbaan-dicht)

Weggebruikers op de A16 moeten rond 07:00 uur rekening houden met een vertraging van meer dan een half uur. De file is meer dan acht kilometer en begint bij Zwijndrecht.
Verkeer vanuit Brabant richting Rotterdam kan omrijden via de A17, A59, A29 en A15. Het is niet bekend wanneer de weg weer open kan.
