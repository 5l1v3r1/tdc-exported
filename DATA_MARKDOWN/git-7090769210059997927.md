
# В Благовещенске ночью в очередной раз горел барак по улице Лазо

Published at: **2019-11-03T02:26:07+00:00**

Author: ****

Original: [Амурская Служба Новостей: новости с характером](https://asn24.ru/news/crime/69741/)

В Благовещенске по улице Лазо сегодня ночью горел очередной барак. Видео с места происшествия опубликовало сообщество @autoroad.blg.«Лазо, 87, опять уже в который раз барак горел, дежурит скорая, два экипажа пожарной техники», — сообщил очевидец.
