
# Φεστιβάλ κούκλας στην Τυφλίδα

Published at: **2019-11-05T09:11:25+00:00**

Author: ****

Original: [euronews](https://gr.euronews.com/2019/11/05/festival-kouklas-tiflida)

Τα φεστιβάλ τέχνης στη Γεωργία, που παρουσιάζουν χειροποίητες κούκλες, αποτελούν πόλο έλξης για καλλιτέχνες από όλο τον κόσμο.
Κάθε λεπτομέρεια και ο τρόπος ένδυσης κάθε μίας από αυτές τις κούκλες είναι ένα έργο τέχνης.
Οι κούκλες από την διάσημη γερμανίδα κατασκευάστρια, Sylvia Weser, για παράδειγμα, μπορεί να κοστίζουν έως και 10.000 δολάρια.
Οι σκελετοί των κούκλων της καλύπτονται από δέρμα και το πρόσωπο και τα χέρια είναι κατασκευασμένα από πορσελάνη. Είναι ζωγραφισμένο στο χέρι προκειμένου να πετύχει τα χαρακτηριστικά σωστά.
Ορισμένες από τις κούκλες είναι διακοσμημένες με ημιπολύτιμους λίθους ή κρύσταλλα.
"Αγαπούσα τις κούκλες μου και ποτέ δεν σκέφτηκα ότι θα τις πουλήσω. Τις έκανα μόνο για τη δική μου συλλογή και σήμερα είμαι προσκεκλημένη στη Μόσχα, στη Νέα Υόρκη και στη Φλόριντα", δήλωσε η Weser.
Οι κούκλες τέχνης που παρουσιάζονται στην έκθεση ποικίλλουν σε στυλ και τεχνική, και αυτό είναι που κάνει τις κούκλες μοναδικές.
