
# Трамп о соглашении по Brexit: Борису следует быть очень осторожным с этим

Published at: **2019-11-01T08:33:00+00:00**

Author: **InfoResist**

Original: [InfoResist](https://inforesist.org/tramp-o-soglashenii-po-brexit-borisu-sleduet-byt-ochen-ostorozhnym-s-etim/)

Новое соглашение по Brexit ограничит возможности США торговать с Великобританией. Об этом заявил хозяин Белого дома Дональд Трам, сообщает BBC.
Ранее Британия и ЕС объявили о достижении долгожданной сделки по Brexit.
