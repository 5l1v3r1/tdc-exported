
# «ВКонтакте» попробовала бороться с оскорблениями в комментариях

Published at: **2019-11-05T10:51:00+00:00**

Author: ****

Original: [Computerworld Россия](https://www.computerworld.ru/news/VKontakte-poprobovala-borotsya-s-oskorbleniyami-v-kommentariyah)


        Новая функция призвана защитить подростков и предотвратить травлю по национальному признаку.
      
4 ноября «ВКонтакте» опробовала алгоритм для борьбы с оскорблениями в комментариях. Когда пользователь писал что-то, решение определяло, есть ли в тексте признаки оскорблений. Если да, автор получал совет отказаться от обидной фразы.
Ранее соцсеть запустила раздел «Центр безопасности» vk.com/safety. Из него можно узнать, как работает модерация, и получить советы по противодействию травле и защите личного пространства.
По результатам эксперимента команда «ВКонтакте» оценит, снижает ли автоматизированное решение количество негатива.
