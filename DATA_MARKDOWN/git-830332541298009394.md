
# Highway 51 near McFarland reopens following rollover crash

Published at: **2019-11-06T17:29:59+00:00**

Author: **Tabatha Wethal, Brandon Arbuckle**

Original: [CHANNEL3000](https://www.channel3000.com/news/crash-closes-stretch-of-highway-51-south-of-mcfarland/1139371993)

McFARLAND, Wis. - All lanes on Highway 51 near McFarland have reopened following a crash that happened Tuesday night.
