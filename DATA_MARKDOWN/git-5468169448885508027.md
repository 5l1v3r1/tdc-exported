
# Zes auto's op elkaar gebotst op A2 tussen Liempde en Best

Published at: **2019-11-07T07:49:17+00:00**

Author: ****

Original: [Blik op nieuws](https://www.blikopnieuws.nl/nieuws/277504/zes-autos-op-elkaar-gebotst-op-a2-tussen-liempde-en-best.html)

Liempde/Best - Op de snelweg A2 bij Liempde gebeurde donderdagochtend een ongeval waarbij zes voertuigen op elkaar botste.
Liempde/Best - Op de snelweg A2 bij Liempde gebeurde donderdagochtend een ongeval waarbij zes voertuigen op elkaar botste.
Alle voertuigen zijn door een bergingsbedrijf afgevoerd. Door het ongeval werden twee rijstroken afgesloten wat voor een flinke vertraging zorgde. Voor zover bekend raakte niemand gewond. Rond 08.15 werd de snelweg weer vrijgegeven.
