
# 【豪門爆姦情2】元祖小開搞不倫　對象竟是大黑松小倆口千金

Published at: **2019-11-05T21:58:00+00:00**

Author: **娛樂組, 攝影組**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191105ent018)

日前爆出「元祖少奶奶」安晨妤與老公張劭緯鬧婚變，安母接受媒體訪問，驚爆女兒「人財兩失」，並說女婿坦承出軌2次、且張家欠安晨妤1,600萬元不還，甚至透露握有不雅照、影片等，引爆一波「元祖家醜」。更醜的是，新聞傳出後事件未因此落幕，本刊取得其中一張獨家照，發現張劭緯出軌對象竟是元祖競爭對手大黑松小倆口的千金邱威樺。
