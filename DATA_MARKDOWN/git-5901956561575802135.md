
# Na Obali eksplodiral bankomat

Published at: **2019-11-06T07:03:00+00:00**

Author: **A. L.**

Original: [Slovenske novice](https://www.slovenskenovice.si/kronika/doma/clanek/na-obali-eksplodiral-bankomat-246316)

ŠKOFIJE – Danes malo po 4. uri je bila policija obveščena, da so v Škofijah razstrelili bankomat. To so nam potrdili pri PU Koper. Pobrali so denar, več pa sledi v kratkem.
