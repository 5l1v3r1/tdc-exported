
# ஜம்மு காஷ்மீரில் லஷ்கர்-இ-தொய்பா பயங்கரவாதி கைது

Published at: **2019-11-02T00:00:00+00:00**

Author: ****

Original: [Dailythnathi.com](https://www.dailythanthi.com/News/India/2019/11/02135235/Indian-Army-and-Jammu-and-Kashmir-Police-have-arrested.vpf)

ஸ்ரீநகர்,காஷ்மீர் மாநிலம் சோபோர் மாவட்டத்தில் ராணுவம் மற்றும் போலீசார் இணைந்து நடத்திய தேடுதல் அதிரடி வேட்டையில் லஷ்கர்-இ-தொய்பா பயங்கரவாத அமைப்பை சேர்ந்த பயங்கரவாதியை கைது செய்தனர்.  கைது செய்த பயங்கரவாதியை உயர்  போலீஸ் அதிகாரிகள் தீவிர விசாரணை நடத்தி வருகின்றனர்.
