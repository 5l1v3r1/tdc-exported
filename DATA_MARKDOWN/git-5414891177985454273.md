
# Каменская «Яворина» отметила свое пятилетие

Published at: **2019-11-06T06:15:00+00:00**

Author: ****

Original: [dndz.tv](http://dndz.tv/article-30042.html)

Объединению украинок г. Каменское (г. Днепродзержинска) исполнилось 5 лет.Активистки ОУ «Яворина» за время существования объединения стали волонтерами, выполняли общественные работы, провели общение с воинами из зоны АТО:- женщины изготовили 15233 м² маскировочных сетей;- для бойцов сплели 200 индивидуальных маскировочных костюмов – кикимор;- для бойцов сделали налокотники, наколенники, зимние костюмы для маскировки и легкие (лесные) костюмы разведчиков;- для бойцов на фронт и в госпиталь изготовили угощения;- проведены различные общественные акции, праздники, мероприятия.Особую роль в общественной организации выполняют члены «Яворины»:- Елена Залевская; - Ирина Мирная; - Ольга Харьковая;- Инна Демидова;- Лидия Ярош;- Надежда Фортуна.
