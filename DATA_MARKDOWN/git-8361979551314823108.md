
# Величезна яма і перекритий квартал: що відбувається на місці прориву труби в центрі Києва

Author: ****

Original: [НВ](https://nv.ua/ukr/kyiv/shcho-vidbuvayetsya-na-misci-prorivu-trubi-v-centri-kiyeva-video-novini-ukrajini-50051868.html)

Як повідомляє Інформатор, один квартал вулиці Саксаганського перекрили для проїзду. На місці прориву вирили величезну яму, там працюють екскаватори.
