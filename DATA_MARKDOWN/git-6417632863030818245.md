
# Ўзбекистон МТЖнинг совуқда ўтган машғулотидан ГАЛЕРЕЯ

Published at: **2019-11-04T14:24:55+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/ozbekiston-mtjning-sovuqda-otgan-mashgulotidan-galereya)

Вадим Абрамов бошчилигида терма жамоамиз бугун ҳам "Жар" стадионининг захира майдонида шуғулланди. Совуқ ҳавода ўтган машғулотдан суратларни сайтимизнинг махсус бўлими орқали томоша қилинг. >>>>> ФОТОГАЛЕРЕЯ

        Черан борасида қўнғироқ, терма жамоада ўйнаши мумкин бўлган Голбан ва Қирғизистонга қарши ўйин нега кўрсатилмаслиги ҳақида — Вадим Абрамов билан интервью
