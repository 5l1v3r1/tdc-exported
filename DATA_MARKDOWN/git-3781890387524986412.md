
# ลิน คว้า 3 เหรียญทอง ประกวด Cake International 2019

Published at: **2019-11-06T02:00:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/article/740190)


        ทั่วโลกทึ่งฝีมือ เค้กดีไซเนอร์ไทย ทีมน้ำตาลลิน กวาด 3 เหรียญทอง จากการประกวด Cake International 2019 ที่อังกฤษ
      
ผ่านไปแล้วกับการประกวด Cake International 2019 ณ NEC เมือง Birmingham ประเทศอังกฤษ งานประกวด.ระดับโลกที่มีผู้ส่งผลงานรวมประกวดจากทั่วโลกหลายพันชิ้น ในหลาย class การประกวด  โดยงานนี้ กลุ่มน้ำตาลไทยรุ่งเรือง ผู้ผลิตน้ำตาลลินและน้ำตาลปั้นตกแต่งเค้กเจ้าแรกของประเทศไทย ได้ส่งทีมไทย เข้าไปร่วมประกวดอีกครั้ง เพราะความมั่นใจในฝีมือที่เต็มเปี่ยมไปด้วยศิลปะความปราณีต  โดยเฉพาะฝีมือของคุณโอปอล์ ที่ถือว่าเป็น Cake designer และครูสอนน้ำตาลปั้นอันดับต้นๆ ของเมืองไทย สร้างสรรค์ผลงานชิ้นเอกมาแล้วมากมาย รวมทั้งยังเป็นกรรมการตัดสินงานประกวด Lin Thailand Sweet Creation ทุกปี จนได้รับการยอมรับจากคนในวงการน้ำตาลปั้นของเมืองไทย  
และในครั้งนี้ คุณโอปอล์ ก็ไม่ได้ทำให้คนไทยผิดหวัง เพราะเขาก็ได้ไปสร้างความตื่นตะลึงให้กับผู้ชมและผู้ประกวดทั้งงาน กับผลงานสุดสวยงามอลังการ ที่ใช้เวลามากกว่า 5 เดือนในการสร้างงานชิ้นนี้  จนคว้ารางวัล เหรียญทอง จากการประกวด Cake International 2019 ณ NEC เมือง Birmingham ประเทศอังกฤษ ใน Class : Decorative Exhibit  กับผลงานชื่อว่า “The Queens”  สื่อสารเรื่องราว ของ Queens แห่งราชวงศ์ อังกฤษ ที่ไม่ใช่แค่รางวัลเหรียญทองที่ได้รับเท่านั้น แต่ ทั้งผู้ชมงาน ผู้เข้าร่วมประกวด และสื่อต่างประเทศต่างให้ความสนใจและทึ่งในฝีมือ ความงดงามอลังการ ของงานชิ้นนี้อย่างมาก ถึงขนาดที่มีผู้ชมงานต้องต่อแถวเพี่อถ่ายรูปร่วมกับผลงาน และสื่ออังกฤษ พากันนำเสนอข่าว ผลงานชิ้นนี้อย่างต่อเนื่อง. นอกจากผลงานของคุณ โอปอล์แล้ว  เค้กดีไซเนอร์ทีมไทย ยังได้รับรางวัลเหรียญทอง อีก 2 ชิ้นงานประกอบไปด้วย 
ศิลปิน เดโชกุล สิงห์อำไพ class: Small decorative exhibit ชื่อผลงาน : Hades World นำเสนอผลงาน แฟนตาซี จากตัวละครหลัก Hades ตามตำนานปกรนัม กรีก
ศิลปิน วริษฐ์ จันทวิชานุวงษ์ class : Decorative Exihibit ชื่อผลงาน : พลับพลาเทวาลัย แนวคิดสื่อถึงความดีในหลักพทธศานา ผ่านศิลปะลายปั้นที่งดงามอย่างไทย
