
# 菊池桃子再婚で高橋克実と的場浩司が火花「僕は夫婦」「僕は幼なじみで恋人」

Published at: **2019-11-05T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/05/0012850351.shtml)

　タレント・菊池桃子（５１）が経済産業省の新原浩朗産業政策局長（６０）と再婚したことを受けて、俳優の高橋克実（５８）と的場浩司（５０）が５日、フジテレビ系の生番組「直撃ＬＩＶＥグッディ！」で、菊池を巡って張り合う一幕があった。
　ＴＢＳ系ドラマ「おじいちゃんは２５歳」（２０１０年）で菊池と夫婦役を演じた高橋は「僕も菊池さんと妻・夫役をやらせていただいたんで、一回。夫なんです。すごいでしょ。これすごいでしょ。僕夫婦です。夫婦」とくどく自慢。
　菊池とは同期で、ＮＨＫ大河ドラマ「信長　ＫＩＮＧ　ＯＦ　ＺＩＰＡＮＧＵ」（１９９２年）とＴＢＳ系連続ドラマ「男嫌い」（１９９４年）で共演している的場は「（『男嫌い』では）僕は幼なじみで恋人なんで…」と張り合い、サバンナの高橋茂雄（４３）が「まあまあまあまあ」と割って入っていた。
