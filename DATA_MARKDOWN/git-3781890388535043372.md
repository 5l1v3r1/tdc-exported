
# นิวเดลีประกาศภาวะฉุกเฉินมลพิษทางอากาศ

Published at: **2019-11-03T04:30:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/foreign/739763)


        ทางการกรุงนิวเดลีซึ่งเป็นเมืองหลวงของอินเดีย ประกาศ "สถานการณ์ฉุกเฉินด้านสุขภาพ" เมื่อปลายสัปดาห์ที่แล้ว เนื่องจากค่ามลพิษทางอากาศสูงเกินมาตรฐานองค์การอนามัยโลกถึง 20 เท่า และเป็นช่วงเทศกาลดีวาลี ที่มีการจุดตะเกียงและดอกไม้ไฟกันเป็นวงกว้าง
      
สำนักข่าวต่างประเทศรายงานจากกรุงนิวเดลี ประเทศอินเดีย เมื่อวันที่ 3 พ.ย. ว่าสำนักงานเทศบาลกรุงนิวเดลีประกาศสถานการณ์ฉุกเฉินด้านสุขภาพเมื่อวันศุกร์ที่ผ่านมา หลังตรวจพบค่าเอคิวไอ ซึ่งเป็นดัชนีวัดความเป็นมลพิษทางอากาศตามมาตรฐานขององค์การอนามัยโลก ( ดับเบิลยูเอชโอ ) สูงถึง 484 ในพื้นที่ ซึ่งค่าเอคิวไอระหว่าง 401 ถึง 500 เกินกว่าเกณฑ์ปลอดภัยของดับเบิลยูเอชโอถึง 20 เท่า และยังถือว่าอยู่ในระดับแย่ที่สุดและเป็นอันตรายที่สุดต่อสุขภาพ ตามการจัดระดับดัชนีวัดคุณภาพอากาศของอินเดียเองด้วย
