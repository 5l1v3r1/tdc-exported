
# Запуск парома через Дунай в Румынию снова откладывается

Published at: **2019-11-04T20:12:00+00:00**

Author: ****

Original: [ТАЙМЕР](http://timer-odessa.net/news/zapusk_paroma_cherez_dunay_v_ruminiyu_snova_otkladivaetsya_473.html)


        Анонсированный заместителем председателя Одесского облсовета Юрием Димчогло запуск с 1 ноября паромной переправы Орловка — Исакча не состоялся.
      
Об этом сообщает издание «Измаил сити».
По данным издания, запуск переправы затягивается из–за юридических формальностей с румынской стороны.Новые сроки запуска пока не называются.
