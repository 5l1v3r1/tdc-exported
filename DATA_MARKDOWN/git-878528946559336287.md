
# だいすけお兄さんスピード婚　一般女性と、交際期間数カ月

Published at: **2019-11-07T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/07/0012855061.shtml)

　ＮＨＫＥテレの「おかあさんといっしょ」に出演していた“だいすけお兄さん”こと、横山だいすけ（３６）が、一般女性と結婚したことが、６日、分かった。自身のブログで同日、婚姻届を提出したことを報告した。
　横山は「お相手は一般女性の方となります。お付き合いは数カ月ですが、どんな時も、この人とだったら幸せな家庭を築けると思い、結婚の運びとなりました」とスピード婚だったことを告白。さらに、９年間「歌のお兄さん」を務めた中で、さまざまな家族に出会ったことを振り返り「自分自身もいつか笑顔溢れる家庭を築きたい！」と夢を描いた。
　なお、所属事務所によると、お相手の女性は妊娠していない。挙式・披露宴などは未定だという。
