
# ФСБ показала видео задержания вымогателей с Северного Кавказа на заправке в Крыму

Published at: **2019-11-02T08:03:00+00:00**

Author: ****

Original: [Фонтанка.ру](https://www.fontanka.ru/2019/11/02/016/)

Группу уроженцев Северного Кавказа задержали в Крыму сотрудники Федеральной службы безопасности. По предварительной информации, они вымогали деньги у местного жителя.
