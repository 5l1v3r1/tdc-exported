
# Главый госканал Беларуси покажет сериал Слуга народа с президентом Украины в главной роли

Published at: **2019-11-06T11:00:00+00:00**

Author: ****

Original: [Українські Новини](https://ukranews.com/news/664199-slugu-naroda-pokazhut-v-belarusi-v-prajm-tajm)


        Главный белорусский государственный телеканал "Беларусь-1" в вечерний прайм-тайм покажет украинский сериал "Слуга народа", где в главной роли снялся Владимир Зеленский, который позже стал президентом Украины. Об этом сообщили в пресс-службе Белтелерадиокомпании.
      

        "Все сезоны этого многосерийного фильма зрители" Беларусь 1 "смогут увидеть в вечерний прайм-тайм - в 20:00 с понедельника по пятницу. В эфир будет выходить по два эпизода картины" Слуга народа"", - сообщает телерадиокомпания.
