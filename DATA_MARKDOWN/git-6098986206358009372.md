
# Полную версию Bastion для iOS отдают бесплатно в App Store

Published at: **2019-11-05T04:15:55+00:00**

Author: **Дарья Громова**

Original: [Лайфхакер](https://lifehacker.ru/bastion-free-ios/)

Обычно игра стоит 379 рублей.
Supergiant Games, разработчики популярной RPG Bastion (вы также можете их знать по игре Transistor), перезалили приложение в App Store. В качестве извинения за неудобства полная версия временно доступна бесплатно для всех желающих.
