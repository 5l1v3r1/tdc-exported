
# Nvidia GeForce 441.12 WHQL: Soporte para el Red Dead Redemption 2 y G-Sync en televisores LG OLED

Published at: **2019-11-04T00:00:00+00:00**

Author: **Borja Rodríguez**

Original: [El Chapuzas Informático](https://elchapuzasinformatico.com/2019/11/nvidia-geforce-441-12-whql-soporte-para-el-red-dead-redemption-2-y-g-sync-en-televisores-lg-oled/)

Nvidia anunció sus nuevos controladores gráficos GeForce 441.12 WHQL Game Ready, los cuales incluyen soporte al inminente lanzamiento del Red Dead Redemption 2 en PC, donde la compañía recomienda una GeForce RTX 2060 para jugarlo en calidad Alta @ 1080p; una GeForce RTX 2070 SUPER para los 2K en calidad Alta; o una GeForce RTX 2080 Ti para jugar a 4K con gráficos Medio-Altos.
Adicionalmente, también incluye soporte para el Need for Speed Heat que se lanzará 3 días después del RDR2, es decir, el 8 de noviembre.
Por otro lado, estos controladores completan el anuncio de LG, donde sus televisores OLED de la serie B9, C9 y E9 serán compatibles con la tecnología Nvidia G-Sync Compatible. Una vez instalado el nuevo firmware del televisor, se requerirá de estos controladores para disfrutar de una experiencia de juego más fluida, sin tearing o stuttering.
Con un menor protagonismo, nuevos/actualizados perfiles de rendimiento para el Red Dead Redemption 2 y Borderlands 3, y se solventan problemas de seguridad.
