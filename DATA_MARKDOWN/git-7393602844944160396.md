
# В Сыктывкаре планируют провести виртуальные соревнования по гребному спорту

Published at: **2019-11-06T20:45:00+00:00**

Author: **Вероника Морохина**

Original: [Комиинформ](https://komiinform.ru/news/188430)

В Сыктывкаре планируют провести виртуальные соревнования по гребному спорту. Такую возможность дают новые тренажеры, которые были закуплены в рамках нацпроекта "Спорт - норма жизни", сообщил Михаил Дудников во время общегородской планерки.
В спортшколе олимпийского резерва "Аквалидер" приобретены австрийские электронные финишные панели, специализированные тренажёрные комплексы для пловцов, специализированные тренажеры для пловцов ("Vasa Ergometer Swim", "TechnoGym Skillrow", "Regent", "Finis Kick Pro", Хюттеля-Мартенса), лопатки, доски для плавания, дыхательные тренажеры и прочее.
На переоснащение потрачено порядка 7 миллионов рублей. Новое оборудование дает возможность не только для тренировок, но и для соревнований. Гребля в закрытых помещениях — официальная дисциплина гребного спорта, утвержденная Министерством спорта РФ в 2015 году, и уже вышла в других регионах на международный уровень.
Виртуальные лодки — гребные эргометры Concept могут соединяться между собой по Wi-Fi, благодаря чему продвижение виртуальных лодок на мониторе отображается в реальном времени. На экране самого тренажера точно отображается скорость, количество гребков, ритм и расход калорий. Одновременно с этим данные передаются на общий сервер, где хранятся результаты подобных соревнований со всей России. Таким образом, гребцы смогут сравнить свои достижения с достижениями других спортсменов.
