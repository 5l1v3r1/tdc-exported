
# మోకాళ్లపై మోకరిల్లి వేడుకున్న సమ్మె కార్మికులు

Published at: **2019-11-07T08:23:20+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=948741)

ADVT
భద్రాద్రి కొత్తగూడెం: తమ కడుపులు కొట్టొద్దంటూ ఆర్టీసీ సమ్మె కార్మికులు.. తాత్కాలిక ఉద్యోగులను వేడుకున్నారు. భద్రాద్రి కొత్తగూడెం జిల్లా భద్రాచలంలో డిపో గేటు ముందు బైటాయించిన కార్మికులు.. తాత్కాలిక ఉద్యోగులు రాగానే మోకాళ్లపై మోకరిల్లి తమ కడుపులు కొట్టొద్దని వేడుకున్నారు.
