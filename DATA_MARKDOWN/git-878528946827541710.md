
# 東国原　五輪は「都市ではなく『国』開催にすべき」

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/01/0012839755.shtml)

　元宮崎県知事でタレントの東国原英夫が１日、ツイッターを更新。東京五輪のマラソン・競歩の札幌移転開催についての考えを記した。
　札幌移転正式決定発表以前の１日午前に投稿したツイートでは、「東京は、札幌移転案を受け入れる方向らしい。まぁ、仕方無いだろう。五輪憲章や開催地契約、各規約上、主催者であるＩＯＣに権限・決定権があるのだから」とＩＯＣの決定を受け入れるしかないことは「仕方なし」とした上で、「しかし、ＩＯＣも決定が急過ぎ・強権ぎ」（※原文ママ）とＩＯＣのやり方には批判的な思いをつづった。
　その上で「何回も言うが、複数都市開催が可能なら、都市では無く『国』開催にすべき。又、今後は、複数国開催や恒久都市開催案等も含め、柔軟に対応すべきである」と意見。「アジェンダ２０２０で是非、議論して頂きたい」と五輪改革案「アジェンダ２０２０」での議論を促したいとツイートした。
　東国原は自身もマラソンにたびたび出場。市民ランナーとして知られる。
