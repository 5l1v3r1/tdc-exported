
# На Орловском шоссе в Туле «ВАЗ» сбил двоих пешеходов

Published at: **2019-11-01T13:27:00+00:00**

Author: ****

Original: [Тульская Служба Новостей](https://www.tsn24.ru/na-orlovskom-shosse-v-tule-vaz-sbil-dvoih-peshehodov.html)

Дорожно-транспортное происшествие на улице Орловское шоссе в Туле произошло вечером 31 октября.
В 17:50 56-летний водитель автомобиля ВАЗ-21150 допустил наезд на двух пешеходов, переходящих проезжую часть по нерегулируемому пешеходному переходу.
По данным УГИБДД УМВД России по Тульской области, 16-летний молодой человек госпитализирован с различными травмами, 45-летний мужчина после оказания медицинской помощи был отпущен.
