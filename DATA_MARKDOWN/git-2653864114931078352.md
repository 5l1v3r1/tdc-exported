
# Череповчане сравнили беседку для молодожёнов с «клеткой блудного попугая Кеши»

Published at: **2019-11-03T20:27:00+00:00**

Author: ****

Original: [35медиа](https://35media.ru/news/2019/11/03/cherepovchane-sravnili-besedku-dlya-molodozhenov-s-kletkoj-bludnogo-popugaya-keshi)

Череповчане высказывают недовольство по поводу новой беседки для молодожёнов. В социальных сетях появляются негативные комментарии.
Больше всего горожан возмущает внешний вид арт-объекта. Многие сравнивают беседку с клеткой блудного попугая Кеши или кладбищенской оградой.
Беседку для молодожёнов возводили около двух месяцев строительные организации Череповца. Всю работу они делали на собственные деньги.
