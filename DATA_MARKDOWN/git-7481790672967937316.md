
# 「当地域にゲリラ攻撃の可能性」村内全域に２回誤放送

Published at: **2019-11-07T20:00:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191107-OYT1T50252/)

　岩手県野田村は７日、「ゲリラ攻撃があった」との趣旨の誤った放送を、防災行政無線を通じて村内全域に流した。
　村の担当者によると、午前１０時５０分頃、全国瞬時警報システム（Ｊアラート）のサイレンが流れた後に、「ゲリラ攻撃情報。当地域にゲリラ攻撃の可能性があります」と放送した。誤放送は２回流れ、約１０分後に訂正の放送を流した。
　当時、業者がＪアラートシステムの更新作業をしており、動作確認中に誤放送が流れたという。
