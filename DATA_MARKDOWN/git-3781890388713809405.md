
# สหรัฐแจ้งยูเอ็นเรื่องการถอนตัวจากข้อตกลงปารีส

Published at: **2019-11-05T00:46:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/foreign/740029)


        รัฐบาลสหรัฐยื่นเรื่องอย่างเป็นทางการต่อสหประชาชาติ แจ้งการขอลาออกจากการเป็นสมาชิกความตกลงปารีส ว่าด้วยการแก้ไขวิกฤติโลกร้อน
      
สำนักข่าวต่างประเทศรายงานจากกรุงวอชิงตัน ประเทศสหรัฐอเมริกา เมื่อวันที่ 4 พ.ย. ว่านายไมค์ ปอมเปโอ รมว.กระทรวงการต่างประเทศสหรัฐ กล่าวเมื่อวันจันทร์ ว่ารัฐบาลวอชิงตันได้ส่งหนังสืออย่างเป็นทางการถึงนายอันโตนิโอ กูเตร์เรส เลขาธิการสหประชาชาติ ( ยูเอ็น ) เพื่อแสดงความจำนงขอถอนตัวออกจากการร่วมเป็นหนึ่งในภาคีข้อตกลงปารีส ฉบับปี 2558  ซึ่งเป็นความตกลงตามกรอบอนุสัญญาสหประชาชาติว่าด้วยการเปลี่ยนแปลงสภาพภูมิอากาศ ( ยูเอ็นเอฟซีซีซี ) ว่าด้วยการควบคุมการปล่อยก๊าซคาร์บอน เพื่อรักษาอุณหภูมิเฉลี่ยของโลกให้อยู่ในระดับต่ำกว่า 2 องศาเซลเซียสเมื่อเทียบกับยุคก่อนปฏิวัติอุตสาหกรรม
