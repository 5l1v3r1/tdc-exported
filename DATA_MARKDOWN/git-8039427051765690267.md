
# విశాఖ జిల్లాలో రోడ్డు ప్రమాదం: దంపతులు మృతి

Published at: **2019-11-01T07:22:48+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=943538)

Ph: 88865 47666ప్రవేశం ఉచితం ADVT
విశాఖ: జిల్లాలోని కసింకోట మండలం, తాళ్లపాలెం జాతీయ రహదారిపై రోడ్డు ప్రమాదం జరిగింది. వేగంగా వస్తున్న పెట్రోల్ ట్యాంకర్.. నడిచి వెళ్తున్న దంపతులను ఢీ కొట్టింది. భార్య, భర్తలు అక్కడికక్కడే మృతి చెందారు. శుక్రవారం తెల్లవారుజామున వారు పొలం పనికి వెళ్తుండగా ఈ ప్రమాదం జరిగింది. మృతులు తాళ్లపాలెం గ్రామానికి చెందిన కరణం సోమినాయుడు (55), కరణం పైడితల్లి( 50) గా గుర్తించారు. సమాచారం అందుకున్న పోలీసులు సంఘటనా ప్రదేశానికి చేరుకుని పోస్టుమార్టం నిమిత్తం మృత దేహాలను ప్రభుత్వం ఆస్పత్రికి తరలించి, కేసు నమోదు చేసి దర్యాప్తు చేపట్టారు.
