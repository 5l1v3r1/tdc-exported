
# Хэмилтон досрочно стал шестикратным чемпионом Формулы-1

Published at: **2019-11-03T20:54:34+00:00**

Author: **Euronews **

Original: [euronews](https://ru.euronews.com/2019/11/03/ru-f1-hamilton-6th-title)

Британский пилот команды "Мерседес" Льюис Хэмилтон в шестой раз в карьере стал победителем чемпионата мира по автогонкам в классе машин "Формула-1".
34-летний британец занял второе место на Гран-при США, победу в гонке одержал партнер британца по команде финн Валттери Боттас. В общем зачете Хэмилтон стал недосягаем для Боттаса.
По количеству чемпионских титулов британец опередил Хуана Мануэля Фанхио, впереди него только Михаэль Шумахер с семью победами.
Ранее Хэмилтон становился чемпионом мира в 2008, 2014, 2015, 2017, 2018 годах.
