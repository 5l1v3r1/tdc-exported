
# В Кривом Роге правоохранители изъяли более 500 пачек контрафактных сигарет

Published at: **2019-11-01T09:44:00+00:00**

Author: ****

Original: [34 телеканал](https://34.ua/v-krivom-roge-pravoohraniteli-izyali-bolee-500-pachek-kontrafaktnyh-sigaret_n82752)

За два дня в Металлургическом и Центрально-Городском района Кривого Рога патрульные совместно с сотрудниками Таможенной службы Украины провели рейды. Правоохранители обнаружили несколько точек продажи нелицензированного табака. Полицейские конфисковали 300 пачек сигарет, которые ввезли в Украины с помощью контрабанды.
Кроме того, в Саксаганском районе стражи порядка нашли прилавок. Реализатор продавал сигареты без акцизной марки Украины. На продавца составили административный протокол. Несколько сотен пачек табачных изделий изъяли.
