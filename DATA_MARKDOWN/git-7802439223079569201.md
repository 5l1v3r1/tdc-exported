
# League of Legends Rambah Mobile, Eksklusif di Oppo

Published at: **2019-11-04T19:27:02+00:00**

Author: **Rachmatunnisa - detikInet**

Original: [detikinet](https://inet.detik.com/games-news/d-4771959/league-of-legends-rambah-mobile-eksklusif-di-oppo)

Jakarta - Riot Games mengumumkan game PvP populer besutannya, League of Legends (LoL), hadir di platform selain PC. Versi terbaru game ini, Wild Rift, ditujukan untuk konsol dan perangkat mobile. Studio game tersebut sudah mengindikasikan kalau Android adalah platform pertama yang menjadi rumah bagi LoL versi mobile. Riot Games bahkan mengerucutkannya lagi dengan memilih OPPO sebagai ponsel pertama yang secara eksklusif kedatangan LoL.Tidak diketahui akan berlangsung berapa lama eksklusivitas ini. Namun sejatinya, pemilihan Oppo sebagai ponsel pertama yang kedatangan LoL tidak mengherankan. Sebelumnya, Oppo memang sudah aktif mendukung Riot Games menggelar sejumlah turnamen LoL. Bahkan, Oppo pada Oktober lalu menyatakan akan mensponsori turnamen game LoL hingga 2024. Pabrikan ponsel adal China ini percaya diri bahwa software dan AI yang dikembangkannya cukup menjadi modal untuk mendukung event dan liga profesional bahkan yang serius. Berbicara Wild Rift, Riot Games dengan tegas meyakinkan bahwa game ini lebih dari sekadar versi yang di-port dari PC. Varian terbaru ini akan punya maps dan kontrol sendiri untuk mode klasik 5v5. Di sisi lain, Riot Games juga mengingatkan bahwa tidak semua champ, skin atau lock dari game klasik ini akan diboyong ke mobile. Selain itu, durasi pertandingan game versi terbaru akan lebih pendek, sekitar 15-20 menit, sesuai dengan feedback dari para fans game ini.League of Legends: Wild Rift dijadwalkan mulai tersedia awal 2020.
