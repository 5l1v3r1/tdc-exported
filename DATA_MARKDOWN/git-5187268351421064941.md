
# Usa o Visual Studio Code? Mais 5 super dicas para dominar este editor

Published at: **2019-11-07T21:00:01+00:00**

Author: **Pedro Pinto**

Original: [Pplware](https://pplware.sapo.pt/truques-dicas/usa-o-visual-studio-code-mais-5-super-dicas/)

O Visual Studio Code é provavelmente um dos melhores IDEs para código e está disponível para os principais sistemas operativos. Este editor gratuito e é direcionado para programadores que procuram um IDE completo e que, ao mesmo tempo, seja fluído.
Depois de publicarmos aqui 5 dicas, conheçam mais 5 super dicas para dominar este editor.
Sabia que através do Visual Studio Code (VSC) pode ter acesso direto ao terminal? É verdade… e é fácil! Esta funcionalidade é extremamente útil quando precisamos de executar ou analisar a nossa aplicação/serviço na linha de comandos.
Para aceder ao terminal via VSC basta, por exemplo, selecionar um diretório ou um ficheiro, clicar com o botão do lado direito do rato e depois escolher a opção “Open in Terminal“.
Caso pretendam comentar um bloco de código o VSC também permite. Para isso basta selecionar o que pretendemos comentar e usar as teclas CTRL + K. EM seguida devem usar o atalho CTRL + C. Para descomentar façam Ctrl+K e depois Ctrl+U.
Considere que pretende proceder a alterações em várias linhas de código de uma única vez. Com o VSC tal operação é simples. Para tal basta selecionar o que se pretende, usando a tecla ALT (ou Option no macOS) de depois acrescentar informação ou remover.
Precisa de realizar uma pesquisa em vários ficheiros que estão abertos no VSC? O editor tem tal funcionalidade. Usando as teclas de atalho Ctrl + F o utilizador realiza uma pesquisa no ficheiro atual. Para pesquisar em vários ficheiros usem o atalho Ctrl + Shift + F.
Que mudar de tema no VSC? São vários os que estão disponíveis. Pesquisando aqui pode escolher o tema que mais gosta e que lhe criará o melhor ambiente para programar. As opções são muitas e algumas delas bastante interessantes.
E são estas as dicas que tínhamos para hoje. Para uma terceira versão desta rubrica gostaríamos de contar com as vossas dicas. Como viram, o VSC é um editor extremamente flexível e prático. Se nunca usaram, experimentem
