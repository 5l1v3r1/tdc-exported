
# 侍ジャパン　ミス逃さず逆転　菊池＆誠也のカープコンビが執念のタイムリー

Published at: **2019-11-05T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/05/0012851637.shtml)

　「プレミア１２・１次ラウンド、日本－ベネズエラ」（５日、桃園国際野球場）
　侍ジャパンが相手のミスを逃さず、逆転に成功した。
　１点を追う五回、１死一塁。この回から登板した右腕ソティリェットに対し、坂本はボテボテの遊ゴロ。だが、二塁へのトスが高く浮き、暴投に。１死一、三塁のチャンスとなり、２番菊池が右翼線へ同点の適時二塁打。さらに１死満塁から４番鈴木が勝ち越しの中前適時打を放った。
　菊池＆鈴木のカープコンビが重苦しいムードを打破。２－１とした。
