
# Kranjčane prebudil alarm, prihiteli so gasilci

Published at: **2019-11-07T05:52:00+00:00**

Author: **N. B.**

Original: [Slovenske novice](https://www.slovenskenovice.si/kronika/doma/clanek/kranjcane-prebudil-alarm-prihiteli-so-gasilci-246656)

KRANJ – Ob 22.21 se je na Slovenskem trgu v objektu sprožil požarni alarm, poroča uprava za zaščito in reševanje.Gasilci GARS Kranj so na kraju ugotovili, da se po hodniku širi dim, vendar vzroka niso našli. Prezračili so prostore in resetirali centralo.
