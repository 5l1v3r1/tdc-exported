
# 【診所職場事｜寶豬趣聞】等收醫療卡｜珍寶豬

Published at: **2019-11-04T14:57:29+00:00**

Author: **珍寶豬**

Original: [經濟一週](https://www.edigest.hk/article/122329/%e5%b0%88%e6%ac%84/%e9%86%ab%e7%99%82%e5%8d%a1-%e8%a8%ba%e6%89%80%e8%81%b7%e5%a0%b4%e4%ba%8b-%e5%af%b6%e8%b1%ac%e8%b6%a3%e8%81%9e-%e7%8f%8d%e5%af%b6%e8%b1%ac/)

有日一位小姐嚟到診所，佢一嘢『啪』咗張卡喺登記處枱面～我望望佢張卡，噢，又係尊貴嘅醫療卡，不過好對唔住，我哋唔收呢張。
撰文：珍寶豬（funnyclinic）｜圖片：電視截圖
我：「我哋唔收呢張卡啊！」佢：「呢張都唔收？收啲咩呀？」我指指貼喺出面嘅醫療卡樣：「你可以望望出面張紙，印晒我哋收邊張邊隻色嘅醫療卡喺度，如果冇你嗰張嘅話，你可以打電話去返卡中心度問吓附近有邊間診所收！」佢：「你肯定你記得晒收邊一張？」呢啲都係從N次失敗中堆成嘅經驗啊~我：「你呢張我哋肯定係唔收嘅！」佢：「你打電話上去問呀，我認為係你自己唔知可以收囉，我唔走啦，出面又曬又熱，我咪喺度等到你收囉！」你鍾意啦，你鍾意喺度等到人老珠黃都得呀，可以睇住你變老，或者成為一舊望卡石都係一件幸福嘅事啊。
我：「我打去問問唔係問題，我而家就打去呀！」佢企喺我面前睇住聽住我打電話，當然最後都係話唔收…我：「小姐，我哋真係唔收呢張啊！」佢：「我係叫你打上去申請收呀！唔係問收唔收呀！」我伸你一腳送你出門口就得。我：「冇得即時申請～你去間收嘅診所睇會快好多喎，或者你喺度睇完我俾返收據你，你再CLAIM返？咁可唔可以呢？」佢：「NO WAY！我喺度等到你申請到為止，本小姐大把時間！」佢自己行去蹺埋腳坐低。醫生睇電腦見到小姐坐咗喺出面,就問我:「係咪有症呀？」我：「佢張醫療卡我哋唔收嘅，佢叫我哋同佢申請收喎，仲話等到肯收為止。」
醫生膊頭郁郁：「佢係客，佢點話點好啦！」我：「即係打上去卡公司同佢搞呀？咁好似…」醫生搶答：「佢話要等咪由佢等，你同佢講多一次唔收就得啦！」嘻嘻～結果小姐坐咗個半鐘嘆夠冷氣就講住粗口走咗。
