
# Пожар в Калифорнии сняли из космоса

Published at: **2019-11-01T18:49:00+00:00**

Author: **Корреспондент.net**

Original: [Корреспондент.net](https://korrespondent.net/world/4156008-pozhar-v-kalyfornyy-snialy-yz-kosmosa)

Масштабный пожар в Калифорнии виден из космоса, сообщает Space.com.
По данным лесного и противопожарного департамента Калифорнии, на данный момент площадь пожара составляет 310 кв. км.
Астронавт NASA Эндрю Морган смог сфотографировать пожары из иллюминатора Международной космической станции. Фото он разместил в своем Twitter.
