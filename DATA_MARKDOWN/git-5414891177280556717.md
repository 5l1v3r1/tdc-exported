
# Для молодежи г. Каменское в центре Мир HUB провели тренинг

Published at: **2019-11-04T12:43:00+00:00**

Author: ****

Original: [dndz.tv](http://dndz.tv/article-30027.html)

02.11.19 молодежный центр города Каменское (Днепродзержинска) провел для молодых людей тренинг - "Сексуальное образование".Большую часть слушателей составили девушки.Тренинг прошел в МЦ "Мир HUB" под руководством тренера Александра Беленинника.
