
# Представлен электрический Ford Mustang… на механике

Published at: **2019-11-06T09:00:27+00:00**

Author: **Артём Баусов**

Original: [iPhones.ru](https://www.iphones.ru/iNotes/predstavlen-elektricheskiy-ford-mustang-na-mehanike-11-06-2019)

На тюнинг-шоу SEMA показали электрическую версию Ford Mustang.
Прототип новенького Ford Mustang Lithium EV Concept собирали совместно с компанией Webasto. Электрокар получил два движка Phi-Power с двумя инверторами мощности, выдающим 912 л.с. и 1356 Нм.
Главной особенностью автомобиля стала шестиступенчатая механическая коробка передач, усовершенствованная версия Getrag MT82. Но больше о ней сейчас нет информации.
Переключаться между четырьмя режимами управления можно с помощью 10,4-дюймового сенсорного дисплея информационно-развлекательной системы. За питание отвечает 800-вольтовая аккумуляторная система.
