
# Αρχιεπίσκοπος για εμπλοκή Εκκλησίας: Όποιου δεν αρέσει να πέσει χώρκα του

Published at: **2019-11-06T20:15:00+00:00**

Author: **sigmalive**

Original: [SIGMALIVE](https://www.sigmalive.com/news/politics/596089/arxiepiskopos-gia-emploki-ekklisias-opoiou-den-aresei-na-pesei-xorka-tou)

«Η Εκκλησία θα δουλεύει σε όλους τους τομείς. Όποιου δεν αρέσει «μπορεί να πέσει χώρκα του».
Σε αυτή τη δήλωση προέβη πριν από λίγο ο Αρχιεπίσκοπος, στο περιθώριο εκδήλωσης στο σωματείο "Εληά Λυθροδόντα", σχολιάζοντας την εμπλοκή που είχε η Εκκλησία στο πρόγραμμα πολιτογραφήσεων. 
"Μερικοί θέλουν την εκκλησία να περιορίζεται μόνο στο Kύριε Έλέησον στις Θείες Λειτουργίες", ανέφερε. Για να προσθέσει ότι η εκκλησία της Κύπρου θα είναι και θρησκευτικός οργανισμός και εθνικός και πολιτιστικός και θα δουλεύει σε όλους τους τομείς. Και όποιου δεν αρέσει «μπορεί να πέσει χώρκα του», είπε.
Εγώ δεν είδα ούτε διαβατήρια, ούτε υπηκοότητες, ανέφερε. Η κυβέρνηση έχει την αστυνομία και την Interpol για να λάβει πληροφορίες, ξεκαθάρισε.
Η Εκκλησία δεν φοβάται δουλεύουμε με ανοιχτά τα χαρτιά. Ούτε θέλουμε να φάμε από κανέναν, τόνισε.
