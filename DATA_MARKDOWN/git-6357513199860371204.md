
# Тизер новых гаджетов для Rainbow Six Siege подтверждает недавнюю утечку

Published at: **2019-11-07T17:59:07+00:00**

Author: **Андрей Маковеев**

Original: [Stopgame](https://stopgame.ru/newsdata/40708)

На прошлой неделе проверенный инсайдер Kormora выдал очередную порцию сведений о будущих новинках для Rainbow Six Siege. Сегодня Ubisoft опубликовала тизер обновления Operation Shifting Tides, который подтверждает утёкшую информацию.В ролике показаны новые гаджеты и оружие:• Снайперская винтовка CSRX 300. Судя по видеоряду, выпущенная из неё пуля пробивает больше одного препятствия.• Подствольный гранатомёт, стреляющий болтами взрывного действия. Болт выпускает разряд, который уничтожает гаджеты в определённом радиусе, не нанося урона оперативникам.• Система «Маг-НЭТ» — девайс, притягивающий и взрывающий снаряды. Согласно описанию с сайта «Осады», устройство реагирует в том числе на дротики Capitão и болты взрывного действия.Напомним, что, если верить утечке, снайперская винтовка со взрывными болтами достанется индийскому штурмовику (судя по заметке с сайта Ubisoft, это женщина с позывным Kali). Ну а «Маг-НЭТ» — гаджет для защитника из Кении.Полноценная презентация Operation Shifting Tides состоится во время финалов киберспортивных состязаний по Siege в Токонамэ. Чтобы не пропустить премьеру, заходите на Twitch-канал Rainbow Six 10 ноября.
