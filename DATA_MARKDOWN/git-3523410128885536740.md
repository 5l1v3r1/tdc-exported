
# 
        تیم کشتی فرنگی امید ایران قهرمان جهان شد

به گزارش بولتن نیوز، تیم کشتی فرنگی زیر ۲۳ سال (امید) کشورمان در رقابت های جهانی مجارستان با کسب ۳ مدال طلا و ۳ مدال برنز بعنوان قهرمانی دست یافت.
