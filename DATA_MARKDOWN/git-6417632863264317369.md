
# Болалар футболи. Тошкентда Ҳамдўстлик кубоги-2019 халқаро турнири старт олди

Published at: **2019-11-02T12:28:29+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/bolalar-futboli-toshkentda-hamdostlik-kubogi-2019-xalqaro-turniri-start-oldi)

ЎФА, ТФА ва "Спартак Тошкент" жамоалари ҳамкорлигида ўтказилаётган мусобақада 8 та жамоа иштирок этмоқда. 
"А" гуруҳи: "Одил Аҳмедов ФА" (терма), "Жас Киран" (Олмата), "Пахтакор" (Тошкент), Академия ЮГ (Бишкек)
"Б" гуруҳи: "Спартак" (Тошкент), ФШМ (Олмата), "Бунёдкор" (Тошкент), "Олға" (Бишкек).
Бугун мусобақа доирасида тўртта ўйин бўлиб ўтди. Унда қуйидаги натижалар қайд этилди:
"Спартак" - ФШМ 1:1
"Жас Киран" - "Пахтакор" 0:0
"Одил Аҳмедов ФА" - Академия ЮГ 5:1
"Олға" - "Бунёдкор" 0:16
Беллашувлар 7 ноябрга қадар давом этади.
