
# Что, если я гей?: в Сети появился новый трек Eminem об ЛГБТ

Published at: **2019-11-01T17:31:42+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-01/chto-esli-ya-gey-v-seti-poyavilsya-otryvok-pesni-eminem-o-lgbt)

В Сети появился новый трек рэпера Eminem, который он записал с Joyner Lucas. Песня получила название What If I Was Gay ("Что, если я гей?"). Судя по записи, этот вопрос себе задают оба артиста. 
"Что, если я ханжа, который боится взглянуть правде в глаза? Постой. Вдруг я тоже гей? (What if I'm a hypocrite who is afraid to just face truth? Wait / What if I told you I'm gay too?)" — читает в нем Eminem.
Отметим, это не первая совместная работа рэперов. Ранее они записали трек "Lucky You" для альбома Eminem Kamikaze.
Рэпера Eminem считают одним из главных гомофобов среди артистов. В своих треках он часто использует оскорбительные слова для обозначения людей с нетрадиционной сексуальной ориентацией.
