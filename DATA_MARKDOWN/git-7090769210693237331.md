
# В районе Тайвань в Благовещенске поставили две новые автобусные остановки

Published at: **2019-11-01T02:20:59+00:00**

Author: ****

Original: [Амурская Служба Новостей: новости с характером](https://asn24.ru/news/society/69635/)

В одном из отдаленных районов Благовещенска появились две новые автобусные остановки. С сегодняшнего дня автобусы маршрутов № 22 и № 28 ездят по улице Муравьева-Амурского от Строителей до Шафира.
Необходимость в оборудовании остановочных пунктов появилась из-за активной застройки данного района, сообщили в пресс-службе городской администрации.Ранее для удобства жителей, проживающих в микрорайоне «Северный», по ул. Муравьева-Амурского были обустроены два остановочных пункта и пешеходный переход с вызывным светофором.
