
# Самым дружелюбным российским городом стал Грозный, недружелюбным — Якутск

Published at: **2019-11-07T18:48:00+00:00**

Author: ****

Original: [EADaily](https://eadaily.com/ru/news/2019/11/07/samym-druzhelyubnym-rossiyskim-gorodom-stal-groznyy-nedruzhelyubnym-yakutsk)

Самым дружелюбным городом России является столица Чечни Грозный, самым недружелюбным столица Якутии — Якутск. Об этом заявляют специалисты портала недвижимости Domofond.ru, которые опросили 90 тысяч обитателей 200 крупных российских городов.
Участники опроса оценивали утверждение «люди, с которыми я живу по соседству, в основном дружелюбные и вежливые» по десятибалльной шкале, после чего эксперты для каждого города вывели средний балл. Оказалось, что в миллионниках живут менее дружески настроенные горожане, чем по стране в целом. А шесть из десяти позиций в верхней части рейтинга (более дружелюбной) заняли южные города.
