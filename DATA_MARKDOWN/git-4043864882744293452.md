
# Кабмин создал комиссию, которая будет проверять знание украинского

Published at: **2019-11-06T15:20:00+00:00**

Author: ****

Original: [Украинская правда](https://www.pravda.com.ua/rus/news/2019/11/6/7231192/)

Кабинет министров создал национальную комиссию по стандартам государственного языка, которая будет проверять уровень владения государственным языком для получения гражданства.
Источник: пресс-служба Кабмина
Дословно: "6 ноября Кабинет министров на своем заседании создал национальную комиссию по стандартам государственного языка, которая также будет проверять уровень владения государственным языком для получения гражданства Украины или занятия определенных законом должностей".
Детали: Кроме того, правительство начало формировать стратегию в области культуры и искусства. Формированию стратегии будет предшествовать проведение социологического исследования для обеспечения доступа граждан к потреблению культурных услуг, доступа к искусству.
