
# Против экс-мэра Каспийска Джамалудина Омарова возбудили уголовное дело

Published at: **2019-11-07T11:41:00+00:00**

Author: **Юлия Никулина**

Original: [Регион Online](https://news-r.ru/news/dagestan/381064/)


        Его ранее задержали силовики, а в доме и в кабинете бывшего чиновника прошли обыски
      
Махачкала, 7 ноября. Появились подробности задержания бывшего главы администрации Каспийска и ныне депутата Народного собрания Дагестана Джамалудина Омарова. Против него возбуждено уголовное дело по ч. 2 статьи 286 УК РФ «Превышение должностных полномочий главой органа местного самоуправления». 
По версии следствия, Омаров, занимая пост мэра, в конце декабря 2010 года незаконно, без проведения торгов, подписал постановление о предоставлении частному лицу участка площадью 3 гектара для строительства гостинично-развлекательного комплекса с лодочной станцией.
- На основании этого документа после был заключен незаконный договор аренды участка, кадастровая стоимость которого превышает 42 миллионов рублей. Это повлекло за собой существенное нарушение интересов муниципалитета, - сообщили РИА «Регион online» в пресс-службе СУ СК РФ по Дагестану.  
Сейчас по уголовному делу проводятся обыски по месту работы и по месту жительства Омарова. Решается вопрос об избрании меры пресечения бывшему мэру Каспийска. Расследование продолжается.
Ранее «Регион online» сообщал, что в Дагестане бывшего мэра Каспийска и ныне депутата Омарова задержали силовики. Есть данные, что именно из-за него в администрации утром 7 ноября прошли обыски.
