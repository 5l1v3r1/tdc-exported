
# Фінський пілот Формули-1 Валттері Боттас виграв Гран-прі США

Published at: **2019-11-04T02:30:00+00:00**

Author: ****

Original: [ZIK.UA](https://zik.ua/news/2019/11/04/finskyy_pilot_formuly1_valtteri_bottas_vygrav_granpri_ssha_1682855)


        В Остіні, на Трасі Америк завершився 19-й етап чемпіонату світу-2019 з автоперегонів у класі болідів Формули-1.
      
Переможцем Гран-прі США став фін Валттері Боттас («Мерседес»), який на 4.148 секунди випередив свого партнера по команді британця Льюїса Гемілтона.
Трійку призерів доповнив голландець Макс Ферстаппен («Ред Булл»), який програв переможцю 5.002 секунди.
