
# Каспарайтис: Вашингтон сделает все для второго чемпионства

Published at: **2019-11-04T14:38:00+00:00**

Author: ****

Original: [Allhockey.ru](https://allhockey.ru/news/show/368952-Kasparajtis_Vashington_sdelaet_vse_dlya_vtorogo_chempionstva)

Олимпийский чемпион Дарюс Каспарайтис прокомментировал лидерство "Вашингтона" в регулярном чемпионате НХЛ."Они уже пару лет считаются одними из главных фаворитов регулярного чемпионата. Там все играют на высоком уровне, потому что выработалась определенная победная культура, которую создал Овечкин, а остальные игроки, которые уже не первый год в "Вашингтоне", ее подхватили. Сохранился костяк с чемпионского сезона, все знают свои способности, почувствовали вкус победы. Эти игроки снова решили основательно взяться за дело и выдают свой лучший уровень. "Кэпиталз" сделают все для второго чемпионства.Мы все болеем за Овечкина, хотим, чтобы он побил рекорд Гретцки. Саша забивает, это хорошо. Он такой игрок, который может провести несколько игр без голов, а потом сразу выдать пять шайб подряд.Что касается Кузнецова, то ему тяжело после такого скандального межсезонья. Ему еще нужно время, чтобы окончательно войти в игру и забыть все проблемы. Все знают, что он талантливый хоккеист, он не может растерять талант за несколько месяцев. У Жени есть все возможности найти свою игру и стать одним из лидеров клуба. Неприятности иногда требуют время для психологического восстановления, надо просто немного подождать", - сказал Каспарайтис.
