
# В Сети появилось видео с места смертельного ДТП в Туле с участием маршрутки

Published at: **2019-11-03T10:10:00+00:00**

Author: ****

Original: [Тульская Служба Новостей](https://www.tsn24.ru/v-seti-poyavilos-video-s-mesta-smertelnogo-dtp-v-tule-s-uchastiem-marshrutki.html)

Дорожно-транспортное происшествие, в результате которого пострадали два человека и один погиб, произошло вечером 1 ноября на Алексинском шоссе в Туле.
Видеозапись с места происшествия на канале YouTube разместила пользователь, представленная как Галина Абрамова.
По предварительной версии, водитель маршрутного автобуса не справился с управлением, из-за чего произошло опрокидывание.
По факту ДТП возбуждено уголовное дело по статье «Нарушение ПДД и эксплуатации транспортных средств, повлекшее по неосторожности смерть человека».
