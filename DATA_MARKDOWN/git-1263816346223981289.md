
# Конфуз на Гран-При по фигурному катанию во Франции

Published at: **2019-11-03T14:32:00+00:00**

Author: **Елена Баранова**

Original: [МОЛНЕТ.RU](https://www.molnet.ru/mos/ru/culture/o_717221)


        Организаторы по ошибке вручили золотую медаль американкой спортсменке вместо завоевавшей ее россиянки Алены Косторной.
      
Первое место Гран-При по фигурному катанию завоевала российская спортсменка Алена Косторная, которая на 20 баллов обошла свою соотечественницу Алину Загитову. Третье место пъедестала заняла фигуристка из США Мэрайе Бэлл. Именно ей организаторы по ошибке и вручили золотую медаль, пишет РИАМО.
Российские спорстменки с юмором отнеслись к сложившейся ситуации. Бэлл, поняв ошибку, перевесила награду на шею Косторной. После этого церемония пошла по ранее утвержденному плану.
