
# Aji Santoso Nilai Stadion GBT Layak Jadi Venue Piala Dunia U-20

Published at: **2019-11-06T21:03:00+00:00**

Author: **Bola.net**

Original: [Bola.net](https://www.bola.net/indonesia/aji-santoso-nilai-stadion-gbt-layak-jadi-venue-piala-dunia-u-20-9043bf.html)

Bola.net - Pelatih Persebaya Surabaya, Aji Santoso menilai Stadion Gelora Bung Tomo (GBT), Surabaya layak menjadi venue Piala Dunia U-20 2021. Apalagi, jika ada pembenahan di beberapa sisi.
Hal itu diungkapkan Aji ketika memimpin latihan Persebaya untuk pertama kalinya di Stadion Gelora Bung Tomo, Surabaya, Rabu (6/11/2019) sore.
"Kalau saya bilang stadion ini cukup bagus, kalau memang ada sedikit perubahan untuk piala dunia ini sangat layak," kata Aji Santoso.
"Makanya kita sebagai warga Surabaya termasuk saya warga Surabaya, mari kita menjaga sama-sama stadion kebanggaan warga Surabaya," imbaunya.
Stadion Gelora Bung Tomo memang menjadi satu dari sepuluh stadion yang disiapkan untuk Piala Dunia U-20. Rencananya akan dilakukan pemugaran di stadion milik Pemkot Surabaya itu.
