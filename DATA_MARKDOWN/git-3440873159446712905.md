
# Как повысятся тарифы на электроэнергию

Published at: **2019-11-01T07:29:00+00:00**

Author: **Рабига Дюсенгулова**

Original: [Tengrinews.kz](https://tengrinews.kz/kazakhstan_news/kak-povyisyatsya-tarifyi-na-elektroenergiyu-383044/)

Председатель Комитета по регулированию естественных монополий МНЭ РК Асан Дарбаев рассказал о повышении тарифов на электроэнергию, передает корреспондент Tengrinews.kz. 
"В части энергетики - там Министерство энергетики... Среднее повышение тарифов по станциям составило где-то 15 процентов. Из 44 станций по 32 станциям, данное повышение незначительно отразится. Повышение будет в районе 6-7 процентов, это максимум", - расссказал он.
По его словам, в конечном счете для потребителя будет повышение в пределах 6-7 процентов.
"Частично это будет в этом году подниматься, часть будет с 1 января 2020 года. Большая часть субъектов по завершении определения тарифов будет обращаться в следующем году. На тот момент будем анализировать и смотреть обоснованность тарифов каждого субъекта. Новая тарифная ставка будет с 2021 года", - заключил Асан Дарбаев.
