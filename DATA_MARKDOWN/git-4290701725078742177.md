
# Misiones bajo alerta por tormentas fuertes hacia la noche de este lunes

Published at: **2019-11-04T17:10:00+00:00**

Author: ****

Original: [El Territorio Misiones](https://www.elterritorio.com.ar/misiones-bajo-alerta-por-tormentas-fuertes-hacia-la-noche-de-este-lunes-49012-et)

Alerta meteorológico vigente por tormentas fuertes para el Este de Chaco, Corrientes, Norte de Entre Ríos, Este de Formosa, Sur de Misiones y Norte de Santa Fe, emitido por el Servicio Meteorológico Nacional (SMN), a las 16.En el norte de Santa Fe, sur de Corrientes y norte de Entre Ríos continúan registrándose áreas de tormentas de variada intensidad. Se prevé que las mismas se generalicen durante el resto del día de este lunes 4, en la zona de cobertura, pudiendo ser algunas localmente fuertes, estar acompañadas de caída de granizo, intensa actividad eléctrica, ráfagas y principalmente abundante caída de agua en cortos períodos. Las condiciones se mantendrían durante gran parte de este martes para la provincia con tormentas hasta la tarde inclusive.
