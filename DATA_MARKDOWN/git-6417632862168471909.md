
# ЕЧЛ. 3та клуб 1/8 финал масаласини муддатидан олдин ҳал қилди

Published at: **2019-11-07T09:02:06+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/echl-3ta-klub-18-final-masalasini-muddatidan-oldin-hal-qildi)

"Бавария" ва ПСЖ ЕЧЛнинг жорий мавсумида ҳозирча йўқотишларсиз одимлашмоқда.
А Сериянинг амалдаги чемпиони "Ювентус" эса 4та ўйиннинг 3тасида ғалаба, 1тасида дуранг қайд этди.
Шундай қилиб, юқоридаги 3 жамоа ЕЧЛнинг кейинги босқичига муддатидан олдин йўлланма олишди.

        
          Европа Чемпионлар Лигаси 4-турдан сўнг гуруҳлардаги вазият
