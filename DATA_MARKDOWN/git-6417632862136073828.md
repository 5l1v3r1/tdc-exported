
# Бугун Ўзбекистон миллий терма жамоаси йиғинни бошлайди

Published at: **2019-11-02T03:00:00+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/bugun-ozbekiston-milliy-terma-jamoasi-yiginni-boshlaydi)

Бош мураббий Вадим Абрамов ва унинг штаби ушбу йиғинга жалб қилинган футболчилардан иборат кенгайтирилган таркибни эълон қилган. Унда 32 футболчи номи қайд этилган бўлиб, йиғин давомида ва Қирғизистонга қарши ўйиндан сўнг Саудия Арабистони терма жамоаси билан учрашувда иштирок этадиган 23 футболчи саралаб олинади.
