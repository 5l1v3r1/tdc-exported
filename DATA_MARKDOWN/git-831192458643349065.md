
# Μερομήνια: Δείτε τι καιρό θα έχουμε μέχρι τα Χριστούγεννα – Πότε έρχονται τα πρώτα χιόνια?…

Published at: **2019-11-01T12:45:00+00:00**

Author: ****

Original: [europost.gr](https://europost.gr/merominia-deite-ti-kairo-tha-echoyme-mechri-ta-christoygenna-pote-erchontai-ta-prota-chionia/)

Μπορεί ο Οκτώβριος να μας χάρισε καλοκαιρινές ημέρες, ωστόσο αυτό, σύμφωνα με τα Μερομήνια, δεν θα έχει συνέχεια στους μήνες που ακολουθούν μέχρι τα Χριστούγεννα.
Αναλυτικά, τα Μερομήνια προβλέπουν :
Για τον ΝοέμβριοΤο φθινόπωρο έχει μπει για τα καλά, με τις βροχές να συνεχίζονται και τα χαρακτηριστικά του καιρού να θυμίζουν σιγά – σιγά χειμώνα. Στο τέλος των πρώτων είκοσι ημερών θα έχουμε βοριάδες και χιονόνερο στην επικράτεια, αλλά και χιονοπτώσεις στα ορεινά της χώρας. Πάντως, η μετάβαση μεταξύ Νοέμβρη και Δεκέμβρη θα γίνει με πολύ καλές συνθήκες για την εποχή.
Για τον ΔεκέμβριοΑρκετοί βοριάδες και κάθετη πτώση της θερμοκρασίας, προβλέπονται από τα μέσα του μήνα και στη συνέχεια. Τα χιόνια θα κάνουν την εμφάνισή τους, έως τα Χριστούγεννα. Ωστόσο, στην τελευταία εβδομάδα του μήνα και του έτους, ο καιρός θα «γλυκάνει» κάπως.
Σύμφωνα με τα μερομήνια, μέσα σε σχετικά υποφερτές συνθήκες θα καλωσορίσουμε το νέο έτος. Θα υπάρξουν βροχές και κρύο, αλλά θα ακολουθήσουν οι πολυπόθητες αλκυονίδες και θα βάλουν τα πράγματα στη θέση τους.
