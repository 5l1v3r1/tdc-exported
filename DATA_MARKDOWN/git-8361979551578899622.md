
# В Москве убили руководителя центра противодействия экстремизму в Ингушетии

Published at: **2019-11-02T20:57:00+00:00**

Author: ****

Original: [НВ](https://nv.ua/world/countries/v-moskve-ubili-rukovoditelya-centra-protivodeystviya-ekstremizmu-v-ingushetii-novosti-rossii-50051301.html)

Об этом сообщает ТАСС, ссылаясь на источник в экстренных службах.
«На улице Академика Анохина застрелен глава центра противодействия экстремизму МВД РФ по Ингушетии Ибрагим Эльджаркиев», — сказал собеседник агентства.
По другим сообщениям, были ранены два человека (вероятно брат Ельджаркиева — ред.), оба они мертвы.
В января 2019 года на Эльджаркиева уже было совершено покушение. Когда он на автомобиле возвращался из Чечни в Ингушетию, его машину обстреляли из автоматов. В результате тяжелые ранения получили трое полицейских, которые ехали в машине вместе с Эльджаркиевим, сам он не пострадал.
