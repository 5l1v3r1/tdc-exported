
# BHIC委任Mohd Fakhrul为财务总监

Published at: **2019-11-04T06:58:56+00:00**

Author: **Justin Lim**

Original: [The Edge Markets](https://www.theedgemarkets.com/article/bhic%E5%A7%94%E4%BB%BBmohd-fakhrul%E4%B8%BA%E8%B4%A2%E5%8A%A1%E6%80%BB%E7%9B%91)

（吉隆坡4日讯）Boustead Heavy Industries Corp Bhd（BHIC）宣布，委任Mohd Fakhrul Arifin Adinan为财务总监。
该集团今日向大马交易所报备，这项任命于11月1日生效。
现年35岁的Mohd Fakhrul自2014年4月加入该集团。此前，他于2005年在安永会计事务所展开他的职业生涯。
截至2时40分，BHIC起4仙或3.17%，挂1.30令吉，市值为3亿2300万令吉。
