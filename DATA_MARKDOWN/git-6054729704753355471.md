
# В Омске президентский пресс-секретарь Песков отказался комментировать решение Верховного суда об отмене срока давности по налоговым преступлениям

Published at: **2019-11-07T19:13:00+00:00**

Author: ****

Original: [БК55](https://bk55.ru/news/article/159711/)


        Это решение судей может серьезно улучшить инвестиционный климат в стране.
      
Сегодня в Омске пресс-секретарь президента РФ Дмитрий Песков заявил СМИ, что создание благоприятного инвестиционного климата является одним из приоритетов для экономического развития России.
Отвечая на вопрос об отношении Кремля к решению Верховного суда РФ не включать в постановление пленума формулировку, которая могла бы отменить срок давности по налоговым преступлениям, Песков заметил, что не намерен комментировать решение суда.
