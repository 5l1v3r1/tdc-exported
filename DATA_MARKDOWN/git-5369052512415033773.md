
# Где на этой неделе пройдут продуктовые ярмарки. Адреса

Published at: **2019-11-04T18:20:00+00:00**

Author: **Александра Шмарова**

Original: [kiev.vgorode.ua](https://kiev.vgorode.ua/news/sobytyia/412464-ne-zabud-skupytsia-5-noiabria-v-kyeve-proidut-sezonnye-yarmarky)
