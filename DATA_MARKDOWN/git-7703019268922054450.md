
# Jueves soleado y con máxima de 33º: así estará el tiempo hoy

Published at: **2019-11-07T00:00:00+00:00**

Author: **Santiago**

Original: [El Liberal](https://www.elliberal.com.ar/noticia/510020/jueves-soleado-maxima-33-asi-estara-tiempo-hoy)

La "Madre de Ciudades" se prepara para retornar a los días de calor intenso: para hoy se espera una jornada soleada y una máxima de 33º.
Para hoy se espera un día despejado por la mañana y nublado al mediodía, acompañado por vientos leves del sector sureste. Mientras tanto, para mañana se anticipa un día despejado con 35º de máxima y vientos del este durante todo el día.
El sábado podrían registrarse algunas lluvias.
