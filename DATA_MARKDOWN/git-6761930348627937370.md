
# Крымчане стали меньше пить – Крымский.Пармезан

Author: ****

Original: [Крым.Реалии](https://ru.krymr.com/a/anons-radio-krynchane-stali-menshe-pit/30249741.html)

Сколько туристов, по версии крымских властей, уже успели побывать на полуострове с начала года? Почему жители Крыма стали употреблять меньше алкоголя? Какое вино больше всего любит российский глава Крыма Сергей Аксенов?

Время эфира в Крыму 18:00 (17:00 по киевскому времени) в эфире Радио Крым.Реалии (105.9 FM), а также на сайте Крым.Реалии. Также программу можно слушать и на волнах украинского Радио НВ. Каждую субботу в 17:00 по киевскому времени.
