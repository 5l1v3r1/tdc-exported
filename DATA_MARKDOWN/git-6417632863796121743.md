
# “Барселона” салкам бир йил деганда ўз майдонида гол ура олмади

Published at: **2019-11-06T03:55:16+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/barselona-salkam-bir-yil-deganda-oz-maydonida-gol-ura-olmadi)

Каталонияликлар 2018 йил февралидан бери илк бор “Камп Ноу”да ўтган расмий ўйинда гол ура олмади. Сўнгги бор “Хетафе” “Камп Ноу”да дуранг қайд этганди.

        Мавзуга доир: 
        
          “Барселона” бундай натижани ЕЧЛда 3-марта қайд этди
