
# В Богородицком районе Scania «впечатал» ВАЗ-2017 в ограждение

Published at: **2019-11-07T11:59:00+00:00**

Author: ****

Original: [Тульская Служба Новостей](https://www.tsn24.ru/v-bogorodiczkom-rajone-scania-vpechatal-vaz-2017-v-ograzhdenie.html)

Дорожно-транспортное происшествие на 250-м км автомагистрали М-4 «Дон» Богородицкого района произошло утром 6 ноября.
В 08:15 57-летний водитель автомобиля Scania допустил столкновение с ВАЗ-2107 под управлением 37-летнего водителя, который наехал на металлическое барьерное ограждение.
По данным группы по пропаганде УГИБДД УМВД России по Тульской области, пострадавший водитель автомобиля ВАЗ-2107 после оказания медицинской помощи был отпущен.
