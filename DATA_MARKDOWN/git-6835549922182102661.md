
# Агент Щербака: Никита остается в КХЛ, спрос на него здесь есть

Published at: **2019-11-04T15:39:00+00:00**

Author: ****

Original: [Allhockey.ru](https://allhockey.ru/news/show/369005-Agent_Scherbaka_Nikita_ostaetsya_v_KHL_spros_na_nego_zdes_est)

Сергей Исаков, агент нападающего Никиты Щербака, рассказал о ситуации его клиента.  Ранее "Авангард" расторг контракт с Щербаком по обоюдному согласию."У нас есть несколько вариантов, мы склоняемся к одному из них. В ближайшие дни все должно разрешиться. Никита остается в КХЛ, спрос на него здесь есть", – сказал Исаков.
