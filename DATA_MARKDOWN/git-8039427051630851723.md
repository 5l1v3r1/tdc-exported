
# వరదలు తగ్గగానే ఇసుక కొరత తగ్గుతుంది: అనిల్‌

Published at: **2019-11-07T13:41:28+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=948829)

ADVT
కర్నూలు: రాష్ట్రంలో వరదలు తగ్గగానే ఇసుక కొరత తగ్గుతుందని మంత్రి అనిల్‌
కుమార్ యాదవ్ అన్నారు. గురువారం కర్నూలులో ఆయన మీడియాతో మాట్లాడుతూ భవన నిర్మాణ కార్మికులను అడ్డంపెట్టుకుని.. టీడీపీ నేతలు శవరాజకీయాలు చేస్తున్నారని మండిపడ్డారు. చంద్రబాబు పాలనలో ప్రాజెక్టుల్లో జలకల లేదని విమర్శించారు. చంద్రబాబు మళ్ళీ నువ్వే రావాలనే డ్రామా కంపెనీ పెట్టారని ఎద్దేవా చేశారు. జగన్‌ పాలన చూసి చంద్రబాబుకు మతి భ్రమించిందన్నారు. ఒక్క మాట చెబితే చాలు.. టీడీపీ నాయకులు వైసీపీలోకి రావడానికి సిద్ధంగా ఉన్నారన్నారు. రివర్స్ టెండరింగ్‌ డబ్బులు అగ్రిగోల్డ్ బాధితులకు ఉపయోగపడ్డాయని మంత్రి అనిల్‌ అన్నారు.
