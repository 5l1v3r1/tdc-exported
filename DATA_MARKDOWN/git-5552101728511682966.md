
# 謝忻萬聖節深V露半球　扮「胸」狠小丑讓人流鼻血

Published at: **2019-11-01T03:07:00+00:00**

Author: **朱晉緯**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191101web001/)

知名藝人謝忻在前陣子和阿翔的不倫戀風波逐漸平息後，也開始復工並活躍於社群媒體。謝忻31日即在臉書粉專中po出萬聖節相關貼文，不僅臉上畫著小丑妝容，還大方露出「半球」，以深V姿態復出。
謝忻在經歷過前陣子與阿翔的風波後，在螢光幕上黯淡了一陣子，但最近於社群媒體依舊活躍，並在臉書上發布「理想中的萬聖節vs.現實中的萬聖節…」，貼出自己2張萬聖節扮裝照，並和小朋友一同扮裝合照。
謝忻一張照片貼出自己穿著深V衣，大膽袒露，「半球」都已經出現，臉上也掛著小丑的妝容。許多網友熱烈回覆「這鬼也太胸狠了吧」「這樣犯規」「頭以下比較嚇人」「流鼻血了」，似乎已經恢復許多人氣。
