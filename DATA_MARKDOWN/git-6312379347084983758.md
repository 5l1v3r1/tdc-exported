
# В Приморском крае двое детей утонули в овраге с водой

Published at: **2019-11-06T11:31:54+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-06/v-primorskom-krae-dvoe-detey-utonuli-v-ovrage-s-vodoy)

В Приморском крае двое детей утонули в овраге с водой. Об этом сообщает РИА "Новости" со ссылкой на региональный СК.
По информации ведомства, тела семилетнего мальчика и трехлетней девочки были обнаружены в овраге с водой в селе Летно-Хвалынское.
По факту случившегося было возбуждено уголовное дело по признакам преступления, предусмотренного частью 3 статьи 109 УК РФ "Причинение смерти по неосторожности".
По предварительным данным, причина смерти детей - утопление.
В настоящее время сотрудники правоохранительных органов выясняют все детали произошедшего.
