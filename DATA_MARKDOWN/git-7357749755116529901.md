
# Hong Kong : ils bravent l’interdiction de manifester masqués

Published at: **2019-11-05T19:12:54+00:00**

Author: ****

Original: [euronews](https://fr.euronews.com/2019/11/05/hong-kong-ils-bravent-l-interdiction-de-manifester-masques)

Les manifestants portant le masque Guy Fawkes, symbole de protestation dans le monde entier, ont défilé dans le centre-ville de Hong Kong mardi pour marquer le premier mois de l'interdiction des masques.
La police n'a pas été vue alors que les manifestants violaient l'interdiction d'utiliser des masques faciaux lors des rassemblements.
