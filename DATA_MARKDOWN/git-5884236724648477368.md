
# Организаторы показали официальный мяч Евро-2020

Published at: **2019-11-07T06:36:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/740596-organizatory-pokazali-oficialnyi-myach-evro-2020)


        Спортивный снаряд получил название Uniforia и был произведен немецкой фирмой Adidas. Мяч будет использоваться на всех матчах турнира.
      
В официальном Twitter Евро-2020 состоялась презентация официального мяча турнира. Об этом сообщает RT.Отметим, что дизайн мяча, который создавала немецкая фирма Adidas. состоит из широких чёрных полос с размытыми краями в виде мазков кисти художника. Спортивный снаряд будет задействован на всех матчах турнира.
