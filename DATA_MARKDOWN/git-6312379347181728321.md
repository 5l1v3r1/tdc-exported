
# Известная футбольная специалистка стала жертвой мошенников

Published at: **2019-11-05T15:29:08+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-05/izvestnaya-futbolnaya-specialistka-stala-zhertvoy-moshennikov)

Двукратная чемпионка России, обладательница Кубка России по футболу среди женщин Наталья Дорошева стала жертвой мошенников.
По данным РЕН ТВ, женщина, работающая в тренерском штабе казахстанского футбольного клуба "СШВСМ-Барыс", передала более 300 тысяч рублей лицам, обещавшим ей денежную компенсацию за приобретение лекарственного препарата.
Отмечается, что известная футбольная специалистка не получила ни лекарства, ни компенсации.
По факту произошедшего сотрудники правоохранительных органов Москвы возбудили уголовное дело по статье "Мошенничество".
