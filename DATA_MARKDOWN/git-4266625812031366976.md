
# Абромавичус: Ценообразование для оборонных предприятий будет пересмотрено

Published at: **2019-11-06T17:00:00+00:00**

Author: ****

Original: [ZIK.UA](https://zik.ua/ru/news/2019/11/06/abromavychus_tsenoobrazovanye_dlya_oboronnih_predpryyatyy_budet_peresmotreno_1685143)


        Гендиректор ГК «Укроборонпром» Айварас Абромавичус сказал, что ценообразование для оборонных предприятий будет пересмотрено.
      
Об этом Айварас Абромавичус сказал во время рабочей поездки Владимира Зеленского на Харьковщину, – информирует телеканал ZIK.
