
# Видео: в Жуковском горит склад на площади 1,75 тыс. квадратов

Published at: **2019-11-03T04:39:13+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-03/video-v-zhukovskom-gorit-sklad-na-ploshchadi-175-tys-kvadratov)

РЕН ТВ публикует кадры с места крупного пожара на подмосковном складе в Жуковском. Площадь возгорания составляет 1750 квадратных метров.
В настоящее время пожар локализован. По предварительным данным, никто не пострадал.
На горящем складе произошло обрушение кровли. Возгоранию был присвоен повышенный ранг сложности.
С огнем борются 59 человек и 16 единиц техники.
По неподтвержденной информации, склад используется для хранения посылок с AliExpress.
