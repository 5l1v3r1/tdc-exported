
# Τράπεζα Πειραιώς: Ποιες χρεώσεις καταργούνται

Published at: **2019-11-01T16:41:00+00:00**

Author: ****

Original: [enikos.gr](http://www.enikos.gr/economy/681667/trapeza-peiraios-poies-chreoseis-katargountai)

Η Τράπεζα Πειραιώς ανακοίνωσε ότι οι πελάτες της δεν θα έχουν οποιαδήποτε επιβάρυνση κατά την ανάληψη μετρητών μέσω ΑΤΜ τρίτων τραπεζών στα νησιά, όπου η ίδια δεν διατηρεί δικό της ΑΤΜ.
Σε ανακοίνωσή της, η Τράπεζα Πειραιώς αναφέρει ότι, επιπλέον, μηδενίζει το κόστος σειράς εργασιών μέσω της ΔΙΑΣ σε ΑΤΜ τρίτων τραπεζών, όπως:
• Ερώτηση Υπολοίπου
• Αλλαγή Προσωπικού Κωδικού Ασφαλείας (PIN)
• Απεμπλοκή Προσωπικού Κωδικού Ασφαλείας (PIN)
Εκτός από τις παραπάνω ρυθμίσεις, η Τράπεζα επανεξετάζει την τιμολογιακή της πολιτική και σε άλλες προσφερόμενες υπηρεσίες.
Τόσο οι αναλήψεις όσο και οι παραπάνω εργασίες παρέχονται ανέξοδα για τους πελάτες της Τράπεζας Πειραιώς στο δίκτυό της. Ταυτόχρονα, η Τράπεζα συνεχίζει να επενδύει για ακόμα καλύτερες υπηρεσίες και προϊόντα για τους πελάτες της, αναφέρει η ανακοίνωση.
