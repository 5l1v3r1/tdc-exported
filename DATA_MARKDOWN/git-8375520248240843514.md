
# В сети появился полноценный трейлер сериала «Ведьмак» с Генри Кавиллом

Published at: **2019-11-01T08:10:37+00:00**

Author: ****

Original: [ГЛАВКОМ](https://glavcom.ua/scotch/showbiz/v-seti-poyavilsya-polnocennyy-treyler-seriala-vedmak-s-genri-kavillom-636905.html)

Netflix выпустил полноценный трейлер сериала «Ведьмак», премьера которого состоится 20 декабря. Это экранизация серии книг польского писателя Анджея Сапковского. Первый сезон сериала основан на событиях первого тома цикла «Последнее желание».
