
# 'ผู้กำกับเรียลิตี้'โต้ลือกิ๊ก'มัยร่าBNK48' ขอเอาผิดคนกุข่าว!

Published at: **2019-11-03T06:49:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/entertainment/739791)


        "สุชาติ แสงชู"ชี้แจงแล้ว หลังถูกแฉหลุดภาพถ่ายเดินควงคู่ "มัยร่าBNK48" เที่ยวงานหนังสือ ทั้งที่มีภรรยาอยู่แล้ว เจ้าตัวยันไปงานคนเดียว พร้อมเข้าแจ้งความเอาผิดคนกุข่าวแล้ว!
      
กลายเป็นประเด็นวิพากษ์วิจารณ์อย่างกว้างขวางกันอีกครั้ง หลังจากที่ก่อนหน้านี้มีเพจแอนตี้ปล่อยข่าวลือหาว่าหนึ่งในสมาชิกไอดอลเกิร์ลกรุ๊ป BNK48 รุ่นที่ 2 อย่างสาว "มัยร่า-มะอิระ คูยามา" มีสัมพันธ์ลึกซึ้งกับ "สุชาติ แสงชู" ผู้กำกับรายการเรียลิตี้ของวง แต่ล่าสุดกลับมีการเผยภาพคล้ายทั้งคู่กำลังเที่ยวงานหนังสือ จนกลายเป็นที่วิจารณ์อย่างุรนแรงในสังคมออนไลน์อยู่ในขณะนี้ตามที่ได้นำเสนอไปแล้วนั้นโดยล่าสุดเมื่อวันที่ 3 พ.ย. ผู้สื่อข่าว "เดลินิวส์ออนไลน์" รายงานว่าโลกออนไลน์ต่างเกิดกระแสแชร์วิจารณ์อีกครั้ง เมื่อ "สุชาติ แสงชู" ได้ออกมาชี้แจงเรื่องดังกล่าวผ่านเฟซบุ๊กส่วนตัวว่า "ผมได้ดำเนินการแจ้งความลงบันทึกประจำวันไปเมื่อวันจันทร์แล้วครับ และผมขอแสดงตัวว่า งานที่ผมเดินทางไปสถานที่แห่งนั้น ผมเดินทางไปคนเดียว ไม่ได้เป็นไปตามข้อเท็จจริงที่เกิดขึ้น"...
ขอบคุณภาพและข้อมูลจาก @Suchart Sangchuข่าวที่เกี่ยวข้อง..-หลุดอีกภาพคล้าย'มัยร่าBNK48' แอบควงหนุ่มมีเมียแล้ว-'จ๊อบซัง'วอนเสพข่าวมีสติ ขอสอบ'มัยร่า'ปมภาพหลุด!
