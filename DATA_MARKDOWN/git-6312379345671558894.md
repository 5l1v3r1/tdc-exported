
# ТЦ загорелся в Петушках

Published at: **2019-11-01T09:40:04+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-01/tc-zagorelsya-v-petushkah)

Торговый центре "Дом Экватор" загорелся в городе Петушки Владимироской области. По данным РЕН ТВ, площадь пожара составляет 900 квадратных метров. 
Возгоранию присвоен второй повышенный номер сложности. 
Информации о возможных пострадавших пока нет.
