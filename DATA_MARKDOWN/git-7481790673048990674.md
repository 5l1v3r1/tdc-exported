
# ラグビー？イチロー？…流行語大賞、候補３０語発表

Published at: **2019-11-06T13:15:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191106-OYT1T50321/)

　今年話題になった言葉を選ぶ「２０１９ユーキャン新語・流行語大賞」（「現代用語の基礎知識」選）の候補３０語が６日、発表された。大きな盛り上がりを見せたラグビー・ワールドカップや、米大リーグ・マリナーズ元外野手・イチローさんの引退記者会見での発言などスポーツ関係の言葉が目立った。年間大賞とトップ１０は１２月２日に発表される。
