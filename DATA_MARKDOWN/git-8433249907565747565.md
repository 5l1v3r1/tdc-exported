
# WhatsApp wirdbequemer und sicherer

Published at: **2019-11-01T10:16:35+00:00**

Author: ****

Original: [Bild](https://www.bild.de/digital/smartphone-und-tablet/handy-und-telefon/neue-version-fuer-android-whatsapp-wird-bequemer-und-sicherer-65749554.bild.html)


        WhatsApp macht seine App sicherer und komfortabler …
      

        Eine aktualisierte Variante von WhatsApp für Android-Geräte bringt mehrere neue Funktionen. Mit der neuen Version 2.19.308 wird es unter anderem möglich, mehrere Sprachnachrichten hintereinander abzuspielen.
      
Hat man in einem Chat gleich eine Reihe von Sprachnachrichten erhalten, genügt es nun, nur die Wiedergabe der ersten zu starten. Die folgenden Mitteilungen werden automatisch danach abgespielt.

        Auch neu ist eine zusätzliche Absicherung der Chats in WhatsApp: Sie können jetzt mit einer Fingerabdruck-Sperre vor unbefugtem Zugriff geschützt werden. In den Einstellungen unter Account → Datenschutz → Fingerabdruck-Sperre findet sich dafür der Punkt „Entsperren mit Fingerabdruck“.
      
Aktiviert man die Funktion, muss vor dem Öffnen eines Chats zunächst der Finger auf den Fingerabdrucksensor gehalten werden. So werden heimliche Mitleser gestoppt, die womöglich das Smartphone in die Hand bekommen.

        Bislang ist es nicht möglich, die Gesichtserkennung zum Freigeben der Chats auszuwählen. Auf dem iPhone geht das bereits seit Längerem.
