
# Названы пять простых способов избавиться от жира без спорта

Published at: **2019-11-04T08:08:00+00:00**

Author: ****

Original: [БНК](https://www.bnkomi.ru/data/news/102271/)

Диетологи дали советы по похудению людям, которые по каким-либо причинам не могут делать физические упражнения.
Рекомендации опубликовал портал Food.news.
Для того чтобы похудеть, необходимо увеличить в рационе количество богатых белком продуктов, таких как мясо, птица, рыба. В качестве альтернативы эксперты назвали тофу, фасоль, горох, нут, орехи, семена чиа, тыквенные семечки и несоленый арахис.
Также, по словам специалистов, следует выработать «прерывистый» режим питания, при котором есть разрешено лишь в определенное время. Например, человек в течение восьми часов потребляет суточную норму калорий, а оставшиеся 16 часов обходится без пищи. Альтернативный вариант — три дня в неделю есть 25 процентов от суточной потребности в калориях, а в остальные дни питаться в обычном режиме, но без переедания.
Специалисты призвали желающих похудеть исключить из рациона сладкие и подслащенные напитки, включая энергетики.
Кроме того, эксперты рекомендуют избегать стрессов, поскольку они пробуждают аппетит. Еще одним правилом для эффективного похудения диетологи назвали хороший регулярный сон, пишет Lenta.ru.
