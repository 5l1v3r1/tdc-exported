
# AMD lanza sus Radeon Adrenalin 19.11.1 para que disfrutes del Red Dead Redemption II

Published at: **2019-11-04T00:00:00+00:00**

Author: **Borja Rodríguez**

Original: [El Chapuzas Informático](https://elchapuzasinformatico.com/2019/11/amd-radeon-adrenalin-19-11-1-red-dead-redemption-ii/)

Hoy parece que es el 'Día D' de los controladores gráficos, y no es para menos, y es que los AMD Radeon Adrenalin 19.11.1 llegan para ofrecer plena compatibilidad al Red Dead Redemption II, título que llegará mañana mismo PC en diferentes plataformas.

        Adicionalmente, se amplía el soporte de la API Vulkan con 7 nuevas extensiones:
      
Para acabar, solventan problemas con The Outer Worlds, donde algunos usuarios estaban experimentando el cierre o congelación del juego cuando abren el inventario del personaje y un mal renderizado de los modelos al hacerlo; el limite de 60 FPS en juegos basados en API Vulkan; la gran caída de rendimiento al realizar streaming con OBS con la codificación AMF; y problemas al conectarse en Twitch para realizar streaming desde la aplicación Radeon.
