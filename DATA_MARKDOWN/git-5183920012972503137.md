
# В Днепре презентовали проект сверхзвуковой противокорабельной ракеты Молния

Published at: **2019-11-03T00:00:00+00:00**

Author: ****

Original: [Контракты.UA](http://kontrakty.ua/article/145360)

В Днепре презентовали проект сверхзвуковой противокорабельной ракеты "Молния"
Ракетами этого типа планируют вооружить авиацию украинской армии.
В Днепре на выставке “Оружие и безопасность-2019“ представили макет сверхзвуковой противокорабельной ракеты “Молния“. Ее скорость составит порядка 2 - 2,5 тысячи километров в час. Об этом сообщает “Днепровская Панорама“.
Ракетами этого типа планируют вооружить авиацию украинской армии, для этого потребуется минимальное переоборудование самолетов.
По мнению военного эксперта Олега Жданова “Молния“ может применяться Украиной для противодействия РФ в Азовском и Черном морях.
