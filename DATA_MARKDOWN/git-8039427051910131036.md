
# తహసీల్దార్‌ విజయారెడ్డి మృతదేహానికి పోస్టుమార్టం పూర్తి

హైదరాబాద్: ఉస్మానియాలో తహసీల్దార్‌ విజయారెడ్డి మృతదేహానికి పోస్టుమార్టం పూర్తయింది. అనంతరం ఎల్బీనగర్‌లోని ఆమె నివాసానికి మృతదేహాన్ని తరలించారు. రేపు మునుగోడు మండలం కలవలపల్లిలో విజయారెడ్డి అంత్యక్రియలు జరగనున్నాయి.
