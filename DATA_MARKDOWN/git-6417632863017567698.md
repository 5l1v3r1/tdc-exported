
# Лига кубоги Фабрегас учун қимматга тушди

Published at: **2019-11-01T15:54:00+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/liga-kubogi-fabregas-uchun-qimmatga-tushdi)

32 ёшли испаниялик орқа бел қисмидан жароҳат олган ва тикланиш муддати очиқланмаган. Жорий мавсум Фабрегас Франция чемпионатининг 8 ўийнида майдонга тушди ва иккита голга ҳаммуаллифлик қилди.
