
# ഡി.കെ ശിവകുമാര്‍ ആശുപത്രിയില്‍

Published at: **2019-11-02T08:00:00+00:00**

Author: **ന്യൂസ് ഡെസ്‌ക്**

Original: [DoolNews](https://www.doolnews.com/dk-shivkumar-at-appolo-hospital-bengaluru.html)

ബംഗളൂരു: കര്‍ണാടക മുന്‍ മന്ത്രിയും പ്രമുഖ കോണ്‍ഗ്രസ് നേതാവുമായ ഡി.കെ ശിവകുമാര്‍ ആശുപത്രിയില്‍. ഇന്നലെ വൈകീട്ട് 6.30നാണ് അപ്പോളോ ആശുപത്രിയില്‍ ശിവകുമാര്‍ ചികിത്സ തേടിയത്.
നടുവേദനയും കടുത്ത ക്ഷീണവും കാരണമാണ് ശിവകുമാര്‍ ചികിത്സ തേടിയത്. ശിവകുമാറിനെ പരിശോധിച്ച ഡോക്ടര്‍മാര്‍ അഡ്മിറ്റാവാന്‍ നിര്‍ദേശിക്കുകയായിരുന്നു.

        
          വാര്‍ത്തകള്‍ ടെലഗ്രാമില്‍ ലഭിക്കാന്‍ ഇവിടെ ക്ലിക്ക് ചെയ്യൂ
        
      
ദല്‍ഹിയില്‍ എന്‍ഫോഴ്‌സ്‌മെന്റ് ഡയറക്ടറേറ്റിന്റെ കസ്റ്റഡിയിലായിരിക്കേ നെഞ്ച് വേദനെയെ തുടര്‍ന്ന് രാം മനോഹര്‍ ലോഹ്യ ആശുപത്രിയില്‍ പ്രവേശിപ്പിച്ചിരുന്നു.
രാജ്യോത്സവ പരിപാടിയില്‍ പങ്കെടുത്തതിന് ശേഷമാണ് ശിവകുമാര്‍ അപ്പോളോ ആശുപത്രിയില്‍ എത്തിയത്. തൈറോയിഡ്, ഹൈപ്പര്‍ ടെന്‍ഷന്‍ എന്നീ രോഗങ്ങളും ശിവകുമാറിനുണ്ട്.

        
          ഡൂൾന്യൂസ് യൂട്യൂബ് ചാനൽ സബ്സ്ക്രൈബ് ചെയ്യാനായി ഇവിടെ ക്ലിക്ക് ചെയ്യൂ
