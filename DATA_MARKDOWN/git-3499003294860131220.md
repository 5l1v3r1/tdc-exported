
# В Зеленограде продали только три места в «народных гаражах»

Published at: **2019-11-07T11:49:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/zelao/v_zelenograde_prodali_tolko_tri_mesta_v_narodnykh_garazhakh/)

В Зеленограде с лета 2019 года власти Москвы пытаются продать через электронные аукционы места в многоуровневых паркингах. Но к сегодняшнему дню покупателей нашлось только три. При этом всего мест для машин 1300.
По данным инвестиционного портала Москвы, с июня по сентябрь на продажу было выставлено более 1300 машино-мест в восьми «народных гаражах», расположенных во 2-м, 8-м, 20-м микрорайонах и на Новокрюковской улице напротив 14-го микрорайона. Их предлагали приобрести по цене от 312 тысяч до 1,1 млн рублей.
Почти все аукционы «провалились» из-за отсутствия заявок от потенциальных покупателей. Интерес вызвали лишь три парковочных места стоимостью 588 тысяч, 655 тысяч и 725 тысяч рублей.
