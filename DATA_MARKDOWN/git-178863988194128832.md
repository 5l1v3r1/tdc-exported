
# На запорожской таможне собака унюхала крупную сумму долларов

Published at: **2019-11-06T10:49:00+00:00**

Author: **Ксения Басанец**

Original: [РИА Мелитополь](https://ria-m.tv/news/170743/na_zaporojskoy_tamojne_sobaka_unyuhala_krupnuyu_summu_dollarov.html)


        Спаниель Венера, "работница" и помощница для таможенников в запорожском аэропорту смогла буквально унюхать в чемодане одного из пассажиров незадекларированные деньги. Об этом сообщили в пресс-службе государственной фискальной службы (ГФС) в Запорожской области.
      
""Деньги не пахнут" – это не аргумент для Венеры…  Во время оформления пассажиров, которые прилетели из Турции, один гражданин Украины избрал упрощенный порядок таможенного контроля. Однако после пересечения "зеленого коридора" Венера сработала на его сумку", - рассказали в ГФС.
Открыв чемодан, таможенники обнаружили в нем 13 тысяч 720 долларов США, что превышает допустимую для свободного передвижения сумму  (без декларирования можно перевозить сумму до 10 тысяч евро –ред.)
По факту составлен протокол о нарушении таможенных правил по статье 471 Таможенного кодекса Украины (нарушение порядка прохождения таможенного контроля – ред.). Разрешенная для ввоза сумма возвращена мужчине.
