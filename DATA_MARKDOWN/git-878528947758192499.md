
# 次課長・河本　ラグビーＷ杯決勝を生観戦も無念のリタイア…「仕事ですから」

Published at: **2019-11-02T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/02/0012843462.shtml)

　次長課長の河本準一が２日、公式ブログを更新し、この日、横浜・日産スタジアムで開催されたラグビーＷ杯日本大会決勝、南アフリカ－イングランドを生観戦したものの、途中で退出を余儀なくされたことを明かしている。
　河本はまず「間に合った！！の巻」と題した投稿で「生まれて初めてラグビーの決勝を見れた！！感動しかありません」と、決勝を初めて生観戦していることを報告。
　約１時間後の投稿「仕事だから仕方がありませんよ。タンメンさん」では「ラグビー決勝前半終わって　南アフリカのリード　イングランドの巻き返しあるか？！」とあおりつつ、「あたし、後半始まって少し経ったら次の仕事場に向かわなければ行けません。仕事ですから」と、仕事のため無念のリタイアを余儀なくされることを明かした。
　河本は「家に帰っても１度最初から見ます　なんせ、今回　全試合眠たい目を擦りながら見ましたから　もちろんバレーも世界陸上も見ましたけど。好きなんですスポーツが」と、後ほど録画で見るとしつつ、「次の現場行きましょう」と仕事に向かっている。
