
# Год спустя: Red Dead Redemption 2 добралась до компьютеров

Published at: **2019-11-05T18:43:30+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-05/god-spustya-red-dead-redemption-2-dobralas-do-kompyuterov)

Red Dead Redemption 2, одна из лучших игр 2018 года, наконец-то добралась до пользователей персональных компьютеров.
Популярный вестерн доступен пользователям Epic Game Store (EGS) и сервиса Rockstar с 5 ноября. Однако в Steam игра должна появиться только в декабре.
