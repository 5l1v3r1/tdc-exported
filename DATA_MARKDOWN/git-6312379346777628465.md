
# В Турции за переход границы из Сирии задержаны россиянки с детьми

Published at: **2019-11-01T17:26:17+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-01/v-turcii-za-perehod-granicy-iz-sirii-zaderzhany-rossiyanki-s-detmi)

Две россиянки задержаны в Турции неподалеку от границы с Сирией за незаконное пересечение границы.
По данным телеканала TRT Haber, вместе с женщинами находились трое детей.
Предполагается, что задержанные могут быть связаны с ИГ (террористическая организация, запрещенная в РФ судом), передает "Интерфакс".
Задержанные переданы иммиграционному ведомству в провинции Килис.
Сообщается, что задержанные могли быть в розыске по линии Интерпола.
