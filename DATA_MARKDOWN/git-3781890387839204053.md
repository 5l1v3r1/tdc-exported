
# รมว.พณ.ลุ้นผลปมตัดสิทธิจีเอสพี! ตั้งวอร์รูมภาคธุรกิจถกข้อสรุป

Published at: **2019-11-01T05:18:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/politics/739452)


        “จุรินทร์”รอลุ้นผลหารือ “ทูตพาณิชย์-ทูตไทยวอชิงตัน-ผู้แทนการค้าสหรัฐ” ปมตัดสิทธิจีเอสพี เผยวอร์รูมภาคธุรกิจได้ข้อสรุปบางอย่างแล้ว เตรียมหารือปลัดพาณิชย์ ปัดตอบผลกระทบนำเข้าถั่วเหลืองจากการแบน 3 สารพิษ
      
เมื่อวันที่ 1 พ.ย. นายจุรินทร์ ลักษณวิศิษฏ์ รองนายกรัฐมนตรี และรมว.พาณิชย์ ให้สัมภาษณ์ถึงกรณีมอบหมายให้ทูตพาณิชย์ประจำกรุงวอชิงตัน ดีซี ประสานงานกับเอกอัครราชทูตไทยประจำกรุงวอชิงตัน ดีซี และทูตแรงงานเพื่อจะนัดหารือกับสำนักงานผู้แทนการค้าสหรัฐ (USTR) ที่กรุงวอชิงตัน ดีซี กรณีสหรัฐอเมริกาตัดสิทธิจีเอสพีสินค้าไทย ในวันที่ 1 พ.ย.นี้ ว่า ยังไม่มีรายงานเรื่องดังกล่าว เพราะเวลายังเหลื่อมกันอยู่ ทั้งนี้ขอให้ทูตพาณิชย์และทูตไทยประจำวอชิงตัน ไปหารือกับ USTR ให้เสร็จก่อน แล้วทางนู้นจึงจะมีการรายงานให้ตนทราบ ซึ่งก็ต้องรอการหารือดังกล่าวก่อน ยืนยันว่าไม่มีการปิดเป็นความลับ อย่างไรก็ตามกระทรวงพาณิชย์ก็ไม่ได้อยู่นิ่ง โดยมีการตั้งวอร์รูมสำหรับภาคธุรกิจที่ได้รับผลกระทบจากการตัดสิทธิจีเอสพีครั้งนี้ ให้มีการหารือร่วมกัน ซึ่งปลัดกระทรวงพาณิชย์ได้หารือแล้วเมื่อวาน (31 ต.ค.) ซึ่งได้ข้อสรุปบางอย่างแล้ว และจะมีการนำมาหารือกับตนอีกครั้งหนึ่ง ซึ่งคาดว่าจะมีการจัดประชุมวอร์รูมเพื่อพิจารณาข้อสรุปดังกล่าว ก่อนที่ตนจะเดินทางไปร่วมการจัดงานแสดงสินค้านำเข้านานาชาติ (China International Import Expo) ที่เมืองเซียงไฮ้เมื่อถามว่ามีกระแสข่าวว่าประเทศไทยจะนำเข้าถั่วเหลืองจากประเทศบราซิลไม่ได้ เนื่องจากมีการแบนไกลโฟเซตนั้น นายจุรินทร์ กล่าวว่า ตนยังไม่ทราบเรื่องดังกล่าว ต้องถามกระทรวงสาธารณสุขซึ่งจะเป็นผู้พิจารณาว่าอันไหนนำเข้าได้อันไหนนำเข้าไม่ได้ในเรื่องของสุขอนามัย และอันไหนกระทบสุขอนามัยหรือไม่กระทบ ส่วนจะมีผลกระทบกับภาคอุตสาหกรรมหรือไม่ ตนไม่ขอตอบก่อน.
