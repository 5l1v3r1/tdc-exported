
# WhatsApp запустил функцию разблокировки по отпечатку пальца для Android

Published at: **2019-11-01T17:25:00+00:00**

Author: **Марина Ибушева**

Original: [Лайкни](https://www.likeni.ru/events/whatsapp-zapustil-funktsiyu-blokirovki-po-otpechatku-paltsa-dlya-android/)

В приложении WhatsApp появилась возможность разблокировки с помощью отпечатка пальца, сообщается в блоге мессенджера.  Новая функция позволит обеспечить дополнительный уровень безопасности и сделать пользование мессенджером удобнее.Чтобы включить функцию аутентификации по отпечатку пальца, необходимо нажать Настройки > Аккаунт > Приватность > Блокировка отпечатком пальца. Затем включить опцию Разблокировать с помощью отпечатка и настроить свой отпечаток пальца. 
Функция блокировки при помощи Touch ID и Face ID на iPhone была запущена ранее в этом году.  Напомним, в октябре WhatsApp начал тестировать функцию автоматического удаления сообщений для чатов.  Источник: блог WhatsApp
