
# Аккерманцы нашли противопехотную мину на площадке для собак

Published at: **2019-11-05T12:22:00+00:00**

Author: ****

Original: [ТАЙМЕР](http://timer-odessa.net/news/akkermantsi_nashli_protivopehotnuyu_minu_na_ploschadke_dlya_sobak_407.html)


        Утром 6 ноября сотрудники правоохранительных органов получили сообщение о найденной на ул. Победы в Белгороде-Днестровском противопехотной мине.
      
По информации издания Odessa Online, потенциально опасный боеприпас находился на территории площадки, предназначенной для выгула собак.
«Полиция обеспечивает охрану места происшествия до приезда бригады взрывотехников», — уточняют журналисты.
