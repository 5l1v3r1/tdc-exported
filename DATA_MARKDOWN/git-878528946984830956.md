
# 成田凌 後輩・黒島結菜の姉御肌を絶賛「進んで手を引っ張ってくれた」

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/01/0012838351.shtml)

　俳優の成田凌（２５）が３１日、都内で開催中の「第３２回東京国際映画祭」で行われた初主演映画「カツベン！」（１２月１３日公開）のレッドカーペットセレモニーに出席した。
　「温かく迎えてもらえてうれしいですね！」と集まった４００人のファンの熱気にテンション爆上がり。同じ所属事務所の後輩女優・黒島結菜（２２）と幼なじみ役で共演しており「劇中でリードしてくれて心から演じられた。進んで手を引っ張ってくれたんですよ」と“姉御肌”を絶賛した。
