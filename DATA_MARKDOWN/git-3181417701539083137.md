
# В Тверской области бесследно пропал деревенский житель

Published at: **2019-11-05T09:34:00+00:00**

Author: ****

Original: [Тверские ведомости](https://vedtver.ru/news/proisshestviya/v-tverskoy-oblasti-bessledno-propal-derevenskiy-zhitel/)

Фото: province.ru
Краснохолмский межрайонный следственный отдел СУ СКР по Тверской области проводит доследственную проверку по факту пропажи жителя Весьегонского района.
Известно, что 63-летний житель деревни Барановка 26 октября покинул свой дом, и до настоящего времени о его судьбе ничего не известно.
По месту жительства пенсионера выезжали следователи. Проведен осмотр и опрошены очевидцы. Проводится поиск.
Любую информацию о пропавшем, которая может помочь в поиске, просьба сообщать в Краснохолмский межрайонный следственный отдел СУ СКР по Тверской области по телефону: 8 (48237) 2-29-01, либо по телефону 02.
