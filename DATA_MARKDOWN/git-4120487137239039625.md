
# 4000名跑步者参加Astra半程马拉松

Published at: **2019-11-04T02:38:00+00:00**

Author: ****

Original: [Shangbao Indonesia](http://shangbaoindonesia.com/read/2019/11/03/sports-1572791723)

在庆祝第6周年成立日，Astra Runners社团举办了以#InspiringMOVE为主题的Astra 半程马拉松活动。 Astra邀请跑步爱好者保护环境，即减少使用塑料. 来自印尼各地区的约4000名跑步者参加了此次Astra Half Marathon活动.图示：Astra 董事经理Suparno Djasmin（中）于周日在Astra Biz 中心 BSD City开设了Astra 半程马拉松活动时的情景.
