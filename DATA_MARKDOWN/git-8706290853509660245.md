
# Φωτιά σε υπόγειο πολυκατοικίας στο Παγκράτι - ΤΩΡΑ

Published at: **2019-11-06T11:49:00+00:00**

Author: ****

Original: [enikos.gr](http://www.enikos.gr/society/682615/fotia-se-ypogeio-ktiriou-sto-pagkrati-tora)

Φωτιά ξέσπασε σε ένα υπόγειο πολυκατοικίας, στην οδό Πρατίνου 20, στο Παγκράτι, όπως μεταδίδει ο ΑΝΤ1.
