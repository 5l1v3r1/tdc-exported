
# В Сыктывкаре установят уличный туалет стоимостью полтора миллиона рублей

Published at: **2019-11-01T07:30:00+00:00**

Author: **Мария Майер**

Original: [Комиинформ](https://komiinform.ru/news/188243)

В Сыктывкаре появится круглогодичный туалет.
Администрация столицы Коми разместила госзакупку на поставку туалетного модуля для общественной территории. Стоимость туалета оценивается в 1 млн 558 тыс. рублей.
В документации говорится, что уличный туалет будет предназначен для круглогодичного использования. Павильон будет состоять из двух отделений для посетителей, одно из них с увели­ченными размерами для лиц с ограниченными возможностями. Кабины будут оснащены сантехническим и гигиеническим оборудованием.

        Эскиз туалета
      
Комплектация туалетного модуля: два унитаза, столешница с раковиной, раковина, тепловентилятор,  два конвектора, диспенсеры для бумаги, мыла, полотенец, а также два зеркала, три вытяжных вентилятора, семь светильников, в том числе три уличных, розетки, звонок вызова и проч.
"Павильон конструктивно должен состоять из утепленной рамы, корпуса, ступени, пандуса, крыши, козырька, декоративных угловых зашивок", - указывается в закупке.
