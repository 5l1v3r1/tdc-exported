
# Гончарук: В бюджете хватит средств для «государства в смартфоне»

Published at: **2019-11-06T17:54:00+00:00**

Author: **InfoResist**

Original: [InfoResist](https://inforesist.org/goncharuk-v-byudzhete-hvatit-sredstv-dlya-gosudarstva-v-smartfone/)

Премьер-министр Алексей Гончарук заявил, что в государственном бюджете хватит средств для создания «государства в смартфоне».  Об этом он сказал на брифинге по итогам заседания Кабинета министров.
По словам премьера, средств, заложенных в госбюджете, «хватит на то, чтобы «государство в смартфоне» появилось уже в следующем году». «Я вам больше скажу: отдельные его элементы появятся уже в этом году», — отметил Гончарук.
Также он добавил, что у каждого министерства есть расходы, которые предусматривают информатизацию и цифровизацию.
Ранее вице-премьер-министр, министр цифровой трансформации Михаил Федоров заявил, что проект государственного бюджета на 2020 год не предполагает финансирования проекта «государство в смартфоне».
