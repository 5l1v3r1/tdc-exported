
# Почти даром: MSC выставила круизы по Карибам всего за $149, по Персидскому заливу - за $339, по Средиземноморью - за $359

Published at: **2019-11-07T00:00:00+00:00**

Author: ****

Original: [Турпром](https://www.tourprom.ru/news/43859/?utm_source=pulse_mail_ru&utm_referrer=https%3A%2F%2Fpulse.mail.ru)

Один из крупнейших в мире операторов морских круизов – итальянская компания «MSC Cruises» сегодня, 7 ноября, заканчивает трехдневную мегараспродажу своих круизов. До 19:00 ещё можно успеть забронировать себе круиз мечты по любому из представленных направлений по суперцене: Карибы, Персидский залив или Средиземное море. Все представленные MSC круизы можно найти у партнёров компании этой ссылке или по этой ссылке.
Так, по этой акции можно всего за $149 можно отправиться в 5-звездочный шикарный морской круиз по Карибам на Багамских островах. Подробнее о круизе, маршруте и включённых в него услугах можно посмотреть тут.
А всего за $339 можно совершить 7-дневное шикарное путешествие по Персидскому залив по маршруту: Дубай + Абу-Даби + Хаур-эль-Факкан + Маскат (Оман) + Дубай (ОАЭ). Подробнее о круизе, маршруте и включённых в него услугах можно посмотреть тут.
И наконец, на распродажу выставлен всего за $359 шикарный 7-дневный круиз по Средиземному морю по маршруту: Венеция (Италия) + Бари (Италия) + Олимпия (Греция) + Санторини (Греция) + Афины + Корфу (Греция) + Котор (Черногория) + Венеция. Подробнее о круизе, маршруте и включённых в него услугах можно посмотреть тут.
⛴ Акция распространяется только на новые бронирования и на выборочные круизы по Карибам на шикарных лайнерах MSC Armonia, MSC Divina, MSC Meraviglia, MSC Seaside, по Персидскому заливу из Абу-Даби и Дубая на лайнерах MSC Lirica, MSC Belissima, а также по Средиземному морю на лайнерах MSC Grandiosa, MSC Opera и MSC Magnifica.
Как приобрести круиз с такой потрясающей скидкой? Очень просто: для этого надо по данной ссылке до 19:00 часов 7 ноября выбрать тариф «Распродажа 96 часов». Стоимость каюты в данном тарифе уже указана с учетом скидок по акции. При этом необходима 100% оплата круиза в день
