
# Крупнейшая операция за последние годы: полиция вывезла из Парижа 1600 мигрантов — видео

Published at: **2019-11-07T18:16:00+00:00**

Author: ****

Original: [НВ](https://nv.ua/world/countries/video-migranty-v-parizhe-policiya-provela-operaciyu-novosti-mira-50052264.html)

Об этом сообщает EUobserver.
Данная операция стала одной из крупнейших за последние годы.
Операция началась через день после того, как правительство Франции объявило меры по сдерживанию миграции.
Мигрантов в приемные центры в районе Парижа сопровождали 600 сотрудников полиции.
Автобусы везли людей из района Порт-де-ла-Шапель, где нелегальные беженцы живут в палатках и вокруг эстакад Парижской кольцевой дороги.
