
# У відповідь на запуски ракет. ВПС Ізраїлю обстріляли сектор Газа

Published at: **2019-11-02T02:46:00+00:00**

Author: ****

Original: [НВ](https://nv.ua/ukr/world/countries/vps-izrajilyu-obstrilyali-sektor-gaza-novini-svitu-50051235.html)

Повідомляється, що по території Ізраїлю було випущено десять ракет.
Система ППО перехопила вісім ракет. Водночас дві інші ракети влучили в житлові будинки.
