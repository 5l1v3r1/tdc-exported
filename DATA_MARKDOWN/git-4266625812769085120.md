
# НВФ проводят работы по минированию местности вдоль линии столкновения на Донбассе, – разведка

Published at: **2019-11-06T14:07:00+00:00**

Author: ****

Original: [ZIK.UA](https://zik.ua/ru/news/2019/11/06/nvf_provodyat_raboti_po_mynyrovanyyu_mestnosty_vdol_lynyy_stolknovenyya_na_1684981)


        Незаконные вооруженные формирования на Донбассе активизировали проведение инженерных работ по минированию местности вдоль линии соприкосновения.
      
Об этом сообщает пресс-центр Минобороны.
«Главное управление разведки Министерства обороны Украины сообщает об активизации командованием российско-оккупационных войск мероприятий по инженерно-фортификационному оборудованию передовых позиций соединений и частей 1 (Донецк) и 2 (Луганск) АК ВС РФ», – говорится в сообщении.
По данным разведки, с начала ноября выросла интенсивность проведения врагом инженерных работ, в т.ч. по минированию местности вдоль линии столкновения, включая т.наз. «серую зону».
Также увеличилось количество небоевых потерь личного состава оккупантов. «Так, 1 ноября военнослужащий одного из подразделений 1 АК получил тяжелые ранения вследствие неосторожного обращения с протипехотной миной во время ее установки. Командование противника принимает меры по сокрытию указанного инцидента», - добавили в ведомстве.
Напомним, с начала суток незаконные вооруженные формирования совершили один обстрел на Луганском направлении.
