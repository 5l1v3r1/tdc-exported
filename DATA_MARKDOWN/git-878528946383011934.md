
# 侍のサブマリン・高橋礼が圧巻投球　五回まで完全　１２／１５が内野ゴロ

Published at: **2019-11-06T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/06/0012854093.shtml)

　「プレミア１２・１次ラウンド、日本－プエルトリコ」（６日、桃園国際野球場）
　先発のサブマリン・高橋礼（ソフトバンク）は圧巻の投球。五回までひとりの走者も許さず、１２のアウトを重ねた。
　直球、スライダー、シンカーでタイミングを外し、１５アウトのうち１２個が内野ゴロ。相手打線を翻弄（ほんろう）して外野に飛ばすことすらさせず、リズム良く味方の攻撃につなげた。
