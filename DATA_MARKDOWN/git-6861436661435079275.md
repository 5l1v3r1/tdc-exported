
# நவம்பர் 14: 6.44-இன்ச் டிஸ்பிளேவுடன் களமிறங்கும் விவோ எஸ்5.!

Author: **Saravanan Saravanan**

விவோ நிறுவனம் வரும் நவம்பர் 14-ம் தேதி புதிய விவோ எஸ்5 என்ற ஸ்மார்ட்போன் மாடலை அறிமுகம் செய்ய திட்டமிட்டுள்ளது, தற்சமயம் இந்த ஸ்மார்ட்போனின் பல்வேறு அம்சங்கள் ஆன்லைனில் வெளிவந்துள்ளது, அதைப் பற்றிய முழுமையான தகவல்களைப் பார்ப்போம்.
விவோ எஸ்5 ஸ்மார்ட்போன் ஆனது 6.44-இன்ச் முழு எச்டி பிளஸ் டிஸ்பிளே வடிவமைப்பைக் கொண்டுள்ளது, பின்பு 2400 x 1080 பிக்சல் திர்மானம் மற்றும் சிறந்த பாதுகாப்பு வசதியின் அடிப்படையில் வெளிவரும் எனத் தகவல் தெரிவிக்கப்பட்டுள்ளது.
இந்த ஸ்மார்ட்போனில் குவால்காம் ஸ்னாப்டிராகன் 712எஸ்ஒசி சிப்செட் வசதி இடம்பெற்றுள்ளது. இதனுடன அட்ரினோ
616ஜிபியு வசதியும் இவற்றில் உள்ளது என்பது குறிப்பிடத்தக்கது. மேலும் ஆண்ட்ராய்டு 9பை இயங்குதளத்தை அடிப்படையாக கொண்டுள்ளது இந்த சாதனம்.
வாயுக்கோள்களை கடந்து சென்று வாயேஜர்-2 விண்கலம் சாதனை: மகிழ்ச்சியில் நாசா.!
இந்த ஸ்மார்ட்போனின் கேமரா அமைப்பு சற்று வித்தியசமாக உள்ளது என்றே கூறலாம், அதன்படி விவோ எஸ்5 சாதனத்தில் 48எம்பி பிரைமரி சென்சார் + 8எம்பி வைடு ஆங்கிள் லென்ஸ் + 5எம்பி டெப்த் சென்சார் என மூன்று கேமராக்கள் இடம்பெற்றுள்ளது. மேலும் 32எம்பி செல்பீ கேமரா, எல்இடி பிளாஸ் உள்ளிட்ட ஆதரவுகளும் இவற்றில் உள்ளது.
விவோ எஸ்5 ஸ்மார்ட்போனில் 8ஜிபி ரேம் மற்றும் 128ஜிபி/256ஜிபி உள்ளடக்க மெமரி வசதி உள்ளது என்றும், பின்பு கூடுதலாக மெமரி நீட்டிப்பு ஆதரவு கொண்டு இந்த சாதனம் வெளிவரும் எனவும் தெரிவிக்கப்பட்டுள்ளது.
வோடபோன் அறிமுகம் செய்த மலிவு விலை திட்டங்கள்: கடுப்பில் ஜியோ.!
