
# В Архангельске откроется бесплатная пятидневная «Школа предпринимательства»

Published at: **2019-11-01T17:16:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Arkhangelsk/738822-v-arkhangelske-otkroetsya-besplatnaya-pyatidnevnaya-shkola-predprinimatelstva)


        Тренинги для архангельских бизнесменов подготовила федеральная Корпорация МСП. У архангелогородцев будет возможность индивидуально проконсультироваться с экспертами. 
      
В Архангельске предприниматели смогут повысить свои знания на «Школе предпринимательства». На тренинги приглашают действующий бизнес, сообщает сайт правительства области. 
Программу тренингов разработала федеральная Корпорация МСП. Учеба рассчитана на четыре дня, обучение будет идти с 25 по 28 ноября. На пятый день, 29 ноября участников ждут индивидуальные консультации с экспертами.
Целевая аудитория обучения - действующие предприниматели, у который уже есть свое дело. «Школа предпринимательства» интересна будет тем, кто хочет развивать или изменить формат своего бизнеса. Участие бесплатное. 
Корпорация МСП подготовит тренинги по проведению маркетинговых исследований и анализа бизнеса. Архангелогородцы научатся управлять товарно-материальными запасами, получат основы финансового планирования и управления предприятием. 
Мероприятие проходит в рамках национального проекта «Малое и среднее предпринимательство и поддержка индивидуальной предпринимательской инициативы».
