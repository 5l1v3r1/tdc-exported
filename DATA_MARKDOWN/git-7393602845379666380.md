
# Старт сыктывкарского Строителя в Суперлиге сезона 2019/20 откладывается

Published at: **2019-11-05T18:35:00+00:00**

Author: **Комиинформ**

Original: [Комиинформ](https://komiinform.ru/news/188436)

Первый матч "Строителя" в этом сезоне Чемпионата России по хоккею с мячом отменен. Сыктывкарцы едут на игру второго тура в Нижний Новгород, сообщает пресс-служба хоккейного клуба.
Оттепель с дождем и неудовлетворительное качество хоккейного поля кировского стадиона "Родина" не позволило сегодня, 5 ноября, провести стартовый матч первого этапа XXVIII Чемпионата России по хоккею с мячом среди клубов Суперлиги "Родина" - "Строитель". Он должен был состояться в 18.30.
Окончательное решение по поводу отмененной встречи и ненадлежащего состояния хоккейного поля примет Комитет по проведению соревнований ФХМР.
На 8 ноября запланирован матч "Старт" - "Строитель". Место проведения: Нижний Новгород, стадион "Труд".
Выехавшие в Киров для поддержки "Строителя" болельщики возвращаются в Сыктывкар.
Напомним, ледовое покрытие хоккейного поля должно отвечать утвержденным стандартам для проведения соревнований по хоккею с мячом в соответствии с Регламентом. Если состояние хоккейного поля не отвечает требованиям утвержденных стандартов, то это может являться одной из причин для отмены или переноса матча.
В случае невозможности проведения матча по метеоусловиям или иным рискам, которые, по мнению главного судьи и судьи-инспектора представляют угрозу жизни и здоровью спортсменов, а также безопасности всех участников соревнований, матч может быть отменен или перенесен.
Ответственность за плохое состояние хоккейного поля, включая плохое качество льда, лежит на организаторах матча.
