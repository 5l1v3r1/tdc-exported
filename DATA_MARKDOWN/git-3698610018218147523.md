
# Гончарук представить проект держбюджету-2020: онлайн

Published at: **2019-11-05T11:02:00+00:00**

Author: ****

Original: [DT.ua](https://dt.ua/ECONOMICS/goncharuk-predstavit-proekt-derzhbyudzhetu-2020-onlayn-328507_.html)

Прем'єр-міністр України Олексій Гончарук представить проект державного бюджету на 2020 рік, рекомендований до прийняття Верховною Радою в другому читанні. Проект держбюджету затвердили під час позачергового засідання Кабміну 3 листопада.
За словами Гончарука, в бюджет заклали "розвиток стратегічно важливих напрямів для людей", один з них — медицина. Глава уряду уточнив, що мова йде про систему екстреної медичної допомоги, конкретних цифр не назвав.
Він додав, що уряд планує заробляти всередині країни за рахунок приватизації, внутрішніх і зовнішніх інвестицій, лібералізації ринків.
