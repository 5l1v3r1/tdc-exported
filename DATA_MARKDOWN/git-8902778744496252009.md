
# Ростуризм исключил из реестра ещё 8 туроператоров

Published at: **2019-11-02T00:00:00+00:00**

Author: **Яна Вараксина**

Original: [Турпром](https://www.tourprom.ru/news/43819/)

Приказ об исключении из реестра еще восьми туроператоров издал в последние дни октября Ростуризм. Согласно тексту приказа, все туроператоры исключены из Реестра в связи с непредставлением сведений о наличии финансового обеспечения на новый срок.
Ни одного крупного игрока рынка среди исключенных нет. В основном это мелкие региональные компании.
Отметим, всего с начала октября Ростуризм в целом исключил из реестра 80 туроператоров. При этом четверо из них добровольно прекратили деятельность. У остальных причина та же – невозможность найти фингаранта.
