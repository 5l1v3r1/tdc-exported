
# Экс-главе Чебаркуля предъявлено новое обвинение

Published at: **2019-11-07T11:01:51+00:00**

Author: ****

Original: [Новый День](https://newdaynews.ru/chel/675979.html)

Экс-главе Чебаркульского городского округа Сергею Ковригину предъявлено новое обвинение.
Как уже сообщал «Новый День», в марте текущего года чиновника обвинили по ч.4 ст.290 УК РФ (получение взятки).
По данным следствия, в 2016 году обвиняемый получил взятку в виде услуг неимущественного характера на сумму более 52 тысяч рублей от представителя коммерческой организации за незаконную передачу объектов муниципальной собственности. Решением Арбитражного суда Челябинской области по иску прокурора области в 2017 году договоры аренды по передаче указанного муниципального имущества признаны недействительными (ничтожными).
По информации СУ СКРФ по Челябинской области, теперь Ковригин обвиняется по трем статьям: ч.2 ст.285 УК РФ (злоупотребление должностными полномочиями), ч.2 ст.286 УК РФ (превышение должностных полномочий), ч.4 ст.290 УК РФ (получение взятки).
«Оперативное сопровождение уголовного дела осуществляется сотрудниками ГУ МВД России по Челябинской области. Расследование уголовного дела находится в завершающей стадии», – сообщили в ведомстве.

        Челябинск, Павел Зварзин
