
# ఢిల్లీలో ‘తెలుగు అకాడమీ’

Published at: **2019-11-04T03:11:47+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=945590)

ADVT
న్యూఢిల్లీ, నవంబరు 3(ఆంధ్రజ్యోతి): ఢిల్లీలో నివసిస్తున్న తెలుగు వారి కోసం తెలుగు అకాడమీని ఏర్పాటు చేస్తామని సీఎం అరవింద్‌ కేజ్రీవాల్‌ ప్రకటించారు. ఢిల్లీలో నివసించే తెలుగు ప్రజలంతా తనను ఆదరించి, గౌరవిస్తున్నందుకు ఽగర్వపడుతున్నానన్నారు. ఆదివారం నాడిక్కడ ఢిల్లీ తెలుగు అకాడమీ ఆధ్వర్యంలో జరిగిన 32వ వార్షిక సాంస్కృతిక వేడుకలను ఆయన ప్రారంభించి మాట్లాడారు. మాడభూషి శ్రీధర్‌, ఎన్‌వీఎల్‌ నాగరాజు ఆధ్వర్యంలో కేజ్రీవాల్‌ను శాలువాతో ఘనంగా సన్మానించారు.
