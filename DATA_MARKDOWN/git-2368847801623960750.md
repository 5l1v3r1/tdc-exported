
# 【直播預告】川普11.4將參加肯塔基州助選集會（中文同聲翻譯）

Published at: **2019-11-01T23:15:03+00:00**

Author: ****

Original: [NTD 新唐人電視台](https://www.ntdtv.com/b5/2019/11/01/a102698416.html)

【新唐人北京時間2019年11月02日訊】美國總統川普11月4日將參加肯塔基州列克星敦魯普體育館（Lexington Rupp Arena）的競選集會，為肯塔基州長馬特·貝文（Matt Bevin）的連任助陣。新唐人電視台和《大紀元時報》將進行全程追蹤和網絡直播，並有同聲翻譯。請鎖定新聞直播間。
據WBKO報導，魯普體育館將於11月4日下午4點開放。 「讓美國繼續偉大」（Keep America Great）集會從傍晚7點開始。官方尚未發布集會議程，但據一份新聞稿，川普期待與肯塔基州人分享他的成就。
貝文正與州檢察長、民主黨候選人安迪·比希爾（Andy Beshear）進行激烈的競爭。
川普曾推文盛讚州長貝文。川普說，貝文為肯塔基人民做的非常棒！貝文繼續保護著你們非常重要的第二修正案。「他在犯罪和邊境問題上都很強勢，他熱愛我們偉大的老兵和軍隊。他擁有我的完全和全面的支持，並且一直都有！」
川普最近一次訪問肯塔基州是在兩個月前，當時他在路易斯維爾舉行的第75屆AMVETS全國代表大會上發表講話。
預計副總統邁克·彭斯（Mike Pence）也將在大選之前對肯塔基州進行訪問。
