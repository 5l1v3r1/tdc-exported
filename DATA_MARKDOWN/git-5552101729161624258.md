
# 【豪門爆姦情5】偷情還拍淫照與影片　元祖小開超愛炫耀婚姻幸福

Published at: **2019-11-05T21:58:00+00:00**

Author: **娛樂組, 攝影組**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191105ent021/)

歌手兼網路電商企業家安晨妤和「富二代」、元祖食品少東張劭緯結婚13年，表面一副甜蜜幸福模樣，沒想到日前卻驚爆婚變。主因是張劭緯結婚以來，背著老婆安晨妤偷歡數女，還拍下淫照與影片，而且不倫的其中一位對象，竟是與元祖敵對的大黑松小倆口千金邱威樺。
