
# В Бурятии пройдет перенастройка эфирного телесигнала

Published at: **2019-11-07T15:14:00+00:00**

Author: ****

Original: [RuNews24](https://runews24.ru/buryatiya/07/11/2019/8401fba9745d69176cde25bf777fafc6)

Перенастройки связаны с началом вещания региональных выпусков на федеральном канале ОТР.
Как сообщают с Минтрансе, у жителей республики могут возникнуть помехи в первом мультиплексе, но не стоит расстраиваться. Проблема будет носить временный характер. Поэтому, для корректной работы мультиплекса стоит в настройках запустить автопоиск каналов и трансляция вернется уже с расширенными возможностями.
Региональное вещание на канале ОТР начнется с начала 2020 года.
Для получения дополнительных консультаций по настройке оборудования можно обратиться по телефону Федеральной горячей линии 8-800-220-20-02, а также региональной горячей линии 8(3012)337777.
Напомним, Бурятия перешла на цифровое вещание в июне 2019 года.
