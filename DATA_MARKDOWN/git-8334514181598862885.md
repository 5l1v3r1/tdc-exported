
# Blast heard in Baghdad, cause unknown: Reuters witnesses

Published at: **2019-11-05T18:15:50+00:00**

Author: ****

Original: [Reuters](https://www.reuters.com/article/us-iraq-security-idUSKBN1XF2BG)

BAGHDAD (Reuters) - A blast was heard in the Iraqi capital Baghdad on Tuesday night, Reuters witnesses said. The cause was as yet unknown. There was no immediate report of a security incident or immediate comment from security officials.
