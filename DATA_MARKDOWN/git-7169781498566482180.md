
# Microsoft провела эксперимент по переводу сотрудников на четырехдневную рабочую неделю

Published at: **2019-11-05T15:46:00+00:00**

Author: **Марина Ибушева**

Original: [Лайкни](https://www.likeni.ru/events/microsoft-provela-eksperiment-po-perevodu-sotrudnikov-na-chetyrekhdnevnuyu-rabochuyu-nedelyu/)

Японское подразделение компании Microsoft подвело итоги эксперимента по переходу на четырехдневную рабочую неделю. За тот месяц, что длился эксперимент, продуктивность сотрудников выросла, а расход элекроэнергии и бумаги сократился. 
В течение августа 2300 сотрудников филиала Microsoft в Японии работали с понедельника по четверг, имея дополнительный выходной в пятницу. За этот период продуктивность, измеряемая продажами на одного сотрудника, выросла на 39,9% по сравнению с августом 2018 года. Дополнительно компания заметила снижение расходов. В частности, на 23,1% снизилось потребление электроэнергии, на 58,7% уменьшился расход бумаги. По итогам эксперимента, 92,1% сотрудников заявили, что им понравилось работать четыре дня в неделю. Microsoft планирует провести еще один подобный эксперимент этой зимой. Напомним, эксперименты по сокращению рабочей недели ведутся последние несколько лет. По мнению экспертов, в будущем с развитием технологий люди смогут работать по 3 дня в неделю. В частности, такие идеи высказывали предприниматель Ричард Брэнсон и основатель и соучредитель Alibaba Group Джек Ма. Источник: CNBC
