
# Скончались двое пострадавших при стрельбе на западе Москвы

Published at: **2019-11-02T16:12:47+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-02/skonchalis-dvoe-postradavshih-pri-strelbe-v-moskve)

Как стало известно РЕН ТВ, скончались двое мужчин, пострадавших при стрельбе на улице Академика Анохина в Москве.
По предварительным данным, ЧП произошло возле подъезда одного из домов. Стрельбу открыл злоумышленник в маске.
Он скрылся с места преступления на черном BMW. Правоохранители ведут его розыск, объявлен план "Перехват".
