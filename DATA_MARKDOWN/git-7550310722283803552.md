
# Британский гонщик Льюис Хэмилтон стал шестикратным чемпионом «Формулы-1»

Published at: **2019-11-04T08:25:00+00:00**

Author: ****

Original: [Новая Газета](https://www.novayagazeta.ru/news/2019/11/04/156624-britanskiy-gonschik-lyuis-hemilton-stal-shestikratnym-chempionom-formuly-1)

Британский пилот команды «Мерседес» Льюис Хэмилтон в шестой раз в карьере выиграл чемпионат мира по автогонкам в классе машин «Формула-1». Теперь Хэмилтон уступает по количеству титулов только Михаэлю Шумахеру, который выигрывал турнир семь раз.
