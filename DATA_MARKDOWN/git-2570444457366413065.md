
# Жительница Саратова обнаружила свою грудную дочь мертвой в коляске

Published at: **2019-11-07T14:57:00+00:00**

Author: ****

Original: [Саратов 24](https://saratov24.tv/news/zhitelnitsa-saratova-obnaruzhila-svoyu-grudnuyu-doch-mertvoy-v-kolyaske/)

Следственным отделом по Заводскому району организована доследственная проверка по факту
гибели ребенка. Об этом сообщает пресс-служба Следственного комитета области.
По предварительным данным, сегодня 7 ноября 2019 года, утром 32-летняя женщина в доме по улице Азина Заводского района обнаружила в коляске свою 7-месячную дочь без признаков жизни. После чего она вызвала "скорую помощь" Врачи констатировали смерть девочки.
"Проведенным следователями осмотром места происшествия признаков насильственной смерти на теле погибшей не обнаружено", - рассказали журналисту телеканала "Саратов 24"в ведомстве.
В настоящее время проводится комплекс мероприятий, направленных на установление всех обстоятельств произошедшего. Назначены судебно-медицинские экспертизы с целью установления причин смерти малолетнего ребенка. По результатам проверки будет принято соответствующее процессуальное решение.
Автор Артур Авдеев
