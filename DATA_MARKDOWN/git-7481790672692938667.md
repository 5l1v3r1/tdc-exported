
# ［深層ＮＥＷＳ］米大統領選「経済情勢悪くなれば…現職は厳しい」

Published at: **2019-11-04T15:04:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/world/20191104-OYT1T50203/)

　自民党の長島昭久衆院議員と海野素央明治大教授、中山俊宏慶応大教授が４日、ＢＳ日テレの「深層ＮＥＷＳ」に出演し、約１年後に迫った米大統領選について議論した。
　トランプ大統領の再選について、長島氏は「経済情勢が悪くなれば現職は厳しい」と述べた。海野氏は、民主党指名争いについて「候補者たちは決め手がない」と語った。中山氏は「民主党は分裂しているがゆえに負けるという可能性はある」と分析した。
