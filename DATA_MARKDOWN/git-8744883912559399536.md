
# HBO представила коробку для спокойного просмотра сериалов

Published at: **2019-11-06T15:04:32+00:00**

Author: **Евгений Федуненко**

Original: [Коммерсантъ](https://www.kommersant.ru/doc/4149764)

Американская компания HBO представила черную картонную коробку The HBO Box, которая предназначена для уединенного просмотра фильмов и сериалов. В качестве презентации компания опубликовала на YouTube видеоролик, в котором рассказала об основных особенностях предмета.
