
# Нефтяники пожаловались Козаку на Министерство финансов

Published at: **2019-11-07T10:27:00+00:00**

Author: ****

Original: [RuNews24](https://runews24.ru/economy/07/11/2019/d4425d536a699de97782cec01e482283)

Соответствующие обращения на имя вице-премьера Дмитрия Козака, министра энергетики Александра Новака и главы антимонопольной службы Игоря Артемьева направило объединение независимых нефтегазовых компаний «Ассонефть», пишет РБК.
Ввести налог на добычу полезных ископаемых для ПНГ предложил Минфин, чтобы компенсировать выпадающие доходы бюджета (600 млрд рублей за 10 лет) из-за льгот по НДПИ для Приобского месторождения «Роснефти» и «Газпром нефти». Льготы одобрил президент России Владимир Путин, однако чиновники еще обсуждают схему компенсации для бюджета.
В данный момент законопроект проходит процедуру публичного обсуждения.
