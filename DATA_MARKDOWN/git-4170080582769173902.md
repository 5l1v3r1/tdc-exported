
# Из-за гибели мужчины в ДТП в Туле возбуждено уголовное дело

Published at: **2019-11-02T18:17:00+00:00**

Author: ****

Original: [Тульская Служба Новостей](https://www.tsn24.ru/iz-za-gibeli-muzhchiny-v-dtp-v-tule-vozbuzhdeno-ugolovnoe-delo.html)

В результате гибели мужчины в ДТП с участием маршрутки в Туле возбуждено уголовное дело.
Напомним, накануне «автолайн», следовавший по маршруту №42, перевернулся на Алексинском шоссе. В результате аварии пострадали три человека – два пассажира и водитель. Один из них скончался в больнице.
По факту ДТП возбуждено уголовное дело по статье «Нарушение ПДД и эксплуатации транспортных средств, повлекшее по неосторожности смерть человека».
