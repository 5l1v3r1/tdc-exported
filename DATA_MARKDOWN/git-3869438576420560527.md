
# В Ковдоре зимой будет светить собственное северное сияние

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [Мурманский Вестник](https://www.mvestnik.ru/newslent/v-kovdore-zimoj-budet-svetit-sobstvennoe-severnoe-siyanie/)

В Ковдоре появился новый арт-объект «Северное сияние». Световая конструкция передает краски соответствующего природного явления, одного из узнаваемых символов Заполярья.Совсем скоро инсталляция заработает и будет радовать яркими красками жителей вплоть до весны.
