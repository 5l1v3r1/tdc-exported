
# Сайфиев қайтди, Кримец эса... Ўзбекистон МТЖ машғулотидан ГАЛЕРЕЯ

Published at: **2019-11-07T17:52:28+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/sayfiev-qaytdi-krimec-esa-ozbekiston-mtj-mashgulotidan-galereya)

Машғулотлар "Жар" стадионининг захира майдонида ўтказилмоқда.
Фарруҳ Сайфиев икки кунлик индивидуал шуғулланишдан кейин умумий гуруҳга қайтгани ва Егор Кримец енгил жароҳат олгани ҳақида аввалроқ хабар берган эдик >>>
