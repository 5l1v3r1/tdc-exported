
# Курганские судебные приставы призвали граждан жаловаться им на незаконные действия коллекторов

Published at: **2019-11-07T07:03:00+00:00**

Author: **Михаил Белокрылов**

Original: [Накануне.ru](https://www.nakanune.ru/news/2019/11/07/22557523/)

Судебные приставы Зауралья организовали телефонную "горячую линию". Туда может позвонить любой желающий и пожаловаться на незаконные действия коллекторов.
Как сообщили Накануне.RU в пресс-службе УФССП России по Курганской области, в ведомстве постоянно ведется работа по консультированию зауральцев по вопросам, касающимся деятельности коллекторских агентств.
"Горячая линия" пройдет 8 ноября с 9.00 до 12.00.
Звонки в эти три часа будут принимать по номеру телефона в Кургане 8 (3522) 49-19-83.
