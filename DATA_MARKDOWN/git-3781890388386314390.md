
# 'โค้ชบอส'สัญญาหมด หลังพาจงอางยึดแชมป์ไทยลีก3

Published at: **2019-11-07T10:10:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/sports/740484)


        "โค้ชบอส" ปฏิภัทร รอบรู้ สุดภูมิใจนำทีมบ้านเกิด ขอนแก่น ยูไนเต็ด คว้าแชมป์ไทยลีก 3 สัญญาหมด พร้อมรับงานทุกทีม
      
"โค้ชบอส" ปฏิภัทร รอบรู้ กล่าวว่า หลังจากที่ตนนำทีม "จงอางผยอง" ขอนแก่น ยูไนเต็ด คว้าแชมป์ไทยลีก 3 เลื่อนชั้นสู่ไทยลีก 2 ได้สำเร็จ ต้องขอบคุณ "เสี่ยต้อม" วัฒนา ช่างเหลา ประธานสโมสร ขอนแก่น ยูไนเต็ด ที่ให้โอกาส รวมทั้งแฟนคลับที่สนับสนุน ภูมิใจมากที่ทำทีมบ้านเกิดกลับมายิ่งใหญ่ ตอนนี้สัญญาของตนกับทีมหมดลงแล้ว ยังขอเป็นกำลังใจให้ทีมขอนแก่นมุ่งหน้าสู่ลีกสูงสุดให้ได้ ส่วนตัวก็เปิดรับข้อเสนอการคุมทีมจากทุกทีม ไม่ได้ตั้งเป้าว่าจะเป็นทีมในลีกไหน หากทีมไหนสนใจ สามารถมาคุยกันได้.
