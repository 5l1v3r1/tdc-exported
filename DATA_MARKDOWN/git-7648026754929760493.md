
# פסטיבל הביצה להומור יהודי יתקיים בגולה בפעם השלישית

Published at: **2019-11-05T07:20:00+00:00**

Author: **ליהיא גלמן רם**

Original: [חדשות פתח תקוה](http://mynetpetahtikva.co.il/%d7%aa%d7%a8%d7%91%d7%95%d7%aa-%d7%95%d7%91%d7%99%d7%93%d7%95%d7%a8/%d7%a4%d7%a1%d7%98%d7%99%d7%91%d7%9c-%d7%94%d7%91%d7%99%d7%a6%d7%94-%d7%9c%d7%94%d7%95%d7%9e%d7%95%d7%a8-%d7%99%d7%94%d7%95%d7%93%d7%99-%d7%99%d7%aa%d7%a7%d7%99%d7%99%d7%9d-%d7%91%d7%92%d7%95%d7%9c%d7%94-%d7%91%d7%a4%d7%a2%d7%9d-%d7%94%d7%a9%d7%9c%d7%99%d7%a9%d7%99%d7%aa-415872/3)

פסטיבל הביצה הינו הפסטיבל היחיד בארץ להומור יהודי. במסגרת הפסטיבל שיימשך כשבועיים בין התאריכים 9-23.11.19  יתקיימו מגוון אירועים בגולה, ביניהם הופעות יחיד, ערבי בידור, סטנד אפ, הצגות ילדים ועוד.
פסטיבל הביצה ביוזמת מועדון הגולה ובתמיכת סגן רה"ע איציק דעי ויו"ר מקפ"ת אליהו גינת,  מתקיים בפעם השלישית והוא פסטיבל ייחודי בעצם העובדה שהוא משלב ותיקים וצעירים, דתיים וחילוניים ומקיים מגוון אירועים לכל הגילאים.
ראש העיר רמי גרינברג בירך על קיום הפסטיבל ואמר: "הומור יהודי היה קיים מאז ומתמיד, בכל התפוצות ובכל התקופות, ואנו החלטנו לקיים גם השנה את פסטיבל הביצה - פסטיבל להומור יהודי, ולתת במה לנישה הייחודית הזו. אני מזמין את כולכם לבוא וליהנות משפע האירועים שיתקיימו במסגרת הפסטיבל".
טקס פסטיבל הביצה, בו יוענקו אות הצפרדע לסטנדאפיסט השנה, יתקיים ביום ראשון ה- 10.11 בשעות הערב.
