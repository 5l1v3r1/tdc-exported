
# Градоначальник Каменского провел объезд города

Published at: **2019-11-07T18:33:00+00:00**

Author: ****

Original: [dndz.tv](http://dndz.tv/article-30061.html)

07.11.19 городской голова г. Каменское (Днепродзержинск) с инспекцией побывал в разных районах города.Улицы и территорию дворов мэр проинспектировал вместе с подчиненными:- ни один район не обошли вниманием;- Андрей Белоусов с ответственными работниками проверили работу коммунальных служб по вывозу опавшей листвы;- проблем с вывозом у КП «УКОЖФ» нет;- вывезли более 25000 кубов листьев с использованием техники, рабочих;- очередную проверку следует ожидать через 3 недели, сказал А. Белоусов.Особые поручения по восстановлению дорожного покрытия получили службы «Горводоканала»  и «Теплосети».
