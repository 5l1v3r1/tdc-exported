
# ФК Буковина зіграє домашній матч проти ФК Верес з Рівного

Published at: **2019-11-06T16:46:00+00:00**

Author: ****

Original: [Погляд](https://pogliad.ua/news/sport/fk-bukovina-zigrae-domashniy-match-proti-fk-veres-z-rivnogo-379917)

У найближчу неділю, 10-го листопада 2019 року, ФСК "Буковина" зіграє черговий домашній матч чемпіонату України серед команд Другої ліги групи А сезону 2019/20 проти одного з лідерів першості - футбольного клубу "Верес" з Рівного.
Початок матчу о 14.00. Гра відбудеться на стадіоні "Буковина".
