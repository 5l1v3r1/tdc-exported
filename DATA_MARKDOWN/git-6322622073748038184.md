
# 云顶马来西亚子公司向建发购帝国1320万股

Published at: **2019-11-05T07:17:59+00:00**

Author: **Chong Jin Hun**

Original: [The Edge Markets](https://www.theedgemarkets.com/article/%E4%BA%91%E9%A1%B6%E9%A9%AC%E6%9D%A5%E8%A5%BF%E4%BA%9A%E5%AD%90%E5%85%AC%E5%8F%B8%E5%90%91%E5%BB%BA%E5%8F%91%E8%B4%AD%E5%B8%9D%E5%9B%BD1320%E4%B8%87%E8%82%A1)

（吉隆坡5日讯）云顶马来西亚（Genting Malaysia Bhd）独资子公司云顶（美国）有限公司周一（4日）以每股9.74每美元的价格，向建发实业III私人有限公司收购帝国度假村（Empire Resorts Inc）的1320万股。
这是云顶马来西亚-建发实业联营，以私有化美国上市赌场营运商帝国的计划。
“随着这项收购，云顶美国如今持有帝国38.3%股权，假设所有流通的优先股全部转换为普通股，则稀释至33.3%。”
云顶马来西亚指出：“云顶美国和建发实业完成拟议收购与合并计划无关，也不附条件。”
今年8月6日，云顶马来西亚向大马交易所报备，该集团与建发实业于8月5日签署具约束力的条款清单，云顶美国将以约1亿2860万美元（5亿3880万令吉）或每股9.74美元，收购帝国的1320万股。
根据云顶马来西亚，将通过云顶美国连同建发实业向帝国提交一份初步非约束提案，以合并的方式收购与建发实业无关的帝国股东持有的流通股。
截至3时，云顶马来西亚起2仙或0.62%，报3.26令吉，交投量有709万9600股。
