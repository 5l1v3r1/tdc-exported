
# Автомобиль ГАИ столкнулся с такси в Москве

Published at: **2019-11-03T13:04:52+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-03/avtomobil-gai-stolknulsya-s-taksi-v-moskve)

Автомобиль Госавтоинспекции столкнулся с такси в Москве. По информации РЕН ТВ, авария произошла на пересечении улиц Вавилова и Профсоюзной. 
Пока неизвестно, есть ли в результате ДТП пострадавшие. 
Причины столкновения выясняются.
