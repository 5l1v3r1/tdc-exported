
# 田口淳之介が活動再開　新曲、ファンミ「しっかり前に進んで」【コメント全文】

Published at: **2019-11-04T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/04/0012847951.shtml)

　大麻取締法違反で先月２１日、懲役６月執行猶予２年の有罪判決を受けた元ＫＡＴ-ＴＵＮの田口淳之介（３３）が４日、公式サイトで活動再開を発表し、「これからしっかり前に進んで行ける様に、努力して参ります」などとコメントした。
　田口は今月７日に自社レーベル「Ｉｍｍｏｒｔａｌ-Ｒｅｃｏｒｄｓ」から初の楽曲となる「Ｖｏｉｃｅｓ」（ボイシズ）を配信リリース。「心の奥に押し殺した自分自身を声と共に解き放て！という強いメッセージを歌とアグレッシブなダンスで表現。困難な状況でも立ち向かう自身の姿を投影する１曲」だといい、ミュージックビデオも同時公開する。
　また、毎年恒例というファンミーティングを行うことも発表。「Ｊｕｎｎｏｓｕｋｅ　Ｔａｇｕｃｈｉ　ｏｆｆｉｃｉａｌ　ＦＡＮ　ｍｅｅｔｉｎｇ　２０１９　～ＣＯＳＭＯＳ　ＣＩＴＹ～」と題して１２月３日に東京・六本木で開催。今年開催予定だったツアーに向けて制作した楽曲を初披露するという。４日からファンクラブの新規入会受付も再開する。
　田口のコメント全文は次の通り。
　　◇　　◇
　僕の新たな１歩を刻む楽曲が完成しました。協力してくれた仲間や支えてくれるスタッフに本当に感謝しています。
　力強い踊りで何事にも立ち向かう強さを表現しました。歌と共に僕からのメッセージを受け取って下さい。
　そしてファンの皆さんには大変お待たせする事になりましたが、僕とファンの皆さんだけの特別な時間を用意することが出来ました。
　この日は皆さん一人一人と心が通じ合える様に、しっかり準備してお待ちしています。
　楽しみにしていてください。
　これからしっかり前に進んで行ける様に、努力して参ります。応援のほど何卒宜しくお願い致します。
　田口淳之介
