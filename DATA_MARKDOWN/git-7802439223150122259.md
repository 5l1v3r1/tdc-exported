
# Microsoft Edge Ganti Logo, Tak Lagi Mirip Internet Explorer

Published at: **2019-11-04T06:03:33+00:00**

Author: **Anggoro Suryo Jati - detikInet**

Original: [detikinet](https://inet.detik.com/cyberlife/d-4770793/microsoft-edge-ganti-logo-tak-lagi-mirip-internet-explorer?utm_term=echoboxauto&utm_campaign=detikcomsocmed&utm_medium=oa&utm_content=detikinet&utm_source=Twitter)

Jakarta - Microsoft memperbarui logo browser Edge setelah lebih dari empat tahun menggunakan logo yang mirip dengan logo browser lama mereka, yaitu Internet Explorer.Logo baru ini ditemukan dalam sebuah mini game yang tersembunyi di dalam Edge versi Canary. Logo baru ini mirip dengan ombak, senada dengan gaya Fluent Design yang diterapkan Microsoft di ikon baru Office.Mini game ini adalah bagian dari perburuan Easter Egg yang tersimpan di dalam bermacam pesan cryptic dari sejumlah puzzle dan gambar. Perburuan mini game itu terbilang sulit, bahkan melibatkan proses render ikon Edge menjadi objek tiga dimensi.Mini gamenya sendiri mirip dengan SkiFree, game klasik yang menjadi bagian dari Microsoft Entertainment Pack 3 untuk Windows yang dirilis 1991 lalu. Anda bisa menggunakan tombol WASD di keyboard untuk bernavigasi dan menghindari bermacam halangan.Logo ombak ini memang masih membentuk huruf 'e' seperti logonya yang sebelumnya. Namun logo baru ini terlihat lebih modern dan tak lagi mirip dengan logo Internet Explorer.Tampaknya logo baru ini digunakan sejalan dengan perubahan besar yang ada di dalam browser bawaan Windows 10 itu, yaitu penggunaan engine Chromium, demikian dikutip detikINET dari The Verge, Minggu (3/11/2019).Penggunaan engine Chromium ini cukup menarik dan mengundang rasa penasaran, salah satunya adalah apakah engine baru ini akan membuat Edge menjadi lebih mirip Chrome atau tidak.Namun untuk menjajalnya, kita harus menunggu sampai Microsoft merilis versi final dari Edge Chromium ini.
