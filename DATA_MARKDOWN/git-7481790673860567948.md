
# 公民権回復手続きを３年以上怠る…市が恩赦調査で気づき謝罪

Published at: **2019-11-02T07:20:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191101-OYT1T50305/)

　京都府舞鶴市は１日、公民権が停止された男性について、本来は５年間で回復すべきところを、先月まで３年５か月にわたって余分に停止状態にしていたと発表した。天皇陛下の「即位礼正殿の儀」に合わせて行われた恩赦の対象者調査で先月下旬に気づき、男性に謝罪した。
　市民課によると、男性は罰金刑に処せられ、２０１１年５月から１０年間の公民権停止となった。ただ刑法には、罰金刑の執行後、再び罰金以上の刑に処せられずに５年が経過すれば、刑の効力を失うとする規定があり、制限された権利や資格が回復することになっている。男性の公民権は１６年５月に回復されなければならなかったのに、市がその必要性を認識していなかったという。
　市は先月２５日付で回復手続きを行い、２７日に謝罪したが、男性はこの間に実施された参院選や市長選など７回の選挙に投票する機会を失った。市は「認識不足が原因で多大なご迷惑をおかけした。回復手続きの状況を定期的に確認して再発を防ぎたい」としている。
