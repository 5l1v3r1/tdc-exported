
# Фонд Рината Ахметова возглавляет тройку благотворительных организаций Украины

Published at: **2019-11-05T13:10:37+00:00**

Author: ****

Original: [34 телеканал](https://34.ua/fond-rinata-ahmetova-vozglavlyaet-trojku-blagotvoritelnyh-organizacij-ukrainy_n82882)

Благодаря помощи Фонда уже шестой год выживает мирное население на востоке страны. Продуктовые наборы регулярно доставляют в населенные пункты вдоль всей линии соприкосновения. Благодаря такой поддержке, спасено более миллиона человек.
Наши корреспонденты вернулись с одной из наиболее пострадавших сел "серой зоны" - Каменки.
Жуткая тишина. Когда-то это была одна из самых оживленных дорог. Трасса "Донецк-Харьков" соединяла два мощных индустриальных города. Остались одни воспоминания. Теперь на участке пути, который идет через село Каменку, слышно только детей.
Каменка - вблизи линии разграничения. За ней - уже опустошены деревни.

        "Там поселок есть Крутая балка, но там выселили людей. Как раз, по дачам оно все проходит", - говорит жительница Каменки Надежда.
      
Среди руин, живет почти сотня людей. Вспоминают, которым было их поселок в мирное время.

        "Вот это кафе у нас было здесь. Продавали напитки, пирожки, кофе. Постоянно останавливались машины", - вспоминает Надежда.
      
Надежда Слаква на этих полях когда-то выращивала рожь, пшеницу, овес. Последний урожай собирали под обстрелами еще в 2014 году. А теперь это опасные участки.
