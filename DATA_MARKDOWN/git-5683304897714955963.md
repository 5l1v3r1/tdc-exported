
# അന്യൻ ലാലേട്ടനായിരുന്നെങ്കിൽ വേറെ ലെവൽ: വിക്രത്തോട് ഭാര്യ

Published at: **2019-11-06T07:43:00+00:00**

Author: **മനോരമ ലേഖകൻ**

Original: [ManoramaOnline](https://www.manoramaonline.com/movies/movie-news/2019/11/06/vikram-tells-about-his-wife-and-mohanlal-adithya-varma-dhruv-vikram.html)

താന്‍ മാത്രമല്ല തന്റെ ഭാര്യയും മോഹന്‍ലാലിന്റെ കടുത്ത ആരാധികയാണെന്ന് നടന്‍ വിക്രം. മകന്‍ ധ്രുവ് വിക്രം നായകനാകുന്ന ‘ആദിത്യവര്‍മ’യുടെ പ്രമോഷനു വേണ്ടി കേരളത്തിലെത്തിയതായിരുന്നു താരം.
