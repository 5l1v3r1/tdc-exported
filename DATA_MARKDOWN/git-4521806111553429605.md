
# Netflix раскритиковали за решение по поводу проката фильма «Ирландец»

Published at: **2019-11-04T09:06:00+00:00**

Author: ****

Original: [RuNews24](https://runews24.ru/world/04/11/2019/367906c2b94b99e986f4f66ddf937778)

По его словам, Netflix послал сигнал кинематографистам, что даже если вы – Мартин Скорсезе, то все равно не получите широкого релиза в кинотеатрах при сотрудничестве с Netflix.
Решение компании глава NATO назвал позором.
Ранее крупные сети кинотеатров Америки не смогли достичь компромисса с Netflix по поводу показа «Ирландца», права на который принадлежат стриминг-сервису.
Камнем преткновения стало время постпремьерной публикации фильма. Сети кинотеатров заинтересованы в трехмесячной отсрочке для проведения успешных показов, Netflix же стремиться начать продавать фильм через 30-45 дней после премьеры.
