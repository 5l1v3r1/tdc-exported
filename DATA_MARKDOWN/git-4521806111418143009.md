
# На Украине предложили создать ракеты для ударов по АЭС России

Published at: **2019-11-02T07:42:00+00:00**

Author: ****

Original: [RuNews24](https://runews24.ru/politics/02/11/2019/e47142ae10cef2d2ded84cf65f60eed6)

По его словам, такой призыв обусловлен «прагматическими соображениями», Киев должен сокращать свою зависимость от Москвы, от российских энергоносителей, замыкать производство оружия в стране, «не бояться строить ракеты».
Загородний отметил, что Украина должна создать ракеты с дальностью до 2,5 тысячи км. 200 штук таких ракет для начала будет достаточно, это позволит наносить ущерб России.
Как ракета с обычной боеголовкой после попадания в АЭС вызовет ядерный взрыв, эксперт не объяснил.
