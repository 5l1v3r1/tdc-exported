
# Боттас выиграл квалификацию Гран-при США Формулы-1

Published at: **2019-11-03T00:45:00+00:00**

Author: ****

Original: [Interfax.ru](https://www.sport-interfax.ru/682807)

Москва. 3 ноября. INTERFAX.RU - Финн Вальттери Боттас из команды Mercedes стал победителем квалификации Гран-при США, очередного этапа чемпионата мира по автогонкам класса "Формула-1".
Вторым финишную черту на трассе в Остине пересек немец Себастьян Феттель (Ferrari). Замкнул тройку призеров голландец Макс Ферстаппен (Red Bull).
Россиянин Даниил Квят, выступающий за команду Toro Rosso, занял 13-е место.
