
# У аэропорта «Мурманск» новый гендиректор

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [Мурманский Вестник](https://www.mvestnik.ru/newslent/u-aeroporta-murmansk-novyj-gendirektor/)

В ПАО «Аэропорт Мурманск» кадровые изменения. Генеральным директором воздушной гавани станет Руслан Степанец, который ранее занимал должность заместителя гендиректора по производству.Занимавший пост гендиректора с 2015 года Андрей Осипов станет руководителем челябинского международного аэропорта имени Курчатова.
