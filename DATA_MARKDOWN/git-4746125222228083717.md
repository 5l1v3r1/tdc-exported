
# 「エモい」ニュースとイケメンコンテンツを発信 Webマガジン『Emo!miu(エモミュー)』サイトオープン

Published at: **2019-11-01T18:10:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147860/full/)

　ティーンをターゲットとしたマーケティング及びプロモーション事業を行っているアイ・エヌ・ジーは1日、エモい（心が揺さぶられる）ニュースと男性インフルエンサー・モデルを起用したWebマガジン『Emo!miu（エモミュー）』を開設したことを発表した。

        【写真】その他の写真を見る
      
　同社では「毎日見たくなる、読みたくなるWEBサイトが見つからない」「短時間で厳選された話題のニュースだけを読みたい」といったティーンの声を参考に、「空いた時間に10分読めば気分がアガる」をコンセプトに同サイトを立ち上げた。 　ユーザー属性はハイティーンから20代女性を想定。男性インフルエンサーによるコンテンツやファッション・ビューティー・カフェグルメといったジャンルからセレクトされたニュースを毎日更新していく予定。 　また、現在掲載中のLol（エルオーエル）、M!LK（ミルク）、こんどうようぢなどの若年層女性に人気のタレントのインタビューをはじめ、男性モデル・インフルエンサーによるスナップ、渋谷マップ（準備中）といったオリジナルコンテンツにも力を入れていくという。 　現在、サイトオープンを記念し、抽選で“エモい”福袋が当たるキャンペーンも開催中。期間は1日から5日午後11時59分まで。
