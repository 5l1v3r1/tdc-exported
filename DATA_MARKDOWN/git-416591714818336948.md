
# Valve pode estar desenvolvendo concorrente para o Google Stadia

Published at: **2019-11-06T15:30:00+00:00**

Author: **Guilherme Preta**

Original: [Olhar Digital](https://olhardigital.com.br/games-e-consoles/noticia/valve-pode-estar-desenvolvendo-concorrente-para-o-google-stadia/92619)

Enquanto o lançamento do Google Stadia não chega, a plataforma pode estar prestar a ganhar uma forte concorrência. A Valve, criadora do serviço para PCSs Steam, pode estar desenvolvendo seu próprio streaming de jogos.
O Steam Database, serviço que rastreia alterações nos sites de bancos de dados da Valve, encontrou uma atualização no site em que os desenvolvedores usam para inscrever seus jogos que menciona o termo “Steam Cloud Gaming”.
A atualização adiciona uma seção para que os parceiros concordem com os termos do adendo ao Steam Cloud Gaming.
