
# Подписана грамота о восстановлении единства западных приходов с РПЦ

Published at: **2019-11-01T14:40:10+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-01/podpisana-gramota-o-vosstanovlenii-edinstva-zapadnyh-prihodov-s-rpc)

Патриарх Кирилл подписал грамоту о восстановлении единства западноевропейской архиепископии с Русской православной церковью. Об этом сообщается на официальном сайте РПЦ.
Документ был подписан в двух экземплярах.
3 ноября один из них вручат главе Архиепископии Иоанну, второй поместят на хранение в архив Московской патриархии.
