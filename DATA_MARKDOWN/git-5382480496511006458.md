
# Теперь не скользко

Published at: **2019-11-02T12:28:00+00:00**

Author: **yk.kz**

Original: [yk.kz](https://yk.kz/news/show/53439)

Работы начались вчера вечером, за ночь резиновыми плитами выложили две трети проходов и часть ступеней.
Плиты делают из старых шин местные подрядчики. Покрытие не сплошное, чтобы можно было легко заменить изношенную часть, - объяснили в отделе ЖКХ.
