
# สหรัฐ-จีนสาดสงครามน้ำลายหลังเวทีอาเซียน

Published at: **2019-11-05T02:09:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/foreign/740034)


        ผู้แทนของสหรัฐและจีนในการประชุมอาเซียนที่ไทย เปิดสงครามน้ำลายนอกรอบในประเด็นทะเลจีนใต้ ขณะที่รัฐบาลวอชิงตันปัดกระแสข่าวความไม่พอใจเรื่องการ "พยายามบอยคอต" ของผู้นำอาเซียน ด้วยการยืนยันเอเชียตะวันออกเฉียงใต้เป็น "พันธมิตรที่สำคัญ"
      
สำนักข่าวต่างประเทศรายงานจากกรุงเทพมหานคร เมื่อวันที่ 5 พ.ย. ว่านายเล่อ หยู่เฉิง รมช.กระทรวงการต่างประเทศจีน ซึ่งเป็นหนึ่งในเจ้าหน้าที่ระดับสูงของรัฐบาลปักกิ่งที่ติดตามนายกรัฐมนตรีหลี่ เค่อเฉียง เยือนกรุงเทพมหานคร กล่าวนอกรอบการประชุมสุดยอดสมาคมประชาชาติแห่งเอเชียตะวันออกเฉียงใต้ ( อาเซียน ) ครั้งที่ 35 ที่ไทยเป็นเจ้าภาพและปิดฉากเมื่อวันจันทร์  ว่ายังคงมี  "บางประเทศนอกภูมิภาค" ซึ่งไม่เกี่ยวข้องกับสถานการณ์ในทะเลจีนใต้แม้แต่น้อย แต่กลับ "กวนน้ำให้ขุ่น" และ "คอยสร้างคลื่นลมแรงอยู่เป็นระยะ" เนื่องจากหากความสงบในทะเลจีนใต้ เป็นบรรยากาศที่ฝ่ายนั้น "รับไม่ได้"
