
# В деревне Конаковского района на пожарище обнаружили тело погибшего мужчины

Published at: **2019-11-03T13:26:00+00:00**

Author: ****

Original: [Тверские ведомости](https://vedtver.ru/news/proisshestviya/v-derevne-konakovskogo-rayona-na-pozharishche-obnaruzhili-telo-pogibshego-muzhchiny/)

Как сообщили на сайте СУ СК РФ по Тверской области, проводится доследственная проверка по факту гибели мужчины, тело которого обнаружили на пожарище в квартире на втором этаже трёхэтажного дома в деревне Игуменка Конаковского района Тверской области.
Пожар произошёл в Игуменке вчера, 2 ноября, вечером. При тушении пожара спасатели обнаружили тело 60-летнего мужчины.
Чтобы определить причины возгорания в трёхэтажном доме, назначена пожарно-техническая экспертиза, для установления точно причины гибели мужчины будет проведена медицинская экспертиза.
По результатам доследственной проверки будет принято процессуальное решение.
