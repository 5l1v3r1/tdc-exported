
# 侍・稲葉監督「結束力を持って世界一を狙う」　５日にベネズエラと初戦

Published at: **2019-11-04T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/04/0012848994.shtml)

　野球国際大会「プレミア１２」のＢ組開幕を翌日に控えた４日、同組に入る日本、台湾、ベネズエラ、プエルトリコの監督が台中市内で監督会議と共同記者会見に臨んだ。
　席上、司会者から日本の強みを問われた指揮官は「日本チームの強みはやはり守備、機動力。投手を中心に守りを固め、機動力を使った攻撃をしていきます」と力を込めた。前回大会は３位に終わり、優勝だけを目指す今大会。「今回はとにかく全員で結束力を持って、世界一を狙います」と決意表明した。
　会見後は、７日の台湾戦の舞台で、公式練習が行われている「台中インターコンチネンタル球場へ移動し、ナインの動きをチェック。「明日になれば緊張感も出てくると思うが、いいムードで練習ができた」と状態に手応えを示した。
　日本は５日にベネズエラ、６日にプエルトリコ、７日に台湾と戦う。
