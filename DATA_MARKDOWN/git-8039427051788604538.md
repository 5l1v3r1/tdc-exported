
# సీఎస్ ఎల్వీ బదిలీ అన్యాయం: బీజేపీ నేత

Published at: **2019-11-04T18:47:51+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=946277)

ADVT
గుంటూరు: ఏపీ ప్ర‌భుత్వ ప్ర‌ధాన కార్య‌ద‌ర్శి ఎల్వీ సుబ్రహ్మణ్యం బదిలీ అన్యాయమని వేమూరు నియోజకవర్గ బీజేపీ ఇన్ చార్జ్ దర్శనపు శ్రీనివాస్ అన్నారు. ఎల్వీ అక్రమ బదిలీని తీవ్రంగా ఖండిస్తున్నామన్నారు. హిందూ దేవాలయాల్లో అన్య మతస్తులు వద్దు అన్నందుకే ఎల్వీపై కక్ష్య కట్టారని, ఏపీలో ప్రభుత్వ సొమ్ముతో జగన్ ఓ మతాన్ని పెంచి పోషిస్తున్నారని విమర్శించారు. జగన్ పతనం ప్రారంభం అయిందని శ్రీనివాస్ అన్నారు.
