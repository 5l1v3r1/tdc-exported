
# 楽天・岡島“ポスト嶋奪取”へ「みんなよりたくさん練習」

Published at: **2019-11-07T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/07/0012854976.shtml)

　「楽天秋季キャンプ」（６日、倉敷）
　楽天・岡島が倉敷での秋季キャンプで約１時間の特守を行い、“ポスト嶋奪取”を誓った。
　昨秋に外野手から本来の守備位置である捕手に再転向。左肩など手術を受け、今季は１軍出場なしに終わったが、嶋の退団でチャンス到来。「みんな狙っているが自分も負けないよう、みんなよりたくさん練習するしかない」と言い切った。
