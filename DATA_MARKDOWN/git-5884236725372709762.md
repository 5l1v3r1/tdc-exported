
# Часть автодорог Карелии останется на зиму под снегом

Published at: **2019-11-03T16:26:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Petrozavodsk/739430-chast-avtodorog-karelii-ostanetsya-na-zimu-pod-snegom)


        На тех участках дорог, которые будут содержаться «в накате», грейдеры нарезают продольные борозды для создания рельефа, производится обработка от наледи. Скорость на данных автодорогах ограничена до 60 км/ч.
      
По информации Упрдор «Кола», часть дорог на территории Карелии в течение зимы будет содержаться под уплотненным снежным покровом. К ним относятся участки трасс А-121 «Сортавала с 197 км по 215 к населенному пункту Ихала, и с 424 км по 445 км, проходящий через Щеккилу, Кутчезеро и Крошнозеро, А-137 Р-21 «Кола» по направлению Тикша — Ледмозеро — Костомукша — граница с Финляндской Республикой и А-215  - Лодейное Поле — Вытегра — Прокшино — Плесецк — Брин-Наволок, подъезд к  Петрозаводску — с 42 по 130 км.
В течение зимнего периода на данных дорогах образуется плотно укатанный снежный слой, который выравнивают дорожной техникой, для предотвращения скольжения грейдеры нарезают продольные тонкие борозды, создающие рифленую поверхность на проезжей части, регулярно выполняется обработка смесями от наледи. При наступлении оттепели снежный покров с дорог в течение нескольких дней счищается.
Для безопасности на данных автодорогах устанавливается ограничение скорости до 60 км/ч.
