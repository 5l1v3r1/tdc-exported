
# 侍・今永は万全アピール　３回１安打無失点６奪三振　圧巻の５連続Ｋ

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/01/0012840518.shtml)

　「侍ジャパン強化試合、日本－カナダ」（１日、沖縄セルラースタジアム那覇）
　侍ジャパン・今永（ＤｅＮＡ）が３回を投げ、１安打無失点６奪三振と好投した。
　一回、先頭の初球にいきなり１５０キロを計測するなど、立ち上がりから快調。無失点に抑えると、二回は緩急を駆使して三者連続三振に仕留めた。三回も先頭を内角直球で見逃し三振。９番はチェンジアップで空振り三振。二回から５者連続三振の後、１番は平凡な投ゴロで打ち取った。
　５日に初戦を迎える「プレミア１２」では、先発の軸として期待される左腕。大会使用球にも順応し、本番へ万全をアピールした。
