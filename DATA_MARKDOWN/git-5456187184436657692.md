
# БТ покажет сериал о смене власти

Published at: **2019-11-03T14:19:00+00:00**

Author: ****

Original: [charter97.org](https://charter97.org/ru/news/2019/11/3/354249/)

Телеканал «Беларусь-1» анонсировал как «главную телепремьеру осени» показ украинского телесериала «Слуга народа». Все три сезона будут показываться с 11 ноября по одному эпизоду с понедельника по четверг в 20.00, сообщает «Радыё Свабода».
