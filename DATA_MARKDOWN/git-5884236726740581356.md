
# Петрозаводчанка привезла домой медаль с Чемпионата мира по гребле

Published at: **2019-11-05T20:43:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Petrozavodsk/740163-petrozavodchanka-privezla-domoi-medal-s-chempionata-mira-po-greble)


        В Гонконге прошел Чемпионат мира по прибрежной гребле. Спортсменка из Карелии приняла в соревнованиях участие в составе сборной России.
      
Спортсменка из Петрозаводска завоевала медаль на Чемпионате мира по гребле. Об этом сообщает пресс-служба Минспорта Карелии.
С 1 по 3 ноября в Гонконге проходил Чемпионат мира по прибрежной гребле, в котором принимала участие в составе сборной России спортсменка из Лахденпохского района Александра Большакова.  
По итогу соревнований женская команда заняла второе место.
Ранее НЕВСКИЕ НОВОСТИ сообщали о том, что юная спортсменка из Карелии победила в турнире по тхэквондо в Минске.
