
# จีนแต่งตั้งผู้แทนพิเศษด้านกิจการยุโรปคนแรก

Published at: **2019-11-02T03:38:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/foreign/739595)


        รัฐบาลปักกิ่งประกาศการแต่งตั้งตำแหน่งทางการทูตตำแหน่งใหม่ คือผู้แทนพิเศษด้านกิจการยุโรป และมอบหมายให้นายอู๋ หงป๋อ อดีตนักการทูตมากประสบการณ์ด้านตะวันตกรับหน้าที่เป็นคนแรก
      
สำนักข่าวต่างประเทศรายงานจากกรุงปักกิ่ง ประเทศจีน เมื่อวันที่ 2 พ.ย. ว่านายเกิ่ง ส่วง โฆษกกระทรวงการต่างประเทศจีน แถลงเมื่อวันศุกร์ ว่ารัฐบาลปักกิ่งแต่งตั้งนายอู๋ หงป๋อ นักการทูตอาชีพซึ่งมีประสบการณ์ด้านการสร้างเสริมความสัมพันธ์ระหว่างจีนกับตะวันตกมาเป็นเวลานาน ให้ดำรงตำแหน่งผู้แทนพิเศษด้านกิจการยุโรปของรัฐบาลปักกิ่งเป็นคนแรก โดยนายเกิ่งกล่าวถึงเหตุผลที่รัฐบาลปักกิ่งตัดสินใจแต่งตั้งตำแหน่งดังกล่าวเป็นครั้งแรก ว่าเพื่อยกระดับและขยายกรอบความรอบมือระหว่างจีนกับยุโรปให้มีความหลากหลายมากขึ้นในทุกมิติ
