
# Умерла звезда Instagram енот Пампкин (фото)

Published at: **2019-11-03T14:32:00+00:00**

Author: ****

Original: [РИА Мелитополь](https://ria-m.tv/news/170423/umerla_zvezda_instagram_enot_pampkin_(foto).html)


        Умерло самое популярное в мире животное - енот по кличке Памкин. Об этом рассказала его хозяйка, Лора Янг, которая в 2014 году нашла его на улице.
      
"Ее смерть стала для нас настоящим шоком, мы все еще пытаемся с этим смириться", - написала Янг.
Пять лет назад Пампкин получила профиль в Instagram, который со временем стал очень популярным. Кроме публикаций снимков милого животного, с помощью аккаунта проводили различные благотворительные кампании.
