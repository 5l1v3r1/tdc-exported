
# L’Italie rend obligatoire un dispositif anti-oubli de bébé dans les voitures

Published at: **2019-11-07T16:39:00+00:00**

Author: **md**

Original: [www.lenouvelliste.ch](https://www.lenouvelliste.ch/articles/monde/l-italie-rend-obligatoire-un-dispositif-anti-oubli-de-bebe-dans-les-voitures-879941)


        Dès le 7 novembre, les automobilistes italiens devront équiper leur véhicule d’un dispositif anti-oubli de bébé sous peine d’une lourde amende. Le pays a connu plusieurs tragédies lors d’épisodes de canicule.
      
Dès ce jeudi 7 novembre, en Italie, les parents, grands-parents et accompagnateurs d’enfants devront équiper leur véhicule d’un dispositif anti-oubli de bébé. Le gouvernement a rendu obligatoire cet appareil, à installer sur les sièges auto pour les petits de moins de quatre ans, rapporte le quotidien La Stampa. Les personnes qui ne respecteront pas cette loi risqueront une lourde amende et le retrait de cinq points sur leur permis de conduire.
