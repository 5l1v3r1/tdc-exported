
# Le HC Le Locle s’incline aux prolongations

Published at: **2019-11-02T22:26:00+00:00**

Author: ****

Original: [RTN](https://www.rtn.ch/rtn/Sport/Hockey/20191102-Le-HC-Le-Locle-s-incline-aux-prolongations.html)

Le HC Le Locle doit se contenter d’un point contre le HC Moutier. Les hockeyeurs de la Mère-Commune ont été défaits 6-5 après prolongations samedi en 2e ligue. Les Loclois ont pourtant réussi à combler un retard de deux buts par deux fois dans la partie (de 3-1 à 3-3 puis de 5-3 à 5-5). Mais les Prévôtois ont assené le coup fatal après moins de deux minutes dans les prolongations. Cette défaite n’a pas d’incidence sur le classement du HC Le Locle qui reste 6e avec le même nombre de points que le 4e mais une rencontre jouée en plus. /lyg
