
# Белоруска завоевала бронзу на этапе Кубка мира по плаванью

Published at: **2019-11-03T19:19:00+00:00**

Author: ****

Original: [charter97.org](https://charter97.org/ru/news/2019/11/3/354283/)

Белорусская спортсменка Алина Змушко на этапе Кубка мира по плаванию в Казани показала 3-й результат в финале на дистанции 50 метров брассом — 31,18, пишет pressball.by.
Победу одержала Дженнифер Кансейсао из Бразилии — 30,68, второе место заняла итальянка Арианна Касильоне — 31,10.
Вчера Алина Змушко был третьей на дистанции 100 метров брассом (1.08,11).
