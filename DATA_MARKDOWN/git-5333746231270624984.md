
# В четырех банках из топ-10 упали ставки по вкладам

Published at: **2019-11-05T23:36:00+00:00**

Author: **Денис Линов**

Original: [Накануне.ru](https://www.nakanune.ru/news/2019/11/06/22557368/)

Средняя максимальная ставка по депозитам в 10 крупнейших банках России в третьей декаде октября составила 6,36% годовых.
Как следует из материалов Центробанка РФ, по сравнению со второй декадой прошлого месяца показатель опустился на 0,12%. Ставка банков из топ-10 идет вниз семь месяцев подряд, сообщают "Ведомости".
Как сообщалось ранее, в сентябре российские банки приступили к снижению ставок по вкладам.
Это произошло после того, как Сбербанк ухудшил условия по своим депозитам. Только за последние десять дней доходность рублевых депозитов снизили шесть из топ-10 финансовых учреждений по величине портфеля вкладов населения.
Так, Россельхозбанк, который предлагал до 8,1% годовых по вкладам, снизил доходность по ним на 0,2–0,8 п. п., до 7,4% годовых. На столько же упали ставки депозитов для клиентов в Райффайзенбанке. Банк "ФК Открытие" снизил ставки на 0,2 п. п., МКБ – на 0,3–0,4 п. п. по промовкладу и депозиту для пенсионеров.
Напомним, 19 сентября Сбербанк снизил ставки по вкладам в рублях и в долларах.
