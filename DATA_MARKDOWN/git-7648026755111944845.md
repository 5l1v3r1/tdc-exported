
# גבר בן 60 מעד במדרגות נעות במלון בפתח תקוה - ונפצע באורח בינוני

גבר כבן 60 נפצע בינוני ברחוב מוטה גור בפתח תקווה.
מנחם סלבוטיצקי חובש באיחוד הצלה סיפר כי הסיבה לפציעה הייתה מעידה במדרגות נעות שבמתחם מלון שבמקום.
"הפצוע סובל מחבלות ופציעות בראשו ובפלג גופו העליון כתוצאה מנפילה במדרגות נעות", אמר סלבוטיצקי. "לאחר הסיוע הראשוני בזירת האירוע הוא פונה בניידת טיפול נמרץ להמשך קבלת טיפול רפואי בבית החולים בלינסון".
