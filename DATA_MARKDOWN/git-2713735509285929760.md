
# Ибрагима Эльджаркиева уже пытались убить в начале года в Ингушетии

Published at: **2019-11-02T17:43:00+00:00**

Author: **Виктория Якушина**

Original: [Регион Online](https://news-r.ru/news/moscow/379348/)


        Тогда главе центра по борьбе с экстремизмом удалось избежать гибели
      
Москва, 2 ноября. Трагическая смерть главы центра по борьбе с экстремизмом по Республике Ингушетия Ибрагима Эльджаркиева и его 30-летнего брата Абдулы Ахмеда произошла вечером, 2 ноября, в Москве. Мужчины были застрелены неизвестными.
Оказалось, это не первое покушение Эльджаркиева.
- В январе 2019 года, после встречи Эльджаркиева с главой парламента Чечни Магомедом Даудовым, неизвестные догнали авто начальника центра Э близ аэропорта Магас и открыли по нему огонь из автоматов. Трое сидевших на заднем сидении сотрудников полиции получили тяжелые ранения. Позднее один из них умер. Сам Эльджаркиева не пострадал. За рулем авто тогда сидел его брат, теперь уже погибший, - сообщает ряд СМИ.
В настоящее время стрелков разыскивают столичные правоохранители.
Ранее «Регион Online» писал, что в Москве водители учинили пальбу возле крупного ТЦ. Пострадал один человек.
