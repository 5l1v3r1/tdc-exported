
# Глава Карелии поручил Министерству здравоохранения создать региональную «карту болезней»

Published at: **2019-11-06T14:05:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Petrozavodsk/740402-glava-karelii-poruchil-ministerstvu-zdravookhraneniya-sozdat-regionalnuyu-kartu-boleznei)


        Артур Парфенчиков сказал создать карту, которая отражала бы количество больных людей. Собирать сведения будут через районные больницы и ФАПы.
      
В Карелии будет создана «карта болезней». Об этом сообщает издание «Республика».
Артур Парфенчиков в ходе рабочей поездки в Костомукшу заявил, что в данный момент идет работа в слепую. Глава Карелии отметил, что для комплексной работы необходимо представлять ситуацию по болезням в других регионах.
Для эффективной борьбы с болезнями и улучшения здравоохранения республики, губернатор региона поручил Минздраву создать региональную «карту болезней», которая отражала бы количество больных людей. Сбор сведений лежит на районных больницы и ФАПах.
