
# Transportistas se dirigen a Ministerio de Energía y Minas para mostrar descontento

Published at: **2019-11-07T15:56:03+00:00**

Author: ****

Original: [Canal Antigua](https://www.canalantigua.tv/transportistas-ministerio-de-energia-y-minas-protesta-caravana/)

Transportistas ya se movilizan en caravanas como forma de protesta por decisiones de las autoridades.
Rechazan el ajuste del seguro para transporte de productos derivados del petróleo.
Y que la SAT no permite las modificaciones de cabezal a camión para las unidades.
