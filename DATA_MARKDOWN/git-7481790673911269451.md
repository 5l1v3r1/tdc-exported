
# 降雨ないのに「土砂災害の危険」、ＦＭに緊急割り込み放送

Published at: **2019-11-05T00:57:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191104-OYT1T50084/)

　三重県名張市は３日、「土砂災害情報相互通報システム」が誤作動し、地元ＦＭ放送に注意喚起のアナウンスが流れるなどしたと発表した。
　同システムは、県と連携して土砂災害の危険が高まった場合に防災情報を発信。市危機管理室によると、雨が降っていないのに、この日午後１時から約１分間、地元のＦＭラジオ局「ａｄｓ．ＦＭ」に緊急割り込み放送で、「名張市に土砂災害の危険が高まっている」と流れた。市が希望者に配布している防災ラジオも自動的に起動した。
　市は県、津地方気象台と連動した別のシステムに切り替えたため、市のシステムの使用を今年度から停止し、関連機器を間もなく撤去する予定だった。県や機器のメンテナンス会社は休みのため、原因は連休明けに調査するという。
