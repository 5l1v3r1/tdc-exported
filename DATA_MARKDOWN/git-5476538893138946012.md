
# Евгений Королек победил в скретче на первом этапе Кубка мира по велоспорту в Минске

Published at: **2019-11-02T14:00:00+00:00**

Author: **sb.by**

Original: [Гродненская правда](https://grodnonews.by/news/sport/evgeniy_korolek_pobedil_v_skretche_na_pervom_etape_kubka_mira_po_velosportu_v_minske.html)

Евгений Королек стал победителем в скретче на первом этапе Кубка мира по велоспорту, проходящего в Минске. Серебро в этой гонке выиграл испанец Себастьян Мора Верди, бронзу – американец Эрик Янг. Для белорусской сборной золото Евгения Королька стало уже второй медалью, завоеванной на домашнем турнире – ранее бронзовую награду в гонке по очкам завоевала Татьяна Шаракова.
