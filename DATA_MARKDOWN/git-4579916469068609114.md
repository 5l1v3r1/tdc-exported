
# Одна из выпущенных по Израилю ракет попала в жилой дом

Published at: **2019-11-01T21:30:00+00:00**

Author: ****

Original: [Фонтанка.ру](https://www.fontanka.ru/2019/11/02/002/)

Одна из 10 ракет, запущенных в пятницу радикалами из сектора Газа по Израилю, попала в жилой дом на юге еврейского государства. Об этом сообщает ТАСС со ссылкой на пресс-службу Армии обороны Израиля.
"На жилой дом на одну семью упала ракета из Газы. К настоящему моменту по израильским гражданским лицам из сектора Газа запущено 10 ракет, восемь перехвачены системой ПВО "Железный купол", – отмечается в сообщении военных. Государственное радио Израиля Kan информирует, что в результате попадания ракеты по дому в граничащем с сектором Газа районе пострадавших нет, зданию причинен ущерб.
В предыдущий раз палестинские радикалы из анклава выпустили ракету по Израилю 31 октября.
