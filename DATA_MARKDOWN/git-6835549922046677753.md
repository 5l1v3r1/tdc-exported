
# Айлендерс обыграли Тампу, Василевский сделал 21 сэйв

Author: ****

Original: [Allhockey.ru](https://allhockey.ru/news/show/368718-Ajlenders_obygrali_Tampu_Vasilevskij_sdelal_21_sejv)

"Айлендерс" переиграл "Тампа-Бэй" в матче регулярного чемпионата НХЛ (5:2).
