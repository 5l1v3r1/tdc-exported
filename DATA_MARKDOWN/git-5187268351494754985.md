
# Muitos falsos positivos no Windows Defender? É hora de adicionar exclusões

Published at: **2019-11-03T09:00:10+00:00**

Author: **Pedro Simões**

Original: [Pplware](https://pplware.sapo.pt/microsoft/windows/falsos-positivos-windows-defender-exclusoes/)

Apesar de não ser a escolha de muitos, o Windows Defender consegue dar já a segurança necessária aos utilizadores do Windows. Esta foi uma aposta da Microsoft que foi claramente ganha e que está cada vez melhor.
Apesar de toda a sua inteligência, esta solução tem ainda algumas melhorias e não são raros os falsos positivos. Para ajudar neste ponto podemos criar exclusões, que assim não vão gerar alarmes. Veja como pode acabar com os falsos positivos no Windows Defender.
Os antivírus são principalmente essenciais no Windows e em outros sistemas. São a certeza de proteção e a salvaguarda contra problemas e ameaças de segurança. Com o Windows Defender a Microsoft conseguiu estabelecer-se e mostrar que também tem uma palavra nesta área.
Não é anormal a existência de falsos positivos nos antivírus. Estes são aplicações ou ficheiros que são erradamente identificados como potenciais perigos. Para os excluir no Windows Defender precisam apenas de aceder a esta app e navegar para o sítio certo.
Devem inicialmente escolher a opção Proteção contra vírus e ameaças. Ai dentro devem depois escolher a opção Gerir definições, na área com o mesmo nome. Por fim, devem escolher Adicionar ou remover exclusões, na área dedicada a este ponto.
Disponíveis estão 4 tipos de exclusões possíveis. Devem recordar-se que ao escolher adicionar uma destas exclusões, estão a mantê-los fora do alcance do Windows Defender. Caso exista um problema de segurança, não haverá nenhuma proteção.
As primeiras duas foram-se em ficheiros ou pastas, devendo o utilizador escolher quais quer isolar do Windows Defender. Deverá escolher cada uma das que quer manter fora desta proteção. O processo poderá ter de ser repetido para abranger todos.
Outra possibilidade de exclusão está igualmente no tipo de ficheiro ou no processar. Estes isolam todos os tipos de ficheiros indicados ou os processos de aplicações que forem registados como exclusão.
O Windows Defender só necessita que indiquem a extensão dos ficheiros a manter excluídos ou o nome dos processos a serem também isolados e colocados fora do controlo do antivírus do Windows.
Esta é a forma simples e direta para controlarem os ficheiros, pastas ou processos que o Windows Defender irá ignorar. É assim que devem tratar os falsos positivos, com o cuidado de garantir que estes não representam nenhum perigo ou problema de segurança.
