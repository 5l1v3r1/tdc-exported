
# เวเนซุเอลา-เอลซัลวาดอร์เนรเทศนักการทูต

Published at: **2019-11-04T00:13:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/foreign/739862)


        เวเนซุเอลาและเอลซัลวาดอร์ต่างเนรเทศนักการทูตของอีกฝ่าย ตอบโต้กันหลังรัฐบาลเอลซัลวาดอร์หันไปยอมรับนายฮวน กวยโด ในฐานะ "ผู้นำที่ชอบธรรม" ของรัฐบาลการากัส
      
สำนักข่าวต่างประเทศรายงานจากกรุงการากัส ประเทศเวเนซุเอลา เมื่อวันที่ 3 พ.ย. ว่ากระทรวงการต่างประเทศเวเนซุเอลาออกแถลงการณ์เมื่อวันอาทิตย์ ว่านักการทูตทั้งหมดของเอลซัลวาดอร์ถือเป็น "บุคคลไม่พึงปรารถนา" และต้องเดินทางออกนอกประเทศภายใน 48 ชั่วโมง พร้อมทั้งวิจารณ์ประธานาธิบดีนายิบ บูเคเล ผู้นำเอลซัลวาดอร์ว่า "เป็นเรื่องน่าเศร้าที่บูเคเลประกาศตัวเป็นทาสรับใช้ของสหรัฐอย่างเป็นทางการ" และยินยอมพร้อมใจเป็นส่วนหนึ่งของแผนยุทธ์ศาสตร์อันล้มเหลวของรัฐบาลวอชิงตัน ในการรุกรานทางการเมืองและเศรษฐกิจต่อเวเนซุเอลา
