
# В России запустили приложение для электронного оформления ДТП. Пока оно работает в пяти регионах

Published at: **2019-11-01T05:45:28+00:00**

Author: **Коммерсант**

Original: [Meduza](https://meduza.io/news/2019/11/01/v-rossii-zapustili-prilozhenie-dlya-elektronnogo-oformleniya-dtp-poka-ono-rabotaet-v-pyati-regionah)

Российский союз автостраховщиков (РСА) запустил мобильное приложение «Помощник ОСАГО» (iOS, Android), которое позволяет оформить ДТП в электронном виде. Пилотный проект заработал с 1 ноября в пяти регионах: Москве, Московской области, Санкт-Петербурге, Ленинградской области и Татарстане.
Заполнить извещение об аварии через приложение можно при соблюдении нескольких условий:
«Пока пользователь мобильного приложения вводит данные для оформления извещения, второй участник, ожидающий получения черновика извещения от ЕПГУ, изображает схему ДТП на листе бумаги», — говорится в справочном разделе РСА. Эту схему затем надо будет сфотографировать. Само ДТП также надо будет зафиксировать с помощью фотографий, тогда лимит страховых выплат составит 400 тысяч рублей (без фото — 100 тысяч).
