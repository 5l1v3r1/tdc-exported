
# ผู้นำเกาหลีใต้-ญี่ปุ่นตกลงหารือเคลียร์ปัญหาการค้า

Published at: **2019-11-04T04:30:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/foreign/739890)


        ประธานาธิบดีเกาหลีใต้และนายกรัฐมนตรีญี่ปุ่นหารือกันนอกรอบเวทีอาเซียนที่ไทย ตกลงให้มีการเจรจาเพื่อคลี่คลายความขัดแย้งทางการค้า ซึ่งมีชนวนมาจากประเด็น "แรงงานทาส" สมัยสงคราม
      
สำนักข่าวต่างประเทศรายงานจากกรุงเทพมหานคร เมื่อวันที่ 4 พ.ย. ว่ากระทรวงการต่างประเทศเกาหลีใต้และกระทรวงการต่างประเทศญี่ปุ่นออกแถงการณ์ในเวลาไล่เลี่ยกัน เมื่อวันจันทร์ ว่าประธานาธิบดีมุน แจ-อิน และนายกรัฐมนตรีชินโซ อาเบะ พบหารือกันอย่างไม่เป็นทางการ นอกรอบการประชุมสุดยอดสมาคมประชาชาติแห่งเอเชียตะวันออกเฉียงใต้ ( อาเซียน ) ครั้งที่ 35 ซึ่งไทยทำหน้าที่ประธานในปีนี้ ว่าผู้นำทั้งสองประเทศเห็นชอบร่วมกันในหลักการ ว่าประเด็นความขัดแย้งใดก็ตามที่เป็นระดับทวิภาคี ต้องได้รับการแก้ไขผ่านการเจรจาปรับความเข้าใจระหว่างสองฝ่าย
