
# Рекламодатели Яндекса получили доступ к TikTok

Published at: **2019-11-07T14:09:00+00:00**

Author: **Марина Ибушева**

Original: [Лайкни](https://www.likeni.ru/events/reklamodateli-yandeksa-poluchili-dostup-k-tiktok/)

Рекламодатели Яндекса получили возможность выйти на новую молодежную аудиторию в TikTok. Сервис Яндекс.Директ теперь предлагает размещение рекламных блоков для показа в видеоленте платформы. 
Видеоролики для TikTok создаются автоматически из картинок и текста с описанием. Затем на рекламный блок автоматически накладывается подходящая аудиодорожка, элементы взаимодействия для пользователей. Далее реклама адаптируется под вертикальный формат экрана смартфона.  Размещение в TikTok доступно в рамках стандартного аукциона Рекламной сети Яндекса в Директе. Рекламодатели могут выбрать таргетинг по поисковым интересам, социально-демографическим характеристикам, географии и другим параметрам.  Алгоритмы Яндекса позволят не переплачивать за нецелевые контакты, а показывать рекламу только тем, кому она действительно будет полезна. Напомним, по данным Sensor Tower, TikTok стал самым скачиваемым приложением в категории социальных сетей в сентябре. Также видеоплатформа обогнала Facebook по числу пользователей от 13 до 16 лет в США. Источник: пресс-релиз Яндекса
