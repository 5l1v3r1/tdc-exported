
# సిరియాలో మిలిటెంట్ల దాడి..ఒకరి మృతి,పలువురికి గాయాలు

Published at: **2019-11-07T05:51:49+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=948635)

ADVT
అలెప్పో (సిరియా): సిరియా దేశంలో తాజాగా మిలిటెంట్లు జరిపిన కాల్పుల్లో ఒకరు మరణించగా, పలువురికి గాయాలయ్యాయి. సిరియా దేశంలోని అలెప్పో నగరంలోని సిద్ధిక్ మసీదు వద్ద మిలిటెంట్లు కాల్పులు జరిపారు. ఈ కాల్పుల్లో ఒకరు మరణించగా, పలువురికి గాయాలయ్యాయి. గాయపడిన వారిని ఆసుపత్రికి తరలించారు. సిరియా మిలటరీ అధికారులు స్పందించి మిలిటెంట్ల దాడిని తిప్పి కొట్టారు. మిలిటెంట్ల తాజా దాడులతో సిరియాలో భయాందోళనలు నెలకొన్నాయి.
