
# യുഎപിഎ പ്രതികളുടെ ജയിൽ മാറ്റം ആവശ്യമില്ല: ഋഷിരാജ് സിങ്

Published at: **2019-11-07T08:16:00+00:00**

Author: **മനോരമ ലേഖകൻ**

Original: [ManoramaOnline](https://www.manoramaonline.com/news/latest-news/2019/11/07/uapa-accused-in-kozhikode-district-jail.html?utm_source=dlvr.it&utm_medium=twitter)

കോഴിക്കോട്∙ യുഎപിഎ ചുമത്തപ്പെട്ട വിദ്യാർഥികൾ കോഴിക്കോട് ജില്ലാ ജയിലിൽ തുടരും. ജയിൽ മാറ്റണമെന്ന സൂപ്രണ്ടിന്റെ ആവശ്യം ജയിൽ മേധാവി ഋഷിരാജ് സിങ് തള്ളി. സുരക്ഷാ പ്രശ്നങ്ങളില്ലെന്ന് അദ്ദേഹം വ്യക്തമാക്കി. ഇരുവരെയും വിയ്യൂർ ഹൈടെക്ക് ജയിലിലേക്ക് മാറ്റണമെന്നായിരുന്നു ആവശ്യം. 
അറസ്റ്റിലായ  സിപിഎം പ്രവർത്തകരെ ഇന്ന് പൊലീസ് കസ്റ്റഡിയിൽ ആവശ്യപെട്ടേക്കും. ഇവരുടെ ജാമ്യ ഹർജി ഇന്നലെ കോടതി തള്ളിയിരുന്നു.  ഡിജിറ്റൽ തെളിവുകളുടെ ഫോറൻസിക് പരിശോധന ഫലം വന്നാൽ കൂടുതൽ ചോദ്യം ചെയ്യാൻ കസ്റ്റഡി അപേക്ഷ നൽകുമെന്ന് അന്വേഷണസംഘം അറിയിച്ചിരുന്നു. കേസിൽ എഫ്ഐആർ റദ്ദാക്കണമെന്നാവശ്യപെട്ട് പ്രതിഭാഗം ഹൈക്കോടതിയെ സമീപിക്കും. പ്രതികൾക്കൊപ്പം പന്തീരാങ്കാവിൽ നിന്നും അറസ്റ്റിലായ മൂന്നാമനു വേണ്ടിയുള്ള അന്വേഷണം തുടരുകയാണ്.
