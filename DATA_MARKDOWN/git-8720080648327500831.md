
# В Самаре закроют ещё один рынок

Published at: **2019-11-06T13:53:17+00:00**

Author: **Максим Фёдоров**

Original: [Другой город](http://drugoigorod.ru/two-rinok-off/)

Компания «Самарские распределительные сети» потребовала закрыть рынок на пересечении Ново-Вокзальной и Московского шоссе.
По словам энергетиков, киоски стоят слишком близко к опоре ЛЭП:
«Собственники несколько раз получали от нас предписания. Из-за несоблюдения правил безопасности загорелся один из киосков. Если бы огонь распространился дальше, то часть города могла остаться без электричества».
После пожара мэрия расторгла договор с владельцами торговой точки, но киоски до сих пор не убрали.
Напомним, на этой неделе полностью снесли рынок на пересечении Авроры и Аэродромной.
