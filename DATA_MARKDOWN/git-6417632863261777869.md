
# Шюррле “Арсенал”га қарши баҳсни ўтказиб юборади

Published at: **2019-11-04T17:45:50+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/shyurrle-arsenalga-qarshi-bahsni-otkazib-yuboradi)

Футболчи учрашув қайдномасидан ўрин олмади. Илгарироқ у индивидуал тарзда машғулотларни бошлаганди.
“Спорт –Экспресс” нашрининг маълумотларига кўра, Шюррле яхши спорт формасида бўлмаганлиги учун бироз унга бироз танаффус беришга қарор қилинган. 
Манба : Спорт Экспресс
