
# Хэмилтон в шестой раз стал чемпионом Формулы-1

Published at: **2019-11-05T08:46:00+00:00**

Author: ****

Original: [Главное™](https://glavnoe.ua/news/n357008045-hemilton-v-shestoj-raz-stal-chempionom-formuly-1)

Британский автогонщик Льюис Хэмилтон команды «Мерседеса» в шестой раз в карьере стал чемпионом «Формулы-1», сообщает theuk.one.
Хэмилтон, завоевавший титул чемпиона мира в третий год подряд, обогнал по количеству титулов аргентинского автогонщика и пятикратного чемпиона «Формулы-1» Хуана Мануэля Фанхио по прозвищу Маэстро.
Теперь британец уступает только семикратному чемпиону немцу Михаэлю Шумахеру.
Пять из шести титулов чемпиона мира (2014, 2015, 2017, 2018 и 2019 года) Хэмилтон получил в команде «Мерседес» в «гибридную эру».
В 2008 году он выиграл чемпионат, являясь пилотом «Макларена» (2008).
