
# Unbekannte setzen Katze in Kühltasche aus

Published at: **2019-11-04T09:27:21+00:00**

Author: ****

Original: [Mitteldeutsche Zeitung](https://www.mz-web.de/leipzig/mitten-auf-einer-wiese-unbekannte-setzen-katze-in-kuehltasche-aus-33409660)

Leipzig - In Leipzig haben Unbekannte eine kleine Katze in einer Kühltasche eingeschlossen und ausgesetzt. Das Tier wurde am Sonntag von einem Passanten auf einer Brache nahe der B-Bahnhaltestelle MDR gefunden. Wie dieser via Facebook berichtet, übergab er die Fundkatze dem Veterinäramt, welches die kleine rote Katze schließlich ins Leipziger Tierheim brachte. Es soll sich um einen ungechipten Kater handeln, ergänzten andere User.
Nun wird nach dem bisherigen Besitzer der Katze gesucht. (mz)
