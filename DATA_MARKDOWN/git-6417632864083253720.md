
# Достон Ибрагимов: “Олимпия терма жамоасида Европа футболи ҳавоси ҳукмрон”

Published at: **2019-11-07T08:05:00+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/doston-ibragimov-olimpiya-terma-jamoasida-evropa-futboli-havosi-hukmron)

ТТЙМИ стадионидаги навбатдаги машғулотлар бошланишидан олдин Достон Ибрагимовни фикрлари билан қизиқдик.- Мана бир неч йиғиндан бери ОТЖ таркибида машғулотларда қатнашяпсиз. Мазкур вақт оралиғида ўзингиз учун нималарни кашф қилдингиз?- Олимпия терма жамоасининг мураббийи ўз вақтида Европанинг номдор клубларида тўп тепган Любинко Друлович саналади. Табиийки ундан кўп янги нарсаларни ўрганса бўлади. Йиғин қувноқ кайфиятда ўтяпти. Дубайдаги турнирда ғолиблик учун ҳаракат қиламиз.- Айтингчи жамоада тўлиқлигича Европа футболи ҳавоси сезилиб турадими?- Ҳа, шундай деса ҳам бўлади.
- Масалан, хорижда футболчилар ўз устиларида тинмай ишлайдилар. Машғулотларга барвақт келиб қўшимча машқларни бажаришади. Миллий терма жамоада ёки олимпия термасида бунга қай даражада гувоҳ бўлдингиз?- Тўғри. Бизда ҳам шундай футболчилар етарлича. Албатта айтганингиздек эринчоқ футболчилар йўқ эмас. Мен ўйлайман ҳаммаси одамнинг ўзига боғлиқ. Инсонда хоҳиш бўлмаса унга ҳеч ким ёрдам бермайди.- Тўғри овқатланиш, кун тартибига риоя қилиш футболчининг ўйнига нечоғлик таъсир ўтказади?- Катта аҳамияти бор. Профессионал футболчининг кун тартиби бўлади. Бошқа спорт турларида ҳам овқатланиш тартиби муҳим роль ўйнайди.Абдулазиз суҳбатлашди
