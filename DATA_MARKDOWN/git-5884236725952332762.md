
# В Колпино «геймеры» похитили диски для приставок уже из нескольких магазинов

Published at: **2019-11-01T15:52:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/738786-v-kolpino-geimery-pokhitili-diski-dlya-pristavok-uzhe-iz-neskolkikh-magazinov)


        Молодые люди промышляют кражами игровых дисков для приставок в магазинах цифровой техники. Преступники попали на камеры, установленные в магазинах.
      
В Колпинском районе Санкт-Петербурга в правоохранительные органы поступила оперативная информация о преступной деятельности целой группы молодежи. Как стало известно НЕВСКИМ НОВОСТЯМ, несколько парней устраивают прогулки по местным магазинам и чаще всего посещают салоны электронной техники. Продавцы после этого не досчитываются выставленных на прилавках игровых дисков для различных приставок. Группу «игроманов» в одном из магазинов сняли камеры видеонаблюдения.
