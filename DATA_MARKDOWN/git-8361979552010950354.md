
# Світоліна вразила новими ефектними ударами на Підсумковому турнірі - відео

Published at: **2019-11-01T20:26:00+00:00**

Author: ****

Original: [НВ](https://nv.ua/ukr/sport/football/efektni-udari-svitolinoji-na-pidsumkovomu-turniri-video-50051204.html)
