
# นายกรัฐมนตรีรัสเซียร่วมประชุมอาเซียนที่ไทย

Published at: **2019-11-01T01:04:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/foreign/739409)


        นายดมิทรี เมดเวเดฟ นายกรัฐมนตรีรัสเซีย เตรียมเยือนไทยในช่วงสุดสัปดาห์นี้ เพื่อเข้าร่วมการประชุมสุดยอดอาเซียนที่กรุงเทพฯ ซึ่งหนึ่งในรายการหารือย่อยที่สำคัญรวมถึงการประชุมเอเชียตะวันออก
      
สำนักข่าวต่างประเทศรายงานจากกรุงมอสโก ประเทศรัสเซีย เมื่อวันที่ 1 พ.ย. ว่าทำเนียบนายกรัฐมนตรีรัสเซียออกแถลงการณ์เมื่อวันพฤหัสบดี ว่านายกรัฐมนตรีดมิทรี เมดเวเดฟ จะทำหน้าที่ผู้แทนของประธานาธิบดีวลาดิเมียร์ ปูติน ในการทางเยือนกรุงเทพมหานครระหว่างวันที่ 3 ถึง 4 พ.ย. นี้ เพื่อเข้าร่วมการประชุมเอเชียตะวันออก ( อีเอเอส ) ครั้งที่ 14 ซึ่งเป็นหนึ่งในรายการหารือกลุ่มย่อยที่สำคัญของการประชุมสุดยอดสมาคมประชาชาติแห่งเอเชียตะวันออกเฉียงใต้ ( อาเซียน ) ครั้งที่ 35 ซึ่งไทยเป็นประธานในปีนี้
