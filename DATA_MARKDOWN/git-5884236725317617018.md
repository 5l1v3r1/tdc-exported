
# Спасатели спустя четыре часа потушили горящий склад в Жуковском

Published at: **2019-11-03T10:10:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/739278-spasateli-spustya-chetyre-chasa-potushili-goryashii-sklad-v-zhukovskom)


        В Московской области рано утром 3 октября загорелся склад. Для локализации пожара сотрудникам МЧС потребовалось чуть больше час. А для ликвидации открытого горения — четыре часа. О пострадавших информации нет.
      
Спасатели спустя четыре часа потушили горящий склад в городском округе Жуковский в Подмосковье. Об этом сообщила пресс-служба ГУ МЧС России по региону.
