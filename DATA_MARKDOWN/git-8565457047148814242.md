
# Jueves con cielo algo nublado y máxima de 18°C

Published at: **2019-11-07T12:17:00+00:00**

Author: ****

Original: [El Patagónico](https://www.elpatagonico.com/jueves-cielo-algo-nublado-y-maxima-18c-n5064239)


        Tras varias jornadas ventosas en la zona, hoy Comodoro Rivadavia amaneció con cielo mayormente despejado y se prevé una temperatura máxima de 18°C. ¿Cómo seguirán los días?
      
Según lo anuncia el Servicio Meteorológico Nacional, mañana viernes el cielo permanecerá parcialmente nublado y la temperatura oscilará entre 7°C y 12°C. Se prevén vientos leves de direcciones variables.
Para el sábado se indica cielo parcialmente nublado, mínima de 9°C y máxima de 21°C. Se registrarán vientos leves del sureste.
El domingo será una jornada con cielo mayormente nublado, mínima de 19°C y máxima de 24°C. Habrá vientos leves del noreste.
