
# Ушел из жизни актер из «Стартрека»

Published at: **2019-11-07T14:59:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/kultura/ushel_iz_zhizni_akter_iz_startreka/)

В возрасте 88 лет ушел из жизни американский актер Уильям Уинтерсоул, передает Deadline.
Причиной смерти издание называет онкологическое заболевание.За свою долгую карьеру Уильям Уинтерсоул исполнил роли в нескольких популярных сериалах: «Стартрек», «Коломбо», «Воздушный волк», «Улицы Сан-Франциско», «Миссия невыполнима», «Дымок из ствола», других.
