
# ലഘുലേഖ കയ്യില്‍ വച്ചാല്‍ മാവോയിസ്റ്റ് ആകില്ല: സംസ്ഥാന യുഎപിഎ സമിതി അധ്യക്ഷന്‍

Published at: **2019-11-03T14:45:00+00:00**

Author: **മനോരമ ലേഖകൻ**

Original: [ManoramaOnline](https://www.manoramaonline.com/news/latest-news/2019/11/03/state-uapa-committee-president-on-uapa-case-against-students.html)

കൊച്ചി∙ വ്യക്തമായ തെളിവില്ലാതെ യുഎപിഎ ചുമത്താനാകില്ലെന്നു സംസ്ഥാന യുഎപിഎ സമിതി അധ്യക്ഷന്‍ ജസ്റ്റിസ് പി.എസ്.ഗോപിനാഥന്‍. ലഘുലേഖ കൈവശം വച്ചാല്‍ മാവോയിസ്റ്റാകില്ല. പന്തീരാങ്കാവ് കേസില്‍ ഇപ്പോള്‍ എഫ്ഐആര്‍ മാത്രമേയുള്ളു. അതില്‍ പറയുന്ന വകുപ്പുകള്‍ നിലനില്‍ക്കണമെങ്കില്‍ ശക്തമായ തെളിവുവേണം. തെളിവില്ലാതെ പ്രോസിക്യൂഷന്‍ അനുമതി നല്‍കാനാകില്ലെന്നും ജസ്റ്റിസ് ഗോപിനാഥന്‍ കൊച്ചിയില്‍ പറഞ്ഞു.
അതേസമയം യുഎപിഎ കരിനിയമമെന്ന് ആവർത്തിച്ച് സിപിഎം നേതാക്കളും രംഗത്തുവന്നു. എന്നാൽ യുഎപിഎ ചുമത്തപ്പെട്ട സിപിഎം പ്രവര്‍ത്തകര്‍ നഗരങ്ങള്‍ കേന്ദ്രീകരിച്ചു പ്രവര്‍ത്തിക്കുന്ന മാവോയിസ്റ്റ് സംഘത്തിന്റെ ഭാഗമെന്നാണ് പൊലീസിന്റെ വാദം. നഗരങ്ങളില്‍ വിവരശേഖരണവും ആശയപ്രചാരണവുമാണ് ഇവര്‍ നടത്തിയിരുന്നതെന്നും അന്വേഷണ ഉദ്യോഗസ്ഥര്‍ ആരോപിച്ചു. കേസില്‍ യുഎപിഎ നിലനില്‍ക്കുമോയെന്ന് അന്വേഷണത്തിനുശേഷം കോടതിയെ അറിയിക്കുമെന്ന് ‍ഡ‍ിജിപി വ്യക്തമാക്കി.
മാവോയിസ്റ്റ് അനുകൂല ലഘുലേഖകള്‍ സൂക്ഷിച്ചെന്ന പേരിലാണു കോഴിക്കോട് തിരുവണ്ണൂർ സ്വദേശി അലൻ ഷുഹൈബ്, മൂർക്കനാട് സ്വദേശി താഹ ഫസൽ എന്നിവരെ പൊലീസ് അറസ്റ്റ് ചെയ്തത്. ഇരുവർക്കുമെതിരെ യുഎപിഎ ചുമത്തിയായിരുന്നു പൊലീസ് നടപടി. സിപിഎം പിബി അംഗം എം.എ. ബേബിയും ജില്ലാ നേതൃത്വവും സിപിഐയും പൊലീസിനെതിരെ വിമർശനവുമായി രംഗത്തെത്തി.
