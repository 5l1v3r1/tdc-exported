
# Σύγκρουση τρένων στον σταθμό Ρέντη

Published at: **2019-11-04T14:51:30+00:00**

Author: **ΤοΒΗΜΑ Team**

Original: [Το Βήμα Online](https://www.tovima.gr/2019/11/04/society/sygkrousi-trenon-ston-stathmo-renti/)
