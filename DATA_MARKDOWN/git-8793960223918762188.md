
# Pesawat yang Ditumpangi PM Jepang Alami Kebakaran

Published at: **2019-11-03T18:11:36+00:00**

Author: **Rakhmad Hidayatulloh Permana**

Original: [detiknews](https://news.detik.com/internasional/d-4770519/pesawat-yang-ditumpangi-pm-jepang-alami-kebakaran)

Tokyo - Pesawat yang ditumpangi Perdana Menteri (PM) Jepang Shinzo Abe mengalami kebakaran kecil. Namun, tak ada korban dalam insiden ini karena petugas sigap dalam memadamkan sumber api.
Sebagaimana dilansir dari AFP, insiden ini terjadi pada hari Minggu (3/11/2019). Saat itu, PM Jepang itu hendak terbang ke Bangkok dari Tokyo menggunakan Boeing 777-300ER. Shinzo direncanakan menghadiri pertemuan di KTT Perhimpunan Bangsa-Bangsa Asia Tenggara (ASEAN).
Sekitar satu jam usai lepas landas, sebuah pengumuman dalam penerbangan mengatakan bahwa oven di pesawat telah memicu kobaran api kecil. Keterangan ini disampaikan oleh media nasional Jepang, Asahi Shimbun.
Sepuluh menit kemudian, pengumuman lain mengatakan api telah padam. "Tak ada asap atau bau terdeteksi," tulis media itu dalam sebuah artikelnya.
