
# Россия может лишить США всех авианосцев сразу

Published at: **2019-11-07T11:35:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/novosti_mira/rossiya_mozhet_lishit_ssha_vsekh_avianostsev_srazu/)

Заместитель президента Российской академии ракетных и артиллерийских наук по информационной политике Константин Сивков в своей статье для «Военно-промышленного курьера» рассказал, как нанести непоправимый ущерб сразу всему авианосному флоту США.
По его словам, добиться этого мог бы удар по верфи в Ньюпорт-Ньюсе в американском штате Вирджиния, так как только она может строить и ремонтировать атомные авианосцы.
Он отметил, что на верфи в Ньюпорте-Ньюсе находятся три стапеля, способные принять такие корабли, причём два из них используются для ремонта и только один – для постройки атомных авианосцев.
