
# Revelan dos nuevos personajes para el próximo Guilty Gear

Published at: **2019-11-04T19:31:16+00:00**

Author: **Rodolfo León**

Original: [Atomix](https://atomix.vg/revelan-dos-nuevos-personajes-para-el-proximo-guilty-gear/)

El nuevo Guilty Gear se está viendo como toda una locura. Este próximo juego de peleas debutó un espectacular tráiler con el regreso de dos personajes: Chipp y Potemkin. Además de los previamente revelados Sol, Ky, May, y Axl, estos seis combatientes conformarán el roster para el demo jugable de este título que debutará en ARCREVO America 2019.
