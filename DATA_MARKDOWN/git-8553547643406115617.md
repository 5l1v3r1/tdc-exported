
# Veeam представила новые решения резервного копирования для облачных платформ Microsoft

Published at: **2019-11-07T15:35:00+00:00**

Author: ****

Original: [Компьютерное Обозрение](https://ko.com.ua/veeam_predstavila_novye_resheniya_rezervnogo_kopirovaniya_dlya_oblachnyh_platform_microsoft_130786)

Компания Veeam Software на конференции Microsoft Ignite 2019 в Орландо, штат Флорида, объявила о выпуске публичной бета-версии нового решения Veeam Backup для Microsoft Office 365 v4.
Также компания представила участникам конференции новое облачное решение для защиты данных для рабочих нагрузок Microsoft Azure, Veeam Backup for Microsoft Azure.
Как отмечается, новые решения поддерживают облачную стратегию компании и предоставляют пользователям Microsoft дополнительную масштабируемость, защиту и улучшенный контроль облачных данных.
Хотя Microsoft гарантирует пользователям Office 365 безотказную работу инфраструктуры, но ответственность за резервное копирование и управление данными лежит на самих заказчиках. Новое решение Veeam Backup для Microsoft Office 365 v4 включает в себя прямую интеграцию с хранилищем BLOB-объектов Azure, предлагая компаниям, желающим хранить свои данные Office 365 в Azure, экономичное, масштабируемое и безопасное решение. В релиз также входят дополнительные улучшения производительности для резервного копирования данных Microsoft SharePoint и OneDrive for Business.
Сообщается, что возможности, заложенные в новую версию Veeam Backup для Microsoft Azure, обеспечивают максимальную простоту использования и развертывание «под ключ» через Microsoft Azure Marketplace, а также восстановление на уровне файлов для снимков и резервных копий Veeam. Впервые в отрасли решение включает в себя инструмент предварительной оценки стоимости использования облачных ресурсов, что дает заказчикам возможности более строгого управления расходами. Интеграция с Veeam Backup & Replication позволяет лучше контролировать облачные данные благодаря защите и управлению резервными копиями Azure вместе с облачными, виртуальными и физическими данными.
«Демонстрируя рост на 114% по сравнению с предыдущим годом, Veeam Backup для Microsoft Office 365 продолжает оставаться самым быстрорастущим продуктом в истории Veeam, — заявил Ратмир Тимашев, сооснователь и исполнительный вице-президент по продажам и маркетингу Veeam Software. — Сегодня сохраняется необходимость в SaaS-решениях для резервного копирования и защиты важнейшей информации, как и раньше, когда заказчики выполняли резервное копирование и защиту данных локально. Решение Veeam Backup для Office 365 призвано удовлетворить эту потребность. Наблюдая стремительный рост популярности Azure среди наших клиентов, мы рады представить собственное решение для резервного копирования, разработанное в сотрудничестве с Microsoft специально для рабочих нагрузок Azure».
