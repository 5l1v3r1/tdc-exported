
# Facebook prueba sistema de reconocimiento facial

Published at: **2019-11-07T09:10:00+00:00**

Author: **ANSA**

Original: [cubasi.cu](http://cubasi.cu/cubasi-noticias-cuba-mundo-ultima-hora/item/100533-facebook-prueba-sistema-de-reconocimiento-facial)


        Facebook está probando un sistema de verificación del perfil de los usuarios con una especie de video-selfie, que implica el reconocimiento facial.
      
Lo informó la desarrolladora Jane Manchun Wong, quien mostró el funcionamiento de este nuevo instrumento gracias a la posibilidad de compartir diversas pantallas.    El sistema podría servir para contrarrestar el fenómeno de las "cuentas falsas".    Para lo que se conoce como imagen animada, para poder determinar la identidad los usuarios deben registrar un breve video-selfie, moviendo el rostro en diversas direcciones.    Una vez completado el procedimiento, Facebook mostraría un mensaje de aviso donde es especificado que nadie tendrá la posibilidad de ver el video precedentemente registrado.    El sistema de reconocimiento facial no es nuevo para la compañía de Mark Zuckerberg. En 2017 la plataforma confirmó que había comenzado a probar una funcionalidad capaz de verificar la identidad de los usuarios a través de la utilización de las fotocámaras.
