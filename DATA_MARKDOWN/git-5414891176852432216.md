
# Сегодня баскетболисты Каменского проведут игру на выезде

Published at: **2019-11-07T07:30:00+00:00**

Author: ****

Original: [dndz.tv](http://dndz.tv/article-30057.html)

07.11.19 БК «Прометей» города Каменское (г. Днепродзержинск) проведут встречу с БК «Запорожье».Вторая подряд выездная игра для каменчан пройдет вне своего поля.Поединок «Прометей»/«Запорожье» стартует в 18:00.В дебютной игре Суперлиги «Прометей» выиграл домашнюю встречу с сегодняшним соперником (28.09.19).
