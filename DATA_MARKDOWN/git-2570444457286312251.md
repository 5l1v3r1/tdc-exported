
# Больницу, где пациенты травят тараканов, продезинфицируют

Published at: **2019-11-05T12:30:00+00:00**

Author: ****

Original: [Саратов 24](https://saratov24.tv/news/bolnitsu-gde-patsienty-travyat-tarakanov-prodezinfitsiruyut/)

Один из пациентов лечебного учреждения сообщил о том, что больные сами намерены травить тараканов, которые ползают повсюду: по стенам и полу, по раковине. Фотографии он отправил в средства массовой информации. Как только публикации появились в СМИ, чиновники решили исправить ситуацию. «Министерство здравоохранения поручило руководителю данного лечебного учреждения провести внеплановую дезинфекцию", - сообщили в пресс-службе ведомства.
