
# Politiet mener denne skiltingen er lite forutsigbar

Published at: **2019-11-06T11:24:14+00:00**

Author: **Martin Andersen**

Original: [adressa.no](https://www.adressa.no/pluss/nyheter/2019/11/06/Politiet-mener-denne-skiltingen-er-lite-forutsigbar-20279350.ece)

Peker på to mulige konfliktområder i kryss der Olav Tryggvasons gate møter Søndre gate.
Adresseavisen fortalte tirsdag om kollektivfelt-skiltingen i Olav Tryggvasons gate i Trondheim. Der ble det nemlig i sommer satt opp feil type skilt.
