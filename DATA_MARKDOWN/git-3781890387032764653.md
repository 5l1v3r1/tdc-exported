
# ค่ายยัน'คาราบาว'วงไม่แตก 'เทียรี่'ซัด'แอ๊ด'แค่แหย่เล่น!

Published at: **2019-11-05T04:40:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/entertainment/740054)


        เดลินิวส์ออนไลน์ โทรถามความจริงให้แล้ว หลังดราม่าหนักปมคาราบาววงแตก ค่ายยันแค่แหย่เล่น
      
ยังคงเป็นประเด็นที่ชาวเน็ตสนใจอย่างมาก สำหรับกรณีที่นักร้องและนักดนตรีคนดัง เทียรี่ เมฆวัฒนา หนึ่งในสมาชิกของวงคาราบาว ได้ออกมาโพสต์ข้อความผ่านทางเฟซบุ๊กส่วนตัวว่า “ตั้งแต่วันนี้ไป ผมไม่ทำตามคำสั่งของพี่แอ๊ดอีกต่อไปนะครับ” ทำให้แฟนๆเกิดความสงสัยว่าเกิดอะไรขึ้น มีปัญหาภายในวงหรือเปล่า ในเวลาต่อมาเทียรี่ก็ยังโพสต์ข้อความต่ออีกว่า “ถึงเวลาจะพูดความจริงนะครับ” ซึ่งได้สร้างความเป็นห่วงให้กับแฟนคลับอย่างมากว่าวงคาราบาวกำลังจะแตกหรือเปล่า ตามที่เสนอข่าวไปแล้วนั้นความคืบหน้าล่าสุดผู้สื่อข่าว “เดลินิวส์ออนไลน์” ได้โทรศัพท์ติดต่อไปยัง ผู้จัดการส่วนตัวของวงคาราบาว ก็ได้รับการเปิดเผยว่า “ตอนนี้ไม่ได้อยู่กับสมาชิกในวงและไม่รู้เรื่องอะไรเลย”ต่อมาผู้สื่อข่าวเลยได้ติดต่อไปที่ต้นสังกัด วอร์นเนอร์ มิวสิก ของวงคาราบาว ก็ได้รับการเปิดเผยว่า “จริงๆเรื่องนี้ได้มีการประสานให้ทีมเพื่อชีวิต ซึ่งเป็นทีมที่ดูแลศิลปินเพื่อชีวิตในค่ายดูเรื่องนี้ให้แล้ว ซึ่งก็ได้รับการยืนยันว่าเรื่องดังกล่าว ไม่มีอะไรเลยจริงๆ เมื่อคืนวงคาราบาวยังไปเล่นเต็มวงที่ลาวกันอยู่แล้ว ตอนนี้อยู่ในช่วงพักผ่อนของแต่ละคน ซึ่งทีมดูแลวงคาราบาวก็ได้แจ้งว่าเหมือนเป็นการแหย่เล่นกันเฉยๆ ซึ่งสมาชิกแต่ละคนก็จะเล่นแบบนี้กันอยู่แล้ว พอแหย่เล่นแล้วก็ลบโพสต์ มันเลยไม่มีอะไร”“อย่างประเด็นพี่เทียรี่จะแถลงนั้นก็ไม่มีอะไรเช่นกัน จริงๆที่ผ่านมาวงคาราบาวเขาทำงานด้วยกัน ไม่มีปัญหาเลย เขาทำงานเยอะมาก ในปีหนึ่ง 365 วันเขาทำงานแทบทุกวันและวันละ 2 งานด้วยซ้ำ ดังนั้นเขาจะสนิทและรู้กันอยู่แล้วใครร้องท่อนไหนยังไง ซึ่งเขาไม่เคยมีปัญหากันมาเลย เวลาเห็นเขาก็ปกติดี”ขณะเดียวกันในเฟซบุ๊กของเทียรี่ ก็ยังไม่ได้มีการลบโพสต์หรือเคลื่อนไหวใดๆเพิ่มเติม มีแต่ชาวเน็ตและแฟนคลับเข้ามาคอมเมนต์ถามว่าเรื่องจริงหรือเปล่า อย่าวงแตกกันนะจำนวนมากข่าวที่เกี่ยวข้องไม่ใช่ครั้งแรก!'คาราบาว'วงแตก 'เทียรี่'ออก-ปมขัดแย้งคาราบาวแตก!'เทียรี่'เดือด ซัดผมไม่ทำตาม'แอ๊ด'สั่งแล้ว
