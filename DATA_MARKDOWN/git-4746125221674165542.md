
# 中山美穂、25年ぶりにNHKホールで歌う 『Covers’Fes.』出演決定

Published at: **2019-11-03T22:01:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147975/full/)

　NHK・BSプレミアムの音楽番組『The Covers』が、年に1度、東京・渋谷のNHK ホールで公開収録を兼ねて開催する『The Covers’Fes.2019』（12月5日開催）に、デビュー35周年イヤーを迎えた中山美穂がスペシャルゲストとして出演する。NHK ホールで歌唱するのは、1994年の『NHK紅白歌合戦』以来、25年ぶり。あの名曲を披露する。

        【写真】その他の写真を見る
      
　今回で6回目を迎える同フェスだが、今年は“令和”初開催。そして初めての年末放送ということを踏まえ、今回は今年から来年にかけてビッグトピックスを携えたアーティストをラインナップ。 　日本を代表するロックバンド・エレファントカシマシのヴォーカル＆ギター宮本浩次。30年以上にわたる長いバンドキャリアの中で、今年ソロデビューを果たし大きな注目を集めている。『The Covers’Fes.』の常連で、ドラマ『歪んだ波紋』（BSプレミアム）の主題歌を手がける田島貴男（ORIGINAL LOVE）。今年デビュー15周年を迎え、精力的に活動を続けるバンド・フジファブリックが出演。それぞれが、この番組でしか見られない、思い入れの深い名曲カバーを披露する。 　さらに、LEGENDゲストとして、来年デビュー40年目を迎えるスターダスト☆レビュー。数々の歌い継がれる名曲を放つ彼らが、『The Covers’Fes.』のステージに初登場を果たす。 ■11月5日まで観覧募集中http://www4.nhk.or.jp/thecovers/収録予定：12月5日（木）収録場所：東京・渋谷　NHKホール（公開収録）放送予定：12月22日（日）　後10：50〜深0：20　NHK-BS プレミアム MC：リリー・フランキー 池田エライザカバーズゲスト：田島貴男（ORIGINAL LOVE）、フジファブリック、宮本浩次ほか（五十音順）スペシャルゲスト：中山美穂LEGEND ゲスト：スターダスト☆レビュー
