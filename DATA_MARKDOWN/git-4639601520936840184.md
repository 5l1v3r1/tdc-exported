
# Σχέδιο της κυβέρνησης για τις πορείες : Πώς θα μπει τέλος στην «ομηρία» του κέντρου

Published at: **2019-11-03T18:15:13+00:00**

Author: **ΤοΒΗΜΑ Team**

Original: [Το Βήμα Online](https://www.tovima.gr/2019/11/03/politics/sxedio-tis-kyvernisis-gia-tis-poreies-pos-tha-mpei-telos-stin-omiria-tou-kentrou/)

Τέλος στην απίστευτη ταλαιπωρία των πολιτών και στην «ομηρία” του κέντρου της Αθήνας- που προκαλούν οι συνεχείς πορείες κυρίως μικρών ομάδων διαδηλωτών θέλει να βάλει ο πρωθυπουργός Κυριάκος Μητσοτάκης.
Συγκεκριμένα, η κυβέρνηση εξετάζει εναλλακτικά μοντέλα διαχείρισης του προβλήματος, ώστε να μην «παραλύει» το κέντρο, κάθε φορά που διεξάγονται συνεχεία πορείες.

Για τα εναλλακτικά μοντέλα πρώτος μίλησε ο κυβερνητικός εκπρόσωπος, Στέλιος Πέτσας.
Ο Στέλιος Πέτσας μιλώντας στον Ant1 ανέφερε ότι θα αλλάξει το καθεστώς προκειμένου να οριοθετηθεί ο χώρος που θα πραγματοποιούνται οι πορείες ώστε οι διαδηλωτές να διαμαρτύρονται χωρίς να δημιουργούν προβλήματα.
Σκοπός της κυβέρνησης, όπως πρόσθεσε, είναι να μην προκαλούνται προβλήματα στους καταστηματάρχες και ταλαιπωρία στους πολίτες.
Ειδικότερα σύμφωνα με ρεπορτάζ της «Καθημερινής», μερικές από τις προτάσεις που έχουν τεθεί στο τραπέζι είναι:
Για μεγαλύτερες σε όγκο διαδηλώσεις
Σημειώνεται πως η κυβέρνηση θα οριστικοποιήσει άμεσα τις προτάσεις της και θα εκκινήσει διάλογο, ενώ δεν αποκλείει , εάν απαιτηθεί, νομοθετική πρωτοβουλία.
