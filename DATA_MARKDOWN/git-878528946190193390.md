
# Ｎ国・丸山議員、「ＮＨＫをぶっ壊す！」の流行語大賞“落選”に不満爆発

Published at: **2019-11-07T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/07/0012855208.shtml)

　ＮＨＫから国民を守る党（Ｎ国）の丸山穂高衆院議員が６日、ツイッターを更新。２０１９年の新語・流行語大賞の候補３０語がこの日発表されたが、今年７月の参院選で注目された「れいわ新選組」の「れいわ旋風」が候補入りしたものの、同じく存在感を発揮したはずのＮ国のキャッチフレーズ「ＮＨＫをぶっ壊す！」が選に漏れたことに不満を爆発させ、「ノミネート基準が不明確すぎる」と苦言を呈した。
　丸山氏は「流行語大賞候補に『れいわ旋風』は入れつつ、小学生まで真似してる『ＮＨＫを～ぶっ壊す！』を入れないところが、まさに政治的に偏っている流行語大賞らしい。過去にも『保育園落ちた日本死ね』『アベ政治を許さない』とか」と皮肉った。さらに「審査員メンバー見ればお察しですが、ノミネート基準が不明確すぎるでしょ」と疑問を呈した。
　６日発表された同賞の候補には、ラグビーＷ杯にちなんだ「ＯＮＥ　ＴＥＡＭ（ワンチーム）」や、台風関連の「計画運休」が入った。大賞の発表は１２月２日。
