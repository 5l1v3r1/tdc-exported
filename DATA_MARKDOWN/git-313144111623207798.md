
# [스포츠 브리핑] 안세현, 월드컵 접영 200m 동메달 외

Published at: **2019-11-03T18:00:26+00:00**

Author: **조선일보**

Original: [Chosun](https://news.chosun.com/site/data/html_dir/2019/11/04/2019110400018.html)

입력 2019.11.04 03:00 안세현, 월드컵 접영 200m 동메달안세현(24·SK텔레콤)이 2일 러시아 카잔에서 열린 2019 국제수영연맹(FINA) 경영월드컵 6차 대회 여자 접영 200m 결승에서 2분10초69 로 헝가리의 호수 커틴커(2분08초23)와 주잔나 야카보스(2분08초76)에 이어 동메달을 목에 걸었다.매킬로이, HSBC 챔피언스 우승남자 골프 세계 랭킹 2위 로리 매킬로이(북아일랜드)가 월드골프챔피언십(WGC) HSBC 챔피언스(총상금 1025만달러) 우승을 차지했다.좋아요 0 Copyright ⓒ 조선일보 & Chosun.com 제휴안내구독신청
