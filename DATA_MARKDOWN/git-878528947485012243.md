
# 巨人阿部２軍監督が始動　円陣で「考える力をつけよう」と若手にメッセージ

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/01/0012839665.shtml)

　巨人の阿部慎之助２軍監督（４０）が１日、川崎市のジャイアンツ球場で行われた秋季練習で“始動”した。
　背番号のないユニホーム姿でグラウンドに登場。首脳陣、ナインらが作った円陣で「考える力をつけよう」と主に若い選手らに向けてメッセージを送った。練習ではベンチで原辰徳監督と談笑。ナインにも声を掛けた。
　練習後、阿部２軍監督は「いろいろ大変だなと」と正直な感想を吐露。「今日１日じゃ分からないこともある。静観して把握したい。一番の課題です」と語った。今季限りで現役を引退した男が、指導者としての道のりをスタートさせた。
