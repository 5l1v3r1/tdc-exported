
# 20 ноября в Туле пройдет ярмарка вакансий для инвалидов

Published at: **2019-11-03T13:22:00+00:00**

Author: ****

Original: [Тульская Служба Новостей](https://www.tsn24.ru/20-noyabrya-v-tule-projdet-yarmarka-vakansij-dlya-invalidov.html)

20 ноября в Туле пройдет ярмарка вакансий квотируемых и специально созданных рабочих мест для трудоустройства инвалидов. Об этом сообщили в пресс-службе городской администрации. На ярмарке также запланированы мастер-классы по составлению резюме и по подготовке к собеседованию с работодателями.
День открытых дверей по вопросу трудоустройства инвалидов проводит городской Центр занятости населения. Мероприятие пройдет с 10.00 до 12.00 по адресу: г. Тула, ул. Демонстрации, д.34.
Напомним,что в 2018 году в Тульской области были трудоустроены 43,7% жителей, имеющих ограниченные возможности здоровья.
