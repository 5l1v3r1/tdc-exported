
# 山口　２回６失点…侍ローテの軸まさかの乱　カナダに３５年ぶり黒星

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/01/0012838444.shtml)

　「侍ジャパン強化試合、日本５－６カナダ」（３１日、沖縄セルラースタジアム那覇）
　野球日本代表「侍ジャパン」が惜敗した。カナダ代表には３５年ぶりの黒星となった。先発マウンドを託された山口俊投手（３２）が大乱調。二回に４安打４四球と乱れて６点を先行された。打線は四回以降に追い上げ、九回には１点差まで迫る粘りを見せた。１日の第２戦は今永昇太投手（２６）が先発する。
　最大６点差を１点差まで追い上げたが、届かなかった。プレミア前最後の実戦となるカナダとの２連戦。１９８４年以来３５年ぶりにカナダに敗れた。それでも稲葉監督は「最初に６点を取られた時は嫌な流れだったが、今は調整段階。また明日もあるので、しっかりやっていきたい」とサバサバと振り返った。
　先発・山口が課題を残した。初回は三者凡退と安定した立ち上がりを見せたが、二回に暗転。先頭への四球から打者１１人の猛攻を受け、一挙６失点。この回だけで４四球と制球も乱れた。「もう一度、細かいことをやっていく。収穫はないですね」。山口は硬い表情でバスに乗り込んだ。
　１９日以来の実戦登板に加え、代表に合流後まだ４日。低めのフォークを見極められ「（フォークに関してボールの違いは）少しありました」と振り返った。１１月５日のベネズエラ戦での先発が最有力となっている右腕。次回の実戦は本番になるが「不安はないです」と前を向いた。
　前回、カナダに敗れたのはロサンゼルス五輪での戦いだった。敗れはしたが、五輪では金メダルに輝いた。今回も糧とする時間は残されている。「修正ポイントは（本人も）分かっている」と山口への信頼を口にした指揮官。プレミア本番での変わり身を信じている。
