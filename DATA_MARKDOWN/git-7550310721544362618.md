
# Либертарианца Михаила Светова отпустили из СК после допроса в качестве свидетеля

Published at: **2019-11-07T04:27:00+00:00**

Author: ****

Original: [Новая Газета](https://www.novayagazeta.ru/news/2019/11/07/156702-libertariantsa-mihaila-svetova-otpustili-iz-sk-posle-doprosa-v-kachestve-svidetelya?utm_source=tw&utm_medium=novaya&utm_campaign=mihaila-svetova-posle-doprosa-otpustili)

Члена Либертарианской партии Михаила Светова около трех часов ночи отпустили из Следственного комитета после допроса в качестве свидетеля по делу о развратных действиях к несовершеннолетним (ст. 135 УК), сообщает в своем твиттере замглавы партии Ярослав Конвей.
