
# Сальвадор высылает весь дипломатический корпус Венесуэлы

Published at: **2019-11-03T06:41:29+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-03/salvador-vysylaet-ves-diplomaticheskiy-korpus-venesuely)

Сальвадор высылает всех дипломатов Венесуэлы. Им дали 48 часов на то, чтобы покинуть территорию страны.
Соответствующее решение приняло правительство центральноамериканской республики, сообщает ТАСС.
Сальвадор не признает президента Венесуэлы Николаса Мадуро главой государства.
Именно с этим связано решение властей этой страны.
