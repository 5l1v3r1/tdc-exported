
# Полицейские Южного округа столицы задержали подозреваемого в вандализме

Published at: **2019-11-01T15:11:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/yuao/politseyskie-yuzhnogo-okruga-stolitsy-zaderzhali-podozrevaemogo-v-vandalizme-123267/)

В ОМВД России по району Царицыно поступило сообщение о том, что на Пролетарском проспекте неизвестный в павильоне наземного городского транспорта разбил стекло, после чего скрылся. Ущерб составил более 1 тысячи рублей.В результате проверочных мероприятий сотрудники полиции задержали подозреваемого. Им оказался 22-летний приезжий.По данному факту отделением дознания ОМВД России по району Царицыно возбуждено уголовное дело по признакам преступления, предусмотренного статьей 214 УК РФ «Вандализм». В отношении подозреваемого избрана мера принуждения в виде обязательства о явке.Пресс-служба УВД по ЮАО(495) 734-61-22
