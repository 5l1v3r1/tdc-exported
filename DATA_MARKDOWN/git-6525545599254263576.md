
# Игорь Гужва: На парламентских слушаниях власть презентовала планы по уничтожению свободы слова

Published at: **2019-11-06T11:59:00+00:00**

Author: ****

Original: [Українські Новини](https://ukranews.com/news/664220-na-parlamentskih-slushaniyah-vlast-prezentovala-plany-po-unichtozheniyu-svobody-slova)


        Главный редактор портала Страна.UA Игорь Гужва дал свою оценку проходящим в Верховной Раде Украины парламентским слушаниям "Безопасность деятельности журналистов".
      

        "
        После выступления Бородянского и Ткаченко слушания по свободе слова в Раде правильнее было бы назвать презентацией планов власти по уничтожению свободы слова.
      
Причём уже никто ничего не стесняется. Говорят прямым текстом", - констатировал Гужва.
