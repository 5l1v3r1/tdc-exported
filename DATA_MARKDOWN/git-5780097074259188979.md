
# ​កម្លាំង​ប្រដាប់​អាវុធ​ចាប់​ផ្តើម​ចល័ត​នៅ​ព្រំ​ដែន​ថៃ មុន​ពេល​ក្រុម លោក សម រង្ស៊ី មក​ដល់​

Published at: **2019-11-01T07:34:40+00:00**

Author: **អ៊ឹង ប៊ុនថន**

Original: [RFI](http://km.rfi.fr/cambodia/troops-getting-toward-border-khmer-thai-01-11-2019)


        ដូច​ការ​គ្រោង​ទុក កង​កម្លាំង​ប្រដាប់​អាវុធ​គ្រប់​ប្រភេទ​ត្រូវ​បាន​បញ្ជា​ឲ្យ​ធ្វើ​ការ​ចល័ត​នៅ​ព្រឹក​ថ្ងៃសុក្រ​នេះ នៅ​ក្នុង​ខេត្ត​បន្ទាយ​មាន​ជ័យ​ជាប់​ព្រំដែន​កម្ពុជា-ថៃ ក្រោម​ហេតុ​ផល​ដើម្បី​ហ្វឹក​ហាត់​ការពារ​សន្តិសុខ​និង​សណ្តាប់​ធ្នាប់​។ ការ​បញ្ចេញ​កង​កម្លាំង​បំពាក់​ដោយ​អាវុធ​គ្រប់​ដៃ និង​គ្រប់​ធន​បែប​នេះ គឺ​មុន​៨ថ្ងៃ ដែល​លោក​ សម រង្ស៊ី ប្រកាស​ថា​នឹង​នាំ​ពលករ​ខ្មែរ​នៅ​ប្រទេស​ថៃ​រាប់​ម៉ឺន​នាក់​អម​ដំណើរ​លោកវិល​ចូល​កម្ពុជា​វិញ​នៅថ្ងៃ៩​វិច្ឆិកា​តាម​ច្រកព្រំដែន​ថៃ។ 
      
កម្លាំង​ប្រដាប់​អាវុធ​ដែល​ត្រូវ​បានបញ្ជាឲ្យ​ធ្វើការ​ចល័ត​នៅពេលនេះ គឺទាំង កងទ័ព កងរាជអាវុធហត្ថ និងនគរបាលដែលសុទ្ធតែបំពាក់​ទៅដោយអាវុធគ្រប់ដៃ និង​គ្រប់ធន់ រួមទាំង​រថគ្រោះទៀតផង។
បើតាម​រដ្ឋបាល​ខេត្ត​បន្ទាយ​មាន​ជ័យ ការបញ្ចេញ​កងកម្លាំង​ប្រដាប់អាវុធនៅថ្ងៃទី០១ ខែវិច្ឆិកានេះ គឺចេញពីទីបញ្ជាការដ្ឋានទិសទី១ និងទីបញ្ជាការដ្ឋាន​កងរាជ​អាវុធហត្ថ ឆ្ពោះទៅតម្រង់ជួរនៅស្នងការដ្ឋាននគរបាលខេត្ត ​ក្រោម​វត្តមាន លោក អ៊ុំ រាត្រី អភិបាលខេត្ត​បន្ទាយមានជ័យ។
លោក អ៊ុំ រាត្រី បានអះអាងថា​ នេះគឺជាការហ្វឹកហាត់រួមគ្នា ជាការបង្ហាញថា កង​កម្លាំង​ទាំងអស់មានសាមគ្គីភាព និង​មាន​សមត្ថភាព​គ្រប់​គ្រាន់​ក្នុងការការពារ​សន្តិសុខ​ផ្ទៃក្នុង ​និង​ការពារព្រំដែន​។
ទោះជាយ៉ាងណាក៏ដោយ ការបញ្ចេញ​កងកម្លាំងបំពាក់​ដោយអាវុធគ្រប់ដៃ និងគ្រប់ធន់បែបនេះ គឺធ្វើ​ឡើង​តែជាងមួយ​សប្តាហ៍ប៉ុណ្ណោះ ដែល​លោក​ សម រង្ស៊ី ប្រកាសថា​នឹង​នាំពលករ​ខ្មែរ​នៅប្រទេសថៃរាប់ម៉ឺននាក់​អម​ដំណើរ​លោកវិល​ចូល​កម្ពុជា​វិញ​នៅថ្ងៃ៩​វិច្ឆិកា​តាម​ច្រកព្រំដែន​ថៃ។
តាំងពីថ្ងៃទី២៥​ខែតុលាមកម្ល៉េះ នៅច្រកទ្វារអន្តរជាតិប៉ោយប៉ែត ក៏មានការដាក់ពង្រាយកម្លាំងប្រដាប់អាវុធឈរ​ជាជួរពុះចែក​ទ្រូង​ផ្លូវជាពីរចំណែក​ ដើម្បី​ត្រួត​ពិនិត្យ​ការចេញ​ចូល​របស់​អ្នកដំណើរ​នៅច្រក​ទ្វារ​អន្តរជាតិមួយនេះ។
ក៏ប៉ុន្តែ​មកដល់​ថ្ងៃទី១វិច្ឆិកានេះ គេនៅមិនទាន់​ប្រាកដទេថា តើ​អាជ្ញាធរថៃ​អនុញ្ញាតឲ្យ​ក្រុម លោក សម រង្ស៊ី ចុះ​ចត​នៅប្រទេសនេះ​ដើម្បីដឹកនាំ​ពលករ​ខ្មែរ​វិល​មកម្ពុជាឬយ៉ាងណាទេ ខណៈ​មន្រ្តី​បក្ស​ប្រឆាំង​អះអាងថា ខ្លួន​កំពុង​ចរចាជាមួយភាគីថៃ។ ចំណែក លោក សម រង្ស៊ី ផ្ទាល់បានអះអាងថា ផែន​ការលំអិត​នៃ​តម្រាយ​ផ្លូវ​ធ្វើ​ដំណើរ​របស់​លោក​នឹង​ប្រកាសឲ្យដឹងនៅថ្ងៃទី៦វិច្ឆិកាខាងមុខ៕
