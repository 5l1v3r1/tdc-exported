
# На SEMA Show представили 902-сильный restomod Chevrolet Camaro 1969 года

Published at: **2019-11-06T00:00:00+00:00**

Author: **Алексей Шилко**

Original: [CAR.RU](https://car.ru/news/automobili/57010-na-sema-show-predstavili-902-silnyiy-restomod-chevrolet-camaro-1969-goda/)


        Частное дизайнерское агентство показало обновленную версию легендарного Chevrolet Camaro.
      
Дизайнерское агентство Ringbrothers в рамках международной выставки автомобилей SEMA Show в Лас-Вегасе провело презентацию новой версии культового Chevrolet Camaro, который был выпущен в 1969 году. Модификация получила название в честь одной из воительниц скандинавской мифологии - Valkyrja.
Подобная машина выпущена в единственном экземпляре по заказу частного клиента из Бельгийского города Антверпен. В качестве обновление силовой установки специалисты установили 6,8-литровый двигатель с 8-ю цилиндрами, мощность которого составляет 902 л.с. Для усиления мощности мотора был добавлен искусственный 2,9-литровый нагнетатель от компании Whipple.
 
Трансмиссия нового авто оснащается механической коробкой передач Tremec с 6-ю ступенями от известной марки Bowler. Данная КПП также изготовлена под заказ в единственном экземпляре. Основа машины была переработана и собрана из отдельных частей Detroit Speed Engineering. Из примечательных особенностей нового авто можно отметить два стильных патрубка выхлопной системы.
Практически все кузовные детали: крыша, сплиттеры, диффузоры, капот выполнены из усиленного углепластика с помощью 3D-принтера.
