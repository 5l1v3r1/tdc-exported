
# 錦戸亮、ソロ初アルバムジャケ写3種公開

Published at: **2019-11-07T16:00:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148226/full/)

　10月1日からソロ活動をスタートした錦戸亮が主宰レーベル「NOMAD RECORDS」からリリースする第1弾アルバム『NOMAD』（12月11日発売）のジャケット写真3種が7日、公開された。

        【写真】その他の写真を見る
      
　本作は初回限定盤A（CD＋DVD）、初回限定盤B（CD＋DVD＋ライブフォトブック）、通常盤（CD）の3形態。リードトラック「ノマド」を含む全11曲入りで、すべての楽曲の作詞・作曲・プロデュースを錦戸自身が手がけている。 　初回限定盤Aの付属DVDには、表題曲「ノマド」およびYouTubeの公式チャンネルで先行配信した「Point of Departure」のミュージックビデオ、アルバム制作に密着した未公開メイキング映像を収録。 　初回限定盤Bには11月から開催される全国ツアー『錦戸亮 LIVE TOUR 2019”NOMAD”』ドキュメンタリー映像がいち早く収められるほか、全48ページのライブフォトブックが付く。通常盤には限定ボーナストラックとして「Potential」が収められる。 ■錦戸亮『NOMAD』収録曲01. ノマド02. ホンキートンクラプソディ03. 狛犬04. オールドスクール05. ヤキモチ06. いとしのエリ07.
