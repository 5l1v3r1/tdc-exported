
# Украинско-американская ракета вывела на орбиту космический корабль: как это было (фото)

Published at: **2019-11-04T03:58:24+00:00**

Author: ****

Original: [ГЛАВКОМ](https://glavcom.ua/scotch/live/ukrainsko-amerikanskaya-raketa-vyvela-na-orbitu-kosmicheskiy-korabl-kak-eto-bylo-foto-637413.html)

В субботу, 2 ноября, украинско-американская ракета-носитель «Антарес» успешно вывела на орбиту грузовой корабль «Сигнус», который доставит на Международную космическую станцию 3750 килограммов полезного груза, в том числе экспериментальную духовку.
Об этом сообщает Конструкторское бюро «Южное», которое разработало основную конструкцию первой ступени ракеты.
Пуск ракеты осуществили из Центра космических полетов на острове Уоллопс, штат Вирджиния. Стыковка с МКС запланирована на 4 ноября.
«Сигнус» по заказу NASA должен доставить на станцию провизию и вещи для экипажа, материалы для научных исследований, оборудование для магнитного альфа-спектрометра, оборудование для станции и выхода в открытый космос, компьютеры и комплектующие.
Также на МКС запланировано тестирование устройства для переработки пластика на нить (филамент) для 3D-печати. Эту нить можно будет использовать на устройстве Made in Space Manufacturing Device - 3D-принтере Refabricator, который работает в орбитальной лаборатории МКС с 2016 года.
