
# 侍ジャパン完封リレーでカナダに快勝　今永ら５投手で１１Ｋ「プレミア１２」に弾み

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/01/0012840931.shtml)

　「侍ジャパン強化試合、日本３－０カナダ」（１日、沖縄セルラースタジアム那覇）
　侍ジャパンが被安打４の完封リレーでカナダに快勝。敗戦した前夜の借りを返し、５日に初戦を迎える国際大会「プレミア１２」へ向けて弾みをつけた。
　日本の強みである投手力を見せつけた。先発の今永（ＤｅＮＡ）は３回を１安打無失点、５者連続を含む６奪三振の快投。２番手の左腕大野（中日）も３回を２安打無失点。七回からは中川（巨人）、山本（オリックス）、山崎（ＤｅＮＡ）が１イニングずつ無失点に抑えた。５投手の継投で計１１三振を奪った。
　打線は迫力を欠いたが、四回に２四球を絡め、浅村（楽天）の二ゴロの間にノーヒットで先制。五回は山田（ヤクルト）の適時打、八回は近藤（日本ハム）の適時打で小刻みに加点した。
　国際大会「プレミア１２」の１次ラウンドは５日の初戦でベネズエラ、６日にプエルトリコ、７日に台湾と対戦する。
