
# Зеленский назначил глав шести областных управлений СБУ

Published at: **2019-11-07T11:18:00+00:00**

Author: ****

Original: [НВ](https://nv.ua/ukraine/events/zelenskiy-naznachil-glav-shesti-oblupravleniy-sbu-novosti-ukrainy-50052153.html)

Соответствующие указы опубликованы пресс-службой Офиса президента.
В частности, Сергей Криворучко назначен начальником управления СБУ Херсонской области, Юрий Борейчук — в Тернопольской области, Евгений Борзилов — Полтавской области, Сергей Заяц — Ивано-Франковской, Павел Данюков — Волынской, Анатолий Денисенко — Винницкой области.
Отдельным указом президент уволил Андрея Олейника с должности начальника управления Службы безопасности в Днепропетровской области.
Также отдельным указом Зеленский назначил Игоря Носко начальником департамента охраны государственной тайны и лицензирования СБУ.
