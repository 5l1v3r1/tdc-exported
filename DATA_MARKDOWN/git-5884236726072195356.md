
# В Мурманске прошел чемпионат по Юкигассен

Published at: **2019-11-04T18:57:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/region/Murmansk/739767-v-murmanske-proshel-chempionat-po-yukigassen)


        В День народного единства мурманчане сразились в Юкигассен. Битва в снежки по строгим правилам прошла на стадионе школы № 50. Для местных жителей этот спорт стал одним из любимых; соревнования по нему проводятся уже 13 год, зимой и даже летом.
      
В День народного единства жители Мурманска пришли на стадион школы на Капитана Орликовой, чтобы сразиться в Юкигассен. Это снежки по правилам; в них здесь играют уже тринадцать лет: есть специальное оборудование и даже фирменный стиль нового для российского Заполярья вида спорта. Есть даже летняя альтернатива снежкам – тряпичные мячики с наполнителем.
Напоминаем, в конце ноября в Кировске стартует международная лыжная гонка.
