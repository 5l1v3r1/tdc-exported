
# Один титул до рекорда Шумахера. Определился чемпион Формулы-1 2019 года

Published at: **2019-11-03T21:17:00+00:00**

Author: ****

Original: [НВ](https://nv.ua/sport/other-sport/formula-1-chempion-2019-goda-lyuis-hemilton-gran-pri-ssha-2019-pobeditel-valtteri-bottas-50051381.html)

Британский гонщик занял второе место по итогам Гран-при США, чего оказалось достаточно, чтобы его не достали конкуренты.
Сам этап выиграл его партнер по команде Валттери Боттас. Третьим к финишу пришел пилот Ред Булл Макс Ферстаппен.
