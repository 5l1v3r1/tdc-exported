
# Эксперт оценил шансы «Зенита» в матче Лиги чемпионов с «Лейпцигом»

Published at: **2019-11-05T13:08:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/739985-ekspert-ocenil-shansy-zenita-v-matche-ligi-chempionov-s-leipcigom)


        Бывший футболист и тренер «Зенита» Алексей Стрепетов в разговоре с НЕВСКИМИ НОВОСТЯМИ поделился мнением о предстоящем матче четвертого тура группового этапа Лиги чемпионов, в котором сине-бело-голубые на своем поле примут «Лейпциг».
      
По его мнению, ключевым аспектом в игре петербуржцев является реализация моментов, с которой есть определенные проблемы.
