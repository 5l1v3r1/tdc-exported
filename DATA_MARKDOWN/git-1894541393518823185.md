
# На железнодорожном вокзале Мариуполя открыт зал ожидания для военнослужащих

Published at: **2019-11-07T11:00:00+00:00**

Author: ****

Original: [ОстроВ](https://www.ostro.org/donetsk/society/news/579187/)

Зал ожидания для военнослужащих открыт на железнодорожном вокзале Мариуполя, - сообщает пресс-центр "Укрзализныци". 
Переоборудование помещения выполнено собственными силами компании. 
Для комфорта посетителей здесь есть комнаты для приема пищи, санузлы, душ, места для сидения, телевизоры, холодильники, розетки для зарядки мобильных устройств, доступ к интернету. Кроме того, военные могут воспользоваться гладильной доской с утюгом, микроволновой печью и чайником.
Первый такой зал ожидания для военнослужащих открылся 1 октября 2019 на Центральном железнодорожном вокзале в Киеве. Аналогичные залы заработали во Львове, Одессе, Харькове и Днепре.
Все залы предназначены для кратковременного пребывания военнослужащих (участников ООС) при наличии удостоверения, удостоверяющий личность, и проездного документа на поезд (кроме пригородного сообщения).
