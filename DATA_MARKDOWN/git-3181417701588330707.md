
# В России тестируют мобильное приложение по онлайн-оформлению ДТП

Published at: **2019-11-01T08:56:00+00:00**

Author: ****

Original: [Тверские ведомости](https://vedtver.ru/news/society/v-rossii-testiruyut-mobilnoe-prilozhenie-po-onlayn-oformleniyu-dtp/)

Фото: btsmithlaw.com
В пяти регионах России с 1 ноября в тестовом режиме запускается мобильное приложение «Помощник ОСАГО». По сообщению «Известий», сервис оформления дорожно-транспортных происшествий по европротоколу будет доступен для жителей Москвы, Подмосковья, Санкт-Петербурга, Ленинградской области и Республики Татарстан. Скачать его можно как  в App Store, так и в Google Play.
Разработанное приложение позволит оформить ДТП по европротоколу за 15 минут без заполнения бумажного извещения.
«Все данные (в/у, СТС, страховка) для оформления извещения поступают из профилей водителей на портале «Госуслуги», - пояснили в Минкомсвязи.
Попавший в аварию водитель сможет выбрать вариант оформления - с фотофиксацией или без нее. Есть нюанс: если фотографии не приложить, то максимальная сумма выплат снизится до 100 тыс. рублей.
