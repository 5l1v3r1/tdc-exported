
# Появилось устройство для борьбы с депрессией

Published at: **2019-11-01T14:35:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/novosti_mira/poyavilos_ustroystvo_dlya_borby_s_depressiey/)

В продажу на рынок Великобритании поступили устройства, которые способны облегчить симптомы депрессивных заболеваний. Учёные рассказали, что новинка помогает бороться с депрессией с помощью слабых электрических разрядов, которые стимулируют область передней части мозга.
Прибор применяет неинвазивный метод лечения – когда на кожу не оказывается хирургического воздействия. Для работы гаджета используется «виртуальный терапевт» на смартфоне. Устройство было одобрено медиками в Европе.
При этом ранее американские учёные разработали способ диагностики депрессии через смартфон.
