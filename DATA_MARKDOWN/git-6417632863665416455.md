
# Обамеянг Анри, Бергкамп ва Алексисни ортда қолдирди

Published at: **2019-11-04T04:08:34+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/obameyang-anri-bergkamp-va-aleksisni-ortda-qoldirdi)

Ушбу баҳсда мезбонлар ҳужумчиси Пьер-Эмерик Обамеянг "Арсенал" таркибидаги ўзининг 50-голини урди. Габонлик ҳужумчи бунча голни 78та учрашувда уришга муваффақ бўлди. 
Маълумот учун, Тьерри Анри бу клубда 50та голни 104та, Алексис Санчес 106та, Деннис Бергакмпга 112та ўйинда урган.
