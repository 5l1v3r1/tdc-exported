
# Хваса из MAMAMOO в аппетитной рекламе рамёна

Published at: **2019-11-01T11:58:07+00:00**

Author: **gunyasha**

Original: [YesAsia](https://www.yesasia.ru/article/753171)

Знаете ли вы, насколько аппетитно может выглядеть рамён? Королева мокбана, Хваса из MAMAMOO, в очередной раз доказала, что способна заставить любую пищу выглядеть аппетитно и привлекательно для зрителей. Она снялась в рекламном ролике Jin-Jjambong рамён.
