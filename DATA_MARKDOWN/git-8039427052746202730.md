
# ఇంటర్ సర్టిఫికేట్లకు దరఖాస్తు చేసుకోండి!

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=948871)

హైదరాబాద్‌: ఇంటర్‌మీడియట్‌ 2019 పరీక్షల్లో పాస్‌ అయిన విద్యార్ధులకు సర్టిఫికెట్‌లు అందడం లేదంటూ మీడియాలో వస్తున్న వార్తలకు బోర్డు ఆఫ్‌ ఇంటర్మీడియట్‌ అధికారులుస్పందించారు. పాస్‌సర్టిఫికెట్‌ల ప్రింటింగ్‌ పనులు కొనసాగుతున్నాయని మరో పదిరోజుల్లోనే విద్యార్దులకు అందించేందుకు కృషి చేస్తామని బోర్డుఅధికారులు తెలిపారు.అయితే ఎవరైనా విద్యార్ధులకు అర్జంట్‌గా పాస్‌ సర్టిఫికెట్‌ అవసరమైతే వెంటనే బోర్డుకు సంబంధించిన coetsbie16@gmail.com మెయిల్ ఐడీ ద్వారా దరఖాస్తు చేసుకోవాలన్నారు. వెంటనే ఇంటర్‌మీడియట్‌బోర్డు ఏ విధమైన ఫీజు వసూలుచేయకుండా ట్రూ ఎక్స్‌ట్రాక్ట్‌ సర్టిఫికెట్‌ను జారీ చేస్తుందని అధికారులు ఒక ప్రకటనలో పేర్కొన్నారు.
