
# イスラム過激派襲撃か、マリで兵士ら５４人殺害

Published at: **2019-11-02T10:44:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/world/20191102-OYT1T50244/)

　【ヨハネスブルク＝木村達矢】アフリカ西部マリ北東部の軍駐屯地で１日、イスラム過激派組織によるとみられる襲撃があり、兵士５３人と市民１人の計５４人が殺害された。マリ政府が２日、明らかにした。
　犯行声明は出ていないが、マリではイスラム過激派組織による襲撃事件が相次いでいる。９～１０月には国際テロ組織「アル・カーイダ」系のイスラム過激派組織による２件の襲撃があり、計約４０人の兵士が殺害された。
