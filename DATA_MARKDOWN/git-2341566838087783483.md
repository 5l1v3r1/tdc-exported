
# 10月車市重回4萬台 Toyota銷售最亮麗

Published at: **2019-11-01T19:36:00+00:00**

Author: **https://www.facebook.com/cnanewstaiwan, 中央通訊社**

Original: [中央社 CNA](https://www.cna.com.tw/news/gpho/201911010003.aspx)

受到9月底颱風假、買氣遞延影響，台灣10月車市銷售表現出乎意料，單月達4.1萬台、月增24.9%、年增17.1%，其中，龍頭豐田（Toyota）單月大賣1萬4179台最為凸出。中央社製圖 108年11月1日
