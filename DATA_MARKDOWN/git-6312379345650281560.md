
# В Турции рассказали о сроках поставок второй партии С-400

Published at: **2019-11-04T09:39:40+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-04/v-turcii-rasskazali-o-srokah-postavok-vtoroy-partii-s-400)

Поставки в Турцию второй партии российских зенитно-ракетных комплексов С-400 могут быть осуществлены позднее графика, заявил глава секретариата турецкой оборонной промышленности Исмаил Демир.
По его словам, отсрочка может быть связана с переговорами по деталям передачи технологий и налаживанием совместного производства, сообщает телеканал NTV.
Кроме этого, Демир подтвердил, что Москва предложила Анкаре закупать российские истребители и сейчас это предложение рассматривается. Решение примут после "всестороннего анализа".
Напомним, первую партию С-400 Турции поставили в июле 2019 года, несмотря на активное противодействие со стороны США.
