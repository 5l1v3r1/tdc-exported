
# Яременко як дипломат для мене перестав існувати, - Жебрівський

Published at: **2019-11-04T23:26:00+00:00**

Author: ****

Original: [ZIK.UA](https://zik.ua/news/2019/11/04/yaremenko_yak_dyplomat_dlya_mene_perestav_isnuvaty__zhebrivskyy_1683691)


        Після того, як народний депутат Богдан Яременко повів себе після оприлюднення скандальної переписки, для мене він перестав існувати як дипломат.
      
Про це в ефірі програми «Треба поговорити» на телеканалі ZIK заявив лідер партії «Республіканська платформа», ексаудитор НАБУ Павло Жебрівський.
