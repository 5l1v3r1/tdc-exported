
# Ўзбекистон U-19 терма жамоасининг Осиё Чемпионати-2020 саралашига йўл олган таркиби маълум

Published at: **2019-11-04T07:01:49+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/ozbekiston-u-19-terma-jamoasining-osiyo-chempionati-2020-saralashiga-yol-olgan-tarkibi-malum)

Саралаш баҳсларига ҳозирлик жараёнини якунлаган Ўзбекистон ёшлар терма жамоаси бугун Хобар шаҳрига йўл олди.Ўзбекистон U-19 терма жамоаси саралашнинг “F” гуруҳидан ўрин олган ва гуруҳда Саудия Арабистони, Ҳиндистон ҳамда Афғонистон терма жамоалари билан баҳс олиб боради.
