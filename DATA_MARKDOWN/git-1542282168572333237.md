
# Το Γκραν Πρι Ρολογιών της Γενεύης

Published at: **2019-11-02T18:31:43+00:00**

Author: **Ioannis Giagkinis**

Original: [euronews](https://gr.euronews.com/2019/11/02/to-gran-pri-rologion-tis-genevis)

Πραγματοποιείται για ακόμα μια χρονιά στη Γενεύη το Γκραν Πρι Κατασκευής Ρολογιών, μια διοργάνωση που πρωτοπραγματοποιήθηκε το 2001.
Σκοπός του Γκραν Πρι είναι να αναγνωστούν οι κορυφαίες κατασκευές ρολογιών στον κόσμο και να βραβευθούν οι καλύτερες δημιουργίες.
