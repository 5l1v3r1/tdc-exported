
# ลูกเรือไอเอสเอสเตรียมอบ'คุกกี้' ในสภาพ'ไร้น้ำหนัก'

Published at: **2019-11-03T02:15:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/foreign/739741)


        นาซาส่งเสบียงและเครื่องมือทางวิทยาศาสตร์ชุดใหม่ซึ่งรวมถึง "เตาอบขนม" ขึ้นสู่สถานีอวกาศนานาชาติ เพื่อการทดลองอบคุกกี้ในสภาวะไร้น้ำหนัก
      
สำนักข่าวต่างประเทศรายงานจากกรุงวอชิงตัน ประเทศสหรัฐอเมริกา เมื่อวันที่ 2 พ.ย. ว่าจรวดแอนทาเรสของบริษัทนอร์ธรอป กันแมน ซึ่งเป็นหนึ่งในคู่สัญญาขององค์การบริหารการบินและอวกาศแห่งชาติของสหรัฐ ( นาซา ) ในการส่งยานแคมปซูลเสบียงขึ้นสู่สถานีอวกาศนานาชาติ ( ไอเอสเอส ) ออกเดินทางจากฐานปล่อยจรวดภายในศูนย์อวกาศของนาซา บนเกาะวอลลอปส์ นอกชายฝั่งรัฐเวอร์จิเนีย ทางตะวันออกของประเทศ เมื่อเวลา 09.59 น. ตามเวลาท้องถิ่นของวันเสาร์ ( 20.59 น. ตามเวลาในประเทศไทย ) เพื่อส่งยานแคปซูลซิกนัส เอ็นจี-12 ลำเลียงเสบียงและอุปกรณ์อื่นขึ้นสู่ไอเอสเอส
