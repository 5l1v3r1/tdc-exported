
# 性侵案發酵？劉強東因「個人原因」辭去政協委員

Published at: **2019-11-07T21:29:03+00:00**

Author: ****

Original: [NTD 新唐人電視台](https://www.ntdtv.com/b5/2019/11/07/a102702175.html)

【新唐人北京時間2019年11月08日訊】據中共央視7日晚的新聞聯播報導，中共全國政協第十三屆常委會第九次會議當天閉幕，會議接受劉強東「因個人原因」請辭第十三屆全國政協委員。
根據中共的《中國人民政治協商會議章程》第38條規定，對違反社會道德或存在與身份不符行為的政協委員，應當及時約談或函詢，經提醒仍不改正的，應當責令其辭去委員。
京東集團創始人劉強東去年開始深陷性侵醜聞，至今仍未脫身。
去年8月底，劉強東在美國明尼蘇達州因涉嫌性侵一名中國籍女大學生，一度被捕。同年12月21日，美國當地檢方決定不予刑事起訴。但今年4月16日，受害女大學生正式對劉強東提起民事訴訟，目前官司還在進行中。今年8月，劉強東的妻子章澤天遠赴英國劍橋大學讀書。
