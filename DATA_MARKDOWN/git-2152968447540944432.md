
# Верижна катастрофа затруднява движението по Яворов

Published at: **2019-11-05T09:14:00+00:00**

Author: **Красимир Ангелов**

Original: [Dnes.bg](https://www.dnes.bg/sofia/2019/11/05/verijna-katastrofa-zatrudniava-dvijenieto-po-iavorov.428587)

Снимка: Dnes.bg
123
Верижна катастрофа затруднява движението по бул. "Пейо К. Яворов" в района на телевизионната кула, видя репортер на Dnes.bg.Ударили са се няколко коли, като по първоначална информация няма данни за сериозно пострадали. По колите обаче има материални щети.Заради пътния инцидент е силно затруднено движението в района. Автомобилите в посока Телевизионната кула минават през кръговото кръстовище.
