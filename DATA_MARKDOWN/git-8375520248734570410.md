
# У Великобританії випадково знайшли колекцію з 135 ретро-автомобілів. Фотогалерея

Published at: **2019-11-04T12:05:35+00:00**

Author: ****

Original: [ГЛАВКОМ](https://glavcom.ua/photo/u-velikobritaniji-vipadkovo-znayshli-kolekciyu-z-135-retro-avtomobiliv-fotogalereya-637553.html)

135 автомобілів зберігалися у власності в Престоні (Великобританія), яка була продана для перепланування. На проданій території було кілька технічних приміщень, де і були виявлені ретро-автомобілі.
Колекція в основному складається з британських і французьких машин 1950-1980-х років.
Всі автомобілі продадуть через аукціон. Вони продаються як є: без документації або гарантії.
