
# “Сельта”нинг янги мураббийи илк учрашувини собиқ клубига қарши ўтказади

Published at: **2019-11-04T19:40:00+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/seltaning-yangi-murabbiyi-ilk-uchrashuvini-sobiq-klubiga-qarshi-otkazadi)

Каталониялик мутахассис Оскар Гарсия виголиклар бош мураббийи этиб тайинланди. У мураббийлик фаолиятини “Барселона” тизимида бошлаган. Қизиқарли томони “Сельта” устози янги жамоасида илк учрашувни ўзининг собиқ жамоаси “Барселона”га қарши бошлайди.
Оскар Гарсия сўнгги икки мавсумни “Сент-Этьен” ва “Олимпиакос”да ўтказди. Мураббийнинг “Сельта” билан шартномаси мавсум охирига қадар тузилди.
Илгарироқ “Сельта” раҳбарияти Фран Эскрибани истеъфога чиқарганди.
