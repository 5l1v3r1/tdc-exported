
# Эксперты выяснили, как можно распознать измену

Published at: **2019-11-07T12:46:00+00:00**

Author: **Александр Воинков**

Original: [ФедералПресс](https://fedpress.ru/news/77/society/2360394)


        МОСКВА, 7 ноября, ФедералПресс. Тревожным звоночком может стать регулярная привычка задерживаться на работе. Необоснованная пропажа денег из семейного бюджета также может свидетельствовать о возможной измене.
      
Специалисты перечислили признаки, помогающие выявить измену. Для этого нужно проанализировать поведение своей второй половинки и трезво оценить ситуацию, включив холодный ум.
По словам экспертов, женщинам стоит начать беспокоиться, если мужчина начал тщательнее следить за своим внешним видом. Также регулярная привычка задерживаться на работе может стать тревожным звоночком.
Не менее подозрительным может стать необоснованная пропажа денег из общего бюджета, передает Infox. Потенциальный изменщик, по словам психологов, будет избегать зрительного контакта. Помимо этого он начнет тщательнее следить за чистотой своего телефона, ежедневно стирая истории своих переписок, пишет Nation News.
Ранее сообщалось об именах мужчин, обреченных на неверность. Эксперты составили список имен.
