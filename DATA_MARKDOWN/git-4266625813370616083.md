
# Зеленский утвердил состав делегации Украины для участия в конференции ЮНЕСКО

Published at: **2019-11-05T09:00:00+00:00**

Author: ****

Original: [ZIK.UA](https://zik.ua/ru/news/2019/11/05/zelenskyy_utverdyl_sostav_delegatsyy_ukrayni_dlya_uchastyya_v_konferentsyy_1683797)


        Президент Украины Владимир Зеленский утвердил состав делегации Украины для участия в работе 40-й сессии Генеральной конференции ЮНЕСКО.
      
Соответствующее распоряжение №359/2019-рп опубликовано на сайте главы государства.
«Образовать делегацию Украины для участия в работе 40-й сессии Генеральной конференции ЮНЕСКО (12 – 27 ноября 2019 года, г.Париж, Французская Республика) в таком составе», – говорится в документе.
В частности, в состав делегации вошли:
Также Президент поручил Министерству иностранных дел Украины утвердить указание делегации Украины для участия в работе 40-й сессии Генеральной конференции ЮНЕСКО.
Кроме того, Зеленский разрешил главе делегации вносить в случае потребности изменения в персональный состав делегации и привлекать экспертов для обеспечения работы делегации.
Напомним, Президент Украины Владимир Зеленский попросил первого заместителя председателя Верховной Рады Руслана Стефанчука срочно собрать очередное совещание депутатов по подготовке закона о народовластии.
