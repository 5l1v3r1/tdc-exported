
# Tekvando Avrupa Şampiyonası'nda İrem Yaman'dan altın madalya

Published at: **2019-11-02T22:30:32+00:00**

Author: **euronews **

Original: [euronews](https://tr.euronews.com/2019/11/02/tekvando-avrupa-sampiyonasinda-irem-yaman-dan-turkiye-ye-altin-madalya)

İtalya'da düzenlenen Tekvando G4 Ekstra Avrupa Şampiyonası'nda milli sporcu İrem Yaman, rakiplerini eleyerek altın madalyanın sahibi oldu.
Bari kentinde düzenlenen şampiyonanın ikinci gününde kadınlar 57 kiloda mücadele veren İrem Yaman, finalde son 2 olimpiyat şampiyonu İngiliz Jade Jones ile karşılaştı. Rakibini 22-9 mağlup eden milli sporcu, böylelikle adını altın madalyaya yazdırdı.
