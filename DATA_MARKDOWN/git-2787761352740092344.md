
# В Дятькове на хрустальном заводе ООО «ДХЗ плюс» идёт реконструкция цеха

Published at: **2019-11-01T10:46:32+00:00**

Author: ****

Original: [Брянск Today](https://bryansktoday.ru/article/111699)

В Брянской области на дятьковском хрустальном заводе ООО «ДХЗ плюс» полным ходом идёт полномасштабная реконструкция цеха выработки. Уже возведена новая стекловаренная печь с суточным объемом 2,5 тонны.
Демонтирован старый лер, который заменят новым, более современным и высокотехнологичным. Ведутся работы по расширению внутреннего пространства цеха.
У стекловаренной печи появится просторная смотровая площадка для экскурсантов, отвечающая всем требованиям безопасности. Теперь посетители и гости предприятия смогут без помех наблюдать процесс рождения хрустальных шедевров, имеющих высокий статус народных художественных промыслов.
