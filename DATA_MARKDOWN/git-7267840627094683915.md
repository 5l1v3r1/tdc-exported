
# Черниговскую ОГА возглавил Андрей Прокопенко

Published at: **2019-11-01T19:10:00+00:00**

Author: ****

Original: [Главное™](https://glavnoe.ua/news/n357007835-chernigovskuju-oga-vozglavil-andrej-prokopenko)

Президент Украины Владимир Зеленский подписал Указ "О назначении А.Прокопенко председателем Черниговской областной государственной администрации".
"Назначить ПРОКОПЕНКО Андрея Леонидовича председателем Черниговской областной государственной администрации", - говорится в тексте Указа.
Документ датирован 31 октября.
