
# Пользователи ВКонтакте пожаловались на сбои в работе соцсети

Published at: **2019-11-04T13:49:00+00:00**

Author: ****

Original: [РИА Новости](https://ria.ru/20191104/1560563123.html)

МОСКВА, 4 ноя — РИА Новости. Пользователи социальной сети "ВКонтакте" из ряда российских городов сообщают о сбое в работе социальной сети, свидетельствуют данные сайта Downdetector, который отслеживает сбои и отключения в популярных интернет-ресурсах.Сбои начались около 16:00 мск. По данным Downdetector, 59 процентов пользователей соцсети жалуются на проблемы с загрузкой сайта, жалобы 37 процентов связаны с доставкой сообщений.С проблемами столкнулись жители Москвы, Санкт-Петербурга, Екатеринбурга, Челябинска, Новосибирска, Красноярска, Нижнего Новгорода, Волгограда, Краснодара и других городов России.
