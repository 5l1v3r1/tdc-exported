
# Web Summit, l’ora di Snowden: “Ormai l’abuso della privacy è stato legalizzato. Dobbiamo riconquistare Internet”

Published at: **2019-11-04T19:03:00+00:00**

Author: **JAIME D&#39;ALESSANDRO**

Original: [Repubblica.it](https://www.repubblica.it/tecnologia/2019/11/04/news/web_summit_2019_edward_snowden-240239971/)

LISBONA - L’Altice Arena, ai margini di Lisbona, è piena. Il Web Summit 2019 apre con un tutto esaurito: 70 mila presenze accreditate da 163 Paesi e lunghe file all’esterno per entrare. Festival nato per le startup nel 2010 e inaugurato sempre dalle startup anche quest’anno. Alle 22 fra le più promettenti, pescate fra le duemila 150 che si alterneranno dal 4 al 7 novembre, sono stati dati alcuni minuti per presentare la propria idea sul palco più importante, poco prima dell’intervento di Edward Snowden.
