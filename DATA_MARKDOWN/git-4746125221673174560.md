
# YouTuberまあたそ&よきき、恋愛関係匂わせ「興味ある?」 大倉士門から「全然、匂ってない!」とツッコミ

Published at: **2019-11-07T22:21:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148181/full/)

　人気YouTuberのまあたそ、よききが7日、東京・SHIBUYA109で行なわれた『SHIBUYA109 LIVE TV ハチスタ』レセプションパーティーに出席。 　同スタジオが渋谷109にできることからよききは「めっちゃ恥ずかしいんですけど、（渋谷109に）女装の服を買いにきたときは…」と赤面エピソードを明かし、初めての渋谷109は「高校生くらいのときに来た」と告白。 　また、一緒に登壇したカップルYouTuberのおたひかに恋愛話が振られると、まあたそが「ここ2人（よききとの関係）は聞かんでも大丈夫ですか？ 興味ある？ ごめんなさい、匂わせちゃってね」と言うとMCの大倉士門から「全然、匂ってない！」とツッコまれ会場を盛り上げた。 　また、モデルでタレントのくみっきーこと舟山久美子も登場し、会場にいるモデルやクリエイターに向けて「私自身も人にやらされるってことよりは、自分で何をしたいかということを形にしてきて、今がある」と明かし、「だからこそ、せっかく、すてきな場所ができたので活用しない訳にはいかないなと思います。みなさんハチスタで発信していってもらえたら」とアドバイスをした。 　同スタジオは、今年40周年を迎えたSHIBUYA109の8階にオープンした新たな放送局。『SHIBUYA109LAND』をコンセプトに、さまざまなコンテンツを展開する同所のエンターテインメントの中心として位置づけられる。 　そのほか、竹内唯人、黒田昊夢、Kaya、浪花ほのか、ミチ、宮瀬いと、山之内すず、ひかりんちょ、なえなの、伊藤桃々、きいた、本田響矢、北出大治郎、ロイ、ミチ、kirari、たけうちほのか、8467、めがねも登壇した。
