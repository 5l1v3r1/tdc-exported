
# Грабители расстреляли мужчину из травмата в центре Москвы

Published at: **2019-11-01T15:57:42+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-01/grabiteli-rasstrelyali-muzhchinu-iz-travmata-v-centre-moskvy)

Несколько злоумышленников, вооруженных травматическим оружием, напали на мужчину на улице Верхние Поля в Москве.
Грабители выстрелили в него, похитили сумку и попытались скрыться. 
Как сообщили в столичном главке МВД, один из нападавших был задержан. Введен план "Перехват" для розыска и задержания остальных преступников.
Пострадавшего осматривают медики, по факту произошедшего проводится проверка.
