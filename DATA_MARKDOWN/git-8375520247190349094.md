
# Бойовики обстріляли позиції Збройних сил під Кримським

Published at: **2019-11-06T12:06:06+00:00**

Author: ****

Original: [ГЛАВКОМ](https://glavcom.ua/news/boyoviki-obstrilyali-poziciji-zbroynih-sil-pid-krimskim-638104.html)

Російські окупаційні війська від початку доби один раз обстріляли позиції Об’єднаних сил поблизу Кримського. Втрат наразі немає.
Про це під час брифінгу повідомив речник Міноборони Максим Праута.
«Від початку поточної доби до 12.30 російсько-окупаційні війська відкривали вогонь один раз на Луганському напрямку», - повідомив речник.
