
# В Харькове полицейские изъяли больше тысячи литров топлива у незаконных заправщиков

Published at: **2019-11-02T16:23:00+00:00**

Author: ****

Original: [Главное™](https://glavnoe.ua/news/n357007892-v-harkove-policejskie-izjali-bolshe-tysjachi-litrov-topliva-u-nezakonnyh-zapravschikov)

2 ноября во время рейда полицейские изъяли из незаконного оборота почти 1400 литров топлива.
Об этом сообщили в отделе коммуникации ГУНП в Харьковской области.
По данным правоохранителей, они отработали места, где вероятно незаконно продавали горюче-смазочные материалы и установили факты реализации продукции без каких-либо разрешительных документов. Общее количество изъятого топлива составила 1360 литров.
Составлены административные протоколы по ст. 160 (торговля с рук в неустановленных местах) Кодекса Украины об административных правонарушениях.
В ведомстве отметили, борьба с незаконной торговлей топлива будет продолжаться.
