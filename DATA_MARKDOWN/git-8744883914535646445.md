
# «Динамо» выиграло Кубок России по хоккею с мячом

Published at: **2019-11-03T12:10:53+00:00**

Author: ****

Original: [Коммерсантъ](https://www.kommersant.ru/doc/4148169)

В финале Кубка России по хоккею с мячом московское «Динамо» победило архангельский «Водник». Команда стала семикратным обладателем трофея. Матч проходил в Москве в ледовом центре «Крылатское».
«Динамо» победило со счетом 4:2. Мячи забили: Владислав Тарасов (9; 30), Янис Бефус (47), Виктор Чернышев (68) — Андрей Климкин (17), Артем Шеховцов (41).
Победителем прошлого Кубка России по хоккею с мячом был хабаровский клуб «СКА-Нефтяник».
