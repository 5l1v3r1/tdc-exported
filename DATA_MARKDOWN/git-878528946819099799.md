
# 侍ジャパン　山口２球でヒヤリも…宝刀フォークでピンチ脱出

Published at: **2019-11-05T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/05/0012851414.shtml)

　「プレミア１２・１次ラウンド、日本－ベネズエラ」（５日、桃園国際野球場）
　野球日本代表・侍ジャパンの１次ラウンド初戦。先発マウンドには山口（巨人）が上がった。
　初回は１番に右前打、２番に左前打。いずれも初球を捉えられ、２球で無死一、二塁のピンチを招いた。３番にも３ボールとなったが、ここからフルカウントまで持っていき、最後はフォークで空振り三振。このアウトで落ち着きを取り戻すと、４番もフォークで空振り三振。５番もフォークで二ゴロに仕留め、無失点でしのいだ。
　山口は１０月３１日に行われた強化試合・カナダ戦で２回６失点。制球が定まらず、状態を不安視されていた。それでも稲葉監督は「この期間でしっかりと修正してくれたと思いますし、期待したい」と語り、初戦のマウンドに送り込んだ。
