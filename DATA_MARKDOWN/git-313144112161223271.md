
# 이인정 국제산악연맹 명예회원

Published at: **2019-11-05T18:33:08+00:00**

Author: **조선일보**

Original: [Chosun](https://news.chosun.com/site/data/html_dir/2019/11/06/2019110600273.html)

입력 2019.11.06 03:33
이인정(74) 아시아산악연맹 회장이 지난 2일 키프로스 라르나카에서 열린 국제산악연맹(UIAA) 총회에서 한국인 최초로 명예회원에 선출됐다. 아시아에서는 세 번째다. 이 회장은 한국대학산악연맹 회장, 대한산악연맹 회장 등을 지냈다.
