
# Американские котлеты из гороха продают россиянам по цене элитной говядины

Published at: **2019-11-05T10:17:00+00:00**

Author: ****

Original: [www.dvnovosti.ru](https://www.dvnovosti.ru/khab/2019/11/05/106521/)

В некоторых магазинах страны был дан старт продаж растительных котлет из США. Мясо изготов­лено преимущественно из гороха, воды, рапсового масла. Производит его компания Beyond Meat, в которую вложились "громкие" инвесторы — основатель Microsoft Билл Гейтс, актер Леонардо ДиКаприо, экс-гендиректор McDonald's Дон Томпсон.
Упаковка с двумя котлетами весом чуть больше 200 граммов будет стоить в рознице 750 рублей, то есть примерно 3 тысячи рублей за килограмм. Готовность закупать растительные котлеты выразили некоторые крупные торговые сети.
Российские мясопереработчики сомневаются в успехе предприятия.
По мнению Дмитрия Гордеева, руководителя Центра экономико-аналитических исследований и IT-технологий ФГБНУ "ФНЦ пищевых систем им. В.М. Горбатова, горох не может заменить собой мясной белок, необходимый для полноценного питания.
Тем временем компания Beyond Meat начала использовать в производстве также муку из сверчков, отходы от какао-­бобов и коноплю.
Напомним, что на фестивале технологий хабаровчанам рассказали, как колбаса и вегетарианство влияют на бизнес. На DеvFest, прошедшем в городе, хабаровчанам представили несколько уникальных вариантов заработка.
