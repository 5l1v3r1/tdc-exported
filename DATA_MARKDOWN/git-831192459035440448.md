
# Μπέσσυ Αργυράκη: Κληρονόμησα 20.000€ και τα ξόδεψα μέσα σε 7 μέρες στη Μύκονο

Published at: **2019-11-07T17:00:33+00:00**

Author: ****

Original: [europost.gr](https://europost.gr/mpessy-argyraki-klironomisa-20-000e-kai-ta-xodepsa-mesa-se-7-meres-sti-mykono/)

Η Μπέσσυ Αργυράκη αποκάλυψε μιλώντας σε τηλεοπτική εκπομπή ότι μέσα σε 7 μέρες ξόδεψε στη Μύκονο την κληρονομιά εξ Αμερικής, 20.000 ευρώ
«Σας πήρα τηλέφωνο για αυτό που άκουσα» ανέφερε χαρακτηριστικά η Μπέσσυ Αργυράκη στην Κατερίνα Καραβάτου και τον Κρατερό Κατσούλη.
«Θέλω να καταγγείλω και να πω ότι τα χάλασα, ναι, σε 7 ημέρες. Ήμουν με την κορούλα μου και μια αγαπημένη μου φίλη, βγήκαμε, ψωνίσαμε… Μπορεί να μην ήταν 20.000, να ήταν λιγότερα, αλλά αποφάσισα αυτά να τα χαλάσω» συμπλήρωσε η δημοφιλής ερμηνεύτρια.
«Να ψωνίσουμε ότι μας αρέσει, να βγούμε και να περάσουμε ωραία. Σαν να ήταν ταγμένα για εκεί. Κάποια χρήματα που δεν περίμενα ότι θα έρθουν και τα απήλαυσα με αυτόν τον τρόπο»
