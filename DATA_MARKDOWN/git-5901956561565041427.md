
# Na poti na poroko umorili tri matere in šest otrok

Published at: **2019-11-05T19:37:00+00:00**

Author: **P. K.**

Original: [Slovenske novice](https://www.slovenskenovice.si/kronika/na-tujem/clanek/masaker-na-poti-na-poroko-umorili-tri-mamice-in-sest-otrok-246013)

LA MORA – Dawne Ray Langford (43), Christine Marie Langford (31) in Rhonito Marie LeBaron (33) so ubili v La Mori v mehiški zvezni državi Sonora, poleg njih pa še šest otrok (med njimi celo Rhonitina šestmesečna dvojčka). Vsi so bili ameriški državljani, člani mormonske skupnosti, in so se vračali s poroke.Med potjo so omenjeno deveterico napadli člani mamilarskega kartela in jim prerešetali osebna vozila, poroča The Sun. Eno je tudi zagorelo in ga je razneslo. O tem, kaj je bil razlog za tako grozovit zločin, ne poročajo.Koliko je človeško življenje vredno v Mehiki, sicer najzgovorneje priča podatek, da so denimo v letu 2018 uradno (po podatkih mehiškega notranjega ministrstva) sprožili kar 33.341 preiskav umorov zaradi narkokartelnih obračunov.
