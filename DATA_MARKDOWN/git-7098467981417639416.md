
# Абхазский боксер Игорь Адлейба даст пресс-конференцию в Sputnik

Published at: **2019-11-04T11:31:00+00:00**

Author: ****

Original: [Sputnik Абзахия](https://sputnik-abkhazia.ru/Abkhazia/20191104/1028749975/Abkhazskiy-bokser-Igor-Adleyba-dast-press-konferentsiyu-v-Sputnik.html)


        В мультимедийном пресс-центре Sputnik Абхазия 5 ноября в 12:00 состоится пресс-конференция чемпиона мира по версии WBO среди молодежи Игоря Адлейба.
      
Пятого ноября в 12:00 в мультимедийном пресс-центре Sputnik Абхазия состоится пресс-конференция чемпиона мира по версии WBO среди молодежи Игоря Адлейба.
Второго ноября в Екатеринбурге абхазский боксер Игорь Адлейба завоевал титул чемпиона мира по версии WBO среди молодежи на турнире RCC Boxing Promotions.
В качестве спикера также выступит тренер боксера Беслан Адлейба.
Приглашаем представителей СМИ принять участие в пресс-конференции, которая пройдет в мультимедийном пресс-центре Sputnik Абхазия по адресу: город Сухум, улица Пушкина, 16 (бизнес-центр "Гудоу Плаза", пятый этаж).
