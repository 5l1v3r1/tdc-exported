
# Мобильный Office с фотосканером, дата выхода Microsoft Edge на базе Chromium и другие анонсы конференции Ignite 2019

Published at: **2019-11-04T14:50:26+00:00**

Author: **Евгений Делюкин**

Original: [vc.ru](https://vc.ru/services/90979-mobilnyy-office-s-fotoskanerom-data-vyhoda-microsoft-edge-na-baze-chromium-i-drugie-anonsy-konferencii-ignite-2019)

Коротко о новых сервисах и изменениях для пользователей и бизнеса, объявленных на Microsoft Ignite 2019.
Microsoft запустила предварительную версию нового приложения Office для мобильных устройств, в котором объединила функции Word, Excel, PowerPoint и Office Lens. По словам представителей компании, теперь пользователям «не придётся загружать по отдельности каждое приложение для продуктивной работы на ходу».
Кроме работы с пакетом офисных приложений в обновлённом Office можно добавлять заметки, создавать и подписывать PDF-документы, сканировать QR-коды и сканировать документы и таблицы, автоматически перенося информацию с фотографий в Word и Excel.
С 4 ноября пользователям доступна публичная превью-версия приложения для Android и iOS.
Запуск Chromium-версии Microsoft Edge состоится 15 января 2020 года. Браузер будет доступен на Windows 7, Windows 8, Windows 10, macOS, iOS и Android. Приложения на всех устройствах пользователя получат получат общую синхронизацию паролей, настроек, избранного и истории.
4 ноября выйдет предрелизная версия браузера Microsoft Edge Release Candidate. Ранее компания представила новый логотип браузера — он полностью отошёл от «наследия» Internet Explorer и выполнен в фирменном стиле Microsoft Fluent Design.
