
# కృష్ణాజిల్లాలో గ్రూపు రాజకీయాలు ఉన్నాయి: టీడీపీ నేత

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=943716)

Ph: 88865 47666ప్రవేశం ఉచితం ADVT
కృష్ణా: జిల్లాలో నాలుగైదు నియోజకవర్గాల్లో గ్రూపు రాజకీయాలు ఉన్నాయని టీడీపీ నేత బచ్చుల అర్జునుడు తెలిపారు. ఈ నెలలో గ్రామకమిటీల ఎన్నికలు నిర్వహిస్తామని ఆయన చెప్పారు. వచ్చే ఎన్నికల్లో టీడీపీ గెలవాలంటే కమిటీలు అవసరమని సూచించారు. నియోజకవర్గాలకు విడిగా కమిటీలు ఏర్పాటు చేస్తున్నామన్నారు. టీడీపీ జిల్లా అధ్యక్షుడుగా ఎప్పటికప్పుడు చంద్రబాబుకు వివరంగా విషయాలు చెపుతూనే ఉన్నానని వ్యాఖ్యానించారు. నాలుగు నియోజకవర్గాలలో ఒక కమిటీ తిరిగి పార్టీలో ఉన్న సమస్యలను తీరుస్తారని చెప్పారు.
