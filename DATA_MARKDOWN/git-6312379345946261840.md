
# В Татарстане потушили пожар на заводе

Published at: **2019-11-03T06:18:20+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-03/v-tatarstane-potushili-pozhar-na-zavode)

В Татарстане произошел пожар на одном из заводов в Нижнекамске. Возгорание удалось оперативно потушить.
На площади 4 квадратных метров загорелся вакуумный газойль. Это произошло из-за разрушения фланцевого соединения, сообщает пресс-служба МЧС России.
Пожар был ликвидирован в 8:19. Никто не пострадал.
Технологический процесс производства на заводе не нарушен. Угрозы для окружающей среды нет.
