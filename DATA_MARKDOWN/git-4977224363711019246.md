
# Radio Slovenija: Pri Zidanem mostu se je iztiril vlak

Published at: **2019-11-06T06:10:23+00:00**

Author: ****

Original: [STA.si](https://www.sta.si/2694441/radio-slovenija-pri-zidanem-mostu-se-je-iztiril-vlak)


        Zidani Most, 6. novembra - Pri Zidanem mostu se je davi iztiril vlak. Poškodovan ni nihče, gmotna škoda bo po prvih ocenah velika. Okoliščine in vzrok nesreče še preiskujejo, poroča Radio Slovenija.
