
# Тобольские полицейские нашли похищенного кота

Published at: **2019-11-05T10:51:59+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-05/tobolskie-policeyskie-nashli-pohishchennogo-kota)

Тобольские полицейские нашли похищенного кота. Об этом пишет "Российская газета".
В полицию с заявлением о пропаже питомца породы британская короткошерстная обратился мужчина. По его словам, кот по кличке Веня был похищен у него из офиса.
При помощи камер видеонаблюдения сотрудники правоохранительных органов установили, что животное забрала неизвестная женщина. Полицейским удалось найти подозреваемую, она созналась в краже и вернула Веню хозяину. По ее словам, кот ей очень понравился, поэтому она решила забрать его себе.
В отношении женщины следователи возбудили уголовное дело по статье "Кража".
