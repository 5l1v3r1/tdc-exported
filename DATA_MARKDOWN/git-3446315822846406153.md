
# Для першокласників Приютівської громади придбали мультимедійні дошки

Published at: **2019-11-04T00:00:00+00:00**

Author: **Наталка Маринець**

Original: [Кировоградская правда](https://pravda-kr.com.ua/najsvizhishe/16598-dlya-pershoklasnikiv-priyutivskoji-gromadi-pridbali-multimedijni-doshki.html)

У Приютівській громаді придбали по одному комплекту мультимедійного обладнання для Ізмайлівської та Войнівської шкіл, а також два комплекти для Користівського ліцею, повідомляють у громаді До кінця тижня, на мультимедійних дошках налагодять програмне забезпечення і встановлять їх у навчальних кабінетах перших класів.
Нагадаємо, раніше такі дошки придбали для першокласників Цукрозаводської, Протопопівської, Головківської і Бутівської шкіл.Відтепер усі кабінети перших класів у навчальних закладах Приютівської громади забезпечені мультимедійними дошками. Їх використання дасть можливість ефективніше проводити заняття, що відповідає вимогам реформи «Нової української школи».
