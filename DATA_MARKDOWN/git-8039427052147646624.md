
# కొనసాగుతున్న దేవరకొండ బంద్

Published at: **2019-11-04T08:28:56+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=946074)

ADVT
నల్గొండ: మరో ఆర్టీసీ కార్మికుడు మృతి ఆర్టీసీ కార్మికులను తీవ్ర విషాదంలో ముంచేసింది. దేవరకొండలో గుండెపోటుతో జైపాల్‌ రెడ్డి అనే ఆర్టీసీ డ్రైవర్ మృతి చెందాడు. ఈ నేపథ్యంలో ఆర్టీసీ కార్మికులు దేవరకొండ బంద్‌కు పిలుపునిచ్చారు. నేడు దేవరకొండ బంద్ ప్రశాంతంగా కొనసాగుతోంది. నిన్న మృతదేహంతో డిపో ఎదుట ఆర్టీసీ కార్మికులు ఆందోళన నిర్వహించారు. పరామర్శించేందుకు వచ్చిన డిపో మేనేజర్‌ను సైతం అడ్డుకున్నారు. జైపాల్‌రెడ్డి మృతదేహానికి విపక్ష నేతలు నివాళులర్పించారు.
