
# 仏女優マリー・ラフォレさん死去…「太陽がいっぱい」で脚光

Published at: **2019-11-04T09:30:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/culture/20191104-OYT1T50108/)

　ＡＦＰ通信によると、フランスの女優マリー・ラフォレさん（８０）が２日、スイス西部で死去した。死因は明らかにされていない。１９３９年、仏西部スラック・シュル・メール生まれ。初出演となった映画「太陽がいっぱい」（６０年）でアラン・ドロンさんの相手役を演じ、脚光を浴びた。６０～７０年代にシャンソン歌手としても活躍した。国籍を取得したスイスで暮らしていた。（パリ支局）
