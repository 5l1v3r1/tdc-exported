
# Ongeluk zorgt voor lange file op de A6

Published at: **2019-11-04T06:26:00+00:00**

Author: ****

Original: [omroep flevoland](https://www.omroepflevoland.nl/nieuws/175507/ongeluk-zorgt-voor-lange-file-op-de-a6)


        Een ongeluk waarbij meerdere voertuigen betrokken waren, heeft maandagochtend voor een lange file gezorgd op de A6 tussen Emmeloord en de Ketelbrug. Op het hoogtepunt was de wachttijd opgelopen tot anderhalf uur.
      
De aanrijding gebeurde rond 06.15 uur bij Emmeloord. De rijbanen werden afgesloten en het verkeer moest over de vluchtstrook.
De ANWB adviseerde automobilisten om te rijden via de N50, de A28 en de A1 (richting Zwolle-Amersfoort). Rond 08:00 uur was de weg weer vrij.
