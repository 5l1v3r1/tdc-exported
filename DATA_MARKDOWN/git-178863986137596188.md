
# Синоптики рассказали, когда в регионе начнется снижение температуры

Published at: **2019-11-07T06:39:00+00:00**

Author: **Владимир Андреев**

Original: [РИА Мелитополь](https://ria-m.tv/news/170837/sinoptiki_rasskazali_kogda_v_regione_nachnetsya_snijenie_temperaturyi.html)


        Бабье лето, к сожалению, не будет долгим.
      
Уже с воскресенья начнется постепенное снижение температуры. 
Сегодня ожидается переменная облачность, без осадков, местами туман, ветер юго-западный 7-12 м / с, температура воздуха ночью 8-13 ° тепла, днем 15-20 ° тепла.
