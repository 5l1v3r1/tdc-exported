
# Челси ўз тарихига яна бир салбий кўрсаткични ёздириб қўйди

Published at: **2019-11-05T21:33:09+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/chelsi-oz-tarixiga-yana-bir-salbiy-korsatkichni-yozdirib-qoydi)

"Челси" учрашувнинг биринчи бўлимида 1:3 ҳисобидаги мағлубият билан якунлади. Шу тариқа "аристократлар" ўз тарихларида илк бор, ЕЧЛ доирасидаги баҳснинг биринчи таймида, ўз дарвозасидан 3 та тўп ўтказиб юборди.
