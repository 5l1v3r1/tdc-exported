
# В Омске работы на фасаде у ж/д вокзала едва не закончились трагедией

Published at: **2019-11-01T13:16:00+00:00**

Author: ****

Original: [БК55](https://bk55.ru/news/article/159435/)


        Есть вероятность, что рабочий больше не сможет ходить.
      
О страшном случае, произошедшем накануне в Ленинском округе, со ссылкой на представителя медицинского учреждения сообщает «СуперОмск». По данным СМИ, во время выполнения работ по реставрации фасада дома № 3 по улице Лобкова с высоты упал рабочий. Предполагается, что мужчина в тот момент находился на высоте около пяти метров.
С очень серьезными травмами, которые в некоторых случаях могут оказаться фатальными, 60-летнего омича доставили в БСМП-1.
Первичная работа стражей порядка по этим обстоятельствам выявила один любопытный факт: оказалось, что вызвали медиков спустя несколько часов после предполагаемого несчастного случая.
Евгений Куприенко
