
# Чуют синтетику

Published at: **2019-11-01T15:39:00+00:00**

Author: **zakon.kz**

Original: [yk.kz](https://yk.kz/news/show/53427)

В Казахстане воспитали собак, способных учуять синтетические наркотики. По словам сотрудников МВД, воссоздать запах наркотических веществ, созданных в лабораторных условиях, достаточно сложно, а потому раньше таких собак в республике не было, передает корреспондент Zakon.kz.
- В каждой области есть центр кинологической службы, в Алматы есть кинологические центры, в которых работают наши специалисты – кинологи. В каждом регионе сейчас по одной собаке мы уже обучили, дополнительно в этом направлении мы работаем, проводим тренинги, обучаем и кинологов, и собак. Такие собаки есть и у департамента полиции на транспорте, поэтому они есть в аэропортах и вокзалах, - рассказал на пресс-конференции начальник департамента по противодействию наркопреступности МВД Казахстана Тлеген Маткенов.
На обучение одной собаки у кинологов уходит не менее полугода. Потребность каждого района – по 3 собаки. Цифру в МВД вывели из раскрываемости. На данный момент, полицейские находят по 10-15 так называемых "закладок" синтетических наркотиков. И раз главные помощники человека работают посменно: отдежурили суточную смену – положено отдыхать, то три собаки на одну область, по мнению Маткенова, должно хватить.
- На данный момент практически по всем фактам выявления синтетических наркотиков участвуют собаки. Для подтверждения, для поиска этих "закладок", собаки пригождаются по всем параметрам, - отметил он еще раз заслуги четвероногих.
