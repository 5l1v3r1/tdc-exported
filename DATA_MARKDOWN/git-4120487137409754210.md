
# 成功企业家张金发伉俪 令郎蔚辰君与张凤珊小姐结婚礼隆重举行

Published at: **2019-11-04T03:18:00+00:00**

Author: ****

Original: [Shangbao Indonesia](http://shangbaoindonesia.com/read/2019/11/03/special-news-1572794197)

【本报周孙毅报道】11月2日晚，我国成功企业家PT.Dwinaga Sakti Abadi、PT.Inti Prima Indonesia (Homyped)总裁、雅加达龙岩同乡会永久荣誉会长张金发先生和夫人何玲玲女士令满郎蔚辰君，与张目真先生和夫人赖美璇女士令长媛凤珊小姐之结婚典礼，在雅加达Mulia国际大酒店礼堂举行。     婚礼隆重，贺客盈门，来自国内外亲朋戚友，各国使节代表，华社领导，工商企业界人士，同乡梓叔等，逾千人出席酒会。     与会来宾除了向男女双方主婚人表达诚挚的祝福外，也祝愿这对新郎新娘恩爱美满、白头偕老、早生贵子。
