
# Nesreča zaprla pomursko avtocesto

Published at: **2019-11-06T12:00:00+00:00**

Author: **T. V.**

Original: [Slovenske novice](https://www.slovenskenovice.si/kronika/doma/clanek/nesreca-zaprla-pomursko-avtocesto-246526)

TURNIŠČE – Pomurska avtocesta je zaradi prometne nesreče, ki se je malo pred 12.30 zgodila med priključkoma Turnišče in Gančani proti Mariboru, zaprta. Nastaja zastoj.Za osebna vozila je možen obvoz po regionalni cesti Turnišče–Renkovci–Gančani, poroča prometnoinformacijski center.
