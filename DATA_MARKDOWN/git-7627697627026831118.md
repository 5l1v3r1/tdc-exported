
# Référendum pour sauver le vieux collège d’Hauterive

Published at: **2019-11-05T17:16:00+00:00**

Author: ****

Original: [RTN](https://www.rtn.ch/rtn/Actualite/Region/20191105-Referendum-pour-sauver-le-vieux-college-d-Hauterive.html)

Le vieux collège d’Hauterive devrait être détruit avant la fin de l’année. Le 23 septembre 2019, un arrêté du Conseil général mandatait le Conseil communal pour demander l’autorisation de démolir le vieux collège.
Des citoyens viennent cependant de lancer un référendum, selon le Bulletin des communes de Neuchâtel. L’objectif : obliger les autorités communales à organiser une votation populaire à ce sujet. Les référendaires ont jusqu’au 13 novembre pour déposer 220 signatures auprès de l’Administration communale. /ara
