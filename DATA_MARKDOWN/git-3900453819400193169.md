
# Опубликованы новые стиллы дорамы «Поймать Ю Рён»

Published at: **2019-11-06T15:08:11+00:00**

Author: **gunyasha**

Original: [YesAsia](https://www.yesasia.ru/article/754618)

Канал tvN опубликовал новые стиллы дорамы «Поймать Ю Рён» (другое название «Поймать призрака»).
Смотрите также: Опубликованы новые стиллы дорамы «Поймать Ю Рён»
С сюжетом можно ознакомиться здесь. Кажется, что сотрудничество с Ю Рён (Мун Гын Ён) заставляет Го Чжисока (Ким Сон Хо) раскрываться с совершенно неожиданных сторон, проявляя всевозможные таланты, которые поражают даже его самого. К сожалению, зачастую это заканчивается его травмами.
Дорама транслируется с 21 октября по понедельникам и вторникам в 21:30 по корейскому времени.
