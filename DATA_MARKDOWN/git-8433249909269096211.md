
# Auf diesen Samsung-Geräten läuft bald kein Netflix mehr

Published at: **2019-11-05T17:48:53+00:00**

Author: ****

Original: [Bild](https://www.bild.de/digital/2019/handy-und-telefon/samsung-auf-diesen-geraeten-laeuft-bald-kein-netflix-mehr-65836554.bild.html)


        Eine technische Bevormundung, die einfach nur nervt!
      

        Netflix unterstützt ab 1. Dezember einige Samsung-Geräte nicht mehr. Das gab Samsung auf der Website des Unternehmens bekannt. Grund: „technische Einschränkungen“.
      
Betroffen seien einige ältere Smart TVs. Bedeutet: Leute, die ein älteres Smart-TV-Modell besitzen, müssen ein neues kaufen – oder auf einen anderen Streamingdienst umsteigen.
Künftig könne aber „von vielen anderen Geräten“, die eine Verbindung zu einem Fernseher herstellen können, das Netflix-Angebot abgerufen werden, heißt es in der Mitteilung von Samsung.
Und weiter: „Sofern Sie über eines der unterstützten Geräte verfügen, wie z.B. eine Spielekonsole, einen Streaming-Media-Player oder eine Set-Top-Box, können Sie Netflix weiterhin auf Ihrem Fernseher sehen.“
