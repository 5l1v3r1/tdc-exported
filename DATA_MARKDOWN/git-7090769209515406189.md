
# Какую работу ищут в Амурской области: составлен топ-10 запросов от соискателей

Published at: **2019-11-06T00:46:18+00:00**

Author: ****

Original: [Амурская Служба Новостей: новости с характером](https://asn24.ru/news/society/69815/)

Аналитики сервиса HeadHunter составили топ-10 самых популярных запросов соискателей из Амурской области, показывающий какую работу ищут в регионе.Оказалось, что чаще всего в Приамурье ищут вакансии для бухгалтера, только за один месяц на сайте таких запросов было более 1 300. На второй строчке по популярности водители (более 500 запросов), на третьей — инженеры (почти 450 запросов). Кроме того, жители Амурской области активно ищут предложения для программиста, директора, механика, юриста, уборщицы и инженера ПТО.
— Конечно, запросов гораздо больше. Например, в Амурской области 166 раз за месяц искали вакансии для торговых представителей, 137 раз — для главных бухгалтеров, 126 раз — для экономистов, — рассказала руководитель пресс-службы HeadHunter ФО Елана Таращук.Еще кандидатов интересуют вакансии с определенным режимом работы: временная (124), вечерняя (75), удаленная работа (50).Анализ запросов показал, что кандидаты ищут не только работу. Например, 445 пользователей искали «ничего не делать», «спать» — 226, «играть» — 111. А счастье и надежду пытались найти 875 посетителей.
