
# В Минюсте нашли ящики со скрытыми письмами

Published at: **2019-11-05T14:03:22+00:00**

Author: ****

Original: [ФАКТИ](https://fakty.com.ua/ru/ukraine/20191105-u-min-yusti-znajshly-yashhyky-z-pryhovanymy-lystamy/)
