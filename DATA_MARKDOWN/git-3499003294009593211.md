
# Министр здравоохранения Подмосковья покинула пост

Published at: **2019-11-07T13:55:00+00:00**

Author: ****

Original: [MOS NEWS](https://mos.news/news/vlast/ministr_zdravookhraneniya_podmoskovya_pokinula_post/)

Министр здравоохранения Московской области Татьяна Мухтасарова досрочно сложила с себя полномочия и ушла с поста, сообщает пресс-служба министерства здравоохранения Подмосковья.Обязанности министра здравоохранения теперь исполняет ее первый заместитель Екатерина Огнева.40-летняя Мухтасарова занимала должность главы Минздрава Подмосковья с июня 2019 года.
