
# Столкнувшиеся большегрузы закупорили несколько рядов на внутренней стороне КАД

Published at: **2019-11-03T19:45:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/739487-stolknuvshiesya-bolshegruzy-zakuporili-neskolko-ryadov-na-vnutrennei-storone-kad)


        Вечером 3 ноября в Санкт-Петербурге на внутренней стороне КАД произошло ДТП с участием двух фур. Встав поперек дороги, они закупорили проезд для другого транспорта.
      
Поздним воскресным вечером, 3 ноября, на внутренней стороне КАД на съезде на Москву столкнулись две фуры. Кадрами с места происшествия поделились очевидцы в группе «ДТП и ЧП | Санкт-Петербург | Питер Онлайн | СПб» «ВКонтакте».
Неизвестно, что стало причиной столкновения, и как один из большегрузов умудрился встать поперек дороги перпендикулярно движению транспорта. Тем не менее, несколько полос оказались плотно заняты. По словам очевидцев, сейчас в этом месте можно проехать только по левому ряду или через съезд на Москву. Выясняются обстоятельства произошедшего.
