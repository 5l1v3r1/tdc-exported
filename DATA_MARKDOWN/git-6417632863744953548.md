
# Сульшернинг иши туфайли Лингард устидан кулишмоқда

Published at: **2019-11-03T06:44:03+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/sulsherning-ishi-tufayli-lingard-ustidan-kulishmoqda)

Биринчи бўлим охирида Жошуа Кинг киритган гол мезбонларга ғалаба келтирди ва "Манчестер Юнайтед" навбатдаги омадсизликка учради.
Ушбу учрашувда қизиқ воқеа юз берди: Иккинчи бўлимда МЮ бош мураббийи Уле-Гуннар Сульшер Жесси Лингардга қоғоз бўлагини бериб, кўрсатмани шу тариқа тушунтиришга ҳаракат қилди.
Бироқ ушбу вазият туфайли ижтимоий тармоқларда Лингарднинг устидан кулишмоқда. Тармоқларда тарқалган суратларни эътиборингизга ҳавола қиламиз.

        "Ахлатнинг бир бўлагисан"
      

        "Гол уриш учун тўпни дарвозага теп"
      

        "Футболдаги ахлат бўлишни бас қил"
