
# Минфин обсуждает возможность фискализации операций по Kaspi Gold

Published at: **2019-11-01T08:23:00+00:00**

Author: ****

Original: [Власть](https://vlast.kz/novosti/35907-minfin-obsuzdaet-vozmoznost-fiskalizacii-operacij-po-kaspi-gold.html)

Министерство финансов обсуждает с Национальным банком и Kaspi Bank возможность фискализировать операции по Kaspi Gold, сообщил вице-министр финансов Руслан Енсебаев.
«Сегодня платежи по Kaspi Gold являются очень популярным механизмом. Мы этот вопрос сейчас обсуждаем с Нацбанком и самим Kaspi Bank. С точки зрения закона о платежных системах этот механизм соответствует требованиям, но с точки зрения Налогового кодекса и фискализации - это не совсем соответствует. Эти операции не фискализируются», - сказал Енсебаев на брифинге о применении онлайн контрольно-кассовых машин.
По его словам, Kaspi Bank скоро разработает своё решение, которое будет бесплатным и удобном для бизнеса, которое будут использовать как онлайн контрольно-кассовый аппарат.
«Механизм Kaspi Gold, который сегодня повсеместно используется, уже будет соответствовать требованиям комитета государственных доходов. То есть со временем Kaspi Gold тоже станет онлайн контрольно-кассовым аппаратом», - отметил Енсебаев.
