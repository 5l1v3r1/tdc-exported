
# Έδεσαν τα κάγκελα με σπάγκο στο πάρκο του Ροδινιού!

Published at: **2019-11-01T19:30:02+00:00**

Author: ****

Original: [Η ΡΟΔΙΑΚΗ](https://www.rodiaki.gr/article/424923/edesan-ta-kagkela-me-spagko-sto-parko-toy-rodinioy)


        Με σπάγκο έδεσαν τα κάγκελα που τοποθέτησαν σε μέρος που έχουν πραγματοποιηθεί έργα στο πάρκο του Ροδινιού.
      
Εκεί, λόγω επικινδυνότητας έπρεπε να τοποθετηθούν τα εν λόγω κάγκελα εως ότου αποκατασταθεί η όποια ζημιά.
Ωστόσο, αρνούμαστε να δεχτούμε ότι η μόνη λύση που μπορούσαν να βρουν είναι αυτή που φαίνεται και στη φωτογραφία που δημοσιεύει σήμερα η "Ροδιακή"...
Τα σχόλια δικά σας!

        Λ.Τ.
