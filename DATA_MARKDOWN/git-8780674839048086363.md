
# ഒറിജലിനെ വെല്ലുന്ന തരത്തിൽ അൻപതിന്റെ വ്യാജൻ; കുടുങ്ങി സാധാരണക്കാർ...

Published at: **2019-11-05T07:23:00+00:00**

Author: **മനോരമ ലേഖകൻ**

Original: [ManoramaOnline](https://localnews.manoramaonline.com/palakkad/local-news/2019/11/05/palakkad-fifty-rupees-fake-currecy.html)

പാലക്കാട്∙ ടൗണിൽ അൻപതിന്റെ കള്ളനോട്ടുകൾ വ്യാപകം. ഒറിജലിനെ വെല്ലുന്ന തരത്തിലാണ് വ്യാജൻ വിലസുന്നത്. രാത്രിയിലാണ് കൂടുതലും ഇവ പ്രചാരത്തിലെത്തുന്നത്.
സാധാരണക്കാരാണ് അധികവും കുടുങ്ങുന്നത്. കഴിഞ്ഞ ദിവസം ഓട്ടോഡ്രൈവർക്കു വാടകയിനത്തിൽ ലഭിച്ചത് അൻപതിന്റെ വ്യാജൻ.
