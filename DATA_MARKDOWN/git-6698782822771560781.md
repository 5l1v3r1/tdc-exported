
# 61% видеостримеров зарабатывают на трансляциях игр

Published at: **2019-11-06T13:58:00+00:00**

Author: ****

Original: [Computerworld Россия](https://www.computerworld.ru/news/61-videostrimerov-zarabatyvayut-na-translyatsiyah-igr)


        Этот рынок может достигнуть 1 млрд руб. в год.
      
По данным компании Wanta, 61% российских видеостримеров монетизируют свой контент, пишет «Коммерсантъ». Из них 96% используют для этого сайты для сбора пожертвований для авторов контента, 15% — подписки, 8% — платные чаты на YouTube, 6% — платежные системы. Заработок отечественных стримеров на донатах в первом полугодии 2019 года на 62,8% по сравнению с аналогичным показателем прошлого года, подсчитали в DonationAlerts.
Главными площадками для видеостриминга Wanta называет Twitch, YouTube и «ВКонтакте». При этом в Twitch и YouTube контент монетизируют 78% и 32% стримеров соответственно, а в «ВКонтакте» — только 8%. Зарабатывать по рекламной модели стримерам геймерам пока сложно: пользователи часто отписываются после стримов с встроенной рекламой. По мнению собеседников издания, хороший потенциал есть у бизнес-модели, по которой стримы используются для удержания пользователя в качестве подписчика.
Исследование Wanta проводилось в июле 2019 года на основе выборки из 112,3 тыс. стримеров.
