
# ปิดตลาดหุ้นไทยภาคเช้าลบ 0.11 จุด

Published at: **2019-11-07T05:32:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/economic/740424)


        หุ้นไทยแกว่งตัวผันผวน ก่อนดัชนีปิดลบ 0.11 จุด น้ำมันดิบเริ่มพลิกมาปรับตัวลง อาจมีแรงขายกลุ่มพลังงานทำกำไรระยะสั้น
      
ตลาดหุ้นไทยภาคเช้า 7 พ.ย.62 บรรยากาศการลงทุนช่วงเช้าดัชนีค่อนข้างผันผวน ส่งผลให้ปิดตลาดภาคเช้าดัชนีอยู่ที่ 1,623.88 จุด ลดลง 0.11 จุด มูลค่าการซื้อขาย 21,406.15 ล้านบาท โดย บล.ฟินันเซีย ไซรัส คาดดัชนีวันนี้จะแกว่งตัวบริเวณ 1,615 - 1,630 จุด โดยยังคงติดตามพัฒนาการของประเด็นการลงนามข้อตกลงทางการค้าระหว่างสหรัฐฯ-จีนซึ่งล่าสุดอาจถูกเลื่อนออกไปเป็นเดือน ธ.ค. ขณะที่ราคาน้ำมันดิบเริ่มพลิกมาปรับตัวลงหลังบวกติดต่อกัน 3 วันก่อนหน้าทำให้กลุ่มพลังงาน อาจมีแรงขายทำกำไรระยะสั้น ส่วนปัจจัยที่ต้องจับตาระยะนี้คือผลประกอบการ บริษัทจดทะเบียนไตรมาส 3/62
