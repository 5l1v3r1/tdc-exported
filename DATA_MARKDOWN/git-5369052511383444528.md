
# Реальность страшнее: возле метро Золотые Ворота проведут перформанс

Published at: **2019-11-01T14:25:00+00:00**

Author: **Александра Шмарова**

Original: [kiev.vgorode.ua](https://kiev.vgorode.ua/news/dosuh_y_eda/412249-realnost-strashnee-vozle-metro-zolotye-vorota-provedut-performans)

Целью страйка является привлечение внимания к экологическим проблемам. По словам активистов, не так страшен праздник Хеллоуин, как климатический кризис. Затопленные дома, голод, проблема с пресной водой, нехватка ресурсов, лесные пожары - все это реальность, которая в будущем только приобретет масштабы. 
Организаторы наденут костюмы, которые отобразят ужасы климатических проблем. Принять участие в необычном перформансе смогут все желающие. Для этого нужно написать активистам в Телеграм или Facebook.
