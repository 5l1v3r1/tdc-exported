
# 巨人・原監督「時間というものを大事に」ブルペンでは戸郷、與那原らの投球に注目

Published at: **2019-11-06T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/06/0012853611.shtml)

　「巨人秋季キャンプ」（６日、宮崎）
　巨人が６日、宮崎県総合運動公園内で宮崎秋季キャンプをスタートさせた。
　野手キャプテンに指名された岡本はスタート前の訓示。「今日は僕たちの練習を歓迎するかのような雲一つない青空ですね。この青空のもと元気を出してがんばりましょう」と円陣でスピーチし、初日の練習を行った。
　原監督は岡本、大城といった主力選手の打撃練習をケージ裏で見守り、ブルペンでは戸郷、與那原らの投球を真後ろで見続けた。
　「（キャンプメンバーに）選ばれたという部分でも、すごいわけだからね。球団も含めて期待しているわけだから、１日１日、時間というものを大事に。今のところ使っているようにみえるし、継続していくことが大事」と１９日までのキャンプを実りあるものにすることを願った。
