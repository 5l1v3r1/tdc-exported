
# Λίβανος : Οι μαζικές διαδηλώσεις δεν έχουν τέλος

Published at: **2019-11-04T19:50:19+00:00**

Author: **ΤοΒΗΜΑ Team**

Original: [Το Βήμα Online](https://www.tovima.gr/2019/11/04/vimatv/livanos-oi-mazikes-diadiloseis-den-exoun-telos/)

Δεκάδες χιλιάδες πολίτες κατέκλυσαν τους δρόμους της Βηρυτού. Για δέκατη ένατη μέρα οι Λιβανέζοι συνεχίζουν το τις διαδηλώσεις.
Η παραίτηση της κυβέρνησης Χαρίρι δεν οδήγησε στην παύση των κινητοποιήσεων.

        Διαβάστε ακόμη : Αυτοκίνητα - Έρχονται τα πάνω – κάτω στον κλάδο
      
Κύριο σύνθημα των διαδηλωτών είναι πλέον η επανάσταση. Το κύμα των διαδηλώσεων ξέσπασε στις 17 Οκτωβρίου όταν η κυβέρνηση του Λιβάνου επιχείρησε να φορολογήσει τη χρήση της εφαρμογής WhatsApp ενώ νωρίτερα είχε επιβάλει ένα πακέτο αυστηρών μέτρων λιτότητας.
Οι Λιβανέζοι θεωρούν υπεύθυνη την πολιτική ελιτ για την οικονομική κατάρρευση της χώρας και την απομύζηση του πλούτου του Λιβάνου. Οι κεντρικοί λεωφόροι της Βηρυτού είναι κλειστές.
Τα σχολεία οι επιχειρήσεις και οι τράπεζες δεν λειτουργούν.
