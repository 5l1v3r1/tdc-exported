
# Всероссийскую олимпиаду будут писать череповецкие школьники

Published at: **2019-11-07T07:44:00+00:00**

Author: ****

Original: [35медиа](https://35media.ru/news/2019/11/07/vserossijskuyu-olimpiadu-budut-pisat-cherepoveckie-shkolniki)

Всероссийскую олимпиаду будут писать череповецкие школьники. В городе стартовал муниципальный этап. 3 600 учеников с пятого по одиннадцатый класс примут в нем участие.
Открыла череду интеллектуальных испытаний олимпиада по литературе. Всего ребята могут показать углубленные знания по 21 предмету. Олимпиады проходят каждый день, кроме выходных. Результаты приходят уже на следующий день. Если ребенок не согласен с результатами, то в течение трех дней можно направить заявление в апелляционную комиссию. Наградят победителей в январе после новогодних праздников. Затем они отправятся представят город на областном этапе.
