
# 世界一ナショナルズ、トランプ氏を表敬…一部選手は拒否

Published at: **2019-11-05T01:35:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/mlb/20191105-OYT1T50118/)

　米大リーグのワールドシリーズを初制覇したナショナルズが４日、ホワイトハウスにトランプ大統領を表敬訪問した。第４５代大統領にちなみ、背番号４５のユニホームを贈られたトランプ氏は「この地域だけでなく全国の野球ファンの心を捉え、みんながナショナルズに夢中になった」と地元ワシントンのチームをたたえた。
　米スポーツ界ではトランプ大統領の人種差別的な発言や移民政策に反発して、ホワイトハウス表敬を拒否する選手が出ており、この日もドゥーリトル投手らが参加しなかった。（時事）
