
# “48グループ史上最高BODY”NMB48横野すみれ、『ヤンジャン』初登場でいきなり表紙

Published at: **2019-11-07T00:00:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148125/full/)

　アイドルグループ・NMB48の横野すみれ（18）が、7日発売の『週刊ヤングジャンプ』49号（集英社）の表紙に登場。この夏のグラビア界を席巻した“48グループ史上最高BODY”を存分に披露した。

        【写真】その他の写真を見る
      
　大阪出身の横野は、白間美瑠に憧れて2018年7月にNMB48に6期生として加入。現在は研究生だが、雑誌で披露した水着グラビアがネットで話題となり、ツイッターは開設3日でフォロワー2万人を突破した。グラビアアイドルの川崎あやも「グラビアアイドルからしたら間違いなく脅威的な存在になると思うんだけど、それ以上にこんなに可愛くてスタイル良い子が水着になってくれるなんて有難や〜」と絶賛している。 　次世代グラビアクイーンに王手をかけた美少女が、『ヤンジャン』初登場にしていきなり表紙＆巻頭グラビアに大抜てき。“全男子が待ち望んだハイスペックナイスバディ”をたっぷり見せつけ、グラビア新女王爆誕を大いに期待させるグラビアとなっている。 　また、同号の巻頭グラビアにはタレントの石田桃香（22）が登場。表紙を飾る28日発売の同誌52号の“予告編グラビア”が先行公開される。
