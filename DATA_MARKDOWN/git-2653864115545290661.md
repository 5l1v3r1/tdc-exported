
# Череповец присоединится к всероссийской акции «Ночь искусств»

Published at: **2019-11-03T08:43:00+00:00**

Author: ****

Original: [35медиа](https://35media.ru/news/2019/11/03/cherepovec-prisoedinitsya-k-vserossijskoj-akcii-noch-iskusstv)

Театрализованные экскурсии, мастер-классы, встречи с писателями. 3 ноября Череповец при поддержке компании «Северсталь» присоединится к всероссийской акции «Ночь искусств». Масштабную программу приготовили для горожан в историко-краеведческом музее.
Для гостей проведут необычную экскурсию «Звуки из прошлого». Путешествие по каждому залу музея будут сопровождать тематические мелодии. Череповчане смогут услышать звуки монастыря и дворянской усадьбы, строящегося города и технических достижений, Октябрьской революции и Великой Отечественной войны, чтобы наиболее полно погрузиться атмосферу времен и эпох. «Ночь искусств» порадует и книголюбов. После экскурсии в музее состоится встреча с писателем Александром Быковым, а также несколько мастер-классов по изготовлению закладок для книг и исторических шаржей. Кульминацией творческой ночи станет заложение капсулы времени в сквере Историко-краеведческого музея.
