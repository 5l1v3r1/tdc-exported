
# 広島・菊池涼、ＦＡ権行使せず…米球界挑戦に含み

Published at: **2019-11-01T10:43:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/npb/20191101-OYT1T50237/)

　国内フリーエージェント（ＦＡ）権を取得した広島の菊池涼は１日、権利を行使しない意向を鈴木球団本部長に伝えた。菊池涼は昨年の契約更改で、今オフにはポスティングシステムで米球界に挑戦する希望を球団に伝えていた。鈴木本部長は「午後に電話があった」とし、ポスティングについて「（申請は１２月５日までで）まだ日にちはある。これから話し合っていく」と語った。
