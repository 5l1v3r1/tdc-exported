
# బస్‌ పాసుల రాయితీలు కొనసాగుతాయ్: కేసీఆర్

హైదరాబాద్: ఆర్టీసీని ప్రభుత్వంలో విలీనం చేయడం తెలివి తక్కువ తనమని తెలంగాణ సీఎం కేసీఆర్ మరోసారి చెప్పుకొచ్చారు. ఒక వేళ ప్రభుత్వంలో విలీనం చేస్తే దుష్పరిణామాలు ఉంటాయన్నారు. ప్రైవేట్‌ రూట్లలో ఇష్టానుసారం చార్జీలు పెంచడానికి లేదని.. చార్జీల నియంత్రణ కమిటీ ఉంటుందన్నారు. బస్‌ పాసుల రాయితీలు కొనసాగుతాయని సీఎం తెలిపారు. ‘మేం బుల్డోజ్ చేయడం లేదు. యాజమాన్యం అదుపాజ్ఞల్లో పనిచేస్తేనే భవిష్యత్‌. లాభాలు వచ్చే రూట్లను ఆర్టీసీకి ఇస్తాం. కఠినమైన రూట్లను ప్రైవేట్ వాళ్లకు ఇస్తాం. ఆర్టీసీని విలీనం చేస్తే అంతటితో ఆగదు’ అని కేసీఆర్ తెలిపారు.
