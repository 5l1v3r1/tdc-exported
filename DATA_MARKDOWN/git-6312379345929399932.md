
# Бузовой дали леща на концерте в Ростове

Published at: **2019-11-04T08:48:56+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-04/buzovoy-dali-leshcha-na-koncerte-v-rostove)

Телеведущая, певица, писательница Ольга Бузова дала концерт в Ростове.
Она исполнила популярные хиты в одном из клубов, после чего - пообщалась с фанатами. По традции, о подарках, которые вручили ей поклонники. На этот раз кто-то выделился: в одной из коробок Бузова нашла леща.
