
# Microsoft создала AI, который заменит инспектора при сдаче на права

Published at: **2019-11-01T09:35:00+00:00**

Author: **Александра Кукуть**

Original: [dev.by](https://dev.by/news/microsoft-sozdala-ai-kotoryi-zamenit-inspektora-pri-sdache-na-prava)

Microsoft работает на проектом HAMS (Harnessing AutoMobiles for Safety), в рамках которого предлагает заменить автоинспекторов при сдаче экзаменов на права смартфоном и AI-системой. Это поможет экзаменуемым меньше нервничать за рулём, пишет The Next Web.
