
# Бывшая журналистка БК55 возглавила пресс-службу «Омскэнерго»

Published at: **2019-11-01T21:13:00+00:00**

Author: ****

Original: [БК55](https://bk55.ru/news/article/159458/)


        Перед приходом в энергокомпанию она работала в правительстве.
      
Бывший советник руководителя Главного управления информполитики региона Ирина Акишева получила новую должность. Теперь она будет взаимодействовать со СМИ в «Омскэнерго», сообщил «Новый Омск».
На этом посту Ирина сменит Евгению Фролову, которая с 2017 года являлась главным экспертом управления по региональной работе департамента по связям с общественностью ПАО «МРСК Сибири»-«Омскэнерго».
Акишева начинала свою карьеру в частности с работы в редакции БК55.
