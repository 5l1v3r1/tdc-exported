
# Два этажа эвакуировали над горящей квартирой в доме по Коломяжскому

Published at: **2019-11-06T00:37:00+00:00**

Author: **Алексей Громов**

Original: [Невские Новости](https://nevnov.ru/740196-dva-etazha-evakuirovali-nad-goryashei-kvartiroi-v-dome-po-kolomyazhskomu)


        Во вторник вечером загорелась квартира в многоквартирном жилом доме. Пьяного хозяина реанимировали с легким отравлением продуктами горения. Очевидцы в Сети поделились кадрами полыхающей квартиры.
      
Минувшим вечером вторника в Приморском районе по адресу Коломяжский проспект, дом № 1 произошел пожар. Загорелась квартира в жилом многоквартирном доме. В Сети появились новые кадры с места происшествия.
