
# 世貿組織授權中國 對美36億美元商品加徵關稅

Published at: **2019-11-02T07:17:00+00:00**

Author: **https://www.facebook.com/cnanewstaiwan, 中央通訊社**

Original: [中央社 CNA](https://www.cna.com.tw/news/firstnews/201911020010.aspx)

（中央社日內瓦1日綜合外電報導）一名貿易官員指出，世界貿易組織（WTO）仲裁人針對美國反傾銷措施引發的多年爭議，今天授權中國對美國每年價值高達35.8億美元進口商品加徵關稅。
中國先前就此案要求世貿組織許可，對美逾70億美元商品徵稅。
不過世貿組織裁定，非法的美國反傾銷措施已造成「中國利益喪生或減損累積」達35億7912.8萬美元，北京可以每年對不超過此總額的商品徵稅。
這項裁定標誌著，世貿組織首次針對貿易爭議授權中國加徵關稅。（譯者：徐睿承）1081102
