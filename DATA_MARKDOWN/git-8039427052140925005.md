
# చట్టానికి ఎవరూ అతీతులు కారని మరోసారి రుజువైంది: సోమిరెడ్డి

నెల్లూరు: జగన్‌పై సీబీఐ కోర్టు తీర్పును స్వాగతిస్తున్నామని టీడీపీ నేత సోమిరెడ్డి చంద్రమోహన్‌రెడ్డి వ్యాఖ్యానించారు. నేడు ఆయన మీడియాతో మాట్లాడుతూ.. చట్టానికి ఎవరూ అతీతులు కారని మరోసారి రుజువైందన్నారు. ఒక్కసారి కోర్టుకు హాజరైతే ప్రభుత్వానికి రూ.60లక్షలు ఖర్చవుతుందని... సీఎం జగన్‌ కోర్టుకు తప్పుడు సమాచారం ఇచ్చారన్నారు. వ్యక్తిగత కేసులకు జగన్‌ ప్రభుత్వ సొమ్మును ఎందుకు వినియోగిస్తున్నారన్నారు. జగన్‌ కేసులను కోర్టులు త్వరితగతిన విచారించాలని సోమిరెడ్డి కోరారు.
