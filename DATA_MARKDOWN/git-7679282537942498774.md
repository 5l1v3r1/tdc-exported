
# UPSC CDS Result 2019: www.upsc.gov.in पर सीडीएस II रिजल्ट घोषित

Published at: **2019-11-01T12:09:56+00:00**

Author: **लाइव हिन्दुस्तान टीम,नई दिल्ली**

Original: [https://www.livehindustan.com](https://www.livehindustan.com/career/story-upsc-cds-result-2019-declared-here-is-direct-link-2826997.html)

UPSC CDS Result 2019: संघ लोक सेवा आयोग (यूपीएससी) ने सीडीएस-2 का रिजल्ट घोषित कर दिया है। उम्मीदवार www.upsc.gov.in पर जाकर या नीचे दिए गए लिंक पर क्लिक कर अपना रिजल्ट चेक कर सकते हैं। रिजल्ट पीडीएफ फॉर्म में दिया गया है। इंटरव्यू (एसएसबी) के लिए चयनित उम्मीदवारों का नाम व रोल नंबर पीडीएफ फाइल में डाला गया है। एसएसबी के लिए कुल 8120 उम्मीदवारों का चयन किया गया है। 
लिखित परीक्षआ 8 सितंबर, 2019 का आयोजित हुई थी।
