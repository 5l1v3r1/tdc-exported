
# Горбатюк может возглавить подразделение ГБР по расследованию дел Майдана — Труба

Published at: **2019-11-05T16:16:00+00:00**

Author: ****

Original: [НВ](https://nv.ua/ukraine/politics/gorbatyuk-mozhet-pereyti-v-gbr-novosti-ukrainy-50051773.html)

Об этом на брифинге во вторник, 5 ноября, заявил директор ГБР Роман Труба, передает Интерфакс-Украина.
По его словам, для получения должности в ГБР Горбатюку придется пройти конкурсный отбор на общих условиях.
«В ближайшее время будет создано в ГБР отдельное структурное подразделение, которое будет заниматься расследованием уголовных производств дел Майдана. Если Горбатюк изъявит желание принять участие в конкурсе — пожалуйста. Но условия для всех равные», — заявил директор ГБР.
По словам Трубы, Горбатюк лично ему не говорил, что хочет стать руководителем структурного подразделения по расследованию «майдановских дел».
Напомним, Горбатюка уволили с должности начальника управления спецрасследований ГПУ.
Горбатюк занимался расследованием дел о преступлениях во время Революции Достоинства.
Горбатюк отказался писать заявление на прохождение переаттестации по установленному руководством Генпрокуратуры образцу, поскольку, по его мнению, он не соответствует действующему законодательству.
После увольнения Сергей Горбатюк обвинил главу Офиса президента Андрея Богдана в стремлении прекратить расследование дел Майдана.
