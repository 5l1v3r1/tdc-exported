
# Типпетт - о матче с Сент-Луисом: для нас это будет хорошим испытанием

Published at: **2019-11-06T16:03:00+00:00**

Author: ****

Original: [Allhockey.ru](https://allhockey.ru/news/show/369202-Tippett_o_matche_s_Sent_Luisom_dlya_nas_eto_budet_horoshim_ispytaniem)

Главный тренер "Эдмонтона" Дэйв Типпетт поделился ожиданиями от матча регулярного чемпионата НХЛ с "Сент-Луисом". Команды в очном матче будут бороться за первое место в таблице Западной конференции."У действующего чемпиона всегда есть особенная аура. Я уверен, "Сент-Луис" каждый матч сталкивается с этим. У них хороший глубокий состав, они играют определенным образом каждую встречу, и я понимаю, как они будут играть с нами. Для "Эдмонтона" это будет хорошим испытанием", - сказал Типпетт.
