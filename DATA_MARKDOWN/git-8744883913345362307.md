
# Льюис Хэмилтон стал шестикратным чемпионом «Формулы-1»

Published at: **2019-11-03T21:06:54+00:00**

Author: ****

Original: [Коммерсантъ](https://www.kommersant.ru/doc/4148262)

Британский пилот Льюис Хэмилтон из команды Mercedes стал шестикратным победителем чемпионата мира по автогонкам в классе «Формула-1». По количеству титулов британец уступает только семикратному немецкому чемпиону Михаэлю Шумахеру.
В прошедшей Гран-при США Льюис Хэмилтон занял второе место. Первым стал его партнер по команде финн Влаттери Боттас. Третье место занял пилот Red Bull Макс Ферстаппен. Российский пилот Toro Rosso Даниил Квят стал 13-м.

        Подробнее о победе Льюиса Хэмилтона в материале “Ъ” «Британская шестерка».
