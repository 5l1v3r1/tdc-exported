
# Microsoft представила логотип нового браузера Edge на движке Chromium

Published at: **2019-11-02T20:23:09+00:00**

Author: **Евгений Делюкин**

Original: [vc.ru](https://vc.ru/design/90943-microsoft-predstavila-logotip-novogo-brauzera-edge-na-dvizhke-chromium)

Теперь никакого сходства с Internet Explorer.
Новый логотип обнаружили пользователи предварительной версии Microsoft Edge в спрятанной мини-игре, посвящённой серфингу. Её можно вызвать командой edge://surf/.
После завершения игры появится объявление с новым логотипом.
Бета-версия Microsoft Edge была выпущена в августе 2019 года. Официальный релиз ожидается в конце 2019 года — начале 2020 года.
