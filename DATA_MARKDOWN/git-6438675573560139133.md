
# Рейс в Доминиканскую Республику вынужденно сел в Шереметьево

Published at: **2019-11-06T05:04:00+00:00**

Author: ****

Original: [РИА Новости](https://ria.ru/20191106/1560616880.html)

МОСКВА, 6 ноя — РИА Новости. Самолет Boeing 777 сообщением Москва — Пунта-Кана совершил посадку в аэропорту Шереметьево из-за технических проблем с одним из двигателей, сообщил РИА Новости представитель экстренных служб."В аэропорту Шереметьево благополучно приземлился Boeing 777, выполнявший рейс № 7571 из Москву в Пунта-Кана (Доминиканская Республика)", — сказал собеседник агентства.По его словам, после взлета в кабине пилотов сработал датчик давления масла в одном из двигателей. По предварительным данным, на борту воздушного судна находились 488 человек."Самолет благополучно приземлился, обстоятельства инцидента выясняются", — добавил собеседник агентства.
