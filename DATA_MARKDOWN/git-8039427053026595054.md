
# ఇసుక కొరతతో మరో కార్మికుడి మృతి

గుంటూరు: ఏపీలో ఇసుక కొరతకు మరో కార్మికుడు బలయ్యాడు. తాడేపల్లి మండలం ఉండవల్లిలో నాగరాజు అనే తాపీ మేస్త్రీ ఇంట్లో ఉరేసుకుని ఆత్మహత్య చేసుకున్నాడు. పనులు లేకపోవడంతో ఆర్థిక ఇబ్బందుల వల్లే నాగరాజు ఆత్మహత్య చేసుకున్నాడని అతని కుటుంబ సభ్యులు తెలిపారు.
