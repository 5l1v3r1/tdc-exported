
# 3 ноября в Тульской области будет облачно

Published at: **2019-11-02T23:30:00+00:00**

Author: ****

Original: [Тульская Служба Новостей](https://www.tsn24.ru/3-noyabrya-v-tulskoj-oblasti-budet-oblachno.html)

По информации Гидрометцентра России, в 3 ноября в Тульской области будет облачно с прояснением. Ночью преимущественно без осадков, днем небольшой дождь (до 1 мм за сутки). Ветер южный 6-11 м/с, днем местами порывы 12-17 м/с.
Температура воздуха по области ночью -3…+2ºС, днем +2…+7ºС. Атмосферное давление 740-745 мм рт. ст. Метеорологическая дальность видимости — 7000 м, во время осадков – 5000 м.
