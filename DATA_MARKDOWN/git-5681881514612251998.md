
# В Ильичёвске покончил с собой подросток

Published at: **2019-11-05T22:41:00+00:00**

Author: ****

Original: [ТАЙМЕР](http://timer-odessa.net/news/v_ilichevske_pokonchil_s_soboy_podrostok_761.html)


        16-летний житель Ильичёвска (Черноморска) покончил с собой, спрыгнув с 16 этажа жилого дома по улице Виталия Шума.
      
От травм, полученных в результате падения, парень скончался на месте. Иных повреждений на его теле не обнаружили, зато нашли предсмертную записку. 
Её содержание пока не разглашается.По предварительной информации, подросток хорошо характеризовался в школе, семья также была вполне благополучной. По информации местных СМИ, парень мог покончить с собой из-за несчастной любви.
