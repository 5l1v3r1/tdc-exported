
# Нападение астанчанина на автозеркала попало на видео

Published at: **2019-11-05T06:54:00+00:00**

Author: ****

Original: [Tengrinews.kz](https://tengrinews.kz/events/napadenie-astanchanina-na-avtozerkala-popalo-na-video-383320/)

Сотрудниками полиции района Есиль задержан мужчина, который во дворе жилого комплекса повредил несколько автомобилей, передает Tengrinews.kz со ссылкой на пресс-службу Департамента полиции столицы.
32-летний житель города попал в объективы камер видеонаблюдения. Мужчина разбил боковые зеркала четырех автомашин, припаркованных во дворе дома. В полицию обратились пока только двое пострадавших.
"Проводится досудебное расследование по статье 293 УК РК "Хулиганство", проверяется причастность задержанного к аналогичным преступлениям", - сообщили правоохранители.
