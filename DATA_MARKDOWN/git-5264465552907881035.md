
# 3 قتلى وسط بغداد في أكبر احتجاج منذ سقوط صدام

Published at: **2019-11-01T00:00:00+00:00**

Author: **دبي - العربية.نت**

Original: [العربية نت](https://www.alarabiya.net/ar/arab-and-world/iraq/2019/11/01/%D8%A7%D9%84%D8%B9%D8%B1%D8%A7%D9%82-%D8%A7%D8%B9%D8%AA%D8%B5%D8%A7%D9%85%D8%A7%D8%AA-%D9%85%D9%81%D8%AA%D9%88%D8%AD%D8%A9-%D9%81%D9%8A-%D8%AC%D9%85%D9%8A%D8%B9-%D8%A7%D9%84%D9%85%D8%AD%D8%A7%D9%81%D8%B8%D8%A7%D8%AA-%D8%A7%D9%84%D8%AC%D9%86%D9%88%D8%A8%D9%8A%D8%A9)

أفادت مصادر "العربية" بمقتل 3 متظاهرين عراقيين، بينهم امرأة، متأثرين بإصابة مباشرة بقنابل غاز مسيل للدموع، وذلك بعد توافد آلاف المحتجين إلى وسط العاصمة بغداد، الجمعة، وسط اعتصامات مفتوحة تشهدها جميع المحافظات الجنوبية.
