
# Red Dead Redemption 2: cómo arrancar el juego y solucionar el error de cierre inesperado en PC

Published at: **2019-11-05T21:06:00+00:00**

Author: **Frankie MB**

Original: [Vida Extra](https://www.vidaextra.com/guias-y-trucos/red-dead-redemption-2-como-arrancar-juego-solucionar-error-cierre-inesperado-pc?utm_source=feedburner&utm_medium=feed&utm_campaign=Feed%3A+vidaextra+%28VidaExtra%29)

Para muchos, la largamente soñada adaptación de Red Dead Redemption 2 a PC ha llegado de la mano de un problema inesperado: es bastante común que el juego se cierre incluso antes de mostrar el logo de Rockstar. Afortunadamente hay soluciones.
Si bien el lanzamiento del ambicioso Western de los creadores de GTA estaba programado para la tarde de hoy, 5 de noviembre, se ha comenzado a detectar una serie de incidentes comunes en la experiencia que impedían ejecutar el propio juego, incluso tras la pre-carga y decodificación de sus más de 100 GB.
Las opciones que nos ofrece el launcher de Rockstar se resumen en tres: Aceptar, Reintentar y Activar el Modo Seguro, sirviendo este último para que el juego arranque con unos ajustes gráficos reducidos. Una lástima si tenemos en cuenta el potentísimo despliegue visual prometido para esta adaptación.
