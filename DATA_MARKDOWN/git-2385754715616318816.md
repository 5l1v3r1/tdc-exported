
# Αυτόν ψάχνει η Αστυνομία για υπόθεση απάτης μέσω διαδικτύου (ΦΩΤΟ)

Published at: **2019-11-03T10:05:00+00:00**

Author: **SigmaLive**

Original: [SIGMALIVE](https://www.sigmalive.com/news/local/595501/afton-psaxnei-i-astynomia-gia-ypothesi-apatis-meso-diadiktyou-foto)

Η Αστυνομία καταζητεί τον Seun Tunbosun, 28 ετών, από τη Νιγηρία, για διευκόλυνση των ανακρίσεων, σχετικά με διερευνώμενη  υπόθεση απάτης μέσω διαδικτύου, που διαπράχθηκε μεταξύ Σεπτεμβρίου – Οκτωβρίου, 2019 στη Λεμεσό.  Παρακαλείται οποιοσδήποτε γνωρίζει οτιδήποτε που μπορεί να βοηθήσει στον εντοπισμό του, να επικοινωνήσει με το ΤΑΕ Λεμεσού στο τηλέφωνο 25805057 ή με τον πλησιέστερο Αστυνομικό Σταθμό, ή με τη Γραμμή Επικοινωνίας του Πολίτη στον αριθμό 1460.
