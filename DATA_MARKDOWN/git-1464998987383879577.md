
# സിപിഎം പ്രവര്‍ത്തകര്‍ക്കെതിരെ യുഎപിഎ; സർക്കാരിനെ വിമർശിച്ച് ചെന്നിത്തല

Published at: **2019-11-02T15:18:00+00:00**

Author: **Ruhasina J R**

Original: [Anweshanam | Part of A.N.N News Private Limited](http://anweshanam.com/index.php/kerala/news/uapa-ramesh-chennithala-against-ldf-government)

തിരുവനന്തപുരം: സിപിഎം പ്രവര്‍ത്തകര്‍ക്കെതിരെ യുഎപിഎ ചുമത്തിയ നടപടി പിണറായി സര്‍ക്കാരിന്റെ കിരാതമുഖം വെളിവാക്കുന്നതാണെന്ന് പ്രതിപക്ഷനേതാവ് രമേശ് ചെന്നിത്തല. ആശയപ്രചാരണം നടത്തുന്നവര്‍ക്കെതിരെയല്ല യുഎപിഎ പ്രയോഗിക്കേണ്ടത്. മാവോയിസ്റ്റുകളെ കൊന്നതില്‍ സിപിഎം പുലര്‍ത്തുന്ന മൗനം കള്ളക്കളിയാണെന്നും ചെന്നിത്തല തിരുവനന്തപുരത്ത് പറഞ്ഞു.
