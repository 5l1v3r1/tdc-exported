
# Самолет с неисправной гидросистемой сел в Шереметьево

Published at: **2019-11-04T14:15:04+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-04/samolet-s-neispravnoy-gidrosistemoy-sel-v-sheremetevo)

Пассажирский Boeing cел в аэропорту "Шереметьево". По данным РЕН ТВ, экипажу в штатном режиме удалось посадить лайнер с неисправностью гидросистемы. 
По информации агентства, лайнер выполняет рейс из немецкого Франкфурта-на-Майне. На борту находились 154 пассажира, из них трое - члены экипажа. 
Обстоятельства выясняются.
