
# Фото: крыша автосервиса загорелась в Томске

Published at: **2019-11-06T07:13:39+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-06/foto-krysha-avtoservisa-zagorelas-v-tomske)

В Томске произошел пожар в здании автосервиса.
По предварительным данным, загорелась крыша здания.
Площадь пожара составила примерно 600 квадратных метров.
По предварительным данным, в результате инцидента никто не пострадал.
В распоряжении РЕН ТВ появились фото с места пожара.
