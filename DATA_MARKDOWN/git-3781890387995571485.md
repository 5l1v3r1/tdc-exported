
# PLN x JIRADTแฟชั่นผสานสองซิกเนเจอร์สตรีทสไตล์

Published at: **2019-11-02T09:05:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/women/739645)


        Absolute Siam Store เปิดตัวคอลเลคชั่นพิเศษต่อเนื่อง “PLN x JIRADT” แฟชั่นผสานสองซิกเนเจอร์สตรีทสไตล์
      
 “Absolute Siam Store” (แอ็บโซลูท สยาม สโตร์) ศูนย์รวมสินค้าแฟชั่นไลฟ์สไตล์และของที่ระลึก นำเสนอสินค้าที่แตกต่างมีจำหน่ายเฉพาะที่สยามเซ็นเตอร์ เมืองแห่งไอเดียที่ล้ำเทรนด์ เพียงแห่งเดียว สร้างสรรค์ครีเอทด้วยดีไซน์แปลกใหม่จากการร่วมคอลลาบอเรชั่นระหว่างแบรนด์ไทยดีไซเนอร์ชื่อดัง หรือสินค้า Exclusive แบบไม่เคยมีมาก่อน ล่าสุด แอ็บโซลูท สยาม สโตร์ สร้างสีสันความสนุกสนานและแรงบันดาลใจให้กลุ่มเทรนด์ดี้ด้วยการนำเสนอคอลเลคชั่นสุดเอ็กซ์คลูซีฟ “PLN x JIRADT” (พีแอลเอ็น x จิรัฎฐ์) เป็นการคอลลาบอเรชั่นระหว่างแบรนด์สตรีทสไตล์ อย่าง  “PLN” กับ  “ปู” จิรัฎฐ์ พรพนิตพันธุ์ สไตล์ไอคอนของคนรุ่นใหม่ ผู้ก่อตั้ง Cheeze & Looker Magazine มีจำหน่ายเฉพาะที่แอ็บโซลูท สยาม สโตร์ ชั้น 1 สยามเซ็นเตอร์ เมืองแห่งไอเดียที่ล้ำเทรนด์
             PLN x JIRADT เกิดขึ้นจากการนำซิกเนเจอร์ของ PLN ที่เน้นความสนุกสนานแต่งตัวยุค Trendy ที่สามารถนำไอเทมของแบรนด์ไปมิกซ์แอนด์แมทช์กับเสื้อผ้าที่มีอยู่เดิมของแต่ละคนเพิ่มความชัดเจนในสไตล์ของตัวเอง เสื้อผ้าแต่ละคอลเลคชั่นจะนึกถึงความสนุกของการทำกิจกรรมต่างๆ เช่น คอลเลคชั่น Holiday Maker และ Staying Salty ที่มีดีไซน์ในโทนสีทรอปิคัล และน่าสวมใส่ไปเที่ยวทะเล หรือ คอลเลคชั่น  Happy Campers ที่จะมีความเป็น Camping Style แต่ก็ยังสามารถนำมาสวมใส่ในชีวิตประจำวันได้ มาผนวกเข้ากับสไตล์ไอคอนอย่าง “ปู – จิรัฎฐ์” ผู้ที่มีมุมมองทางความคิดและทัศนคติในการใช้ชีวิตที่แฝงไว้ด้วยแง่คิดและปรัชญา รวมไปถึงการแต่งกายที่เต็มไปด้วยอัตลักษณ์เฉพาะตัว สามารถหยิบจับเสื้อผ้าหลากสไตล์มามิกซ์แอนด์แมทช์ให้เข้ากันได้อย่างลงตัว
   
         ปารีสา จาตนิลพันธุ์ ผู้อำนวยการอาวุโส ฝ่ายพัฒนาธุรกิจค้าปลีก  บริษัท สยาม ซีเล็คเต็ด จำกัด เล่าถึงคอลเลคชั่นของ PLN x JIRADT ว่าเป็นการนำเสนอความสนุกและเรื่องราวชีวิตผ่านการคอลลาบอเรชั่นของกลุ่มคนที่ต่างคนต่างมากจากที่หลากหลายแล้วมาอยู่ร่วมกันในบ้านหลังเดียวกัน บ้านหลังใหญ่ที่ไม่มีรั้วหรือแผงกัน แต่เป็นบ้านที่เต็มเปี่ยมไปด้วยบรรยากาศการทำงานที่สนุกนานและสายสัมพันธ์ที่แน่นแฟ้นเสมือนคนในครอบครัว โดยมี “Hidden Backyard” เป็นคอนเซปต์หลักของการออกแบบ บอกเล่าถึงสนามหญ้าหลังบ้าน พื้นที่ที่ทุกคนในครอบครัวได้มาเล่นสนุก ทำกิจกรรมต่างๆร่วมกัน เสื้อผ้าที่ดีไซน์จึงต้องเป็นแบบที่ทุกคนที่อยู่ในสวนหลังบ้านสามารถสวมใส่ร่วมกันได้ จึงมาลงตัวที่เสื้อผ้าในรูปแบบฟรีสไตล์ใส่ได้ทั้งชายและหญิง เสื้อผ้าลำลองในโครงเสื้อที่โปร่งสบาย
อีกหนึ่งความพิเศษของคอลเลคชั่นนี้คือ ด้วยคำว่า Hidden Backyard ทำให้นึกถึงกลุ่มคนที่อยู่สวนหลังบ้านว่าถ้าเป็นบ้านของ JIRADT คนกลุ่มนี้จะเป็นใคร?  JIRADT ที่เติบโตมาในวงการแมกกาซีนเกือบ 20 ปี เส้นทางที่กว่าจะประสบความสำเร็จดังทุกวันนี้เป็นเพราะมีครอบครัวที่ดี มีทีมงานเบื้องหลังที่ทำหน้าที่บันทึกวัฒนธรรมการแต่งกายแฟชั่นสายสตรีทให้กับเมืองไทย ไม่ว่าจะเป็นช่างภาพ สไตลิสต์ คอลัมนิสต์ กองบรรณาธิการ กราฟิกดีไซเนอร์และฝ่ายต่างๆ ที่ทำงานร่วมกันมาตลอด ซึ่งกลุ่มคนเหล่านี้เปรียบเสมือนคนในครอบครัวของ JIRADT กลุ่มคนที่เล่นสนุกกันอยู่หลังบ้าน ความพิเศษของคอลเลคชั่นนี้คือการนำกลุ่มคนเหล่านี้มาถ่ายทอดความสัมพันธ์ของคนในครอบครัวผ่านการเป็นโมเดลถ่ายแบบของคอลเลคชั่นพิเศษนี้ จากเบื้องหลังสู่ด้านหน้า จากสวนหลังบ้านมาเปิดประตูเชิญชวนให้ทุกคนได้มารู้จักกัน
มาร่วมเป็นส่วนหนึ่งของครอบครัว PLN x JIRADT กับคอลเลคชั่นสุดพิเศษที่ Absolute Siam Store คัดสรรมานำเสนอให้เหล่าสตรีทสไตล์ได้สวมใส่แล้ววันนี้ที่ชั้น 1 สยามเซ็นเตอร์ เมืองแห่งไอเดียที่ล้ำเทรนด์
