
# Полиция Екатеринбурга проводит проверку по факту избиения кондуктора

Published at: **2019-11-03T17:36:36+00:00**

Author: ****

Original: [Коммерсантъ](https://www.kommersant.ru/doc/4148197)

Полиция в Екатеринбурге проводит проверку по факту избиения пассажиром кондуктора автобуса, сообщили в городском УМВД. Пассажир набросился с кулаками на кондуктора в салоне транспортного средства. Инцидент произошел в автобусе №25, когда тот находился в районе перекрестка 8 Марта—Малышева, передает e1.ru.
«Пострадавший кондуктор 1996 года рождения направлен на прохождение судебно-медицинского обследования для установления тяжести причиненного вреда здоровью»,— приводит «РИА Новости» выдержку из сообщения полиции.
