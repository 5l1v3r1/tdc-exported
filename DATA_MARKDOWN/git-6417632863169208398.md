
# U-19 Осиё Чемпионати-2020 саралашини ғалаба билан бошладик

Published at: **2019-11-06T14:29:53+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/u-19-osiyo-chempionati-2020-saralashini-galaba-bilan-boshladik)

Ўзбекистон U-19 терма жамоаси саралашнинг “F” гуруҳидан ўрин олган ва гуруҳда Саудия Арабистони, Ҳиндистон ҳамда Афғонистон терма жамоалари билан баҳс олиб боради. U-19 Осиё Чемпионати-2020 саралаш баҳслари Саудия Арабистонидаги Хобар шаҳрида ўтказилади.Илк учрашувни вакилларимиз ҳиндистонлик тенгқурларига қарши ўтказиб 2:0 ҳисобида ғалабага эришишди.
