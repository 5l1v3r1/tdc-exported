
# Verwirrung

Published at: **2019-11-05T09:50:32+00:00**

Author: ****

Original: [Mitteldeutsche Zeitung](https://www.mz-web.de/burgenlandkreis/verwirrung-33416076)

Kürzlich bekamen wir eine Einladung aus einer Behörde geschickt, die zum Pressetermin bat. Das Gespräch sollte 14 Uhr stattfinden. Ein paar Stunden später flatterte die nächste Mail ins Postfach, der Termin werde auf 16 Uhr geändert.
Ein paar Tage später kam eine Erinnerungsmail aus besagter Behörde. Nun hieß es wieder, das Gespräch finde 14 Uhr statt. Aber Pustekuchen, auch diese Mail wurde kurz drauf korrigiert. Sie ahnen es: auf 16 Uhr.
Wie gut, dass heutzutage der Kalender im Internet solche Zeitverschiebungen klaglos hinnimmt. Auf einem Schreibtischkalender in Papierform wäre das vermutlich so ein Durcheinander geworden, dass wir garantiert den Termin verpasst hätten.
