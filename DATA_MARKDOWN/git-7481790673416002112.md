
# 東京パラへ、車いすバスケ女子日本代表が合宿

Published at: **2019-11-01T11:21:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/para-sports/news/20191101-OYT1T50250/)

　東京パラリンピックに出場する車いすバスケットボール女子日本代表は１日、「ナショナルトレーニングセンター（ＮＴＣ）イースト」（東京都北区）での合宿を公開した。
　チームは今月下旬にタイで開幕する国際大会で強豪の中国や豪州などと対戦する。
　主将の藤井郁美は「中国に勝てば、（目標とする）東京パラの銅メダルが見えてくる」と意気込みを語った。
