
# Россиян обяжут выбрать тип трудовой книжки

Published at: **2019-11-07T19:54:00+00:00**

Author: ****

Original: [35медиа](https://35media.ru/news/2019/11/07/rossiyan-obyazhut-vybrat-tip-trudovoj-knizhki)

Россиян обяжут выбрать тип трудовой книжки. Комитет Госдумы по труду и соцполитике рекомендовал принять соответствующий законопроект.
Он предусматривает обязательное письменное волеизъявление человека о том, переходит он на электронную трудовую или остается с бумажной. При этом осуществить обратный переход будет невозможно. По данным СМИ, такое решение связано с позицией Главного правового управления президента. В случае окончательного принятия поправок бумажные трудовые книжки заменят на электронные в январе 2021 года. Сведения в Пенсионный фонд будут поступать в онлайн-режиме. Работник может не переходить на новую систему, для этого он должен подать соответствующее заявление в отдел кадров. Лицам, впервые поступающим на работу, бумажные трудовые книжки оформляться не будут.
