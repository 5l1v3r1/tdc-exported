
# リトグリ５周年ツアーファイナル　改修・代々木の「こけら」「落とし」絶叫

Published at: **2019-11-04T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/04/0012847056.shtml)

　女性ボーカルグループ・Ｌｉｔｔｌｅ　Ｇｌｅｅ　Ｍｏｎｓｔｅｒが３日、東京・国立代々木競技場第一体育館で、デビュー５周年記念ツアーのファイナルを迎えた。
　東京五輪へ向けた改修工事を終えての、こけら落とし公演。アサヒ（２０）は「『こけら』を背負って、『こけら』と言うから、みんなで『落とし』を」と客席の１万人と“レア”なコール＆レスポンス。五輪ではハンドボールで使用される会場に、ラグビーＷ杯のＮＨＫテーマソング「ＥＣＨＯ」など２３曲を響かせた。
