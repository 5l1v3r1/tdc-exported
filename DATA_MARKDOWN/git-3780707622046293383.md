
# Для нового ямальского проекта СПГ построят танкер в России

Published at: **2019-11-05T05:36:00+00:00**

Author: **Редакция ФедералПресс**

Original: [ФедералПресс](https://fedpress.ru/news/78/economy/2357502?utm_source=yxnews&utm_medium=desktop&utm_referrer=https%3A%2F%2Fyandex.ru%2Fnews)


        САНКТ-ПЕТЕРБУРГ, 5 ноября, ФедералПресс. Пилотный танкер-газовоз для нового ямальского СПГ проекта построят в России. Заказ получили судостроители Приморья.
      
Пилотный газовоз для нового проекта «Арктик СПГ2» построят на ССК «Звезда» в приморском поселке большой Камень. Соответствующее соглашение подписали ПАО «Совкомфлот» и предприятие. Лизинговое финансирование будет осуществлять группа ВЭБ.РФ. Всего судоверфь «Звезда» построит для проекта пятнадцать газовозов арктического класса.
Танкер-газовоз займется доставкой сжиженного газа с территории Гыданского полуострова потребителю. Два года назад «Совкомфлот» ввел в эксплуатацию первый арктический газовоз «Кристоф де Маржери». Судно осуществляет перевозку сжиженного газа с полуострова Ямал. Компания уже около десяти лет осуществляет перевозку грузов по Северному морскому пути.
«Арктик СПГ2» - второй по счету проект по производству сжиженного природного газа на Ямале. Как и в «Ямал СПГ», основным оператором является компания «НОВАТЭК». Ресурсной базой для гыданского проекта служит Салмановское нефтегазоконденсатное месторождение. Начало добычи газа запланировано на 2023 год. Со временем, когда «Арктик СПГ2» выйдет на проектную мощность, он должен будет обойти завод в Сабетте по годовому объему производства сжиженного газа.
