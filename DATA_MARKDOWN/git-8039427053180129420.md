
# లక్ష్మీపార్వతికి కీలక పదవి.. ప్రభుత్వం ఉత్తర్వులు జారీ

Published at: **2019-11-06T17:23:16+00:00**

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=948028)

అమరావతి: వైసీపీ నేత లక్ష్మీపార్వతికి కీలక పదవి దక్కింది. ఆమెను ఏపీ తెలుగు అకాడమీ చైర్ పర్సన్‌గా నియమిస్తూ ప్రభుత్వం ఉత్తర్వులు జారీ చేసింది. లక్ష్మీపార్వతి ప్రస్తుతం వైసీపీ ప్రధాని కార్యదర్శిగా ఉన్నారు.
