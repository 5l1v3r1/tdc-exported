
# Абромавичус: Ціноутворення для оборонних підприємств буде переглянуте

Published at: **2019-11-06T15:22:00+00:00**

Author: ****

Original: [ZIK.UA](https://zik.ua/news/2019/11/06/abromavychus_tsinoutvorennya_dlya_oboronnyh_pidpryiemstv_bude_pereglyanute_1685089)


        Гендиректор ДК «Укроборонпром» Айварас Абромавичус сказав, що ціноутворення для оборонних підприємств буде переглянуте.
      
Про це Айварас Абромавичус сказав під час робочої поїздка Володимира Зеленського на Харківщину, – інформує телеканал ZIK.
