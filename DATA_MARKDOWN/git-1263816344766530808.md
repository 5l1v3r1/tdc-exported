
# На сайте метро появился тарифный калькулятор поездок по МЦД

Published at: **2019-11-07T06:49:39+00:00**

Author: **Инна Лутовинова**

Original: [МОЛНЕТ.RU](https://www.molnet.ru/mos/ru/important/o_717681)


        Для того, чтобы рассчитать стоимость поездок, нужно открыть схему диаметров и построить маршрут. Сайт подскажет, сколько минут займет поездка в выбранном направлении, а ниже укажет стоимость.
      
На сайте московского метро появился тарифный калькулятор, который поможет посчитать стоимость поездки по городу с учетом МЦД, сообщает официальный портал мэра и правительства города. 
Также с помощью калькулятора можно сравнить, сколько пассажиры заплатят за проезд по ж/д направлениям, и сколько заплатят после открытия МЦД.
