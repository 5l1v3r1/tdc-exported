
# В Брянске сняли на видео проезд на красный маршрутки №35

Published at: **2019-11-05T13:04:38+00:00**

Author: ****

Original: [Брянск Today](https://bryansktoday.ru/article/111985)

В Брянске сняли на видео очередного летчика, видимо, по ошибке устроившегося водителем маршрутного такси. Ролик выложили в соцсети Вконтакте сегодня, 5 ноября. 
Водитель маршрутки №35 лихо промчался на красный сигнал светофора в районе диагностического центра возле Кургана Бессмертия. Номер героя - Т203ОВ 32.
- Он чуть не спровоцировал аварию, - добавил автор, выложивший видео с лихачом. Вероятно, его полет оценят по достоинству сотрдуники ГИБДД.
