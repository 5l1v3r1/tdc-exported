
# МЮнинг 3 муҳим футболчиси Борнмутга қарши ўйинни ўтказиб юбориши мумкин

Published at: **2019-11-01T12:00:04+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/myuning-3-muhim-futbolchisi-bornmutga-qarshi-oyinni-otkazib-yuborishi-mumkin)

Ҳужумчи Маркус Рэшфорд, ҳимоячилар Виктор Линделёф ва Харри Магуайрлар бу ўйинда енгил жароҳат олишди.
"Ҳали бу ҳақда гапиришга эрта. Умид қиламизки, улар эртанги ўйинда майдонга тушишади.
Улар кечадан бери тикланиш жараёнидалар ва бугун енгил машғулотлар олиб боришди. Умид қиламанки, улар тайёр бўлишади. Ҳозир эса аниқ бир гап айта олмайман", - дейди Сульшер.
Шунингдек, ўйинни ҳимоячиси Аксель Туанзебе ва хавбек Неманья Матичлар ўтказиб юборишади.
Эслатиб ўтамиз, "Борнмут" ва "Манчестер Юнайтед" ўртасидаги ўйин эртага Тошкент вақти билан 17:30да бошланади.
