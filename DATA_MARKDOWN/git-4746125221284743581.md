
# ADDICTION、デビュー曲が週間3位に アバンティーズ・エイジさん誕生日発売で兄のK.Bが感慨「より精進していきたい」

Published at: **2019-11-07T21:29:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148268/full/)

　5人組ダンス＆ボーカルグループのADDICTIONのメジャーデビューシングル「Further away/Destiny」が11／11付オリコン週間シングルランキングで3位を獲得した。 　ADDICTIONはK.B、MYUJI、ASCH、TAKT、JOSHUAの5人からなるダンス＆ボーカルグループ。結成5年にして同曲で悲願のメジャーデビューを果たした。10月29日と30日にはツイッターの日本トレンドで「＃ADDICTION」がランクインした。くしくも30日はメンバーのK.Bの実弟で、1月1日に休暇旅行先のサイパンで高波にさらわれ亡くなった人気YouTuber・アバンティーズのエイジさん（享年22）の誕生日でもあった。 　3位を獲得した勢いのまま、11月17日新潟公演を皮切りに初の全国ツアー『Further away 〜Touch the sky〜』も開催する。 ■メンバーコメント【MYUJI】　オリコンウィークリーシングルチャート3位！ 全然まだまだ信じられない気持ちです。僕達のファンTRIP（ファン）がたくさん応援してくれたおかげです。TRIPがメジャーデビューをウィークリー3位という素晴らしい形で背中を押してくれました。僕達もみなさんの背中を押せる人間、そして、音楽を作っていけるように頑張ります。 　本当にありがとうございます。これからもADDICTIONのこと、よろしくお願いします！【K.B】　メジャーデビューさせていただくことでさえ本当にうれしいことなのに、デビューシングルがオリコンウィークリーのトップ3に入るなんて本当にまだ夢のようです。これはこのシングルに携わってくださった関係者の方々、サポートしてくださってスタッフの方々、そして何より僕らを大きな愛で支えてくださったTRIP（ファン）の皆さんのおかげの結果でしかありません。今はそんな皆さんにただただ感謝の気持ちでいっぱいです。 　まだまだ至らないことばかりな僕らですが、そんな僕らにできることはよい音楽を作って届けていくことだと思います。これからもっとよい結果を残して恩返しできるよう、より精進していきたいと思います。今回は本当にありがとうございました。 【ASCH】　オリコンウィークリーシングルチャート3位ありがとうございます。メジャー1stシングルがこのような結果を残せたこと本当にうれしいです。TRIP（ファン）のおかげです。そして、サポートしてくれているスタッフさんのおかげです。 　僕達はいろいろな方に支えられて今回このような結果を残すことができました。これからも皆様の心に届けられる音楽を発信していきますのでこれからもADDICTIONをよろしくお願い致します。 【TAKT】　今回メジャーデビューシングルがオリコンウィークリーシングルチャート3位にランクインしたこと、夢のようでまだ信じられない気持ちもありますが心の底からうれしいです。メジャーデビューも今回の3位という結果も僕たちのことを 支えてくださるスタッフさん、応援してくれるTRIP（ファン）がいて手にすることが出来たものです。本当にありがとうございます。 　この結果に見合うアーティストになれるよう僕たちを応援してくださる皆様の心に寄り添えるような楽曲、パフォーマンスをこれからも追求していきます。これからもADDICTIONをよろしくお願いします。 【JOSHUA】　オリコンウィークリーシングルチャート3位ありがとうございます！ メンバー、スタッフさん、なによりもいつもそばにいてくて、支えてくれたTRIP（ファン）のみんながいたからこそ、初メジャーシングルを素晴らしい結果で終えることができました。本当にありがとうございました！ 　これからもどんな時もTRIPのみんなのそばにいれるよう、もっともっと頑張りたいと思います。本当にありがとうございます！ これからもよろしくお願いします!!
