
# Жители Брянска превращают макулатуру в коробки для пиццы

Published at: **2019-11-06T13:45:29+00:00**

Author: ****

Original: [Брянск Today](https://bryansktoday.ru/article/112063)

Региональный оператор по обращению с ТКО в Брянской области АО «Чистая планета» рассказал, как брянцы могут превратить бумагу в коробки для пиццы. И напоминает, что каждый из нас может помочь в сохранении брянского леса.
- Если хотите, чтобы макулатура принесла пользу и была переработана, отнесите в сетчатый контейнер для раздельного сбора отходов, - говорится в сообщении регоператора.
Всю собранную брянцами макулатуру отправляют на суражское предприятие «Пролетарий». Там из бумажного вторсырья делают гофрокартон и гофротару – коробки для пиццы, кондитерских изделий и  дроугие упаковки.
