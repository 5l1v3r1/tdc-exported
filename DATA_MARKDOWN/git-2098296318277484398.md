
# Музыкант Linkin Park Майк Шинода записал для «Аванпоста» оригинальный трек

Published at: **2019-11-01T00:00:00+00:00**

Author: **Кот-император**

Original: [Мир фантастики](https://www.mirf.ru/news/muzykant-linkin-park-majk-shinoda-zapisal-dlya-avanposta-originalnyj-trek)

Вокалист, гитарист, клавишник и один из основателей группы Linkin Park Майк Шинода записал для отечественной фантастической картины «Аванпост» новый трек — и по такому поводу создатели фильма поделились новым роликом, смонтированным специально под композицию.
