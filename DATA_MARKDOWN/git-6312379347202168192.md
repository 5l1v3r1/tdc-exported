
# Пассажирские поезда по железной дороге в Крым поедут в этом году

Published at: **2019-11-01T15:53:35+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-01/passazhirskie-poezda-po-zheleznoy-doroge-v-krym-poedut-v-etom-godu)

Первые пассажирские поезда в Крым поедут в этом году, сообщили в Минтрансе.
По данным ведомства, перевозчиком станет компания "Гранд Сервис Экспресс". Отмечается, что первые поезда в республику запустят в текущем году после ввода в эксплуатацию железнодорожной части Крымского моста.
Поезда поедут по направлениям Москва–Симферополь и Санкт-Петербург–Севастополь.
В четверг ФГУП "Крымская железная дорога" опубликовала в соцсети официальное расписание движения поездов по Крымскому мосту. Так, ежедневно по нему будут проезжать восемь пар поездов, которые будут ходить по 11 маршрутам из материковой части страны в четыре города полуострова.
В октябре по железнодорожной части моста пустили два специальных состава, чтобы проверить конструкцию на прочность. Поезда двигались параллельно и протянули за собой 46 полностью загруженных вагонов.
