
# Sara Carreira convida fãs a participarem no seu EP

Published at: **2019-11-07T13:50:27+00:00**

Author: **A Televisão**

Original: [A Televisão](https://www.atelevisao.com/famosos/sara-carreira-convida-fas-a-participarem-no-seu-ep/)


        Com lançamento marcado do primeiro EP para o próximo dia 22, Sara Carreira lançou um desafio aos fãs onde dá a oportunidade de fazerem parte deste projeto.
      
A cantora reservou uma faixa extra para uma participação dos fãs e quem estiver interessado basta enviar uma mensagem em formato áudio, a explicar o que significa para si as músicas da filha de Tony Carreira.
O envio deve ser feito pelo What’s App para o número 914 402 885.
