
# Оплата у транспорті і вхід у музеї: коли у Києві запустять е-квиток для учнів

Published at: **2019-11-01T16:22:56+00:00**

Author: ****

Original: [ФАКТИ](https://fakty.com.ua/ua/ukraine/20191101-oplata-u-transporti-i-vhid-u-muzeyi-koly-u-kyyevi-zapustyat-e-kvytok-dlya-uchniv/)

У грудні 2019 року розпочнеться тестування електронних учнівських квитків для школярів 20 навчальних закладів Києва.
У грудні 2019 року стартує тестування електронних учнівських квитків для школярів 20 навчальних закладів Києва. Це стане своєрідним електронним ключем учня до різних сервісів та можливостей. Картку можна будет використовувати для безоплатного проїзду у громадському транспорті та пільгового входу до закладів культури.
Про це під час прес-брифінгу повідомив директор Департаменту інформаційно-комунікаційних технологій Юрій Назаров.

У березні 2020 року планують випустити 100% електронних учнівських квитків. Запуск електронного квитка у промислову експлуатацію та скасування готівкової оплати проїзду заплановано на квітень 2020 року.
До речі, раніше Факти ICTV писали, як оформити електронний учнівський квиток.
