
# 内山信二がラブラブ生告白　婚約者とは出がけと帰宅時に「ちゃんとチューして」

Published at: **2019-11-05T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/05/0012852278.shtml)

　婚約したことが明らかになったタレントの内山信二（３２）が５日、ＴＯＫＹＯ　ＭＸの生番組「バラいろダンディ」で、出がけと帰宅時にフィアンセとキスしていることを打ち明けた。
　内山は「まだ入籍してないんで、まだ婚約なんで。新婚ではないですけど、でも今でも出る時と帰った時はちゃんとチューして、アレしますよ」と、同棲していることと、出がけと帰宅時にはキスすることを告白。
　番組では元ＴＢＳアナウンサーの安東弘樹（５２）が「いつもニコニコしていますよ、基本的に。いつも帰ったらハグしますね、お互い」と、妻とのラブラブぶりを明かしており、内山は「（自分たちは）今、一番いい時期。今後はやっぱりどうなるか分かりませんけど、でも僕は安東さんみたいな家庭を目指したいなと思っております」とブチ上げていた。
