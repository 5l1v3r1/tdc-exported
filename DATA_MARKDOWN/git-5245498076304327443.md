
# В Коми лисы «мстят» метеорологам

Published at: **2019-11-01T09:49:00+00:00**

Author: ****

Original: [БНК](https://www.bnkomi.ru/data/news/102180/)

В Сыктывкаре на метеоплощадку по вечерам повадилась ходить лиса. Хищница отгрызла уже два датчика. Видео с «места преступления» разместили в группе «Коми ЦГМС» в соцсети «ВКонтакте».
