
# Свекольные пельмени и шербет из антоновки подали на приеме в Кремле

Published at: **2019-11-04T13:10:20+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-04/svekolnye-pelmeni-i-sherbet-iz-antonovki-podali-na-prieme-v-kremle)

Во время торжественного приема по случаю вручения наград за вклад в укрепление единства российской нации, который состоялся в Кремле 4 ноября по случаю Дня народного единства, гостям подавали различные уникальные угощения.
Так, участникам приема предлагали свекольные пельмени с перепелкой, салат из карельской форели с огурцами, шербет из антоновских яблок, воронежскую говядину с печеным картофелем, морского окуня с овощами, имбирный мусс с пряной сливой и мороженое из белого шоколада.
Награды в Георгиевском зале вручил президент РФ Владимир Путин. Перед награжденными также выступили Сергей Волчков, Раяна Асланбекова, хор Краснодарской филармонии и солисты танцевального ансамбля "Буин-хан", пишет ТАСС.
