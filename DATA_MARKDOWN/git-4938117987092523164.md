
# General Motors отзывает более 600 тысяч внедорожников Chevrolet и GMC

Published at: **2019-11-02T00:00:00+00:00**

Author: **Марина Мерш**

Original: [CAR.RU](https://car.ru/news/automobili/56661-general-motors-otzyivaet-bolee-600-tyisyach-vnedorozhnikov-chevrolet-i-gmc/)


        По информации, обнародованной руководством компании, причиной принятия такого решения становится наличие возможности резкого непреднамеренного торможения.
      
Также сообщается, что причиной такой проблемы становится поломка датчика определения скорости вращения колес. Если он выходит из строя, следствием становится возможность резкого торможения колеса, совместно с неожиданным смещением его в сторону, что может стать причиной создания аварийной ситуации. Хотя на данный момент, о случаях аварийности или травматизма, информации нет.
Под действие программы отзыва попали 638 тысяч 58 машин моделей Chevrolet Silverado 1500 и GMC Sierra 1500, которые были выпущены в период с 2014 по 2018 год. Помимо этого, сюда относятся Chevrolet Suburban (2015-2020 МГ), Tahoe и Yukon (2015-2020 МГ). Общей чертой для них становится наличие силовой установки в виде 5,3 литрового мотора, и системой полного привода.
 
Чтобы полностью устранить имеющуюся проблему, следует осуществить перепрошивку программного обеспечения для модуля электронного торможения автомобиля.
Из всех перечисленных моделей в России осуществляются продажи только одной модели - Chevrolet Tahoe.
