
# США уведомили ООН о своем выходе из Парижского соглашения

Published at: **2019-11-05T04:02:00+00:00**

Author: ****

Original: [RuNews24](https://runews24.ru/world/05/11/2019/82ac137deb92215de1b5b8686e33d315)

Госсекретарь США Майк Помпео сообщил, что Штаты уведомили ООН о своем выходе из Парижского соглашения по климату. Помпео отметил, что Америка лидирует в мире по сокращению любых выбросов в атмосферу.
На своей странице в соцсети Twitter американский госсекретарь написал, что Вашингтон начинает формальный процесс выхода из Парижского соглашения.
Еще два года назад, в 2017 году, президент Соединенных Штатов Дональд Трамп объявил о выходе страны из Парижского соглашения по климату. Однако это вызвало критику международного сообщества.
Парижское соглашение по климату должно прийти на смену Киотскому протоколу в 2021 году. Участники планируют не допустить роста средней температуры на Земле к 2100 году более чем на два градуса по Цельсию по сравнению с доиндустриальной эпохой.
