
# Сбербанк планирует запустить помощника врача с ИИ

Published at: **2019-11-07T04:11:54+00:00**

Author: ****

Original: [Коммерсантъ](https://www.kommersant.ru/doc/4149979)

Сбербанк хочет запустить помощника врача с искусственным интеллектом, рассказал «РИА Новости» первый зампред правления банка Александр Ведяхин. Проект позволит быстрее и качественнее диагностировать заболевание и предлагать необходимое лечение.
«Мы уже обсуждаем с несколькими субъектами России старт этой работы на уровне администрации региона»,— сообщил он. Для создания такого помощника важно, чтобы был хороший набор размеченных опытными врачами данных для обучения ИИ, сказал господин Ведяхин.
«Искусственный интеллект — это ядерное топливо для нашего будущего. Нет, наверное, ни одной задачи ни в народном хозяйстве, ни в науке, ни в одной области деятельности человека, которая бы не могла существенно улучшиться при внедрении искусственного интеллекта»,— рассказал ранее Александр Ведяхин в интервью “Ъ FM”.
