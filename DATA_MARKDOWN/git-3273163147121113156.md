
# В одном из домов Мариуполя устранили аварийную ситуацию

Published at: **2019-11-07T19:13:00+00:00**

Author: ****

Original: [i24.com.ua](http://i24.com.ua/news/socium/v-odnom-iz-domov-mariupolya-ustranili-avarijnuyu-situaciyu)

Спасательная аварийно-водолазная служба городского совета оперативно устранила аварийную ситуацию в многоквартирном доме по ул. Куприна, 23а. Сотрудники демонтировали разрушенные конструкции козырьков.
По данному адресу были разрушены объекты в первом и втором подъездах. Они угрожали безопасности жителей. На месте работала мобильная аварийно-спасательная команда. При помощи автовышки, они оперативно провели весь комплекс необходимых работ. Разрушенные конструкции козырьков демонтированы.
