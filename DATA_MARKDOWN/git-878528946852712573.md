
# 笛木優子、４０歳で第１子妊娠　昨年一般男性と結婚

Published at: **2019-11-02T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/02/0012841768.shtml)

　ユミン名義で韓国でも活躍中の女優・笛木優子（４０）が２日、インスタグラムで「第一子を授かる事が出来ました」と妊娠を発表した。
　笛木は１８年６月に３年交際した１歳上の一般男性と結婚。「私事で恐縮ですが、この度、第一子を授かる事が出来ました」と喜びの報告した。
　今後も「体調と相談しながらではありますが、出来る範囲でお仕事させて頂いております」と無理のない範囲での仕事も継続する。「皆様のご理解心より感謝申し上げます」と記した。
