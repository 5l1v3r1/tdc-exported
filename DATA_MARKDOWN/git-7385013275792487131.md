
# 「天華百剣 -斬-」初のライブイベント「天華百剣 -斬- 音楽祭～繚乱の宴～」来年1月18日(土)に開催！　イベントも続々決定！

Author: ****

Original: [アキバ総研](https://akiba-souken.com/article/42237/)

iOS／Android端末向け美少女剣撃アクションRPG「天華百剣 -斬-」初のライブイベント「天華百剣 -斬- 音楽祭～繚乱の宴～」が2020年1月18日(土)に開催されることが決定した。 
「天華百剣 -斬-」は、「巫剣」（みつるぎ）と呼ばれる刀剣をモチーフにしたキャラクターたちを操作し、斬撃や奥義などを駆使しながら「禍憑」（まがつき）と呼ばれる異形の敵を撃破していく iOS／Android端末向け美少女剣撃アクションRPG。すべての「巫剣」には個別のストーリーを楽しめるアドベンチャーパートがフルボイスで用意されており、「天華百剣 -斬-」の世界をより深く楽しむことができる。ダウンロード数は全世界累計で700万を突破しているほか、楽曲展開をはじめコミカライズなどのメディアミックスも幅広く展開している。2018年4月のゲームリリース1周年以降続々とキャラクターソングを発表してきた「天華百剣 -斬-」が、ファンからの熱い要望を受け、作品初のライブイベントを開催することが決定した。本イベントには、TVアニメ「天華百剣 ～めいじ館へようこそ！～」の主題歌「紅、 華を咲かせて」を歌う御華見衆 椿組をはじめ、 キャラクターボイスを担当する声優が総勢12名出演する。また、スペシャルゲストとしてテーマソング「全部君がいたから知ったんだ」を歌う鈴木このみさんの出演も決定している。なお、 11月20日発売のCD「紅、 華を咲かせて」に本イベントのチケット最速先行応募券が封入予定となっているので、ぜひチェックしていただきたい。また、上記ライブイベントの前にも、CDリリースを記念したインストアイベントや、公式YouTube Live生放送「ざんなま♪」の公開生放送イベントも続々と決定している。 詳しくは「天華百剣」公式サイトをご確認いただきたい。
