
# 'Kwon HyunBin' จัดเต็ม แฟนมีตติ้งในไทยสุดอบอุ่น

Published at: **2019-11-04T00:45:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/entertainment/739533)


        VIINI (บีนี่) หรือ Kwon HyunBin (ควอน ฮยอนบิน) ศิลปินหนุ่มสุดหล่อ โปรยเสน่ห์แฟนมีตติ้งที่ไทยสุดประทับใจ
      
จัดเป็นศิลปินหนุ่มมากความสามารถที่มีแฟนๆชื่นชอบอย่างมาก สำหรับหนุ่มสุดฮอต VIINI (บีนี่) หรือ Kwon HyunBin (ควอน ฮยอนบิน) จากสังกัด YGX Entertainment ที่ล่าสุดบินลัดฟ้ามาจัดงานแฟนมีตติ้งที่ไทยในงาน “2019 VIINIFUL DAY IN BANGKOK” โดยผู้จัด RegiStar (รีจิสตาร์) จัดเต็มแสงสีเสียงสุดประทับใจ
เริ่มต้นงานแฟนมีตติ้งด้วยบรรยากาศคึกคัก หนุ่มบีนี่เปิดเวทีด้วยเพลง GENIE พร้อมกล่าวทักทายว่า “สวัสดีครับ ผมบีนี่ครับ”จากนั้นหนุ่มบีนี่จึงขออัพเดทเรื่องราวของตัวเองให้ทุกคนได้ฟังในช่วง VIINIFUL : TIME ถึงบทบาทการเป็นศิลปินเดี่ยวเต็มตัวในสเตจเนม VIINI (บีนี่) กับมินิอัลบั้มชุดแรก DIMENSION ที่เจ้าตัวตั้งใจและมีส่วนร่วมในการทำงานอย่างเต็มที่
จากนั้นหนุ่มบีนี่ก็ยังได้ตอบคำถามที่แฟนคลับส่งมาทางโซเชียลแบบจุใจ ต่อมาในช่วง Joyful : TIME กับเกม VIINI’s Caricature หนุ่มบีนี่ต้องวาดรูปตามโจทย์แล้วให้แฟนๆช่วยกันทายให้ถูกต้อง ส่วนช่วง Sixth Sense Game น้องตี๋จะต้องเอามือล้วงเข้าไปในกล่องและทายว่าของในนั้นเป็นอะไร หลังจากจบช่วงเล่นเกมหนุ่มบีนี่ก็เปลี่ยนเวทีให้กลับมาร้อนแรงอีกครั้งในเพลงสนุกๆอย่าง BITTERSWEET และ AFFECTION หลังจากนั้นเข้าสู่ช่วงอีเว้นท์สำหรับแฟนๆใน Wonderful : TIME ที่หนุ่มบีนี่ขอสวมบทเป็น GENIE VIINI ทำตามคำอธิษฐานของแฟนคลับผู้โชคดี และหลังจากตามใจแฟนๆมาตลอดทั้งงานแล้ว หนุ่มบีนี่ก็มีคำอธิษฐานที่อยากจะขอบ้างนั่นก็คืออยากให้ทุกคนในฮอลล์ร้องเพลง GENIE ไปพร้อมๆกัน ซึ่งแฟนคลับทุกคนต่างก็ช่วยกันประสานเสียงร้องท่อนฮุคพร้อมกับตะโกนเรียกชื่อ VIINI จนดังสนั่น
ช่วงท้ายงานแฟนคลับขอทำเซอร์ไพรส์ให้หนุ่มบีนี่บ้างกับ VCR สุดซึ้งพร้อมชูป้ายข้อความภาษาเกาหลีที่แปลว่า “พวกเราเชื่อว่า VIINI ของเราจะเก่งขึ้นไปอีกในทุกๆ วัน” ทำเอาหนุ่มบีนี่ยิ้มปลื้ม ก่อนจะบอกความในใจว่า “ผมรู้สึกขอบคุณแฟนชาวไทยมากๆ สำหรับการต้อนรับที่อบอุ่น แทนที่ผมจะเป็นคนให้ความสุขกับทุกคน แต่กลายเป็นว่าแฟนๆ กลับเป็นคนที่มอบความสุขและโมเมนท์ดีๆให้กับผม ดังนั้นผมสัญญาว่าจะตอบแทนทุกคนด้วยการตั้งใจทำงานให้ดีที่สุดครับ” จากนั้นหนุ่มบีนี่ขอทิ้งทวนความมันส์แบบเต็มแม็กซ์ในเพลง FIANCÉ ของศิลปินรุ่นพี่ MINO ก่อนจะจัดเต็มไฮทัชกับแฟนคลับทุกที่นั่ง เป็นการปิดท้ายงานแฟนมีตติ้งครั้งนี้ไปแบบน่าประทับใจ
