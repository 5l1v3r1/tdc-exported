
# В Хакасии экс-полицейский обвинил высокопоставленных сотрудников МВД в наркоторговле

Published at: **2019-11-05T12:04:00+00:00**

Author: ****

Original: [Новая Газета](https://www.novayagazeta.ru/news/2019/11/05/156647-v-hakasii-eks-politseyskiy-obvinil-vysokopostavlennyh-sotrudnikov-mvd-v-narkotorgovle)

Бывший работник управления по борьбе с наркотиками МВД по Хакасии Юрий Зайцев рассказал, что наркополицейские занимаются продажей синтетических запрещенных веществ в интернете. Об этом он заявил в своем видеообращении к президенту Владимиру Путину на YouTube.
Самого Зайцева, по данным интернет-издания «19rus.info», ранее обвинили в получении взятки и разглашении гостайны, после чего поместили под домашний арест. Полицейский считает обвинения сфабрикованными.
