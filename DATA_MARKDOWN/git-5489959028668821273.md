
# Klausul Pelepasan, Celah untuk Membajak Lautaro Martinez dari Inter Milan

Published at: **2019-11-02T19:41:00+00:00**

Author: **Bola.net**

Original: [Bola.net](https://www.bola.net/italia/klausul-pelepasan-celah-untuk-membajak-lautaro-martinez-dari-inter-milan-1b18b8.html)

Jika itu dilakukan, Inter berisiko kehilangan kans mendapatkan Arturo Vidal atau Ivan Rakitic dari Barcelona. Namun, selama bisa mempertahankan Lautaro, Inter takkan mempermasalahkannya.
