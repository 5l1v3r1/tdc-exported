
# 千葉雄大の“いたずら”ウィンクショットにファン悶絶 「食べちゃいたい」「30歳でこの可愛さ」

Published at: **2019-11-01T14:19:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2147819/full/)

　あす2日にスタートするテレビ朝日系土曜ナイトドラマ『おっさんずラブ-in the sky-』（毎週土曜　後11：15〜深0：05）の公式インスタグラムが10月31日、同ドラマで副操縦士・成瀬竜を演じる俳優・千葉雄大（30）のオフショットを投稿した。 　「HAPPY HALLOWEEN」とかぼちゃの形をしたアイシングクッキーを手に持った千葉のソロショットをアップ。パイロットの制服姿の千葉が、猫の手のようなポーズでウィンクをするお茶目な1枚となっている。ハッシュタグでは「#副操縦士の成瀬　#悪戯なウィンクを頂きました　#ニャニャニャルセ　#みなさんも　#ハッピーハロウィン」と呼びかけた。 　ファンからは「あざとい！」「悪戯されたい」「食べちゃいたい」「心臓止まる」「30歳でこの可愛さ」「どこで会えますか　フライトスケジュール教えて」「罪深すぎな可愛さ」「顔面力が強い」といった声が寄せられている。
