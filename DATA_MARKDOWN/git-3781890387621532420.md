
# ตักบาตรทางน้ำเชิงวัฒนธรรม งานจุลกฐินสามัคคี ปีที่ 3

Published at: **2019-11-02T16:01:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/article/739715)


        สาธุชนร่วมตักบาตรทางน้ำเชิงวัฒนธรรมในงาน "จุลกฐิน" สามัคคี ปีที่ 3 ยกฉัตรยอดพระเจดีย์ฐิตปุญญานุสรณ์ พร้อมพิธีแสดงมุฑิตาสักการะ "หลวงปู่ทองสุข" เจริญอายุวัฒนมงคล 78 ปี พรรษา 55 วัดป่าวุฑฒาราม
      
บริเวณหนองน้ำล้านห้าหน้าวัดป่าวุฑฒาราม ต.หนองกุงแก้ว อ.ศรีบุญเรือง จ.หนองบัวลำภู นายสุภโชค พนาลิกุล นายอำเภอศรีบุญเรือง พร้อมนางศรัณยา สุวรรณพรหม ประธานสภาวัฒนธรรมจังหวัดหนองบัวลำภู สาธุชนจากทั่วสารทิศร่วม 2 พันคน ที่ทราบข่าวหลั่งไหลมาร่วมทำบุญตักบาตรทางน้ำ ซึ่งถือว่าเป็นหนึ่งเดียวของอีสานในงาน "จุลกฐิน” สามัคคี ปีที่ 3 ร่วมยกฉัตรยอดพระเจดีย์ฐิตปุญญานุสรณ์ พร้อมพิธีแสดงมุฑิตาสักการะ พระครูพันธสมณวัฒน์ (หลวงปู่ทองสุข ฐิตปุญโญ) เจริญอายุวัฒนมงคล 78 ปี พรรษา 55 "เนื้อนาบุญ" ผู้เป็นทายาทแห่งองค์ตถาคต ที่ควรค่าแก่การกราบไหว้บูชา สำหรับประเพณีตักบาตรทางน้ำ ที่วัดป่าวุฑฒารามแห่งนี้ถือเป็นอัตตลักษณ์ของคนลุ่มภู และถือได้ว่าเป็นหนึ่งเดียวของภาคอีสาน ที่คณะศิษยานุศิษย์ที่เคารพและศรัทธาธรรมแด่หลวงปู่ทองสุขฯ พระผู้ปฏิบัติดีปฏิบัติชอบอีก 49 รูป ซึ่งถือได้ว่าเป็นเทศกาลสำคัญ จะมีเพียงปีละ 1 ครั้งเท่านั้น สำหรับกิจกรรมสืบสานวัฒนธรรมทางสายน้ำด้วยการทำบุญตักบาตรทางน้ำ โดยหลวงปู่ทองสุขฯ จะนั่งเรือโขนมังกรสกุณี ตามด้วยเรือเทพกินราออกรับบิณฑบาตจากสาธุชนริมสระน้ำล้านห้าซึ่งเป็นหนองน้ำขนาดใหญ่ ระยะทางร่วม 200 เมตร ก่อนที่จะมีพิธีทอดกฐินซึ่งเป็นจุลกฐินสามัคคี ที่มีในจังหวัดหนองบัวลำภู หนึ่งเดียวเช่นกัน 
นางศรัณยา สุวรรณพหรม ประธานสภาวัฒนธรรมจังหวัดหนองบัวลำภู กล่าวถึงประเพณีการตักบาตรทางน้ำซึ่งเป็นอัตลักษณ์หนึ่งของจังหวัดหนองบัวลำภู ที่มุ่งส่งเสริมแหล่งท่องเที่ยวเชิงวัฒนธรรมและสิ่งดีๆ ทางวัฒนธรรมของจังหวัดหนองบัวลำภู ไม่ว่าจะเป็นแหล่งท่องเที่ยวทางธรรมชาติ ที่มีอยู่ในทุกอำเภอ แหล่งท่องเที่ยวทางธรรมะที่มีวัดเก่าแก่คู่บ้านคู่เมืองอยู่หลายวัด และมีแหล่งท่องเที่ยวเชิงศิลปวัฒนธรรมที่มีอายุเก่าแก่ร่วม 2.5 พันปี อย่างวัฒนธรรมกุดกวางสร้อย อายุคราวเดียวกันกับบ้านเชียงสุภัชรกานต์ แก้วสิงห์ ผู้สื่อข่าวเว็บไซต์เดลินิวส์ออนไลน์ จ.หนองบัวลำภู
