
# Зеленський затвердив склад делегації України для участі в конференції ЮНЕСКО

Published at: **2019-11-05T08:51:00+00:00**

Author: ****

Original: [ZIK.UA](https://zik.ua/news/2019/11/05/zelenskyy_zatverdyv_sklad_delegatsii_ukrainy_dlya_uchasti_v_konferentsii_1683793)


        Президент України Володимир Зеленський затвердив склад делегації України для участі в роботі 40-ї сесії Генеральної конференції ЮНЕСКО.
      
Відповідне розпорядження №359/2019-рп опубліковано на сайті глави держави.
«Утворити делегацію України для участі в роботі 40-ї сесії Генеральної конференції ЮНЕСКО (12 – 27 листопада 2019 року, м.Париж, Французька Республіка) у такому складі», – йдеться у документі.
Зокрема, до складу делегації увійшли:
Також Президент доручив Міністерству закордонних справ України затвердити вказівки делегації України для участі в роботі 40-ї сесії Генеральної конференції ЮНЕСКО.
Крім того, Зеленський дозволив главі делегації вносити у разі потреби зміни до персонального складу делегації та залучати експертів для забезпечення роботи делегації.
Нагадаємо, Президент України Володимир Зеленський попросив першого заступника голови Верховної Ради Руслана Стефанчука терміново зібрати чергову нараду депутатів з підготовки закону про народовладдя.
