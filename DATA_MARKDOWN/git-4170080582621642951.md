
# В Туле задержан подозреваемый в убийстве 28-летнего мужчины

Published at: **2019-11-07T08:33:00+00:00**

Author: ****

Original: [Тульская Служба Новостей](https://www.tsn24.ru/v-tule-nashli-podozrevaemogo-v-ubijstve-28-letnego-muzhchiny.html)

Уголовное дело по факту убийства молодого мужчины возбуждено в Туле. В конце октября в одной из квартир города было обнаружено тело с колото-резаной раной в области грудной клетки. По подозрению в убийстве задержан 55-летний туляк: он признался в совершении преступления.
Так, по информации СУ СК РФ по региону, мужчины распивали спиртные напитки, в процессе возникла ссора, и обвиняемый нанес потерпевшему удар ножом в область сердца. Он скончался на месте происшествия. Расследование уголовного дела продолжается.
