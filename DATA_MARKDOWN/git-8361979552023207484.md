
# Нацкомсвязи определилась. Где и когда в Украине будут тестировать 5G

Published at: **2019-11-06T15:41:00+00:00**

Author: ****

Original: [НВ](https://nv.ua/biz/tech/testirovanie-5g-v-ukraine-stalo-izvestno-gde-i-kogda-novosti-ukrainy-50051993.html)

Тестирование технологии 5G в Украине проведут «дочка» компании Ericsson и мобильный оператор lifecell с 1 декабря 2019 года по 31 мая 2020 года. Соответствующее решение обнародовала Национальная комиссия по госрегулированию в сфере связи и информатизации.
Тестирование пройдет на территории магазинов сети lifecell по адресам:
— Киев, Бесарабская пл., 2;
— Киев, ул. Антоновича, 176 (Оушен Плаза);
— Днепр, бул. Европейский, 2;
— Харьков, ул. Ярослава Мудрого, 33а;
— Львов, пр. Свободи, 6/8;
— Одесса, ул. Дерибасовская, 19;
— Черкассы, ул. Смелянская, 36.
Управление оборудованием базовой станции Radio DOT System будет осуществляться специализированным тестовым ядром сети (5G EPC) с ограниченной функциональностью: будет доступна только сеть интернет, без возможности осуществления голосовых вызовов.
Как сообщалось, в Министерстве цифровой трансформации прогнозируют появление в Украине связи пятого поколения в 2022 году.
