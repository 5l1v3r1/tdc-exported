
# فقدان الوزن المفرط .. ناقوس خطر

Published at: **2019-11-06T00:00:00+00:00**

Author: ****

Original: [الوطـــن الإلكترونية](http://alwatan.kuwait.tt/articledetails.aspx?id=603060&yearquarter=20194)

د ب أ- قال المركز الاتحادي للتوعية الصحية إن فقدان الوزن غير المبرر يدق ناقوس الخطر وينذر بالإصابة بأمراض خطيرة.وأوضح المركز الألماني أن فقدان الوزن المفرط رغم غياب الحمية الغذائية قد يشير إلى التهابات الأمعاء المزمنة، مثل داء كرون، والتهاب القولون التقرحي، أو اضطرابات الأيض، مثل السكري، أو الاضطرابات الهرمونية مثل فرط نشاط الغدة الدرقية.كما قد يكون السبب الإصابة بسرطان القولون، أو سرطان الكبد، أو سرطان البنكرياس.ومن ناحية أخرى قد تكون الأمراض النفسية والأمراض النفس جسدية سبباً في فقدان الوزن المجحف، مثل الاكتئاب واضطرابات الأكل، أو فقدان الشهية العصبي، أو الشره العصبي، البوليميا.ويجب استشارة الطبيب لتحديد سبب فقدان الوزن والعلاج في الوقت المناسب.
