
# МЮнинг асосий футболчиларидан бири Партизанга қарши ўйинда иштирок этмайди

Published at: **2019-11-07T05:41:00+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/myuning-asosiy-futbolchilaridan-biri-partizanga-qarshi-oyinda-ishtirok-etmaydi)

"Манчестер Юнайтед"нинг марказий ҳимоячиси Виктор Линделёф Европа Лигаси 4-тур доирасидаги ўйинни ўтказиб юборади.
Виктор танасининг орқа қисмидан жароҳат олган.
25 ёшли швед жорий мавсум МЮ сафида барча турнирларда 12та ўйинда майдонга тушиб, 2та сариқ карточка олган.
Эслатиб ўтамиз, "Манчестер Юнайтед" ва "Партизан" жамоалари ўртасидаги ўйин бугун Тошкент вақти билан соат 01:00да бошланади.
