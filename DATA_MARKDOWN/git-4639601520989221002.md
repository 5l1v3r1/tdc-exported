
# Ρις Γουίδερσπουν : Η σιωπή ήταν προϋπόθεση για να έχω δουλειά

Published at: **2019-11-06T19:55:22+00:00**

Author: **ΤοΒΗΜΑ Team**

Original: [Το Βήμα Online](https://www.tovima.gr/2019/11/06/culture/ris-gouiderspoun-i-siopi-itan-proypothesi-gia-na-exo-douleia/)

«Η σιωπή ήταν προϋπόθεση για να έχω δουλειά», δήλωσε η ηθοποιός Ρις Γουίδερσπουν, επισημαίνοντας ότι δεν θα μπορούσε να μιλήσει νωρίτερα για την σεξουαλική παρενόχληση που δέχτηκε ότι ήταν μόλις 16 ετών.
