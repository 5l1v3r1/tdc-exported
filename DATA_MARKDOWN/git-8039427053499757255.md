
# నెట్‌లో సీడీఎస్‌-2 మెరిట్‌ జాబితా

Published at: **2019-11-05T01:44:56+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=946507)

ADVT
న్యూఢిల్లీ, నవంబరు 4: కంబైన్డ్‌ డిఫెన్స్‌ సర్వీస్‌-2 (సీడీఎస్‌-2) రాత పరీక్షలో ఉత్తీర్ణత సాధించి ఇంటర్వ్యూకు ఎంపికైనవారి జాబితాను యూపీఎ్‌ససీ తమ వెబ్‌సైట్‌ యూపీఎ్‌ససీ.జీవోవీ.ఇన్‌లో ఉంచింది. మొత్తం 8,120 మంది ఇంటర్వ్యూకు అర్హత సాధించినట్లు కమిషన్‌ పేర్కొంది. వచ్చే ఏడాది ప్రారంభం కానున్న కోర్సులకు వీరిని సర్వీస్‌ సెలక్షన్‌ బోర్డు ఇంటర్వ్యూ చేయనుంది. అభ్యర్థులంతా రెండు వారాల్లో జాయిన్‌ఇండియన్‌ఆర్మీ.నిక్‌.ఇన్‌ వెబ్‌సైట్లో రిజిష్టర్‌ చేసుకోవాలని కమిషన్‌ సూచించింది.
