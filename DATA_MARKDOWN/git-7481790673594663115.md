
# 宇野、ＳＰ４位…女子は樋口５位・坂本６位

Published at: **2019-11-02T03:00:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/winter/20191102-OYT1T50164/)

　フィギュアスケートのグランプリ（ＧＰ）シリーズ第３戦、フランス杯は１日、グルノーブルで開幕し、男子ショートプログラム（ＳＰ）で今季ＧＰ初戦の宇野昌磨（トヨタ自動車）はジャンプで２度転倒し、７９・０５点で４位だった。世界選手権２連覇中のネーサン・チェン（米）が１０２・４８点でトップ。女子ＳＰは樋口新葉（明大）が６４・７８点で５位。坂本花織（シスメックス）は６位、白岩優奈（関大ク）は７位だった。シニアＧＰデビュー戦で１６歳のアリョーナ・コストルナヤ（ロシア）が７６・５５点で首位に立ち、平昌五輪金メダルのアリーナ・ザギトワ（同）は７４・２４点で２位発進。（時事）
