
# కొత్త ఇండియా మ్యాప్‌ను ఆవిష్కరించిన కేంద్ర ప్రభుత్వం

Published at: **2019-11-02T17:57:18+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=944604)

Ph: 88865 47666ప్రవేశం ఉచితం ADVT
న్యూఢిల్లీ: జమ్మూ కశ్మీర్, లద్దాఖ్‌లు కేంద్ర పాలిత ప్రాంతాలుగా ఏర్పాటవడంతో భారత ప్రభుత్వం ఇందుకు సంబంధించి ఇండియా మ్యాప్‌లో మార్పులు చేసింది. ఈ మార్పులతో కూడిన నూతన మ్యాప్‌ను శనివారం విడుదల చేసింది. జమ్మూ కశ్మీర్‌లో రాష్ట్రపతి పాలనకు తెరపడిన రెండు రోజుల తరువాత ప్రభుత్వం ఈ మ్యాప్‌ను విడుదల చేసింది. కాగా..రెండు కేంద్రపాలిత ప్రాంతాలకూ లెఫ్టినెంట్ గవర్నర్‌లు శుక్రవారం ప్రమాణస్వీకారం చేశారు. ప్రస్తుతం భారత్‌లో 28 రాష్ట్రాలు, 9 కేంద్ర పాలిత ప్రాంతాలు ఉన్నాయి.
