
# Аҳмед Тиллаев U-19 Осиё Чемпионати саралаш баҳсларида комиссар сифатида ишлайди

Published at: **2019-11-06T06:34:16+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/ahmed-tillaev-u-19-osiyo-chempionati-saralash-bahslarida-komissar-sifatida-ishlaydi)

Жумладан, Индонезияда “K” гуруҳи учрашувлари ўтказилади. Мазкур гуруҳда Индонезия, КХДР, Гонконг ҳамда Шарқий Тимор терма жамоалари қитъа чемпионати финал босқичига чиқиш учун ўзаро баҳс олиб боришади.Қайд этиш жоиз, Индонезия мезбонлик қилаётган U-19 Осиё Чемпионати саралашининг “K” гуруҳи ўйинларида ҳамюртимиз Аҳмед Тиллаев комиссар сифатида фаолият олиб боради.Бугун бўлиб ўтадиган КХДР – Гонконг учрашуви ҳамюртимиз учун халқаро майдонда комиссар сифатидаги илк учрашув бўлади.

        Бугун 17:00. Ўзбекистон U-19 – Ҳиндистон U-19
