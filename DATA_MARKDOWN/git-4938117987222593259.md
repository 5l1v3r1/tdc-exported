
# Студенты Московского политеха обучат Hyundai Creta ездить силой мысли

Published at: **2019-11-06T00:00:00+00:00**

Author: **Ирина Котова**

Original: [CAR.RU](https://car.ru/news/tehnologii/56995-studentyi-moskovskogo-politeha-obuchat-hyundai-creta-ezdit-siloy-myisli/)


        Студенты Московского политехнического университета решили создать дополнительное оборудование для корейского кроссовера Hyundai Creta, который считается достаточно востребованным на российском транспортном рынке.
      
По сути студенты планируют создать своеобразный вид беспилотника, взяв за основу корейский кроссовер. Первой задачей для студентов становится разработка многофункционального автономного оборудования, управлять которым можно будет с помощью голосовых команд водителя.
Следующим этапом, к реализации которого молодые ученые планируют приступить в 2020 году, станет модернизация оборудования, позволяющая управлять автомобилем уже не с помощью голоса, а с помощью нейроимпульсов. Цель у данного проекта социальная, так как студенты планируют разработать и создать комфортный автомобиль для инвалидов.
 
Кроссовер для выполнения всех разработок студентам был предоставлен руководителями концерна "Хенде Мотор СНГ".
