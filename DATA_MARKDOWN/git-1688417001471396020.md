
# HBO показала картонную коробку для просмотра видео

Published at: **2019-11-06T15:54:00+00:00**

Author: **Корреспондент.net**

Original: [Корреспондент.net](https://korrespondent.net/lifestyle/4157517-HBO-pokazala-kartonnuui-korobku-dlia-prosmotra-vydeo)

Американская телесеть HBO презентовала необычный гаджет - картонную коробку для уединенного просмотра видео. Ролик появился на официальном YouTube-канале платформы.
The HBO Box позиционируется, как полезный гаджет для студентов. В нем удобно спрятаться от соседей и смотреть сериалы.
Для этого в коробке есть специальные отверстия для ног, вентиляционные отверстия и полка для ноутбука. “Гаджет” на 100% состоит из картона, обеспечивает приватность и шумоизоляцию.
