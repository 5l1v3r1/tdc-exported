
# Гамильтон стал шестикратным чемпионом Формулы-1

Published at: **2019-11-04T07:05:00+00:00**

Author: ****

Original: [Главное™](https://glavnoe.ua/news/n357007989-gamilton-stal-shestikratnym-chempionom-formuly-1)

Британский автогонщик Мерседес Льюис Гамильтон досрочно стал шестикратным чемпионом международных соревнований по автогонкам "Формула-1".
Об этом передает BBC.
По количеству чемпионских титулов британец опередил Хуана Мануэля Фанхио, впереди него только Михаэль Шумахер с семью победами.
