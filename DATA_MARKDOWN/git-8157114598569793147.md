
# Como verano: Revisa el pronóstico del tiempo para este miércoles en Santiago

Published at: **2019-11-06T00:00:00+00:00**

Author: **Yem Manríquez**

Original: [www.adnradio.cl](https://www.adnradio.cl/noticias/nacional/como-verano-revisa-el-pronostico-del-tiempo-para-este-miercoles-en-santiago/20191106/nota/3975494.aspx)

Una máxima de 31°C se sentirá este miércoles 6 de noviembre en Santiago, con un inicio de mañana con tan solo 8°C.
Según informó la Dirección Meteorológica de Chile, desde este miércoles habrá altas temperaturas desde las regiones de Coquimbo a la Metropolitana.
Este jueves será el día más caluroso, con 11°C y un tope de 32°C. Mientras que el viernes comenzará con 11 grados, llegando a una máxima de 30 grados.
De igual manera, este sábado seguirá despejado, marcando 11°C y 31°C. Finalmente, para este domingo se espera una mínima de 11 grados y un peak de 30 grados.
