
# Бускетс соғлиғи билан боғлиқ муаммога дуч келди

Published at: **2019-11-06T03:59:46+00:00**

Author: ****

Original: [CHAMPIONAT.asia](https://championat.asia/uz/news/buskets-sogligi-bilan-bogliq-muammoga-duch-keldi)

Бускетс ошқозонидаги муаммо сабаб “Славия” билан баҳснинг 68-дақиқасида майдонни тарк этди. У ўйин давомида бош айланиш ҳолатига дуч келган. Шундан сўнг шифокорлар унга ўйинни сўнгига қадар майдонда қолишга рухсат бермаган. Унинг ўрнига ўйинни Иван Ракитич давом эттирди.
