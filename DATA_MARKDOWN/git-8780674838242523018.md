
# അമ്മയുടെ മുന്നിലിട്ട് മകനെ വെട്ടിയ കേസിൽ ഒരാൾ കൂടി അറസ്റ്റിൽ

Published at: **2019-11-05T21:52:00+00:00**

Author: **മനോരമ ലേഖകൻ**

Original: [ManoramaOnline](https://localnews.manoramaonline.com/alappuzha/local-news/2019/11/05/alappuzha-criminal-arrest.html)

കായംകുളം ∙ കൃഷ്ണപുരത്ത് വീട് കയറി ആക്രമിച്ച് പ്രായപൂർത്തിയാകാത്ത മകനെ അമ്മയുടെ മുന്നിലിട്ട്  കഞ്ചാവ് സംഘം വെട്ടിയ കേസിലെ ഒരു പ്രതിയെ കൂടി കായംകുളം പൊലീസ് അറസ്റ്റ് ചെയ്തു. ഓച്ചിറ  വയനകം മാടവന കിഴക്കതിൽ പുത്തൻവീട്ടിൽ ഹരീഷിനെ (27)ആണ് എസ്ഐ കെ.സുനുമോന്റെയും, എസ്ഐ.സാമുവലിന്റെയും നേതൃത്വത്തിലുളള പൊലീസ് സംഘം അറസ്റ്റ് ചെയ്തത്.
കഴിഞ്ഞ ജൂലൈ രണ്ടിനു പുലർച്ചെയാണ് 11അംഗ സംഘം കൃഷ്ണപുരം ഞക്കനാൽ സ്വദേശിയായ പതിനേഴുകാരനെ മാരകമായി വെട്ടി പരുക്കേൽപിച്ചത്. സഹോദരനെ അന്വേഷിച്ചെത്തിയവരോട് ഇയാൾ വീട്ടിൽ ഇല്ലെന്നു പറഞ്ഞ് അമ്മ തിരിച്ചയച്ചിരുന്നു. പിന്നീടു സംഘമായി എത്തിഅമ്മയുടെ മുന്നിലിട്ടു മകനെ വെട്ടുകയായിരുന്നു.
