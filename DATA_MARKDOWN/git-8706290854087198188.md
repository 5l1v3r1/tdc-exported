
# Μητσοτάκης: Η δική μας γενιά οφείλει να δείξει τι πετύχαμε σε δύο αιώνες ελεύθερης ζωής - ΒΙΝΤΕΟ

Published at: **2019-11-07T12:10:00+00:00**

Author: ****

Original: [enikos.gr](http://www.enikos.gr/politics/682836/mitsotakis-i-diki-mas-genia-ofeilei-na-deixei-ti-petychame-se-dyo)

"Η δική μας γενιά οφείλει να δείξει τι πετύχαμε σε δύο αιώνες ελεύθερης ζωής" τόνισε ο Πρωθυπουργός Κυριάκος Μητσοτάκης μιλώντας στη Βουλή, κατά την έναρξη των εργασιών της Επιτροπής «Ελλάδα 2021».
"Οι κύκλοι της ιστορίας μοιάζουν να ειρωνεύονται τις ιστορίες των ανθρώπων", είπε. "Σήμερα καλούμαστε να επανατοποθετήσουμε την Ελλάδα στον κόσμο. Τότε ήταν η γέννηση, τώρα είναι η αναγέννηση".
"Προσδοκώ ότι τα 200 χρόνια από την ελληνική επανάσταση θα συνδεθούν με τα χρόνια που έχουμε μπροστά μας", επισήμανε ο κ. Μητσοτάκης.
