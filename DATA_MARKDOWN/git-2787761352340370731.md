
# Взрыв на танкере в Приморье

Published at: **2019-11-02T11:32:52+00:00**

Author: ****

Original: [Брянск Today](https://bryansktoday.ru/article/111775)

По неподтвержденным данным, на судне произошел хлопок газа, который находится в танках.
- Находкинская транспортная прокуратура с привлечением контролирующих органов организовала проверку по факту произошедшего, — сообщили в пресс-службе дальневосточной транспортной прокуратуры.
По данным ведомства, угрозы затопления судну нет, разлива нефтепродуктов не допущено.
