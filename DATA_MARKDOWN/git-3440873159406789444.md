
# 200 фур из-за бурана застряли на трассе в Актюбинской области

Published at: **2019-11-04T11:03:00+00:00**

Author: **Земфира Мусабаева**

Original: [Tengrinews.kz](https://tengrinews.kz/kazakhstan_news/200-fur-iz-za-burana-zastryali-trasse-aktyubinskoy-oblasti-383253/)

Около двухсот машин попали из-за бурана в снежный плен в Иргизском районе Актюбинской области, передает корреспондент Tengrinews.kz.
По сообщению КЧС МВД, затор образовался на участке с 1153 по 1240 километр автодороги Самара-Шымкент. Судя по фотографиям из социальных сетей, в заторе стоят в основном большегрузы.
