
# Женщина едва не лишилась кисти, подняв брошенную ей под ноги петарду

Published at: **2019-11-02T13:31:19+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-02/zhenshchina-edva-ne-lishilas-ruki-podnyav-broshennuyu-ey-pod-nogi-petardu)

РЕН ТВ стали известны обстоятельства ЧП во Владивостоке, где пострадала женщина. По информации РЕН ТВ, местная жительница Галина Ч. подняла брошенную ей под ноги петарду с надписью "Хеллоуин".
Предмет взорвался у нее в руках. Пострадавшую женщину с повреждением руки доставили в больницу, сейчас ей проводят операцию.
Инцидент произошел на Полярной улице возле школы. Правоохранители выясняют все обстоятельства произошедшего.
