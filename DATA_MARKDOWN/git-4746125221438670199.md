
# 佐々木みゆ『万引き家族』続編を是枝裕和監督に直談判「ちょっと考えてみようかな」

Published at: **2019-11-06T18:56:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148082/full/)

　『万引き家族』（2018年）で日本映画として21年ぶりに『カンヌ国際映画祭』の最高賞となるパルムドールを受賞した是枝裕和監督（57）が6日、都内で行われた最新作『真実』の特別編集版公開記念舞台あいさつに日本語版吹替を担当した佐々木みゆ（8）と登壇した。

        【写真】その他の写真を見る
      
　全編フランスにて撮影された本作。第17回カンヌ国際映画祭でグランプリを受賞した『シェルブールの雨傘』（63年）で主演を務めたカトリーヌ・ドヌーヴが、自身とも重なるような国民的大女優・ファビエンヌ役を演じる。彼女の娘・リュミール役には『イングリッシュ・ペイシェント』（96年）でアカデミー賞助演女優賞を受賞したジュリエット・ビノシュ。吹替版では、宮本信子がファビエンヌ、宮崎あおいがリュミール、佐々木がリュミールの娘・シャルロットの声を担当している。特別編集版は、本作に登場する女性たちを支え、見守る男性陣によりスポットを当てたもの。 　『万引き家族』にも出演していた佐々木は、MCから「次は是枝監督のどんな映画に出たいか？」と聞かれると「『万引き家族2』に出させてください」と回答。是枝監督は「クランクアップの日から2を撮りたいと言っているので、ちょっと考えてみようかな」と佐々木の思いに応える意欲も見せた。 　次回作について佐々木から是枝監督に質問がとぶと、是枝監督は「まだ全然決めてなくて、ちょっとお休みしています。いつもは次の準備をしているんだけど休憩中だから、一緒に遊ぼうね！」と話していた。 ※宮崎あおいの「崎」は「たつさき」が正式表記
