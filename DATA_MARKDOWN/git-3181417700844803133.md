
# В Ржеве начался новый этап работ по созданию мемориала Советскому солдату

Published at: **2019-11-01T12:54:00+00:00**

Author: ****

Original: [Тверские ведомости](https://vedtver.ru/news/society/v-rzheve-nachalsya-novyy-etap-rabot-po-sozdaniyu-memoriala-sovetskomu-soldatu/)

Фото: архив редакции
На площадке под Ржевом, где создаётся мемориал Советскому солдату, начался новый этап работ - монтаж каркаса для центральной фигуры. Вскоре на нём начнут закреплять первые бронзовые элементы – отлитые в Подмосковье фигуры журавлей.
