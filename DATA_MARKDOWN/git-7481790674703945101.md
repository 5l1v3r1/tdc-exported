
# ボルボ新型セダン、被害軽減ブレーキ装備…対向車を感知

Published at: **2019-11-05T07:52:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/economy/20191105-OYT1T50182/)

　ボルボ・カー・ジャパンは５日、新型セダン「Ｓ６０」を発売した。対向車を感知して作動する被害軽減ブレーキ機能を標準装備したほか、先代モデルより全高を４５ミリ低くし、スポーティーなデザインにした。ガソリン車とプラグインハイブリッド車（ＰＨＶ）があり、価格は消費税込みで４８９万～９１９万円。
　ボルボは月々の定額制で車を利用できる「サブスクリプション」と呼ばれる販売方法も強化すると発表した。Ｓ６０は、整備費や税金、諸経費など含めて月額７万３８１０円（頭金なし）から提供する。
