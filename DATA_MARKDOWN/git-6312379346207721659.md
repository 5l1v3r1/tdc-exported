
# В Сети появилось видео с места хлопка на танкере в Находке

Published at: **2019-11-02T04:45:57+00:00**

Author: ****

Original: [РЕН ТВ](http://ren.tv/novosti/2019-11-02/v-seti-poyavilos-video-s-mesta-hlopka-na-tankere-v-nahodke)

В Сети появилось видео с места взрыва на танкере в Находке. Жертвами инцидента стали три человека.
Тела двух погибших уже подняты на поверхность, ведутся поиски третьего.
По данным Следственного комитета РФ, в заливе Находка произошел хлопок газовоздушной смеси у мыса Астафьева. Горения не наблюдалось.
Дальневосточная транспортная прокуратура начала проверку по факту гибели моряков.
Отметим, что местные жители рассказали о взрыве, "от которого утром зашатались стены домов".
