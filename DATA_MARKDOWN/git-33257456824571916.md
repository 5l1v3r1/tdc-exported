
# Xiaomi представила свои первые умные часы. Они могут звонить и проигрывать музыку без телефона

Published at: **2019-11-05T11:16:32+00:00**

Author: **iXBT.com**

Original: [Meduza](https://meduza.io/news/2019/11/05/xiaomi-predstavila-svoi-pervye-umnye-chasy-oni-mogut-zvonit-i-proigryvat-muzyku-bez-telefona)

Китайская компания Xiaomi представила свои первые умные часы Xiaomi Mi Watch по цене 185 долларов. Об этом сообщает iXBT.com.
Часы работают под управлением операционной системы MIUI for Watch, диагональ экрана составляет 1,78 дюйма, корпус выполнен из нержавеющей стали. Часы дают возможность совершать голосовые звонки без смартфона (eSIM), оснащены модулем NFC и собственными музыкальным и видеоплеером. Также гаджет можно использовать для управления всеми устройствами «умного дома».
