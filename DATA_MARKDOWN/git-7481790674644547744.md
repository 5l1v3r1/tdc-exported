
# 北朝鮮の非核化「遅すぎる」米国務長官が不満

Published at: **2019-11-02T13:41:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/world/20191102-OYT1T50260/)

　【ワシントン＝海谷道隆】米国のポンペオ国務長官は１日の米ラジオのインタビューで、北朝鮮との非核化交渉について「あまりにも進み方が遅すぎる」と述べ、北朝鮮の消極姿勢に不満を示した。数か月で具体的な進展が得られるように交渉に注力する考えも表明した。
　ポンペオ氏は、「我々はトランプ大統領と金正恩（キムジョンウン）朝鮮労働党委員長が合意した北朝鮮の非核化を実現させるため、懸命に取り組んでいる」と訴えた。今後も引き続き交渉に取り組むとして、「数か月で良い結果が得られることを願っている」と語った。
　非核化に関する米朝の実務者協議は１０月５日にスウェーデンで行われたが、北朝鮮側は「決裂した」と主張し、再開のめどは立っていない。その後も、北朝鮮は弾道ミサイル発射などの挑発行為を繰り返している。
