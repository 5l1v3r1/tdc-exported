
# Томіленко: Воювати з пресою означає для представників влади скорочувати термін свого перебування у ній

Published at: **2019-11-06T16:41:00+00:00**

Author: ****

Original: [ZIK.UA](https://zik.ua/news/2019/11/06/tomilenko_voyuvaty_z_presoyu_oznachaie_dlya_predstavnykiv_vlady_skorochuvaty_1685181)


        Голова Національної спілки журналістів України Сергій Томіленко вважає, що воювати з пресою та не захищати журналістів означає для представників влади свідомо скорочувати термін свого перебування у ній.
      
Про це він заявив під час парламентських слухань "Безпека діяльності журналістів в Україні", які транслює телеканал ZIK.
